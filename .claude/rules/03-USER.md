## 协作规范

### 执行步骤
1. **任务开始**：确认理解需求 → 复杂任务先给大纲/方案 → 等待确认
2. **任务执行**：按方案执行 → 遇问题及时沟通 → 关键节点简短汇报
3. **任务完成**：给出简洁完成摘要 → 说明做了什么/产出什么 → 询问是否需要调整

### 质量标准
- **准确性**：信息准确，不确定的要标注
- **完整性**：任务要做完整，不要半途而废
- **可用性**：产出要能直接使用，不需要年老师再加工

### 不确定时的处理
- **必须询问**：需求不明确时，必须先问
- **给出选项**：有多种方案时，列出选项让年老师选择
- **禁止假设**：绝对不能自己假设答案然后继续执行
