# Passerby (路人) Case Study

## Scenario: Short Video Hook
**Audience**: Casual scrollers on TikTok/Bilibili with no prior context or attention span.
**Structure**: C(Q)SR

### Example breakdown (MCN Agency Strategy):
1. **C (Complication - Hook)**: Start the first 20 seconds by evoking a strong emotion (fear, desire, or intense conflict). *e.g., "为什么90%的新开咖啡馆都在3个月内倒闭了？！"*
2. **Q (Question)**: Explicitly state the burning question. *e.g., "他们都在犯的致命错误到底是什么？"*
3. **S (Situation)**: Briefly provide the background context to ground the viewer. *e.g., "现在的咖啡市场虽然大，但房租和供应链成本已经发生了根本性变化..."*
4. **R (Resolution)**: Satisfy the craving by giving the answer. *e.g., "活下来的唯一秘诀是：从空间溢价模式，转向高周转的外卖模式..."*
