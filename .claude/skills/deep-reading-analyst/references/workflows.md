# deep-reading-analyst 工作流

> **何时读取**：需要执行深度阅读分析任务时


## 工作流决策树

```
用户提供内容
    ↓
询问：目标 + 深度 + 偏好框架
    ↓
选择深度级别
    ↓
执行对应工作流
```


## 深度级别选择

| 级别 | 时间 | 框架 | 适用场景 |
|------|------|------|---------|
| **Level 1** 快速 | 15min | SCQA + 5W2H | 快速了解、日常阅读 |
| **Level 2** 标准 | 30min | L1 + 批判 + 逆向 | 学习参考、方案评估 |
| **Level 3** 深度 | 60min | L2 + 模型 + 系统 + 六帽 | 深度研究、决策支持 |
| **Level 4** 研究 | 120min+ | L3 + 跨源对比 | 论文综述、竞品分析 |


## Level 1 快速分析（15 分钟）

### 步骤

1. **结构理解**（5min）
   - 核心论点（一句话）
   - 结构概览（主要论据）
   - 关键概念（3-5 个）

2. **SCQA 分析**（5min）
   - S：背景
   - C：冲突/问题
   - Q：核心问题
   - A：答案/方案

3. **5W2H 检查**（5min）
   - 已覆盖：标注✅
   - 部分覆盖：标注⚠️
   - 缺失：标注❌

### 输出模板

```markdown
## 核心论点
[一句话概括]

## SCQA 结构
- S: [背景]
- C: [冲突]
- Q: [问题]
- A: [答案]

## 信息完整性
✅ 已覆盖：[What, Why, How]
⚠️ 部分覆盖：[Who, When]
❌ 缺失：[Where, How much]

## TOP 3 洞察
1. [洞察 1]
2. [洞察 2]
3. [洞察 3]

## 立即行动
[1 个可立即执行的行动]
```


## Level 2 标准分析（30 分钟）

### 步骤

在 Level 1 基础上增加：

4. **批判性思维**（10min）
   - 论证强度评分（1-10）
   - 论据质量评估
   - 逻辑漏洞识别
   - 替代视角

5. **逆向思考**（5min）
   - 如何确保失败？
   - 哪些假设可能错误？
   - 缺失的风险？

### 输出模板（增加部分）

```markdown
## 批判性分析

### 论证强度：[X/10]

**优势**：
- [优势 1]
- [优势 2]

**弱点**：
- [弱点 1]
- [弱点 2]

**逻辑谬误**：
- [如有]

## 逆向分析

### 失败模式
1. [失败模式 1] → 预防：[...]
2. [失败模式 2] → 预防：[...]

### 缺失风险
- [风险 1]
- [风险 2]
```


## Level 3 深度分析（60 分钟）

### 步骤

在 Level 2 基础上增加：

6. **思维模型**（15min）
   - 选择 3-5 个相关学科模型
   - 每学科视角分析
   - 跨模型洞察

7. **第一性原理**（10min）
   - 识别核心假设
   - 验证假设有效性
   - 回归基本原理

8. **系统思考**（10min）
   - 绘制关系图
   - 识别反馈回路
   - 寻找杠杆点

9. **六顶思考帽**（10min）
   - 六视角全面分析

### 输出模板（增加部分）

```markdown
## 多学科视角

### 思维模型应用
1. **[经济学：机会成本]**
   洞察：[...]

2. **[心理学：损失厌恶]**
   洞察：[...]

3. **[生物学：生态位]**
   洞察：[...]

**跨模式洞察**：[综合发现]

## 第一性原理

**核心假设**：
1. [假设 1] → 有效：[是/否/条件]
2. [假设 2] → 有效：[是/否/条件]

**基本原理**：[剥离假设后的真理]

## 系统图

```
[变量 A] ←增强— [变量 B]
    ↑           |
    |           |
  调节        增强
    |           |
    └────<──────┘

**杠杆点**：[小改变大影响的位置]

## 六顶思考帽

🤍 事实：[客观数据]
❤️ 感受：[直觉反应]
🖤 风险：[下行因素]
💛 收益：[上行潜力]
💚 创意：[替代方案]
💙 流程：[元思考/下一步]
```


## Level 4 研究分析（120 分钟+）

### 步骤

在 Level 3 基础上增加：

10. **跨源对比**（30min+）
    - 搜索 2-3 个相关来源
    - 提取各来源 SCQA
    - 对比共识与分歧
    - 综合形成完整视角

### 输出模板（增加部分）

```markdown
## 多源对比

### 来源 1：[本文]
SCQA：[概括]
核心主张：[...]

### 来源 2：[搜索结果 1]
SCQA：[概括]
核心主张：[...]

### 来源 3：[搜索结果 2]
SCQA：[概括]
核心主张：[...]

## 综合分析

**共识**：[各方同意的点]
**分歧**：[各方不同的点]
**独特价值**：[各来源的贡献]
**整合视角**：[你的综合结论]
```
