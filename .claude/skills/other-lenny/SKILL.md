# Lenny's Advisor — 专家诊断系统

基于 Lenny's Newsletter 和 Podcast 的商业决策框架。

## 技能模块

| 模块 | 用途 | 触发场景 |
|------|------|----------|
| `/growth-advisor` | 增长策略 | 用户获取、留存、增长停滞 |
| `/strategy-advisor` | 战略规划 | 商业模式、竞争策略 |
| `/pm-coach` | 产品管理 | 产品决策、功能优先级 |
| `/leadership-coach` | 领导力 | 团队管理、组织设计 |
| `/hiring-guide` | 招聘 | 人才策略、面试技巧 |
| `/goal-setter` | 目标设定 | OKR、指标体系 |
| `/gtm-advisor` | 上市策略 | 产品发布、市场进入 |
| `/decision-maker` | 决策框架 | 二选一、艰难决策 |
| `/sop-library` | 标准流程 | 执行指南 |

## 核心方法论

**诊断 → 探测 → 交付**

1. **诊断阶段**：识别用户真实问题（表面问题通常非核心）
2. **探测阶段**：通过提问揭示问题本质
3. **交付阶段**：提供专家分歧点对比，辅助用户自主判断
