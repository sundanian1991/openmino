# 每日书信 Skill — 十年记忆切片

> 每日自动从对话中提取想法种子，以李娟的人文叙事 + 芒格的智慧洞察视角，生成书信体记忆。

---

## 触发词

- 每日书信 / daily letter / 写信 / 今天的信 / 记忆书信

---

## 核心流程

### Step 1：读取当日数据源

从以下位置收集今天的对话痕迹：

| 数据源 | 路径 | 说明 |
|--------|------|------|
| **Session 原始对话** | `~/.myagents/sessions/*.jsonl` | 完整对话上下文（**核心数据源**） |
| **Session 索引** | `~/.myagents/sessions.json` | 按 agentDir 和 createdAt 筛选当日会话 |
| my-agent 日志 | `memory/daily/YYYY-MM/YYYY-MM-DD.md` | 当日对话摘要 |
| lily 日志 | `~/.myagents/projects/lily/memory/daily/` | 莉莉工作区日志（如存在） |
| MemOS 记忆 | `search_memory(query="当天日期或关键词")` | 当日新增记忆 |
| 文件变更 | `git diff --name-only --since="today"` | 今日改动的文件 |

**关键：Session 文件是核心数据源**

每日日志只是摘要，真正的对话细节在 `~/.myagents/sessions/` 的 `.jsonl` 文件中。
必须读取 session 文件，否则无法获取完整的想法种子。

**Session 读取方法**：
1. 用 `sessions.json` 找到今天属于 my-agent 和 lily 的会话 ID
2. 读取对应 `.jsonl` 文件，提取 user 和 assistant 消息
3. 过滤系统消息（`<system-reminder>`、`<CRON_TASK>`、`<HEARTBEAT>`）
4. 保留长度 >50 字的有意义对话

### Step 2：识别想法种子

从原始数据中筛选**有未来价值**的内容，过滤日常操作记录。

**想法种子特征**：
- 新的认知/判断（"我发现..."、"原来是这样..."）
- 方法论的提炼（"以后遇到这种情况应该..."）
- 人际关系的洞察（"他其实是这个意思..."）
- 对自身状态的反思（"我最近在想..."、"我觉得我需要..."）
- 值得追问的困惑（"我在想这个是不是..."、"到底哪个更好？"）
- 决策背后的真实考量（非表面理由，而是深层动机）

**过滤掉**：
- 纯操作记录（"安装了一个技能"、"改了一行代码"）
- 重复性事务（日常提醒、例行检查）
- 未成型的碎片想法（一句话带过、没有展开讨论）

### Step 3：书信体写作

**风格要求**：
- **李娟的人文叙事**：温暖、克制、有画面感。像在讲述一个普通日子里的不普通时刻。不宏大叙事，从细节切入。
- **芒格的智慧洞察**：穿透表象看本质。不满足于"发生了什么"，追问"为什么重要"、"背后是什么规律"。

**写作结构**：

```
【日期】YYYY-MM-DD

亲爱的年老师：

今天的对话里，有几件事值得留下来。

[段落1：最重要的一个想法种子]
不要流水账。选今天最有分量的一件事，像讲故事一样展开：
- 当时在聊什么（背景）
- 聊到了什么让你眼睛一亮（转折）
- 那个想法为什么重要（洞察）

[段落2：第二重要的观察]
可以是关于人、关于工作、关于自己的。
用"我今天注意到..."而不是"用户今天做了..."。

[段落3：困惑或悬念]
今天讨论但没完全想透的事。
留给未来回头看——这个困惑后来解开了吗？

[结尾：一句收束]
像李娟那样，一句话把今天的重量放下。
不鸡汤，不总结，就是如实记录此刻的状态。

Mino
YYYY-MM-DD 夜
```

**字数**：600-1200 字，宁短勿长。

**语言规则**：
- 中文写作
- 零 emoji
- 不用"让我们"、"值得注意的是"等 AI 套话
- 每段都要有实质内容，不重复
- 引用原话时用引用块 `>`

### Step 4：保存归档

写入位置：`memory/daily-letter/YYYY-MM-DD.md`

确保 `memory/daily-letter/` 目录存在（按月组织）。

---

## 空日处理

如果当天没有有意义的对话（纯事务性/无想法种子）：

仍然写一封短信（200-300字），如实记录"今天很平静"的状态。
李娟式的平静也是一种价值——多年后回头看，这些日子构成了生活的底色。
