# Design Philosophy — baoyu-slide-deck

> 设计理念与指南


## 核心设计哲学

**Decks designed for reading and sharing, not live presentation**:

- Each slide self-explanatory without verbal commentary
- Logical flow when scrolling
- All necessary context within each slide
- Optimized for social media sharing


## 受众特定原则 (Audience-Specific Principles)

| Audience | Design Principles |
|----------|-------------------|
| **Beginners** | More whitespace, larger fonts, step-by-step flow, explanatory captions |
| **General readers** | Balanced density, accessible language, visual metaphors |
| **Experts** | Higher data density, technical terminology, compact layouts |
| **Executives** | Key insights first, minimal detail, high-level metrics |


## 视觉层次 (Visual Hierarchy)

**优先级顺序**：
1. **主标题** — 每 slide 核心信息（不超过 10 个词）
2. **副标题/引言** — 补充说明（1-2 句）
3. **关键点** — 3-5 个 bullet points
4. **支撑内容** — 数据、图表、引用

**原则**：
- 每个 slide 只有一个焦点
- 使用尺寸、颜色、位置建立层次
- 避免"所有东西都重要"


## 内容密度指南 (Content Density)

| Density | 每 Slide 内容 | 适用场景 |
|---------|-------------|---------|
| **Minimal** | 1 个核心观点 + 大量留白 | 章节过渡、引用页、强调页 |
| **Balanced** | 2-3 个关键点 + 支撑数据 | 标准内容页 |
| **Dense** | 4-6 个要点 + 多个数据点 | 数据汇总、对比页、技术细节 |

**建议**：
- 全文 80% 使用 Balanced
- 10% Minimal（过渡、强调）
- 10% Dense（数据汇总）


## 颜色选择原则

**基于 Mood 的颜色策略**：

| Mood | 主色调 | 强调色 | 使用场景 |
|------|-------|-------|---------|
| **professional** | 深蓝、灰 | 金色、白色 | 企业、金融、咨询 |
| **warm** | 大地色系、橙色 | 棕色、米色 | 教育、生活方式 |
| **cool** | 蓝色、青色 | 灰色、白色 | 科技、医疗 |
| **vibrant** | 高饱和度多色 | 对比色 | 创意、营销 |
| **dark** | 深灰、黑色 | 霓虹色 | 娱乐、游戏 |
| **neutral** | 黑白灰 | 单强调色 | 极简、高管简报 |


## 排版选择原则

| Typography | 特征 | 适用场景 |
|------------|------|---------|
| **geometric** | 现代无衬线、干净 | 科技、SaaS、产品 |
| **humanist** | 友好、易读 | 教育、生活方式 |
| **handwritten** | 手写体、有机感 | 创意、教程 |
| **editorial** | 杂志风格、戏剧性 | 营销、发布会 |
| **technical** | 等宽、代码感 | 开发者文档、学术 |
