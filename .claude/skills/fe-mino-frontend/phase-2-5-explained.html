Phase 2.5 审美判断力层

Canvas-Design 核心能力的深度吸收

什么是 Phase 2.5？

在风格选择(Phase 2)和代码生成(Phase 3)之间，插入设计哲学层 — 确保每个设计都有"灵魂"而非机械执行规范。设计模式提供技术规范（色彩值、字体、间距），设计哲学提供审美判断（为什么这样用、怎么用才有气质）。

三步流程

18维审美定位

  • 在8个坐标轴上精确定位
  • 不是"选盒子"，是"定坐标"
  • 温度、密度、权威感、时间感...
  • 无限可能的审美空间

2设计哲学宣言

  • 基于8维定位，输出500字哲学
  • 3句话格式：运动名称 + 空间/形式
  • + 色彩/情绪 + 工艺标准
  • 每次设计都有独特的美学定位

3工艺检查清单

  • 视觉5项：呼吸感/对齐/对比/层级/无AI感
  • 信息3项：3秒法则/数据诚实/文字极简
  • 技术3项：视口适配/响应式/性能

Step 1: 8维审美定位（核心方法）

温度

6/10 偏冷

密度

极简（单点） 密集（系统图）
5/10 中等

权威感

平等（对话） 权威（发布）
4/10 中低

时间感

永恒（经典） 当下（流行）
7/10 当下

精度

有机（手绘） 精确（像素）
8/10 高精度

戏剧性

克制（隐藏） 张扬（表达）
3/10 低

材质感

平面（数字） 物质（纸张）
5/10 轻微

色彩策略

单色 多色 / 无彩色
6/10 低饱和多色

8维定位示例 对应设计决策

Step 2: 设计哲学宣言（示例）

实验室仪器 — 数据即证据

哲学核心：这是内部管理工具，不是对外展示品。像实验室仪器，功能性优先，装饰性让位于信息传达。

空间与形式：信息密度中等，保留呼吸空间。数据排列精确，像科学记录，不是艺术表达。每个像素都有目的。

色彩系统：低饱和多色区分维度，但克制使用。色彩是信息的区分工具，不是情绪的表达工具。

工艺标准：每个选择都能说出"为什么"。没有装饰，没有多余，每个元素的存在都服务于可读性和精度。

Step 3: 11项工艺检查清单

视觉工艺（5项）
  • 呼吸感 — 元素周围有足够留白
  • 对齐精度 — 所有元素视觉对齐
  • 层次清晰 — 一眼看懂主次
  • 色彩克制 — 每种颜色都有明确目的
  • 无AI感 — 无渐变滥用、阴影堆砌
信息工艺（3项）
  • 3秒法则 — 打开3秒内理解重点
  • 数据诚实 — 图表比例准确
  • 文字极简 — 能用视觉不用文字
技术工艺（3项）
  • 视口适配 — 完美适配视口
  • 响应式 — 多高度兼容
  • 性能 — 无不必要的动画

三重验证机制

结构层验证

  • 工作流必经关卡
  • Phase 2 → Phase 2.5 → Phase 3 唯一路径
  • 无法跳过审美判断直接生成代码
  • 结构上的必然性

输出层验证

  • 每个设计携带哲学签名
  • HTML注释：Philosophy/Method/Standard
  • 可反向追问验证
  • 随时回答"概念线索是什么"

质量层验证

  • 11项工艺检查逐项通过
  • 不符合标准必须返工
  • 工艺是底线，不是可选
  • 每次输出符合博物馆级标准

"真正的吸收 = 可验证的依赖。不穿越 Phase 2.5，无法生成设计。"
