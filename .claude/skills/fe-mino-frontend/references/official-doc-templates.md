## 5维决策速查

```
内容分析 → 提取关键词 → 确定情绪 → 选择模板 → 生成
```

| 情绪关键词 | 模板 | Header颜色 |
|-----------|------|-----------|
| 汇报、评估、战略、年度、金融 | 青绿报告 | `#0d7680` |
| 通知、清退、处罚、整改、禁止 | 红头通知 | `#8B0000` |
| 欢迎、庆祝、激励、突破、成功 | 陶土欢迎 | `#E2725B` |

## 1. 青绿报告

### 5维特征

| 维度 | 选项 |
|------|------|
| 信息结构 | 洞察 + 排名 |
| 情绪气质 | 权威 |
| 信息密度 | 中 |
| 视觉元素 | 简单图标 |
| 色彩倾向 | 冷 |

### 设计参数

```css
/* Header */
--header-bg: #0d7680;
--header-accent: #14a3b0;
--header-top-line: 4px;

/* 强调色 */
--accent: #0d7680;
--accent-light: rgba(13, 118, 128, 0.1);

/* 表格 */
--table-header-bg: #0d7680;
--table-header-color: #fff;

/* KPI栏 */
--kpi-bar-bg: #f4f9f9;
--kpi-bar-border: 2px solid #0d7680;
```

### 版式结构

```
┌─────────────────────────────────┐
│ Header（青绿 + 4px上划线）        │
│ 标题 + 副标题 + 日期              │
├─────────────────────────────────┤
│ KPI栏（4宫格数据概览）            │
├─────────────────────────────────┤
│ Section 1: 执行摘要               │
│ ├─ 结论框（核心判断）             │
│ └─ 关键指标卡片                   │
├─────────────────────────────────┤
│ Section 2: 详细分析               │
│ ├─ 数据表格（斑马纹）             │
│ └─ 趋势图表（可选）               │
├─────────────────────────────────┤
│ Section 3: 建议与后续             │
│ └─ 行动项列表                     │
├─────────────────────────────────┤
│ Footer（青绿底 + 部门信息）        │
└─────────────────────────────────┘
```

### 适用场景

- `formal-report` - 正式汇报、专项汇报
- `supplier-evaluation` - 季度评估、年度评估
- `supplier-reserve` - 储备评估、准入建议
- `supplier-allocation` - 分量调整说明


```
│ Body                             │
│ ├─ 导语（一句话说明）             │

│ ├─ 附件清单（如有）               │
│ └─ 执行要求                      │
├─────────────────────────────────┤
│ 落款（右对齐）                    │

│ └─ 印章位置（预留）               │
├─────────────────────────────────┤
│ 抄送（如有）                      │
└─────────────────────────────────┘
```

### 适用场景

- `notification` - 会议通知、变更通知、紧急通知

- `punishment-dispute` - 处罚通知、整改要求

## 附录：模板选择决策树

```
需要前端展示？
    ├─ 是 → 选择以上9个模板
    │       ├─ 向上汇报/评估 → 青绿报告
```
