## 字体选择决策树

```
什么场景？
├─ 商业/咨询汇报 → 严肃咨询(01) / 高管大字报(08)
├─ 新闻/对外发布 → 编辑风格(04) / 新闻速写(07)
├─ 学术/论文 → 学术严谨(05)
├─ 科技产品 → 现代科技(06)
├─ 仪表板/看板 → 数据终端(03) / 紧凑仪表板(11)
├─ 数据故事 → 数据散文(10) / 亲和数据(02)
├─ 脑暴/草图 → 手绘草图(09)
├─ 年度报告/出版 → 出版级精致(12)
└─ 团队内分享 → 亲和数据(02)

需要打印吗？
├─ 是 → 学术严谨(05) / 严肃咨询(01) / 出版级精致(12)
└─ 否 → 其余

必须用系统字体（无网络加载）？
├─ 是 → 严肃咨询(01)[Arial/Helvetica] / 编辑风格(04)[Georgia]
└─ 否 → 可用 Google Fonts 系列
```
