# Brand Guide — McKinsey PPT Design

> 团队级规范。适用于所有 PPT 项目，每次生成必须遵守。

## Color Palette

| Color Name | Hex | RGB | Usage |
|-----------|-----|-----|-------|
| **NAVY** | #051C2C | (5, 28, 44) | Primary: titles, circles, primary elements |
| **WHITE** | #FFFFFF | (255, 255, 255) | Backgrounds, text on navy |
| **BLACK** | #000000 | (0, 0, 0) | Lines, text separators |
| **DARK_GRAY** | #333333 | (51, 51, 51) | Body text, descriptions |
| **MED_GRAY** | #666666 | (102, 102, 102) | Secondary text, labels |
| **LINE_GRAY** | #CCCCCC | (204, 204, 204) | Table row separators |
| **BG_GRAY** | #F2F2F2 | (242, 242, 242) | Background panels, takeaway areas |
| ~~CYAN~~ | #00A9F4 | — | **DEPRECATED** — 不要使用 |

**Key Rule**: NAVY (#051C2C) 用于所有圆形标记、action titles、section headers、TOC 高亮。

### Accent Colors（3+ 并列项时使用）

| Accent | Hex | 配对浅色背景 | 用途 |
|--------|-----|------------|------|
| ACCENT_BLUE | #006BA6 | #E3F2FD | 第1项 |
| ACCENT_GREEN | #007A53 | #E8F5E9 | 第2项 |
| ACCENT_ORANGE | #D46A00 | #FFF3E0 | 第3项 |
| ACCENT_RED | #C62828 | #FFEBEE | 第4项/警告 |

**规则**：只有幻灯片有3+并列项时才用 accent colors。卡片内 body text 始终用 DARK_GRAY。

## Typography

```
英文标题：Georgia (serif)
英文正文：Arial (sans-serif)
中文全部：KaiTi 楷体（备选：SimSun 宋体）
```

### 字号层级（严格遵守，不得使用其他字号）

| 字号 | 类型 | 使用场景 |
|------|------|---------|
| 44pt | 封面大标题 | Cover slide only，Georgia |
| 28pt | 章节标题 | TOC title，Georgia |
| 24pt | 副标题 | Cover subtitle |
| 22pt | Action Title | 所有内容页标题，bold，Georgia |
| 18pt | Sub-Header | 栏头、章节名 |
| 16pt | 强调文字 | 底部 takeaway，bold |
| 14pt | 正文 | **PRIMARY SIZE**，所有主体内容 |
| 9pt | 脚注 | Source 出处，仅用灰色 |

## Design Philosophy

1. **极简** — 无阴影、无3D、无渐变、无装饰性色块
2. **一致** — 同类内容同字号同颜色，全局统一
3. **层级** — 标题22pt → 栏头18pt → 正文14pt → 脚注9pt
4. **平面** — 纯实色填充，无任何效果
