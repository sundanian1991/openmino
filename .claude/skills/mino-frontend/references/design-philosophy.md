# 设计哲学与内容基因

> **用途**：深度设计时的参考方法论。mino-frontend 默认不执行这些步骤，只在需要时读取。
>
> **触发场景**：用户说"深度设计"、"帮我选风格"、或内容复杂需要额外设计判断时。

---

## 一、内容基因提取（Content DNA）

> **来源**：canvas-design 的 Conceptual DNA 方法论

### 核心区别

- Phase 1 识别内容"有什么"（数据/流程/清单）
- Content DNA 识别内容"是什么气质"

同样的"供应商排名"，"垄断困境排名"和"健康竞争排名"应走向完全不同的设计方向。

### 提取四维 DNA

| 维度 | 问题 | 输出 |
|------|------|------|
| **情绪向量** | 内容的核心情绪是什么？ | 紧迫/沉稳/庆祝/反思/好奇/对抗 |
| **视觉重量** | 视觉上应该"重"还是"轻"？ | 重（压迫感/权威感）/ 轻（呼吸感/友好感） |
| **叙事节奏** | 内容的节奏是快还是慢？ | 快（多信息高密度）/ 慢（少信息大留白）/ 阶梯（逐步深入） |
| **视觉隐喻** | 有没有可以转化为视觉的抽象概念？ | 金字塔/战场/旅途/迷宫/里程碑/信号灯/树/流 |

### 提取方法

1. **情绪向量** — 读完全部内容后，第一情绪反应是什么？不是"分析情绪"，是"感受情绪"。
2. **视觉重量** — 如果这些内容有物理重量，它重吗？沉重 = 视觉重（大色块、粗线条、深色系），轻松 = 视觉轻（大留白、细线条、浅色系）。
3. **叙事节奏** — 内容像什么节奏的音乐？快 = 多信息紧凑排列，慢 = 大留白单焦点，阶梯 = 分阶段逐步深入。
4. **视觉隐喻** — 内容中有没有可以被图形化的概念？竞争 = 战场/金字塔，成长 = 旅途/里程碑，风险 = 信号灯/迷宫。

### 提取示例

**刘乾坤带节奏分析**：对抗 + 重 + 阶梯 + 迷宫 → "诊断式解剖"
**游泳训练计划**：好奇 + 轻 + 快 + 旅途 → "友好引导"
**供应商排名（健康）**：庆祝 + 轻 + 快 + 领奖台 → "轻松专业"
**供应商排名（垄断）**：紧迫 + 重 + 阶梯 + 金字塔 → "冷峻诊断"

## 二、内容基因 × 风格推理

| 基因组合 | 推荐风格 | 典型场景 |
|---------|---------|---------|
| 对抗 + 重 + 阶梯 + 迷宫 | 克制专业主义 / 咨询模式 | 行为分析、问题诊断 |
| 好奇 + 轻 + 快 + 旅途 | 品牌模式 / 柔和几何 | 学习指南、入门教程 |
| 庆祝 + 轻 + 快 + 领奖台 | 金融时报 / 极简现代 | 成果展示、排名报告 |
| 紧迫 + 重 + 快 + 信号灯 | 数据模式 | 风险预警、异常监控 |
| 沉稳 + 中 + 慢 + 深度 | 纸墨文学 / 复古报刊 | 深度分析、年度回顾 |
| 好奇 + 中 + 阶梯 + 树 | Anthropic 品牌 | 技术文档、知识库 |

## 三、设计哲学宣言

选定设计模式后，用 3 句话定义美学方向：

**格式**：
```
[运动名称] — [1-2 个词的美学定位]

1. 空间与形式：这一设计如何处理空间关系
2. 色彩与情绪：色彩如何服务于概念线索
3. 工艺标准：这一设计应该给人什么质感
```

**示例**（金条供应商排名，金融时报模式）：
```
竞技场 — 数据即战场

1. 空间与形式：金字塔结构，头部供应商占视觉重心，留白制造距离感
2. 色彩与情绪：青绿色代表冷静判断，暖米色保持权威感，陶土色仅用于异常值
3. 工艺标准：像 FT 数据可视化一样精确、克制、值得信赖
```

## 四、空间使用法

### 内容类型 → 位置模型

| 模型 | 划分方式 | 适用内容 |
|------|----------|---------|
| **中心模型** | 中心 + 边缘 | 金句、结论、单一焦点 |
| **上下模型** | 顶部 + 主体 + 底部 | 标题+内容+结论 |
| **左右模型** | 左侧 + 右侧 | 对比、因果关系 |
| **网格模型** | 2-3 列均分 | 清单、表格、多模块 |

### 视觉层级 → 权重分配

**1 层**：金句/结论 100% 居中，四周留白 >60%
**2 层**：主 70% + 次 30%，或 60:40 对比
**3 层**：标题 8% + 内容 65% + 结论 27%

### 核心原则

```
中心区域 = 核心内容
边缘区域 = 边界信息
四角区域 = 元信息（页码、日期、来源）
层级之间 = 大间距（≥24px）
层级内部 = 小间距（≤12px）
焦点元素 = 周围留白最多
```

## 五、极致工艺检查清单

**视觉**：呼吸感 / 对齐精度 / 层次清晰 / 色彩克制 / 无 AI 感
**信息**：3 秒法则 / 数据诚实 / 文字极简
**技术**：视口适配 / 响应式 / 性能

---

*设计哲学 vs 设计规范：规范说"用 #E2725B，字号 28px"，哲学说"陶土色是唯一的温度来源，大字号是自信的表达"。规范保证一致性，哲学保证气质。*
