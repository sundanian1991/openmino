# 人物观察分析

> **"表面看他说什么，深层看他为什么这么说"**

---

- 年老师提供对话记录，要求分析某人行为/潜台词

### 3. 深度分析（核心）

#### 表面行为拆解
- 逐句或逐段分析"说了什么"
- 识别具体的话术和表达方式
- 注意语气、用词、时机

#### 潜台词解读
- **为什么这么说？** — 识别潜在动机
- **真实意图是什么？** — 看穿表面客气的实际施压/展示
- **对谁说？谁在看？** — 分析"观众意识"

#### 行为模式匹配
对照已知模式，判断是哪种手法：

| 模式 | 特征 | 典型话术 |
|------|------|----------|
| **抛问题** | 只出数据不出结论，让别人背锅 | "帮我看下，我该如何解答" |
| **表演型同步** | 全程有观众意识，向上展示 | 5分钟连发两条、CC上级 |
| **公开锁定** | 群里@人，制造公开压力 | 不私聊，直接在群里问 |
| **软硬施压** | 阴阳怪气+公开追问切换 | "您"+"帮我看下"+@追问 |
| **正面对冲** | 用好消息平衡坏消息 | 先表扬再提要求 |
| **任务下放+CC展示** | 给下属任务+向上展示培养 | "我们一起"+CC上级 |
| **责任转移** | 成了是"我们一起"，没成是"他们还没" | 分散责任风险 |

判断是：
- **深化**：同一模式的更极致表现
- **变种**：在不同关系/场景中的变体
- **补充**：发现全新的行为模式

#### 关系网络分析
- 这个行为在不同关系中的呈现
- 与其他角色的互动模式
- 识别权力动态和利益同盟

## 核心能力

### 1. 潜台词识别
- 区分"表面客气"vs"实际施压"
- 识别"责任转移"的各种话术
- 看懂"向上展示"的时机和方式
- 注意"观众意识"——他在让谁看到这个行为？

### 2. 模式提炼
- 从单个案例到通用模式
- 识别行为的一致性和变化
- 建立人物的行为标签体系
- 判断新模式与已有模式的关系

### 3. 关系网络分析
- 理解人物在组织中的位置
- 分析与不同角色的互动模式
- 识别权力动态和利益同盟

### 4. 档案管理
- 结构化存储人物信息
- 版本化追踪观察记录
- 支持跨案例对比分析

## 索引机制优势

1. **自动上下文**：每次触发技能自动读取最近观察，无需重复说明
2. **时间结构**：时间倒序自动组织，最新观察置顶
3. **持续追踪**：长期观察自然沉淀，不活跃内容自动下沉
4. **快速回顾**：一目了然看到最近观察和待观察事项

## 分析原则

1. **不轻易下结论**：说"可能是"而非"一定是"
2. **证据说话**：每个判断都要有对话内容作为支撑
3. **关系敏感**：同样的话对不同关系可能有不同含义
4. **演变追踪**：关注人物行为模式是否有变化
