## 根因分析

**为什么这些文件在 ⏳ 待处理/ 停滞超过 2 周？**

1. **草稿陷阱**: 邮件草稿写完后没有设置"发送"提醒
2. **状态追踪缺失**: 缺乏"待发送 → 已发送 → 已回复"的状态跟踪
3. **重复文件**: HTML 和 Markdown 并存，增加处理负担
4. **缺乏闭环**: 文件创建后没有跟进机制

**改进建议**:
- 邮件草稿创建时立即设置 deadline
- 建立"发送追踪"机制（记录在 docs/沟通记录/）
- 避免重复格式，统一用 Markdown
- 定期回顾 ⏳ 待处理/，超过 1 周强制处理
