Plan

两周学习路线

两个阶段，从 Agent/MCP 到安全与审查，每个阶段包含阅读和实践任务。

Week 1 · Agent + MCP
改进 Subagent 策略与 MCP 配置
明确自主级别，优化 MCP 配置并添加验证脚本。阅读 Anthropic 内部指南、Claude Code Best Practices、Specs Are the New Source Code。
Week 2 · 安全 + 审查
Semgrep 安全扫描与 AI 代码审查
集成 Semgrep 到工作流，改进代码审查流程：不仅是语法检查，还包括逻辑和安全性审查。
