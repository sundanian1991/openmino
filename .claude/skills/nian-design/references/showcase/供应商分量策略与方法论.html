供应商分量策略
与拿捏方法论

从"管人"到"经营生态"的能力跃迁。不是分蛋糕，是把合适的供应商放在合适的地方。

SM
01 · 供应商画像
每一个维度都是决策坐标
对每家供应商建立多维画像——配合度、业务结构、学习能力、业务经验、上人速度、历史表现。
维度分类用途
配合度高 / 中 / 低决定信任度和信息开放程度
业务结构头部 / 中部 / 尾部决定分量权重和期望值
学习能力强 / 中 / 弱决定新项目试水意愿
业务经验拉新 / 企微 / 大额 / 创新型决定业务匹配度
上人速度快 / 中 / 慢决定应急响应能力
历史表现稳定 / 波动决定风险预判

核心观察

独特之处
以业绩论英雄——不管背景、不管历史关系，只看数据。
规则运行在"阳光"下——相对透明，供应商知道规则、知道标准。
口碑效应——做好就有，躺平就没。供应商之间的口碑传播比我们的说教更有效。
对冲策略 · 降价时的处理
承诺"不是永远降，后面可能回调"— 给供应商希望
用自建团队数据作为客观基准："自建能做到 110% 增速，你们只有 45%"
一视同仁 — 所有供应商统一降价，不接受就不给业务
承诺"看好生命线"— 不会降到供应商活不下去

拿捏能力的终极体现
清退不是目的，是手段。四阶段递进，每阶段都给供应商回头的机会。

阶段触发条件具体动作供应商感受
警告首次轻微违规口头/邮件警告，要求整改还有机会
约谈再次违规或整改不到位正式约谈，记录在案开始紧张
限制新进持续不改善不允许接新业务、新增人力真正痛了
清退严重违规或长期不达标启动清退流程最后通牒
限制新进的杀伤力
收入增长被锁死 · 产能只能持续流失 · 不主动清退，让供应商"慢慢流血"
给供应商两难选择：闹→被拉黑 vs 走→日后好相见。
无损退出技巧
告知公司制度，说明行业影响。承诺如果自己申请，不加黑名单。利用黑名单行业威慑力，给台阶下——"日后好相见"。

07 · 总结
供应商管理的本质不是"管"，
而是"经营"
五大能力共同构成一个自我运转的供应商管理生态系统。
能力一
分量策略
经营资源分配，让每家供应商都有动力。把合适的供应商放在合适的地方。
能力二
拿捏能力
经营关系平衡，让供应商在有限奖励下依然愿意投入。胡萝卜+大棒，业绩论英雄。
能力三
对标能力
经营效能基准，让"做不到"有据可查。自建团队是检视供应商的客观标尺。
能力四
培育能力
经营能力提升，让新供应商快速出产能。培育新商就是培育未来的竞争。
能力五
进化能力
经营知识沉淀，让经验变成可复制的制度。方法论本身是最大的杠杆。
核心认知
可替代的供应商管理只用"管"，不可替代的供应商管理在"经营"——
经营资源分配、经营关系平衡、经营效能基准、经营能力提升、经营知识沉淀。
