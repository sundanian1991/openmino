P1 · Dualism — 双元对比
01
Heritage Future

111 年的历史不是束缚，是基座。每款新产品必须同时回答：它尊重了什么传统？突破了什么边界？

02
Organic Engineered

自然的形态通过精确的几何被提炼。每一个弧度都有功能理由——不是为了好看，是为了好用。

03
Nature Tech

泥土色的基底上，用精准的橙色信号做功能提示。有机的轮廓里，嵌入工程优化的结构。

04
Warmth Precision

功能上的精确性和情感上的温度必须同时存在。大到戴手套也能操作的拉链头，形状要让人愿意触碰。

P2 · Separated Fields — 分隔场域
F
Form Follows Function
每个组件的形态由其功能决定。装饰不是加法——是未被发现的减法。去掉，直到不能再去掉。
G
Grid as Rhythm
网格不是牢笼。8px 是步长，不是精度。用间距而非分割线建立层级——野外你不会在开阔地和营地之间画线。
T
Typography as Voice
三种字体，三种角色，永不混用。Georgia 是宣言，Inter 是叙述，JetBrains Mono 是证据。每页不超过 3 字号 2 字重。
P3 · Color Pillars — 色彩编码
Nature · 户外 DNA
olive
#4A5D3A — 品牌主色，承载户外基因与可持续理念。80% 视觉面积由此色系定义。
Material · 工艺温度
earth
#8B7355 — 辅助品牌色，传递材质温暖与自然纹理。用于工艺展示和历史叙事。
Tech · 技术精确
glacier
#2A4A5A — 场景色，代表冬季/高海拔/技术。用于数据面板和技术规格展示。
P4 · The Fifth Element — 第五元素

点阵 H 是 Haglöfs 品牌的 第五元素——一个贯穿系统的签名视觉符号。从 Hero 区到数据卡片，从装订线到页脚，它不做装饰，只做标记。在 OPPO 品牌中，SDL 设计了视觉界面模块（Visual Interface Module）作为第五元素；在 Stadium 中，它是一组运动数字。每个品牌系统都需要一个属于自己的"第五元素"。

Synthesis · SDL Design Principles
阅读 + 分析 · 核心结论
4
01

Dualism — 双元对比是品牌能量的来源。四组张力对同时工作，不是妥协，是同时拉向相反方向的力量。

02

Separated Fields — 用间距建立层级，不用分割线。网格是步长，不是精度。8px 步长允许 4px 或 12px 的偏离。

03

Color Coding — 颜色必须承载功能。80% brand primary + 10-15% scene + 5-10% accent。accent 仅做信号不做装饰。

04

The Fifth Element — 每个品牌需要一个贯穿系统的签名元素。点阵 H、视觉界面模块、一组数字——不做装饰，只做标记。
