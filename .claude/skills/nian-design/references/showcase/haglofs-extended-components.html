10 / Typography
字体完整展示
三种字体角色。Georgia = 展示层（品牌陈述、产品名、核心判断）。Inter = 内容层（正文、导航、描述）。JetBrains Mono = 数据层（规格、标签、编码、数字）。永不混用角色。
Font Roles — 字体角色分配
每种字体有明确的角色边界。Georgia 只做展示，Inter 只做内容，JetBrains Mono 只做数据。混用角色 = 系统失效。

11 / Extended Components
扩展组件
从工业设计系统移植并适配为户外品牌调性。保留结构，替换视觉语言。按钮、输入框、分段控制等标准 UI 组件。
Buttons — 按钮
Primary 用于主操作，Secondary 用于次操作，Ghost 用于取消，Accent 用于警告操作。所有按钮 44px 最小高度。
Inputs — 输入框
下划线样式为主，全边框为辅。Focus 态用 olive 边框，Error 态用 error 色边框+提示文字。
[Error: 请输入有效数字]
Segmented Control — 分段控制
用于场景切换（Forest/Alpine/Desert）或时间维度切换（Q1/Q2/Q3/Q4）。Active 态用 dark 背景反白。
Tab Panel — 标签页
多维度内容切换。Active 态用 2px 底部线指示。Tab 标签用 JetBrains Mono ALL CAPS。

12 / Detail Panel
详情面板

Detail Panel — 供应商穿透面板
固定遮罩层 + 居中面板，内容可独立滚动。适用于需要聚焦查看的场景：供应商详情、规则说明、数据明细。
