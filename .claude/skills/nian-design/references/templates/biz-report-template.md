# 业务报告模板

> 参照 H086（三字号规则 + 8:1 比例）

---

## 适用场景

- 月度/季度业务数据报告
- 供应商 / 团队 / 产线运营分析
- 数据驱动决策简报

---

## 推荐节奏骨架

1. **Hero** — 核心结论 + 关键数字（单行 112px 大字号，8:1 比例关系）
2. **KPI Bar** — 4 格指标矩阵（最高值高亮，其余平铺）
3. **Section n** — 逐段深入，每段以 tag（10px 数据字体）引导
4. **Ranking / Distribution** — 排名列表 + 占比条形 + 分群可视化
5. **Finding Cards** — 关键发现（左边框颜色区分类型）
6. **Outro** — 收束：行动建议 / 下一步 / 参考

---

## 关键组件组合

- Tag（`<div class="tag">`，数据字体 10px，uppercase，横线前缀，作 section 分隔符）
- 三字号体系：`h1`（112px / 8vw 起，展示数字）、`b`（14px，正文）、`l`（10px uppercase，标签/注释），核心比例 8:1:0.7
- Metric Bar（4 列等宽 KPI 网格，首列高亮，含值 + 标签 + 单位）
- Stat Pair（2 列对开数据块）
- Ranking（仿表格列表，含排名/姓名/数值/进度条）
- Finding Card（左边框色条，标题 + 说明 + 标签）
- Segment Bar / Dot Strip（占比可视化）

---

## 对照 H086

H086 是典型的"自上而下"业务报告：Tag + Hero 定调 → KPI 总览 → 细分 section（构成/产出/对比/诊断） → Finding 汇总 → Outro。报告节奏为"总→分→收"，不设目录页，以 Hero 直接切入核心数据。
