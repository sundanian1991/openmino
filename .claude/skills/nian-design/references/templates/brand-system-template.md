# 品牌系统模板

> 参照 H062/H063（Tension Pair 方法论 + 色调层级权重）

---

## 适用场景

- 品牌设计语言（Brand Design Language）定义
- 品牌视觉识别系统（VIS）文档
- 品牌多触点应用规范

---

## 推荐节奏骨架

1. **Brand Heritage / Origin** — 品牌年份、起源故事、核心精神
2. **Tension Pair** — 定义品牌张力对（如 Heritage vs Innovation / Craft vs Tech），作为系统驱动力
3. **Principles** — 3–6 条核心设计原则，每原则配说明 + 视觉示例
4. **Color System** — 色调层级权重：主色 / 副色 / 品牌辅助色 / 信号色，用 Color Bar 展示比例关系
5. **Typography** — 展示字体 / 正文 / 数据字体三层体系 + 字重映射
6. **Space & Layout** — 间距系统 + 网格 + 容器规范
7. **Applications** — 数字产品 / 零售体验 / 营销物料 / PPT / 邮件等触点变体

---

## 关键组件组合

- Sidebar（固定侧栏，垂直书写品牌标记 + 章节锚点）
- Hero Split（左文右图或图文混合，展示品牌声明 + 核心数字）
- Principle Card（编号 / 大标题 / 元标签 / 说明 / 分类标签）
- Color Bar（水平条形图，展示色盘层级权重，hover 展开比例）
- Meta Strip（标签 + 数值对，如产品数/年份/区域）

---

## 对照 H062/H063

H062 侧重品牌声明 + 原则体系 + 色调布局；H063 增加侧栏导航 + 更精细的色调层级权重可视化。两者构成品牌系统的"宣言层 + 规范层"双段式结构。
