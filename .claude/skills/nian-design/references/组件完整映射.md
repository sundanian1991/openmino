## 选型流程（完整版）

```
① 定形式族 → Hero / 数据图表 / 表格 / 卡片 / 导航 / 装饰 / 品牌
② 定变体   → 族内选 ID，A 优先
③ 定场景色 → olive / earth / glacier
④ 查 CSS   → ID → CSS 类名（此表）
⑤ 拼 HTML  → 按类名写结构
⑥ 验节奏   → 相邻 section 不得同族
```
