Haglöfs — Design Philosophy
R2 Preamble Block — 阅读前言

北欧设计有一种罕见的品质：它不急于证明自己。当你走进一家斯德哥尔摩的工作室，你不会看到炫耀性的标识或炫目的色彩。你会看到光线穿过亚麻窗帘，工具整齐地挂在木板上，一张桌子上放着半杯咖啡和一本翻到一半的笔记本。这就是 Haglöfs 设计语言的起点——不是在电脑前，而是在车间里。不是从趋势报告开始，而是从问题开始："这个人需要在山里待三天，他需要什么？"

T1 Marginalia Text — 边注文本

Haglöfs 的设计系统建立在四组张力之上，每一组都在同一个产品中同时存在且相互制衡。这不是一种"平衡"——平衡意味着妥协。这是一种"紧张"，是两股力量同时拉向相反方向时产生的能量。Heritage 与 Future：111 年的历史不是束缚，而是基座。每一款新产品都必须同时回答两个问题：它尊重了什么传统？它突破了什么边界？

Organic 与 Engineered：自然的形态通过精确的几何被提炼。一条登山裤的剪裁曲线，来自对数十种山地地形的运动分析；一个背包的背负系统，其曲线呼应了人体脊柱的自然弧度。这些不是装饰性的选择——每一个弧度都有功能理由。

引用

Nature does not hurry, yet everything is accomplished. We build gear the same way.

Alfred Haglöfs · 1914

第三组张力是 Nature 与 Tech。这是最明显的一对，也是最难做好的。许多户外品牌要么倒向自然（沉闷的绿色、粗糙的质地、功能让位于浪漫），要么倒向技术（全是 Gore-Tex 标签和防水指数，失去了与山的情感连接）。Haglöfs 的做法是让它们同时工作：泥土色的基底上，用精准的橙色信号点做功能提示；有机的轮廓里，嵌入工程优化的结构。

最后一组是 Warmth 与 Precision。这是最微妙的一对——它关乎语气。品牌的文字、图像、交互、材质选择，都必须同时传递功能上的精确性和情感上的温度。一个背包的拉链头，要大到戴手套也能操作（功能性），但它的形状要让人愿意去触碰（温暖感）。

AN 四组张力是品牌系统的核心架构——一切设计决策都能回溯到其中某一对的拉扯。
NF 自然界没有直线。Haglöfs 的几何不是强加的秩序，是对自然形态的功能性提炼。

T3 Annotated Highlights — 标注高亮

理解 Haglöfs 的设计系统，首先要理解一个核心前提： 颜色不是装饰，是信号定义每个颜色在系统中都有功能角色。Olive 代表户外 DNA，Earth 代表材质温暖，accent-orange 仅用于功能警示。 在大多数品牌系统中，颜色被用来"美化"——选择一个漂亮的调色板，然后应用到所有地方。Haglöfs 的做法相反：颜色的使用是被严格限制的。 Brand primary 三色占了 80% 的视觉面积洞察这意味着品牌的第一视觉印象几乎完全由 darkgray、olive、earth 三色定义。Accent 色只在 5-10% 的面积中出现，且必须承载功能信号。， 这意味着什么？意味着当你看到 Haglöfs 产品上的一个橙色元素时，你不需要思考就知道它有功能意义——它在告诉你"这里需要注意"。

同样的逻辑应用于间距系统。 8px 网格是否过于机械？追问网格是骨架，不是牢笼。8px 是步长，不是精度。在需要时，4px 或 12px 的偏离是被允许的——只要你能说出为什么。Haglöfs 不迷信网格，迷信"有理由"。 间距遵循地形逻辑——段落之间需要"开阔地"（空间-2xl 48px），相关内容需要"紧密扎营"（空间-sm 8px）。如果需要分割线，说明间距选错了。这不是设计规则的卖弄，而是一个来自实际使用场景的洞察：在野外，你不会在开阔地和营地之间画一条线——你会用自然过渡的距离感。

字体系统的三工分离不是美学偏好，是信息层级的硬约束。 Georgia : Inter : Mono = 1 : 10 : 1 的使用频率比洞察在典型页面上，Georgia 只用于标题（约 5-10% 的文本量），Inter 覆盖 80%+ 的内容，Mono 仅用于数据标签。这个比例本身就建立了信息的视觉层级。。 每页最多三种字号、两种字重——这不是限制，是判断力训练。当设计师被迫在三个字号内完成所有层级，他们必须做出真正的取舍。

U1 Key Term Cards — 关键术语
张力驱动设计
通过对立力量的紧张关系产生视觉能量。不追求"平衡"（妥协），追求"紧张"（同时拉向相反方向的能量）。四组张力对：Heritage/Future、Organic/Engineered、Nature/Tech、Warmth/Precision。
参见 R2 · T1
颜色信号规则
颜色必须承载功能，不做装饰。Brand primary 占 80%，Scene 占 10-15%，Accent 占 5-10% 且仅用于警示。视觉层级依赖灰度（4 级），不依赖色相。
