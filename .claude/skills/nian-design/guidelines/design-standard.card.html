Design Standard — 设计标准
Section 结构
min-height100vh每个 section 占满屏
padding128px 96px上下 128px，左右 96px
max-width1100px内容区最大宽度
section 数6-9 个不追求覆盖每一行原文
底色系统（不用白色）
olive-tint#F0F3EE数据/分析类 section
earth-tint#F5F0E8人文/品牌类 section
glacier-tint#EFF2F4技术/冷静类 section
dark#1E1D1AHero + 收尾
交替规则明→暗→色→明不连续同色，深色 ≤ 1/3
字号层级
H152px / 300Hero 标题，letter-spacing: -2px
Section title30px / 300section 标题
Stat number42px / 200数据数字，等宽字体
Body15px / 400正文，line-height: 1.7
Label10px / uppercase标签，letter-spacing: 1.5px
分隔方式（不用 border）
Stat gridgap: 1px + background卡片间用 1px 间隙做分隔
Key pointsgap: 1px + background同上
Timelinegap: 1px + background同上
卡片内部不用 border靠底色差异区分
/* Stat grid 分隔 */\n.sg { display: grid; gap: 1px; background: var(--border); }\n.sg4 { grid-template-columns: repeat(4, 1fr); }\n.sg-card { background: var(--bg); padding: 40px 32px; }
Ghost 水印
字号25vw非常大，制造空间感
opacity0.04几乎不可见
positionabsolute不影响布局
内容英文单词ALLIANCE / NEXT / CONTENT 等
使用方式：做任何 nian-design 产出时，参照此标准。不满足标准的 showcase 逐步迁移。新 showcase 必须按此标准制作。
