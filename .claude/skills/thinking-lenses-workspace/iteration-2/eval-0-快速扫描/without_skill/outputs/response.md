# 快速扫描：A供应商产能下滑

## 缺失信息

回答这个问题之前，需要先确认三点：

1. **哪家供应商** — 6家重点（毅航、毛毛虫、伽玛、赛维斯、岐力、翰锐）还是其他？
2. **什么指标** — 日接听量、完成率、在线人数、工单处理量？不同指标对应不同原因
3. **下滑幅度** — 10%以内是波动，20%以上是异常，判断标准不同

## 诊断框架

如果确认是真实下滑（非波动），按三个方向排查：

**人的问题**
- 关键坐席流失/请假集中
- 排班不合理（如新人占比突然升高）
- 培训期间产能自然下降

**流程/系统问题**
- 业务规则变更导致单通时长增加
- 系统故障或卡顿
- 新上线流程不熟练

**外部因素**
- 业务量本身下降（不是产能下滑，是需求下滑）
- 季节性因素
- 上游/甲方侧变化

## 建议动作

1. 先拉数据确认 — 对比近两周 vs 前两周的核心指标（日均量、在线人数、单通时长）
2. 拆维度看 — 按天、按班次、按业务线定位是全面下滑还是局部问题
3. 直接问供应商 — 要解释比猜原因高效得多
