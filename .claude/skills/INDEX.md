# Skills索引

> 所有可用技能及其使用场景

---

## 文档处理

| Skill | 功能 | 使用场景 |
|-------|------|---------|
| **pdf** | PDF文本/表格提取、创建、合并拆分 | 处理供应商合同、述职文档 |
| **docx** | Word文档创建、编辑、修订跟踪 | 写制度文档、SOP、报告 |
| **pptx** | PPT创建、编辑、模板操作 | 述职答辩、业务汇报 |
| **xlsx** | Excel表格创建、公式、数据分析 | KPI报表、供应商绩效统计 |
| **summarize** | URL/文本摘要提取 | 快速了解长文档核心内容 |
| **nano-pdf** | PDF表格提取、创建、合并拆分 | 轻量级PDF处理 |
