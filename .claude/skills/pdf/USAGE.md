# pdf Skill - 使用指南

> PDF文档处理工具

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **提取文本** | 从PDF提取可搜索的文字 |
| **提取表格** | 识别并导出表格数据 |
| **创建PDF** | 生成新PDF文档 |
| **合并PDF** | 多个PDF合并为一个 |
| **拆分PDF** | 按页码范围拆分文档 |
| **填写表单** | 自动识别并填写可填字段 |

---

## 注意事项

- **表格识别**：复杂表格可能需要人工核对
- **扫描质量**：扫描版PDF提取准确率较低
- **文件大小**：大文件处理时间较长，请耐心等待
