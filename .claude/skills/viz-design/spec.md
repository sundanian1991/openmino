{
  // 散点图配置
  xAxis: { name: '线路费%', min: 30, max: 70 },

## 图5：提前还款率 vs 纠偏系数 — 双轴组合图

### 数据

| 月份 | 首贷实际提前还款率 | 首贷固定假设 | 纠偏系数 |
|------|-----------------|------------|---------|
| 01月 | 9.1% | 7% | 0.769 |
| 02月 | 10.4% | 7% | 0.673 |
| 03月 | 9.9% | 7% | 0.707 |
| 04月 | 10.0% | 7% | 0.700 |
| 05月 | 11.6% | 7% | 0.603 |

### 配置
- 双Y轴：左轴 %（柱状），右轴 纠偏系数（折线）
- 柱状：实际值用橙色，固定值用灰色半透明
- 折线：深红色，带数据标注
- 上下方向反向（提前还款率上升 + 纠偏系数下降 → 视觉交叉）
