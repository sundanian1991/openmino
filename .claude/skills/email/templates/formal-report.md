# 正式汇报邮件模板

## 使用说明
用于向上汇报、专项汇报、周报月报、述职汇报

<p>现向您汇报XXX事项。</p>

<p>XXX（简述为什么做这件事、想达到什么目标）。</p>

<p><strong>关键进展：</strong></p>

<p>一是XXX（重点突出关键数据/结论），二是XXX，三是XXX。</p>

<p><strong>数据支撑：</strong></p>

<ul>
  <li>KPI1：XX（环比+X%，同比+X%）</li>
  <li>KPI2：XX</li>
  <li>KPI3：XX</li>
</ul>

<p><strong>存在风险：</strong></p>

<p>一是XXX，二是XXX。如无风险可说明"无重大风险"。</p>

<p><strong>下一步计划：</strong></p>

<ul>
  <li>XXX（明确时间节点）</li>
  <li>XXX</li>
  <li>XXX</li>
</ul>

<p>如有疑问，随时沟通。</p>
