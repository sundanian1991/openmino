# Leader Interrogation — 领导模拟质询法

> 在低风险环境中暴露思维漏洞，建立经得起质疑的逻辑体系


---

## 核心流程

1. **配置领导风格** → 职级/风格/焦点/强度
2. **提交方案材料** → 文档/大纲/核心逻辑
3. **首轮质询（7 轮）** → 压力测试暴露漏洞
4. **生成漏洞报告** → 逻辑/数据/执行/话术
5. **修复 + 二次验证** → 确认改进效果
6. **最终评估** → 汇报准备度评分

---

## 适用场景

- 向上汇报前压力测试
- 资源申请方案预演
- 跨部门协同方案评审
- 晋升述职材料打磨

---

*不是为了应付领导，是为了建立更严谨的思维体系。*
