# 偷师三行法 - 模板库

> 针对不同场景的即用模板

---

## 一、通用三行模板

### 基础模板

```
【第一行：我看到了什么？】
[人物/主体] + [做了什么动作] + [产生了什么结果]
只记录事实，不加评价。

【第二行：我能用在哪儿？】
下次 + [具体场景] + [如何应用]
绑定100%会遇到的真实场景。

【第三行：我何时试一次？】
[具体日期] + [具体事件] + [如何实践]
设定死线，关联任务。
```

---

## 二、场景专用模板

### 模板1：话术/沟通技巧类

```
【第一行：我看到了什么？】
在[场景]中，[人物]遇到[情况]时，说的是"[原话]"，
而不是"[常见的错误说法]"，
结果[对方的反应/结果]。

【第二行：我能用在哪儿？】
下次我遇到[类似情况]时，
不说"[错误说法]"，
而是说"[正确话术框架]"。

【第三行：我何时试一次？】
[具体日期]的[具体事件]中，
如果遇到[类似情况]，就用这个话术。
```

**示例场景**：
- 客户投诉处理
- 价格异议回应
- 向上汇报
- 跨部门沟通

---

### 模板2：方法论/流程类

```
【第一行：我看到了什么？】
[人物]处理[任务]时，
没有用[常规做法]，
而是用了[新方法/工具/流程]，
结果是[具体效果/数据]。

【第二行：我能用在哪儿？】
下次我做[类似任务]时，
不用[旧方法]，
而是先[新方法的第一步]，再[第二步]。

【第三行：我何时试一次？】
[具体日期]开始做[某任务]时，
用这个新方法，一周后复盘效果。
```

**示例场景**：
- 数据分析流程
- 项目复盘方法
- 会议组织技巧
- 时间管理方法

---

### 模板3：思维/认知类

```
【第一行：我看到了什么？】
面对[问题]，[人物]的第一反应是[常规反应]，
但他马上调整为[不同视角]，
然后用[什么方式]解决了问题。

【第二行：我能用在哪儿？】
下次我遇到[类似问题]时，
不按[本能反应]做，
而是先用[新视角]问自己：[关键问题]。

【第三行：我何时试一次？】
下次[类似情况]发生时，
给自己30秒停顿，
用这个新视角重新审视问题。
```

**示例场景**：
- 应对突发问题
- 向上管理思维
- 跨部门协作
- 冲突处理

---

### 模板4：工具/技巧类

```
【第一行：我看到了什么？】
[人物]用[工具/技巧]处理[任务]，
比传统方法快/好[具体倍数/程度]，
核心是用了[什么核心技巧]。

【第二行：我能用在哪儿？】
下次我做[类似任务]时，
不用[旧方式]，
而是直接用[工具/技巧]，重点是[核心技巧]。

【第三行：我何时试一次？】
[具体日期]做[某任务]时，
用这个工具，记录时间对比。
```

**示例场景**：
- Excel技巧
- AI工具使用
- 快捷键技巧
- 效率工具

## 三、进阶模板

### 模板5：复合型萃取（用于复杂场景）

```
【第一行：我看到了什么？】
[场景描述]
人物做了A动作 → 导致B结果
然后做了C调整 → 产生D效果
最终达成E目标

核心技巧：
- 技巧1：[描述]
- 技巧2：[描述]
- 技巧3：[描述]

【第二行：我能用在哪儿？】
场景A：[具体应用方式]
场景B：[具体应用方式]
场景C：[具体应用方式]

【第三行：我何时试一次？】
近期机会：
1. [日期] - [事件]：试技巧1
2. [日期] - [事件]：试技巧2
3. [日期] - [事件]：完整实践

一周后复盘效果。
```

---

### 模板6：逆向萃取（从失败中学习）

```
【第一行：我看到了什么？】
[人物/我]在[场景]中做了[动作]，
结果是[失败/负面效果]。
复盘发现：错误原因是[根本原因]。

【第二行：我以后怎么避免？】
遇到[类似情况]时，
不做[错误动作]，
而是改为[正确做法]。

【第三行：我何时验证改变？】
下次[类似场景]出现时（预计[时间]），
用新方法应对，记录结果。
```

## 四、年老师专用模板

### 模板7：供应商管理场景

```
【第一行：我看到了什么？】
面对供应商[情况]（如：违约/涨价/投诉），
常规做法是[错误做法]，
但[某人]用了[新做法]：
- 第一步：[具体动作]
- 第二步：[具体动作]
- 结果：[供应商反应/最终效果]

【第二行：我能用在哪儿？】
下次我管理[供应商类型/类似情况]时，
不用[旧方式]，
而是用这套方法：
1. [具体应用点1]
2. [具体应用点2]

【第三行：我何时试一次？】
[日期]处理[具体供应商/事件]时，
完整用这套方法，对比效果。
```

---

### 模板8：晋升述职场景

```
【第一行：我看到了什么？】
[某人]在述职中讲[案例]时，
不是说"我做了XX"，
而是用"面临XX挑战→用了XX方法→达成XX结果"的结构，
评委问的问题都是"怎么复制的"而不是"你真的做的吗"。

【第二行：我能用在哪儿？】
我准备述职案例时，
不用"流水账"写法，
而是用"挑战-方法-结果-可复制"四段式。

【第三行：我何时试一次？】
今晚就用这个结构重写[某案例]，
明天找人要反馈。
```

## 五、快速模板（30秒版）

### 极简模板

```
【我看到了什么？】
[谁]在[什么情况]下，做了[什么]，结果是[怎样]。

【我能用在哪儿？】
下次[类似场景]，我用[这个方法]。

【我何时试一次？】
[日期]的[具体事件]，我就试。
```

## 六、配合AI的增强模板

### AI辅助萃取模板

```
【原始素材】
[粘贴内容/描述场景]

【第一行：我看到了什么？】
（AI辅助提取事实）
请提取：谁、做了什么、结果是什么

【第二行：我能用在哪儿？】
（AI辅助场景匹配）
请匹配：我工作中100%会遇到的类似场景

【第三行：我何时试一次？】
（AI辅助行动规划）
请给出：近期可能用到这个的具体机会

【执行检查】
- 场景是否具体？[是/否]
- 日期是否明确？[是/否]
- 能否立刻行动？[是/否]
```
