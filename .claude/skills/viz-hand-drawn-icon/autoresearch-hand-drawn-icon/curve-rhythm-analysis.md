## 可编码规律总结

### 规律 1：控制点间距 = 复杂度倒数的函数

```javascript
function getBaseSpacing(complexity) {
  // complexity: 'simple' | 'medium' | 'complex'
  const spacing = {
    simple: { min: 20, max: 50, avg: 35 },
    medium: { min: 10, max: 30, avg: 20 },
    complex: { min: 2, max: 15, avg: 8 }
  };
  return spacing[complexity];
}
```

### 规律 2：间距变化 = 波浪式节奏（不是纯随机）

```javascript
function getNextSpacing(prevSpacing, trend) {
  // trend: 'increasing' | 'decreasing' | 'switch'
  const variation = prevSpacing * 0.3; // 30% 波动
  const direction = trend === 'switch'
    ? Math.random() > 0.5 ? 1 : -1
    : trend === 'increasing' ? 1 : -1;
  return prevSpacing + direction * variation * Math.random();
}
```

### 规律 3：小数点 = 1-2 位随机（避免 .00）

```javascript
function addRandomDecimal(num) {
  const decimal = Math.random() > 0.7 ? 1 : 2; // 70% 2位，30% 1位
  return parseFloat(num.toFixed(decimal));
}
```

### 规律 4：线条粗细 = 物体大小的函数

```javascript
function getStrokeWidths(size) {
  // size: 'small' | 'medium' | 'large'
  const widths = {
    small: { main: 1.5, structure: 1.0, detail: 0.7 },
    medium: { main: 3.0, structure: 2.0, detail: 1.0 },
    large: { main: 5.0, structure: 3.5, detail: 2.0 }
  };
  return widths[size];
}
```

### 规律 5：不对称约束 = 左右 ±(1-5px) 偏差

```javascript
function addAsymmetry(coord, isLeft, isRight) {
  if (!isLeft && !isRight) return coord;
  const deviation = (Math.random() * 4 + 1) * (isLeft ? 1 : -1);
  return coord + deviation;
}
```
