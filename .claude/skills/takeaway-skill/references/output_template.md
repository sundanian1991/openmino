# takeaway-skill output template

Notes:

- This is a safe template with no third-party website examples.
- Use placeholders instead of real site names, brand names, screenshots, or protected copy.
- This file is designed to stay public.

Suggested output structure:

1. `Distilled object`
   - [reference object type]
   - [target layer]

2. `One-line judgment`
   - [the most important takeaway]

2.5 `Pattern tags`
   - [object type]
   - [interaction tags]
   - [style tags]

3. `Most valuable takeaway`
   - [mechanism / structure / rhythm / visual language]

4. `Can take directly`
   - [item 1]
   - [item 2]
   - [item 3]

5. `Should not copy directly`
   - [brand-tied item 1]
   - [brand-tied item 2]
   - [copyright-sensitive item 3]

6. `Can take after adaptation`
   - [mechanism stays, expression changes]

7. `Evidence still insufficient`
   - [missing evidence]

7.5 `Scores`
   - evidence confidence: [high / medium / low]
   - originality distance: [far / medium / near]
   - archive value: [high / medium / low]

8. `Adaptation suggestions`
   - [direction 1]
   - [direction 2]

9. `Downstream handoff`
   - target workflow:
     - [workflow type]
   - smallest valid result:
     - [smallest result]
   - stop-loss:
     - [when to stop guessing]

10. `Implementation brief`
   - regions:
     - [region 1]
     - [region 2]
   - components:
     - [component 1]
     - [component 2]
   - states:
     - [default / hover / scroll / expanded]
   - lock first:
     - [skeleton / crop / hierarchy]
   - do not touch yet:
     - [decorative details / guessed parameters / mixed layers]
   - recommended container:
     - [HTML / CSS / JS / SVG / design file]
