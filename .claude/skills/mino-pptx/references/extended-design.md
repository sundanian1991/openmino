## 二、风格配方速查

| 风格 | 圆角 | 间距 | 最适合 |
|------|------|------|--------|
| **Sharp** | 0" ~ 0.05" | 紧凑 0.1"~0.2" | 数据报表、专业报告 |
| **Soft** | 0.08" ~ 0.12" | 适中 0.15"~0.25" | 企业汇报（默认推荐）|
| **Rounded** | 0.15" ~ 0.25" | 宽松 0.25"~0.4" | 产品介绍、营销 |
| **Pill** | 0.3" ~ 0.5" | 开放 0.3"~0.5" | 品牌展示、发布会 |

### 组件映射表

| 组件 | Sharp | Soft | Rounded | Pill |
|------|-------|------|---------|------|
| 卡片 | 0.03" | 0.1" | 0.2" | 0.3" |
| 按钮 | 0" | 0.05" | 0.1" | 0.2" |
| 图片框 | 0" | 0.08" | 0.15" | 0.25" |
| 徽章 | 0.02" | 0.05" | 0.08" | 0.15" |

**关键规则**：外容器圆角 ≥ 内元素圆角（防止视觉溢出）

## 三、幻灯片布局速查

### Cover（封面）

```
非对称左右布局          居中布局
[标题  |  图片]         [背景图]
[副标题|      ]          [主标题]
                         [副标题]
```

- 主标题：72-120pt（至少是副标题2-3倍）
- 副标题：28-40pt
- 必须有视觉元素（装饰圆、线条、图案）

### TOC（目录）

```
垂直列表（3-5项）       双栏网格（4-6项）
01 Section One         01 Sec    02 Sec
02 Section Two         03 Sec    04 Sec
03 Section Three
```

- 编号要突出（粗体/强调色）
- 2-3秒可扫描完

### Section Divider（章节分隔）

```
左对齐+色块            居中大字
███ 02                02
███ SECTION           SECTION
███ 简介              简介
```

- 章节号是最大视觉元素
- 留白充足（这是"停顿"页）

### Content（内容页）

| 子类型 | 适用场景 |
|--------|---------|
| Text | 要点列表（必须有图标/装饰） |
| Mixed Media | 左文右图 |
| Data | 图表+关键结论（必须标数据来源） |
| Comparison | 左右对比卡片 |
| Timeline | 流程步骤 |

**禁止**：纯文字页（必须有至少一个非文字元素）

### Summary（总结页）

```
要点式                 行动项
KEY TAKEAWAYS          NEXT STEPS
✓ Point 1              [1] Action 1
✓ Point 2              [2] Action 2
✓ Point 3              Contact: xxx
```

## 四、字体层级速查

| 元素 | 字号 | 备注 |
|------|------|------|
| 封面主标题 | 72-120pt | 全页最大 |
| 章节号 | 72-120pt | Section Divider |
| 内容页标题 | 36-44pt | 每页顶部 |
| 副标题 | 28-40pt | |
| 章节标题 | 20-24pt | 内容页内 |
| 正文 | 14-16pt | **禁止加粗** |
| 数据高亮 | 60-72pt | 关键数字 |
| 注释/来源 | 10-12pt | 最小字号 |

**关键原则**：
- 正文和列表 → 左对齐（**禁止居中**）
- 只有标题可以居中
- 标题与正文字号差距 ≥ 20pt

## 五、关键防坑清单

### 致命错误（会导致文件损坏）

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| `color: "#FF0000"` | `color: "FF0000"` |
| `color: "00000020"`（含透明度）| `color: "000000"` + `opacity: 0.12` |
| `async function createSlide()` | `function createSlide()` |
| 复用 options 对象 | 用工厂函数：`const makeShadow = () => ({...})` |

### 设计错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| 正文居中 | 正文左对齐 |
| 纯文字页 | 每页至少一个非文字元素 |
| 连续相同布局 | 每页布局要有变化 |
| 标题下加装饰线 | 用留白或背景色代替 |
| 正文加粗 | 正文常规字重，只有标题加粗 |
| 低对比度 | 图标和文字都要强对比 |

### QA 必查

```bash
# 内容检查
python -m markitdown output.pptx

# 占位符检查
python -m markitdown output.pptx | grep -iE "xxxx|lorem|ipsum|placeholder"
```

**必须完成一次"修复-验证"循环才能算完成**

## 六、间距速查

基于 10" x 5.625" 幻灯片：

| 用途 | 推荐值 |
|------|--------|
| 图标到文字 | 0.08" ~ 0.15" |
| 列表项间距 | 0.15" ~ 0.25" |
| 卡片内边距 | 0.2" ~ 0.4" |
| 元素组间距 | 0.3" ~ 0.5" |
| 页面安全边距 | 0.4" ~ 0.6" |
| 大区块间距 | 0.5" ~ 0.8" |
