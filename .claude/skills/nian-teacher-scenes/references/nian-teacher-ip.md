# 年老师 IP

## 角色定义

年老师是一个安静的观察者、思考者。不是吉祥物，不是Logo，是一个在自然场景中认真观察、思考、记录的文艺青年。

它可以因为场景不同而有细微形体差异：在森林里更放松、在书房里更专注、在户外更随性。变化要小，识别必须稳定。

## 必须保留的识别点

- 黑框方形眼镜（粗框，方形，黑色）。
- 浅灰蓝色连帽卫衣，帽子拉起。
- 黑色双肩背包。
- 东亚男性面孔，五官线条柔和，神态平和带浅微笑。
- 整体风格偏休闲户外、文艺气质。
- 绘画风格：宫崎骏/吉卜力风格手绘插画，线条柔和色彩饱满。

## 2.0 形体弹性

默认形态：

- 正常人类比例，不是Q版也不是写实。
- 站姿自然放松，不僵硬。
- 高度通常保持在画面高度的 15%-25%，比小黑更大一些，因为是人类角色。

允许变化：

- 在森林里可以更放松，比如倚靠树木。
- 在思考时可以手托下巴或抬头仰望。
- 在记录时可以拿着笔和本子。
- 在户外行走时可以有动态姿势。
- 背包可以调整位置，有时背在身后，有时放在身边。

禁止变化：

- 不要变成卡通Q版大头娃娃。
- 不要变成写实照片风格。
- 不要变成二次元动漫风格。
- 不要戴帽子（只有连帽卫衣的帽子）。
- 不要做夸张表情或动作。
- 不要穿西装或过于正式的服装。
- 不要每张都完全同一个姿势，否则会死板。

## 年老师必须承担核心动作

年老师不能只是站在旁边看。

合格动作：

- 在森林小径上独行，观察周围环境。
- 坐在木屋窗边，安静思考。
- 手拿笔记本，记录观察到的细节。
- 与自然物件互动：捡起落叶、观察蘑菇、触摸树皮。
- 背包行走，探索新的路径。
- 在户外写生或拍照。
- 与动物安静共处（鸟停在肩上、猫蜷在脚边）。

不合格动作：

- 站在角落里看场景。
- 举牌子解释主题。
- 对着镜头摆拍。
- 和场景没有情感连接。
- 去掉年老师后图意仍然完整。

判断标准：

```text
如果删掉年老师，画面的观察感/思考感还完全成立，说明年老师失败。
```

## 性格

- 安静，不张扬。
- 内敛，但有强烈的价值判断。
- 像一个低调的自然观察者。
- 在安静中发现美。
- 它的气质来自"认真观察和思考"，不是表情和装饰。
- ISFP探险家特质：审美敏感、用行动表达、活在当下。
