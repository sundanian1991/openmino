# weekly-report — 周报生成技能

> 输入：零散工作事项 → 输出：大厂风格周报（纯文字版 + 口述稿）

---

## 快速使用

```
/weekly-report

[粘贴你的本周工作事项]
```

---

## 输出内容

1. **书面周报** — 纯文字版，方便 Excel 粘贴同步
2. **口述稿** — 3 分钟版本，周会发言用

---

## 示例

### 输入
```
/weekly-report

本周工作：
1. 商家白条 AI 加微项目启动，2 家 AI 赛马并行，期末留 1 家
2. 采购融资 3 人扩量测试，选定泉州速腾
3. 白条分期新增 15 人，选定毛毛虫（述标第 2 名），人力储备被挖，9 号上线有风险，延期 2 天便于招聘出班
4. 低风险客群专营，选定博岳和赛维斯
5. 配合综管 3 月实施清正行动，审计问题回头看，需相关同事知悉配合
6. 新增定价委员会机制，待讨论完善
```

### 输出

**书面周报**（纯文字版）+ **口述稿**（3 分钟）

详见 SKILL.md 完整规则。

---

## 适用场景

| 场景 | 说明 |
|------|------|
