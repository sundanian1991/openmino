# 策展视觉语言

## 概述

使用这个 skill 建立研究型展览应如何被观看和操作。视觉语言必须服务策展判断、阅读秩序、材料身份和观众定位。

## 定义参照系

用功能性语言命名视觉和编辑参照系。例如：当代学术编辑式、档案阅览室、田野笔记、机构藏品界面、口述史聆听室、分析型地图册、展墙展签系统。

不要只从审美偏好选择风格。参照系应解释文本、证据、图像、导航和交互如何被组织。

## 材料身份

先分类材料，再放入页面：

- 证据材料：档案图像、地图、文档、物件、访谈文本、引文。
- 定位材料：地图、时间线、索引、章节标记。
- 分析材料：机制图、关系图、比较图、网络图。
- 氛围材料：纹理、运动、声音、图像碎片。

除非页面类型明确需要，材料不应成为整页情绪背景。如果一张图像正在承担整页成立感，应回到结构阶段重新检查。

## 设计决策

需要定义：

- 标题、章节、正文、图注、标签、导航和证据层的字体层级。
- 背景、正文、弱化文字、强调色、激活态、材料边框、风险或不确定性的色彩系统。
- 阅读列、侧栏材料、索引模块和来源区域的版式节奏。
- 进入、切换、展开、返回、来源揭示和跨单元关系的交互语法。
- 会扭曲项目气质的排除方向。

## 输出

输出视觉语言说明，可直接指导设计和 CSS。内容应包括设计参照系、材料规则、字级角色、色彩方向、交互行为和具体排除项。

## 参考范式

《汴京十二时辰》使用“当代学术编辑式”视觉语言：文字主导页面，历史地图作为材料而不是默认背景，交互服务阅读和下钻。这是一个参考案例，不是所有项目必须采用的风格。
