# pptx Skill - 使用指南

> PPT演示文稿创建与编辑

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **创建新演示** | 从模板或空白创建 |
| **编辑现有** | 修改内容、样式、布局 |
| **提取文本** | 从.pptx提取文字内容 |
| **合并文件** | 多个演示合并为一个 |
| **拆分演示** | 一个演示拆成多个文件 |

---

## 年老师场景

### 述职答辩
```
 Skill
   ↓
创建述职PPT模板
   ↓
填充三案例内容
   ↓
调整视觉风格（Clean风格）
   ↓
导出答辩用版本
```

### 业务汇报
```
 Skill
   ↓
读取旧汇报
   ↓
更新本月数据
   ↓
生成新版本
   ↓
保留演讲者备注
```

---

## 调用示例

"用pptx skill创建一个述职答辩模板，包含封面、三个案例、总结页"

---

## 注意事项

- **保持风格统一**：使用相同字体、配色方案
- **少文字多图表**：述职场景尤其重要
- **演讲者备注**：添加关键要点提醒
- **Clean审美**：充足留白、圆角、中性色
