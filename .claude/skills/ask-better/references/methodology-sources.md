# 方法论来源与致谢

本技能基于以下 7 本提问方法论著作构建：

## 核心来源

1. **Frank Sesno — Ask More**
   - 12 类问题分类体系（诊断型、战略型、共情型、桥接型、对抗型、创造型、使命型等）
   - 提供了意图识别的理论框架

2. **Curtis Newbold — 7 Rules of Asking Questions**
   - 7 条提问原则（开放性、简洁性、中立性等）
   - 提供了提问设计的质量标准

3. **T.J. Fadem — The Art of Asking**
   - 25 种提问题型字典
   - 提供了具体题型的参考库

4. **Jeffrey Baucom — The Ask**
   - "为什么是我 / 为什么是现在 / 问完希望对方做什么" 三问框架
   - 提供了场景澄清的结构

5. **Chris Voss — Never Split the Desk**
   - 重新框定、 calibrated questions 技巧
   - 提供了谈判场景的提问策略

6. **Sheila Heen & Douglas Stone — Thanks for the Feedback**
   - 触发反应、边界线的处理
   - 提供了处理敏感话题的视角

7. **Amanda Ripley — Talk/High Conflict**
   - 高冲突场景下的降阶对话
   - 提供了对抗性场景的化解方式

## 使用说明

这些理论在本技能中的应用方式：

* **12 类问题** → 用于意图路由（阶段 2）
* **7 条原则** → 用于提问质量检验（阶段 3）
* **25 种题型** → 用于具体措辞参考（阶段 3）
* **三问框架** → 用于场景澄清（阶段 1）
* **重新框定** → 用于中国语境适配（强制规则）

## 中国语境适配

以上理论均来自西方语境。本技能根据中国社会的文化特征进行了本地化适配，详见 `china-context.md`。

**主要调整**：

* 权力距离：中国职场层级意识更强 → 三版提问的梯度设计
* 关系优先："面子"文化 → 开场用语和拒绝处理
* 套路敏感：中国用户对"话术"的反感 → 禁止话术套路规则
