# Data Audit Checklist (Terminal Output)

Use this guide when performing Phase 7 data audits.

---

## Quick Audit Flow

```
1. 打印审计头部信息
2. 列出所有待验证数据点
3. 逐项验证，每项打印结果
4. 完成后打印汇总统计
5. 列出问题清单
6. 询问用户修复方式
```

## ANSI Terminal Colors

```bash
# 定义颜色
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
RED='\033[0;31m'
BLUE='\033[0;34m'
CYAN='\033[0;36m'
BOLD='\033[1m'
NC='\033[0m' # No Color

# 使用示例
echo -e "${GREEN}✓ 已验证${NC}: 全年收入 3,649亿元"
echo -e "${YELLOW}⚠️ 疑似${NC}: Q4亏损161亿元 (需确认)"
echo -e "${RED}✗ 错误${NC}: 市场份额68% (源数据无此字段)"
echo -e "${BLUE}🔍 验证中${NC}: 正在查找..."

# 删除线 (使用 Unicode 或特殊字符)
echo -e "${GREEN}✓${NC} 完成~~: 全年收入 3,649亿元"
```

## Validation Methods

| Method | When to Use | Output |
|--------|-------------|--------|
| 直接匹配 | 数值在源数据中完全一致 | ✓ 已验证 |
| 计算验证 | 百分比/增长率等派生值 | ✓ 已验证 (计算正确) |
| 阈值检查 | 差异在5%以内 | ⚠️ 疑似 (轻微差异) |
| 差异过大 | 差异超过5% | ✗ 错误 |
| 未找到 | 源数据无此字段 | ✗ 错误 / ⚠️ 疑似 |
