# 供应商问询函模板

> 适用于供应商突发人力异常、质量下滑、合同违约等需要正式问询的场景

---

## 适用场景

- 人力突然大幅减少
- 服务质量明显下滑
- 关键指标异常波动
- 合同履约出现违约
- 需要供应商正式说明情况

---

## 模板结构

```
【主题】关于[供应商名称][事项]问询函

尊敬的 [供应商称呼]：

【背景】说明观察到的异常情况
【疑问】请供应商说明的具体问题（列点）
【时限】要求回复的时间
【后续】如有问题如何沟通
【姿态】表达合作意愿

此致
[发件方署名]
```

---

## 正式版本示例

**主题：关于[供应商名称]人力情况问询函**

尊敬的 [供应商名称]负责人：

我司注意到，贵司近期向我司[业务类型]提供的人力坐席数量呈持续下降趋势，尤其近两日出现明显骤降，已对正常业务运营造成直接冲击。

此前贵司向我司报送的人力数据持续处于正常偏高水平，本次[异常事项]发生较为突然，我司需及时了解情况，以便做好业务应对。

请贵司就以下事项于收函后[三个工作日]内书面回复：

1. 当前实际在岗[数量/人员]情况及分布
2. [异常事项]的具体原因及时间节点
3. 后续[补充/恢复]计划及预计时间

如涉及特殊情况或存在实际困难，请贵司主动与我司沟通协商，我司愿共同寻求解决方案。

我司高度重视与贵司的合作关系，期待贵司积极回应，共同保障业务平稳运行。

此致

[发件方署名]
[日期]

---

## 关键要素

| 要素 | 说明 |
|------|------|
| 背景 | 客观描述观察到的异常，不带情绪 |
| 疑问 | 列点说明需要回复的具体问题 |
| 时限 | 明确回复期限，保留主动权 |
| 姿态 | 表达合作意愿，避免对立 |
| 签名 | 根据场景选择合适署名 |

---

## 质量检查

- [ ] 背景描述是否客观、不带预设立场？
- [ ] 疑问是否具体、可回复？
- [ ] 时限是否合理（通常3-5个工作日）？
- [ ] 是否留有沟通协商空间？
- [ ] 语气是否既严肃又不失合作意愿？
