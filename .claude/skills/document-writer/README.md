# 公文起草技能 (Document Writer)

> 专业的公文/邮件/审批材料起草工具

---

## 概述

**用途**：起草各类公文、邮件、审批材料
**输出**：Markdown 格式（可预览、可编辑、可转换 HTML）
**特点**：专业规范、模板齐全、持续打磨

---

## 工作流

```
用户说："写一个审批邮件"
  ↓
选择对应模板（分量扩项/内部协调）
  ↓
按模板格式起草
  ↓
输出 Markdown 格式
  ↓
用户预览 → 提出修改 → 调整内容
  ↓
确认无误 → 转换 HTML → 调用 email skill 发送
```

---

## 与 email skill 的区别

| 维度 | document-writer | email |
|------|-----------------|-------|
| **功能** | 起草文档 | 起草 + 发送 |
| **输出** | Markdown 格式 | 发送邮件 |
| **签名** | 可选 | 自动追加 |
| **定位** | 专注写作质量 | 完整邮件工作流 |

---

## 模板分类

### 通用公文（4个）
- 正式汇报、通知、协作、内部简洁

### 供应商管理（6个）
- 储备、引入、运营、评估、清退、结算

### 扩展模块（6个）
- 商务谈判、分量扩项、内部协调、合规管控、处罚争议、风险通报

---

## 质量标准

段落逻辑优先，视觉装饰为零，适度留白舒适。
