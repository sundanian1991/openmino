# CLAUDE.md — Commands

> Commands目录的标准配置文件

---

## Summary

Commands目录存放所有Claude命令配置文件，按功能分类但存放在同一目录中。Commands系统提供标准化的工具和工作流程，帮助用户完成各种任务。

---

## Members

### 核心文件

| 文件 | 用途 |
|------|------|
| `INDEX.md` | 命令系统索引和分类 |
| `README.md` | 目录说明和结构 |
| `CLAUDE.md` | 本文件 - 配置说明 |

---

### Core Commands (核心高频)

| 文件 | 用途 |
|------|------|
| `wake.md` | 会话启动 - 同步代码并初始化 |
| `plan.md` | 规划模式 - 分析现状、设计方案、等待确认 |
| `checklist.md` | 任务核实 - 提取对话任务，逐项确认状态 |
| `think.md` | 思考显性化 - 展示推理过程，让思考可见 |

---

### Memory Commands (记忆与学习)

| 文件 | 用途 |
|------|------|
| `observer.md` | 对话观察 - 记录事实与洞察 |
| `UPDATE_MEMORY.md` | 每周汇总 - 从事实洞察生成周文档 |
| `learn.md` | 模式提取 - 从会话中提取可复用技能 |
| `log-day.md` | 日志记录 - 时间轴+标签，整理到daily |

---
