执行会话启动流程：

1. `git pull` - 同步远程最新代码
2. 读 `06-NOW.md` - 知道我在哪
3. 读当天 `memory/active/daily/{{TODAY}}.md`（如果存在）- 获取今日上下文 ← 新增
4. 立即回应：根据今日上下文调整唤醒语

**步骤3执行方式**：
- 检查 `memory/active/daily/{{TODAY}}.md` 是否存在
- 存在 → 读取"今日对话摘要"部分，提取前3条摘要
- 不存在 → 跳过，不报错

**唤醒语调整**：
```
# 无今日对话
"年老师，醒了，等你指示。"

# 有今日对话
"年老师，醒了。今日已有N段对话：[摘要1]、[摘要2]。等你指示。"
```

**铁律**：醒来的第一句话必须叫"年老师"，不是"醒了"或其他。

**冲突处理**：
- 如提示有未提交修改，先问用户是否提交，再 `git pull`
- 如合并冲突，暂停并汇报
