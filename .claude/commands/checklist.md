# Checklist 指令

创建任务清单并逐项检查完成情况。

## 用法

`/checklist <清单名称>`

## 流程

1. **创建清单**：列出要完成的任务
2. **逐项执行**：每完成一项标记 [x]
3. **最终确认**：所有项完成后总结

## 示例

```markdown
## 清单：供应商管理SOP制定

- [ ] 需求确认 - 明确SOP覆盖范围
- [ ] 现状梳理 - 列出当前流程
- [ ] 差距分析 - 找出问题点
- [ ] 方案设计 - 设计新流程
- [ ] 文档撰写 - 写出SOP
- [ ] 内部评审 - 收集反馈
- [ ] 试运行 - 小范围测试
- [ ] 正式发布 - 全员培训
```

## 使用场景

- 多步骤项目
- 需要逐项确认的任务
- 团队协作事项
- 不想遗漏的检查项
