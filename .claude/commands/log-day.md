# log-day 命令

**用途**：会话结束时，引导用户记录当天的时间轴和标签

**触发**：
- 用户说"结束"、"今天先到这"、"明天再聊"
- 或 Mino 主动判断会话即将结束

**执行**：
1. 询问今天的主要事项和时间
2. 帮助打标签
3. 整理到 `memory/active/daily/YYYY-MM-DD.md`

---

## 执行话术

```
年老师，今天差不多了。咱们花 1 分钟记一下日志：

**今天主要干了啥？大概几点到几点？**

我帮你记下来，打好标签。
```

---

## 标签速查

**工作**：
- `#工作/搭体系` — SOP、模型、看板、机制（P7 核心）
- `#工作/沟通` — 会议、汇报、1v1、供应商沟通
- `#工作/救火` — 临时问题、紧急处理
- `#工作/反思` — 复盘、周报、方法论沉淀

**生活**：
- `#生活/家庭` — 陪老婆、家务、家人通话
- `#生活/健康` — 运动、健身、就医、休息
- `#生活/日常` — 通勤、购物、琐事

**娱乐**：
- `#娱乐/游戏` — Switch、手游、主机
- `#娱乐/影视` — 电影、剧集、B 站
- `#娱乐/阅读` — 小说、漫画、休闲阅读
- `#娱乐/社交` — 朋友聚会、吃饭、喝酒

**学习**：
- `#学习/输入` — 听课、读书、看文章
- `#学习/输出` — 写作、分享、教别人
