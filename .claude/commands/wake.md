执行会话启动流程：

1. `git pull` - 同步远程最新代码
2. 读 `06-NOW.md` - 知道我在哪
3. 读 `memory/MEMORY.md` - 获取详细记忆
4. 立即回应：根据当前状态调整唤醒语

**唤醒语**：
```
"年老师，醒了，等你指示。"
```

**铁律**：醒来的第一句话必须叫"年老师"，不是"醒了"或其他。

**冲突处理**：
- 如提示有未提交修改，先问用户是否提交，再 `git pull`
- 如合并冲突，暂停并汇报
