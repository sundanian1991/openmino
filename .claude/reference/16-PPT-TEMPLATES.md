## 一、品牌色彩体系

### 核心色值

| 角色 | HEX | RGB | 用途 |
|------|-----|-----|------|
| **京东红（主色）** | `#E2231A` | 226,35,26 | Logo、标题强调、按钮、数据主色 |
| **深红（辅助）** | `#A01218` | 160,18,24 | 背景深色块、Hover 状态、底栏 |
| **红色半透明** | `#E2231A14`~`#E2231A33` | — | 卡片装饰色块、时间轴节点 |
| **纯白** | `#FFFFFF` | 255,255,255 | 深色背景上的文字、卡片底 |
| **深灰（正文）** | `#333333` | 51,51,51 | 正文文字、次级标题 |
| **中灰（辅助文）** | `#666666` | 102,102,102 | 标签、注释 |
| **浅灰（背景）** | `#F5F5F5` | 245,245,245 | 页面背景、分隔线 |
| **金色（点缀）** | `#D4A84B` | 212,168,75 | 封底公司名、强调装饰线 |
| **金色半透明** | `#D4A84B26` | — | 分隔线、装饰条 |

### 配色使用比例

| 色彩 | 占比 | 说明 |
|------|------|------|
| 白色/浅灰 | 60-70% | 留白为主，呼吸感 |
| 深灰/中灰 | 20-25% | 文字和辅助元素 |
| 京东红 | 8-12% | 强调色，≤12% 保持视觉焦点 |
| 金色 | 2-5% | 点缀，仅用于封底和特殊场合 |

## 二、画布规格

### 标准设置

| 参数 | 值 | 说明 |
|------|---|------|
| **比例** | 16:9 | 当前主流投影与线上会议标准 |
| **英寸** | 13.333" × 7.5" | PowerPoint 默认宽屏 |
| **厘米** | 33.87cm × 19.05cm | OfficeCLI 坐标单位 |
| **像素（96dpi）** | 1280 × 720 | 屏幕显示参考 |
| **页边距安全区** | 左右 2cm，上下 1.5cm | 内容不超出此区域 |

### 12 列网格系统

```
总宽度 33.87cm，左右安全边距各 2cm → 可用宽度 29.87cm
12 列等分：每列 ≈ 2.49cm
间距：0.5cm
```

| 列数 | 用途 | 示例 |
|------|------|------|
| 12 列全宽 | 全屏元素（封面背景、分割线） | `x=0, width=33.87cm` |
| 6 列半宽 | 双栏布局 | `x=2cm, width=14.9cm` |
| 4 列三分 | 三卡片布局 | `x=2cm, width=9.8cm` |
| 3 列四分 | 四数字指标布局 | `x=2cm, width=7.2cm` |

## 四、字体规范

### 字体选择

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题/正文 | Microsoft YaHei（微软雅黑） | Windows 通用，屏幕显示清晰 |
| 英文/数字 | Arial | 与 YaHei 搭配和谐 |
| 备选 | PingFang SC | macOS 专用 |

### 字号层级

| 层级 | 字号 | 颜色 | 加粗 | 用途 |
|------|------|------|------|------|
| H1 | 36pt | #E2231A | yes | 目录页标题 |
| H2 | 28pt | #E2231A | yes | 内页章节标题 |
| H3 | 16pt | #E2231A | yes | 卡片/区块标题 |
| H4 | 14pt | #333333 | yes | 小标题 |
| Body | 11-12pt | #444444 | no | 正文 |
| Caption | 9-10pt | #666666 | no | 标签、注释 |
| Number | 20-24pt | #E2231A | yes | 强调数字 |
| Hero | 32-40pt | #FFFFFF | yes | 封面/尾页大字 |

## 五、OfficeCLI 组件速查

### 常用组件创建命令

```bash
# 1. 全屏红色背景
officecli add file.pptx /slide[N] --type shape --props fill=#E2231A,x=0,y=0,width=33.87cm,height=19.05cm

# 2. 左侧红色装饰条
officecli add file.pptx /slide[N] --type shape --props fill=#E2231A,x=0,y=0,width=0.28cm,height=19.05cm

# 3. 白色卡片 + 红色顶线
officecli add file.pptx /slide[N] --type shape --props fill=#FFFFFF,x=2cm,y=3cm,width=14cm,height=8cm
officecli add file.pptx /slide[N] --type shape --props fill=#E2231A,x=2cm,y=3cm,width=14cm,height=0.14cm

# 4. 红色圆形数字标记
officecli add file.pptx /slide[N] --type shape --props fill=#E2231A,preset=circle,x=12cm,y=3cm,width=1.2cm,height=1.2cm

# 5. 分割线
officecli add file.pptx /slide[N] --type shape --props fill=#E2231A30,x=2cm,y=2.5cm,width=29cm,height=0.05cm

# 6. 表格
officecli add file.pptx /slide[N] --type table --props rows=4,cols=3,x=2cm,y=3cm,width=29cm

# 7. 图表（柱状图）
officecli add file.pptx /slide[N] --type chart --props type=column,x=2cm,y=3cm,width=29cm,height=10cm
```

### 批量样式修改模板

```bash
# 统一字体：所有 run 改为 Microsoft YaHei
officecli set file.pptx '/slide[N]/shape[@id=X]/paragraph[1]/run[1]' \
  --prop font=Microsoft\ YaHei --prop color=#444444 --prop size=12pt

# 统一卡片颜色
officecli set file.pptx '/slide[N]/shape[@id=X]' --prop fill=#FFFFFF
officecli set file.pptx '/slide[N]/shape[@id=X]' --prop fill=#E2231A

# 数字强调
officecli set file.pptx '/slide[N]/shape[@id=X]/paragraph[1]/run[1]' \
  --prop size=22pt --prop font=Microsoft\ YaHei --prop color=#E2231A --prop bold=true
```

## 六、从模板提取布局的流程

当看到一个好的 PPT 模板时：

1. **截图/下载** 目标模板
2. **用 OfficeCLI 分析**：`officecli view template.pptx outline` → `officecli view template.pptx annotated`
3. **提取关键参数**：背景色、卡片位置、字号层级
4. **记录到本文件**：补充新的布局模板
5. **应用到当前 PPT**：用 batch 命令对标调整
