## 二、承诺事项

本人郑重承诺，在使用甲方提供的企业微信账号期间，严格遵守以下规定：

### 2.1 账号使用规范

1. **账号归属**：本人知悉并同意，所使用的企业微信账号归甲方所有，本人仅有使用权，无所有权。
2. **本人使用**：账号仅限本人使用，不转借、出租、出售给任何第三方。
3. **专号专用**：账号仅用于工作目的，不用于处理私人事务。
4. **密码安全**：
   - 妥善保管账号密码，不泄露给他人
   - 不使用过于简单的密码（至少8位，含数字+字母）
   - 定期更换密码（至少每90天一次）
   - 不在第三方平台使用相同密码
5. **配置管理**：不擅自修改账号配置（头像、昵称、权限等），如需修改需经甲方批准。

### 2.2 设备使用规范

1. **指定设备**：账号仅在甲方指定的设备上登录，不在其他设备登录。
2. **设备备案**：使用前完成设备备案（设备号、IMEI），更换设备需提前申请。
3. **安全软件**：工作设备安装指定的安全软件，开启设备锁屏密码。
4. **禁止操作**：不在Root/越狱设备上登录账号。

### 2.3 数据安全规范

1. **禁止导出**：不导出、复制、截屏、拍照任何客户数据。
2. **禁止传输**：不通过非授权渠道（个人微信、私人邮箱等）传输客户数据。
3. **禁止存储**：不在个人设备、私人云盘存储任何客户数据。
4. **数据脱敏**：涉及客户敏感信息时，遵守脱敏展示规则。
5. **操作留痕**：知悉所有操作将全程留痕，接受甲方审计。

### 2.4 服务品质规范

1. **话术规范**：使用甲方提供的标准话术库，不使用未经审核的话术。
2. **响应时效**：按甲方要求的时效标准响应客户消息（VIP客户5分钟内，普通客户30分钟内）。
3. **服务态度**：保持专业、礼貌的服务态度，不使用辱骂、威胁性语言。
4. **禁止承诺**：不向客户做出未经授权的承诺。
5. **合规营销**：不频繁群发消息，不骚扰客户，不夸大宣传。

### 2.5 离职交接规范

1. **提前通知**：离职前提前通知甲方，完成工作交接。
2. **账号归还**：离职时立即归还账号，配合甲方完成账号回收。
3. **数据清除**：离职后删除本地所有客户数据，不保留任何备份。
4. **保密义务**：离职后仍需保守甲方商业秘密和客户信息。

## 三、违约责任

本人知悉并同意，如违反上述承诺，甲方有权采取以下措施：

| 违规等级 | 违规行为 | 处罚措施 |
|----------|----------|----------|
| 轻度（L） | 首次轻微违规、操作失误 | 警告 + 培训 |
| 中度（M） | 话术违规、响应超时、服务态度问题 | 罚款1000-5000元 + 通报 |
| 重度（H） | 私自导出数据、未授权换设备、连续失联 | 罚款5000-20000元 + 警告 + 暂停 |
| 严重（S） | 泄露数据、私下收款、恶意删客户 | 立即解约 + 全额赔偿 + 法律追责 |

**本人知悉**：
1. 罚款将从本人工资/服务费中扣除，由所属供应商代为执行。
2. 给甲方造成损失的，本人承担全额赔偿责任。
3. 涉嫌违法犯罪的，甲方将移送司法机关处理。

5. **违约责任**：违反保密义务的，甲方有权追究法律责任。
