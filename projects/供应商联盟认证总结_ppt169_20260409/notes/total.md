# 03_方法论总览

# 04_坐席分级管理

[过渡] 让我详细展开一下坐席分级管理的核心机制。

ABC分级竞争机制有三个核心原则：第一是同等级竞争，C级不能越级和A级竞争，保证低能力坐席的竞争信心；第二是清晰的晋升路径，C打赢C级可升B级，B打赢B级可升A级；第三是名单量差异化，A级比C级多15条，额外奖励给能力突出的批次。

[停顿] 但这里有一个重要的失败教训需要分享。我们曾参照王者荣耀段位给坐席分级，刚开始效果很好，坐席会主动加班冲业绩。但问题是分级后直接对应固定名单额度，低段位坐席会认为自己永远升不上去，产生不公平感，慢慢产生倦怠，效果逐渐下滑。

要点：(1) 同等级竞争保证公平感 (2) 清晰晋升路径维持动力 (3) 固定名单额度导致"阶层固化"是失败核心
时长：3分钟

# 05_激励体系设计

[过渡] 接下来讲一下日周维度的即时激励方案。

日维度分为三个时段：上午开出2单可抽一次16到88元不等的红包，下午开出3单、6单各可再抽一次。周维度在大盘人均不到2单的周期，周开12单即可抽一次50到300元不等的奖品，抽到直接现场拿走。

[数据] 这种即时正向反馈能显著提升员工内驱力。但关键是不要天天做激励，一般放在月头第一周、月尾做，带动整体开单节奏即可。频繁做会降低激励价值。另外，团队PK活动比发一百块现金更有效——输的一方需要表演节目、给赢方买水、鞠躬，靠活动刺激性调动情绪。

要点：(1) 分时段即时激励 (2) 控制频率避免价值稀释 (3) 团队PK比现金更有效
时长：2分钟

# 06_名单管控策略

[过渡] 名单是电销的核心资源，精细化管控至关重要。

名单全生命周期管理包括六个关键环节：新名单静置2-3天不动用，给名单"冷静期"；单名单拨打频次控制在40次以内，超频停止拨打；18:30-19:30是一天中接通率第二高的黄金时段，全部用来拨打新名单；每周一全量回收所有坐席手中的名单，请假就直接清零；给客户分ABCD标签，优先A级优质客户；5万额度以上的高转化名单，优先分配给态度、能力、结果指标都最优的坐席。

[停顿] 这些规则的核心是：让名单资源不被闲置，让高价值资源匹配高能力坐席，最大化整体产出。

要点：(1) 名单静置、频次控制、黄金时段 (2) 周末回收、客户分级、高价值匹配 (3) 最大化名单资源产出
时长：2分钟

# 07_三会管理流程

[过渡] 三会是执行落地的关键抓手，各有明确定位。

早会控制在15分钟以内，核心做三件事：复盘昨日业绩数据、拆解今日目标、明确今日关键动作。负面问题放在晚会上说，避免影响当天状态，不安排培训，只做结果对齐和打鸡血。

午会一般安排在午休结束后，控制在10分钟以内，核心是告知下午业绩缺口，做叫醒动作。要求全员站着开，帮助坐席快速进入工作状态，周五可增加小游戏环节提神。

晚会时间可以拉长，核心做复盘培训：逐一分析当天最高和最低业绩的原因，听录音做辅导，筛选次日名单、定明日目标，汇总所有坐席提出的问题，哪怕是基础问题也要逐个解决，不遗留问题到第二天。

要点：(1) 早会对齐目标、午会唤醒状态、晚会复盘培训 (2) 时间控制严格、各有定位 (3) 问题不过夜、闭环解决
时长：2分钟

# 09_岐力落地实施方案

[过渡] 岐力作为本次认证的第一名，制定了详细的标准化落地三阶段方案。

第一阶段是建立初步认知，管控准入端，执行话术模式表和人员画像要求，采用新人新办法老人老办法。目标是将新员工手动流数据降低50%，让管理层养成每日数据查看习惯。

第二阶段用2-3个月做培优，覆盖25%-40%员工，用标准化制度管理，打造核心团队，安排专项预算支持。给存量30%潜力员工倾斜资源，选拔核心团队。

第三阶段将验证成熟的管理制度、工具迭代后推广到全项目，实现全团队标准化。保障措施是已经启动专项小组，由组织发展部和合规部牵头，分阶段由不同部门牵头推进，所有任务绑定绩效考核。

要点：(1) 分阶段推进：认知→培优→推广 (2) 专项小组保障、绩效考核绑定 (3) 新员工手动流数据降低50%
时长：2分钟

# 10_汇讯学习总结

[过渡] 汇讯在管理工具方面有很深的积累，同时也识别出了待补充的部分。

现有工具已经落地了BI看板和WPS智能看板，可实现多维度、分周期、分组到人数据追踪，且能和钉钉绑定定时推送。回去后计划补充现场大屏实时展示。

但同时也识别出了四个待补充的管理工具：夕会问题记录表，确保每日问题闭环解决；新人30天成长记录表，追踪新人培训进度；60秒非有效名单回收流程，提升坐席名单使用效率；标准化质检流程，第一时间同步问题解决方案。

[停顿] 汇讯还提出了几个管理问题：招聘目前由人力部门做面试、业务部门做培训，在人力需求大的背景下，探讨项目经理和主管是否需要直接参与面试；需要主办方分享最优标准化带教流程；提议参会职场共建共享话术库，由罗明老师审核后统一更新。

要点：(1) 已落地BI+WPS看板 (2) 待补充4个管理工具 (3) 提出3个管理问题需要探讨
时长：2分钟

# 12_落地推进计划

[过渡] 基于各供应商的收获，我们制定了统一的三个月推进计划。

第一个月是基础搭建：完成名产梳理打标，搭建标准化SOP和话术库框架，建立数据体系和评估标准。核心目标是完成基础数据梳理，建立标准化框架。

第二个月是落地执行：落地执行新流程，新人团队全面执行新标准，老团队逐步推进可落地的措施。核心目标是新团队全面执行，老团队逐步跟进。

第三个月是复盘优化：建立复盘机制，完成试点验证，用新成果倒逼老团队改变。核心目标是验证效果形成闭环，全面推广准备就绪。

要点：(1) 第一个月：基础搭建 (2) 第二个月：落地执行 (3) 第三个月：复盘优化
时长：2分钟

# 13_关键成功要素

[过渡] 为了确保标准化运营落地效果，我们制定了六个关键量化指标。

人员管理方面，ABC梯队建设覆盖率要达到100%。激励体系方面，日周月分层激励参与率要大于80%。数据驱动方面，每日数据查看执行率要达到100%。

话术迭代方面，月度话术更新及时率要达到100%。三会执行方面，早会午会晚会达标率要大于95%。名单效率方面，名单利用率要提升大于30%。

[数据] 以上指标为标准化运营落地的关键成功要素，需定期追踪、公开公示、与绩效挂钩。只有通过可量化的指标体系，才能确保标准化运营落地效果可追踪、可验证。

要点：(1) 6个关键量化指标 (2) 定期追踪、公开公示、与绩效挂钩 (3) 确保落地效果可追踪、可验证
时长：2分钟
