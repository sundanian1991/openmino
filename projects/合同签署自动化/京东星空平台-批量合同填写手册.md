## 组件交互速查

| 组件 | 类型 | 交互方式 |
|------|------|---------|
| 合作意向 | 自定义 findRelated 弹窗 | 点击 input → 弹窗搜索 → 选 radio → 确定 |
| 合同模板 | el-select remote | `vue.$props.remoteMethod()` + `vue.$emit()` |
| 合同类型/流向/期限 | el-select 普通 | click input → 遍历 `.el-select-dropdown__item` |
| 合同开始/结束时间 | el-date-picker | fill + Enter 确认 |
| 客方签约方标识 | 浮动 listitem | `page.locator(...).first().click()` + `getByRole('listitem')` |
| 我方签约方标识 | 浮动 listitem（不可见） | **必须** `page.evaluate()` 绕过可见性检查 |
| 开票税率 | el-select 被遮挡 | `click({ force: true })` |
| 模板变量 | el-input | `getByRole('textbox', { name: '请输入${xxx}' }).fill()` |

## 已知坑与对策

1. **合作意向对话框未关闭**：搜索词避免括号（用"汇讯"而非"汇讯(湖北)"），确定后加 `Escape` 兜底
2. **我方签约方标识不可见**：Playwright 认为 `input[placeholder="请选择签约方标识"]:last` 不可见，必须用 `page.evaluate(() => document.querySelectorAll(...)[last].click())`
3. **开票税率被遮挡**：发票类型的 tags 元素覆盖在税率 select 上，必须 `force: true`
4. **合同模板 disabled 状态**：必须用 Vue remoteMethod，不能直接 click
5. **时间字段**：fill 后必须 Enter 确认，否则不会生效
6. **fullPage screenshot 会破坏页面**：不要在有未保存表单时用 fullPage screenshot

### 核对脚本（browser_run_code）

```
async(page)=>{
  const F=['${party_b_name}','${party_b_address}',...,'${uniform_credit_code}'];
  const R={};
  const B=page.getByRole('button',{name:'编辑'});
  for(let i=OFFSET;i<Math.min(OFFSET+3,await B.count());i++){
    await B.nth(i).click();
    await page.waitForTimeout(3500);
    const v={};
    for(const f of F){
      try{v[f]=await page.getByRole('textbox',{name:'请输入'+f}).inputValue({timeout:2000});}
      catch(e){v[f]='ERR';}
    }
    R[v[F[0]]]=v;
    await page.getByRole('button',{name:'取消'}).click({timeout:3000});
    await page.waitForTimeout(2500);
  }
  return JSON.stringify(R);
}
```
