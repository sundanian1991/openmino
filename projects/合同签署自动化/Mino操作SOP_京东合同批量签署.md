## 单家操作流程（9步）

### Step 1: 打开浏览器 + 导航到签约页面

```
browser_navigate → https://xingkong.jd.com/qudao/793559463196729346
browser_snapshot → 确认已登录
```

如果未登录 → 暂停，等年老师手动登录后继续。

### Step 2: 点击"新建E签"

```
browser_click → ref: 包含"新建E签"文本的button
wait → 2s
snapshot → 确认表单页面已加载
```

页面结构：5个区块的表单。

### Step 3: 选择合作意向

这是**弹出对话框**模式，不是页面内下拉。

**操作序列**：
1. 找到"合作意向"标签旁的输入框 → click
2. 等待弹出对话框（含搜索框 + 列表 + 分页）
3. 在搜索框输入供应商名称
4. 等待列表刷新（通常只剩1-2条匹配结果）
5. 选中匹配行的 radio
6. 点击"确定"按钮

**自动带出字段**：伙伴名称、合作模式（渠道代理）、客方签约主体信息

**关键**：搜索后才精准，否则需要翻页找。

### Step 4: 选择合同模板（Remote Select）

这是 **el-select remote** 组件，普通点击无法触发搜索。

**已验证方案：通过 Vue 实例设值**

```javascript
// 1. 找到合同模板的 el-select 组件
// 2. 获取其 __vue__ 实例
// 3. 调用 remoteMethod 搜索
// 4. 等待 options 加载
// 5. 设置 value 为目标选项的 value
```

目标选项：`客户服务外包合同-202604`

**注意**：需要先 snapshot 确认 el-select 的 DOM 位置，因为页面可能有多个 el-select。

### Step 5: 填写合同信息

| 字段 | 操作方式 | 目标值 |
|------|---------|--------|
| 合同类型 | 普通 el-select click → 选"渠道主协议" | 渠道主协议 |
| 资金流向 | 普通 el-select click → 选"支出" | 支出 |
| 合同期限 | 普通 el-select click → 选"固定期限" | 固定期限 |
| 是否宿迁承接对账 | **选"固定期限"后自动出现** → radio选"是" | 是 |
| 客方签约方标识 | 下拉选"乙方" → 点"完成" | 乙方 |
| 我方签约方标识 | 下拉选"甲方" → 点"完成" | 甲方 |

**合同期限选"固定期限"后**：
- 自动出现"是否宿迁承接对账"字段
- 日期字段可能自动带出（2026-06-01 至 2027-05-31），需 snapshot 确认

### Step 7: 填写发票信息

| 字段 | 操作方式 | 目标值 |
|------|---------|--------|
| 发票类型 | 普通 el-select | 增值税专用发票 |
| 开票税率 | 普通 el-select（可能被遮挡，需scroll或force click） | 6% |

**开票税率注意**：下拉可能被页面底部遮挡。解决：
1. 先 scroll 到发票区域
2. 或用 evaluate 直接触发 el-select 的 visible-change

### Step 8: 截图确认 + 保存

```
browser_take_screenshot → 保存当前表单状态
browser_click → 点击"保存"按钮
wait → 3s
snapshot → 确认保存成功（页面跳转回列表 或 弹出成功提示）
```

### Step 9: 继续下一家

保存成功后，页面回到合同列表。直接回到 Step 2 点击"新建E签"。

## 加速策略

### 当前效率

- 第1家（河北创博）：探索+调试，约20分钟
- 后续每家预估：**3-5分钟**（纯执行，无探索）

### 加速方法

1. **批量 evaluate 设值**：Step 6 的23个模板变量，用一个 evaluate 批量完成
2. **模板化操作**：每家的操作序列完全一致，用固定步骤执行
3. **减少 snapshot**：只在关键节点 snapshot（选模板后、填完变量后、保存后）
4. **并行数据准备**：在操作当前家的同时，用 evaluate 预读下家的 Excel 数据

### 可进一步优化的点

- **日期字段**：如果选"固定期限"后日期自动带出，可以跳过
- **合同金额**：固定为0，可能在选择合作意向后自动带出
- **合作意向搜索**：如果供应商名精确匹配，搜索后通常只有1条结果，可以直接选中

## 已踩过的坑

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 合同模板下拉点不开 | remote select，需要输入触发搜索 | Vue实例设值或remoteMethod搜索 |
| 开票税率下拉被遮挡 | 页面底部遮挡 | scroll到发票区域再点击 |
| 合同期限选后多出字段 | "固定期限"触发显示"是否宿迁承接对账" | 正常现象，继续填 |
| 模板变量label含特殊字符 | `${}` 在选择器中需要转义 | 用evaluate遍历DOM设值 |
