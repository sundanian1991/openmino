## 视觉策略

### 图表类型（每块数据要传达什么判断）

| 数据 | 要表达的判断 | 图表类型 | 视觉策略 |
|------|-------------|---------|---------|
| 周末 vs 工作日个人使用占比 | 周末个人使用明显上升 | 折线图（7日×1线） | 周末灰色底，关键线加粗 |
| 一天中请求类型分布 | 每个时间的活动模式不同 | 折线图（多条线） | 核心线（食谱/新闻）高亮，其余灰化 |
| 夜间周末高端职业占比 | 高端职业更易在非传统时间工作 | 柱状图（4个分位） | 按分位排序，颜色渐变 |
| 税务截止日前后占比 | 截止日前巨大峰值 | 折线图（2线：US vs 全球） | 截止日竖线标注 |
| 产出物分类占比 | 解释/文档/指导是主力 | 水平条形图 | 排序：从高到低 |
| 产出物用途拆分 | 工作/个人/课程 | 堆叠条形图 | 三种颜色：蓝/灰/绿 |
| Token 消耗 vs 工资 | 高工资→高 Token | 散点图 + 拟合线 | 对数刻度，标注异常值 |
| 自主性 vs 表面 | Claude Code 自主性更高 | 分组柱状图（Chat vs Code） | 两种颜色 |
| 职业暴露分布 | 大多数预期 AI 能做更多 | 分布图（双色散点） | 双色区分 today/12m |
| 自动化与乐观的关联 | 自动化越深越乐观 | 回归系数图（6个维度） | 正负用色区分，置信区间 |

### 核心视觉形式（本次新增）

| 规则 | 说明 |
|------|------|
| **折线图 + 灰色区域** | 周末用灰色底色块突出，工作日白底，比多色填充更清晰 |
| **散点图 + 拟合线** | 工资 vs Token 用对数刻度，标注异常值（药剂师等） |
| **分组柱状图** | Chat vs Code 用双色，按输出类型分组 |
| **回归系数图** | 6 个维度（工资、安全、意义等），正/负用红/灰区分 |
| **分布图** | today vs 12m 双色，点表示职业，线表示变化 |

### 施工规则（通用化）

1. **总览横条**：多个章节的报告用横条摘要，不用网格
2. **图表驱动**：每个章节至少 4 张图，图是主体不是装饰
3. **折线图优先**：时间序列数据用折线图，关键线加粗/灰化其余
4. **散点图 + 对数刻度**：工资/Token 这种量级差异大的数据
5. **分组柱状图**：两个表面/两个时间点对比
6. **结论先行**：页首必须有总览 + 三章摘要
7. **附录分离**：方法细节和数据来源放附录
