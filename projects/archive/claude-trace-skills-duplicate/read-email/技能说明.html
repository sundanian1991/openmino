read-email

Version 1.0.1 Python + SOAP macOS / Windows

核心能力

突破"只能读标题"限制，通过 SOAP 协议直接访问京东邮件系统，获取完整的邮件正文、附件信息、文件夹结构。支持搜索、回复、转发、批量操作等全功能邮件管理。

智能启发式

pending-important

自动识别"最近收到但未回复的重要邮件"，基于以下启发式规则：

时间衰减：越近的邮件权重越高
重要性标记：高优先级邮件加权
回复状态：未回复的优先展示
发件人权重：上级/重要联系人加权

person-unhandled

查找来自特定人的未处理邮件，用于"某人找我但我还没回复"的场景：

按发件人筛选
仅展示未回复邮件
按时间倒序排列
包含邮件摘要

使用示例

# 查看文件夹结构 bash scripts/run.sh folders # 搜索最近1天的邮件 bash scripts/run.sh search --after 2026-06-24 # 查看邮件详情 bash scripts/run.sh detail --id "邮件ID" # 查找待处理重要邮件 bash scripts/run.sh pending-important # 查找某人的未处理邮件 bash scripts/run.sh person-unhandled --person "张三" # 回复邮件（需交互确认） bash scripts/run.sh reply --id "邮件ID" --content "回复内容"

安全规则

交互确认规则

  • 所有状态修改操作需要交互确认
  • 回复/发送/转发前必须预览内容
  • 批量删除前必须确认影响范围
  • 不自动执行任何写操作

执行规则

结果优先，不展示实现细节

使用本地脚本执行

关键突破

突破"只能读标题"限制

京东邮件系统默认界面只展示邮件列表，本技能通过 SOAP 协议获取完整邮件数据：

完整邮件正文（HTML/纯文本）
附件列表和大小信息
邮件头信息（CC/BCC/Reply-To）
完整的文件夹层级结构

Endpoint: https://mail-skill.jd.com/mail/api/clawmail/mailpost
