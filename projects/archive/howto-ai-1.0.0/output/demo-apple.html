06 — Build 路径指引
自建技能：临时数据查询汇总
从需求定义到上线运行，六步引导你完成第一个自建技能。
  1. 定义查询模板 列出最常收到的 5-10 种数据查询（供应商状态、绩效排名、合同到期、费用汇总、准入进度），每种写明输入字段和期望输出格式。
  2. 映射数据源 标注每种查询的数据来源（系统名/表名/导出路径），明确数据格式（Excel/CSV/API）和更新频率。
  3. 编写 Skill 描述文件 在 SKILL.md 中定义技能名称、触发条件、输入输出规范、处理逻辑。参考现有技能的结构。
  4. 编写处理 Prompt 将查询逻辑转写为 AI 可执行的 prompt：解析用户问题、匹配查询模板、提取参数、格式化输出。
  5. 本地测试 用 3-5 个真实查询案例测试，验证输出准确性和格式一致性。记录边界情况（模糊查询、多条件组合）。
  6. 迭代优化 根据测试结果调整 prompt，补充异常处理逻辑，增加用户反馈入口。

10 — 定时任务配置
让 AI 定时为你工作
以下 prompt 可直接复制到定时任务平台，每周一早 8:00 自动执行供应商准入材料审核。
Prompt / 每周一供应商准入审核
你现在是供应商管理岗的 AI 助手。请执行以下任务：

1. 读取本周新提交的供应商准入材料（路径：~/supplier-intake/本周/）
2. 使用 doc-xlsx 技能解析所有 Excel 文件，提取：
   - 营业执照信息（公司名、有效期、经营范围）
   - 资质证书信息（类型、等级、有效期）
   - 财务数据（注册资本、营收、利润率）
3. 使用 grill-with-docs 技能逐项核验：
   - 营业执照是否在有效期内（≥6个月）
   - 资质等级是否满足准入要求（对照准入标准文档）
   - 财务指标是否达标（注册资本≥100万，连续2年盈利）
4. 输出审核意见表：
   | 序号 | 审核项 | 结果 | 备注 |
   对每项标注：通过 / 不通过 / 需补充材料
5. 汇总异常项，生成待处理清单

输出格式：Markdown 表格 + 异常清单。语言：中文。
JoyClaw（小龙虾）

在 JoyClaw 定时任务面板中创建新任务：

触发条件：0 8 * * 1（每周一早 8:00）

技能链：doc-xlsx → grill-with-docs

将上方 Prompt 粘贴到任务指令框中，保存即可。

MyAgents

在 MyAgents 会话配置中设置定时触发：

Cron 表达式：0 8 * * 1

将上方 Prompt 作为会话首条消息模板，配置周一自动触发。

系统 Crontab

终端执行：crontab -e

添加以下行：

0 8 * * 1 cd ~/projects/ai-agents/my-agent && claude --skill doc-xlsx,grill-with-docs -p "$(cat ~/cron-prompts/supplier-audit.txt)" >> ~/cron-logs/audit-$(date +\%Y\%m\%d).log 2>&1

先将 Prompt 保存到 ~/cron-prompts/supplier-audit.txt
