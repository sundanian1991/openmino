# Intent — 岐力职场复贷 SABC 诊断

## 可视化目的
认知缺口：岐力复贷团队健康状况如何？哪些人需要干预？

## 信息类型
差异（等级分布差异、参评/非参评差异、司龄×等级交叉差异）

## 想传达什么
岐力复贷团队参评率低、C 级膨胀、新人质量差，需要立即干预

## 结论
读者应得出判断：岐力团队结构性问题严重，新人培训和低活跃激活是当务之急

## 叙事弧线
诊断型：全景扫描 → 问题定位 → 归因分析 → 行动建议

## 图表选择

| 图 | 类型 | 匹配理由 |
|----|------|---------|
| 图1 KPI 卡片 | 大号数字+红绿灯 | 关键数字不需要图表，直接用文字 |
| 图2 等级构成 | 水平条形图 | 类别比较（S/A/B/C 人数） |
| 图3 司龄×等级 | 热力图 | 二维矩阵模式识别 |
| 图4 参评vs非参评 | 对比表 | 两组多指标并排比较 |
| 图5 GMV分布 | 水平条形图 | 等级间 GMV 比较 |
| 图6 预警名单 | 数据表格 | 精确值查阅 |
| 图7 行动建议 | 文字卡片 | 结论收口 |

## 为什么不是别的
- 等级分布不用饼图（违反四条禁忌）
- GMV 分布不用折线图（不是时间序列）
- 不用双轴图（违反四条禁忌）
- 不用 3D 效果（违反四条禁忌）
