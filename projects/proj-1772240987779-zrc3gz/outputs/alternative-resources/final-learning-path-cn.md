### 方法 1：B 站缓存（最简单）

**步骤**：
1. 打开 B 站 App
2. 找到目标视频
3. 点击「缓存」按钮
4. 选择清晰度（推荐 1080P）

**优点**：简单、快速
**缺点**：仅限 B 站 App 内观看

---

### 方法 2：yt-dlp 下载（推荐技术用户）

**安装**：
```bash
# macOS (Homebrew)
brew install yt-dlp

# Windows (Chocolatey)
choco install yt-dlp

# Linux
sudo pip install yt-dlp
```

**下载 B 站视频**：
```bash
# 下载单个视频（1080P）
yt-dlp -f "best[height<=1080]" "https://www.bilibili.com/video/BV1Zn93YiEBZ/"

# 下载整个合集
yt-dlp -f "best[height<=1080]" "https://space.bilibili.com/3546759227771697/video"

# 下载字幕
yt-dlp --write-sub --skip-download --sub-langs zh-Hans "URL"
```

**优点**：支持多平台、可批量下载
**缺点**：需要命令行基础
