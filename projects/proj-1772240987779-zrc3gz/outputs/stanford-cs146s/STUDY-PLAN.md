# CS146S 学习计划 - 年老师版

## 学习目标

**核心目标**：掌握 AI 辅助开发技能，提升工作效率

**具体目标**：
1. 理解 Claude Code 的工作原理和最佳实践
2. 学会使用 MCP 工具扩展能力
3. 掌握 AI IDE 的配置和使用
4. 了解 AI 开发的安全风险和防护
