# 04_timeline

[Transition] 这张时间线展示了两年的关键节点。

起点是2024年四季度，DeepSeek带头降价，百万Token从几十元降到几毛钱。到2025年四季度，智谱率先推出Coding Plan，从按量付费转向固定月费订阅。中间经历了补贴收紧、Kimi切换Token计量。

转折点是2026年2月——OpenClaw在GitHub上拿到355K stars，Agent自动循环调用让Token消耗增长6到10倍。这直接催热了Coding Plan需求，也暴露了"不限量"模式的不可持续性。

随后，MiniMax将"Coding Plan"正式更名为"Token Plan"，腾讯最后入局。到了4月，阿里上线企业版Token Plan，小米以极低价入局。同时Anthropic从标准订阅中移除了OpenClaw访问——成本撑不住了。

[Pause]

要点：① OpenClaw是转折点 ② 7家全部上线 ③ 收紧信号出现

# 05_pricing_comparison

[Transition] 看一个很实际的问题：同样花40到50块钱，各平台给的到底多不多？

MiniMax Starter只要29元，但只有40 prompts每5小时。腾讯云39元给3500万Token，大约70轮深度问答。小米39元给6000万Credits。阿里40元给18000次API请求。Kimi和智谱都是49元。

[数据] 看起来百炼的18000次比MiniMax的40次多了几百倍，但这是不同计量单位，直接比数字毫无意义。这也是为什么90%的开发者容易踩坑。

要点：① 计量单位不同不能直接比数字 ② 90%开发者踩坑 ③ 同样价格实际差异巨大
时长：2分钟

# 06_measurement_systems

[Transition] 这五种计量单位并存，是最大的信息壁垒。

API请求次数，数字看起来很大但实际消耗不透明。Prompt次数，一次提问大约等于1200到1600次API调用，稍微直观一些。Token计量最透明，直接按输入输出计费。Credit计量抽象了一层但可以预测。时间窗口配额限制的是节奏而不是总量。

演变方向很清晰：从API次数到Prompt到Token再到统一Credits。方向是越来越透明、越来越可预测。

[数据] 但即使在Agent场景下，只有5-15%的Token真正用于代码生成，85-95%是上下文开销——文件读取、工具输出、上下文重发。

要点：① 五种计量单位并存 ② 演变方向是越来越透明 ③ 85-95%的Token是上下文开销
时长：2分钟

# 08_five_forces

[Transition] 用波特五力来分析这个市场的竞争格局。

行业内竞争极高——七家平台同池价格战，首月7.9元起，全线跟进补贴。买方议价力强——用户切换成本几乎为零，API Key一换就走。替代品威胁高——开源模型加本地部署持续压制定价上限。

供应商议价力中偏高——GPU是刚性成本，但国产芯片替代在降低依赖。新进入者威胁中等——门槛高但小米等新玩家仍在进入。

核心判断：七家同池价格战加上用户零切换成本，等于买方强势，定价权争夺白热化。

要点：① 行业内竞争极高 ② 买方议价力强 ③ 定价权争夺白热化
时长：2分钟

# 09_three_layer_drivers

[Transition] 定价模式的演变由三层驱动同时作用。

第一层是成本驱动。国产芯片放量、DeepSeek V4开源的MoE架构效率革命，都在降低Token成本。第二层是竞争驱动。价格战补贴圈用户，然后收紧测试价格弹性，最后精细分层。第三层是价值驱动——从卖Token到卖工作流再到卖生态。MiniMax的Token Plan不只卖编程，打包了图片、语音、音乐、视频。

要点：① 成本驱动 ② 竞争驱动 ③ 价值驱动
时长：1.5分钟

# 10_three_phase_rhythm

[Transition] 这是互联网定价的经典三段式，所有赛道都走这条路。

第一阶段补贴期，2024到2025年，抢用户规模建习惯，百万Token几毛钱。第二阶段收紧期，2025年下半年开始，取消首购优惠、停售低档、限时抢购，测试价格弹性。第三阶段分层期，2026年现在，极速版98到899元，企业版198到1398元，精细分层收割利润。

[Pause] 关键启示：进入新市场不要一开始就想好定价。先补贴跑量，再收紧测试弹性，最后精细分层。

要点：① 补贴→收紧→分层 ② 所有赛道通用 ③ 先进入者有先发优势
时长：2分钟

# 12_five_insights

[Transition] 第一条：计量单位即定价权。谁定义计量单位谁就掌握定价权，模糊计量利早期获客，透明计量利长期信任。

第二条：补贴-收紧-分层。所有互联网赛道都走这三步，先跑量再测弹性最后精细分层。

第三条：入口到生态。用编程高频刚需获客，用多媒体低频高价值场景提高ARPU。从"编程工具"升级为"AI工作室"。

第四条：国内外差异。国内七家抢用户比谁更便宜，海外面对Claude三巨头比谁更强。同一产品两套价格体系的合理性来源于竞争环境差异。

第五条：Agent时代成本不可逆增长。70%的coding agent token是浪费，实际账单是标价的2到5倍，订阅费是地板不是天花板。

要点：① 五条规律可跨场景复用 ② 适用于所有"算力即服务"商业模式
时长：3分钟

# 13_agent_cost

[Transition] 最后我们深入看一下Agent场景下的成本结构。

[数据] 一个开发者8个月Claude Code消耗100亿Token，按API价格计算超过15000美元。但Max plan只花了800美元，节省了93%。这说明什么？订阅模式下的成本放大效应。

Token构成中，文件读取占35-45%，工具输出15-25%，上下文重发15-20%，推理10-15%。真正用于代码生成的只有5-15%。也就是说，85-95%的Token消耗是Agent的上下文开销。

Cursor 2025年6月转credit-based后，重度用户一周超支350美元，引发了退款风波。这就是"订阅费是地板不是天花板"的真实写照。

要点：① 85-95%是上下文开销 ② 实际账单是标价2-5倍 ③ 订阅模式不可持续
时长：2.5分钟

# 14_future_trends

[Transition] 基于以上分析，我们对未来做四个趋势预判。

趋势一：Token成本将持续下降。国产芯片放量、MoE效率革命、推理芯片竞争，三股力量同时作用。利润空间将来自服务溢价而非算力差价。

趋势二：国产芯片替代加速。英伟达市占率从66%降到54%，预计今年跌破50%。华为、寒武纪、平头哥形成国产算力三角。国内厂商成本结构会和海外产生分化，定价策略可以更激进。

趋势三：开源模型重塑定价权。DeepSeek模式是开源加极致低价加不卖订阅。厂商的定价权将来自服务便利性而非模型独占性。

趋势四：Agent成本指数增长不可逆。推理需求已达训练的4到5倍，IDC预测2028年推理占73%。不限量模式不可持续，所有厂商最终都会转向精细计量。

[Pause] 核心结论：Coding Plan是过渡产品，Token Plan是终态。我们核心学到三件事——计量单位即定价权、补贴-收紧-分层节奏、入口到生态升级路径。

要点：① 四趋势 ② 核心结论 ③ 三件事
时长：3分钟
