## 一、先理解本次培训的定位

本次培训不是"工具使用教程"，而是 **"大额税票贷场景下的企微客户经营实战"**。

- 受众：11家供应商的项目经理和直接负责人（约20人），他们已经做过大额税票贷
- 关键背景：大额税票贷的**成交周期长、决策成本高、客户信任建立慢**，纯电话外呼容易在中途流失
- 你的任务：展示**企微怎么补位**——在电话触达之外，怎么用企微延长接触面、积累信任度、提高转化率

**核心要求：偏业务场景，不要偏工具功能介绍。** 大家不需要知道"企微怎么发消息"，需要知道"发什么、什么时候发、发给谁、怎么衡量效果"。

## 三、模块三：企业微信客户长期运营（45分钟）

### 核心议题
大额税票贷业务中，企业微信怎么成为电话外呼的有效补充？新动力是怎么做的？

### 分享内容清单（请按此结构准备PPT）

**3.1 为什么大额业务必须用企微经营（10分钟）**
- [ ] 大额税票贷的成交周期特征：从首触到放款平均需要多少天？
- [ ] 中间流失发生在哪些节点？（资料提交后等审批？审批被拒后没有跟进？）
- [ ] 电话外呼的局限：一次通话只能传递有限信息，企业主决策需要"反复确认"
- [ ] 企微的价值定位：**不是替代电话，而是延长接触面、积累信任度、提高转化率**
- [ ] 数据对比：用企微经营的客户 vs 纯电话跟进客户的转化率差异（如果有数据）

**3.2 客户分层与标签体系（10分钟）**
- [ ] 企微标签设计：
  - 按客户意向度（高/中/低）
  - 按资金需求紧迫度
  - 按企业规模
  - 按历史交互记录
- [ ] 标签怎么打：系统自动打 vs 坐席手动打？标准是什么？
- [ ] 分群策略：不同标签的客户进入不同的企微运营策略
- [ ] 互动设计：展示真实的标签体系截图（脱敏）

**3.3 企微运营SOP（15分钟）★ 核心**
- [ ] **首触加微**：
  - 什么时候加企微效果最好？（首通电话后？资料提交后？）
  - 加微话术怎么设计？通过率多少？
  - 展示真实加微话术（脱敏）
- [ ] **日常触达节奏**：
  - 朋友圈发什么内容？（行业资讯/产品优势/客户案例/活动通知）
  - 发几条？什么时间发？
  - 群发消息怎么发？频率多少？
  - 不同标签客户的差异化内容
  - 展示真实朋友圈截图（脱敏）
- [ ] **1v1私聊触发条件**：
  - 什么情况下主动私聊？（客户查看朋友圈后？客户点赞后？审批状态变化后？）
  - 私聊说什么？展示真实对话截图（脱敏）
- [ ] **跟进节点设计**：
  - 资料提交后：企微上怎么催进度、怎么安抚等待焦虑
  - 审批通过/被拒后：怎么通知、怎么引导下一步
  - 放款后：怎么维护关系、为复贷铺垫
- [ ] **沉默客户激活**：
  - 多久没互动算沉默？（7天？14天？30天？）
  - 激活策略：发什么内容、什么节奏？
  - 展示激活成功的真实案例（脱敏）

**3.4 内容素材库与话术（5分钟）**
- [ ] 朋友圈文案模板：准备 **3-5个可直接复用的模板**
- [ ] 1v1私聊开场白模板：准备 **2-3个开场白**
- [ ] 客户常见问题回复库：TOP5问题及标准回复
- [ ] 展示企微截图（脱敏），让现场直观看到"真实对话长什么样"

**3.5 效果衡量与问题规避（5分钟）**
- [ ] 看哪些指标：加微率、互动率、转化率、复贷率
- [ ] 常见踩坑（必须有真实教训）：
  - 过度群发导致客户删除/拉黑
  - 内容太"广告"导致反感
  - 坐席企微运营和工作量怎么平衡（每天花多少时间？）
- [ ] 合规边界：企微沟通中的合规红线

### 互动设计
- 讲完后预留 **5分钟Q&A**
- 预判会被问到的问题，提前准备好答案：
  - "你们加微通过率多少？"
  - "坐席每天花多少时间在企微上？"
  - "内容谁负责生产？坐席自己写还是统一提供？"
  - "企微运营对业绩提升的贡献怎么量化？"
  - "如果客户不通过企微好友申请怎么办？"

## 五、备场建议

1. **不要讲"企微功能介绍"**：大家不是来学"怎么发朋友圈"的，是来学习"在大额税票贷场景下怎么用企微经营客户"的
2. **多用截图、少用文字**：一张真实的对话截图比十页PPT更有说服力
3. **坦诚讲踩过的坑**：不要只讲成功案例，也要讲"我们试过的、失败了的、后来调整的"
4. **准备可复用的模板**：参会者最想要的是"直接能抄的作业"，尽量提供可直接复用的话术和模板
5. **控制时间**：45分钟很紧，建议内部彩排一次，确保每个章节不超时不赶时间
