## 风格族分类

### 族 A: 极致简洁白（White Minimalism）

| 风格 | 来源 | 核心特征 |
|------|------|---------|
| **Notion 温暖极简** | Notion | 暖灰白 + NotionInter + 极细边框(0.1透明度) + 压缩字距 |
| **Vercel 极客克制** | Vercel | 纯白 + Geist字体 + shadow-as-border + 工作流色彩 |
| **Airbnb 温暖杂志** | Airbnb | 纯白 + Cereal VF + Rausch红 + 圆润摄影优先 |
| **Webflow 工具精确** | Webflow | 纯白 + WF Visual Sans + 蓝强调 + 保守圆角 |

| **Spotify 沉浸剧院** | Spotify | 深黑(#121212) + 药丸形(9999px) + 重阴影 + 渐变强调 |
