# 权责不对等的政治本质

## 核心洞察

权责不对等不是"沟通问题"，是"权力结构问题"。调整意味着权力重新分配，阻力来自组织政治而非能力或意愿问题。

## 背景

在供应商管理岗的实际工作中，年老师多次尝试与策略组沟通权责问题（03-11分析、03-22记录、03-29再次讨论），但重复沟通无效。用户判断这是"政治问题"——权力重新分配的阻力太大。

## 具体表现

| 策略组 | 服务组 |
|--------|--------|
| 决策权（定价、名单、数据分配） | 建议权 |
| KPI: 费效比优化 | KPI: 供应商稳定/生态健康 |
| 压降价格 = 达成目标 | 供应商流失 = 承担风险 |

**结果**: 策略组达成目标，服务组承担后果。

## 为什么是政治问题

1. **调整阻力**: 让出定价权 = 策略组放弃核心权力
2. **KPI冲突**: 策略组KPI（费效比）与服务组KPI（供应商稳定）根本性冲突
3. **组织设计**: 权责分工是既定结构，涉及上级决策
4. **刘伟佳的行为模式**: 责任边界管理、分析外化、模糊化处理 — 这是理性自保逻辑，不是恶意

## 含义

这改变整个应对策略：

❌ **错误的路径**:
- 试图说服策略组让权
- 反复反馈问题期待自然调整
- 在不可控领域消耗精力

✅ **正确的路径**:
- **接受现实**: 这不是你能单边解决的
- **聚焦可控**: 在可控领域做到极致
- **保护自己**: 建立记录、明确边界、不揽责
- **等待机会**: 重大事故/VP关注时才有调整窗口

## 如何应用

当遇到权责问题时，先问自己：
1. 决策权在谁？ → 策略组/上级 = 不可控
2. 改变需要什么？ → 权力重新分配 = 政治问题
3. 我能控制什么？ → 自己的判断、表达、记录
