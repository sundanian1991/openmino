# B3-出差巡检准备

> 出发前系统化准备巡检清单、重点问题和检查流程

## 触发信号
- 月度巡检计划：每月2-3次现场巡检，每次1-2个站点
- FCI<75预警：能力指数低于75分触发专项巡检
- 红黄灯巡检：红黄灯供应商需现场复核
- 重大风险发现：群内异常信号/客诉/合规问题需现场核实

## 核心问题
- 巡检目标是什么？（验证数据/发现问题/验收整改）
- 重点检查哪几家供应商？为什么？
- 12项巡检标准如何快速检查？
- FCI能力指数怎么算？红黄绿判定规则是什么？
- 巡检后输出什么？整改要求怎么写？

## 思考步骤

**Step 1：确定巡检目标与名单**
```
巡检类型矩阵：
┌─────────────┬──────────────┬──────────────┬──────────────┐
│ 触发条件     │ 巡检类型     │ 核心目标     │ 检查重点     │
├─────────────┼──────────────┼──────────────┼──────────────┤
│ 月度常规     │ 例行巡检     │ 验证数据真实性 │ 12项标准全覆盖 │
│ FCI<75      │ 专项巡检     │ 定位能力短板  │ FCI七大项最低分 │
│ 红黄灯       │ 复核巡检     │ 验证预警真实性 │ 红色项整改情况 │
│ 重大风险     │ 紧急巡检     │ 核实问题      │ 相关项深度检查 │
└─────────────┴──────────────┴──────────────┴──────────────┘

巡检名单排序（按优先级）：
1. 🔴 红色FCI（<60）：必须现场，24-48h内出发
2. 🟡 FCI 60-75 + 红黄灯：优先安排，本周内
3. 🔵 首次预警供应商：常规巡检覆盖
4. 🟢 尾部供应商（后30%）：抽查
5. 🟢 S/A级供应商：可选，了解标杆做法

每次巡检≤2个站点（避免走马观花）
```

**Step 2：准备FCI速记卡（工具箱§十二）**
```
FCI能力指数（7大项加权）：
A. 资质（10分）：证照齐全/无异常
B. 绩效（25分）：业绩达成/排名趋势
C. 质检（25分）：客诉率/A类次数/整改率
D. 管理（15分）：团队稳定/出勤/主管胜任
E. 技术（10分）：系统工具/信息化
F. 财务（10分）：薪资发放/社保缴纳
G. 负面（5分）：G项红色一票否决

计算：
FCI总分 = A×10% + B×25% + C×25% + D×15% + E×10% + F×10% + G×5%

判定标准：
≥75分：合格（绿灯）
60-75分：预警（黄灯）
<60分：危险（红灯）

红黄绿判定（工具箱§十二）：
红=0且黄≤2：合格（可保持）
红≥2或G有红：危险（需立即整改）
```

**Step 3：12项巡检标准清单（工具箱§五）**
```
巡检检查表（每站点30-60分钟）：

空间布置（A资质）：
□ 场地面积是否符合合同？
□ 工位是否整齐/卫生？
□ 品牌露出是否规范（禁用竞品logo）？

人员状态（B绩效+D管理）：
□ 现场人数与人力表是否一致？
□ 员工精神面貌？（疲惫/松散/有序）
□ 主管是否在场？是否熟悉业务？

薪资（F财务）：
□ 薪资发放是否及时？（问员工）
□ 社保缴纳是否合规？（查记录）

招聘（B绩效+D管理）：
□ 招聘渠道是否合规？（禁用竞品渠道）
□ 面试流程是否规范？

品牌（A资质）：
□ 是否使用甲方logo？
□ 是否使用竞品logo？（严重违规）

管理（D管理）：
□ 主管是否清楚本月业绩目标？
□ 是否有培训记录/考试？
□ 是否有违纪处理记录？

活动（C质检）：
□ 是否有营销活动？是否合规？
□ 是否有客诉？如何处理？

作息（D管理）：
□ 上下班时间是否执行？
□ 加班是否合理/有记录？

培训（D管理）：
□ 是否有培训计划/记录？
□ 新员工是否培训上岗？

招聘（重复，重点）：
□ 是否存在虚假招聘？（承诺未提供的福利）

品牌滥用（A资质）：
□ 是否使用竞品名称/logo？（红线）

质检合规（C质检）：
□ 是否有A类客诉？如何处理？
□ 整改是否完成？
```
**检查方式**：
- 看：现场观察（空间/人员状态/品牌）
- 问：随机问员工2-3人（薪资/作息/培训）
- 查：查记录（入职/离职/薪资/培训/客诉）
- 核：核对人力表 vs 实际人数

**Step 4：重点问题预判（根据该供应商历史）**
```
针对每家供应商准备：
1. 该供应商近3个月异常信号：
   - 出勤异常？业绩断崖？流失加速？
   - 问题：_________________
   - 现场核实：_________________

2. 上次整改项：
   - 整改要求：[上次约谈的X条要求]
   - 完成情况：[是否按期完成？]
   - 现场验证：_________________

3. 双维度诊断：
   - 自身峰值比：[XX%] → [正常/下滑/严重未发挥]
   - 头部差距：[XX%] → [接近头部/有差距/差距大]
   - 类型判断：[标杆型/尽力型/问题型/暂时波动]
   - 现场验证点：_________________

4. 主管胜任5项：
   - 达标项：[X/5项]
   - 不达标项：[具体哪几项]
   - 现场观察：[主管状态/团队氛围]
```

**Step 5：巡检报告与整改要求**
```
巡检后2分钟内填写FCI速记卡（工具箱§十二）：
站点：[XXX]
日期：[YYYY-MM-DD]
FCI总分：[XXX分]
趋势：[↑/↓/→]
最低分项：[B3现场管理]
红色项：[1项 — 品牌违规使用竞品logo]
黄色项：[2项 — 出勤不稳定/培训缺失]
根因：[一句话，如：主管更换频繁导致管理松弛]
状态：[预警/危险/合格]

输出：
□ FCI速记卡（内部存档）
□ 巡检报告（operations-missing.md现场巡检报告模板）
□ 整改通知（如有问题，compliance-missing.md）
```

## 关键变量
- **巡检类型**：例行/专项/复核/紧急，决定检查深度
- **FCI分数**：决定巡检优先级（红>黄>绿）
- **历史问题**：上次整改项是否完成？是本次重点
- **供应商等级**：S/A级关注标杆做法，C/D级聚焦问题
- **时效要求**：红色FCI需24-48h内出发

## 常见误区
- ❌ 巡检前不做准备：现场发现问题效率低
- ❌ 12项标准全覆盖：应聚焦该供应商的历史问题
- ❌ 只看表面不看记录：问员工+查记录交叉验证
- ❌ 巡检后不及时填写：2分钟内填FCI速记卡，否则遗忘
- ❌ 发现问题不要求整改：必须有整改通知+截止时间
- ❌ 不跟踪闭环：整改后需复检，验证效果

## 决策输出

**巡检前准备清单**：
```
□ 确定巡检名单（按优先级排序：红FCI > 黄FCI > 首次预警）
□ 查该供应商近3个月数据（排名/异常信号/主管胜任）
□ 准备FCI速记卡（工具箱§十二模板）
□ 准备12项检查表（打印或手机）
□ 预约时间（提前1天通知供应商）
□ 准备整改通知模板（如有问题）
```

**巡检后FCI速记卡**（2分钟内填完）：
```
站点：毅航-杭州
日期：2026-04-19
FCI总分：68分
趋势：↓（上月72分）
最低分项：D管理（12/15分）
红色项：0项
黄色项：2项（D1出勤不稳定/D3培训缺失）
根因：主管本月更换，团队磨合期
状态：预警（60-75）
建议：约谈主管，要求提交整改计划（出勤+培训）
```

**现场问题记录**：
```
问题1：[品牌违规使用竞品logo] — 现场发现3处京东logo
问题2：[出勤率不足85%] — 问员工确认加班多离职多
问题3：[培训记录缺失] — 查近2周无培训记录

整改要求：
1. 品牌整改：48h内清除所有竞品logo，拍照反馈
2. 出勤恢复：下周出勤率≥90%，提交改善计划
3. 培训补录：3天内组织培训，留存记录
截止时间：2026-04-22 12:00
```

## 下一步衔接
