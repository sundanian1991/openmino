## 使用说明

### 调用决策树

```
开始（年老师日常工作）
  ↓
遇到问题 → 判断场景类型
  ├─ 同事@/领导问/供应商来电 → D类模板（30秒-5分钟快速响应）
  ├─ 看到异常信号/需要决策 → A类模板（10-15分钟系统分析）
  ├─ 拿到任务/新项目/模糊问题 → B类/C类模板（拆解规划）
  └─ 新供应商接触 → E类模板（初筛/转正评估）
```

### 与 official-doc 衔接

每个模板的"下一步衔接"字段已写明对应的 `official-doc` 模板文件名：
- 想清楚 → 输出结构化结论
- 调 official-doc 技能 → 生成公文
- 交付：邮件/报告/通知
