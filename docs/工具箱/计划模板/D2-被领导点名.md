# D2-被领导点名

> 军哥或王易人直接@你要解释某个供应商问题时，需要快速组织口径并判断 escalation 级别

## 触发信号
- 军哥问："毅航这个月为什么排到后30%？怎么回事？"
- 王易人说："跟进下赛维斯的客诉问题，给我个说法。"
- 领导在群里点名："@年老师，解释一下这个数据。"
- 领导私聊："刚才那个事，你怎么看？"

## 核心问题
- 领导问这个问题的真实意图是什么？（要事实？要判断？要解决方案？）
- 我掌握的信息足够吗？缺什么？
- 这个事的严重等级是什么？需要我当场决策还是回去查证？
- 回复的框架是什么？（事实→根因→已做动作→下一步）

## 30秒快速判断

**Step 1：识别提问层级**
| 提问人 | 意图判断 | 回复要求 | 时限 |
|--------|---------|---------|------|
| **军哥**（部门负责人） | 要根因+决策依据+风险判断 | 事实准确、根因清晰、有数据支撑 | 24h内 |
| **王易人**（服务组负责人） | 要事实+已做动作+下一步计划 | 简洁清晰、动作明确、不模糊 | 当场或2h内 |
| **其他领导** | 视情况而定 | 先确认意图再回复 | 先问清楚 |

**Step 2：紧急度分级**
- 🔴 **红色（立即）**：涉及合规红线、诚信问题、重大客诉
- 🟡 **黄色（2h内）**：业绩异常、排名大幅下滑、连续预警
- 🟢 **绿色（当天）**：常规进度询问、数据核对

**Step 3：信息完整性检查**
```
我有以下信息吗？
□ 基础事实（排名/业绩/问题发生时间）
□ 根因判断（工具箱§十六异常信号对应项）
□ 已做动作（约谈/整改/沟通记录）
□ 数据支撑（双维度/主管胜任/异常信号数据）
```
- 全有 → 可立即回复
- 缺1-2项 → "我需要核实，X点前回复你"
- 缺关键项 → "这个事情我需要先了解清楚，下班前给你完整答复"

**Step 4：回复框架（一句话模板）**
```
[领导称谓]，[供应商][问题简述]。根因是[1-2个关键判断]。我已[已做动作]。下一步[计划动作+时间点]。
```
示例：
- "军哥，毅航本月落入后30%（首次预警）。根因是新人占比45%导致人效稀释+出勤率下滑至87%。我已约谈主管，要求下周提交整改计划。"
- "易人哥，赛维斯客诉已核实，是招聘违规（用京东logo）。我今日已发整改通知，要求48h反馈。后续会纳入数据质量分考核。"

## 5分钟深入版（如有时间）

**Step 1：事实层（30秒内能说的）**
- 时间线：问题何时发生？数据何时变化？
- 量化指标：排名变化、GMV达成率、异常信号触发项
- 对比参照：同产线均值、头部供应商差距

**Step 2：根因层（需工具箱判断）**
- 引用工具箱§十六异常信号手册，匹配最可能的根因
- 区分：供应商自身问题 vs 甲方问题（名单/产品/系统）
- 用双维度/主管胜任数据支撑判断

**Step 3：动作层（已做+待做）**
- 已做：何时沟通？约谈谁？发了什么通知？
- 待做：下一步具体动作 + 完成时间点
- 需要领导支持：什么情况下需要你介入？（高层约谈/清退决策/资源调整）

**Step 4：风险与预案**
- 最坏情况：如果不整改会怎样？（进入连续预警→清退评估）
- 备选方案：如果供应商不配合，备选是谁？
- 需要领导决策点：明确写出"需要你确认的事项"

## 关键变量
- **领导层级**：军哥（战略层）要根因+风险；易人哥（执行层）要动作+进度
- **问题性质**：合规问题（红线级）> 业绩问题（运营级）> 进度问题（事务级）
- **历史背景**：该供应商是否有诚信记录？是否连续预警？
- **资源影响**：该供应商的份额是否重大？是否有备选？

## 常见误区
- ❌ 只摆数据不讲根因：领导要的是"为什么"不是"是什么"
- ❌ 把球踢回去："你觉得怎么办？" — 需给出建议方案
- ❌ 模糊表态："我们会跟进" — 需明确"我做什么+何时完成"
- ❌ 忽略历史背景：该供应商是否是诚信问题惯犯？
- ❌ 不标注风险等级：领导需要知道这事有多紧急

## 决策输出
**向军哥回复（战略层）**：
```
[军哥]，[供应商][问题简述+影响范围]。根因分析：[工具箱§十六匹配的根因，2-3点]。当前状态：[已做动作+供应商反馈]。风险判断：[如果不整改的后果]。建议：[我的方案+需要你支持的事项]。
```
示例：
```
军哥，毅航本月首次落入后30%（影响约800万月GMV）。根因是：①新人占比45%导致人效稀释（新人拉低幅度32%）；②出勤率下滑至87%。我已约谈主管，要求下周提交整改计划。风险：如果连续2个月后30%将进入正式约谈。建议：若下月未回升，考虑启动清退评估，需要您事前确认。
```

**向易人哥回复（执行层）**：
```
[易人哥]，[供应商][问题简述+当前状态]。我已[已做动作]，计划[下一步+时间点]。需要您支持：[如需审批/协调的事项]。
```
示例：
```
易人哥，赛维斯客诉已核实（招聘违规，用京东logo）。我今日已发整改通知，要求48h反馈。后续会纳入数据质量分考核。需要您支持：若整改不力的处罚决定，需您签字确认。
```
