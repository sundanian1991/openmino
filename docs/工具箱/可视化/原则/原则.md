## 一、前认知属性（Preattentive Attributes）

大脑在 **200ms 以内**、不需要有意识处理就能自动捕捉的视觉特征。这是所有"突出手法"的科学基础。

### 感知优先级

| 优先级 | 属性 | 处理速度 | 可视化应用 | 对应手法 |
|--------|------|---------|-----------|---------|
| 1 | **位置** | 最快 | 坐标轴定位、散点图 | 网格权重、留白包围 |
| 2 | **颜色色相** | 很快 | 分类高亮 | 颜色隔离 |
| 3 | **颜色深浅** | 快 | 数值编码 | 背景色块区隔 |
| 4 | **大小/长度** | 快 | 柱高、气泡大小 | 字号对比 |
| 5 | **粗细** | 快 | 线条权重 | 线条权重 |
| 6 | **形状** | 中 | 散点分类 | 点标注 |
| 7 | **方向/角度** | 中 | 较少用 | 引导线 |
| 8 | **封闭** | 慢 | 背景框定 | 分层叠加 |

**核心规则**：突出手法必须按优先级使用。先用位置/颜色，不够再加字号/粗细。叠加不超过 2 种。

### 处理时间对照

```
< 200ms（前认知）：颜色、大小、位置、粗细
200-500ms（认知）：文字阅读、数字比较
> 500ms（后认知）：逻辑推理、模式发现
```

出图目标：让观众在 **前认知阶段** 就抓住核心信息，不需要进入认知阶段。

## 二、格式塔原理（Gestalt Principles）

人脑自动组织视觉元素的 7 条规律。

### 1. 相似性（Similarity）

> 同色/同形/同大小的元素被视为一组

| 应用 | 手法 |
|------|------|
| 同色 = 同类 | 颜色隔离中，灰色元素自动被视为"背景组" |
| 同形 = 同类 | 数字卡片统一圆角 + 统一间距 |
| 不同 = 对比 | 核心元素用不同颜色/大小，自动跳出来 |

### 2. 接近性（Proximity）

> 靠近的元素被视为相关

| 应用 | 手法 |
|------|------|
| 卡片间距 = 关系远近 | 核心卡片间距小，次要卡片间距大 |
| 分组留白 | 用留白代替边框来分组 |
| 标签贴近 | 标注紧贴对应元素 |

### 3. 封闭性（Enclosure）

> 被框住的元素被视为一组

| 应用 | 手法 |
|------|------|
| 背景色块 | 浅色背景框出关键区间 |
| 卡片边框 | 数字卡片的圆角边框 |
| 太阳辐射 | 同心圆建立"核心-外围"关系 |

### 4. 连续性（Continuity）

> 沿路径/方向的元素被视为相关

| 应用 | 手法 |
|------|------|
| 引导线 | 箭头引导视线 A→B→C |
| 折线趋势 | 线条的连续走向暗示趋势 |
| 装饰弧线 | 右上角淡弧线增加层次 |

### 5. 连接性（Connection）

> 有连线的元素被视为关联

| 应用 | 手法 |
|------|------|
| 点+引线 | 关键节点用引线标注 |
| 流程图 | 箭头连接步骤 |
| 关系图 | 连线表示关系 |

### 6. 闭合性（Closure）

> 大脑自动补全不完整的图形

| 应用 | 手法 |
|------|------|
| 太阳辐射光晕 | 淡色圆形不需要完整闭合 |
| 虚线分隔 | 虚线暗示边界但不生硬 |
| 不完整圆环 | 留白的圆环更有呼吸感 |

### 7. 对称性（Symmetry）

> 居中/对称的元素被视为重要

| 应用 | 手法 |
|------|------|
| 居中大数字 | 指标卡居中 = 核心 |
| 对称布局 | 左右对比卡片 |
| 辐射中心 | 太阳辐射的核心点 |

## 三、视觉编码通道

数据映射到视觉属性的规则。

### 数据类型 → 编码方式

| 数据类型 | 推荐编码 | 不推荐 |
|----------|---------|--------|
| 分类（名义） | 颜色色相、形状 | 颜色深浅 |
| 分类（有序） | 颜色深浅、大小 | 颜色色相 |
| 数值（定量） | 位置、长度、面积 | 形状 |
| 时间 | 位置（X 轴） | 颜色 |

### 编码冲突检查

| 冲突 | 问题 | 解决 |
|------|------|------|
| 颜色 + 形状同时编码分类 | 认知超载 | 只用颜色 |
| 大小 + 颜色同时编码数值 | 双重编码 | 大小为主，颜色为辅 |
| 位置 + 颜色编码同一维度 | 冗余 | 保留位置，去掉颜色 |

## 四、框架式温暖设计

年老师确认的风格方向：温暖、克制、有品质感。

### 核心元素

| 元素 | 实现 | 格式塔原理 |
|------|------|-----------|
| 暖色渐变背景 | `linear-gradient` 多色阶 | 封闭性建立氛围 |
| 太阳辐射光晕 | `radial-gradient` 淡色圆形 | 闭合性 + 对称性 |
| 装饰弧线 | `border-radius` 椭圆弧线 | 连续性 |
| 页眉标签 | 小圆点 + 大写字母 | 相似性 |
| 标题局部高亮 | `<em>` 橙色文字 | 相似性对比 |
| 数字卡片 | 白色 + 淡边框 + 圆角 | 封闭性 + 接近性 |

### 五种变种

| 变种 | 做法 | 适用场景 |
|------|------|---------|
| **光晕版** | 单侧放射渐变 | 单页报告、指标卡 |
| **网格版** | 淡色网格背景 | 数据仪表盘 |
| **波浪版** | 底部波浪曲线 | 首页/落地页 |
| **几何版** | 淡色几何散点 | 创意报告 |
| **极简版** | 圆角 + 淡边框 | 正式汇报 |

## 五、配色优先级

日常出图 90% 应该在前 3 个里选。

| 优先级 | 组合 | Ramp | 感觉 |
|--------|------|------|------|
| **1** | 单色深浅 | Warm | 统一、高品质 |
| **2** | 暖+灰 | Warm + Stone | 温暖不花哨，首选 |
| **3** | 纯灰 | Stone 单色 | 极简不抢 |
| **4** | 暖+红 | Warm + Coral | 温暖+紧迫 |
| **5** | 单色深浅 | Coral | 负面、风险 |
| **6** | 单色深浅 | Sky | 干净、冷静 |
| **7** | 暖+蓝 | Warm + Sky | 专业+亲和 |
| **8** | 暖+琥珀 | Warm + Amber | 暖+警觉 |
| **9** | 暖+绿 | Warm + Sage | 暖+自然 |
| **10** | 暖+绿 | Warm + Teal | 语义明确需要 |
