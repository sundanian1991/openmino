# D2 被领导点名

> 领导突然问起某个供应商的情况，30秒内组织口径，既不能答不上来，也不能说错话。

## 触发信号

- 军哥（卞海军）问："XX供应商最近怎么回事"
- 王易人说："你跟进下XX供应商"
- 军哥在会上点名："XX的数据怎么样"
- 王易人转述军哥的话："军哥问XX了"
- 群里领导@你："@年老师 XX情况说明一下"

## 核心问题

领导点名问供应商，不是要你汇报所有情况，是要你回答一个具体关切。核心问题有三个：

1. 他为什么问？（出了什么事，或者听到了什么）
2. 他想知道什么结论？（是定性的判断，还是定量的数据）
3. 我说的会不会和已有信息矛盾？（领导可能已经从别人那里听到了版本）

## 30秒快速判断

> 谁在问？问谁？为什么问？我要回答到什么程度？

**第一秒 — 识别来者层级：**

| 来者 | 通常意图 | 你的权限边界 |
|------|---------|-------------|
| 军哥（上上级） | 听到风声、要决策依据、会上被问到 | 只汇报事实和你的判断，不代替王易人表态 |
| 王易人（直属上级） | 分配任务、了解情况、让你去处理 | 可以给判断，但涉及清退/处罚要确认 |
| 其他领导 | 一般是临时问一句 | 给简短结论即可 |

**第二秒 — 判断问询性质：**

| 问法 | 性质 | 要求 |
|------|------|------|
| "XX供应商怎么回事" | 出了问题，领导已知部分情况 | 必须主动说问题 + 原因 + 你的处理 + 当前状态 |
| "你跟进下XX" | 有事要处理，领导还没细说 | 先接任务，再确认具体要跟什么 |
| "XX的数据怎么样" | 要数据支撑某个判断 | 给关键指标 + 同比变化 + 你的判断 |
| "XX能不能用" | 要一个定性结论 | 给结论 + 依据 + 前提条件 |

**第三秒 — 快速扫描该供应商状态：**

脑中过一遍这5个维度（3秒内）：

| 维度 | 想什么 |
|------|--------|
| 排名 | 这家最近排名？趋势是升是降？ |
| 出勤 | 有效出勤率是否低于85%？ |
| 业绩 | 近4周GMV有没有断崖（降>20%）？ |
| 流失 | 单周离职有没有超过5%？ |
| 重大事件 | 最近有没有巡检问题、合规风险、约谈记录？ |

**一句话回复口径模板：**

> "[供应商]目前[定性状态]，[1个关键数据]，[如果有问题：已经在做XX处理/如果没问题：持续关注中]。"

**举例：**
- 军哥问"赛维斯怎么回事" → "赛维斯3月出勤率掉到78%，低于85%预警线，已和供应商确认是春节后扩编培训期导致，预计4月中旬恢复。"
- 王易人说"跟进下毛毛虫" → "好的，毛毛虫最近主要问题是金条产线转化率波动较大，我这周和他们的主管约了沟通。"

## 5分钟深入版

如果30秒回复后领导追问，或者问题本身比较复杂，按以下步骤准备详细口径。

## 思考步骤

### 步骤1：摸清领导的信息源

**想什么**：领导为什么突然问这个？他从哪里听到的？

**检查什么数据**：
- 最近3天的内部群消息，有没有人提过这家供应商
- 最近1周的巡检报告，有没有这家的问题
- 策略组（刘伟佳）最近有没有对这家发过什么
- 供应商自己的群里有没有异常动态

**判断什么**：
- 领导主动问 → 说明他听到了什么，你的回答要和已知信息对得上
- 领导转述问 → 可能是更上层在问，回答要更谨慎
- 领导分配任务 → 他可能比你知道得少，需要你补充完整背景

**举例**：军哥问"伽玛怎么回事"，你先想：最近伽玛有什么事？查一下发现上周伽玛周度GMV比4周均值降了25%，触发了业绩断崖信号。军哥可能是看到了周报。

### 步骤2：拉取该供应商核心数据

**想什么**：回答领导需要带数据，但不能带太多，抓关键。

**检查什么数据**（按优先级）：

| 指标 | 数据源 | 关注点 |
|------|--------|--------|
| 综合排名 | 月度排名表 | 当前位次 + 环比变化 |
| 出勤率 | 出勤统计 | 是否低于85% |
| GMV趋势 | 周度数据 | 近4周均值 vs 最新一周 |
| 离职率 | 人事数据 | 单周是否超5% |
| 巡检/整改 | 巡检记录 | 有无未关闭项 |
| 合规状态 | 合规台账 | 有无风险标记 |

**判断什么**：
- 哪些数据是领导关心的？按他的问法推测重点
- 哪些数据是亮点、哪些是雷？主动暴露雷比被动问出来好
- 数据口径要准确：说"截至X月X日"比说"目前"靠谱

### 步骤3：组织汇报口径

**想什么**：怎么把信息组织成领导一听就懂的结论？

**判断什么**：

按"结论先行 + 关键数据 + 原因分析 + 你的动作"组织：

> 结论："伽玛目前状态可控，但需要关注。"
> 数据："3月排名第5，环比下降2位。金条产线GMV近4周降了18%，接近20%预警线。"
> 原因："主要因为2月底流失了一批熟手，新人还在培训期。"
> 你的动作："上周已经和伽玛老板沟通过，要求4月底前补齐人力。本周会做一次现场巡检确认整改进度。"

### 步骤4：评估是否需要升级

**想什么**：这个事情需不需要王易人知道？需不需要正式处理？

**判断什么**（对照决策授权矩阵）：

| 情况 | 处理方式 |
|------|---------|
| 只是领导随口问 | 口头回答，不用升级 |
| 涉及供应商需正式约谈 | 自主执行，事前报备王易人 |
| 涉及要求更换主管 | 建议权，需王易人确认后执行 |
| 涉及清退评估 | 建议权，需王易人确认 + 军哥事前沟通 |
| 涉及约谈供应商高层 | 建议权，王易人必须陪同 + 军哥事前报备 |

**举例**：军哥问完伽玛，你说完口径后，如果发现伽玛的情况持续恶化，需要主动和王易人说："伽玛的情况我向军哥汇报了，目前在做XX处理。如果4月底没改善，建议启动约谈。"

### 步骤5：确认后续动作并闭环

**想什么**：回答完之后，怎么确保这件事不会在你手上断掉？

**检查什么**：
- 领导有没有给具体指示？记录下来
- 你承诺的动作有没有时间节点？设提醒
- 需不需要发正式文档留痕？

**判断什么**：
- 如果领导说"继续跟进" → 定一个回顾时间点，主动回来汇报
- 如果领导给了具体指示 → 照办 + 回复执行结果
- 如果涉及跨部门/供应商 → 发正式文档，口头说了不算

## 关键变量

| 变量 | 判断标准 | 影响 |
|------|---------|------|
| **领导关注程度** | 随口问一句 vs 连续追问 vs 会上点名 | 决定回复深度和后续跟进力度 |
| **供应商严重程度** | 有无触发3个突变信号（出勤<85%、GMV降>20%、离职>5%） | 决定是否需要升级处理 |
| **你的了解程度** | 你是否一直在跟这家供应商？ | 了解多→直接给结论；了解少→先说"我确认一下" |
| **信息一致性** | 你的数据和领导听到的版本是否一致？ | 不一致时先对齐事实，别急着解释 |
| **时间压力** | 领导要当场回答还是可以准备一下？ | 当场→30秒版；可以准备→5分钟版+正式文档 |

## 常见误区

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 被领导气场带着走，顺着领导的话说 | 领导的判断可能有偏差，你附和了就等于背书 | 先核实事实，有不同看法用数据说明 |
| 只报问题不给方案 | 领导会觉得你只会发现问题不会解决问题 | 每个问题后面跟上"我建议..."或"我已经..." |
| 报喜不报忧 | 小问题捂成大问题，最终你背锅 | 主动暴露风险，但要同时带上应对方案 |
| 越权承诺 | "我会要求XX供应商..."实际上需要王易人确认 | 清楚自己的权限边界，超出权限的说"我建议XX，需要和王易人确认" |
| 回答完就结束 | 领导以为你在跟，其实你没有下一步 | 明确说"我会在X号之前给你一个回复" |
| 在军哥面前绕过王易人 | 军哥直接找你，你没告诉王易人 | 回复军哥后，第一时间同步王易人 |

## 决策输出

**回复格式示例：**

> 军哥随口问：
> "军哥，赛维斯目前出勤率78%，低于85%的预警线。原因是3月扩编培训期，预计4月中旬恢复。我上周已经和他们沟通过，这周会做现场确认。"
>
> 王易人分配任务：
> "好的易人，岐力这边我跟进。目前岐力主要问题是财富产线转化率不稳定，我本周和岐力主管约一次沟通，周五前给你结果。"
>
> 需要升级的情况：
> "军哥，翰锐3月GMV连续4周下降，累计降幅超过20%，触发了业绩断崖预警。我已经做了首次预警沟通，供应商反馈是主管离职导致团队不稳。如果2周内没有改善，我建议启动正式约谈，需要和王易人确认。"
