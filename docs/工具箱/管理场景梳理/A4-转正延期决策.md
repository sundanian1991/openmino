# A4 转正/延期决策

> 试运行到期前，用6月里程碑框架逐项打分，决定留、延还是退。

## 触发信号

- 新供应商试运行满3个月（中期检查）
- 新供应商试运行满6个月（终期决策）
- 王易人问"某某供应商怎么样了"
- 月度排名出炉，新供应商出现在后30%

## 核心问题

> 这家供应商在试运行期间，每个阶段是否达到了对应的量化检查点？没达到的部分是能力问题还是时间问题？

## 思考步骤

### 步骤1：拉出时间线，确认当前所处阶段

**想什么**：这家供应商跑了多久？每个阶段的达标情况如何？

**检查什么数据**：
- 试运行起始日期
- 月度GMV、出勤率、流失率、合规记录
- 每月排名数据

**判断什么**：
- 第1-2月（引入期）→ 出勤率是否≥85%？业务是否跑通无重大合规问题？
- 第3-4月（成长期）→ GMV是否达到中位供应商的60%以上？
- 第5-6月（稳定期）→ 月度排名是否进入前70%？

### 步骤2：逐项核对主管胜任5项指标

**想什么**：供应商派来的主管能不能带队伍？这是决定留不留的关键。

**检查什么数据**：
- 流失率（团队稳定度 → 达标线：<8%/月）
- 出勤率（出勤稳定度 → 达标线：>90%）
- 人效走势（是否连续2个月下降）
- 预警后48h内回复率（响应质量 → 达标线：>80%）
- 整改承诺按期完成率（整改执行力 → 达标线：>70%）

**判断什么**：
- 5项全过 → 主管合格，问题在其他地方
- 3-4项过 → 主管有短板，看短板是哪项（稳定度和出勤是硬伤，其余可救）
- ≤2项过 → 主管不行，建议更换（需王易人确认）

### 步骤3：找到短板根因

**想什么**：没达标的部分，是供应商能力问题还是我们的支持问题？

**检查什么数据**：
- 培训记录：培训是否到位？供应商是否参加？
- 系统对接：是否有过系统故障影响产能？
- 分量情况：试运行期间是否给了足够的量？
- 产线匹配：该供应商擅长什么产线？当前分配是否对口？

**判断什么**：
- 根因在供应商（主管能力、团队招聘、执行力）→ 需要约谈整改或换主管
- 根因在我们（分量不够、培训缺失、系统问题）→ 先补课再观察
- 根因在外部（市场波动、政策变化）→ 调整预期，延长观察

### 步骤4：对比同类供应商同期表现

**想什么**：别只看绝对数，看相对位置。同样的6个月，其他供应商表现如何？

**检查什么数据**：
- 毅航、毛毛虫、伽玛等头部供应商当年引入时的同期数据
- 当前中位供应商的GMV基准线
- 同批次引入的其他新供应商对比

**判断什么**：
- 低于同类同期 → 倾向不转正
- 接近同类同期 → 倾向转正或延期
- 高于同类同期 → 强烈建议转正

### 步骤5：做出决策，准备材料

**想什么**：基于以上分析，结论是什么？需要谁审批？

**判断什么**：
- **转正**：6月末排名稳定在前70%，5项指标≥3项达标 → 正式纳入
- **延期**：有2-3项指标接近达标但未过，且根因可修复 → 延长1-2个月，明确延期考核标准
- **清退**：≤2项达标且无改善趋势 → 启动清退评估（转A6模板）

**决策授权**：
- 转正/延期：年老师建议 → 王易人确认
- 清退：年老师建议 → 王易人确认 → 军哥事前沟通

## 关键变量

| # | 变量 | 判断标准 | 数据来源 |
|---|------|---------|---------|
| 1 | 阶段达标率 | 引入期/成长期/稳定期，每个阶段核心指标是否过线 | 月度运营数据 |
| 2 | 主管胜任评分 | 5项指标中几项达标（≥3项=及格，5项=优秀） | 人事数据+预警记录 |
| 3 | 排名趋势 | 逐月排名是上升、持平还是下降 | 月度排名表 |
| 4 | 根因归属 | 问题出在供应商能力、我方支持还是外部环境 | 培训记录+系统日志+分量数据 |
| 5 | 同类对比 | 与头部供应商同期数据对比的相对位置 | 历史数据 |

## 常见误区

1. **只看最后一个月的数据** — 试运行是6个月的过程，要看趋势不是看切片。第1个月烂、第6个月达标，比第1个月好、第6个月烂要乐观
2. **被供应商态度带偏** — "态度很好、很配合"不等于业务达标。先看数据，再听解释
3. **忽略主管评估** — 供应商换了主管但没告诉你，数据波动可能就是这个原因。每次评估前确认主管是否有变动
4. **延期不设标准** — 延期必须明确"延期期间考核什么、什么时候再评估"，否则无限延期
5. **不敢启动清退** — 试运行6个月没达标的供应商，转正后大概率还是不行。数据不过硬就别留情面

## 决策输出

> **示例：赛维斯试运行6个月转正评估**
>
> **结论**：建议转正
>
> **依据**：
> - 引入期：出勤率88%（≥85%过线），无合规问题
> - 成长期：第4月GMV达到中位的67%（≥60%过线）
> - 稳定期：第5、6月排名分别为第18名、第15名（前70%过线，32家供应商）
> - 主管胜任：5项达标4项（流失率偏高7.2%，接近但未过8%线）
> - 同类对比：优于毛毛虫同期表现
>
> **风险提示**：流失率7.2%接近红线，转正后需持续关注团队稳定性
>
> **建议**：转正后首月重点跟进流失率，若突破8%则启动主管更换建议
