# D4 供应商群异常信号

> 30秒判断是真问题还是撒娇，决定是记录、关注、还是出手

## 触发信号

- 某供应商24h未回复消息（工作群、企微、电话）
- 供应商态度突变（配合→推诿、积极→消极）
- 群里突然沉默或突然频繁发言
- 供应商开始推诿责任、甩锅
- 原本活跃的老板突然不说话了
- 供应商在群里发牢骚/暗示不满
- 多个供应商同时反映类似问题（可能不是个体问题）

## 核心问题

> 这个信号是**偶发波动**还是**趋势性变化**？背后是管理问题、情绪问题、还是博弈策略？要不要现在就介入？

## 30秒快速判断

**谁在发信号？**

| 来源 | 可信度 | 典型行为 |
|------|--------|---------|
| 重点供应商（毅航、毛毛虫等头部6家） | 高，必须重视 | 排名波动、态度变化直接影响产线KPI |
| 中腰部供应商 | 中等，观察为主 | 多数是情绪表达或资源争夺 |
| 小供应商 | 低，记录即可 | 容易过度反应，不用被带节奏 |

**什么类型的信号？**

| 信号类型 | 识别方法 | 严重度 |
|---------|---------|--------|
| **失联** | 24h未回复 | 看日常沟通频率——日常高频的突然断联更严重 |
| **态度转向** | 配合度骤降、开始推诿 | 可能是排名下滑后的情绪反应，也可能是真在酝酿走人 |
| **群内发难** | 在公开场合质疑规则/政策 | 要区分：有道理的质疑 vs 情绪宣泄 vs 故意施压 |
| **集体信号** | 多家同时反映同类问题 | 高度重视——可能是政策/流程本身出了问题 |
| **沉默退让** | 原本争取资源的突然不争了 | 最危险——可能已经在找下家或放弃这个渠道 |

**紧急吗？**

| 信号 | 判断 | 紧急度 |
|------|------|--------|
| 头部供应商24h失联+排名在下滑 | 可能在酝酿重大决策 | 紧急 |
| 供应商老板突然沉默（原本日均1h沟通） | 关系出问题或战略转向 | 紧急 |
| 多家同时投诉同一政策 | 政策可能不合理 | 重要 |
| 一家在群里发牢骚 | 多数是情绪 | 常规 |
| 某供应商突然不争取资源了 | 可能在退出 | 需关注 |

**一句话回复口径**：

> 失联型："确认一下XX的情况，最近的[具体事项]进度怎么样了？"
> 发难型："你说的这个我记下了，回头核实一下。"
> 沉默退让型（私信）："[名字]，最近怎么样？上次说的[具体事]推进了吗？"

## 5分钟深入版

**Step 1：确认信号真实性**
- 是真的异常还是节奏变化？（周末/节假日不算失联）
- 查最近3天沟通记录——是突然变化还是渐变的？
- 有没有外部因素？（行业变动、政策调整、竞对挖人）

**Step 2：关联背景数据**

| 查什么 | 为什么 | 去哪查 |
|--------|--------|--------|
| 最近排名 | 排名下滑是态度变化的最常见原因 | 月度评估数据 |
| 最近分量变化 | 减量容易引发情绪反应 | 分量分配表 |
| 最近质检/合规记录 | 被处罚过的供应商容易消极 | 质检台账 |
| 结算状态 | 结算延迟/金额争议会直接影响态度 | 结算台账 |
| 人员变动 | 主管/核心人员离职 = 管理真空 | 供应商报备 |

**Step 3：判断信号性质**

| 性质 | 判断依据 | 处理方式 |
|------|---------|---------|
| 情绪宣泄 | 无具体诉求，只是表达不满 | 记录，不急着介入，下次沟通时自然带过 |
| 博弈策略 | 有具体诉求（要量/要支持），用消极表达施压 | 看诉求是否合理，合理就谈，不合理就不接 |
| 真实困难 | 有具体问题（人手不足、系统故障），态度是附带品 | 先解决问题，态度会自然恢复 |
| 退出前兆 | 沉默+不争取+不配合，且持续1-2周 | 紧急约谈，启动风险预案 |
| 关系裂痕 | 对你个人态度变化（从热络到冷淡） | 找机会私下聊，别在群里对峙 |

**Step 4：决定介入程度**

| 介入级别 | 适用场景 | 动作 |
|---------|---------|------|
| 仅记录 | 偶发、情绪性、无实质影响 | 备忘录记一笔，下次沟通留意 |
| 主动关注 | 持续2-3天、有趋势性 | 加大沟通频率，多问几个问题 |
| 私下沟通 | 持续1周、可能影响业务 | 找对应人私聊，问真实情况 |
| 正式约谈 | 持续2周+影响指标 | 约谈邀约，要求说明情况 |
| 升级上报 | 可能导致供应商退出/重大风险 | 报王易人，准备风险处置方案 |

## 思考步骤

### Step 1：信号确认（5秒）

> 想：这个信号是真的异常吗？
> 检查：排除正常因素（周末、假期、已知请假）
> 判断：是异常 / 是正常波动 / 不确定需观察

### Step 2：关联分析（30秒）

> 想：这个供应商最近有没有"该高兴"或"该不满"的事？
> 检查：排名变化、分量调整、结算状态、质检结果
> 判断：有明确诱因 / 无明显诱因（更需警惕）

### Step 3：性质判断（30秒）

> 想：是真问题还是撒娇？个体问题还是系统性问题？
> 检查：其他供应商有没有类似信号？同类供应商排名如何？
> 判断：个体情绪 / 个体困难 / 系统性问题 / 退出前兆

### Step 4：介入决策（30秒）

> 想：要不要现在就出手？用什么方式？
> 检查：介入级别（记录/关注/私聊/约谈/升级）
> 判断：选定介入方式 + 时间节点

### Step 5：执行+记录

> 想：动作是什么？记什么？
> 检查：沟通记录要留痕，异常信号要建立追踪档案
> 判断：随手记 / 建立异常追踪 / 触发正式流程

## 关键变量

| 变量 | 判断标准 | 影响 |
|------|---------|------|
| **供应商等级** | 头部6家 vs 中腰部 vs 小供应商 | 头部异常 = 优先处理，小供应商 = 标记观察 |
| **信号持续时间** | 1天 / 3天 / 1周+ | 越久越不是情绪问题 |
| **排名趋势** | 上升 / 稳定 / 下滑 | 下滑中的异常更可能是退出前兆 |
| **替代性** | 高（可替代）/ 低（独家能力） | 低替代性 = 话语权在供应商，需更谨慎 |
| **信号密度** | 单点 vs 多点同时出现 | 多点 = 系统性问题 |

## 常见误区

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 所有异常都高度重视 | 精力分散，真正重要的反而不敏感 | 分级处理，头部优先 |
| 觉得供应商在撒娇就忽视 | 可能错过真实问题 | 先记录观察，持续3天以上要关注 |
| 在群里公开回应发难 | 把局面搞僵，给其他供应商围观 | 私下沟通，群里只做必要回应 |
| 等"铁证"再出手 | 等到出事就晚了 | 趋势性信号就够，不需要铁证 |
| 自己扛着不报上级 | 小问题拖成大事故 | 涉及头部供应商退出风险 → 立即报王易人 |

## 决策输出

**场景：毅航老板连续3天没联系你（原本天天聊）**

> 30秒：毅航（头部6家，金条主力）失联型，紧急。主动出击："最近几天没聊，[具体事项]推进得怎么样了？"
> 分析：查排名/分量/结算/人员变动。排名稳定+无负面 → 观察1天；有负面因素 → 今天打通电话；电话不通+有负面 → 报王易人。

**场景：伽玛在群里频繁质疑考核标准**

> 30秒：伽玛（头部6家）群内发难型，不紧急但需处理。群里："我记下了，回头核实。" 转移到私聊。
> 分析：查伽玛排名/标准是否变更/其他供应商反馈。只有伽玛质疑+排名下滑 → 私聊拿数据说话；多家反馈 → 标准可能有问题的，走 `internal-coordination.md`。

**场景：翰锐突然不争资源了（原本很积极）**

> 30秒：翰锐（头部6家）沉默退让型，最危险。私信："最近感觉你那边节奏有变化，遇到困难了吗？"
> 分析：查3个月排名趋势/人员流失率/竞对挖人。排名持续下滑+不争取+人员流失 → 退出前兆，正式约谈。
