# A2 约谈准备

> 约谈前备数据、约谈时用结构、约谈后盯结果。

## 触发信号

以下任一信号出现即启动本模板：

| 信号类型 | 具体表现 |
|---------|---------|
| **连续预警** | 连续2个月在后30% → 正式约谈 |
| **A级骤降** | 上月前30%本月跌到后30% → 立即约谈 |
| **周度三重信号** | 出勤异常+业绩断崖+流失加速同时出现 |
| **质检红线** | 质检得分连续2月低于合格线，或发生重大客诉 |
| **整改不力** | 上次约谈后提交的整改计划未按期执行 |

## 核心问题

> 约谈要达成什么结果？是拿整改承诺、调分量、还是启动清退预沟通？

## 思考步骤

### 第1步：明确约谈目标

**想什么**：这次约谈的底线结果和理想结果分别是什么？

**检查什么数据**：
- 供应商在A1模板中的异常级别和原因定位
- 历史约谈记录（有没有约谈过、上次约谈结果）

**判断什么**：

| 异常级别 | 底线目标 | 理想目标 |
|---------|---------|---------|
| 连续2个月后30% | 拿到书面整改计划+时间表 | 供应商承认问题并提出可量化改进方案 |
| A级骤降 | 搞清楚骤降原因+短期补救措施 | 供应商承诺1个月内恢复到中间区间 |
| 周度三重信号 | 应急止血方案（48小时内） | 供应商给出人力补充和业务恢复时间线 |
| 整改不力 | 明确后果（减量/清退预警） | 重新提交整改计划并加码约束条件 |

### 第2步：准备数据弹药

**想什么**：约谈时展示什么数据最有说服力？对方会怎么反驳？

**检查什么数据**：

| 数据维度 | 具体内容 | 用途 |
|---------|---------|------|
| **排名轨迹** | 近3-6个月排名变化折线 | 证明趋势而非单次波动 |
| **人效对比** | 该供应商 vs 头部供应商（毅航/伽玛）人均GMV | 用排名思维：看相对位置而非绝对值 |
| **产能利用率** | 自身峰值比 + 头部差距 | 定性类型（标杆/尽力/问题/波动） |
| **质量数据** | 质检得分、客诉率 | 排除"以量换质" |
| **人力数据** | 出勤率、离职率、新员工占比、招聘到位速度 | 判断是管理问题还是招人问题 |
| **分量变化** | 近3个月分量是否有调整 | 排除分量减少导致的合理下降 |

**判断什么**：
- 数据是否足以支撑约谈论点
- 对方可能的反驳点（"最近招不到人"、"产品政策变了"），准备应对数据

### 第3步：设计约谈结构

**想什么**：约谈不是审讯，要让供应商自己说出问题和计划。

**约谈四段结构**：

| 阶段 | 时长 | 内容 | 要点 |
|------|------|------|------|
| **开场定调** | 5分钟 | 说明约谈背景和目的 | 不绕弯子，直接说数据 |
| **数据呈现** | 10分钟 | 展示排名、人效、质量等核心数据 | 用相对排名说话，不给绝对分数 |
| **原因探讨** | 15分钟 | 问供应商怎么看、什么原因、怎么解决 | 引导式设问：先给再问，不直接下结论 |
| **承诺确认** | 10分钟 | 确认整改计划、时间节点、验收标准 | 必须拿到书面承诺 |

**判断什么**：
- 先给数据再问原因（引导式设问），不是直接说"你业绩差"
- 约谈目标是让对方自己承认问题，不是逼对方认错

### 第4步：准备引导式问题清单

**想什么**：不同原因维度准备不同的引导问题。

**检查什么数据**：第2步准备的各维度数据

**设计什么问题**：

| 原因维度 | 引导问题示例 |
|---------|-------------|
| 人效下降 | "毛毛虫3月人均GMV是X，头部平均是Y。你觉得差距主要在哪？" |
| 产能不足 | "赛维斯目前座席使用率82%，但出勤率只有78%。团队稳定性有什么困难？" |
| 质量下滑 | "岐力质检得分连续2个月在合格线以下。培训侧有什么调整计划？" |
| 流失严重 | "翰锐上个月离职了8个人。离职面谈有什么共性反馈？" |
| 外部因素 | "产线政策3月做了调整，你们切换进度怎么样？有没有遇到卡点？" |

**判断什么**：
- 每个问题都有数据支撑，不是空问
- 问题设计让对方无法回避但也不会当场翻脸

### 第5步：确定整改验收标准

**想什么**：整改计划不能是"加强管理"这种空话，必须有可量化的验收节点。

**检查什么数据**：异常原因对应的核心指标

**设定什么标准**：

| 整改类型 | 量化标准 | 验收周期 |
|---------|---------|---------|
| 人效提升 | 人均GMV提升X%，排名脱离后30% | 1个月 |
| 出勤恢复 | 有效出勤率恢复到≥90% | 2周 |
| 质量达标 | 质检得分≥合格线，客诉率≤X% | 1个月 |
| 人力补充 | 新员工招聘到位X人，通过培训X人 | 2周-1个月 |
| 流失控制 | 月离职率降至≤X% | 1个月 |

### 第6步：约谈后跟踪设计

**想什么**：约谈不是结束，跟踪才是关键。怎么确保整改不变成一纸空文？

**检查什么数据**：整改计划中的时间节点

**设定什么跟踪机制**：

| 时间点 | 动作 | 形式 |
|--------|------|------|
| 约谈后3天 | 收到书面整改计划 | 邮件确认 |
| 约谈后1周 | 第一次进度检查 | 微信/电话 |
| 约谈后2周 | 第二次进度检查+数据验证 | 对比周度报表 |
| 约谈后1个月 | 正式验收 | 数据对比+书面评估 |

**判断什么**：
- 跟踪频率与异常级别匹配（红色级别跟踪更密）
- 每次跟踪都有数据验证，不是口头汇报

## 关键变量

| # | 变量 | 判断标准 | 用途 |
|---|------|---------|------|
| 1 | **异常级别** | 黄色/橙色/红色 | 决定约谈正式程度和授权层级 |
| 2 | **历史约谈次数** | 首次/二次/多次 | 决定话术严厉程度和后果说明 |
| 3 | **供应商配合度** | 积极承认/找借口/推诿 | 决定后续跟踪密度 |
| 4 | **整改计划质量** | 可量化/有时间表/有负责人 | 决定是否需要退回重写 |
| 5 | **上级知情状态** | 已报备/未报备 | 约谈前必须确认授权 |

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 没准备数据直接约谈 | 数据是约谈的弹药，没有数据就没有发言权 |
| 约谈变成训话 | 引导式设问，让对方自己说问题，不是你告诉对方有什么问题 |
| 接受"加强管理"式整改 | 必须要可量化的指标+时间节点+责任人 |
| 约谈完不跟踪 | 约谈只是开始，跟踪机制才是保障 |
| 忘记报备 | 正式约谈需事前报备王易人，橙色级别及以上不能跳过 |
| 把A级骤降当普通预警处理 | A级骤降是红色级别，48小时内必须行动 |

> **供应商**：华啸
> **约谈原因**：连续2个月排名后30%（橙色级别）
> **约谈目标**：拿到书面整改计划，确认1个月内脱离后30%

> **跟踪计划**：4-25收整改计划 → 4-29第一次检查 → 5-06数据验证 → 5-22正式验收
> **如果整改失败**：升级为清退评估（需王易人确认）
