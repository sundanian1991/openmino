# A1 业绩异常判断

> 看到信号后，判断严重程度、定位原因、决定下一步。

## 触发信号

以下任一信号出现即启动本模板：

| 信号类型 | 具体表现 |
|---------|---------|
| **排名下降** | 月度排名落入后30%，或从上月位置下降≥5名 |
| **周度断崖** | 本周GMV比近4周均值下降>20% |
| **峰值比低** | 自身峰值利用率<70% |
| **头部差距大** | 与头部供应商峰值利用率差距>15个百分点 |
| **A级骤降** | 上月前30%本月直接跌到后30% |
| **出勤异常** | 有效出勤率<85% |
| **流失加速** | 单周离职超过5人 |

## 核心问题

> 这个异常是短期波动还是结构性问题？需要什么级别的干预？

## 思考步骤

### 第1步：确认信号真实性

**想什么**：数据是否准确？是否因统计口径、结算周期、数据延迟导致"假异常"？

**检查什么数据**：
- 本月排名 vs 近3个月排名趋势
- 本周GMV vs 近4周GMV逐周数据
- 数据来源是否完整（有无缺产线、缺站点）

**判断什么**：
- 排除数据质量问题后，异常是否依然成立
- 如果数据有误 → 修正后重新评估，不走后续步骤

### 第2步：判断异常级别

**想什么**：这个供应商当前处于排名触发规则的哪个档位？

**检查什么数据**：
- 上月排名位置 + 本月排名位置
- 是否连续落入后30%
- 是否出现A级骤降（前30% → 后30%）

**判断什么**（对照排名触发规则）：

| 供应商状态 | 判断标准 | 异常级别 |
|-----------|---------|---------|
| 首次预警 | 本月落入后30%，上月不在 | 黄色 — 关注 |
| 连续预警 | 连续2个月在后30% | 橙色 — 需干预 |
| 清退评估 | 连续3个月在后30% | 红色 — 需决策 |
| A级骤降 | 上月前30%本月跌到后30% | 红色 — 立即行动 |

### 第3步：定位原因维度

**想什么**：异常是产能问题、人效问题、还是外部因素？

**检查什么数据**：

| 维度 | 具体数据 |
|------|---------|
| 产能 | 有效出勤率、座席数变化、新员工占比 |
| 人效 | 人均GMV、转化率、与头部对比 |
| 质量 | 质检得分、客诉率 |
| 业务 | 分量是否有调整、产线是否有切换、产品政策变化 |
| 人事 | 离职率、招聘进度、培训投入 |

**判断什么**：
- 是单一维度问题还是多维度叠加
- 原因是供应商自身可控的，还是外部不可抗的

### 第4步：产能利用率定性

**想什么**：结合自身峰值比和头部差距，判断供应商属于哪种类型？

**检查什么数据**：
- 自身峰值利用率（当前产出 / 自身历史峰值）
- 头部差距（自身峰值利用率 - 头部平均峰值利用率）

**判断什么**（双维度矩阵）：

| 自身峰值比 | 头部差距 | 类型 | 解读 |
|-----------|---------|------|------|
| 高 | 高 | 标杆型 | 自己发挥好，也跟得上头部 → 不需要干预 |
| 高 | 低 | 尽力型 | 已经尽力，但产线/产品本身人效偏低 → 调预期或调分量 |
| 低 | 低 | 问题型 | 自己没发挥好，跟头部差距也在拉大 → 重点干预 |
| 低 | 高 | 暂时波动 | 短期因素导致发挥不好，底子不差 → 密切关注 |

### 第5步：检查周度突变信号

**想什么**：是否有短期恶化信号叠加？这决定干预的紧迫度。

**检查什么数据**：
- 有效出勤率是否<85%
- 本周GMV是否比近4周均值下降>20%
- 单周离职是否超过5人

**判断什么**：
- 3个信号全无 → 按常规节奏处理
- 有1个 → 本周内完成沟通
- 有2个及以上 → 48小时内沟通

### 第6步：决定下一步动作

**想什么**：基于异常级别 + 原因定位 + 供应商类型，该做什么？

**判断什么**（对照决策授权矩阵）：

| 异常级别 | 动作 | 权限 |
|---------|------|------|
| 黄色（首次预警） | 私下沟通问原因 | 自主执行 |
| 橙色（连续预警） | 正式约谈，要求整改计划 | 自主执行，事前报备王易人 |
| 红色（清退评估/A级骤降） | 评估延长或启动清退 / 立即约谈 | 建议权，王易人确认，视情况事前沟通军哥 |

## 关键变量

| # | 变量 | 判断标准 | 数据来源 |
|---|------|---------|---------|
| 1 | **月度排名位置** | 前30% / 中间 / 后30% | 月度排名表 |
| 2 | **连续落后月数** | 1个月 / 2个月 / 3个月 | 近3个月排名 |
| 3 | **自身峰值利用率** | ≥70%正常 / <70%偏低 | 产能报表 |
| 4 | **周度GMV降幅** | <20%波动 / ≥20%断崖 | 周度报表 |
| 5 | **离职率** | 单周<5人正常 / ≥5人加速 | HR报表 |

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 看到排名下降就约谈 | 先确认是首次还是连续，首次私下问原因即可 |
| 只看排名不看原因 | 毅航排名下降可能是分量减少导致，不一定是管理问题 |
| 忽略暂时波动型 | 赛维斯峰值比低但头部差距高，底子不差，别过度反应 |
| 红色级别自己拍板 | 清退评估必须王易人确认，A级骤降约谈要事前报备 |
| 只看单周数据 | 至少看近4周趋势，排除结算周期等短期因素 |

## 决策输出

> **供应商**：毛毛虫
> **异常级别**：橙色（连续2个月后30%）
> **原因定位**：人效下降 + 新员工占比高（42%）
> **供应商类型**：问题型（自身峰值比低 + 头部差距大）
> **突变信号**：出勤率82%（<85%阈值）
> **下一步动作**：正式约谈，要求提交整改计划
> **授权情况**：事前报备王易人
> **约谈时间**：本周内
