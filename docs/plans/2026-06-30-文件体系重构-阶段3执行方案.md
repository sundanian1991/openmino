## 0. 第一性原理

阶段 3 的本质：**"在不破坏 agent 运行时的前提下，把死的东西归位。"**

每个文件的安全等级由**唯一一个问题**决定：**"它现在还在被活脚本/hook/skill 读写吗？"**
- 在 → 🔴 禁动
- 不在 → 🟢 可归档
- 在但已停摆 → 🟡 归档需配套动作（停 cron / 改 skill 路径 / 更新配置）

**绝对原则**：宁可不动，不可切断依赖。所有 🟡 项的配套动作必须与归档**同批完成**。
