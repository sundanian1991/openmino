# B6 降价谈判准备

> 谈价格之前，先把底牌想清楚——自己的底线、对方的痛点、可交换的筹码。

## 触发信号

- 年度/季度合同续签，需要重新议价
- 供应商成本结构明显变化（规模效应应降价、原材料降了但报价没动）
- 策略组（刘伟佳）反馈某产线单客成本偏高
- 王易人要求优化供应商成本
- 新供应商引入导致竞争格局变化
- 供应商主动提涨价（需要反制或谈判）

## 核心问题

> 降价的目标幅度是多少？对方能接受的底线在哪？我们手里有什么筹码可以交换？

## 思考步骤

### 第1步：确认谈判必要性

**想什么**：降下来的收益 vs 谈判成本+关系损伤，划不划算？

**检查什么数据**：
- 当前单客成本 vs 同产线其他供应商均价
- 该供应商占该产线的分量比例
- 替换成本和风险（切换周期、培训成本、质量波动）
- 该供应商排名和表现

**判断什么**：

| 供应商类型 | 是否值得谈 | 原因 |
|-----------|----------|------|
| 标杆型（排名前30%） | 谨慎 | 砍价容易伤关系，可能被竞品挖走 |
| 中等（排名中间） | 可以谈 | 有替换空间，供应商也想保量 |
| 问题型（排名后30%） | 换思路 | 不是降价问题，是是否继续合作的问题 |

### 第2步：摸清成本结构

**想什么**：供应商报价由什么构成？哪些部分有弹性？

**判断什么**：

| 计价方式 | 降价切入点 | 谈判策略 |
|---------|-----------|---------|
| 按件计费 | 提量换降价——量上去，单件成本应降 | "加20%的量，单价降5%" |
| 按人计费 | 提效换降价——人效上去，隐性成本降 | "人均GMV达XX后，管理费降Y%" |
| 阶梯价 | 看是否跨了阶梯——量变但价没跟上 | "已进第二阶梯，应用第二阶梯价" |

**举例**：赛维斯金条线按件计费55元/单，月均8000单。加量到10000单，边际管理成本几乎不增，固定成本被摊薄——"量加25%，单价降到52元"合理。

### 第3步：盘点谈判筹码

**想什么**：手里有什么可以交换？不是白砍价，是做交易。

| 筹码类型 | 具体内容 | 供应商在乎程度 |
|---------|---------|--------------|
| **分量承诺** | 保证未来N月量不减少 | 高 |
| **增量机会** | 降价可以给更多分量 | 高 |
| **新产线机会** | 优先给新产线份额 | 中 |
| **结算周期** | 缩短结算周期（30天→15天） | 高 |
| **排名数据** | 暗示不降价可能调分量 | 中 |
| **竞品报价** | 其他供应商同产线报价（匿名） | 中 |

**原则**：分量承诺 > 增量机会 > 新产线机会（供应商最在乎确定性）；不要用虚筹码（"以后再说"）。

### 第4步：确定谈判目标区间

| 目标 | 设定方法 | 举例 |
|------|---------|------|
| **理想目标** | 同产线均价再低5% | 均价48元 → 45.6元 |
| **可接受目标** | 同产线均价水平 | 48元 |
| **底线** | 现状价格降5%且不附带额外条件 | 55×0.95 = 52.25元 |

**关键原则**：
- 底线必须内部确认（王易人知道且同意）
- 谈不到底线宁可不谈，不接受不痛不痒的结果
- 算总账：赛维斯月均8000单，55→52元，月省2.4万，年省28.8万

### 第5步：准备谈判话术框架

**开场（5分钟）**：先肯定合作 → 再点明目的 → 不一上来就报数字，先让对方表态

**推进（核心阶段）**：
- 先问对方想法，对方报数字后再还价，不先亮底牌
- 用筹码交换让步，不是单方面要求（"价格到XX，我们可以做YY"）
- 每次让步都换对方让步

**收尾（5分钟）**：能确认就确认；不能确认则约定时间节点；结束前说清下一步。

**举例**：赛维斯说"成本涨了没法降" → 不直接反驳，问"成本涨了哪些部分？一起看看优化空间" → 把"你降不降"变成"怎么一起降成本"。

### 第6步：准备B计划

| 情况 | B计划 |
|------|------|
| 对方拒绝降价但愿其他让步 | 接受非价格让步（结算周期缩短、服务条款优化） |
| 对方拒绝所有让步 | 不续签或减量，给到愿意降价的供应商 |
| 对方强硬且是唯一供应商 | 先稳住，同时加速培养替代，不急翻脸 |
| 对方要求涨价 | 拿数据和排名说话——表现没提升就没涨价理由 |

### 第7步：内部确认

| 事项 | 需确认的人 | 确认内容 |
|------|----------|---------|
| 降价目标区间 | 王易人 | 目标价、底线价、可交换筹码 |
| 分量承诺 | 王易人 + 策略组（刘伟佳） | 是否真的能给更多量 |
| 结算周期调整 | 财务 | 是否可以缩短结算周期 |
| 替代方案 | 王易人 | 谈不拢则分量转给谁 |

**原则**：承诺出去的筹码必须内部已确认能做到。

## 关键变量

| # | 变量 | 判断标准 | 数据来源 |
|---|------|---------|---------|
| 1 | **价格偏离度** | 当前价 vs 产线均价偏离>10% → 值得谈 | 成本分析表 |
| 2 | **供应商类型** | 标杆/中等/问题 → 决定谈判策略 | 月度排名表 |
| 3 | **替代可行性** | 同产线有≥1家供应商可接手 → 底气足 | 产能报表 |
| 4 | **分量弹性** | 该供应商分量占比<30% → 调整空间大 | 分量分配表 |
| 5 | **总成本影响** | 年化节省>10万 → 值得花时间谈 | 算总账 |

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 一上来就报目标价 | 先让对方表态，不先亮底牌 |
| 只谈价格不谈交换 | 每个让步都要换筹码 |
| 对标杆供应商硬砍 | 毅航排名前3、只贵2元——不该砍 |
| 忽略供应商成本压力 | 硬压只会导致偷工减料 |
| 承诺了做不到的筹码 | 下次谈判信用归零 |
| 谈完不记录 | 必须书面确认，口头不算数 |
| 只看单价不看总成本 | 单价降但量减，总成本反升 |

## 决策输出

> **B6 谈判准备卡**
>
> **供应商**：赛维斯 | **产线**：金条线 | **当前单价**：55元/单 | **产线均价**：48元/单（偏离14.6%） | **月均单量**：8000单
>
> **谈判目标**：理想45.6元（均价-5%） / 可接受48元（持平） / 底线52元（降5.5%）
>
> **筹码组合**：承诺金条线分量不减 + 降价到位则信用卡线增量10% + 结算周期30→20天（需财务确认）
>
> **B计划**：谈不拢则10%量转给伽玛
>
> **内部确认**：王易人已确认目标区间，待确认分量承诺和结算周期
> **年化节省**：52元成交 → 月省2.4万 → 年省28.8万
