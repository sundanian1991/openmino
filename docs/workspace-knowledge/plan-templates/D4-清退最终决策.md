# D4 清退最终决策

> 连续3月后30%，但可能还有改善趋势。留还是退？这不是数据问题，是判断问题。清退涉及人员安置、产能缺口、合同善后。要有方案才去找王易人，不能只带问题去。

---

## 问题拆解

清退是供应商管理中最重的决策。把"该不该清退"拆成3个判断：

1. **还有没有救？** — 看趋势，不看绝对值
2. **清退的代价能不能承受？** — 产能缺口、人员安置、合同善后
3. **清退后的替代方案是什么？** — 没有替代就暂缓

**核心矛盾**：数据说"该清"，但现实说"清不起"。

---

## 归因路径（决策树）

### 第一步：趋势判断（有没有救）

> 核心工具：排名思维 — 不看绝对分数，看相对排名的趋势。

| 趋势 | 位置 | 判断 |
|------|------|------|
| 排名在升 | 后30%但从第28升到第22 | **有救**，给最后一次机会 |
| 排名持平 | 后30%且在第25-28波动 | **看情况**，需判断是平台期还是停滞 |
| 排名在降 | 后30%且从第25降到第30 | **没救趋势**，重点评估替代方案 |
| 垫底稳定 | 稳定在第30-32名 | **大概率要清**，启动替代准备 |

**辅助判断**（趋势不明确时）：

| 指标 | "有救"信号 | "没救"信号 |
|------|-----------|-----------|
| 主管变动 | 新主管到位且有效 | 频繁换主管或无人可用 |
| 团队规模 | 在扩编或稳定 | 持续缩编 |
| 配合态度 | 主动改善、积极配合 | 敷衍、拖延、反驳 |
| 质检趋势 | 合格率在上升 | 合格率持续下降或波动 |
| 甲方支持需求 | 主动要求支持 | 不提需求也不改善 |

### 第二步：替代方案评估

> 没有替代方案不能清退。这是硬约束。

| 等级 | 情况 | 处理 |
|------|------|------|
| **A** | 有1家以上可立即承接 | 可以启动清退 |
| **B** | 有潜在承接方但需时间 | 先培养替代，同步压减目标供应商 |
| **C** | 无替代方案 | 暂缓清退，加快寻源，进入D1监控 |

**替代检查清单**：同产线富余产能、新供应商试运行状态、产能缺口测算、过渡期安排、合同终止条款。

### 第三步：决策分支

**分支一：有改善趋势** — 给6周观察期，明确恢复目标（排名回到前70%），每周跟踪，未达标进分支三/四。

**分支二：无改善 + 无替代** — 暂缓清退。加快寻源、压减分量、重点监控。培养替代期间不接受"给机会"请求。

**分支三：无改善 + 有替代** — 启动清退，过渡期4-8周。第1-2周通知+交接准备，第3-4周业务转移+人员安置，第5-6周验收+合同终止，第7-8周收尾归档。同步通知：王易人（审批）→ 策略组 → 质检组 → 财务。

**分支四：诚信红线** — 直接清退，不需要观察期。保留证据，同步法务评估。

| 红线行为 | 处理 |
|---------|------|
| 伪造业绩数据 | 直接清退 |
| 泄露客户信息 | 直接清退 + 法务跟进 |
| 串通舞弊/利益输送 | 直接清退 + 上报合规 |
| 虚假招聘/吃空饷 | 直接清退 |
| 隐瞒重大合规事件 | 直接清退 |

### 第四步：人员安置评估

> 清退涉及真实的人。3200+人的团队，人员安置必须提前想清楚。

关键问题：多少人？分布哪些产线？能否转移？转移意愿？培训周期？产能损耗？

**原则**：不承诺安置，但给过渡期。人员是供应商员工，甲方没有安置义务，但出于业务连续性给合理过渡期。

## 权衡因素

| 正方（清退） | 反方（不清退） |
|-------------|--------------|
| 信号效应（**高**）— 清退一家，其他31家都会认真对待 | 产能缺口（**高**）— 量谁来接？ |
| 业绩压力（**高**）— 后30%拖累整体KPI | 人员风险（**高**）— 大规模变动引发舆情/纠纷 |
| 资源优化（**中**）— 量给高效供应商 | 寻源成本（**中**）— 新供应商稳定需3-6个月 |
| 合规要求（**中**）— 不清退反有风险 | 替代不确定/关系成本/管理精力（**中**） |

**核心权衡**：短期阵痛 vs 长期健康。关键是**时机** — 有替代的时候清，没替代的时候准备。

## 叙事草稿

### 向上汇报（给王易人）

> "王总，我建议启动对翰锐的清退流程。原因有三：
> 第一，连续4个月排名后30%且排名持续下滑（从第25降到第30）。
> 第二，已进行2轮整改，效果不达标，对方改善承诺未兑现。
> 第三，翰锐目前负责金条约12%的量，赛维斯、岐力确认可分别承接5%和7%。
> 过渡方案：第1-2周通知+交接报备军哥；第3-4周业务转移，翰锐人员4周过渡期；第5-6周验收+财务结算。
> 风险点：翰锐可能申诉（已准备完整数据），短期产能波动预计<5%。
> 您看方案有没有需要调整的？"

**结构**：三段式 — 原因（数据硬）+ 方案（时间节点具体）+ 风险（不隐瞒）。

### 对供应商（清退通知）

> "XX，今天找你沟通一件重要的事。经过综合评估和多轮沟通整改，目前合作效果仍未达到预期，公司计划在6周内完成业务交接。过渡安排：（具体计划）。人员方面由你们自行安排，我们给4周过渡期。有什么问题现在沟通。"

**关键**：不拖泥带水，态度尊重，给对方留体面但方向不变。

### 内部同步

> "翰锐清退流程已启动。策略组：下周一前完成名单调整。质检组：重点监控承接质量。财务：启动结算对账。有问题随时找我。"
