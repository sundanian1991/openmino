# Q3 供应商来电/来消息

> 一句话：供应商主动找你 — 先听他说完，再判断他真正要什么

## 30秒判断三件事

1. **他说的是什么，真正要的是什么**：说"人不好招"可能在铺垫减量，说"能不能多给点量"可能察觉到要被减了，说"流程能不能快一点"可能是别家给了更好的条件
2. **他现在的状态**：正常沟通还是带情绪？带情绪的要先稳住，不当场答应任何事
3. **这事我能决定吗**：减量增量=不能自己定，流程加速=看情况，信息反馈=马上能做

## 一句话口径

- 供应商要量："我理解你的诉求，我拉一下最近的数据，看完了跟你聊。"（不拒绝不承诺，用数据争取时间）
- 供应商反映问题/投诉："你说的情况我记下了，我确认一下，X号之前给你回复。"（先记下来，别现场定性）
- 供应商要支持/资源："这个我需要内部对一下，回头给你准信。"（涉及其他部门或预算的，绝不自己拍）

## 需要调的数据

- 该供应商当前量和产能利用率：经营分析表
- 最近3个月业绩趋势：排名数据
- 同产线其他供应商对比：横向排名表

## 转给谁

- 减量/增量/清退决策 → 王易人
- 涉及合同条款变更 → 法务 + 王易人
- 投诉性质的问题 → 王易人，必要时军哥
