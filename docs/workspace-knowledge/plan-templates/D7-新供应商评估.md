# D7 新供应商评估

> 准入9维度评估怎么打分、怎么对标、怎么决策？不是机械算分，是判断能不能适配你的体系。

## 触发信号

- 业务线提出新供应商需求 / 供应商主动寻求合作 / 产线扩容 / 现有供应商退出替补 / 年度寻源计划

## 问题拆解

1. **硬伤红线**：有没有一票否决的致命问题？
2. **适配判断**：软实力能不能适配现有管理方式和节奏？
3. **战略价值**：引入这家能填补什么空白？带来什么新能力？

## 归因路径（决策树）

### 第一层：硬伤红线（一票否决）

| 硬伤维度 | 标准 | 检查方法 |
|---------|------|---------|
| 资质合规 | 营业执照异常、无相关经营许可、涉及金融业务违规 | 天眼查/企查查 |
| 法律风险 | 涉诉金额超注册资本30%、被执行人、重大行政处罚 | 中国裁判文书网、天眼查 |
| 经营年限 | 成立不满2年 | 营业执照 |
| 财务健康 | 资不抵债、连续亏损3年、现金流断裂迹象 | 供应商提供近2年财报 |
| 诚信记录 | 提供虚假数据/地址、隐瞒关键信息 | 现场考察+同行验证 |
| 利益冲突 | 同一实控人、与内部人员利益关联 | 天眼查+和现有供应商比对 |

### 第二层：软实力评估（9维度对标）

> 过了红线后，按9维度逐项评估。不是机械打分，是对标现有同产线供应商。

| 维度 | 评估要点 | 权重 |
|------|---------|------|
| 过往业绩 | 做过什么产线/规模/效果，和现有供应商起步期对比 | 高 |
| 同类型对标 | 同类金融电销经验，客单价/转化率水平 | 高 |
| 管理方式 | 内部管人/KPI/培训方式，能否适配你的管理节奏 | 高 |
| 人员数量 | 现有多少人，能扩到多少，1个月内到位？ | 中 |
| 公司背景 | 股东结构、融资情况、业务布局 | 中 |
| 历史交付风格 | 合作甲方评价，有无中途退出记录 | 中 |
| 配合商务 | 合同条款/结算周期接受度、配合度 | 中 |
| 团队架构 | 有无专门培训/质检/数据分析团队 | 中 |
| 资质合规（加分） | ISO认证、信息安全等级保护等额外资质 | 低 |

**评估结论**：

| 结论 | 标准 | 动作 |
|------|------|------|
| 全面优于同产线均值 | 多数维度达到或超过现有水平 | 加速引入，给更多初始分量 |
| 部分优势部分短板 | 核心维度达标但有明确短板 | 引入但限制初始分量，短板设改善计划 |
| 勉强达标 | 核心维度刚过及格线 | 谨慎引入，最小试单，严格观察期 |
| 达不到现有水平 | 多数维度低于现有均值 | 不引入，放入观察池 |

**对标参照系**：金条→头部6家；企金/信用卡→前5家均值；财富→前3家均值。

### 第三层：战略价值评估（加分项）

| 战略价值 | 加分程度 |
|---------|---------|
| 填补空白产线 | 高 — 直接引入 |
| 带来新能力（AI/自动化） | 中 — 提升优先级 |
| 区域互补 | 中 — 降低集中风险 |
| 成本优势（同等质量） | 中 — 需验证非以质量换价 |
| 生态韧性（增加多样性） | 低 — 长期价值 |

## 引入节奏控制

| 原则 | 理由 |
|------|------|
| 同时引入不超过2家 | 管理精力有限 |
| 错开引入至少1个月 | 给每家足够关注度 |
| 避开旺季 | 旺季管理成本高，适配期易出问题 |

**初始分量**：全新类型5%起步；有同类经验8-10%；其他产线迁移10-15%。

**管理成本**：新供应商前3个月管理成本是成熟供应商的3-5倍（每天额外约30分钟/家）。

## 准入诊断6问

| 问题 | 目的 | 判断标准 |
|------|------|---------|
| 为什么必须是这家？ | 验证必要性 | 答不上来→不紧迫 |
| 没有这家会怎样？ | 验证依赖度 | 没影响→不急；有影响→提高优先级 |
| 谁最需要说名字？ | 验证需求来源 | 业务线真实需求 vs 供应商自推 |
| 最小试单是什么？ | 验证试错成本 | 成本高→保守；低→可试 |
| 有成功案例吗？ | 验证可行性 | 有同类型→加分；无→提高风险等级 |
| 3年后还需要吗？ | 验证长期价值 | 不需要→不值得投入管理成本 |

## 权衡因素

| 正方（引入） | 反方（不引入/缓引入） |
|-------------|---------------------|
| 产能不足有需求 | 管理精力有限，管不好等于白引 |
| 资质不错错过可惜 | 市场上不只这一家，不急 |
| 对标有优势 | 历史数据可能美化，需验证 |
| 战略价值高 | 初始管理成本高，可能影响现有管理质量 |

## 叙事草稿

### 向上汇报

> "建议引入XX供应商做[产线]业务。9维度评估中，过往业绩和同类型对标达到现有供应商中上水平，管理方式适配度较高，红线全部通过。战略上能[填补空白/增加多样性/降低头部依赖]。建议初始分量X%，按6月里程碑管理：出勤率>=85% → GMV>=中位60% → 排名前70%。需要你支持的：[现场考察/合同审批/分量确认]。"

### 对供应商

> "经过评估，我们决定给你试单机会。初始分量X%，按[产线]分配。前两个月观察期，关键是出勤率和转化率达标。你的管理团队我们比较认可，但培训体系需要再加强。有问题随时找我。"

### 内部同步

> 引入需同步：策略组（新分量从哪调）、质检（纳入质检计划）、业务管理（增加日常关注）。

## 常见误区

| 误区 | 正确做法 |
|------|---------|
| 只看供应商自报数据 | 交叉验证：天眼查+同行+现场考察 |
| 急着引入不管时机 | 同时不超过2家，错开至少1个月 |
| 初始分量给太多 | 5-10%起步，看效果再加 |
| 跳过现场考察 | 前3个月至少去1次现场 |
| 被供应商老板个人魅力带跑 | 评估公司不是评估老板 |
