# N1 月度复盘叙事

> 每月主动输出自己的观察总结，不等策略组先说。把"一线管理者看到的"变成"军哥记住的"。

---

## 叙事主题

> 不是"本月业绩XX万"——是**"本月我观察到的最重要的一件事是什么"**。
>
> 策略组的月度通报会说GMV、转化率、拨打量。那些是他们的主场。你的叙事要从一线管理者的视角提炼一个判断——这个月供应商生态里发生了什么结构性变化？哪些趋势值得军哥记住？
>
> 好的叙事主题像一句话新闻标题，不是流水账。例：
> - "这个月最大的变化不是谁升了谁降了，是赛维斯的主管更替——这比排名波动重要十倍"
> - "新人稀释效应在加速：6家里有3家新人占比超过30%，这不是单家问题，是系统性压力"
> - "毅航连续三个月领跑的同时头部差距在拉大——标杆越强，后面的压力越大"

---

## 三层叙事

### 第一层：数据层 — 发生了什么

> 用工具箱的双指标卡片（自身峰值比 + 头部差距）描述现状，不要堆砌所有数据。

**必选数据（每份月度复盘必须包含）**：

| 数据项 | 来源 | 怎么呈现 |
|--------|------|----------|
| 整体格局 | 月度排名追踪表 | S/A/B/C各几家，和上月比是扩张还是收缩 |
| 双指标卡片 | 工具箱第十二节 | 每家一行：业绩达成 + FCI能力 + 状态判断 |
| 变化最大的一家 | 排名环比 | 升最多或降最多的那家，一句话说清楚为什么 |
| 后30%动态 | 触发规则检查 | 有没有新增/连续/回正，决定下个月的管理节奏 |

**选填数据（有异常才说）**：

| 数据项 | 什么时候说 |
|--------|-----------|
| 熟手人均GMV异动 | 自身峰值比 < 65% 或环比下降 > 10% |
| 新人拉低幅度 | 新人占比 > 30% |
| 流失率异动 | ≥ 8%/月 |
| 主管胜任评估 | ≤ 2项达标 |
| 合规红线触发 | 质检A类 ≥ 1 或信息安全管控 < 70% |
| 隐性退步/隐性进步 | 排名和人效方向相反 |

**数据呈现原则**：

- 策略组说"本月GMV总额XX万，同比增长XX%"——你不要重复这个
- 你说"赛维斯本月熟手人均GMV 37.1万，峰值比88%，但50个新人拉低了总人均26%——真实管理水平没有下降，是扩编期的阵痛"
- 数据层的价值不在于有多少数字，在于**选择哪些数字说、怎么解读**

### 第二层：判断层 — 意味着什么

> 基于年老师的四维框架（合规、运营效率、组织治理、风险控制）给出判断。不是策略组的"转化率下降了2个百分点所以有问题"，是"从运营一线看，这个数据变化意味着什么"。

**判断来源矩阵**：

| 判断维度 | 你看到的（一线视角） | 策略组看不到的 | 怎么说让军哥记住 |
|---------|---------------------|---------------|-----------------|
| **合规** | 巡检发现外网管控率60%、坐席私下用手机 | 质检分数只是结果，不知道现场怎么发生的 | "华啸不是坐席行为问题，是技术基础设施缺失——20个工位根本没有管控能力" |
| **运营效率** | 供应商扩编但老员工人均拨打量在降 | 只看总量GMV不知道产能为什么高/低 | "同样扩50人，岐力2周到位、翰锐还在招。扩编期管理能力差距暴露最明显" |
| **组织治理** | 核心主管上个月提离职被挽留、老板最近不主动联系你 | 只看流失率数字不知道流失原因 | "赛维斯主管胜任只有2项达标，整改执行力33%——问题不是不知道做什么，是管不动团队" |
| **风险控制** | 听说竞品在大量挖人、某供应商老板态度变冷 | 没有前瞻性风险预警，只有事后数据 | "最近XX公司在大量招聘电销人员，可能导致我们供应商人员流失加剧，建议提前做梯队备份" |

**判断输出的核心格式**：

每个判断由三部分组成：事实 + 推断 + 信心度。

> **判断示例1**：
> - 事实：岐力主管胜任评估2/5，流失率12%，整改闭环率33%
> - 推断：管理能力系统性缺失，不是"做得不好"，是"管不动"
> - 信心度：高（数据 + 一线沟通双重支撑）
>
> **判断示例2**：
> - 事实：毛毛虫最近两次沟通都很安静，没有主动找你
> - 推断：可能在考虑减少投入或退出，结合其排名仍在S级，更可能是对当前份额不满
> - 信心度：中（信号强烈但未验证，需进一步确认）

**判断层的禁忌**：

- 不要说"需要关注"——这是废话，所有供应商都需要关注。要说"我判断XX，因为YY"
- 不要说"建议加强管理"——这不是判断，是口号。要说"管理能力缺失，具体表现为XX"
- 不要回避不确定——不确定的事标注"信心度：中"，比假装确定更有信用

### 第三层：建议层 — 该怎么做

> 具体的、可执行的建议。不是"请领导指示"，是"我建议做X，因为Y，预计Z"。

**建议格式**：

每条建议必须回答四个问题：对谁、做什么、为什么、什么时候。

| 建议类型 | 格式 | 示例 |
|---------|------|------|
| **约谈建议** | 约谈[谁]，因为[事实]，要求[产出]，[日期]前完成 | "约谈岐力，因为主管胜任≤2项，要求提交管理团队调整方案，5月10日前" |
| **资源调整建议** | 给[谁][加/减]什么，因为[数据]，预计[效果] | "建议给广达增加20%名单量，因为头部差距连续2月在缩小（75%→82%），产能利用率在提升" |
| **风险预警建议** | [谁]可能在[X周/月]后出现[Y问题]，建议提前[Z] | "赛维斯新员工占比35%且有2名熟手离职，如不干预预计6月排名跌入后30%" |
| **标杆配对建议** | 用[标杆]的做法配对[对标方]，聚焦[具体做法] | "毅航的晨会目标拆解复制到赛维斯，聚焦复贷分组拨打" |
| **豁免建议** | [谁]享受考核豁免，因为[甲方责任]，豁免[什么] | "赛维斯本月扩编由甲方发起，建议人效维度豁免2个月，标注扩编豁免期" |

**建议优先级**：

| 优先级 | 判断依据 | 示例 |
|--------|---------|------|
| P0 立即 | 合规红线触发 / A级骤降 / 预判3月内会暴雷 | "华啸外网管控60%，立即整改" |
| P1 本月 | 连续预警 / 主管不胜任 / 资源匹配失衡 | "岐力主管胜任问题，本月约谈" |
| P2 下月 | 趋势性下滑 / 新人稀释 / 标杆配对 | "赛维斯新人培养计划，下月跟踪转正" |
| P3 观察 | 单月波动但根因明确 / 隐性进步需确认 | "翰锐排名降但人效升，观察是否主动缩编提质" |

## 对标策略组

> 月度复盘的核心目的之一：让军哥记住**你的观察**，而不只是策略组的数据。

### 策略组月度通报会说什么

| 内容 | 策略组怎么说 | 你的叙事补充什么 |
|------|------------|----------------|
| 整体业绩 | "本月GMV总额XX万，达成率XX%" | "达成率的背后，是6家里3家新人占比超30%——总量达标但结构性压力在累积" |
| 转化率 | "产线平均转化率XX%，环比升/降X个百分点" | "转化率下降的那家，是因为新主管上任还在磨合——这层数据上看不到" |
| 供应商排名 | "第一名XX，最后一名XX" | "排名之外，毛毛虫虽然还在S级，但最近两次没有主动联系我——信号比排名早" |
| 异常说明 | "XX供应商本月业绩下滑，原因待确认" | "我已经和XX确认了原因：核心主管离职，新主管还在交接。预计下月中恢复" |

### 怎么说让军哥记住你的观察

**三原则**：

1. **比策略组早**：他们看数据趋势，你看一线信号。你说"我预判这家3个月后会出问题"，3个月后真出了——军哥记住的是你的预判，不是策略组的事后分析。
2. **比策略组深**：他们说"转化率下降2个百分点"，你说"转化率下降是因为新主管上任后培训节奏乱了，老员工拨打量降了15%"。根因叙事 > 结果叙事。
3. **比策略组有立场**：他们的通报是"描述现状"，你的复盘是"判断+建议"。军哥听完策略组的数据通报，再听你的判断，两件事拼在一起才是全貌。

**话术示例**：

> "策略组会告诉您这个月谁排第一谁排最后。我补充三个一线观察：
> 第一，赛维斯从第3掉到第5，表面看是新人稀释，但老员工人均拨打量也在降——不只是新人问题，是管理节奏松了。
> 第二，毅航连续三个月领跑的同时头部差距在拉大，标杆越强后面的压力越大，下个月可能出现'追赶者掉队'的情况。
> 第三，岐力主管胜任只有2项达标，整改执行力33%。我判断不是能力问题是管不动团队，建议本月约谈谈管理团队调整。"

## 预判机制

> 月度复盘不是总结过去，是**为下个月设定议题**。

### 下月预判模板

| 预判类型 | 格式 | 怎么验证 |
|---------|------|---------|
| 排名预判 | "[谁]可能[升/降]到[X]名，因为[Y]" | 下月排名出来后对照 |
| 风险预判 | "[谁]可能在[X周/月]后触发[Y]条件" | 触发时间到了检查 |
| 结构性变化预判 | "[趋势]正在发生，[Z]月后会体现在排名上" | 后续月份的趋势确认 |
| 机会预判 | "[谁]处于隐性进步期，如果[给什么]，可能[变成什么]" | 给了之后的效果 |

**预判的纪律**：

- 每个月度复盘最多3条预判，不要贪多
- 每条预判必须有时间锚点（"X周后"/"X月后"）
- 上个月的预判必须在下个月的复盘中验证——对了说对了，错了说错了，不藏着
- 预判错了不可怕，预判对了加分巨大。关键是**有预判**——有预判说明你在思考，没有预判说明你在等数据喂你

## 执行节奏

| 时间节点 | 动作 | 耗时 | 产出 |
|---------|------|------|------|
| 月初第1周 | 导出数据 → 更新排名 → 双指标卡片 → 叙事主题提炼 | 60min | 数据层完成 |
| 月初第1周 | 基于数据 + 一线观察写判断层 | 30min | 判断层完成 |
| 月初第2周 | 写建议层 → 上月预判验证 → 本月预判 | 20min | 建议层 + 预判完成 |
| 月初第2周 | 合成月度管理报告 → 给军哥的月度简报 | 20min | 输出物就绪 |
| **合计** | | **约2小时** | |

## 质量自检

完成月度复盘后，对照以下清单逐项检查：

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 有没有一句话叙事主题？ | 15字以内，说清楚"这个月最重要的一件事" |
| 2 | 数据层有没有重复策略组的数据？ | 不重复GMV总额、产线转化率等策略组主场数据 |
| 3 | 判断层每条判断都有事实+推断+信心度？ | 三要素缺一不可 |
| 4 | 建议层有没有"请领导指示"？ | 没有。每条建议都是"我建议X因为Y预计Z" |
| 5 | 上月预判有没有验证？ | 有。对了说对了，错了说错了 |
| 6 | 本月预判有没有时间锚点？ | 每条预判都带"X周/月后" |
| 7 | 对标策略组说清楚了差异？ | 军哥听完知道"策略组看什么、我看什么、为什么不同" |
