# Q5 周度突变信号

> 周度扫描时发现异常指标，快速判断该不该介入、怎么介入。

## 30秒判断三件事

1. **信号是真异常还是甲方导致的？**：先查本周是否调整了名单分配、缩量、上线新策略。甲方决策导致的波动不是供应商的问题，先内部对齐再沟通。
2. **是哪种信号、紧迫度多少？**：出勤<85%（问一声）/ GMV比近4周均值降>20%（当天了解）/ 单周离职>5%（可能管理崩了，最危险）。
3. **供应商自己知道吗？**：出勤异常他们通常自己知道但不一定报；业绩断崖他们可能还在解释中；流失加速往往他们自己也在慌。

## 一句话口径

- **出勤异常**："这周出勤降了不少，是什么情况？"（不要说"你出勤不达标"，先听原因）
- **业绩断崖**："今天看到数据波动比较大，现在方便聊一下吗？"（当天联系，语气平但不拖）
- **流失加速**："最近人员变动有点频繁，运营上遇到什么困难了？"（不是质问，是打开话题）
- **如果是甲方决策导致**：不联系供应商，先内部确认影响范围，再决定是否需要同步给供应商。

## 需要调的数据

- 近4周出勤率趋势：供应商出勤日报汇总表
- 近4周GMV均值 vs 本周GMV：产线日报数据
- 本周离职人数 vs 月度离职率：人事变动台账
- 甲方本周是否有策略调整：策略组/王易人确认

## 转给谁

- 连续2周触发业绩断崖且原因不明 → 报王易人，建议约供应商管理层面谈
- 流失加速+出勤异常同时出现 → 报王易人，可能需要现场走访
- 确认是甲方策略导致 → 内部先闭环，年老师自主判断是否需要同步供应商
- 疑似供应商系统性造假 → 触发Q7红线流程，立即报王易人
