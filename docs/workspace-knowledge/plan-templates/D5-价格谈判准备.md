# D5 价格谈判准备

> 供应商说"成本涨了要涨价"，30分钟内完成立场判断、弹药准备、底线设定。

## 触发信号

- 供应商老板/商务提出调价要求（涨价/降价）
- 季度/年度合同续签涉及价格调整
- 新产线定价谈判
- 供应商暗示"结算太低影响投入"（培养悖论定律的前兆）
- 结算政策调整后供应商反弹

## 问题拆解

> 他说的成本上升，是真的还是试探？我让不让步，让多少？让步的同时能不能换回来对等承诺？

三个核心问题：

1. **成本判断**：他的成本真的涨了吗？涨了多少？同产线其他供应商的成本趋势如何？
2. **替代成本**：如果不让步，替换这家供应商的成本有多高？有没有第二梯队可以顶上？
3. **博弈空间**：他的真实底线在哪？我让步的同时能换来什么？

## 归因路径（决策树）

### 第一层：判断涨价原因是否真实

> 不要默认供应商说的是真的，也不要默认是假的。用数据验证。

**核实方法清单：**

| 核实维度 | 怎么查 | 判断标准 |
|---------|--------|---------|
| 同产线供应商成本趋势 | 问其他供应商最近成本变化 | 普遍上涨 → 市场因素；只有他家涨 → 个体因素 |
| 行业公开数据 | 查招聘网站薪资、办公租金行情 | 人均成本涨幅和行业匹配吗？ |
| 供应商历史报价 | 翻历史合同和结算记录 | 上次涨价是什么时候？频率正常吗？ |
| 供应商扩缩编情况 | 查近期人员变动 | 扩编期成本短期上升是正常的 |
| 结算占比分析 | 算结算金额占供应商营收比例 | 如果占比很高，涨价动机更强 |

**判断结论：**

| 判断 | 特征 | 信心度 |
|------|------|--------|
| **确实成本上升** | 同产线多家供应商反映、行业数据可验证、涨幅合理 | 高 |
| **部分真实部分试探** | 有一定成本压力，但涨幅夸大 | 中 |
| **纯试探底线** | 只有他家反映、行业无变化、近期排名下滑想保量 | 低 |

### 第二层：根据原因选策略

**路径A — 确实成本上升（市场性涨价）**

> 不强硬压价，但也不能全盘接受。适度让步 + 要求对等承诺。

| 动作 | 具体做法 |
|------|---------|
| 承认成本压力 | "我知道最近行业成本确实有变化"（不卑不亢） |
| 限定让步幅度 | 不超过成本上升幅度的50%-70%（他也要承担一部分） |
| 要求对等承诺 | 涨价可以，但要承诺：出勤率不降、产能提升X%、人员稳定 |
| 设定观察期 | 先给3个月过渡价，到期看效果再定是否永久调 |
| 保持竞争性排名 | 调价后排名机制不变，不会因为涨了价就照顾排名 |

**路径B — 部分真实部分试探**

> 拆分处理：真实的部分承认，试探的部分挡回去。

| 动作 | 具体做法 |
|------|---------|
| 拿出数据 | "我了解了一下行业情况，人均成本大概涨了X%，你说的Y%是不是包含了你自己的管理效率折损？" |
| 分段谈判 | "成本部分我理解，但管理效率的部分，你先优化内部，我们半年后再看" |
| 给台阶 | 不直接说"你在试探"，说"有些成本项我需要你提供明细，我们一项一项对" |
| 引入对标 | "同产线其他供应商目前的价格在X-Y区间，你的诉求超出这个区间了" |

**路径C — 纯试探底线**

> 用数据证明当前价格合理，不给让步。但要保留关系空间。

| 动作 | 具体做法 |
|------|---------|
| 拆穿试探（温和） | "你的成本数据和行业数据差距比较大，能具体说说哪些项涨了吗？" |
| 强调排名逻辑 | "价格调整要结合排名来看，你们目前排名第X，在这个位置上，当前价格是有竞争力的" |
| 指出替代选项（暗示） | "第二梯队有几家在追赶，价格更有优势"（不直接威胁，让他自己算账） |
| 给出替代方案 | "价格暂时调不了，但如果你能提升转化率X个百分点，从绩效侧可以补回来" |
| 记录信号 | 试探底线 = 对当前合作状态有不满或对外部有新选项，记入供应商画像 |

### 第三层：判断替代成本

> 让不让步不只是看供应商的态度，还要看你有没有后手。

| 替代成本评估 | 高（换不了） | 中（能换但有代价） | 低（随时换） |
|-------------|-------------|-------------------|-------------|
| 供应商排名 | 前30% | 中段 | 后30% |
| 产线依赖度 | 独家或主力之一 | 多家分担 | 可替代 |
| 人员规模 | 500人以上 | 100-500人 | 100人以下 |
| 管理成本 | 深度绑定，替换成本极高 | 中等，3个月过渡期 | 低，1个月可切换 |
| 梯队备份 | 第二梯队未就绪 | 第二梯队部分就绪 | 第二梯队已就绪 |

**替代成本高时**：即使对方在试探，也要温和处理，不能撕破脸。但启动梯队培育计划。
**替代成本低时**：态度可以更硬，"你不愿意干有的是人干"（不说出来，但底气在这）。

## 权衡因素

| 正方（让步） | 反方（不让步） |
|-------------|---------------|
| 供应商确实难，不涨可能投入下降 | 一家涨了，其他家会跟风 |
| 头部供应商要稳住，不能因小失大 | 让步空间有限，预算不是无限的 |
| 涨价换承诺，可能是划算的交易 | 培养悖论定律 — 做大了反而议价 |
| 适度让步显示甲方的诚意和长期合作意愿 | 数据说明当前价格合理，让步没有依据 |
| 替代成本高，硬顶可能导致产能缺口 | 让步要留痕，不能每次都让 |

## 谈判弹药准备

> 进谈判前，确保手上有这些东西。

**必带数据（谈判前30分钟准备）：**

| 数据项 | 来源 | 用途 |
|--------|------|------|
| 该供应商近6个月排名 | 月度排名表 | 谈判底气 — 排名好说话才有分量 |
| 同产线其他供应商价格区间 | 合同/结算记录 | 对标 — 你的价格在什么水平 |
| 该供应商结算金额趋势 | 财务结算数据 | 算账 — 涨价影响多少钱 |
| 人员/出勤/转化率趋势 | 运营数据 | 交换 — 涨价可以但你要承诺这些 |
| 行业人均成本参考 | 招聘网站/同行交流 | 验证 — 他说涨了到底涨没涨 |

**话术弹药：**

| 场景 | 话术 |
|------|------|
| 承认压力但不全盘接受 | "成本压力我理解，但涨幅需要你提供明细，我们一项一项看" |
| 引入对标 | "同产线目前X家供应商的价格在A-B区间，你的诉求超出这个范围了" |
| 要求对等承诺 | "价格调整可以谈，但需要你承诺出勤率不低于X%、转化率提升Y个百分点" |
| 设定观察期 | "先给3个月过渡价，到期我们看效果再定" |
| 温和拒绝 | "当前价格体系是基于市场竞争形成的，暂时没有调整计划" |
| 给台阶 | "你在其他方面（培训投入/管理效率）如果能提升，我可以从绩效侧帮你争取" |

## 叙事草稿

### 向上汇报（给王易人/军哥）

> 谈判结果要让上级知道三件事：为什么调/不调、对方什么态度、你的判断依据。

**涨价方案汇报（同意调价时）：**

> XX供应商提出涨价X%，经核实同产线确实存在成本上升趋势。考虑到该供应商排名前X、产线占比X%、替换成本较高，建议同意调价X%（低于对方诉求的Y%），条件是出勤率不低于X%、转化率季度提升Y个百分点。先执行3个月过渡价，到期评估效果。预计年化增加成本约XX万元。

**拒绝涨价汇报（不让步时）：**

> XX供应商提出涨价X%，经核实当前价格在同产线处于合理区间，成本上升幅度低于其诉求。该供应商排名第X，产线内有多家替代选项。建议维持现有价格，同时关注其后续投入变化。已暗示可从绩效侧优化收益，对方态度偏软，判断试探成分居多。

### 对供应商

**同意部分调价：**

> "成本压力我理解，行业确实有变化。但涨幅这块我们需要一项一项对，你先发一份明细过来。我这边可以给到X%的调整幅度，前提是出勤率和转化率要保证。先按这个执行3个月，到期我们看效果。"

**维持现状：**

> "你提的需求我认真看了，也做了内部评估。当前价格体系是综合市场竞争形成的，暂时没有调整空间。但如果你能在转化率/人效上再提一提，从绩效侧可以补回来。这个方案你先考虑一下。"

### 内部同步

> 如果调价影响结算，需要同步财务（预算影响）、策略组（分量调整是否联动）、质检（是否同步调整质量要求）。

## 关键变量

| 变量 | 判断标准 | 影响 |
|------|---------|------|
| **供应商排名** | 前30% / 中段 / 后30% | 排名越高谈判底气越足，但也越不能撕破脸 |
| **产线依赖度** | 主力 / 普通 / 可替代 | 依赖度越高让步空间越大 |
| **梯队备份状态** | 已就绪 / 部分就绪 / 未就绪 | 备份就绪则态度更从容 |
| **涨价幅度** | <5% / 5%-15% / >15% | 小幅可谈，大幅需要硬数据支撑 |
| **供应商态度** | 柔和协商 / 正式提出 / 带威胁 | 态度越硬你越需要数据准备充分 |
| **预算约束** | 有空间 / 刚好 / 已超 | 预算决定你的最终天花板 |
| **上级态度** | 王易人知情并授权 / 未沟通 | 重大调价必须先沟通上级 |

## 常见误区

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 不核实就信了供应商的成本数据 | 多花冤枉钱 | 先核实，再谈判 |
| 当场答应 | 后续反悔更被动 | "我内部评估一下，X天后回复" |
| 只谈价格不谈条件 | 涨了价供应商该怎样还怎样 | 涨价必须绑定对等承诺 |
| 一家涨了其他家不知道 | 其他家觉得不公平或跟风 | 调价信息保密，但要有统一逻辑 |
| 谈完不记录 | 下次谈判没有基线 | 每次谈判结果写入供应商画像 |
| 替代成本高就全盘接受 | 被绑架 | 即使替代成本高也要培育梯队 |
