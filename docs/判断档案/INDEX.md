## 视图 3 · 按主题聚类
*元视角——跨样本的规律，可能是更高层的判断力*

### 🎯 主题 · "判断对象 vs 合理近似物"（3 条 · 升级候选）
AI 系统性地用"可编码/可合理化的近似物"替代真实判断对象
- [基准翻转](2026-07-04-haglofs-basis-assumption-flip.md) · 把"质感"近似成"叙事一致性"（可编码）
- [审美锚点](2026-06-21-nian-aesthetic-anchor-recovery.md) · 把"调性"近似成"色相覆盖"（可合理化）
- [排除方向](2026-07-05-42zhangjing-exclusion-as-entry.md) · 把"砍回到一件事"近似成"五模块平铺"（可结构化）
- **通用规则候选**：判断对象若被替换成"看起来合理但偏离真实"的近似物，所有后续工作会在错误根基上累积

### 🎯 主题 · "先诊断根因再行动"（3 条 · 升级候选）
遇到问题先归因（A 还是 B），再决定方向（加东西还是建约束）
- [灰蒙蒙根因](2026-06-21-nian-color-rules-vs-palette.md) · 资源不足 vs 规则缺失 → 建约束
- [905≠905](2026-06-29-insightasset-purify-before-extract.md) · 吞吐量 vs 输入质量 → 净化
- [完成失真](2026-07-01-refactor-completion-means-landed.md) · 过程完成 vs 状态完成 → 落库
- **通用规则候选**：遇到"X 不够/不对"的反馈，先诊断是"资源/速度/进度"问题还是"规则/质量/状态"问题，前一类加东西，后一类建约束

### 🎯 主题 · "AI 系统性偏好"（贯穿 8 条 · 元规则候选）
AI 有一组稳定的默认偏好，识别它们 = 判断力建设的核心

| AI 系统性偏好 | 出现在 |
|---|---|
| 叙事化倾向（用叙事近似质感） | 基准翻转 |
| 完整性幻觉（自报统计偏大） | AI 自报 |
| 资源堆砌（加东西而非建约束） | 灰蒙蒙根因 |
| 全量处理（不净化就处理） | 905≠905 |
| 主体扁平化（提问者当中性） | 三层声音 |
| 过程完成即标记完成 | 完成失真 |
| 主流技术匹配（shadow/对称框） | shadow / 对比框 |
| 框架展示倾向 | 故事线选择 |
| 流程优化倾向（裁剪已确认流程） | 流程省略 |

- **元规则候选**：AI 的"默认答案"来自训练数据高频匹配（叙事/对称/主流技术/完整覆盖），但这些默认往往与"克制/editorial/真实判断对象"冲突。判断 AI 产出质量的标准之一：识别它落入了哪个系统性偏好
