## 优化后

**邮件主题**：关于2026年2月赛马结果及3月专项提升期安排的通知

**邮件正文**：

```
俊哥：

同步下2026年2月赛马结果及3月安排：

**触发情况：**
- 拓银、华道2月月均人力目标＜20人，触发赛马淘汰机制
- 华道已提出撤场
- 拓银需进入专项提升期

**专项提升期要求（1个月）：**
- 人日均激活 ≥ 大盘均值的70%
- 排名不为倒数第1

**真正淘汰条件（需同时满足）：**
- 进入专项提升期后，排名仍为倒数第1
- 人日均激活 < 大盘均值的70%

**规则说明：**
- 2月赛马标准：人日均核卡
- 3月赛马标准已更新为：人日均激活
- 详细规则参考：赛马规则文档

如有疑问，随时沟通。
```

## 优化点

| 优化维度 | 原稿问题 | 优化方案 |
|---------|---------|---------|
| **结构** | 内容混杂，重复表述 | 按"触发情况-要求-淘汰条件-规则说明"分层 |
| **重点** | 核心规则淹没在文字中 | 加粗关键术语，独立板块呈现 |
| **冗余** | 专项提升期要求重复2次 | 合并为1次，补充说明3月标准变更 |
| **可读性** | 段落过长，缺乏视觉分隔 | 短句+项目符号，每块不超过3行 |
| **专业性** | "＜"符号不规范 | 统一使用"<" |
| **信息完整** | 缺少邮件主题 | 补充明确主题 |

## 使用说明

此邮件已添加到 `supplier-evaluation.txt` 模板库，使用时：
1. 邮件主题写明"关于[时间]赛马结果及[周期]安排的通知"
2. 正文按触发情况→要求→淘汰条件→规则说明分层
3. 重要数据/要求可加粗突出
4. 最后添加"如有疑问，随时沟通"缓冲语
