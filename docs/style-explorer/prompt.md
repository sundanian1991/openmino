# 审美风格探索器 — 需求文档

## 🎯 目标

帮助用户探索自己真正喜爱的 UI 审美风格，并生成一份**审美偏好报告**，明确指出：
- 用户喜欢什么风格（如：Minimalist + Clean）
- 配色偏好（如：蓝色系、低饱和度）
- 排版偏好（如：大标题、宽松留白）
- 适合的行業场景（如：SaaS、企业官网）

## 🚫 排除项

- 不要渐变色
- 不要深色主题（默认浅色）
- 不要过于复杂/实验性的动画
- 不需要用户注册/登录（本地运行）

## 📦 交付物

1. **Next.js 应用** — 可本地运行（`npm run dev`）
2. **风格数据库** — 基于 ui-ux-pro-max 的 67 种风格、96 种配色、57 种字体
3. **交互流程** — 浏览 → 筛选 → 收藏 → 测评 → 生成报告
4. **报告导出** — PDF 或 Markdown 格式的审美偏好报告
5. **部署方案** — 可直接运行，或上传 GitHub 供他人下载

## ✅ 完成标准

| 维度 | 标准 |
|------|------|
| **功能** | 用户能浏览风格、做筛选、完成测评、生成报告 |
| **数据** | 包含 67 种风格、96 种配色、57 种字体、100 条规则 |
| **交互** | 流畅、直观，小白也能用 |
| **报告** | 清晰、可操作，能直接用到后续项目中 |
| **部署** | 本地能跑起来，或者 GitHub 仓库可下载 |

## 📊 核心流程

```
用户进入应用
     ↓
浏览风格库（67 种风格卡片）
     ↓
筛选/搜索（按行业、 mood、颜色）
     ↓
收藏喜欢的风格（3-5 个）
     ↓
进入测评流程（10-15 道选择题）
     ↓
生成审美偏好报告
     ↓
导出报告（PDF/Markdown）+ 配置（CSS 变量/JSON）
```
