## 三、供应商触发规则状态机

基于**月度排名后30%**和**数据质量分**，定义9种状态和流转规则：

### 状态流转图

```
                   月度正常
                 (排名前70%)
                     │
         ┌───────────┼───────────┐
         ↓           ↓           ↓
     首次预警       A级骤降      诚信预警
  (本月后30%      (前30%→       (质量≤70
   上月不在)      后30%)        或1项不实)
     → 私下沟通      → 立即约谈     → 警告+3月加倍核查
         │           │               │
         └─────┬─────┘               │
               ↓                     ↓
           连续预警                 诚信红线
         (连续2月后30%)            (≥2项不实
         → 正式约谈+整改             或同一项连续2次)
               │                    → 约谈高层
               ↓                      │
           清退评估                  信任重建
         (连续3月后30%)            (连续3月质量=100)
         → 延长或清退             → 取消加倍核查
               │
               ↓
            (离开系统)
```

### 关键触发条件速查

| 状态 | 触发条件 | 你要做的事 | 完成标志 |
|------|---------|-----------|---------|
| **月度正常** | 排名不在后30% | 无需动作 | — |
| **首次预警** | 本月后30% 且 上月不在 | 私下沟通：问原因，记录原因 | 沟通记录已存档 |
| **连续预警** | 连续2个月后30% | 正式约谈：要求提交整改计划 | 整改计划已收到 |
| **清退评估** | 连续3个月后30% | 评估：延长观察期 / 启动清退 | 评估结论已记录 |
| **A级骤降** | 上月前30% 本月跌到后30% | **立即约谈**（不等连续） | 原因已查明 |
| **诚信预警** | 数据质量分≤70 或 1项不实 | 警告+标记：3个月加倍核查 | 警告已送达 |
| **诚信红线** | ≥2项不实 或 同一项连续2次 | 约谈供应商高层，告知造假后果 | 约谈记录已存档 |
| **回正** | 连续2个月后30%后回到前70% | 取消预警，归档记录 | 预警取消 |
| **信任重建** | 连续3个月数据质量=100 | 取消加倍核查，恢复正常 | 重建完成 |

### 核心规则

- **A级骤降必须立即行动**，不等三个月连续
- **诚信预警必须记录**，标记3个月加倍核查
- **整改计划必须评估质量**，不合格退回重写
- **信任重建需连续3个月**数据质量满分

## 四、产能双维度诊断矩阵

区分"供应商自身问题"还是"产线/甲方问题"的核心工具。

### 计算公式

```
自身峰值比 = 本月熟手人均GMV ÷ 该供应商近6个月最高熟手人均GMV
头部差距   = 本月熟手人均GMV ÷ 同产线当前第1名的熟手人均GMV
```

**关键规则**：
- 必须用**熟手人均**（入职>30天），排除新人稀释干扰
- 新人占比>30%时，必须标注新人拉低幅度
- 总人数变化>20%时，同时看总量GMV和熟手人均GMV

### 四象限判断矩阵

```
              头部差距
              ↑
        >80% ┌─────┬─────┐ 接近头部
  标杆型 (Ⅰ)│     │尽力型│  >85%
   双高      │     │高/低 │  正常发挥
  标杆分享   │     │评估产│
  60-80% ┌────┼─────┼─────┤ 65-85%
   有差距  │暂时│问题型│   有下滑空间
  (对标)  │波动│(Ⅲ)  │   关注趋势
        │低/高│双低  │
   <60% ┌────┼─────┼─────┤ <65%
   差距显│    │     │     │ 严重未发挥
   著    │    │     │     │ 立即约谈
        └────┴─────┴─────┘
           <60% 60-80%
```

### 四象限详解

| 象限 | 类型 | 特征 | 解读 | 动作 |
|------|------|------|------|------|
| **Ⅰ** | 标杆型 | 自身峰值比>85% + 头部差距>80% | 自己发挥好，也跟得上头部 | 标杆分享 |
| **Ⅱ** | 尽力型 | 自身峰值比>85% + 头部差距<80% | 自己已尽力，但产线/产品人效偏低 | 评估产线适配 |
| **Ⅲ** | 问题型 | 自身峰值比<65% + 头部差距<60% | 自己没发挥好，差距也在拉大 | **立即约谈** |
| **Ⅳ** | 暂时波动 | 自身峰值比<65% + 头部差距>80% | 短期因素导致，底子不差 | 观察2周 |

### 常见认知误区

- ❌ 用**总人数**算人均GMV（应强制用熟手人数）
- ❌ 新人占比>30%不看新人拉低幅度（必须标注）
- ❌ 总人数变化>20%只看总量（应同时看总量和熟手人均）
- ❌ 将问题型（双低）误判为产线问题（实际是内部管理问题）

## 七、核心价值提炼

### 工具箱的核心

> 将年老师的"手感"系统化为**可执行、可验证的制度**

- **9种状态触发**：首次预警、连续预警、清退评估、A级骤降、诚信预警/红线、回正、信任重建
- **31个异常信号**：周度3个（出勤/业绩/流失）+ 月度多个（峰值比/头部差距/诚信等）
- **双维度诊断**：自身峰值比 + 头部差距，区分"自己问题"vs"产线问题"
- **5项主管胜任评估**：流失率、出勤、人效趋势、响应质量、整改执行力

### official-doc 的核心

> 提供**100+场景的模板库**，确保输出规范、专业、要素齐全

- **29个文件**：通用公文4个 + 生命周期6个 + 扩展模块6个 + 高频13个
- **6种语气变体**：正式/柔和/紧急/协商/告知/质询，快速适配场景
- **写作标准**：段落加粗关键词、禁止装饰、数据结构化、留白适度

### 提效方案的核心

- 现有供应商人员大换血后的重建期
