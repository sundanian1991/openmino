## 2. 关键阈值速查

### 2.1 SLA 核心指标（来源：SM-2026-003 §2）

| 指标 | 达标 | 预警 | 红线 |
|------|------|------|------|
| 日均出勤率 | **≥95%** | <90% | <85% |
| 产能达标率 | **≥90%** | <80% | <70% |
| 数据报送及时率 | **≥98%** | <95% | <90% |
| 数据准确率 | **100%** | <98% | <95% |

### 2.2 合规指标（来源：SM-2026-003 §3）

| 指标 | 达标 | 预警 | 红线 |
|------|------|------|------|
| 信息安全违规 | **0次** | — | ≥1次严重违规 → 清退 |
| 人员持证率 | **100%** | <95% | <90% |
| 录音留存率 | **100%** | <98% | <95% |
| 话术合规率 | **≥95%** | <90% | <85% |

### 2.3 评估评级阈值（来源：SM-2026-030 §5.1）

| 评级 | 总分 | 含义 |
|------|------|------|
| A（优秀） | **≥90分** | 持续超出预期 |
| B（良好） | **80-89分** | 稳定达标 |
| C（达标） | **70-79分** | 基本达标 |
| D（预警） | **60-69分** | 未达标，需整改 |
| E（红线） | **<60分** | 严重不达标，启动清退评估 |

**一票否决**（直接降E）：信息安全严重违规、持证率<90%、质检月均分<70、数据造假

### 2.4 集中度管控（来源：SM-2026-001 §19）

| 管控维度 | 上限 |
|---------|------|
| 单供应商坐席规模占比 | **≤15%**（总纲）/ ≤20%（日常管理） |
| 单供应商产能占比 | **≤20%** |
| 前三大供应商合计占比 | **≤50%** |
| 单业务线单一供应商依赖度 | **≤60%** |
| 新供应商临时容忍 | **≤40%**（6个月内须降至20%） |

### 2.5 站点缩减触发线（来源：SM-2026-040 §3.4）

| 业务线 | 缩点触发线（连续2月） | 限留站点线 |
|--------|---------------------|-----------|
| 金条人工 | **<60席** | <40席限留1个站点 |
| 金条人机 | **<40席** | <30席限留1个站点 |
| 菁卡拉新 | **<30席** | <20席限留1个站点 |
| 企金/固收 | **<30席** | <20席限留1个站点 |

触及后给 **1个月整改缓冲期**。

### 2.6 准入评估阈值（来源：SM-2026-010 §5.2）

| 总分区间 | 决策 | 后续动作 |
|---------|------|---------|
| **≥70分**（均分≥3.3） | 准入 | 正常引入 |
| **60-69分** | 有条件准入 | 6月里程碑跟踪，C1审批 |
| **<60分** | 不准入 | 纳入储备库 |

**准入门槛（须同时满足）**：有效营业执照、坐席≥50人、持证率100%、独立质检团队、数据安全通过评估、无重大违规记录

### 2.7 试用期与里程碑（来源：SM-2026-010 §5.4）

| 节点 | 时间 | 检查内容 |
|------|------|---------|
| 试用期 | **30天** | 质检抽检比例 **2倍** |
| M1 | 上线后1个月 | 到岗率≥90%，培训通过率≥80% |
| M2 | 上线后2个月 | 产能达承诺值 **60%** 以上 |
| M3 | 上线后3个月 | 月度评估不低于C级 |
| M4 | 上线后4个月 | 产能达承诺值 **80%** 以上 |
| M5 | 上线后5个月 | 综合评估有提升趋势 |
| M6 | 上线后6个月 | **转正评估** |

### 2.8 主管胜任5项达标线（来源：供应商管理思路梳理 §2.3）

| 指标 | 达标线 |
|------|--------|
| 团队流失率 | **<8%/月** |
| 日均有效出勤率 | **>90%** |
| 人效趋势（近3月人均GMV） | **不连续2月下降** |
| 响应质量（预警后48h回复率） | **>80%** |
| 整改执行力（按期完成率） | **>70%** |

≤2项达标 = 不胜任信号

### 2.9 产能双维度阈值（来源：供应商管理思路梳理 §2.2）

| 维度 | 区间 | 判定 |
|------|------|------|
| 自身峰值比 | >85% | 正常发挥 |
| 自身峰值比 | 65%-85% | 关注趋势 |
| 自身峰值比 | **<65%** | 严重未发挥 |
| 头部差距 | >80% | 接近头部 |
| 头部差距 | 60%-80% | 有差距不大 |
| 头部差距 | **<60%** | 差距显著 |

### 2.10 整改与验收阈值（来源：供应商管理思路梳理 §2.5）

| 维度 | 通过标准 |
|------|---------|
| 目标达成 | **≥80%** |
| 动作执行 | **≥70%** 按期完成 |
| 持续性 | 整改后 **2个月** 数据未回落 |
| 根因消除 | 原失分项提升 **≥1个等级** |

### 2.11 数据真实性阈值（来源：供应商管理思路梳理 §2.7）

| 触发项数 | 数据质量分 | 判定 |
|---------|-----------|------|
| 0项 | 100分 | 可信 |
| 1项 | 70分 | 存疑，3个月加倍核查 |
| ≥2项 | 40分 | 不可信，排名加星号，不作为奖惩依据 |

### 2.12 新供应商首月标准（来源：SM-2026-010 §8.3）

| 指标 | 标准 |
|------|------|
| 首月人员流失 | **≤20%** |
| 首月出勤率 | **≥85%** |
| 响应业管需求 | **2小时内** |
| 整改计划提交 | 发现问题 **24小时内** |

### 2.13 保密与数据归档（来源：SM-2026-040 §7）

| 事项 | 期限 |
|------|------|
| 保密延续义务 | 退出后 **2年** |
| 录音留存 | **≥6个月** |
| 清退全流程记录归档 | **3年** |
| 清退复盘报告归档 | 长期 |

## 3. 角色职责速查

### 3.1 审批权限矩阵

| 动作 | 供应商管理岗 | C3审批 | C2审批 | C1审批 |
|------|-------------|--------|--------|--------|
| 新供应商引入（单业务线） | 发起+执行 | 审批 | 终审 | — |
| 全业务线/超集中度引入 | 发起 | 审核 | 审批 | **终审** |
| 站点缩减 | 发起 | 审批 | 抄送 | — |
| 单业务线退出 | 发起 | 审批 | 终审 | — |
| 全业务线强制清退 | 发起 | 审批 | 终审 | — |
| 有条件准入（60-69分） | 发起 | 审核 | 审批 | **终审** |
| 紧急引入（7天周期） | 发起 | — | — | **审批** |
| 清退禁入期豁免 | 建议 | 审核 | 审批 | **终审** |
| 扩项审批（≥B级） | 发起 | 审核 | 终审 | — |
| 分量调整≥50%幅度 | 拟定 | 审批 | 终审 | — |

### 3.2 岗位职责分工

| 角色 | 核心职责 |
|------|---------|
| **策略岗** | 发起引入需求、提供业务缺口分析、产能预测 |
| **供应商管理岗** | 制度制定/修订、准入评审、日常监控、月度评估、清退执行、整改下发 |
| **业务运营岗（业管）** | 产能管理、日常交付监督、寻源邮件发送、系统建档、账号开通、数据回收 |
| **质检团队** | 质量抽检、质检月报、重大合规通报、标准校准 |
| **结算岗** | 清退结算、违约金核算、压3个月执行 |
| **站点对接人（供应商）** | 信息报送、沟通协调、异常报备、花名册维护 |
| **站点负责人（供应商）** | 统筹本站点全部业务、月度总结提交 |

### 3.3 升级路径

| 异常级别 | 响应时间 | 升级路径 |
|---------|---------|---------|
| 一般异常 | 当日处理 | 站点自行处理 |
| 重要异常 | 2小时内 | 升级至供应商管理岗 |
| 紧急异常 | 立即 | 供应商管理岗 + 业务运营岗 |
| 合规红线 | 立即 | 供应商管理岗 → C3 → C2（紧急24h内） |
