## 一、核心计算公式（8个）

### 1.1 产能利用率双维度公式

**可视化类型**：公式说明卡片

**内容结构**：
```
┌──────────────────────────────────────────┐
│ 产能利用率双维度计算                      │  ← H2标题
│ ──────────────────────────────────────   │
│                                          │
│ 维度A：自身峰值比 = 当前产能 / 自身历史峰值 │  ← Body
│                                          │
│ 维度B：头部差距 = (头部峰值 - 当前产能)    │
│              / 头部峰值                   │
│                                          │
│ 评估标准：                                │  ← Caption说明
│ • 双高：标杆型（维持）                     │
│ • 一高一低：潜力型（分享）/ 问题型（约谈） │
│ • 双低：问题型（立即行动）                 │
└──────────────────────────────────────────┘
```
