# 一、核心决策工具（11个）

## 1.1 供应商对比矩阵（四象限散点图）

**元数据**
```yaml
id: 1.1
name: 供应商对比矩阵
type: 核心决策工具
priority: ⭐⭐⭐⭐⭐
core_value: 异常发现的第一入口
form: 四象限散点图 + 气泡
```

**ASCII布局**
```
┌──────────────────────────────────────┐
│  供应商对比矩阵                       │
│  ─────────────────────────────────  │
│                                      │
│  Y轴：自身峰值比（高→低）             │
│  X轴：头部差距（大→小）               │
│                                      │
│  四象限背景色差异：                   │
│  - 左下(问题型)：焦橙20% → 最醒目     │
│  - 右上(标杆型)：橄榄绿15% → 次醒目   │
│  - 其他：暖灰背景                     │
│                                      │
│  气泡：                               │
│  - 位置 = (X, Y)坐标                 │
│  - 大小 = 产能规模                    │
│  - 标签 = 供应商名 + 峰值比%          │
└──────────────────────────────────────┘
```

**视觉呈现策略**
| 元素 | 规范 |
|------|------|
| 轴标签 | system-ui 10px `#6b6b6b` |
| 象限背景 | 问题型 rgba(212,132,90,0.2) / 标杆型 rgba(143,155,127,0.15) |
| 气泡描边 | 1px solid `#1a1a1a` |
| 气泡填充 | `#faf9f7` |
| 气泡标签 | 供应商名(10px bold) + 峰值%(9px) |

**数据结构**
```json
{
  "title": "供应商对比矩阵",
  "xAxis": {"label": "头部差距", "direction": "大→小"},
  "yAxis": {"label": "自身峰值比", "direction": "高→低"},
  "quadrants": [
    {"id": "p0", "name": "问题型", "background": "warning-20"},
    {"id": "p1", "name": "标杆型", "background": "success-15"},
    {"id": "p2", "name": "普通型", "background": "warm-gray"},
    {"id": "p3", "name": "稳健型", "background": "warm-gray"}
  ],
  "suppliers": [
    {"name": "华啸", "x": 0.25, "y": 0.25, "size": 48, "value": "58%", "type": "p0"},
    {"name": "毅航", "x": 0.75, "y": 0.85, "size": 60, "value": "95%", "type": "p1"}
  ]
}
```

**逻辑链条**
```
查看矩阵 → 发现左下象限(P0)有华啸 → 点击气泡 → 跳转单体深度看板
```

**交互关系**
- 点击气泡 → 跳转 1.3 单体深度看板

```yaml
id: 1.2
```

```
┌──────────────────────────────────────┐
│  10月供应商排行榜                     │
│  ─────────────────────────────────  │
```

```json
  "rankings": [
```

```json
    {"type": "info", "message": "FCI改善中"}
```

**逻辑链条**
```
查看单体看板 → 核心指标异常 → 查看趋势确认持续性 → 查看约谈历史
→ 决策：是否再次约谈/份额调整/清退
```

**交互关系**
- 返回按钮 → 回到 1.1/1.2
- 导出按钮 → 生成 PDF 报告
