# 筹备期 · 培训通知模板

> **定位**：培训通知不只是一封邮件——它是供应商对培训的第一印象，决定了参与意愿和准备态度。
> **核心原则**：通知分为正式邀请（给供应商老板）+ 内部通告（给京东同事），各用不同语气和结构。

---

## 一、线上圆桌通知模板

### 供应商版（正式邀请）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 X 期供应商联盟圆桌 · 邀请函
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

致 [供应商名称] [联系人]：

供应商联盟第 X 期圆桌将于 [日期] [时间] 举办，本期主题为 [主题名称]，由 [主讲人公司] [主讲人姓名] 主讲。

【为什么这场值得参加】
[一句话说明本期内容对供应商的价值——"如果你在 XX 问题上头疼过，这场对你很关键"]

【基本信息】
- 时间：[日期] [时间]（预计 60 分钟）
- 形式：腾讯会议/飞书视频（链接：[链接]）
- 主讲人：[姓名]，[公司名称]，[角色]
- 主题：[主题名称]

【参与要求】
- 欢迎全体供应商成员参加，不需准备，只需准时上线
- 请于 [日期] 前回复是否参加，方便我们安排

【下期预告】
下期圆桌暂定 [日期]，主题 [主题名称]，主讲人为 [供应商名称]。

如有疑问，请联系 [运营官姓名]（微信/电话：[联系方式]）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 同事版（内部通告）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 X 期供应商联盟圆桌 · 同事邀请
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

X 哥/X 姐：

第 X 期供应商联盟圆桌定在 [日期] [时间]，主题是 [主题名称]。主讲人是 [供应商名称] 的 [姓名]，讲 [内容]。

对你来说，价值在于：[一句话说明为什么对这位同事有价值——"能听到多家供应商在同一件事上的真实做法，这是平时对一接触不到的"]

本期圆桌结构：
- 30 分钟主讲分享
- 10 分钟同事点评（你只需要听，讲真实感受，无需备课）
- 20 分钟自由讨论

时间你定，我配合。如果 [日期] 方便，回复我一下，我把会议链接发你。

[运营官姓名]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 二、线下大额培训通知模板

### 供应商版（完整邀请函）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
大额税票贷供应商联盟培训 · 邀请函
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

致 [供应商名称] [负责人]：

京东供应商联盟定于 [日期] 举办"大额税票贷供应商培训"，本期由 [主讲方] 主讲，聚焦 [主题概述]。

【培训安排】
日期：[日期]（[周几]）
时间：[时间] - [时间]（预计 [时长]）
地点：[场地名称]（[详细地址]）
形式：线下培训 + 分组圆桌

【培训内容】
模块一：[模块名]（主讲：[讲者]）
模块二：[模块名]（主讲：[讲者]）
模块三：[模块名]（主讲：[讲者]）
模块四：[模块名]（主讲：[讲者]）

【参会要求】
1. 每家供应商派 1-2 名负责人/主管参加
2. 请于 [日期] 前填写附件中的"课前预习表"并返回
3. 培训当天请携带"现场对照记录表"（已附）
4. 培训结束后 7 天内需提交"30 天行动回收表"

【附件】
- 附件 1：课前预习框架
- 附件 2：现场对照记录表
- 附件 3：培训议程与分组安排

如有疑问，请联系 [组织者姓名]（微信/电话：[联系方式]）。

[组织者姓名]
[日期]
```

## 三、培训预告（群内/朋友圈）

### 群内预告（轻量版）

```
各位供应商伙伴：

X 月供应商联盟圆桌定了：[日期] [时间]，主题是[主题名称]，由[主讲人公司][主讲人姓名]主讲。

本期亮点：[一句话概括为什么值得关注]。

会议链接：[链接]

下期预告：[日期]，由[供应商名称]主讲[主题]。
```

### 朋友圈/群发（成果展示版）

```
供应商联盟 X 期圆桌圆满结束。

本期主题：[主题]，由[主讲人公司][主讲人姓名]分享了[核心观点]。现场 X 家供应商参与，讨论热烈。

供应商反馈精选：
- "[某供应商的真实反馈]"
- "[某供应商的真实反馈]"

下期预告：[日期]，主题[主题名称]，主讲人[姓名]（[公司名称]）。

期待更多供应商加入。
```

## 四、会前须知模板（线下大额培训专用）

> **用途**：邀请函发出、参会者确认后，发给已确认参会者的详细须知。让参会者清楚"来干什么、怎么来、带什么"。
> **发送时机**：培训前 5-7 天
> **萃取依据**：年老师二期实际做过会前须知，含基本信息+议程+行程后勤三块，但未沉淀成模板。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[培训名称] · 会前须知
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[供应商名称] [姓名]：

感谢确认参加 [培训名称]。以下是参会须知，请仔细阅读。

【一、基本信息】
- 日期：[日期]（[周几]）
- 时间：[开始时间] - [结束时间]（[总时长]）
- 地点：[场地名称]，[详细地址]
- 集合点：[具体集合位置，如"XX酒店大堂"/"XX职场前台"]
- 联系人：[组织者姓名] [手机号]（全天可联系）

【二、议程安排】
[简版议程表，含每个模块的时间/主题/主讲人，不需要太详细]
例：
  09:00-09:30  开场+破冰
  09:30-10:10  模块1：[主题]（主讲：[姓名]）
  10:10-10:18  追问讨论
  ...

【三、行程后勤】
- 交通：[到达方式建议，如"宿州东站打车约15分钟"/"提供接站服务，请告知到达车次"]
- 住宿：[住宿安排，如"已预订XX酒店X月X日-X日，房型X，费用由XX承担"]
- 餐饮：[餐饮安排，如"X日欢迎晚宴X点/培训日含午餐/有饮食禁忌请提前告知"]
- 天气：[当地天气提示，如"近期降温，建议带外套"]

【四、参会准备】
- 请携带：[笔记本/笔/课前预习表（如已发）]
- 着装建议：[商务休闲/无特殊要求]
- 请提前：[阅读课前预习材料（如已发）/思考你想解决的问题]

【五、其他】
- 如需变更参会人员或时间，请尽快联系 [联系人]
- 培训结束后将发放"事后行动回收表"，请配合在 7 天内提交

期待与您相见。

[组织者姓名]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 五、行程确认函模板（线下大额培训专用）

> **用途**：针对需要安排交通/住宿的参会者，单独确认行程细节，避免现场混乱。
> **发送时机**：参会者填完信息收集表后，培训前 3 天最终确认
> **萃取依据**：年老师二期实际做过行程确认，但未沉淀成模板。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[培训名称] · 行程确认
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[供应商名称] [姓名]，您好：

根据您填写的信息，您的参会行程已安排如下，请确认：

【交通】
- 到达：[日期] [时间] [方式]（如：5月7日 15:20 高铁 Gxxxx 宿州东站）
- 接站：[是/否]（如：是，[接站人姓名] [手机号] 在出站口举牌等候）
- 返程：[日期] [时间] [方式]

【住宿】
- 酒店：[酒店名称] [地址]
- 房型：[单间/标间/大床]
- 入住：[日期]  离店：[日期]（[X]晚）
- 费用：[由XX承担/自理]
- 备注：[特殊要求]

【餐饮】
- [日期] 晚餐：[安排]（忌口：[已记录的饮食禁忌]）
- [日期] 午餐：[安排]

【请确认】
请回复"确认无误"或告知需要调整的地方。如有变动，请在 [日期] 前告知。

[组织者姓名] [手机号]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 六、模板使用说明（完整版）

| 场景 | 选哪个模板 | 填充项 | 发送时机 |
|------|---------|--------|---------|
| 线上圆桌正式邀请 | 供应商版（正式邀请） | 日期/时间/主题/主讲人/链接 | 圆桌前 2 周 |
| 请京东同事参与 | 同事版（内部通告） | 日期/时间/主题/价值点 | 圆桌前 3 周 |
| 线下大额培训邀请 | 完整邀请函 | 日期/时间/地点/模块/主讲/附件 | 培训前 10 天 |
| **线下培训详情** | **会前须知** | 基本信息/议程/行程后勤/参会准备 | **培训前 5-7 天** |
| **线下培训行程** | **行程确认函** | 交通/住宿/餐饮确认 | **培训前 3 天** |
| 群内预告 | 群内预告（轻量版） | 日期/时间/主题/主讲人/链接 | 圆桌前 3 天 |
| 对外宣传 | 成果展示版 | 主题/数据/反馈/下期预告 | 培训后 3 天 |

**线下大额培训通知三件套**：邀请函（拉人来）→ 会前须知（让人准备好）→ 行程确认（让人安心来）。三步缺一不可。
