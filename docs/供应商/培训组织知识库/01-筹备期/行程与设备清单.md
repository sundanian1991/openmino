# 筹备期 · 行程与设备清单

> **定位**：线下培训的第一道屏障——场地不对、设备坏了、没人管饭，所有内容准备归零。用这张清单每次出发前 30 分钟逐个确认。
> **原则**：先确认最贵的事（供应商来了没处坐比课件没准备好严重 10 倍）。

---

## 一、场地方接待能力确认清单

> ⚠️ **萃取升级**：原清单只确认"场地硬件"，但年老师真实经验是——**行程难点不在交通住宿，在"场地方接待能力的预确认"**。20人去毅航宿州，毅航要接待一整天，如果会议室不够、早会旁听影响他们工作、午饭没安排，现场就乱。所以这里升级为"接待能力"视角。

### 到达前 7 天确认（硬件 + 接待能力）

#### A. 场地硬件

| 事项 | 确认点 | 谁负责 | 状态 |
|------|--------|--------|------|
| 场地名称 | [场地名称]，[详细地址] | [组织者] | |
| 会议室容量 | 能坐 [X] 人（目标：[X] 人） | [供应商] | |
| 投影/屏幕 | 品牌：[X]，分辨率：[X]，能否投屏 PPT？ | [供应商] | |
| 网络 | Wi-Fi 名称：[X]，密码：[X]，速度足够共享屏幕？ | [供应商] | |
| 白板/大白纸 | 有几块？马克笔颜色：[X] | [供应商] | |
| 麦克风 | 有线/无线？几个？ | [组织者] | |
| 电源插座 | 够用吗？需要带排插？ | [供应商] | |
| 座位排列 | U 型/课桌式/圆桌？ | [供应商] | |

#### B. 接待能力（关键·萃取新增）

| 事项 | 确认点 | 为什么重要 |
|------|--------|-----------|
| **会议室是否够大** | 目标人数能不能坐下？是否需要加座/换更大场地？ | 20人去毅航，会议室坐不下就全乱 |
| **早会旁听影响** | 旁听会不会影响场地方正常工作？他们当天有别的事吗？ | 不要到了才发现人不在、活没停 |
| **餐饮承载** | 场地方能提供几人用餐？还是需外卖/外出？谁出钱？ | 午饭没安排=下午全乱 |
| **住宿接待** | 周边酒店接待能力？需要统一订房还是自理？ | 多家供应商住宿要提前锁 |
| **交通接送** | 场地方能否协助接送站？还是组织者全包？ | 宿州不是主场，交通要盯 |
| **场地方人员配合** | 当天场地方有没有人对接？谁负责开门/设备/引导？ | 别到了找不到人 |

**确认话术**（给场地方）：

> "X总，我们大概[X]人过去，想提前跟你确认几个事：会议室能坐下吗？当天你们正常办公会不会受影响？午饭你们能安排还是我们自己解决？住宿有没有合作酒店推荐？当天谁跟我们对接？"

## 四、签到与到场管理

### 线上

- 提前 5 分钟在群里发会议链接
- 开始前检查到场人数
- 未到的私信提醒一次

### 线下

- 提前准备签到表（含供应商名称、姓名、角色）
- 提前 10 分钟到场检查设备
- 早到的人安排自由交流区

## 五、紧急预案

| 情况 | 预案 |
|------|------|
| 投影坏了 | 备用方案：主讲人口述，大家看打印版数据卡片，白板补充 |
| 主讲人迟到 | 先开始其他模块，调整议程顺序 |
| 网络断了 | 手机热点备用，预下载 PPT |
| 有人临时不能来 | 群内通知替补，或调整分组 |
| 录屏/录音失败 | 安排专人做笔记，录屏是保底 |
