# 持续工具层 · 对外宣传模板与案例

> **定位**：培训做完了不等于影响力结束了——对外宣传是让培训成果持续发酵、让更多供应商主动加入的关键环节。
> **核心原则**：宣传不是"我们做得好"，是"供应商怎么做的，你可以从中学到什么"。

---

## 一、宣传阵地与时机

| 阵地 | 目标受众 | 内容 | 时机 |
|------|---------|------|------|
| 供应商微信群 | 已加入联盟的供应商 | 下期预告、本期亮点 | 培训前3天/后1天 |
| 朋友圈 | 潜在供应商 + 同事 | 成果展示 + 下期预告 | 培训后3天 |
| 京东内部群 | 同事 | 供应商反馈亮点 | 培训后1天 |
| 供应商1v1 | 观望中的供应商 | "上期讲到一个你感兴趣的点…" | 培训后3天 |

---

## 二、培训成果宣传模板

### 朋友圈/群发（成果展示版）— 通用模板

```
供应商联盟第 X 期圆桌圆满结束。

本期主题：[主题]，由[主讲人公司][主讲人姓名]分享了[核心观点]。

参与供应商：[X] 家。

供应商反馈精选：
- "[某供应商的真实反馈]"
- "[某供应商的真实反馈]"

下期预告：[日期]，[主题]，由[供应商名称]主讲。

如有供应商伙伴想加入，欢迎私信我。
```

### 朋友圈（数据亮点版）— 有数据时用

```
供应商联盟第 X 期培训成果：

[核心数据点]，[核心数据点]。

[供应商名称]的[具体做法]，在[X]个月内把[数据]从[X]提升到[X]。

下期预告：X月X日，主题[XX]，主讲人[XX]。

期待更多供应商伙伴加入。
```

## 三、宣传案例（二期毅航专场）

### 稿件一：毅航数据分析成果

```
供应商联盟第二期线下分享在毅航宿州职场圆满结束。

毅航CFO曹总、运营总监李晓燕、培训导师贺学庆等4位主讲人
全天7场分享，覆盖经营回顾、基本法激励、新人培养、客群经营、
日常管理五大模块。

核心数据：毅航复贷S率50%，远超大盘平均水平。

毅航从百人团队到千余人团队的成长路径——每步都踩过坑，每步都讲过真话。

下期预告：5月29日，供应商联盟第三期圆桌——降本增效实战。
```

### 稿件二：曹总扭亏故事

```
供应商联盟第二期线下分享会在毅航宿州职场圆满结束。

毅航CFO曹总分享：毅航从百人到三千人的成长路径——
每一个转折都踩过坑，每一个坑都讲过真话。

核心收获：
① 新人留存不是从入职开始，是从入职前就开始了——"新人第一周淘汰率"是供应商最真实的训练温度计。
② 提成结构不是越高越好——头部与尾部的合理差距才能激发竞争。

下期预告：5月29日，降本增效实战，由供应商联盟轮值分享。
```

## 四、宣传质量的检测标准

| 维度 | 标准 |
|------|------|
| 有真实数据 | 每个宣传稿至少有 1 个具体数字或可量化的成果 |
| 有真实反馈 | 引用的反馈必须是供应商的真实反馈，不是自己编的 |
| 有下期预告 | 每篇宣传结尾都有下期预告——不是为了自我吹嘘，是为了吸引更多人参与 |
| 讲供应商，不讲自己 | 主体是供应商的成功故事，不是"我们办的培训" |

## 五、宣传日历

| 时机 | 动作 | 内容 |
|------|------|------|
| 培训后 1 天 | 群内发"成果简报" | 本期亮点 + 供应商反馈精选 |
| 培训后 3 天 | 朋友圈发"数据亮点版" | 核心数据 + 下期预告 |
| 培训后 5 天 | 1v1 私信发"本期反馈" | 给观望中的供应商私信："上期讲到一个你感兴趣的……" |
| 培训后 7 天 | 内部群发"成果汇报" | 给领导的数据模板 |

## 六、战略张力诊断（萃取新增·待决策）

> ⚠️ **萃取新增·关键**：这里点破一个矛盾，需要年老师决策，工具无法替代。

### 矛盾点

| 目标 | 现状 | 矛盾 |
|------|------|------|
| **建品牌形象**（年老师认定的宣传核心价值） | 渠道全是**圈内**：供应商群、朋友圈、京东内部群、1v1 | 想破圈建品牌，但所有渠道都是私域/熟人渠道，**触达不到圈外潜在供应商** |

### 两个方向（需年老师决策）

**方向 A：保持圈内口碑（轻运营）**

- 维持现有四个阵地，专注"让圈内人觉得专业、让参与供应商有面子"
- 不追求外部品牌，靠口碑自然扩圈（供应商推荐供应商）
- 优点：投入低，现有模板够用
- 缺点：增长靠口碑，速度慢；无法形成行业影响力

**方向 B：建外部品牌（重投入）**

- 新增对外公开渠道：公众号/行业内容平台/案例库对外页面
- 把培训成果做成可对外传播的内容（脱敏后的方法论文章）
- 优点：能触达圈外，建立行业品牌，助力供应商招募
- 缺点：需投入内容生产能力；脱敏/合规需把关；周期长

### 决策建议

> 这个决策不急，但影响宣传体系怎么建。建议：先用方向 A 跑通 2-3 期（验证口碑扩圈效果），同时观察有没有供应商主动问"你们这个联盟能不能对外讲"，再决定是否升级到方向 B。
