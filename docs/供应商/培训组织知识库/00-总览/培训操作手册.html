培训组织全生命周期

筹备期
探需求（1v1微信）→ 定人定时定题（锁定四周）→ 建群发公告（物料6/7）→ 主讲人备提纲（物料1+审提纲）
执行期
线上圆桌：开场3min → 主讲30min → 讨论25min → 收尾3min
线下大额：开场 → 各模块按议程推进 → 分组圆桌（按层级分）→ 京东收口
收尾期
录屏+纪要（3天内）→ 反馈评分（物料8）→ 确认下期 → 归档入库 → 30天回访

线上圆桌（A 模式）

频次每月 1 次
时长60 分钟
主讲供应商轮值
受众10-20 家供应商
运营投入月均 ≤ 2 小时
闭环纪要 + 反馈评分

线下大额培训（B 模式）

频次按需
时长半天到全天
主讲头部供应商 + 京东同事
受众20+ 人（分组混编）
运营投入单次较高，频次低
闭环预习→记录→回收→30天回访
