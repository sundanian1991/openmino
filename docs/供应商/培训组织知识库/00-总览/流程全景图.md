# 供应商培训组织 · 流程全景图

> **定位**：一张图看懂供应商培训组织的全生命周期，从筹备到收尾到持续运营。
> **架构（萃取确认）**：**线上圆桌为主引擎（轻量高频/月度），线下大额培训为辅（季度深度/仪式感）**。两者不是并列，是主辅关系。

---

## 体系架构：线上主 · 线下辅

```
┌─────────────────────────────────────────────────────────┐
│                    内容资产流向                            │
│                                                         │
│  线上圆桌积累  →  选出亮点供应商  →  线下深度拆解          │
│  (月度/轻量)      (讲师发现)         (季度/仪式感)         │
│       ↑                                    │            │
│       └──────── 内容资产化 ← 反哺线上 ←─────┘            │
│                  (案例入库)                              │
└─────────────────────────────────────────────────────────┘
```

**为什么是线上主线下辅**：
- 线上圆桌是**可复制、低成本、高频次**的运营引擎，是联盟日常运转的主体
- 线下大额培训是**高投入、深度落地**的升级形式，用于挖掘出的亮点供应商深度拆解
- 线下产出的内容资产化后，反哺线上案例库，形成正向循环

## 流程总览

### 模式 A：线上圆桌（60 分钟轻量型）

适合月度供应商实战分享，轻运营、快闭环。

```
         ┌─────────────────────────────────────────────────────┐
         │                 筹备期                     收尾期    │
         └─────────────────────────────────────────────────────┘

  探需求     定人定时定题    建群发公告    主讲人备提纲
 (1v1微信)    (锁定4周)    (物料6/7)   (物料1+审提纲)

     │           │            │            │
     ▼           ▼            ▼            ▼
  ┌──────────────────────────────────────────────────────────┐
  │                    执行期 — 圆桌当天                      │
  │  ┌──────────┐   ┌──────────┐   ┌──────────┐           │
  │  │ 开场 3min │ → │ 主讲 30min│ → │ 讨论 25min│ → 收尾 3min│
  │  │ (主持人) │   │ (主讲人) │   │ (全员)   │   │ (主持人) │
  │  └──────────┘   └──────────┘   └──────────┘           │
  └──────────────────────────────────────────────────────────┘
     │            │            │            │
     ▼            ▼            ▼            ▼
  录屏+纪要    反馈评分     确认下期      归档入库
 (3天内)      (物料8)     (排下一期)    (内容库)
```

**运营投入**：运营官月均不超过 2 小时。

### 模式 B：线下大额培训（深度落地型）

适合深度培训专场，含课前预习、事中记录、事后行动回收。

```
          ┌────────── 筹备期 ──────────┐
          │                            │
    场地/行程确认                    课前预习表发放
   (场地/设备/住宿)               (4模块靶向问题引导)

    课程设置编排                    主讲备课协作
   (主题×时间×讲者)               (5页模板填空)
          │                            │
          └────────────┬───────────────┘
                      ▼
          ┌────────── 执行期 ──────────┐
          │                            │
    现场培训 + 分组圆桌               事中对照记录表
   (各模块按议程推进)             (边听边记:要点/差距/可抄动作)

    主持人控场 + QA收集            现场Q&A管理
   (物材追问清单)               (收集→回答→归档)
          │                            │
          └────────────┬───────────────┘
                      ▼
          ┌────────── 收尾期 ──────────┐
          │                            │
    事后行动回收表                    30天行动计划
   (选1-2个可抄动作)             (动作/人/场景/标准/阻碍)

    效果追踪                        培训总结报告
   (30天后跟进回访)                (成果/亮点/不足/优化)
```

## 两模式差异对比（主辅已明确）

| 维度 | 线上圆桌（A）·**主引擎** | 线下大额培训（B）·**辅助升级** |
|------|-------------|----------------|
| **定位** | **月度常规运营主体** | **季度深度仪式感** |
| 频次 | 每月1次 | 按需（季度级） |
| 时长 | 60分钟 | 半天到两天 |
| 主讲 | 供应商轮值 | 头部供应商+京东同事 |
| 受众 | 10-20家供应商 | 20+人（分组混编） |
| 闭环 | 纪要+反馈评分+T+30回访 | 预习→记录→回收→30天跟进 |
| 核心目标 | 广度覆盖、轻运营、可复制 | 深度落地、强制行动回收 |
| 运营投入 | 月均2小时 | 单次投入较高，但频次低 |
| **关系** | **内容来源 + 讲师试炼场** | **亮点深度拆解 + 内容资产化** |

## 核心原则

1. **供应商讲为主，京东讲为辅**：60%以上内容由供应商分享实战案例
2. **事前有预习，事中有对照，事后有回收**：三份表单形成强制闭环
3. **讲已做到的，不讲想做的**：确保内容真实可参考
4. **培训不是终点，30天后才是验收点**：行动回收+效果追踪是效果管理的核心
