## 使用示例

**快速查找**：
```
找"海腾" → 查别名映射表 → 陕西海腾
找"锦瑞" → 查别名映射表 → 河北锦瑞
找"伽玛" → 查按简称表 → 伽玛.md
```

**按业务找供应商**：
```
需要企金供应商 → 查看"企金业务"分类
需要金条人工 → 查看"金条人工"分类
```

**按规模找供应商**：
```
找大型供应商 → 查看"大型"分类
找待启动供应商 → 查看"待启动"分类
```
