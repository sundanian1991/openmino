## 第一章 总则

### 1.1 目的

建立统一、透明、可量化的供应商月度评估体系，作为供应商分级管理、资源分配、续约决策的核心依据。

### 1.2 适用对象

所有承接京东2B电销业务的BPO供应商，按站点单独评估。

> **站点结构说明**：每家供应商公司最多设立 4 个站点。站点为最小考核单元，独立出勤、独立产能、独立评估排名。一个站点可承接多条产品线（如金条+信用卡），日报按产品线分列，但评估按站点总量考核。同一供应商下的不同站点独立排名，互不影响。

### 1.3 基本原则

- **统一标准**：所有站点适用同一套评分规则
- **量化为主**：以客观数据为评分依据，减少主观判断
- **透明公开**：评分规则提前公布，评估结果排名公示
- **结果应用**：评估结果直接关联续约、份额调整、预警清退

### 1.4 评估周期

- 每月评估一次
- 评估周期为自然月（1日-月末）
- 评估结果于次月第1周公布

## 第二章 评估模型

### 2.1 评分框架

总分 **100分**，四大维度：

| 维度 | 权重 | 分值 | 数据来源 |
|------|------|------|----------|
| 交付质量 | 40% | 40分 | 质检团队月度质检数据 |
| 业务达成 | 30% | 30分 | 站点每日报送+业务系统 |
| 合规表现 | 20% | 20分 | 供应商管理台账 |
| 管理配合 | 10% | 10分 | 供应商管理记录 |

### 2.2 设计逻辑

| 维度 | 回答的问题 |
|------|-----------|
| 交付质量 | "做得好不好"——客户体验的直接反映 |
| 业务达成 | "量完没完成"——业务目标的达成情况 |
| 合规表现 | "有没有踩红线"——制度执行情况 |
| 管理配合 | "配不配合"——管理成本高低 |

## 第三章 评估指标细则

### 3.1 交付质量（40分）

**数据来源**：质检团队每月出具的质检月报。

**说明**：供应商管理岗不参与质检执行，仅引用质检数据并按以下规则计算得分。

#### 3.1.1 质检评分换算

质检月度平均分（百分制）→ 折算为交付质量得分（满分40分）：

| 质检月均分 | 交付质量得分 | 说明 |
|-----------|-------------|------|
| ≥95分 | 40分 | 优秀 |
| 90-94分 | 36分 | 良好 |
| 85-89分 | 32分 | 达标 |
| 80-84分 | 28分 | 关注 |
| 75-79分 | 24分 | 预警 |
| 70-74分 | 20分 | 严重预警 |
| <70分 | 0分 | 红线 |

**计分方式**：
- 质检月均分≥95：40分
- 90≤质检月均分<95：36分
- 85≤质检月均分<90：32分
- 80≤质检月均分<85：28分
- 75≤质检月均分<80：24分
- 70≤质检月均分<75：20分
- 质检月均分<70：0分（触发红线评估）

#### 3.1.2 特殊情况

| 情况 | 处理方式 |
|------|----------|
| 质检未提供月度数据 | 该项记0分，同时书面催办质检 |
| 质检数据明显异常（偏离均值>2个标准差） | 启动复核程序 |
| 站点质检覆盖不足（抽检量未达要求） | 按实际覆盖率折算，覆盖率<80%的按80%折算 |

---

### 3.2 业务达成（30分）

**数据来源**：站点每日报送数据+业务系统数据。

#### 3.2.1 日均出勤率（10分）

| 日均出勤率 | 得分 |
|-----------|------|
| ≥95% | 10分 |
| 90%-94% | 8分 |
| 85%-89% | 6分 |
| 80%-84% | 4分 |
| <80% | 0分 |

#### 3.2.2 产能达标率（10分）

| 产能达标率 | 得分 |
|-----------|------|
| ≥90% | 10分 |
| 80%-89% | 8分 |
| 70%-79% | 6分 |
| 60%-69% | 4分 |
| <60% | 0分 |

#### 3.2.3 数据报送及时率（5分）

定义：按时报送次数 ÷ 总应报送次数 × 100%

| 报送及时率 | 得分 |
|-----------|------|
| 100% | 5分 |
| ≥95% | 4分 |
| 90%-94% | 3分 |
| <90% | 0分 |

#### 3.2.4 数据准确率（5分）

定义：经核实的准确数据条数 ÷ 总报送数据条数 × 100%

| 数据准确率 | 得分 |
|-----------|------|
| 100% | 5分 |
| ≥98% | 4分 |
| 95%-97% | 3分 |
| <95% | 0分 |

---

### 3.3 合规表现（20分）

**数据来源**：供应商管理台账。

#### 3.3.1 信息安全（8分）

| 情形 | 得分 |
|------|------|
| 当月0违规 | 8分 |
| 1次轻微违规（已整改） | 4分 |
| 1次及以上严重违规 | 0分（触发红线评估） |

#### 3.3.2 人员持证率（6分）

| 持证率 | 得分 |
|--------|------|
| 100% | 6分 |
| ≥95% | 4分 |
| 90%-94% | 2分 |
| <90% | 0分（触发红线评估） |

#### 3.3.3 人员变动报备合规率（3分）

定义：按时报备的人员变动次数 ÷ 总人员变动次数 × 100%

| 报备合规率 | 得分 |
|-----------|------|
| 100% | 3分 |
| ≥90% | 2分 |
| <90% | 0分 |

#### 3.3.4 录音留存率（3分）

| 留存率 | 得分 |
|--------|------|
| 100% | 3分 |
| ≥98% | 2分 |
| <98% | 0分 |

---

### 3.4 管理配合（10分）

**数据来源**：供应商管理岗日常记录。

#### 3.4.1 异常事项报备及时性（4分）

| 情形 | 得分 |
|------|------|
| 全部按时报备 | 4分 |
| 漏报1次（含超时） | 2分 |
| 漏报≥2次 | 0分 |

#### 3.4.2 整改响应（3分）

| 情形 | 得分 |
|------|------|
| 全部在规定时间内响应 | 3分 |
| 超时1次 | 2分 |
| 超时≥2次 | 0分 |

#### 3.4.3 会议参与（3分）

| 情形 | 得分 |
|------|------|
| 全勤 | 3分 |
| 缺席1次 | 2分 |
| 缺席≥2次 | 0分 |

## 第四章 评估流程

### 4.1 数据收集（次月1-3日）

| 步骤 | 责任人 | 产出 |
|------|--------|------|
| 质检提供月度质检数据 | 质检团队 | 各站点质检月报 |
| 站点提交月度总结 | 站点负责人 | 月度总结报告 |
| 供应商管理整理合规台账 | 供应商管理岗 | 合规记录汇总 |
| 供应商管理整理管理配合记录 | 供应商管理岗 | 配合度记录 |

### 4.2 评分（次月3-5日）

供应商管理岗根据上述数据，按第三章评分规则计算各维度得分，填写《月度评估评分表》。

### 4.3 审核（次月5日）

- 供应商管理岗完成评分后，提交业务对接人审核确认
- 如有异议，按申诉机制处理

### 4.4 公布（次月第1周内）

- 各站点得分及排名在群内公布
- 排名方式：按总分从高到低排列
- 同分站点按"交付质量→业务达成→合规表现"顺序排列

## 第五章 评级与结果应用

### 5.1 评级标准

| 评级 | 总分 | 含义 |
|------|------|------|
| **A（优秀）** | ≥90分 | 持续超出预期 |
| **B（良好）** | 80-89分 | 稳定达标 |
| **C（达标）** | 70-79分 | 基本达标，有改进空间 |
| **D（预警）** | 60-69分 | 未达标，需整改 |
| **E（红线）** | <60分 | 严重不达标，启动清退评估 |

**一票否决规则**：即使总分≥60分，如触发以下任一红线，直接降为E级：
- 信息安全发生严重违规
- 人员持证率<90%
- 质检月均分<70分
- 数据造假

> **SLA与评估方案的关系说明**：SLA定义供应商服务质量底线（"能不能接受"），评估方案定义量化评分（"得多少分"）。两套体系独立运作，详见《供应商管理SLA》第五章。

### 5.2 结果应用

#### 5.2.1 月度排名

- 排名结果群内公开
- 排名变化趋势（较上月升降）一并公示

#### 5.2.2 月度沟通

| 评级 | 沟通方式 |
|------|----------|
| A级 | 群内公示表扬，一对一沟通感谢 |
| B级 | 群内公示，无额外沟通 |
| C级 | 约谈站点负责人，了解原因，提出改进建议 |
| D级 | 约谈站点负责人，要求提交整改计划，整改周期30天 |
| E级 | 启动清退评估流程（参照《供应商管理办法》） |

#### 5.2.3 长期应用

| 情形 | 后果 |
|------|------|
| 连续3个月A级 | 续约优先权 |
| 连续2个月D级 | 升级为E级处理 |
| 连续3个月E级 | 启动清退流程 |
| 全年平均分排名末位 | 合约到期不续约 |

## 第六章 申诉机制

### 6.1 申诉条件

站点对评估得分有异议的，可在评估结果公布后 **3个工作日内** 发起申诉。

### 6.2 申诉流程

```
站点提出书面申诉（说明异议项+理由）
  │
  │ （1个工作日内）
  ↓
供应商管理岗受理
  │
  │ （2个工作日内）
  ↓
指定独立复核人复核
  │
  │ （3个工作日内）
  ↓
复核结果为最终结果
```

### 6.3 复核人规则

- 复核人由供应商管理岗指定，须与被评估站点无直接利益关联
- 复核内容限于申诉所列异议项
- 复核结果为最终结果，不再二次申诉

> **质检标准争议**：如对质检评分标准本身有异议（非评分结果），按《质量管理规范》第六章三方校准流程处理（质检+供应商+供应商管理）。

### 6.4 申诉率监控

- 如某站点连续3个月申诉率（申诉指标数÷总评估指标数）>50%，供应商管理岗介入了解原因
- 可能原因：评估标准不清晰、数据源有误、站点对规则理解有偏差
- 由供应商管理岗组织校准
