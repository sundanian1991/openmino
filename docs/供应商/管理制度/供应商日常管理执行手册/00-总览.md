# 供应商日常管理执行手册 — 总览

> 以年老师个人使用为主的操作指引
>
> 核心目标：打开就知道今天该做什么、怎么做、用什么工具、谁需要参与
>
> 版本：V1.0 | 2026-04-23

---

## 定位与使用说明

**不是什么**：不是重写制度文件（管理办法、运营要求V5.0、全生命周期指引V2.1 保持不动）。

**是什么**：把已有积累变成"拿来就用"的操作手册。每个模块统一结构：**触发条件 → 操作步骤 → 工具/模板 → 协作角色 → 输出物**。

**怎么用**：

| 场景 | 做法 |
|------|------|
| 每天打开 | 看"今日扫描"检查清单，确认有无异常信号 |
| 周度例行 | 看 01-监控扫描，按3信号扫描 |
| 月度排名后 | 看 02-预警约谈，按排名结果触发对应流程 |
| 遇到具体问题 | 按模块索引直接跳转到对应文件 |

---

## 整体架构

```
触发信号（数据/排名/异常）
    │
    ├─ 周度扫描 ──→ 01-监控扫描
    │                    │
    │              发现异常？
    │              是 ↓        否
    │         02-预警约谈     记录正常
    │              │
    │         03-整改闭环（如需）
    │              │
    │         04-供应商沟通（贯穿全程）
    │
    ├─ 月度周期 ──→ 05-分量决策 → 06-巡检管理 → 07-向上汇报
    │
    └─ 事件触发 ──→ 08-新供应商跟进 / 09-数据真实性防控 / 10-HRBP式扶持
```
