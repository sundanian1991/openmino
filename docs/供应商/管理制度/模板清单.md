# C2电销BPO供应商管理配套模板清单

> 共13个模板（SM-TPL-001 至 SM-TPL-013）

| 编号 | 模板名称 | 用途说明 |
|------|----------|----------|
| SM-TPL-001 | 每日出勤报送表 | 供应商每日报送坐席出勤数据（应到/实到/请假/旷工等） |
| SM-TPL-002 | 每日业绩报送表 | 供应商每日报送各业务产线业绩数据（外呼量/接通量/转化量等） |
| SM-TPL-003 | 周报模板 | 供应商每周提交工作概述、业绩汇总、人员变动、合规执行等 |
| SM-TPL-004 | 月度总结模板 | 供应商每月提交业绩总结、人员管理、合规执行、自评与计划 |
| SM-TPL-005 | 花名册模板 | 供应商维护全体人员信息（编号、姓名、身份证、岗位、资质等） |
| SM-TPL-006 | 人员变动报备表 | 供应商实时报送人员入职、离职、调岗、证书到期等变动 |
| SM-TPL-007 | 异常处理记录 | 记录一般/重要/紧急异常的发现、处理、整改全过程 |
| SM-TPL-008 | 月度评估评分表 | 供管岗用于月度综合评估打分（交付质量/业务达成/合规/管理配合） |
| SM-TPL-009 | 排名公告模板 | 月度评估结果公示，包含各供应商得分、评级、排名 |
| SM-TPL-010 | 申诉表 | 供应商对评估结果有异议时提交申诉 |
| SM-TPL-011 | 整改计划模板 | D/E级供应商提交整改方案（问题分析、措施、时间表） |
| SM-TPL-012 | 管理办法确认书 | 供应商签署确认已知悉并遵守《供应商管理办法》 |
| SM-TPL-013 | SLA确认书 | 供应商签署确认已知悉并遵守《供应商管理SLA》 |
