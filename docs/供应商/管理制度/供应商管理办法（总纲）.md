# C2电销BPO供应商管理办法（总纲）

**文件编号**：SM-2026-001
**版本**：V1.0
**生效日期**：2026年XX月XX日
**发文单位**：C2电销供管岗

---

## 第一章 总则

### 第一条 目的与依据

为加强电销BPO供应商管理，规范供应商全生命周期管理流程，保障业务稳定运行，依据《电销业务合规管理办法》《供应商合作管理制度》等相关规定，特制定本办法。

### 第二条 适用范围

本办法适用于金融产品电销外呼业务涉及的全体供应商（含人工外呼供应商和人机协同供应商），涵盖金条、信用卡、财富等全部业务产线。

### 第三条 管理原则

| 原则 | 含义 |
|------|------|
| **合规底线** | 信息安全、持证上岗、数据真实性为不可触碰的红线 |
| **动态管理** | 建立准入→运营→评估→清退的全周期管理机制 |
| **量化评估** | 所有评价基于客观数据，评分有据可查 |
| **分级管控** | 根据评估等级实施差异化管控措施 |

---

## 第二章 供应商全生命周期管理

### 第四条 准入阶段

**准入条件**（须同时满足）：

1. 具备有效营业执照及相关业务经营资质
2. 坐席规模不低于50人
3. 人员持证率（金融相关资质）达到100%
4. 具备独立质检团队或第三方质检协议
5. 数据安全防护能力通过我方评估
6. 无重大商业违规或法律诉讼记录

**准入流程**：

```
供应商申请 → 资质审核 → 现场考察 → 试运行评审 → 转正评估 → 正式签约
```

| 阶段 | 时限 | 评审主体 |
|------|------|----------|
| 资质审核 | 10个工作日 | 供管岗 |
| 现场考察 | 5个工作日 | 供管岗 + 业务方 |
| 试运行评审 | 30天 | 供管岗 + 质检 + 业务方 |
| 转正评估 | 5个工作日 | 供管岗 |

### 第五条 运营阶段

**日常管理**：

1. 信息报送：供应商须按要求报送每日出勤、业绩数据、花名册变动等信息
2. 合规监控：持续监控信息安全、人员资质、话术规范等合规指标
3. 异常处理：建立三级异常响应机制（一般/重要/紧急）
4. 定期评估：每月进行一次综合评估，评估结果与分级挂钩

**运营准入门槛维持条件**：

| 指标 | 达标线 |
|------|--------|
| 月度综合评估等级 | 不低于C级 |
| 数据报送及时率 | ≥98% |
| 数据准确率 | ≥99% |
| 人员持证率 | 100% |

### 第六条 评估阶段

**评估周期**：每月一次综合评估，每季度一次深度评估

**评估维度**：交付质量（40分）、业务达成（30分）、合规表现（20分）、管理配合（10分），满分100分

**评级等级**：A/B/C/D/E五级，详见《供应商月度综合评估方案》

### 第七条 清退阶段

**清退条件**：

| 清退类型 | 触发条件 |
|----------|----------|
| 即时清退 | 触碰合规红线（一票否决项） |
| 预警清退 | 连续2个月评估等级为E级 |
| 整改清退 | 整改期满（最长60天）未达到整改目标 |

**清退流程**：

```
清退决策 → 提前30天通知 → 业务交接 → 数据归档 → 关系终止
```

**清退后管理**：

1. 供应商须配合完成业务交接，确保交付连续性
2. 所有业务数据、客户信息须在清退后15个工作日内完成移交
3. 清退后1年内不得重新申请准入

## 第三章 权责划分

### 第八条 供管岗职责

| 职责 | 说明 |

| 准入管理 | 供应商准入审核、试运行跟踪、转正评估 |

| 评估考核 | 月度评估实施、评级结果应用 |

|------|------|

| 数据提供 | 提供业务相关数据，配合评估 |
| 问题反馈 | 发现供应商问题及时通报业管 |

### 第十条 质检团队职责

| 职责 | 说明 |
|------|------|

| 数据产出 | 提供质检报告、数据分析 |

|------|------|

| 应急响应 | 现场异常情况第一时间响应处置 |

## 第四章 合规红线

### 第十二条 信息安全红线

**严禁行为**：

1. 泄露客户个人信息（含姓名、电话、地址、资产信息等）
2. 未经授权的数据下载、复制、导出
3. 将客户数据存储于非授权系统或外部设备
4. 与竞争对手共享客户数据
5. 数据传输未加密

**触发后果**：立即启动清退流程，情节严重者追究法律责任

### 第十三条 持证上岗红线

**要求**：全体坐席人员须在入职30日内取得金融相关资质证书

**严禁行为**：

1. 无证人员上岗作业
2. 证书过期未续期继续上岗
3. 持假证或借用他人证书
4. 证书信息与实际作业人员不符

**触发后果**：

| 违规情形 | 后果 |
|----------|------|
| 无证人员上岗 | 每发现1人扣5分，暂停新产能分配 |
| 证书过期 | 限5个工作日内完成续期，逾期视为无证 |
| 假证/借证 | 直接清退 |

### 第十四条 数据造假红线

**严禁行为**：

1. 篡改、伪造业绩数据
2. 篡改、伪造出勤记录
3. 伪造质检合格记录
4. 篡改、删除通话录音
5. 伪造培训记录

**触发后果**：直接清退，扣除全部履约保证金，3年内不得重新准入

### 第十五条 录音篡改红线

**严禁行为**：

1. 对通话录音进行任何形式的剪辑、拼接
2. 删除、销毁通话录音
3. 录音留存时长不符合规定（须留存≥6个月）
4. 录音存储于非授权系统

**触发后果**：直接清退，情节严重者追究法律责任

## 第五章 预警与清退机制

### 第十六条 预警机制

**预警分级**：

| 预警等级 | 触发条件 | 响应时限 | 处置措施 |
|----------|----------|----------|----------|
| 黄色预警 | 单项指标低于达标线 | 3个工作日 | 书面提醒，要求提交改进计划 |
| 橙色预警 | 单项指标低于预警线，或连续2周黄色预警 | 24小时 | 约谈供应商负责人，限制新产能 |
| 红色预警 | 单项指标低于红线，或橙色预警未改善 | 即时 | 暂停新产能，启动清退评估 |

### 第十七条 整改机制

**整改触发**：月度评估等级为D级或E级

**整改流程**：

```
整改通知 → 整改计划提交（5个工作日）→ 整改计划审核 → 整改执行（30-60天）→ 整改验收
```

**整改期间管控**：

| 整改等级 | 产能管控 | 新业务 | 评估频率 |
|----------|----------|--------|----------|
| D级整改 | 冻结新增 | 暂停新业务分配 | 每半月评估 |
| E级整改 | 逐步缩减 | 暂停所有新业务 | 每周评估 |

### 第十八条 清退决策机制

**即时清退决策**：触碰红线后48小时内形成清退决议

**预警清退决策**：连续2个月E级后5个工作日内形成清退决议

**清退决议须包含**：

1. 清退理由（量化依据）
2. 清退时间节点
3. 业务交接方案
4. 责任追究说明

## 第六章 集中度管控

### 第十九条 集中度管控原则

为防范业务过度集中风险，实施以下集中度管控：

| 管控维度 | 上限 |
|----------|------|
| 单供应商坐席规模占比 | ≤15% |
| 单供应商产能占比 | ≤20% |
| 前三大供应商合计占比 | ≤50% |
| 单业务线单一供应商依赖度 | ≤60% |

### 第二十条 集中度预警

| 预警条件 | 处置措施 |
|----------|----------|
| 单供应商占比接近上限（80%） | 暂停新产能分配，启动扩项评审 |
| 已达上限 | 不再增加产能，推动业务分流 |

### 第二十一条 扩项审批

供应商扩项（增加业务产线或增加坐席规模）须满足：

1. 当前评估等级不低于B级
2. 无橙色及以上预警
3. 扩项后仍符合集中度管控要求
4. 通过扩项专项评审
