## 质量标准：教科书级交付物金字塔

### 第一层：3 秒法则

**要求**：核心信息 3 秒内可理解

**验证方法**：
- 标题即结论（"FCI速记卡"而非"FCI相关卡片"）
- 焦点元素一眼可见（中心位置、暖棕#c98a6a强调）
- 次要信息自动退后（caption 9px 斜体 #6b6b6b）

### 第二层：3 步决策链

**结构**：
```
第1步（1秒）：这是什么？
  → 标题 + 核心视觉元素

第2步（2秒）：说明了什么？
  → 逻辑主链（3 个递进节点）

第3步（3秒）：对我意味着什么？
  → 行动提示 + 使用场景
```

### 第三层：14 点质量清单

| 类别 | 检查项 | 标准 |
|------|--------|------|
| **字体** | 标题 | Georgia 18px bold |
| | 正文 | system-ui 10.5px 500 |
| | 数值 | monospace 11px 600 #c98a6a |
| | 说明 | 9px 400 italic #6b6b6b |
| **容器** | 外框 | rx=10-12 stroke=1.5px #1a1a1a |
| | 卡片 | rx=8 stroke=1px #d4d1c7 fill=#f5f4f1 |
| | 背景 | #faf9f7 |
| **色彩** | 主色 | 黑白灰三色 |
| | 强调色 | #c98a6a（唯一彩色，≤10%面积） |
| | 语义色 | 橄榄绿/焦橙，最多2个 |
| **布局** | 画布宽 | 680px 固定 |
| | 边距 | 左右 40px |
| | 间距 | 4pt 基线网格（4/8/12/16...） |
| **逻辑** | 信息流 | 符合阅读动线（Z字/F型） |
| | 层级 | H1→H2→H3→Body 清晰 |

## 设计系统：infographic-svg 规范

### 色彩系统

```
主色调（黑白灰）：
  Canvas背景  #faf9f7  (米白)
  容器描边    #1a1a1a  (深灰 1.5px)
  卡片描边    #d4d1c7  (暖灰 1px)
  卡片填充    #f5f4f1  (暖灰)
  文字主色    #1a1a1a
  文字次色    #6b6b6b

强调色（唯一彩色）：
  暖棕强调    #c98a6a  (仅用于数值/重点)

语义色（每个图最多2个）：
  成功/正向   #8f9b7f  (橄榄绿)
  警告/注意   #d4845a  (焦橙)
  用户输入    #6a8fb5  (灰蓝)
```

### 字体系统

```
H1 标题：Georgia 18px bold #1a1a1a
H2 副标题：system-ui 12px 500 #1a1a1a
H3 标签：system-ui 10.5px 500 #1a1a1a
Body 正文：system-ui 10px 400 #1a1a1a
Value 数值：monospace 11px 600 #c98a6a
Caption 说明：system-ui 9px 400 italic #6b6b6b
```

### 容器系统

```
大容器（外框）：rx=10-12 stroke=1.5px
卡片（内框）：rx=8 stroke=1px
间距基线：4pt (4/8/12/16/20/24/28/32px)
```

### 画布规格

```
宽度：680px 固定
高度：根据内容 auto
边距：左右 40px / 上下 25px
内容区：600px 宽
```
