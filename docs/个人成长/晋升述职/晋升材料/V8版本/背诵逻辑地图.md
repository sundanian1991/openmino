# V8 版 · 背诵逻辑地图

---

## 整体结构（三段式）

```
开场（我是谁）
    ↓
主体（我做了什么）
    - 三件事（质量问题、标准问题、参照问题）
    - 一件补位（线路危机）
    ↓
收尾（未来做什么 + 总结）
```

**记忆口诀**：开场用钩子，三件事分层，补位显担当，收尾金句收

---

## 逐段背诵逻辑

### 第一段：开场（4句话）

**结构**：打招呼 → 引子 → 自我介绍 → 核心工作 → 钩子问题

```
各位领导好，我是XX（第一句：礼貌）
今天的述职分三个部分（第二句：预告）
简单自我介绍下（第三句：过渡）
- 2020年加入，第二次晋升
- 管理33家供应商、3239人、4条线
- 核心工作五个字：选、用、育、留、汰

去年5月我转到电销服务组时（钩子）
面对的情况是四个团队带着不同的管理习惯合并在一起
这一年我从三个层面来解决这个问题
```

**记忆锚点**：
- "三个部分"对应后面"三件事"
- "五个字"是贯穿全文的核心
- "四个团队合并"是所有问题的源头

---

### 第二段：第一件事（供应商引入）

**结构**：问题 → 做法 → 结果 → 价值

```
第一个层面的问题是质量问题
- 供应商参差不齐
- 缺乏激活机制（舒适区）

我的做法是
- 引入环节花心思
- 统一标准甄选
- 支持快速达到TOP水平

结果是8家里6家进前三

价值：不是招进来
而是让存量看到——有人能做的更好
```

**记忆锚点**：
- "第一个层面"对应"质量问题"
- 数字"8家里6家"必背
- 金句"有人能做的更好"

---

### 第三段：第二件事（制度统一）⭐ 重点

**结构**：精力投入 → 三个问题 → 做法 → 结果

```
第二个层面花了主要精力——制度和标准统一

三个问题（必背顺序）：
1. 制度版本老旧（三年前）
2. 标准不统一（各用各的）
3. 流程效率低（线下邮件）

做法是
- 编写《供应商全生命周期管理指引》
- 选、用、育、留、汰（五个字再次出现）
- 流程线上化

结果是
- 更准更快
- 四个团队用同一套语言
- 上集团制度中心
```

**记忆锚点**：
- "花了主要精力"提醒这是重点段落
- 三个问题按时间线：旧版本 → 标准乱 → 效率低
- "五个字"呼应开场，形成闭环

---

### 第四段：第三件事（自营团队）

**结构**：问题 → 公司决策 → 过程 → 结果 → 价值

```
第三个层面的问题是交付标准缺乏参照
- 不知道天花板在哪
- 供应商觉得"已经尽力了"

公司决定：在宿迁组建自营团队（注意：不是"我的做法"）

过程：3个月、20人、多种招聘路径
- 社招太慢果断放弃
- 周边公司退项
- 物流招聘团队
- 科技贷后渠道

结果：3个月达成120%

价值：不是多了20人
而是证明还有提升空间
反向传导给2000人
同时作为试验田
```

**记忆锚点**：
- 注意这里是"公司决定"，不是个人决策
- 数字"120%"和"2000人"必背
- 四个招聘路径快速带过即可

---

### 第五段：第四件事（线路危机）

**结构**：背景 → 主动承担 → 做法 → 结果 → 升华

```
还有一个补位的事情（过渡词）

背景：
- 去年7月运营商压降客诉
- 线路并发降到30%
- 预估每周损失37亿GMV

主动承担：
- 不是我职责范围
- 但云业务4年积累的资源
- 对接C2，拉通三大运营商

结果：3周恢复，行业最早

升华：
表面是解决危机
深层是资源管控转化为竞争优势
```

**记忆锚点**：
- "补位"是关键词，体现担当
- "37亿"和"3周"必背
- "不是我职责"→"主动承担"→凸显品质

---

### 第六段：未来展望

**结构**：钩子 → 解法 → 目标

```
三件事做完，体系运转起来了（过渡）

但我发现一个潜在的矛盾（钩子）
- 持续引入和汰换
- 会不会让供应商池越来越小？
- 腰部活不下去怎么办？

所以下一步是BPO联盟（解法）
- 从"筛选合格者"转向"培养合格伙伴"
- 输出京东运营体系

目标：2026年Q2，10家核心供应商

个人层面：AI Agents + 传帮带
```

**记忆锚点**：
- "潜在的矛盾"制造转折
- "筛选→培养"是对比金句
- 时间和数字"2026年Q2，10家"

---

### 第七段：收尾（金句）

**结构**：三件事总结 → 阶段定位 → 下一步

```
回顾这一年，说到底是在做三件事：（放慢）

找对人——把好准入
立好规矩——统一制度
让能力长在组织上——沉淀流程

但这只是第一阶段（转折）

接下来会在BPO联盟这个新方向上继续探索
把基础打扎实

谢谢各位领导
```

**记忆锚点**：
- 三件事金句练到肌肉记忆
- "只是第一阶段"体现格局
- 最后回到"BPO联盟"呼应未来

## 背诵技巧

### 1. 分段背诵法

不要一次性背完整篇，按这个顺序：

```
第1天：开场（4句话）+ 收尾（金句）
第2天：第一件事 + 第二件事
第3天：第三件事 + 第四件事
第4天：未来展望 + 完整串联
第5天：完整演练3遍
```

### 2. 关键词串联法

每段只记3个关键词，现场展开：

| 段落 | 关键词1 | 关键词2 | 关键词3 |
|------|---------|---------|---------|
| 第一件事 | 质量问题 | 8家里6家 | 有人能做的更好 |
| 第二件事 | 三个问题 | 五个字 | 集团制度中心 |
| 第三件事 | 参照系 | 公司决定 | 120%传导2000人 |
| 第四件事 | 37亿 | 补位 | 3周恢复 |
| 未来 | 潜在矛盾 | 筛选→培养 | 2026Q2 |

### 3. 数字锚点法

必背数字，卡壳时从数字想起：

```
33家、3239人、4条线（开场）
五个字：选、用、育、留、汰（贯穿）
8家里6家（第一件事）
三个问题（第二件事）
120%、2000人（第三件事）
37亿、3周（第四件事）
2026年Q2、10家（未来）
```

### 4. 逻辑转折点

这些词后面必须有停顿：

- "今天的述职分三个部分" → 停顿，让评委知道结构
- "这一年我从三个层面来解决" → 停顿，开启主体
- "第二个层面花了主要精力" → 停顿，强调重点
- "还有一个补位的事情" → 停顿，转折
- "但我发现一个潜在的矛盾" → 停顿，制造悬念
- "回顾这一年，说到底是在做三件事" → 长停顿，全场安静

## 应急背诵锦囊

### 如果真的忘词了

| 位置 | 应急话术 |
|------|----------|
| 开场忘 | "各位领导好，我是孙大年，管理33家供应商3239人团队"（直接跳核心） |
| 三件事忘 | "第一件事解决质量问题，第二件解决标准问题，第三件解决参照问题"（用问题类型代替） |
| 具体数据忘 | 直接跳过数字，说"大部分新供应商都达到了优秀水平"（模糊处理） |
| 金句忘 | "这一年做了三件事：找人、立规矩、沉淀能力"（用口语化版本） |

### 保底结构（如果真的大脑空白）

```
各位领导好，我是孙大年。

去年我主要做了三件事：
第一，引入优质供应商，激活竞争；
第二，统一管理制度，建立标准；
第三，组建自营团队，验证天花板。

未来要做BPO联盟，从筛选转向培养。

谢谢各位领导。
```

## 最后提醒

1. **不要追求一字不差**，意思对、逻辑通即可
2. **数字必须准确**，8/6、120、37亿、3周、2026Q2/10家
3. **金句必须完整**，三件事和终极金句练到肌肉记忆
4. **停顿比语速重要**，该停的地方停住，让评委消化
5. **眼神交流**，不要盯着天花板背，要扫视评委
