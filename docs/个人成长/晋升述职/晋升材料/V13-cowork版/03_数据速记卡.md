## 使用说明

- **演讲前**：快速浏览，确保每个数字都能脱口而出

- **背诵时**：用这张卡自测，遮住右边看左边能不能说出数字

## 十一、容易混淆的数字

| 数字 | 含义 | 别混淆 |
|------|------|--------|
| **4家** | 清退的供应商数量 | 不是引入的（引入是8家） |
| **8家** | 引入的新供应商数量 | 不是清退的 |
| **15天** | 我们恢复周期 | 行业是30天 |
| **30天** | 行业恢复周期 / 原引入周期 | 别搞混 |
| **17次** | 巡检次数 | 不是供应商数量 |
| **33家** | 供应商总数 | 不是团队人数 |
| **3239人** | 团队总人数 | 不是供应商数量 |

## 使用建议

1. **打印出来**：面试前打印，放在手边
2. **逐行自测**：遮住右边，看左边能不能说出数字
3. **重点标记**：用记号笔标出你经常卡住的数字
4. **当天速查**：面试前10分钟快速扫一遍
