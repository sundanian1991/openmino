# 晋升述职演讲节奏建议（V7版）

---

## 整体风格定位

| 维度 | 建议 |
|------|------|
| **基调** | 自信但不张扬，谦虚但不卑微 |
| **语速** | 整体偏慢，重点处更慢 |
| **停顿** | 段落间留1-2秒空白，让评委消化 |
| **眼神** | 每段至少扫视全场一次，重点处定点 |

## 逐段拆解

### 开场白

**原文**：
> 各位领导好，我是金科事业群电销服务组的孙大年。
>
> 2020年加入公司，在云业务做了四年企业服务，去年5月转到电销服务组。这次述职我想用三个词概括自己的工作：**找对人，立好规矩，让能力长在组织上**。
>
> 先说"找对人"。我现在管理33家外包供应商、3239人的销售团队，覆盖消金、企金、财富、信用卡4条业务线。供应商管理的核心就在"选"——人选对了，后面的事半功倍；人选错了，后面的事倍功半。

**节奏建议**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "各位领导好" | 轻微点头，目光环视 | 建立连接 |
| "孙大年" | 语速正常，不要加速 | 自信从容 |
| "用三个词概括" | **停顿1秒**，引起注意 | 制造悬念 |
| "**找对人，立好规矩，让能力长在组织上**" | **逐词加重+放慢**，全文核心框架 | 让评委记住你的框架 |
| "先说'找对人'" | 语气平稳，自然过渡 | 开启第一个板块 |
| "事半功倍 / 事倍功半" | **对比加重**，强调入口重要性 | 逻辑点睛 |

**时长**：约45秒

### 核心贡献-总起

**原文**：
> 去年电销整体业绩超额130%达成。这个成绩是策略、业管、产品全链条协作的结果。我在最前端的供应商环节，负责"入口把关"。这个环节的重要性在于：前面卡不严，后面再努力也很难带动。

**节奏建议**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "超额130%达成" | 语速稍快，轻描淡写 | 成绩不邀功 |
| "全链条协作的结果" | 语气真诚，不抢功 | 格局体现 |
| "入口把关" | **加重**，定位清晰 | 价值锚点 |
| "前面卡不严，后面再努力也很难带动" | **放慢+加重**，强调入口重要性 | 逻辑点睛 |

**时长**：约25秒

### 第一件事：供应商引入

**原文**：
> **为什么要做这件事？**
>
> 去年年初盘了一下现有供应商，发现一个问题：合作久了，部分供应商进入"舒适区"——产能稳定但缺乏突破动力。我们需要新鲜血液来搅动这池水。
>
> **做了什么？**
>
> 牵头引入了8家新供应商，全部按统一标准进行甄选。不是简单的"招进来"，而是设计了季度赛马机制——新老供应商同台竞技，按产能、转化效率、合规表现排名。
>
> **结果如何？**
>
> 8家新供应商里，6家进过季度前三，占比75%。更重要的是，这种竞争压力传导给了原有供应商，整体产能水位被抬高了。
>
> **这件事的价值在哪？**
>
> 不是"我招了多少家供应商"，而是建立了一种"持续汰换、优胜劣汰"的机制。让这个3000多人的渠道池始终保持活力。

**节奏建议**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "**为什么要做这件事？**" | **加重**，抛出困难开场 | 故事结构清晰 |
| "舒适区" | 加引号手势（如有），或轻微语气变化 | 生动形象 |
| "搅动这池水" | 语气带点生动 | 画面感 |
| "**做了什么？**" | 自然过渡 | 结构清晰 |
| "**结果如何？**" | 自然过渡 | 结构清晰 |
| "8家...6家...75%" | **停顿+加重**，让数字自己说话 | 实力不言而喻 |
| "**这件事的价值在哪？**" | **加重**，引出升华 | 结构清晰 |
| "不是...而是..." | 对比句式，先抑后扬 | 意义提炼 |
| "3000多人的渠道池始终保持活力" | **放慢+加重**，价值升华 | 格局提升 |

**时长**：约60秒

### 第二件事：制度统一（重点段落）

**原文**：
> **先说背景——这件事花了我去年主要精力。**
>
> 电销服务组是几个老团队合并而来的。合并前，每个团队有自己的供应商管理习惯、评价标准、审批流程。打个比方：A团队看供应商重产能，B团队重合规，C团队重响应速度——标准都不一样。
>
> **年初是什么状态？**
>
> 四个团队带着各自的业务经验和惯性。同样一个供应商，在这个团队是"优质"，在另一个团队可能连门槛都进不了。每次跨团队协调，都要先花时间解释"你们那个标准是什么意思"。
>
> **我做了什么？**
>
> 牵头做了三件事：统一标准、流程线上化、沉淀手册。
>
> **年底是什么状态？**
>
> 四个团队像磨合很久了一样——用同一套语言说话，按同一套标准执行。每个人知道自己该干什么，每个场景知道怎么做。
>
> **这件事的价值在哪？**
>
> 对我来说：减少了大量协调沟通的时间...
> 对团队来说：可以专注在业务交付上...
> 对组织来说：经验沉淀成制度和流程...

**节奏建议**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "**先说背景——这件事花了我去年主要精力。**" | **加重+放慢**，强调投入 | 让评委知道这是重头戏 |
| "打个比方" | 语气轻松，准备举例 | 具象化 |
| "A团队...B团队...C团队..." | 语速稍快，带过例子 | 画面感 |
| "**年初是什么状态？**" | **加重**，制造对比 | 故事结构 |
| "四个团队带着各自的业务经验和惯性" | 语气略紧，描述问题 | 困难感 |
| "你们那个标准是什么意思" | 加引号，模仿语气 | 生动 |
| "**我做了什么？**" | 自然过渡 | 结构清晰 |
| "统一标准、流程线上化、沉淀手册" | 三个词清晰有力 | 行动明确 |
| "选、用、育、留、汰" | **逐字放慢**，五个字像五颗钉子 | 体系感 |
| "**年底是什么状态？**" | **加重**，对比转折 | 故事高潮 |
| "像磨合很久了一样" | **放慢**，形容转变 | 成果感 |
| "每个人知道自己该干什么" | 语气笃定 | 秩序感 |
| "**这件事的价值在哪？**" | **加重**，分层升华 | 结构清晰 |
| "对我来说...对团队来说...对组织来说..." | **逐层放慢**，三个维度 | 全面升华 |

**⚠️ 关键技巧**：这是老板重点关注的段落，务必把"年初混乱→年底有序"的对比讲清楚，价值升华的三个维度层层递进。

**时长**：约80秒

## V7版结构特点

### 每个案例的统一结构

```
【困难开场】为什么要做这件事？/背景是什么？/年初是什么状态？
↓
【行动过程】做了什么？/过程是怎样的？
↓
【结果呈现】结果如何？/年底是什么状态？
↓
【价值升华】这件事的价值在哪？/深层看是什么？
```

### 衔接技巧

| 类型 | 用法 | 示例 |
|------|------|------|
| 设问衔接 | 用问题引出下一段 | "为什么要做这件事？" |
| 逻辑衔接 | 用逻辑关系过渡 | "除了引入新供应商，我们还需要..." |
| 时间衔接 | 用时间节点过渡 | "去年7月..." |
| 总结衔接 | 用小结引出下一部分 | "具体做了四件事" |

## 应急技巧

| 情况 | 应对方式 |
|------|----------|
| **忘词了** | 停顿，喝口水，微笑。不要"呃...啊..." |

| **评委表情严肃** | 不受影响，保持自己的节奏 |

## 最后提醒

1. **背熟开场45秒**——开场稳了，后面就顺了
2. **金句"三件事"要练到肌肉记忆**——找对人，立好规矩，让能力长在组织上
3. **四个关键数字要清晰**：8家里有6家、年初→年底对比、120%、37亿GMV
4. **重点段落是"制度统一"**——务必把"年初混乱→年底有序"的对比讲清楚
5. **每个案例的"价值升华"是亮点**——不要跳过，这是思维深度的体现
6. **提前到场适应场地**——站在讲台上找一下感觉
7. **答辩环节不要抢话**——等评委问完，思考1秒再答
