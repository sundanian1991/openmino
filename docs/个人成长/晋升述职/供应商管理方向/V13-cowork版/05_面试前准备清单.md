## 📅 面试前 6-4 天：背诵练习

### ✅ 第一阶段：分段背诵（第6天）

- [ ] 背诵 **开场**（70秒）→ 必须滚瓜烂熟

## 🚨 紧急情况应对

### 情况1：突然忘词了

**应对**：
- 深呼吸1秒
- 看一眼数据速记卡/案例卡片
- 继续讲，不要卡住太久
- 评委更看重你的整体表现，不是逐字背诵

### 情况2：评委挑战你的数据

**应对**：
- 保持冷静，不要争辩
- 说"这个数据是我根据XX测算的，可能有不准确的地方"
- 回到核心观点：数据背后的逻辑和结论

### 情况3：评委问了一个你没准备的问题

**应对**：
- 不要慌，先思考5秒
- 用STAR框架组织答案
- 如果真的不会，诚实说"这个问题我还没有深入思考，但我会从XX角度来考虑"

### 情况4：时间不够，被评委打断

**应对**：
- 立即停止，不要强行讲完
- 直接跳到收尾"回顾这一年..."
- 被打断不代表不好，可能评委已经了解了
