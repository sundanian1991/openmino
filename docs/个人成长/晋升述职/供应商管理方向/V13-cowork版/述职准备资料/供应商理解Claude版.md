# 供应商管理岗位深度理解——Claude版

> 基于与用户的多轮对话整理
> 更新时间：2025年1月

---

## 一、岗位全景

### 1.1 管理规模

| 维度 | 规模 |
|------|------|
| **供应商数量** | 32家（人工）+ 1家（自建） |
| **人员规模** | 累计3000+人 |
| **业务覆盖** | 4大产线、17条细分业务 |
| **管理周期** | 全生命周期：引入→商务→合同→巡检→处罚→合规→扩张→分量→转正→清退 |

### 1.2 一天的工作

**早晨：**
- 看所有供应商的排名数据（周/月趋势）
- 了解谁最好、谁最差

**白天：**
- 开会/打电话给供应商老板施压
- 处理合规问题、客诉
- 接内部同事需求，了解新项目，寻源供应商
- 谈价格、更新供应商了解

**不定期：**
- 出差巡检现场

## 二、核心能力体系

### 2.1 最不可替代的能力：分量策略

**本质：** 把合适的供应商放在合适的地方，让每家都能产出最大价值

**供应商画像维度：**
- 配合度：高/中/低
- 业务结构：头部/中部/尾部
- 学习能力：强/中/弱
- 业务经验：拉新/企微/大额/创新型
- 上人速度：快/中/慢
- 历史表现：稳定/波动

**决策维度：**
1. 业务类型：拉新vs复购
2. 业绩情况：近期表现
3. 上人速度：多快能上人
4. 历史配合：过往配合度
5. 战略分配：调整供应商结构的需求

**动态管理：**
- 不都给，也不一直不给
- 让供应商保持期待
- 头部和尾部重点管理
- 中间供应商可以激励成头部（标杆打造）

**网状结构思维：**
```
         拉新型          企微型
            |              |
     头部：A、B        头部：E、F
     中部：C、D        中部：G、H
     尾部：I、J        尾部：K、L

         大额型          创新型
            |              |
     头部：M、N        头部：Q、R
     中部：O、P        中部：S、T
     尾部：U、V        尾部：W、X
```

### 2.2 核心竞争力："拿捏供应商"

**核心逻辑：** 胡萝卜+大棒，业绩论英雄

**胡萝卜（分量）：**
- 业绩好就有更多量
- 新项目优先考虑
- 信息同步（怎么决策的）
- 内部规划透露（但不详细）

**大棒（合规+分量削减）：**
- 不听话→频繁复盘问询
- 挑肥拣瘦→新项目不找他
- 业绩差→分量削减
- 严重违规→清退+拉黑

**独特之处：**
- 以业绩论英雄，不管背景/历史
- 规则运行在"阳光"下（相对透明）
- 口碑效应：做好就有，躺平就没

**对冲策略（降价时）：**
- 降价不是永远降，后面可能回调
- 承诺看好生命线（如6000元活下来线）
- 放大客户获客额度

### 2.3 自建团队能力

**选人画像（坐席）：**
1. 很想赚钱
2. 能接受周六加班
3. 年轻有冲劲
4. 没有其他干扰
5. 人相对纯粹

**选人画像（主管）：**
- 有充分的带团队经验
- 有从0到1搭建团队的能力

**可复制性：**
- 可以作为项目经理/总监
- 从零开始搞定场地、装修、招聘
- 看到"业务以外"的不足点
- 让专业的人去落地业务

**"业务以外"的点：**
- 团队设置
- 绩效设计
- 团队结构
- 选人标准
- 场地选址
- 装修布置
- 设备采购
- 招聘渠道

**宿迁自建团队战绩：**
- 面试近200人，百里挑一选20人
- 看了300份简历
- 在宿迁待了一个月
- 高频争取、协调
- 结果：费效进入健康区间（1.2倍以内），产能环比+110%

### 2.4 无损退出能力

**让供应商心甘情愿自己申请：**

**两难选择：**
- 闹→被京东拉黑
- 走→日后好相见

**操作手法：**
1. 告知公司制度
2. 说明在行业的影响
3. 承诺不加黑名单（如果自己申请）
4. 利用黑名单的行业威慑力

### 2.5 京东适应体系

**自己总结的文档：**
- 京东电销团队怎么运转
- 过去踩过的坑
- 内部怎么配合
- 管理要求和处罚措施

**对新供应商：**
- 充分信息交流
- 答疑解惑
- 前置沟通（京东看什么、在乎什么、怎么工作）
- 快速出产能的方法

**价值：**
- 供应商容易理解和吸收
- 减少适应时间
- 提升配合度

## 三、核心成就

### 3.1 最大成就：宿迁自建团队（0→1）

**背景：**
- 老板在京东立足的核心项目
- 如果做不成，整个组的价值会被审视
- 入职前就深度参与决策

**具体动作：**
- 团队设置、绩效、结构设计
- 亲自面试近200人
- 看了300份简历，百里挑一选20人
- 在宿迁待了一个月
- 高频争取、协调

**角色定位：**
- 团队的"初代灵魂"
- 类似政委，凝聚人心
- 不是显性的，但深入骨髓

**结果：**
- 费效进入健康区间（1.2倍以内）
- 产能环比+110%（vs BPO 45%）
- 成为供应商降价的对标尺

### 3.2 降价15%（商务压降）

**背景：**
- 8月份复盘：自建产能+110%，BPO产能+45%
- 自建费效进入健康区间
- BPO结算成本有优化空间

**动作：**
- 9月份基于利润空间及产能盘点
- 对同类名单结算单价从0.15%降至0.1275%
- 降幅15%

**结果：**
- 当月降本100万（671万→571万）
- 驱动来自于自建职场的"效能标杆"

**逻辑链条：**
```
自建团队做得好 → 拿着数据去谈 → 供应商害怕 → 降价
```

**坦诚的真相：**
- 更多是安抚供应商
- 打信息差
- 承诺"不是永远降"
- 承诺看好生命线

### 3.3 金企融合+企贷大额（新增项目）

**金企融合：**
- 支撑400人团队组建
- 项目统管角色
- 从零开始招聘供应商
- 协调产品、策略、业务
- 安抚/施压供应商

**企贷大额：**
- 200人规模
- 余额净增：4.47亿
- 放款金额：9.28亿（人工2亿，人机7.28亿）
- 当月净增1月突破1亿
- 余额费效：T10授信口径1.8%
- 拉新佣金：804万

## 四、不可替代性

### 4.1 如果没有我

**会发生什么：**
- 至少两个人可以接手
- 业务不会停

**但是：**
- 供应商会"难受"
- 重新筛选供应商
- 重新适应过程
- 损失一部分效率和产出

**具体损失：**
- 业管换了→对接重新磨合
- 商务换了→争取机会的渠道少了
- 没把合适供应商放在合适地方→**潜在产出损失**

### 4.2 补位能力

**如果有5家突然退出：**
- 至少有5-10家可以随时补位
- 快速决策谁适合什么业务

## 五、三个案例方向

### 5.1 案例1：选汰机制（制度相关）

**核心：** 激活竞争，打破渠道舒适区

**关键动作：**
- 选：统一准入标准，流程线上化，30天→15天，引入8家
- 用：赛马机制+保护期
- 育：自建宿迁职场0→1
- 汰：清退4家违规供应商，零中断

**关键结果：**
- 新供应商75%进季度TOP3
- 清退零纠纷
- 宿迁3个月达成120%业绩
- 单价下降15%，每月降本100万

**价值：** 从"依赖外包"到"自主可控+多元竞争"

### 5.2 案例2：合规风控（合规相关）

**核心：** 建立精细化管控体系

**背景：** 新增1000+人，缺乏统一标准和过程监控

**关键动作：**
- 事前：制定五项制度
- 事中：巡检17次覆盖20家，红灯4次全部化解
- 事后：递进式清退规则，6家全部整改闭环

**关键结果：**
- 全年零纠纷
- 清退4家无产能中断

**价值：** 从"事后救火"到"过程防控"

### 5.3 案例3：线路应急（线路相关）

**核心：** 用"主动补位"化解核心业务中断危机

**背景：** 核心业务线路中断，日影响1300万呼叫、37亿周GMV

**关键动作：**
- 链：快速链接运营商资源
- 判：建立线路价值评估模型
- 建：沉淀《线路应急管理预案》

**关键结果：**
- 15天完成恢复（行业30天）
- 避免约80亿GMV损失

**价值：** 从"被动救火"到"主动防控"

## 六、三个案例的完整信息（已深挖完成）

### 6.1 案例1：选汰机制（制度相关）

**引入标准（8家）：**
1. 有现成团队做相关业务
2. 线上数标（统一框架和内容）
3. 评估维度：过往业绩、同类型业务对标、管理方式、人员数量、公司背景、历史交付风格、配合的商务、内部团队架构

**引入周期优化：**
- 之前：邮件审批4-5天，总体约1个月
- 现在：审批1天，寻源7天，培训上线总计约2周
- **30天 → 15天**

**寻源储备机制：**
- 事先寻源、储备、分类
- 一两周/一个月都有沟通
- 了解最新案例更新
- 匹配合适供应商，让供应商在自己擅长的业务上精深

**分量打分维度：**
- 业绩：30%
- 配合：15%
- 上人数字：15%
- 质检客诉：25%
- 战略打分：15%（确认是否要扶持、学习能力、额外技能）

**季度TOP3标准：**
- 季度内至少有一个月进入综合排名前三
- 综合排名包含：名单产能、名单转化、赛马、质检质量

**老供应商主动优化：**
- 定期跟头部和尾部沟通
- 找过用户的供应商基本都有改进
- 头部和尾部差异在缩小
- 要求向优秀团队学习
- 整体标准交付提升

**自建职场人效：**
- 业绩增速：自建+110% vs 外包+45%
- 费效：自建虽成本更高，但产出更好
- 成为供应商降价的对标尺

**降价15%（商务压降）：**
- 背景：8月复盘自建产能+110%，BPO+45%，自建费效进入健康区间
- 动作：9月对同类名单结算单价从0.15%降至0.1275%
- 结果：当月降本100万（671万→571万）
- 覆盖：所有金条供应商都降了
- 理由：
  1. 自建团队压力（110% vs 45%）
  2. 所有供应商一视同仁
  3. 绝对价值仍可接受
  4. 不接受就不给业务

### 6.2 案例2：合规风控（合规相关）

**新增1000+人分布：**
- 大约覆盖15家供应商
- 主要是新人

**巡检覆盖：**
- 7次巡检
- 每次看2-3家供应商
- 每次3-5天
- 覆盖地区：承德、石家庄、合肥、宿迁等高频区
- 偏远地区（内蒙、云南、黑龙江）去得少

**巡检标准：**
- 现场空间布置
- 龙虎榜、公开业绩
- 话术管理
- 活动激励了解程度
- **作息薪资：给供应商100元，员工最少拿60元（检查薪资流水）**
- 员工满意度
- 每天工作时间
- 培训方式
- 招聘方式
- 品牌滥用检查

**红线（红灯）：**
- 借用京东名义招聘
- 京东未授权使用的logo、商标
- 招聘用子公司主体或不用自己主体

**化解方式：**
- 当场说问题+解决方法
- 24小时内发邮件
- 要求3-7天整改
- 后续提供整改情况
- 有张贴的撕掉，有实用的改掉

**递进式清退：**
- 警告
- 约谈
- **限制新进：一段时间内不允许接其他业务，只能持续流失，不允许新增人力**
- 清退

**最难整改：**
- 现场硬件设施/工位不达标
- 100多名员工需要渐进式调整
- 花了约1个月时间

### 6.3 案例3：线路应急（线路相关）

**领先行业15天的关键差异：**

**1. 找资源（结构性差异）：**
- 全国范围内突发政策监管针对三大运营商
- 每家供应商有自己的指标
- 成熟供应商指标用光了
- 部分供应商有指标但没用
- 快速找到这样的供应商，让他具备做业务的能力

**2. 控制客诉比：**
- 剔除不适合的人
- 管理外呼频次
- 过程监控
- 每天沟通调整，快速迭代
- 高频管理调度
- 数据和过程指标监控
- 供应商指标是次日前一天反映客诉结果
- 调整晚了会从好跌到差

**1300万呼叫/37亿GMV：**
- 1300万：每天的电话拨打量（后台统计）
- 37亿：一周的GMV（按7天算）
- 实际损失：保留了约20%的外呼量（300万），损失了约80%

**线路费涨幅：**
- 市场价格上涨200%-300%
- 找了涨幅相对少的线路

**价值评估模型：**
- **成本（30%）：** 线路费（找涨幅相对少的）
- **稳定性（40%）：** 过往做过类似案例、直联、有落地机房
- **覆盖率（30%）：** 资源丰富程度、重点覆盖的城市/运营商、管控严格程度

**主动补位原因：**
- 团队里没有专职的人
- 只有一个业务运营（偏运营，不是偏商务）
- 这是边缘角色
- 没有应急角色
- 业务紧迫

**预案推动：**
- 每天组织复盘会
- 分析GMV指标、客诉比、客诉分数
- 梳理流程：预警→应急等级→对应动作
- 动作：降名单、降频控、降人力

## 七、核心价值提炼

**一句话总结：**
通过供应商全生命周期管理，建立"多元竞争+自主可控"的渠道生态，实现成本和效率双优化。

**三个关键词：**
1. **拿捏**：让供应商在有限奖励下依然愿意投入
2. **对标**：通过自建团队建立效能基准，倒逼供应商优化
3. **生态**：从甲乙方关系到合作伙伴关系

### 结构优势分析

| 特征 | 为什么好 |
|------|----------|
| **岗位认知1句话** | 让评委快速理解：你是做什么的、你的工作价值是什么、你沉淀了什么方法论 |
| **结果前置** | 评委一眼看到你做到了什么，震撼数字抓住注意力 |
| **挑战具体** | 不用抽象形容词（如"散僵盲弱乱"），用具体痛点让评委感受到问题 |
| **贡献标题1句话** | "通过XX，实现XX"——清晰展示你的思路和价值 |
| **举措层层剥洋葱** | 大块（破垄断/建标杆）→小块（引入提速/赛马激活），每层都有数据，逻辑清晰 |
| **方法论可复用** | 每个举措都有标签（如【引入提速】），看起来是专家思考，不是凑数的 |
| **每个动作都有结果** | 证明了"我是这么做的，带来了这个结果"，动作和价值强关联 |
| **优势4个关键词** | 精炼展示核心能力，容易记住 |
| **不足1个关键词** | 展示自我认知，不贪多，显得真实 |
