# 述职讲解框架（精简版）

> ⚠️ 这是**讲解提示卡**，不是逐字稿！
> 演讲时扫一眼就知道讲什么、怎么讲。

---

## 📋 整体结构（8分钟）

```
开场（60秒）→ 量化指标（40秒）→ 三个案例（240秒）→ 未来规划（60秒）→ 收尾（30秒）
```

---

## 一、开场（60秒）

### 讲什么

| 要点 | 核心内容 |
|------|---------|
| **我是谁** | 姓名 + 转岗时间 |
| **管什么** | 33家供应商、3239人、4条线、17条业务 |
| **怎么管** | "选、用、育、留、汰" |
| **做得怎么样** | Q3/Q4/年度 A，2次匠心独运奖 |
| **今天讲什么** | 量化指标 + 三个案例 + 未来规划 |

### 怎么讲

1. 先讲管什么（最大销售渠道）
2. 再讲怎么管（五个字）
3. 然后证明做得好（绩效）
4. 最后预告结构

### 评委可能问

| 问题 | 回答 |
|------|------|
| "33家怎么管？" | 全生命周期管理，后面案例详细讲 |

---

## 二、量化指标（40秒）

### 讲什么

| 层级 | 讲解要点 |
|------|---------|
| **第一层：业务增量** | 新增3个项目，___人，___亿GMV |
| **第二层：降本增效** | 单价降___%；30天→15天 |
| **第三层：风险防控** | 清退4家零纠纷；15天恢复（行业30天） |

### 怎么讲

- 只讲结果数字
- 过程留到案例讲
- 数字要准、语速放慢

### 评委可能问

| 问题 | 回答 |
|------|------|
| "新增的3个是哪几个？" | 金企融合、企贷大额、金条 |
| "单价降怎么算的？" | 自建团队人效120%，倒逼降价 |

---

## 三、案例一：激活竞争（80秒）

### 讲什么（STAR）

| 要素 | 讲解要点 |
|------|---------|
| **S（问题）** | "散僵盲弱乱"——四团队标准不一、老商依赖固化、新商成活率低 |
| **T（目标）** | 建立全生命周期管理机制 + 自建团队 |
| **A（做法）** | 选（30→15天）、用（赛马）、育（宿迁0→1）、汰（清退零中断） |
| **R（结果）** | 新商75%进TOP3；宿迁120%业绩；清退零纠纷 |
| **价值** | 从"依赖外包"到"自主可控" |

### 怎么讲

1. 先讲问题（让评委理解必要性）
2. 再讲做法（选、用、育、汰环环相扣）
3. 然后讲结果（用数字证明）
4. 最后升华（不是管理，是生态重构）

### 钩子解释

| 钩子 | 怎么解释 |
|------|---------|
| 30→15天 | 3环节线上化，各砍一半 |
| 75%进TOP3 | 5维度筛选；老商3家降价、2家整改 |
| 宿迁120% | 投入___人、___月；人效110% |

## 四、案例二：合规风控（90秒）

### 讲什么（STAR）

| 要素 | 讲解要点 |
|------|---------|
| **S（问题）** | 千人增长，33家供应商、3000+人，缺标准、缺监控，风险累积 |
| **T（目标）** | 全周期合规管控，千人增长下零重大风险 |
| **A（做法）** | 事前（五项制度）、事中（巡检17次、红灯4次）、事后（递进式清退，6家全闭环） |
| **R（结果）** | 全年零纠纷；清退4家零中断；制度成部门标准 |
| **价值** | 从"事后救火"到"过程防控" |

### 怎么讲

1. 先讲挑战（千人增长=质量+风险平衡）
2. 再讲体系（事前事中事后三道防线）
3. 然后讲效果（巡检不是走过场）
4. 最后升华（合规是护城河，不是成本）

### 钩子解释

| 钩子 | 怎么解释 |
|------|---------|
| 千人增长零风险 | 100%培训、巡检全覆盖、红线熔断 |
| 巡检17次、红灯4次 | 8项清单；24小时整改+7天复查 |
| 递进式清退 | 警告→约谈→限制→清退，7-30天/阶段 |
| 6家全闭环 | 最难___家，___问题，___天解决 |

## 五、案例三：线路应急（80秒）

### 讲什么（STAR）

| 要素 | 讲解要点 |
|------|---------|
| **S（问题）** | 线路中断风险，日影响1300万呼叫、37亿周GMV；行业恢复30天；无专职负责人 |
| **T（目标）** | 主动补位，15天恢复，建立长效机制 |
| **A（做法）** | 链（链接运营商）、判（评估模型）、建（预案+标准化） |
| **R（结果）** | 15天恢复（领先15天），避免80亿损失；预案成部门标准 |
| **价值** | 从"被动救火"到"主动防控" |

### 怎么讲

1. 先讲紧急性（37亿、30天，让评委理解这不是小事）
2. 再讲为什么是你（无专职负责人，业务等不了）
3. 然后讲关键动作（运营商直连、决策授权、双班并行）
4. 最后升华（不是解决一次，是建立机制）

### 钩子解释

| 钩子 | 怎么解释 |
|------|---------|
| 领先15天 | 运营商直连、决策授权、双班并行 |
| 37亿怎么算 | 1300万呼叫×___%接通×___%转化×___元单价 |
| 主动补位 | 原属___团队但无专职负责人；业务紧迫 |
| 预案成标准 | 复盘→发文→培训，已触发___次 |

## 六、未来规划（60秒）

### 讲什么

| 要素 | 讲解要点 |
|------|---------|
| **战略定位** | 从"价格压降"到"价值共创" |
| **核心举措** | 输出京东体系、供应商分级、学习联盟 |
| **三个突破** | 谈判→生态、单一→网络、价格→体系 |
| **量化目标** | 2026年Q2运转，覆盖10家 |
| **个人规划** | AI Agents探索 + 传帮带 |

### 怎么讲

1. 先讲方向（不是压价，是共建生态）
2. 再讲怎么做（输出能力、分级管理、联盟学习）
3. 然后讲目标（Q2、10家）
4. 最后讲个人（技术趋势+组织贡献）

### 评委可能问

| 问题 | 回答 |
|------|------|
| "BPO联盟和赛马矛盾吗？" | 不矛盾，分层：核心10家联盟，其他23家赛马 |
| "AI Agents能做什么？" | 智能巡检、风险预警、知识沉淀 |

## 七、收尾（30秒）

### 讲什么

| 要素 | 核心内容 |
|------|---------|
| **回顾** | 这一年做了三件事 |
| **第一件事** | 找对人 —— 选汰机制激活竞争，依赖→自主 |
| **第二件事** | 立好规矩 —— 全周期合规管控，救火→防控 |
| **第三件事** | 能力长在组织上 —— 自建团队+机制沉淀 |

### 怎么讲

1. 停顿2秒（全场安静）
2. 逐字清晰（三个金句）
3. 微笑收尾

## 九、讲解技巧

| 技巧 | 做法 |
|------|------|
| **眼神交流** | 看着评委，不是稿子 |
| **停顿** | 讲完一部分，停1秒 |
| **重音** | 关键数字加重语气 |
| **手势** | 有PPT的话，引导看重点 |
| **语速** | 不快不慢，评委能听清 |

## 十、核心提醒

**基于材料，不等于材料**

- 材料是笔记 → 讲解是理解
- 看框架提示 → 用自己的话
- 每次略有不同 → 更自然

**练习方法**

1. 看一眼框架要点
2. 用自己的话解释
3. 录音自查（语速、停顿、重音）
4. 每次略有不同（更自然）
