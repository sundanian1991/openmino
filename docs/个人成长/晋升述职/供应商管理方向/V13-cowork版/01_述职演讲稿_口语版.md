## 演讲稿（全文约8分钟）

各位评委好，我是孙大年。

我先简单介绍一下自己。我2020年加入京东云，今年5月转岗到金科数据科技业务部，负责电销供应商管理。

我管的是什么？33家外包供应商，3239人的电销团队，覆盖4大业务产线、17条细分业务——这是金科最大的销售渠道。

我的核心工作，概括为五个字：选、用、育、留、汰。通过供应商的"选用育留汰"，确保供应链能持续、稳定、高效地支撑电销业务。

（绩效和奖项大家都能看到，我就不念了，直接进入正题）

今天的述职，我分三个部分来讲：先说量化指标，然后讲三个核心成果，最后聊聊未来规划。

---

先说量化指标。我用三层结构来总结今年的贡献。

第一层是业务增量。新增了3个项目——金企融合、企贷大额、金条——共 ___ 人，GMV增量 ___ 亿。

第二层是降本增效。通过自建团队的标杆效应，反向压降供应商单价 ___ %；同时把供应商引入周期从30天缩短到15天。

第三层是风险防控。清退4家供应商零纠纷；线路应急15天恢复，比行业快一倍，避免损失约80亿GMV。

这些数字是怎么来的？下面我通过三个核心成果来讲清楚。

**核心成果一：从"依赖外包"到"自主可控"——供应商生态体系搭建**

我看到了什么问题？

年初接手时，渠道管理表面上是什么？四个团队标准不一、老供应商依赖固化、新供应商成活率很低。但这些只是表象。

本质问题是什么？**供应链过度依赖外包，没有核心产能掌控力。** 老供应商躺在舒适区里，新供应商根本活不下来，整个渠道缺乏竞争活力。一旦某家供应商出问题，业务就会受影响——这不是一个健康的生态。

我的判断是什么？

要解决这个问题，不能只做"修修补补"。我的判断是：必须从**供应链结构**入手，两手抓——一手建立全生命周期管理体系，让供应商能进能出、能上能下；一手搭建自建团队，既是标杆也是筹码。只有自己有产能，才有真正的议价权。

我做了什么？

基于这个判断，我做了四件事：统一准入标准、设计赛马机制、自建宿迁职场、建立清退机制。引入周期从30天缩短到15天，引入8家新供应商，清退4家违规供应商。

带来了什么价值？

结果是什么？新供应商75%进季度TOP3，倒逼老供应商主动优化；宿迁职场3个月达成120%业绩，验证了人效天花板。

但更重要的是——**建立了供应商生态体系**。自建团队成为标杆和筹码，让整个渠道从"依赖外包"到"自主可控"。这套体系是可以复制的，明年、后年都能持续产出。

这是第一个成果，通过体系搭建实现从依赖到自主的跨越。

第二个成果，我想讲讲怎么建立风控体系。

**核心成果二：从"事后救火"到"过程防控"——合规风控体系搭建**

我看到了什么问题？

去年业务新增了1000多人，整个渠道池达到3000+人的规模。但过去没有统一的管理制度，33家供应商各自为政，过程监控几乎是空白。

这意味着什么？**规模越大，风险累积越快**。不是已经出事了，而是风险在暗处积累——随时可能爆雷。

我的判断是什么？

如果是100人的规模，靠人盯人还能管得住。但到了3000+人的规模，必须靠**体系**。

我的判断是：不能等问题发生了再救火，必须建立"事前、事中、事后"三道防线，让合规从"成本负担"变成"业务护城河"。

我做了什么？

基于这个判断，我建立了全周期合规管控体系：事前制定五项制度，事设计巡检加预警机制，事后建立递进式清退规则。全年巡检17次覆盖20家供应商，红灯预警4次全部化解，问题供应商6家全部整改闭环。

带来了什么价值？

结果是：全年零纠纷；清退4家违规供应商无产能中断；沉淀的五项制度和管控机制，成为了部门标准。

但更重要的是——**建立了可复制的合规风控体系**。不是解决了一个问题，而是建立了一套可以持续运作的机制。明年规模继续增长，这套体系也能兜得住。

这是第二个成果，通过风控体系实现从救火到防控的跨越。

第三个成果，我想讲一个突发危机事件的处理。

**核心成果三：危机处理与机制沉淀——从"被动救火"到"主动防控"**

我看到了什么问题？

核心业务线路突然中断，每天影响1300万呼叫、37亿周GMV。行业恢复周期约30天，而且团队当时没有专职应急负责人。

这意味着什么？**每一天都在烧钱**，而且烧的是业务最核心的钱。

我的判断是什么？

这不是"技术问题"，而是"业务连续性问题"。等流程走完，业务早完了。

我的判断是：必须主动补位，用最快速度解决燃眉之急，同时建立长效机制——不能每次都靠"临时救火"。

我做了什么？

基于这个判断，我做了三件事：快速链接运营商资源解决燃眉之急，建立线路价值评估模型优化资源配置，沉淀《线路应急管理预案》推动制度标准化落地。

带来了什么价值？

结果是：15天完成恢复，领先行业15天，避免约80亿GMV损失；预案成为部门标准流程。

但更重要的是——**建立了危机处理机制**。不是解决了这一次危机，而是沉淀了一套标准流程。下次再遇到类似问题，组织能自主应对，不依赖个人。

三个成果讲完了，分别对应了体系建设、风险防控、危机处理。

最后聊聊未来规划。

业务攻坚方面：BPO联盟机制。

我的战略定位是：从"价格压降"转向"价值共创"。

核心举措有三个：第一，输出京东运营体系；第二，供应商分级认证；第三，建立学习联盟机制。

我想实现三个突破：从商务谈判到生态经营，从单一供应商到供应商网络，从价格驱动到体系驱动。

量化目标是：2026年Q2运转，覆盖核心供应商10家。

个人规划有两方面：

第一，AI Agents应用探索——把个人经验通过AI Agents实现智能化升级。

第二，传帮带——把自身沉淀持续为内外部赋能。

以上就是未来规划。

最后，我想用三句话总结一下这一年。

第一，找对人——通过"选汰机制"激活渠道竞争，从"依赖外包"到"自主可控"。

第二，立好规矩——建立全周期合规管控体系，从"事后救火"到"过程防控"。

第三，让能力长在组织上——通过自建团队和机制沉淀，让组织不依赖个人。

以上是我的述职，谢谢各位评委。

## 练习建议

### 第一天：读熟全文

1. 完整读一遍，标记不熟的句子
2. 重点练习不熟的部分
3. 读到顺畅为止

### 第二天：加入停顿和重音

**关键停顿点**（每处停顿1秒）：
- "今天的述职，我分三个部分" → 停顿
- "先看量化指标" → 停顿
- "下面我详细讲三个案例" → 停顿
- "第一个案例" → 停顿
- "这件事的价值是什么？" → 停顿（每个案例结束前）
- "回顾这一年，我做了三件事" → 停顿2秒

**关键重音点**（加重语气）：
- 所有数字：33家、3239人、30天、15天、75%、120%...
- 关键词：选、用、育、留、汰
- 金句：找对人、立好规矩、让能力长在组织上

### 第三天：录音自查

1. 录下来听一遍
2. 检查：语速太快？太慢？哪里不自然？
3. 调整后再录，直到听起来像在说话

### 第四天：对镜练习

1. 对着镜子念
2. 观察自己的表情和肢体
3. 确保看起来自信、自然

### 第五天：完整演练

1. 找同事模拟评委
2. 完整讲一遍
3. 根据反馈调整

## 演讲技巧

### ✅ 这样念

| 技巧 | 做法 |
|------|------|
| **语速** | 正常说话速度，不快不慢 |
| **停顿** | 每个部分之间停1秒，让评委消化 |
| **重音** | 关键数字加重语气 |
| **眼神** | 念稿时偶尔抬头看评委，不要一直低头 |
| **手势** | 有PPT的话，用手势引导评委看重点 |

### ❌ 不要这样念

| 错误 | 后果 |
|------|------|
| 语速太快 | 评委听不清，显得紧张 |
| 一口气念到底 | 评委跟不上，显得赶时间 |
| 声音太平 | 没有重点，显得无聊 |
| 全程低头 | 评委觉得你在念稿 |
| 像背书 | 显得机械，不自然 |
