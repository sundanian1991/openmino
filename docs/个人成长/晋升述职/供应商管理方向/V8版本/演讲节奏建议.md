# V8 版演讲节奏建议

---

## 整体风格定位

| 维度 | 建议 |
|------|------|
| **基调** | 自信但不张扬，谦虚但不卑微 |
| **语速** | 整体偏慢（150-180字/分钟），重点处更慢 |
| **停顿** | 段落间留1-2秒，金句后留2秒 |
| **眼神** | 每段至少扫视全场一次，重点处定点 |

---

## 逐段节奏拆解

### 开场白（70-75秒）

**原文**：
> 各位领导好，我是金科事业群电销服务组的孙大年，2020年加入京东，这是第二次晋升。我现在管理33家外包供应商、3239人的销售团队，覆盖消金、企金、财富、信用卡4条业务线。供应商管理的核心我概括为五个字：**选、用、育、留、汰**。去年5月我转到电销服务组时，面对的情况是**四个团队带着不同的管理习惯合并在一起**——之前各自有各自的供应商评价标准、审批流程、协作方式，合并后需要大量时间对齐，效率损耗很大。这一年我从三个层面来解决这个问题，建立一套统一的供应商管理体系。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "各位领导好" | 轻微点头，目光环视 | 建立连接 |
| "孙大年" | 语速正常，不要加速 | 自信从容 |
| "第二次晋升" | 平稳语气，不卑不亢 | 定位清晰 |
| "选、用、育、留、汰" | **逐字放慢+加重**，全文核心 | 让评委记住 |
| "不同的管理习惯" | **停顿1秒**，制造悬念 | 引出问题 |
| "效率损耗很大" | 语气略紧，描述问题 | 困难感 |

---

### 第一件事：供应商引入（60-65秒）

**原文**：
> 第一个层面的问题是**供应商质量参差不齐，缺乏持续激活机制**。有些供应商合作久了进入舒适区，产能稳定但缺乏突破动力。我的做法是在引入环节花心思，按统一标准甄选，找到更匹配的新供应商，并在过程中给予足够支持，让他们能快速达到老商TOP水平。结果是8家新供应商里6家进过季度前三。这不是简单的"招进来"，而是让存量供应商看到——**有人能做到更好，你还想留在牌桌上就得跟上**。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "质量参差不齐" | **加重**，问题开场 | 结构清晰 |
| "舒适区" | 加引号语气，生动形象 | 画面感 |
| "快速达到老商TOP水平" | **加重**，核心动作 | 价值锚点 |
| "8家...6家" | 停顿，让数字自己说话 | 实力不言而喻 |
| "有人能做到更好" | **放慢+加重**，价值升华 | 意义提炼 |

---

### 第二件事：制度统一 ⭐ 核心（105-110秒）

**原文**：
> 第二个层面我花了主要精力——**制度标准需要统一和升级**。主要面临三个问题：第一，**制度版本老旧**。之前的版本是三年前的，不太符合现在业务的实际需要；第二，**标准不统一**。不同业务线如果各用各的，同一个事情会有四套标准，不利于横向对标；第三，**流程效率低**。依赖线下邮件，没有线上化，审批链条长，缺乏存档追溯。我的做法是牵头编写《供应商全生命周期管理指引》，把"选、用、育、留、汰"每个环节的触发条件、判断规则、执行动作写清楚，同时推动流程线上化，沉淀操作手册。结果是新供应商引入从30天缩短到15天，四个团队用同一套语言说话，这套指引后来还上了集团制度中心实现跨部门复用。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "花了主要精力" | **加重+放慢**，强调投入 | 重头戏 |
| 三个问题逐一点明 | 每点**停顿1秒**，清晰有力 | 结构感 |
| "制度版本老旧" | 客观陈述，不针对人 | 安全表达 |
| "标准不统一" | 加重，结构性问题 | 困难感 |
| "流程效率低" | 加重，效率导向 | 业务语言 |
| "选、用、育、留、汰" | **逐字放慢**，体系感 | 专业度 |
| "集团制度中心" | **加重**，成果亮点 | 价值证明 |

**⚠️ 关键技巧**：这是老板重点关注的段落，务必把三个问题点讲清楚，价值在"统一"而非"推翻"。

---

### 第三件事：自营团队（85-90秒）

**原文**：
> 第三个层面的问题是**交付标准缺乏参照，不知道天花板在哪**。供应商可能觉得"已经尽力了"，但真的到天花板了吗？我需要建立一个"参照系"来验证。做法是在宿迁组建自营团队，3个月从0到1建成20人团队，过程中尝试了多种招聘路径——社招太慢就果断放弃，转而从周边公司退项释放、对接物流招聘团队、借用科技贷后渠道等多条路径快速完成组建。结果是3个月内达成对标业绩120%。**这个价值不是20人的团队，而是证明了供应商交付标准还有20%的提升空间**，这个结果反向传导给外部2000人规模的供应商，同时自营团队也作为新话术、新流程的试验田，降低试错成本。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "缺乏参照" | **加重**，问题开场 | 结构清晰 |
| "参照系" | 加引号语气，概念清晰 | 思维深度 |
| "社招太慢果断放弃" | 语速稍快，体现决断 | 行动力 |
| "120%" | **最慢语速+最重语气** | 核心数字 |
| "不是...而是..." | 对比句式，先抑后扬 | 价值提炼 |
| "20%的提升空间" | 加重，杠杆效应 | 影响范围 |

---

### 第四件事：线路危机（85-90秒）

**原文**：
> 还有一个补位的事情。去年7月运营商突然压降客诉量，线路供给降至日常的30%，业务面临停摆风险，预估每周损失37亿GMV。线路管理不是我原本的职责范围，但我之前在云业务做了4年，积累了运营商资源。危机时刻，我主动承担线路应急调度，对接云那边的运营商组C2，协同拉通三大运营商资源，建立资源价值判断标准，核心资源超配150%作为储备。3周内业务完全恢复正常，是行业里最早恢复的。表面看是解决了一次危机，深层看是**把资源管控能力转化成了市场竞争优势**——同行还在中断的时候，京东已经恢复运营。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "补位的事情" | 轻描淡写带过 | 不抢功 |
| "37亿GMV" | **加重**，危机 stakes | 紧张感 |
| "不是我原本的职责范围" | 语气真诚，放低姿态 | 谦虚 |
| "4年积累的运营商资源" | **放慢+自信**，长期主义 | 能力铺垫 |
| "主动承担" | **加重**，有担当 | 品质体现 |
| "行业最早恢复" | 语气笃定，从容 | 结果导向 |
| "资源管控能力转化成市场竞争优势" | **最慢语速+最重语气** | 战略升华 |

---

### 未来展望（80-85秒）

**原文**：
> 这三件事做完，供应商管理体系运转起来了。但我发现一个潜在的矛盾——持续引入和汰换，会不会让供应商池越来越小？如果腰部供应商活不下去，这套机制还能持续吗？所以下一步的重点是**BPO联盟机制**——从"筛选合格者"转向"培养合格伙伴"。给核心供应商提供非物质资源支持，输出京东的运营体系、培训资源、数据洞察，帮助他们提升能力。不是降低标准，而是让更多供应商有能力达到标准。这是从"交易关系"转向"生态关系"。预计2026年Q2运转，覆盖10家核心供应商。个人层面，想把过去几年的经验通过AI Agents变成智能化工具，让供应商管理少一些对"人"的依赖，多一些对"体系"的依赖，同时做好传帮带。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "潜在的矛盾" | **停顿1秒**，制造转折 | 思维深度 |
| "筛选合格者转向培养合格伙伴" | 对比语气，展示战略思维 | 格局提升 |
| "交易关系转向生态关系" | **放慢**，格局升华 | 战略视野 |
| "AI Agents" | 语气带点兴奋，面向未来 | 展现激情 |
| "少一些...多一些..." | 对比句式，节奏美感 | 逻辑清晰 |

---

### 收尾（70-75秒）

**原文**：
> 回顾这一年，说到底是在做三件事：**找对人**——把好供应商准入关，引入优质新商并支持他们快速达标；**立好规矩**——把不同团队的管理习惯统一成清晰的制度，让每个人知道该干什么；**让能力长在组织上**——把个人经验沉淀成可复用的流程、标准和参照系。但这只是第一阶段。**供应商管理的终极问题不是"选谁"，而是"如何让更多人有资格被选中"**。接下来会在BPO联盟这个新方向上继续探索，把基础打扎实。谢谢各位领导。

**节奏要点**：

| 位置 | 处理方式 | 效果 |
|------|----------|------|
| "说到底就是三件事" | **停顿2秒**，全场安静 | 制造期待 |
| "找对人" | **逐字加重，最慢语速** | 首尾呼应 |
| "立好规矩" | **逐字加重，最慢语速** | 首尾呼应 |
| "让能力长在组织上" | **逐字加重，最慢语速** | 全文高潮 |
| "只是第一阶段" | 转折，引出未来 | 格局提升 |
| "如何让更多人有资格被选中" | **最重语气**，金句收尾 | 余韵悠长 |
| "谢谢各位领导" | 微微鞠躬 | 礼貌结束 |

## 应急技巧

| 情况 | 应对方式 |
|------|----------|
| **忘词了** | 停顿，喝口水，微笑。不要"呃...啊..." |
| **被打断提问** | 先听完，"这是个好问题"，再回答 |
| **时间不够（剩余2分钟）** | 跳过未来展望细节，直接收尾金句 |
| **时间不够（剩余1分钟）** | 只讲三句话收尾，保证完整闭环 |
| **评委表情严肃** | 不受影响，保持自己的节奏 |
| **设备故障** | "那我脱稿说吧"，展现应变能力 |

## 最后提醒

1. **背熟开场70秒**——开场稳了，后面就顺了
2. **金句"三件事"要练到肌肉记忆**——找对人，立好规矩，让能力长在组织上
3. **关键数字要清晰**：3个问题点、120%、37亿GMV
4. **重点段落是"制度统一"**——务必把三个问题点讲清楚
5. **过渡钩子"潜在的矛盾"是关键**——自然引出BPO联盟
6. **收尾金句"如何让更多人有资格被选中"是升华**——不要跳过
7. **提前到场适应场地**——站在讲台上找一下感觉
8. **答辩环节不要抢话**——等评委问完，思考1秒再答
