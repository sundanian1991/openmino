## 答辩高频问题速答

**Q：为什么制度统一花主要精力？**
→ 四团队合并+三套问题（版本老旧、标准不一、效率低）+最难是让制度被接受

**Q：赛马机制是你设立的吗？**
→ 不是，我在引入环节花心思，支持新商快速达标，8家里6家进前三

**Q：自营团队为什么一定要建？**
→ 缺乏参照系，需要控制变量验证标准，120%证明还有20%空间

**Q：线路危机为什么是你上？**
→ 非我职责+有云业务4年资源+业务停摆损失比个人风险大

**Q：BPO联盟和现有机制矛盾吗？**
→ 不矛盾，是进化。从筛选到培养，标准不降，让更多人有能力达标
