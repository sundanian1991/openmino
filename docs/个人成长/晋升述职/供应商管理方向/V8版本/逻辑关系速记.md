```

### 三件事的递进关系

```

把好入口关             让管理有章可循          让标准可验证
8家引入6家TOP         四套→一套              120%证明空间

## 制度统一三个问题点（必背）

```
1. 制度版本老旧（三年前版本）
2. 标准不统一（四套标准）
3. 流程效率低（线下邮件，无线上化）

对应解法：
1. 编写新指引（与时俱进）
2. 统一标准（横向对标）
3. 线上化（流程提效）
```

## 答辩Q&A预判速记

### Q1：为什么制度统一花主要精力？
**速记**：四团队合并+四套习惯+三年旧制度+线下流程
**金句**：不是推翻，是翻译；不是否定，是统一

### Q2：自营团队的价值是什么？
**速记**：不是20人团队，是证明20%提升空间
**金句**：参照系的价值，反向传导2000人

### Q3：线路危机为什么你能解决？
**速记**：非我职责+有云业务4年资源+主动补位
**金句**：业务停摆损失比个人风险大

### Q4：BPO联盟和现有机制矛盾吗？
**速记**：不矛盾，是进化+从筛选到培养+标准不降
**金句**：不是降低标准，是让更多人有能力达到标准
