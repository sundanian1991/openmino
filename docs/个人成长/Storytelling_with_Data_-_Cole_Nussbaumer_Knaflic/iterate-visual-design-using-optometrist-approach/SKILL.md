# Iterate Visual Design Using Optometrist Approach

Apply this skill during the refinement phase of any data visualization, graph, chart, or slide when you need evidence-based decisions about design improvements.

## When to Use
- You have an initial draft but are uncertain about optimal design choices
- Multiple small tweaks could improve clarity (e.g., color, label placement, gridlines)
- You can duplicate and preserve versions during editing
- Time allows for iterative refinement

## How to Execute
1. Create **Version A** (your current best draft).
2. Duplicate it as **Version B**.
3. In Version B, make **only one intentional modification** (e.g., change bar color, remove y-axis line, adjust title position).
4. Place Version A and Version B **side-by-side**.
5. Ask: “Which version better supports my message and is easier to understand?”
6. Select the superior version as your new baseline.
7. Repeat steps 2–6 iteratively—always changing only one element per cycle.
8. Stop when no further improvements are evident or time runs out.
9. Always retain prior versions to enable reverting if needed.

> **Note**: This method prevents conflating multiple changes and ensures each decision is grounded in direct visual comparison.
