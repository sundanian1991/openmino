# Enforce Zero Baseline Rule for Bar Charts

## When to Use
Use this skill:
- When designing or auditing any bar chart (vertical, horizontal, or column) that compares categorical magnitudes
- If the chart represents data as bar lengths
- When verifying ethical and accurate data visualization practices

Do NOT use for line graphs, scatterplots, or other non-bar chart types.

## How to Execute
1. **Set the baseline to zero**: Always start the value axis (y-axis for vertical bars/columns, x-axis for horizontal bars) at 0.
2. **Reject truncated axes**: Never crop the axis to exaggerate differences between bars.
3. **Correct misleading charts** by:
   - Extending all bars down to the zero baseline
   - Moving axis labels to the left side (for vertical charts) so viewers interpret scale before seeing data
   - Adding data labels inside bars if needed to reduce visual clutter
   - Removing the axis entirely only if every bar is clearly labeled with its exact value
4. **Apply ethical judgment**: Manipulating baselines to "emphasize a point" misleads audiences and undermines credibility.
5. **Remember the exception**: This rule does not apply to line graphs, where position—not length—encodes values. Even then, use non-zero baselines transparently and sparingly.

> **Key Principle**: Human perception interprets bar length from a common origin. If that origin isn’t zero, the visual comparison becomes mathematically false.
