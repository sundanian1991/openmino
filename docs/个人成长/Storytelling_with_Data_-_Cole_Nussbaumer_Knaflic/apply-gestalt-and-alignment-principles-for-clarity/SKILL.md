# Apply Gestalt and Alignment Principles for Clarity

Use this skill during the design or editing of graphs, tables, dashboards, slides, or infographics that contain multiple elements and aim to communicate structure intuitively.

## When to Use
- The layout feels messy or uncomfortable to view
- You need to group related items without heavy borders or gridlines
- Audience attention must be guided through implicit visual structure
- You’re working in left-to-right reading contexts (adapt for RTL if needed)

## How to Execute
1. **Apply Gestalt principles** to replace explicit clutter with perceptual grouping:
   - **Proximity**: Place related items close; increase spacing between unrelated groups
   - **Similarity**: Use consistent color, shape, or size to indicate relatedness
   - **Enclosure**: Use subtle background shading instead of borders
   - **Closure**: Remove outer frames—audiences perceive completeness
   - **Continuity**: Align elements along smooth paths; remove unnecessary axis lines
   - **Connection**: Use lines (e.g., in line graphs) to strongly associate elements
2. **Optimize alignment**:
   - Left-justify text and titles for clean vertical edges
   - Upper-left-justify legends and labels to match natural eye path
   - Avoid diagonal text (slows reading significantly)
   - Use software grids for precise alignment
3. **Use white space intentionally**:
   - Keep margins clear
   - Isolate critical elements (e.g., a single number) with surrounding space
   - Never fill empty space just because it exists
4. **Combine both**: Aligned elements within generous white space create intentional, comfortable design.

> **Note**: Eliminate redundant cues (e.g., gridlines + borders + color coding for same grouping).
