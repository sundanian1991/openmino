## 发送邮件

### 方式一：Python 脚本（推荐）

```python
import smtplib
from email.mime.text import MIMEText

smtp_server = 'smtp.yeah.net'
smtp_port = 465
username = 'sundanian@yeah.net'
password = 'GFTHt35ZqjdxFUfP'

content = """你的邮件内容"""

msg = MIMEText(content, 'plain', 'utf-8')
msg['Subject'] = '邮件主题'
msg['From'] = username
msg['To'] = '收件人邮箱'

server = smtplib.SMTP_SSL(smtp_server, smtp_port)
server.login(username, password)
server.sendmail(username, ['收件人邮箱'], msg.as_string())
server.quit()
```

### 方式二：macOS Mail + AppleScript

```applescript
tell application "Mail"
    set newMessage to make new outgoing message with properties {subject:"主题", visible:false}
    tell newMessage
        make new to recipient at end of to recipients with properties {address:"收件人@域名.com"}
        set content to "邮件内容"
    end tell
    send newMessage
end tell
```

## 快速发送命令

```bash

from email.mime.text import MIMEText

content = open('文件路径').read()
msg = MIMEText(content, 'plain', 'utf-8')
msg['Subject'] = '主题'
msg['From'] = 'sundanian@yeah.net'
msg['To'] = '收件人@域名.com'

server = smtplib.SMTP_SSL('smtp.yeah.net', 465)
server.login('sundanian@yeah.net', 'GFTHt35ZqjdxFUfP')
server.sendmail('sundanian@yeah.net', ['收件人@域名.com'], msg.as_string())
server.quit()
print('发送成功')
EOF
```

## 故障排查

### 问题：发送失败

1. 检查密码是否正确
2. 检查网络是否能访问 smtp.yeah.net:465
3. 确认邮箱没有异地登录限制

### 问题：收不到邮件

1. 检查垃圾邮件
2. 确认收件人邮箱地址正确
3. 检查 SMTP 是否真正发送成功
