# 洞察2: 「问题-方案-效果-不足」四段式结构

**类型**: playbook
**来源**: 电销复盘文档分析
**置信度**: high
**日期**: 2026-04-08

---

## 核心洞察

每个业务专项都采用统一的四段式结构，这种结构实现了「闭环思维」—— 不仅展示成绩，也诚实面对问题，形成完整的叙事闭环。

---

## 框架结构

```
┌─────────────────────────────────────────────┐
│  核心命题/背景                               │
│  → 为什么要做这件事？                         │
├─────────────────────────────────────────────┤
│  解决思路/核心举措                           │
│  → 做了什么？具体怎么做？                      │
├─────────────────────────────────────────────┤
│  进展与效果验证                              │
│  → 效果如何？数据验证？                        │
├─────────────────────────────────────────────┤
│  不足与下一步                                │
│  → 还有什么问题？下一步计划？                   │
└─────────────────────────────────────────────┘
```

---

## 原文示例拆解

以「供应商联盟专项」为例:

### 1. 核心命题
> 电销采用效果付费模式，供应商人员质量直接决定投产比。当前面临"规模-质量"跷跷板困境：为保规模交付新人成长慢流失大，产能交付不稳定。

**技巧**:
- 用比喻命名困境（"跷跷板"）
- 一句话说清核心矛盾

### 2. 解决思路
> 机制设计：建立供应商联盟，将自建电销验证有效的核心能力标准化封装，通过"京东搭台、供应商互训"模式，降低单家供应商培训成本...

**技巧**:
- 分维度阐述（机制设计、生态运转、知识产权保护）
- 用引号标出核心策略（"京东搭台、供应商互训"）

### 3. 进展与效果
> 培训认证：3月12-13日完成首期联盟培训认证（2天），6家供应商100%通过...
> 效果数据：新人达标周期60天→50天，缩短10天；新人流失率降低15%...

**技巧**:
- 时间+动作+结果
- 用箭头表示变化（60天→50天）

### 4. 不足
> 不足：联盟机制首次运转，供应商落地进度存在差异，部分工具的使用仍需持续跟进验证。

**技巧**:
- 不回避问题
- 指出具体不足（落地进度差异、工具使用）

## 应用指南

### 改写示例: 供应商风险评估

**原文（假设）**:
> 岐力近期业绩波动较大，需要关注。

**改写后**:

**【核心命题】**
岐力本月名产环比下降18%（1850元→1520元），低于大盘平均水平，存在产能下滑风险。

**【解决思路】**
- 数据诊断：分析近30天拨打数据，识别接通率下降为主要原因（45%→38%）
- 现场走访：本周内完成职场走访，排查人员流失/培训问题
- 对标学习：安排岐力负责人参观标杆职场（毅航），学习管理经验

**【效果验证】**
- 短期：两周内接通率恢复至42%以上
- 中期：月底前名产回升至1700元以上

**【不足】**
- 历史数据样本不足，无法准确判断是短期波动还是趋势性下滑
- 岐力管理层配合度存疑，需持续跟进

---

## 为什么这个结构有效

1. **完整性**: 覆盖背景→行动→结果→反思
2. **诚实性**: 主动暴露不足，建立可信度
3. **可复制**: 结构化便于他人理解
4. **闭环性**: 从问题出发，回到问题解决
