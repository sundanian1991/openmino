# 联动业管的切入点：共享数据看板

不是要求配合，而是提供对方也觉得有用的工具。

## Context

供应商管理与业管的工作有重叠（数据相同），需要形成互相促进的局面。

## Implication

跨部门联动不能从"我要你配合"出发，要从"这个工具对你也有用"出发：

| 你的视角 | 业管视角 |
|----------|----------|
| 供应商整体表现、趋势、风险 | 日常执行、问题处理、现场情况 |

**切入点**：先建一份共享数据看板，业管也能用。他们看到供应商是C级 → 知道要重点盯；看到S级 → 知道可以放心。

**节奏**：先降低门槛（共享看板），再逐步深化（联合评分、共同汇报）。
