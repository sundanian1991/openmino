# 技能搜索规范

> 项目时间：2026-03-18

---

## 背景

需要系统化的技能发现机制，避免盲目安装和资源浪费。

---

## 技能市场

**claw123.ai** — OpenClaw 精选技能导航站
- 5000+ 技能
- 中文描述
- 分类索引

---

## API使用

```bash
curl -s https://claw123.ai/api/skills.zh.json
```

**字段说明**：
- `name` — 技能名
- `description_zh` — 中文描述
- `category_zh` — 分类
- `url` — SKILL.md 源地址

---

## 安装流程

1. **搜索**：需要某功能时 → curl 获取技能列表
2. **推荐**：推荐最匹配的 1-3 个
3. **确认**：等年老师确认后再安装
4. **部署**：fetch SKILL.md → 按说明复制到 `.claude/skills/`

**核心原则**：按需安装，不批量装
