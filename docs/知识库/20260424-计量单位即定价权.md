# 计量单位即定价权

谁定义计量单位，谁就掌握定价权和用户感知。AI 行业三套计量体系并存（API 请求次数、Prompt 次数、Token），直接比数字毫无意义，90% 用户踩坑。

## Context

研究国内 AI 大模型厂商从 Coding Plan 到 Token Plan 的定价演变时发现：MiniMax 从 Prompt 计量切换为 Token 计量，Kimi 同样从 Prompt 切换为 Token。这个看似技术性的变化，本质是定价权的争夺。

## Implication

新产品定价时，计量单位的选择决定了：
- **用户感知**：数字大（API 请求次数）感觉划算 vs 数字精确（Token）感觉透明
- **利润空间**：模糊计量（1 Prompt ≈ 15 次调用）有利于厂商，透明计量有利于用户
- **竞争壁垒**：一旦用户习惯了某种计量方式，切换成本很高

规则：模糊计量利于早期获客，透明计量利于长期信任。选择取决于你处在哪个阶段。
