# 管理价值 = 建机制让别人干

从执行者到管理者的关键转变——搭台让别人唱戏。

## Context

BPO联盟需要牵头，但没有经验。核心困惑：怎么让别人干活，自己做协调，别人干好的活算自己的专项贡献。

## Implication

牵头任何专项，核心产出不是"做成了什么"，而是"建了什么机制"：

| 产出物 | 内容 |
|--------|------|
| 议题库 | 要解决哪些问题 |
| 分工表 | 每个议题谁牵头、谁配合、交付什么 |
| 节奏表 | 多久开一次会、会前准备、会后跟进 |
| 成果标准 | 每个议题的交付物定义 |

机制建好了，专项自己转。汇报时说"我搭建了运作机制，明确了N个议题，分配了牵头方"——这就是管理价值。
