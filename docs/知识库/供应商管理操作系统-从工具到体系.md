# 供应商管理的"操作系统"比工具更有价值

打分是手段，差异化管理动作才是目的。

## Context

之前做了 ABC 打分模板（60:30:10权重，13项指标），但工具本身不是管理亮点。Q2 绩效面谈后，需要从"工具"升级到"体系"。

## Implication

真正的价值链条是：

> 打分 → 分级 → 差异化管理策略 → 资源分配决策 → 可汇报的体系化成果

以后做任何管理体系建设，都要问"这个工具用完之后呢？"——工具是起点，不是终点。

**汇报故事线**：不要说"我做了打分模板"，要说"我建了管理系统，实现了从人盯人到体系管人的转变"。
