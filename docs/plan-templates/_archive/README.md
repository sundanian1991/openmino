# 归档 — 旧体系模板（已废弃）

> 旧版 Plan 模板体系（v1.0），基于早期工具箱版本构建。
>
> **状态**：已废弃，仅供历史回溯参考
>
> **当前标准**：请使用根目录下的 A/B/C/D/E 系列 22 个模板（v2.0）

---

## 旧体系说明

旧体系采用三层次分类：
- **Q卡**（快速判断卡）：7个，30秒级响应
- **D框架**（决策思考框架）：8个，15分钟级深度分析
- **N框架**（周期性叙事框架）：3个，月/季度级叙事构建

共 18 个模板，覆盖了当时识别出的核心场景。

---

## 为何废弃

1. **命名冲突**：旧体系 D1-D8 与新体系 D1-D4 编号冲突，易用错
2. **覆盖不全**：旧体系缺失新供应商初筛、新项目对接、制度流程梳理等场景
3. **硬知识分散**：旧模板未严格引用工具箱章节号，新模板每个都标注 `工具箱§X`
4. **结构不统一**：旧模板格式不统一，新模板统一为"触发信号→核心问题→思考步骤→关键变量→常见误区→决策输出→下一步衔接"7段式
5. **深度不一致**：旧体系 Q 卡过简（无思考步骤），D 框架过深（15分钟），新体系根据场景定制响应深度（D类30秒+5分钟两档，A/B/C类结构化步骤）
