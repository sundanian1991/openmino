# A1-业绩异常判断

> 看到排名下降、周度断崖、峰值比低、头部差距大时，系统化判断异常原因

## 触发信号
- 月度排名：某供应商排名进入后30%（首次预警/连续预警）
- 周度扫描：本周GMV比近4周均值下降>20%（业绩断崖）
- 月度分析：自身峰值比<65% 或 头部差距<60%
- 隐性信号：排名升但人效降（假好）或排名降但人效升（真进步）

## 核心问题
- 这是真异常还是正常波动？
- 异常的根因是什么？（新人稀释/管理松弛/名单问题/产线问题）
- 这个异常的严重等级是什么？（首次预警/连续预警/A级骤降）
- 我下一步应该做什么？（观察/约谈/整改/清退）

## 思考步骤

**Step 1：确认异常真实性（排除干扰）**
```
先看3个基础事实：
1. 数据来源可靠吗？（系统自动数据 vs 供应商填报）
2. 人数变动是否影响？（总人数变化>20%需同时看总量和熟手人均）
3. 新人占比高吗？（新人>30%必须用熟手人均GMV，计算新人拉低幅度）
```
- 若数据可疑 → 触发【A5-合规问题定级】（数据真实性核查）
- 若新人稀释严重 → 标注"新人拉低幅度"，判断是短期现象还是管理问题

**Step 2：定位异常类型（工具箱§十六 + §七）**
```
类型判断矩阵：

【业绩断崖型】（周度↓>20%）
→ 触发：周度扫描信号（工具箱§二）
→ 根因排查：名单问题？出勤突降？离职潮？系统故障？
→ 动作：按A级骤降逻辑立即约谈（不等连续）

【持续下滑型】（连续2个月排名下降）
→ 触发：月度排名趋势
→ 根因排查：管理松弛/新人稀释/产能下降/头部差距拉大
→ 动作：正式约谈，要求整改计划

【A级骤降型】（上月前30%本月后30%）
→ 触发：工具箱§一.1（立即约谈，不等连续）
→ 根因排查：重大事件（主管更换/名单突变/合规事件）
→ 动作：立即约谈查明根因

【隐性退步型】（排名升≥3名 且 人效降≥10%）
→ 触发：排名升但人效降（假好）
→ 根因排查：总量涨但人均降（消化能力不足）
→ 动作：提前预警，关注持续性

【隐性进步型】（排名降≥3名 且 人效升≥10%）
→ 触发：排名降但人效升（真进步）
→ 根因排查：总量降但人均升（可能名单减少但能力提升）
→ 动作：评估是否加资源（头部差距是否缩小？）
```

**Step 3：产能双维度诊断（工具箱§七）**
```
计算两个维度：
1. 自身峰值比 = 本月熟手人均GMV ÷ 近6个月最高熟手人均GMV
   >85%：正常发挥
   65%-85%：有下滑空间，关注趋势
   <65%：严重未发挥，内部出问题

2. 头部差距 = 本月熟手人均GMV ÷ 同产线第1名熟手人均GMV
   >80%：接近头部水平
   60%-80%：有差距但不大
   <60%：跟头部差距显著

交叉判断：
┌─────────────┬─────────────┬─────────────────────────────┐
│ 自身峰值比 │ 头部差距   │ 类型   │ 解读               │
├─────────────┼─────────────┼─────────────────────────────┤
│ 高（>85%） │ 高（>80%） │ 标杆型 │ 自己发挥好，也跟得上头部 │
│ 高          │ 低（<60%） │ 尽力型 │ 已尽力，但产线/产品人效偏低 │
│ 低（<65%） │ 低          │ 问题型 │ 自己没发挥好，差距也在拉大 │
│ 低          │ 高          │ 暂时波动 │ 短期因素，底子不差         │
└─────────────┴─────────────┴─────────────────────────────┘

→ 问题型（双低）：立即约谈，定位内部管理问题
→ 尽力型（高/低）：评估产线适配性，是否需调整名单/产品
→ 暂时波动（低/高）：观察2周，看是否恢复
→ 标杆型（高/高）：标杆分享，不干预
```

**Step 4：主管胜任评估（工具箱§八）**
```
5项指标逐项核对：
1. 团队稳定度：流失率 <8%/月？
2. 出勤稳定度：有效出勤率 >90%？
3. 人效趋势：近3个月人均GMV不连续2个月下降？
4. 响应质量：预警后48h内回复 >80%？
5. 整改执行力：承诺动作按期完成 >70%？

达标项数：
5项 → 胜任，问题在业务/名单层面
3-4项 → 需关注，约谈时针对性提问管理问题
≤2项 → 不胜任信号，约谈时谈管理团队调整
```

**Step 5：综合判断与动作决策**
```
根据以上分析，填入判断卡：

【异常类型】：[业绩断崖/持续下滑/A级骤降/隐性退步/隐性进步]
【触发状态】：[首次预警/连续预警/A级骤降/无触发]
【根因定位】：[新人稀释/管理松弛/名单问题/产线问题/系统故障/其他]
【双维度类型】：[标杆型/尽力型/问题型/暂时波动]
【主管胜任】：[5项/3-4项/≤2项]
【严重等级】：🔴高（A级骤降/诚信问题）/🟡中（连续预警）/🟢低（首次预警）

下一步动作：
□ 立即约谈（A级骤降/业绩断崖）
□ 正式约谈+整改计划（连续预警）
□ 私下沟通问原因（首次预警）
□ 观察2周（暂时波动）
□ 触发【A9-HRBP式扶持】（排名下滑但未预警）
□ 触发【A5-合规问题定级】（数据异常）
```

## 关键变量
- **触发状态优先级**：A级骤降 > 连续预警 > 首次预警 > 无触发
- **异常幅度**：周度断崖>20% vs 月度排名下滑3名，响应速度不同
- **根因可逆性**：名单问题（甲改）/管理问题（供应商改）/产线问题（需双方调整）
- **主管胜任**：≤2项需考虑管理团队调整
- **时间窗口**：A级骤降必须立即（24-48h内约谈），首次预警可3-5天

## 常见误区
- ❌ 只看总人均GMV不看熟手人均：新人占比高会导致总人均虚低
- ❌ 将问题型误判为产线问题：双维度双低是内部管理问题，不是产线问题
- ❌ A级骤降等待连续3个月：应立即约谈，不等
- ❌ 忽略隐性退步：排名升但人效降，说明总量涨但人均降，消化能力不足
- ❌ 只关注业绩不关注主管胜任：主管能力是持续性问题根因

## 决策输出

**业绩异常判断卡**（200字内）：
```
供应商：[XXX]
异常信号：[周度断崖↓25% / 月度首次预警 / A级骤降]
触发状态：[首次预警]
双维度：[自身峰值比62%（低）+ 头部差距55%（低）→ 问题型]
根因判断：[新人占比48%导致人效稀释 + 出勤率下滑至84%]
主管胜任：[3项达标，流失率12%不达标]
严重等级：🟡中（连续预警）
下一步：[立即约谈，要求下周提交整改计划，重点关注新人培养和出勤恢复]
```

## 下一步衔接
- 首次预警 → 调用【A2-约谈准备】→ 私下沟通问原因
- 连续预警 → 调用【A2-约谈准备】→ 正式约谈+整改计划 → 触发【A7-整改计划评估】
- A级骤降 → 调用【A2-约谈准备】→ 立即约谈（24-48h内）
- 主管胜任≤2项 → 约谈时加入管理团队调整议题
- 数据异常 → 调用【A5-合规问题定级】
