# B6-降价谈判准备

> 与供应商谈判价格时，系统化准备底线、策略和谈判要点

## 触发信号
- 年度合同续签：价格重新谈判
- 分量调整：增量时供应商要求涨价
- 成本优化：主动要求供应商降价（年度降本X%）
- 竞争对手降价：其他供应商报价更低，需要匹配

## 核心问题
- 谈判目标是什么？（降本X% / 维持现价 / 接受涨价）
- 供应商的成本结构是什么？合理利润空间多少？
- 我的底线是什么？（低于X%无法接受）
- 谈判策略是什么？（先礼后兵/逐步让步/备选威胁）
- 谈判后如何验证？合同怎么签？

## 思考步骤

**Step 1：明确谈判背景与目标**
```
谈判类型：
□ 年度续签降价：每年固定周期，目标降本5-10%
□ 增量压价：新增份额时要求价格优惠
□ 被动应对：供应商主动提价，需还价
□ 竞争对标：其他供应商报价更低，要求匹配

目标量化：
- 理想目标：降价[XX%]
- 可接受目标：降价[X%]
- 底线：降价<[X%] 或 维持现价
- 备选方案：如果谈不拢，备选供应商是谁？（中部供应商[X]）

成本分析：
当前单价：[XXX元/人/月]
当前总量：[XXX人] → 月成本：[XXX万元]
目标单价：[YYY元/人/月] → 月节省：[ZZZ万元/年]
```

**Step 2：供应商成本结构与利润分析（工具箱§十九）**
```
成本构成（估算）：
1. 人力成本：薪资+社保+福利 [占比约60-70%]
   - 人均薪资：[XXX元]
   - 社保公积金：[XXX元]
   - 其他福利：[XXX元]

2. 场地成本：租金+水电+物业 [占比约10-15%]
   - 人均场地成本：[XXX元]

3. 管理成本：主管/行政/培训/IT [占比约10-15%]
   - 人均管理成本：[XXX元]

4. 利润空间：行业平均净利率 [约5-10%]
   - 当前利润率：[估计XX%]

供应商报价合理性判断：
□ 报价明显高于行业：有降价空间
□ 报价接近行业：需从量上争取
□ 报价已很低：维持或少量降价，强调长期合作

供应商痛点识别（谈判筹码）：
□ 该供应商是否依赖我们？（我方业务占比>50%？）
□ 该供应商是否有空余名额？（产能富余可增量换降价）
□ 该供应商是否在爬坡期？（需扶持，可承诺增量换优惠）
□ 该供应商是否有竞品威胁？（备选供应商是否可用？）
```

**Step 3：谈判策略与话术设计**
```
策略矩阵（根据供应商类型）：

头部供应商（S/A级）：
策略：尊重+长期合作+增量换优惠
话术：
"你司是我们头部合作伙伴，希望长期共赢。今年我们业务预计增长[XX%]，如果你能在价格上支持[降X%]，我们可承诺[增量Y%]，实现双赢。"
底线：降价3-5%或维持现价+增量承诺

中部供应商（B级）：
策略：对标+竞争压力+增量条件
话术：
"我们对比了同类型供应商[竞品名]，他们的报价是[XXX]。我们希望保持合作，但价格需匹配市场。降价[X%]可考虑增量。"
底线：降价5-8%

尾部供应商（C/D级）：
策略：业绩导向+明确条件
话术：
"你的排名在后30%，我们考虑清退。如果你能承诺[排名进前70%]且降价[X%]，我们可继续合作。否则将引入新供应商。"
底线：降价8-15%或触发清退

供应商主动提价：
策略：先问原因+数据反驳+备选威胁
话术：
"理解你的成本压力，但我们看行业整体成本是降的。你的报价比市场高[X%]，我们难以接受。如果无法维持现价，我们需评估其他选项。"

谈判让步策略：
让步原则：每次让步都要换取对方承诺
让步阶梯：[理想→可接受→底线]，分2-3次逐步释放
每次让步对应：对方需承诺[具体动作，如：降价X%+保证交付+整改计划]
```

**Step 4：谈判团队与流程**
```
参与人：
□ 我方：年老师（主谈）+ [易人哥/军哥，视金额而定]
□ 对方：供应商[老板/主管]
□ 时间：建议45-60分钟

谈判流程：
1. 开场（5min）：表明来意 + 共同目标（长期合作）
2. 我方立场（10min）：展示数据分析 + 我方目标
3. 供应商回应（15min）：听对方说，不打断
4. 讨价还价（20min）：逐步让步，交换条件
5. 达成共识（5min）：确认降价幅度 + 承诺事项
6. 后续（5min）：合同签署 + 执行时间点

关键红线：
□ 不口头承诺：任何让步必须书面确认
□ 不单独决策：大额降价需上级在场或事后确认
□ 不情绪化：保持专业，数据说话
```

**Step 5：谈判结果与执行**
```
结果三分类：
1. 达成协议：
   - 降价幅度：[X%]
   - 生效时间：[YYYY-MM]
   - 附加条件：[供应商承诺XXX]
   - 输出：合同补充协议 / 分量调整通知

2. 暂时僵局：
   - 双方差距：[我方要X% vs 对方愿Y%]
   - 下一步：冷却[1-2天]后二次谈判
   - 备选启动：启动备选供应商评估

3. 谈判破裂：
   - 原因：[价格分歧过大/对方不接受条件]
   - 动作：启动清退评估（工具箱§一）或引入新供应商（B1）
   - 业务切换：T+[XX]天过渡期

验证机制：
□ 降价后跟踪：未来3个月成本是否真降？
□ 供应商履约：是否保证交付质量？（不因降价降低服务）
□ 承诺事项：对方承诺是否兑现？（如：排名提升）
```

## 关键变量
- **供应商等级**：头部（降3-5%）/ 中部（降5-8%）/ 尾部（降8-15%+业绩条件）
- **业务占比**：我方业务占供应商>50% → 有更强议价权
- **备选可用性**：有备选则压力更大，无备选则需谨慎
- **利润空间**：行业平均净利率5-10%，低于5%难再降
- **谈判时机**：年度续签（固定周期）> 分量调整（增量换优惠）> 被动应对（防御）

## 常见误区
- ❌ 只谈价格不谈条件：降价必须换取对方承诺（业绩/交付/整改）
- ❌ 情绪化对抗：供应商一强硬就退让，或激化矛盾
- ❌ 无底线设定：被供应商牵着走，没有明确底线
- ❌ 不记录：口头达成无书面确认，后续扯皮
- ❌ 忽视备选：有备选不用，失去谈判筹码
- ❌ 降价不跟踪：降了价但供应商降低服务质量，需验证

```
供应商：[XXX]
类型：[头部/中部/尾部]
当前单价：[XXX元]
当前月成本：[XXX万元]
我的目标：降价[ X% ] → 年省[XXX万元]
可接受：降价[ X% ]
底线：降价<[X%] 或 维持现价

供应商痛点：
□ 依赖我们（业务占比XX%）
□ 产能富余（空余名额XX人）
□ 需扶持（爬坡期）
□ 有竞品威胁（备选[YYY]可用）

谈判策略：[尊重+增量 / 对标+竞争 / 业绩导向]
底线守护：[任何让步需书面确认 / 大额需上级在场]
备选方案：[如果谈崩，启动[ZZZ]供应商引入]
```

**谈判 checklist**：
```
谈判前：
□ 成本分析完成（人力/场地/管理/利润）
□ 目标/可接受/底线三档明确
□ 备选供应商评估完成（如需要）
□ 谈判策略与话术准备
□ 参与人确认（我方+对方+时间）

谈判中：
□ 保持专业，数据说话
□ 每次让步换取对方承诺
□ 不单独决策，大额需上级
□ 记录对方承诺点

谈判后：
□ 24h内发出确认邮件（降价幅度+生效时间+承诺事项）
□ 合同补充协议签署
□ 系统价格配置更新
□ 未来3个月跟踪验证（成本+质量）
```

**降价通知（给供应商）**：
```
[供应商名]，你好。
经双方协商，就[项目名]价格调整达成如下协议：
原单价：[XXX元/人/月]
新单价：[YYY元/人/月]（降幅[Z%]）
生效时间：[YYYY-MM-01]
请你司确认并签署补充协议。
同时，我方期望：[供应商承诺事项，如：交付质量保证/排名提升目标]
```
