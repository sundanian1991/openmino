# 麦肯锡图表工具使用指南

> TR Strategy Chart Suite v2.8 | 18 种策略图表一键生成

---

## 30 秒上手

1. 双击 `麦肯锡图表.html` 打开（需要联网）
2. 左侧选图表类型 → 自动加载模板数据
3. 把你的 Excel 数据复制粘贴到数据框（Tab 分隔，直接从 Excel 复制即可）
4. 点「确认并刷新图表」
5. 右上角导出 4K PNG 或 SVG

---

## 第一部分：图表选型决策表

按**你要表达什么**选图，不按个人喜好。

| 你要表达什么 | 选什么图 | 你的场景举例 |
|-------------|---------|-------------|
| 从 A 到 B 的过程拆解 | **瀑布图** | 供应商成本拆解、利润桥接 |
| 各部分对总体的贡献 | **堆叠柱状图** | 区域贡献、渠道占比 |
| 两组/三组数据的同期对比 | **分组柱形图** | 供应商 YoY 对比 |
| 实际 vs 目标的偏差 | **差值箭头图** | 预算达成、KPI 偏差 |
| 两个群体的结构对比 | **蝴蝶图** | 供应商 A vs B 能力对比 |
| 战略四象限分析 | **SWOT** | 竞争态势、供应商定位 |
| 趋势 + 另一个指标的叠加 | **折线+柱形组合图** | 成本+增长率 |
| 横向排名（标签文字长） | **分组条形图** | 部门/供应商横向排名 |
| 多维能力评估 | **雷达图** | 供应商能力六维对比 |
| 市场定位矩阵 | **气泡矩阵** | 供应商份额×增长×规模 |
| 资金/流量流向 | **桑基图** | 成本流向、人效分配 |
| 单一指标趋势 | **折线图** | 月度走势 |
| 单一指标排名 | **柱状图** | 供应商排名 |
| 构成占比（≤5项） | **饼图** | 业务构成 |
| 两变量相关性 | **散点图** | 客流量 vs 转化率 |
| 三维对比 | **分组气泡图** | 供应商多维打分 |
| 层级占比 | **旭日图** | 组织→团队→个人 |
| 占比方块图 | **矩形树图** | 预算分配 |
| 渠道贡献趋势 | **堆叠面积图** | 流量来源月度变化 |

## 第二部分：数据准备速查

### 通用规则

- 第一行是表头，后续是数据行
- 支持 CSV（逗号分隔）和 TSV（Tab 分隔），不要混用
- **直接从 Excel 复制粘贴即可**（Excel 复制 = Tab 分隔）
- 数字自动识别，文字保持原样

### 每种图的数据格式

**瀑布图**（收入/成本拆解）：
```
项目,数值
2023年营收,100
价格效应,20
销量效应,-10
产品结构优化,5
汇率影响,-8
2024年营收,0        ← 末行填 0，工具自动算出总量
```

**差值箭头图**（实际 vs 预算）：
```
项目,实际值,预算值
销售额,120,100
成本,85,80
```
自动显示标签：`120\n(+20.0%)`

**蝴蝶图**（双群体对比）：
```
类别,群体A,群体B
18-24岁,35,40
25-34岁,50,45
```
两侧都填正数，工具自动处理对称显示。

**SWOT**（纯文本格式，不是表格）：
```
S: 优势1, 优势2, 优势3
W: 劣势1, 劣势2
O: 机会1, 机会2
T: 威胁1, 威胁2
```

**桑基图**（流向）：
```
来源,去向,数值
总收入,主营业务成本,40
总收入,毛利润,60
毛利润,运营支出,30
```
自动提取节点，不需要单独定义。

**气泡矩阵**（波士顿矩阵）：
```
产品,市场份额(%),年增长率(%),营收规模
产品A,25,15,100
产品B,10,5,50
```
三列数据分别映射到 X轴、Y轴、气泡大小。

**旭日图**（层级占比）：
```
层级1,层级2,数值
全球,北美,40
全球,欧洲,30
北美,美国,30
北美,加拿大,10
```
自动构建树形结构。

## 第三部分：调参到汇报级

默认参数够用，但汇报/PPT 级需要调这些：

| 参数 | 默认 | 汇报建议 | 什么时候调 |
|------|------|---------|-----------|
| 画布尺寸 | 1000×600 | PPT: 1280×720 或 1920×1080 | 嵌入 PPT 前必须调 |
| 标题字号 | 20 | 16-22 | 画布小时降到 16 |
| 图例位置 | 底部居中 | 底部居中 | 系列少（≤2）可改右上 |
| 网格线 | 虚线 | 虚线（最专业） | 极简风格选「无」 |
| 数值标签 | 显示 | 关闭 | 超过 8 个柱体时建议关 |
| 柱体宽度 | 50% | 40-60% | 多系列时缩小 |
| 分类间距 | 20% | 15-30% | 数据密降到 10% |
| X轴旋转 | 0° | 45° | 标签文字重叠时 |
| 排序 | 原始顺序 | 降序 | 展示 Top N 排名时 |
| Y轴刻度 | 自动 | **起始必须填 0** | 柱状图的铁律 |

**Y轴起始 = 0 是柱状图铁律**。不归零会放大差异，汇报会被质疑专业性。

## 第四部分：导出适配

| 输出场景 | 格式 | 画布尺寸 | 操作 |
|---------|------|---------|------|
| PPT 嵌入 | 4K PNG | 1920×1080 或 1280×720 | 点「导出 4K PNG」 |
| 印刷/矢量编辑 | SVG | 任意 | 点「导出 SVG」，无损 |
| 邮件/即时通讯 | 4K PNG | 800×500 | 缩小画布再导出 |
| 报告文档 | SVG | 1000×600 | SVG 嵌入 Word 可无损缩放 |

## 第五部分：常见问题排查

| 症状 | 原因 | 解决 |
|------|------|------|
| 粘贴数据后图表空白 | 分隔符混用（逗号和 Tab 混在一起） | 清空数据框，重新从 Excel 复制粘贴 |
| 标签文字重叠 | X轴标签太长 | X轴旋转调 45° 或 90° |
| 图例遮住数据 | 图例位置和数据区域冲突 | 图例改「底部居中」 |
| 柱体太窄/太宽 | 系列多+间距大 | 柱体宽度调 60-70%，间距降到 10% |
| Y轴起点不是 0 | 自动计算会取数据下限 | Y轴「起始」手动填 0 |
| 导出图片模糊 | 用了截图而非导出功能 | 必须用右上角「导出 4K PNG」 |
| SWOT 不显示 | 格式错误 | 检查是否用 S:/W:/O:/T: 前缀，冒号后直接写内容，多项用逗号+空格分隔 |
| 瀑布图总量不对 | 末行填了非零值 | 末行数值必须填 0 |
| 桑基图线条混乱 | 来源/去向名称拼写不一致 | 确保节点名称完全一致（大小写、空格） |
| 差值图百分比不对 | 实际/预算列顺序反了 | 确保列顺序：项目, 实际值, 预算值 |

## 第六部分：供应商管理场景速查

| 汇报主题 | 推荐图表 | 数据列 |
|---------|---------|--------|
| 供应商绩效排名 | 柱状图（降序） | 供应商名称, 综合得分 |
| 业绩同比 | 分组柱形图 | 供应商名称, 2024年得分, 2025年得分 |
| 成本结构 | 瀑布图 | 成本项目, 金额 |
| 预算达成率 | 差值箭头图 | 供应商名称, 实际值, 预算值 |
| 供应商能力对比 | 雷达图 | 评估维度, 供应商A得分, 供应商B得分 |
| 市场定位 | 气泡矩阵 | 供应商名称, 市场份额(%), 增长率(%), 管理人数 |
| 成本流向 | 桑基图 | 来源, 去向, 金额 |
| 供应商 SWOT | SWOT | S/W/O/T 纯文本 |
