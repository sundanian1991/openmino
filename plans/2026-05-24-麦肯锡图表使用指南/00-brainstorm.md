# 麦肯锡图表工具 — 头脑风暴

## 核心问题

年老师有一个功能丰富的策略图表生成器，想知道怎么用好它。这不是"功能列表"问题，而是"什么场景用什么图、数据怎么准备、输出怎么调到汇报级别"的实操问题。

---

## 工具能力盘点

### 18 种图表类型

| 分类 | 图表 | 核心用途 |
|------|------|---------|
| **战略** | 瀑布图 | 收入拆解、利润桥接 |
| | 堆叠柱状图 | 区域/产品贡献对比 |
| | 分组柱形图 | 同期对比（QoQ/YoY） |
| | 差值箭头图 | 实际 vs 预算偏差分析 |
| | 蝴蝶图 | 双群体对比（性别/竞品） |
| | SWOT | 战略四象限 |
| **组合** | 分组条形图 | 横向对比（适合长标签） |
| | 折线+柱形 | 量+率的组合展示 |
| **高级** | 堆叠面积图 | 渠道/来源贡献趋势 |
| | 分组气泡图 | 三维对比（GDP/增长率/人口） |
| | 散点图 | 两变量相关性 |
| | 矩形树图 | 预算/份额占比 |
| | 桑基图 | 资金/流量流向 |
| | 旭日图 | 层级占比 |
| | 雷达图 | 多维能力对比 |
| | 气泡矩阵 | 波士顿矩阵（份额×增长） |
| **基础** | 柱状图 | 单指标排名 |
| | 折线图 | 趋势 |
| | 饼图 | 构成占比 |

### 控制能力

- 画布尺寸：自定义 px（适配 16:9 PPT、A4 报告）
- 标题：文字/字号/对齐（左/中/右对齐坐标轴）
- 图例：右上/顶部居中/底部居中
- 网格线：虚线/实线/无
- 坐标轴：Y 轴起始/结束/步长，X 轴同
- 视觉：数值标签、标签字号、柱体宽度、分类间距、X 轴旋转、排序
- 导出：4K PNG（pixelRatio 4）、SVG（矢量无损）

## 数据格式规则

### 通用规则
- CSV（逗号分隔）或 TSV（Tab 分隔）均可
- 第一行是表头，后续是数据
- 数字自动识别，文字保持原样
- 可直接从 Excel 复制粘贴（Excel 复制默认是 Tab 分隔）

### 各图表数据结构

| 图表 | 列结构 | 注意事项 |
|------|--------|---------|
| 瀑布图 | 项目, 数值 | 首行=起始总量，末行留0自动算总量 |
| 堆叠柱状图 | 类别, 系列1, 系列2... | 支持排序 |
| 分组柱形图 | 类别, 系列1, 系列2... | 2-3个系列最佳 |
| 差值箭头图 | 项目, 实际值, 预算值 | 自动算差值百分比 |
| 蝴蝶图 | 类别, 左侧, 右侧 | 自动取绝对值对称 |
| SWOT | S:/W:/O:/T: 前缀 | 唯一的纯文本格式 |
| 桑基图 | 来源, 去向, 数值 | 自动提取节点 |
| 气泡矩阵 | 产品, 份额%, 增长率%, 规模 | 波士顿矩阵布局 |
| 旭日图 | 层级1, 层级2, 数值 | 自动构建树形结构 |

## 隐藏技巧（代码级发现）

1. **瀑布图末行填 0**：代码自动累加计算最终总量，不需要手动算
2. **差值箭头图自动算%**：标签显示 `实际值\n(+偏差%)`，不用手动算
3. **蝴蝶图自动取绝对值**：左侧数据自动取负再显示正值，直接填正数即可
4. **分类间距滑块**：0-80% 范围，20% 是默认值，密数据降到 5-10%
5. **标题右对齐+图例右上**：代码有冲突处理，图例自动下移避让
6. **散点图/气泡图双轴网格线**：X/Y 都可以独立控制网格
7. **排序只在柱状图/堆叠柱状图生效**：其他图表类型忽略了排序参数

## 主题与张力

### 主题
1. **图表选型**：什么业务场景选什么图，这是使用效率的核心
2. **数据准备**：Excel → 图表的数据流转效率
3. **汇报级调优**：默认参数到汇报级别的差距在哪
4. **导出适配**：不同输出场景（PPT/报告/邮件）的尺寸和格式选择

### 张力
- **便利性 vs 精细度**：模板数据一键出图，但汇报级需要调参
- **通用性 vs 专业性**：18 种图表覆盖广，但每种深度有限（如饼图只有环形一种）

### 缺口
- 缺少配色方案切换（固定麦肯锡色系）
- 缺少图表组合（如仪表板模式）
- 没有数据校验提示（输错格式会静默失败）
