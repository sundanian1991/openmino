# 芯片供给即定价天花板

谁控制算力供给，谁就控制定价下限。DeepSeek V4 官方直接说"受限于高端算力，预计下半年昇腾 950 批量上市后价格大幅下调"——芯片放量节奏直接绑定 Token 价格。

## Context

研究 AI 定价演变时发现一个被低估的变量：芯片供给。

- 英伟达在中国市占率从 66%(2024) 降到 54%(2025)，全面禁止 AI GPU 对华出口
- 华为昇腾 950PR 4 月量产，算力是 H20 的 2.87 倍，价格仅 1/3
- DeepSeek V4 同时适配华为 + 寒武纪，Day 0 支持
- 2025 年国产芯片自给率 41%，伯恩斯坦预测 2027 年达 55%

## Implication

Token 定价的底层逻辑：
- 算力成本下降 → Token 成本下降空间打开 → 价格战有底气
- 国产芯片替代 → 供给端不再被卡脖子 → 定价策略可以更激进
- 推理专用芯片竞争 → 推理成本加速下降 → Agent 时代经济性改善

规则：观察一个 AI 产品的定价走势，不要只看市场竞争，要看它的芯片供应链。算力供给是定价的硬约束，市场竞争只是软约束。
