# 供应商分层落地 — 任务计划

## Phase 1：公平性分析（1-2小时）

### 1.1 逐维度适用性分析
- [ ] 业绩完成（40%）— 不同业务线基线怎么定？绝对值 vs 达成率
- [ ] 规模（15%）— 大/中/小供应商怎么分组？组内排名还是组间排名？
- [ ] 稳定性（15%）— 新供应商没有历史数据怎么评？用什么替代？
- [ ] 响应速度（10%）— 怎么量化？事件记录 vs 主观打分？
- [ ] 配合度（10%）— 怎么量化？谁来打分？
- [ ] 资源投入（5%）— 怎么衡量？投入金额？投入人力？
- [ ] 行为指标（5%）— 具体包括哪些？数据从哪来？

### 1.2 供应商分类
- [ ] 定义"内部/外部"供应商的分类标准
- [ ] 定义大/中/小供应商的分组标准（人数？业务量？）
- [ ] 定义新/老供应商的分界线（合作时长？）

**验收**：每个维度都有"适用性问题 + 调整方案"的分析表

---

## Phase 2：调整方案设计（1-2小时）

### 2.1 业绩维度调整
- [ ] 按业务线设基线（金条/企金/信用卡/财富各一套标准）
- [ ] 用"达成率"替代"绝对值"，消除规模影响
- [ ] 用"环比改善"作为加分项，鼓励进步

### 2.2 规模维度调整
- [ ] 按人数分组：小型（<100人）、中型（100-500人）、大型（>500人）
- [ ] 组内排名，不跨组比较
- [ ] 或：用"人均产出"替代"总规模"

### 2.3 稳定性维度调整
- [ ] 新供应商（合作<6个月）：用"试用期表现"替代历史数据
- [ ] 老供应商：用流失率 + 公司存续年限

### 2.4 软性指标量化
- [ ] 响应速度：用系统记录的响应时间，不靠主观打分
- [ ] 配合度：用具体事件记录（配合了几次/推诿了几次）
- [ ] 资源投入：用供应商在职场的投入人数/设备/培训次数

**验收**：每个维度都有"原始方案 + 调整后方案 + 调整理由"

---

## Phase 3：落地模板制作（1-2小时）

### 3.1 Excel 评估模板
- [ ] Sheet 1：供应商基础信息（名称、业务线、规模、合作时长）
- [ ] Sheet 2：业绩评分（按业务线自动匹配基线）
- [ ] Sheet 3：规模评分（自动分组 + 组内排名）
- [ ] Sheet 4：稳定性评分（新/老供应商自动切换计算逻辑）
- [ ] Sheet 5：软性指标评分（响应速度、配合度、资源投入、行为指标）
- [ ] Sheet 6：综合评定（自动计算总分、排名、分级）
- [ ] 公式验证：所有公式 0 错误

### 3.2 场景卡片
- [ ] 场景：坐席 C 占比 >30% → 供应商改善计划
- [ ] 场景：供应商连续 2 季度 C 级 → 清退评估
- [ ] 场景：新供应商试用期结束 → 首次评级

**验收**：Excel 模板输入数据后自动出等级，场景卡片可直接使用

## Phase 4：试评验证（1小时）

### 4.1 选 3-5 家供应商试评
- [ ] 选 1 家大供应商（毅航）
- [ ] 选 1 家小供应商（岐力）
- [ ] 选 1 家新供应商
- [ ] 选 1 家问题供应商（C 级坐席占比 >30%）
- [ ] 选 1 家标杆供应商（S 级坐席占比高）

### 4.2 验证结果
- [ ] 分级结果是否合理？
- [ ] 有没有"大供应商天然 S 级"的问题？
- [ ] 有没有"小供应商永远升不上去"的问题？
- [ ] 新供应商是否被公平对待？

**验收**：试评结果分布合理（S 级不超过 15%，C 级不超过 20%）
