# 供应商分层落地 — 需求

## 目标

把老板给的 7 维度供应商分层方案（业绩40%/规模15%/稳定性15%/响应速度10%/配合度10%/资源投入5%/行为指标5%）落地为可执行的评估体系。

核心问题：这套方案对所有供应商是否公平？需要哪些调整才能适用于不同类型、不同规模、不同业务线的供应商？

## 排除项

- 不是重新设计方案，是在老板方案基础上做适配性调整
- 不涉及坐席层面的 SABC 评级（那是另一套体系）
- 不涉及储备库建设（那是 P3）

## 交付物

1. **公平性分析文档** — 7 个维度对不同类型供应商的适用性分析
2. **调整方案** — 每个维度的适配规则（分组/基线/替代指标）
3. **落地模板** — 可直接使用的 Excel 评估模板
4. **场景卡片** — C 级供应商的管理动作（坐席 C 占比触发）

## 完成标准

- [ ] 每个维度都有"对谁有利/对谁不利/怎么调整"的分析
- [ ] 大/中/小供应商、新/老供应商、不同业务线都有公平的评估路径
- [ ] Excel 模板能自动计算，输入数据直接出等级
- [ ] 年老师确认方案可行

## 约束

- 数据来源：坐席 SABC 评级表（已有）、业务系统数据、日常管理记录
- 时间：Q2 内完成首评
- 老板方案为基础，不能推翻
