## 主题识别

### 主题 1: 风格 × 情绪 决策矩阵

**当前问题**：14种风格是「经验分类」，不是「决策逻辑」

**新模型**（二维决策）：

```
            情绪（内容气质）
         紧迫    中性    庆祝    沉思
       ┌───────┬───────┬───────┬───────┐
  数据 │ 数据模式 │ 数据模式 │ 金融时报 │ 数据模式 │
       │ (红)   │ (灰)   │ (青绿) │ (深灰) │
       ├───────┼───────┼───────┼───────┤
  品牌 │ 自信宣言 │ 品牌模式 │ 品牌模式 │ 纸墨文学 │
       │ (黑金) │ (陶土) │ (陶土) │ (墨色) │
       ├───────┼───────┼───────┼───────┤
  咨询 │ 咨询模式 │ 咨询模式 │ 咨询模式 │ 咨询模式 │
       │ (红)   │ (灰)   │ (蓝)   │ (灰)   │
       └───────┴───────┴───────┴───────┘

风格类型（设计系统）
```

**关键洞察**：
- 风格类型 = 设计系统（数据/品牌/咨询 = 3种基础）
- 情绪 = Header颜色变体（红/灰/陶土/青绿/金/墨 = 6种颜色）
- 组合 = 3 × 4 = 12种，比14种更简洁，且有逻辑

### 主题 2: Header颜色动态选择

**当前**：固定颜色（克制专业主义 = 炭黑）

**新逻辑**（内容驱动）：

| 情绪标签 | Header颜色 | 使用场景 |
|---------|-----------|---------|
| 紧迫/警示 | `#8B0000`（深红） | 处罚、风险、异常 |
| 中性/专业 | `#111`（炭黑） | 数据通报、常规汇报 |
| 庆祝/激励 | `#E2725B`（陶土） | 业绩达成、激励海报 |
| 权威/正式 | `#0d7680`（青绿） | 金融数据、战略汇报 |
| 奢华/高端 | `#D4AF37`（金） | 颁奖、年度盛典 |

**决策流程**：
```
内容 → 提取情绪关键词 → 匹配情绪标签 → 确定Header颜色 → 选择基础风格
```

### 主题 3: 统一灰阶系统

**当前**：14种风格各自有灰阶，混乱

**新方案**：所有风格共用「克制灰阶」

```css
:root {
  /* 核心灰阶（所有风格共用）*/
  --gray-50: #FAFAFA;   /* 斑马纹、卡片 */
  --gray-100: #F5F5F5;  /* hover */
  --gray-200: #E5E5E5;  /* 边框 */
  --gray-400: #999999;  /* 次要文字 */
  --gray-600: #666666;  /* 正文 */
  --gray-900: #111111;  /* 主标题、表头 */

  /* 强调色（按情绪选择）*/
  --accent-urgent: #8B0000;   /* 紧迫 */
  --accent-neutral: #E2725B;  /* 中性（陶土）*/
  --accent-celebrate: #D4AF37; /* 庆祝（金）*/
  --accent-authority: #0d7680; /* 权威（青绿）*/
}
```

### 主题 4: better-icons 整合

**当前**：mino-frontend 没提 better-icons

**整合方案**：

```
mino-frontend 生成流程：
1. 内容分析
2. 风格选择
3. 【新增】图标需求识别
4. 【调用】better-icons search <关键词>
5. 【嵌入】SVG inline
6. 生成HTML
```

**图标使用原则**（克制专业主义）：
- 无图标 ＞ 有图标
- 简单几何图标 ＞ 复杂插画
- 手绘风格 ＞ 矢量线条
- 单色 ＞ 多色

### 主题 5: 14种风格简化

**当前14种**：克制专业主义、品牌模式、数据模式、金融时报、咨询模式、自信宣言、现代工坊、分类标签、柔和几何、趣味拼接、复古报刊、极简现代、纸墨文学、Anthropic品牌

**简化方案（5种基础 × 4种情绪 = 20种组合）**：

| 基础风格 | 核心特征 | 情绪变体 |
|---------|---------|---------|
| **数据模式** | 表格、数字、黑白灰 | 紧迫/中性/庆祝/权威 |
| **品牌模式** | 陶土色、手绘图标、人文感 | 紧迫/中性/庆祝/权威 |
| **咨询模式** | So What结论、结构化 | 紧迫/中性/庆祝/权威 |
| **金融时报** | 青绿、KPI网格、信赖感 | 紧迫/中性/庆祝/权威 |
| **极简宣言** | 黑金/黑陶土、大字号、冲击力 | 紧迫/中性/庆祝/权威 |

**删除**：
- 现代工坊、分类标签、柔和几何、趣味拼接、复古报刊、纸墨文学、Anthropic品牌
- 原因：与上述5种重叠或过于细分

## 张力与缺口

### 张力 1: 简化 vs 保留

**问题**：年老师之前用过的风格（如复古报刊）要不要保留？
**解决**：标记为「扩展风格」，不删除，但默认不推荐

### 张力 2: 自动选择 vs 用户控制

**问题**：情绪自动识别可能不准
**解决**：
- 默认自动识别
- 提供 `--emotion=urgent` 等手动覆盖选项
- 生成前确认情绪标签

### 张力 3: better-icons 依赖

**问题**：better-icons 是独立技能，如何无缝整合？
**解决**：
- mino-frontend 内部调用 better-icons CLI
- 如果 better-icons 不可用，降级为无图标模式
