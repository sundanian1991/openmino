# 参数系统框架 — mino-frontend

> **目的**：解决执行不稳定问题，为每种设计风格定义具体参数值
>
> **核心论点**：方法论是骨架，参数系统是血肉。骨架相同但血肉不同，输出就不稳定。

---

## 一、问题诊断

### 当前问题

| 症状 | 根因 |
|------|------|
| 同样的方法论，输出不一致 | 参数值定义不一致 |
| 每次生成需要"调参数" | 参数值未固化 |
| 方法论执行不稳定 | 参数系统缺失 |

### 从过往输出中发现的问题

| 参数 | 游泳 | 刘乾坤v3 | 权责不对等 | 问题 |
|------|------|----------|-----------|------|
| **T1 字号** | 2rem~2.5rem | 1.8rem~2.8rem | 1.5rem~2.25rem | ❌ 不一致 |
| **T2 字号** | 1.5rem~1.75rem | 1rem~1.3rem | 1.125rem~1.375rem | ❌ 不一致 |
| **页面 padding** | 1.5rem~3rem | 1.5rem~3rem | 2rem~4rem | ❌ 不一致 |
| **陶土色装饰线** | 2.4px | 2px（非陶土色） | 无 | ❌ 不一致 |

## 二、参数系统结构

### 三层参数模型

```
┌─────────────────────────────────────────────────────────────┐
│                    参数系统三层结构                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【Layer 1：全局参数】                                       │
│  所有风格共享的基准参数                                       │
│  ├── 色彩系统（主色、辅助色、陶土色）                        │
│  ├── 间距基准（4pt 系统）                                    │
│  └── 响应式断点                                              │
│                                                              │
│  【Layer 2：风格参数】                                       │
│  每种设计风格独有的参数                                       │
│  ├── 字体层级（T1-T6 px 值）                                │
│  ├── 边框系统（粗细、圆角）                                  │
│  ├── 间距比例（页面 padding、卡片 gap）                      │
│  └── 陶土色使用规则                                          │
│                                                              │
│  【Layer 3：内容参数】                                       │
│  由内容特征决定的参数                                         │
│  ├── 信息密度（高/标准/低）→ 间距调整                        │
│  ├── 情绪气质（紧迫/中性/庆祝/沉思）→ 陶土色用量             │
│  └── 阅读方式（扫视/阅读/凝视）→ 字号层级差                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 参数优先级

```
内容参数 > 风格参数 > 全局参数

例：
- 全局：间距基准 4pt
- 风格（克制专业主义）：页面 padding = 6×4pt = 24px
- 内容（高密度）：页面 padding 调整为 4×4pt = 16px
```

## 三、参数类型定义

### 3.1 色彩系统（全局参数）

**克制专业主义色彩系统**：

```css
:root {
  /* 文字层级 */
  --text-primary: #111111;
  --text-secondary: #666666;
  --text-tertiary: #999999;

  /* 背景 */
  --bg: #FFFFFF;
  --surface: #FAFAFA;

  /* 边框 */
  --border: #E5E5E5;

  /* 陶土色（焦点色） */
  --accent: #E2725B;
  --accent-light: rgba(226, 114, 91, 0.08);
}
```

**陶土色使用规则**：
- 一页一处陶土色强调（紧迫情绪可放宽到 2-3 处）
- 不用于装饰性元素（分隔线、背景色块）
- 不用于正文文字
- 用途优先级：P0 异常/风险 > P1 排名/等级 > P2 行动建议

---

### 3.2 字体层级系统（风格参数）

**克制专业主义字体层级**：

| 层级 | 用途 | 基准值 | clamp 值 | 层级差 |
|------|------|--------|----------|--------|
| **T1** | 页面标题 | 28px | `clamp(1.75rem, 4vw, 2rem)` | 基准 |
| **T2** | 模块标题 | 18px | `clamp(1.125rem, 2vw, 1.25rem)` | -10px |
| **T3** | 卡片标题 | 14px | `clamp(0.875rem, 1.5vw, 1rem)` | -4px |
| **T4** | 正文 | 13px | `clamp(0.8125rem, 1.3vw, 0.9rem)` | -1px |
| **T5** | 辅助文字 | 11px | `clamp(0.6875rem, 1vw, 0.75rem)` | -2px |
| **T6** | 页码/标签 | 10px | `clamp(0.625rem, 0.8vw, 0.7rem)` | -1px |

**层级差原则**：
- T1 → T2：-10px（大跳跃，区分页面标题和模块标题）
- T2 → T3：-4px（中跳跃，区分模块和卡片）
- T3 → T4：-1px（小跳跃，卡片内层级）
- T4 → T5：-2px（中跳跃，正文和辅助）
- T5 → T6：-1px（小跳跃，辅助和标签）

**响应式调整**：
- 在 500px 高度以下，所有字号缩小 10%
- 在 800px 高度以上，所有字号放大 10%

---

### 3.3 边框系统（风格参数）

**克制专业主义边框系统**：

| 元素 | 边框粗细 | 边框颜色 | 圆角 |
|------|---------|---------|------|
| **卡片** | 1px | `#E5E5E5` | 4px |
| **表头分割线** | 2px | `#111111` | 0 |
| **表格行分割线** | 1px | `#E5E5E5` | 0 |
| **顶部装饰线** | 2.4px | `#E2725B` | 0 |
| **强调卡片** | 1px | `#E2725B` | 4px |
| **色块区域** | 0 | — | 0 |

**边框选择规则**：
- 默认：1px 浅灰边框（`#E5E5E5`）
- 强调：陶土色边框（`#E2725B`）
- 分割：2px 黑色线（表头用）
- 装饰：陶土色顶部线（2.4px）

---

### 3.4 间距系统（风格 + 内容参数）

**4pt 基准系统**：

```css
:root {
  --space-xs: 4px;   /* 元素内部最小间距 */
  --space-sm: 8px;   /* 相关元素间距 */
  --space-md: 12px;  /* 分组内部间距 */
  --space-lg: 16px;  /* 分组之间间距 */
  --space-xl: 24px;  /* 区块之间间距（层级分隔） */
  --space-2xl: 32px; /* 大区块之间间距 */
  --space-3xl: 48px; /* 页面边缘间距 */
}
```

**克制专业主义间距比例**：

| 密度 | 页面 padding | 卡片内 padding | 卡片 gap | 区块 gap |
|------|-------------|---------------|----------|---------|
| **高密度** | 24px | 12px | 12px | 16px |
| **标准密度** | 32px | 16px | 16px | 24px |
| **低密度** | 48px | 24px | 24px | 32px |

**密度映射规则**（由内容特征决定）：

| 情绪 × 阅读方式 | 密度选择 |
|----------------|---------|
| 紧迫 × 扫视 | 高密度 |
| 紧迫 × 阅读 | 标准密度 |
| 中性 × 扫视 | 标准密度 |
| 中性 × 阅读 | 标准密度 |
| 中性 × 凝视 | 低密度 |
| 庆祝 × 凝视 | 低密度 |
| 沉思 × 凝视 | 低密度 |

---

### 3.5 陶土色使用系统（内容参数）

**陶土色使用场景**：

| 场景 | 使用方式 | 面积限制 |
|------|---------|---------|
| **顶部装饰线** | 2.4px 实线，陶土色 | ≤ 5px 高 |
| **页码徽章** | 24px 圆形填充 | ≤ 24px × 24px |
| **强调卡片边框** | 1px 实线，陶土色 | 卡片周长 |
| **强调卡片背景** | `rgba(226, 114, 91, 0.08)` | 卡片面积 |
| **列表项圆点** | 4px 圆点 | ≤ 4px × 4px |
| **关键词强调** | 文字颜色 | 仅关键词 |

**一页一处规则**：

| 情绪 | 陶土色使用量 |
|------|------------|
| **紧迫** | 2-3 处（高对比，密集标记） |
| **中性** | 1 处（标准焦点） |
| **庆祝** | 1 处（大字号金句） |
| **沉思** | 1 处（极简，大面积留白 + 小焦点） |

## 四、参数协调规则

### 4.1 边框 × 间距协调

| 边框粗细 | 配套间距 | 视觉效果 |
|---------|---------|---------|
| 1px | 标准（16px） | 轻量、透气 |
| 2px | 加大（24px） | 强调、分隔 |
| 陶土色 1px | 加大（24px） | 焦点、吸引 |

### 4.2 字号 × 间距协调

| 字号层级差 | 配套行高 | 配套段落间距 |
|-----------|---------|-------------|
| T1-T2（-10px） | 1.2 | 24px |
| T2-T3（-4px） | 1.3 | 16px |
| T3-T4（-1px） | 1.5 | 12px |

### 4.3 陶土色 × 信息密度协调

| 信息密度 | 陶土色使用方式 |
|---------|--------------|
| 高密度 | 微点缀（条件格式、小圆点） |
| 标准密度 | 轻混合（边框、徽章） |
| 低密度 | 重混合（大面积背景、大字号） |

## 五、执行检查清单

### 参数执行检查

- [ ] **色彩系统**：使用了正确的色彩变量？
- [ ] **字体层级**：T1-T6 值符合风格定义？
- [ ] **边框系统**：粗细、颜色、圆角符合规则？
- [ ] **间距系统**：密度选择符合内容特征？
- [ ] **陶土色**：一页一处，用在了正确位置？

### 稳定性检查

- [ ] **同类页面参数一致**：所有封面页用相同的 T1 值？
- [ ] **同风格参数一致**：所有克制专业主义页面用相同的间距？
- [ ] **跨文件参数一致**：同一个项目的所有文件用相同的参数？

## 六、参数系统与六层决策的关系

```
【六层决策】                  【参数系统】

L1 内容元素识别    →    决定可视化级别（参数系统 Layer 3）
L2 情绪气质        →    决定密度和陶土色用量（参数系统 Layer 3）
L3 阅读方式        →    决定字号层级差（参数系统 Layer 3）
L4 密度与版式      →    选择间距值（参数系统 Layer 2+3）
L5 可视化叠加      →    选择边框和装饰（参数系统 Layer 2）
L6 焦点选择        →    选择陶土色位置（参数系统 Layer 3）

【执行流程】
六层决策 → 输出内容结构 + 选择参数值 → 生成代码
```
