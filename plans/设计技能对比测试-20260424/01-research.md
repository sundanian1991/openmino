## 测试方案

**统一测试 Brief**：

> 你是供应商管理负责人，需要制作一份「供应商管理月度概览」，向领导汇报 3 月份供应商表现。
>
> 数据：8 家供应商，核心指标包括坐席数、接通率、质检合格率、产能达成、满意度。
> 目的：让领导一眼看清哪些供应商表现好，哪些需要关注。
> 要求：信息清晰、视觉专业、可直接使用。

**测试数据**：详见 workspace 中的测试材料文件

**目录结构**：
```
plans/设计技能对比测试-20260424/          ← kw-workflow 文件
workspace/19-测试-设计技能对比-20260424/  ← 测试材料 + 技能产出
```

## 评估维度

1. **视觉效果**（0-5）：美观度、设计感、专业度
2. **信息传达**（0-5）：数据是否一目了然，重点是否突出
3. **实用性**（0-5）：能否直接用于日常工作
4. **调整成本**（0-5）：修改数据和内容的难易程度（越低越好）
5. **综合满意度**（年老师主观打分 0-10）
