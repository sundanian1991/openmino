# nothing-design — 可复用 Prompt 模板

> 适用场景：需要 Nothing 风格（瑞士排版/单色/工业设计感）的 UI 页面、数据看板、仪表盘、信息展示页。
> 创建日期：2026-05-26

---

## Prompt 模板

复制以下内容到其他对话，填空即用：

```
做一个 [用途描述] 页面，给 [受众] 看。

平台：[HTML / PPT / 其他，默认 HTML]

数据：
- [指标名]：[数值]（[对比/状态，可选]）
- [指标名]：[数值]（[对比/状态，可选]）
- [指标名]：[数值]（[对比/状态，可选]）

信息优先级：
1. [最重要的内容] —— [为什么重要]
2. [次要内容] —— [展示方式示意]
3. [第三层内容]

其他要求：[可选：约束、风格倾向、特殊处理]

模式：[dark / light，不写则技能会问]
```

---

## 完整示例

```
做一个供应商季度KPI看板页，给管理组看。

平台：HTML

数据：
- 综合达成率：96.8%（目标90%，超目标）
- 客诉率：0.23%（目标<0.5%，良好）
- 人效：156单/天（目标140，超目标）
- 预算使用率：67%（正常）
- 在岗人数：342人

信息优先级：
1. 综合达成率96.8% —— 核心KPI，最醒目，大字号展示
2. 人效156单 + 客诉率0.23% —— 次要KPI，并排展示
3. 预算使用率67% + 在岗人数342人 —— 支撑信息，放底部

模式：dark
```

## 使用说明

| 变量 | 必填 | 说明 | 不填怎么办 |
|------|------|------|-----------|
| 用途描述 | 必填 | 概括页面用途，如"季度KPI看板"、"实时监控面板" | 技能产出会偏离你的真实场景 |
| 受众 | 推荐 | 谁看这个页面，影响信息密度和层级 | 默认"一般用户"，信息层级可能不够精准 |
| 平台 | 可选 | HTML / PPT / 其他 | 默认 HTML |
| 数据条目 | 推荐 | 3-5条为宜，过多会压缩层级差异 | 技能无法展示数据 |
| 数据状态 | 可选 | 超目标/警告/正常等，技能会着色 | 技能按中性色展示 |
| 信息优先级 | **核心** | 1/2/3 层，直接驱动视觉层级 | 技能自动按数据顺序分配层级 |
| 模式 | 可选 | dark / light | 技能会主动问你 |
| 其他要求 | 可选 | 特殊约束或风格倾向 | 忽略 |
