## 一、四大管理角度

### 1.1 站点管理

**管理目标**：确保每个站点健康、稳定、可持续运营

| 管理问题 | 风险维度 | 定义 | 阈值 | 优先级 |
|---------|---------|------|------|--------|
| 站点太小不经济 | 微小站点 | 站点人力 <20人 | 0 | P0 |
| 站点太大有风险 | 超大站点 | 站点人力 >100人 | 0 | P0 |
| 站点不稳定 | **站点波动** | 月度环比波动 >30% | 连续2个月 | P1 |
| 站点太饱和 | **站点饱和度** | 站点人力 >200人 | 0 | P2 |
| 站点项目单一 | **项目单一站点** | 站点覆盖项目数 =1 | 0 | P1 |

### 1.2 人员管理

**管理目标**：确保人力配置合理、流失可控、效率达标

| 管理问题 | 风险维度 | 定义 | 阈值 | 优先级 |
|---------|---------|------|------|--------|
| 人员流失 | **人员流失率** | 月度流失 >10% | 0 | P1 |
| 人效低下 | **人均产出** | 人均产出低于平均水平 | <平均值80% | P2 |
| 人员冗余 | **人员冗余度** | 某供应商人力连续3个月无产出 | 0人产出 | P2 |
| 增长过快吃不消 | **增长预警** | 连续3个月增长 >50% | 0 | P2 |

### 1.3 趋势管理

**管理目标**：及时发现异常趋势，提前预警

| 管理问题 | 风险维度 | 定义 | 阈值 | 优先级 |
|---------|---------|------|------|--------|
| 持续下降 | **连续下降** | 月度环比连续下降 ≥2个月 | 0 | P0 |
| 爆发式增长 | **爆发增长** | 单月增长 >100% | 0 | P2 |
| 震荡波动 | **趋势波动** | 连续3个月上下波动 >30% | 0 | P2 |

### 1.4 风险管理

**管理目标**：识别集中度风险、供应链脆弱性、过度依赖

| 管理问题 | 风险维度 | 定义 | 阈值 | 优先级 |
|---------|---------|------|------|--------|
| 供应商过度依赖单一项目 | **供应商集中度（项目）** | 某供应商在单一项目的人力占比 | >30% | P0 |
| 供应链太脆弱 | **供应链单薄** | 某项目的供应商数量 | ≤2家 | P0 |
| 供应商地域集中 | **地理集中度（城市）** | 某供应商在某城市的站点占比 | >50% | P0 |
| 项目过度依赖TOP供应商 | **项目集中度** | 某项目TOP3供应商占比 | >80% | P1 |
| 某省人力过于集中 | **省份集中度** | 某省人力占总人力占比 | >50% | P1 |
| 供应商业务不灵活 | **业务灵活性低** | 供应商只做1条产品线 | products.length=1 | P1 |
| 供应商业务太分散 | **业务分散风险** | 供应商做>5条产品线 | products.length>5 | P2 |

## 四、最终建议：10+2 风险维度

### 4.1 核心风险维度（10个）- 必须实现

| # | 风险维度 | 定义 | 阈值 | 计算 |
|---|---------|------|------|------|
| 1 | 微小站点 | 站点人力 <20人 | 0 | `workplaces.filter(w => w.headcount < 20)` |
| 2 | 超大站点 | 站点人力 >100人 | 0 | `workplaces.filter(w => w.headcount > 100)` |
| 3 | 供应商集中度 | 供应商在单一项目的人力占比 | >30% | `供应商在项目A的人力 / 供应商总人力` |
| 4 | 供应链单薄 | 某项目的供应商数量 | ≤2家 | `projects.filter(p => p.supplierCount <= 2)` |
| 5 | 地理集中度 | 供应商在某城市的站点占比 | >50% | `供应商在城市A的站点数 / 供应商总站点数` |
| 6 | 项目集中度 | 某项目TOP3供应商占比 | >80% | `项目TOP3供应商人力 / 项目总人力` |
| 7 | 连续下降 | 月度环比连续下降 | ≥2个月 | `连续2个月环比 <0` |
| 8 | 业务灵活性低 | 供应商只做1条产品线 | 0 | `suppliers.filter(s => s.products.length === 1)` |
| 9 | 业务分散风险 | 供应商做>5条产品线 | 0 | `suppliers.filter(s => s.products.length > 5)` |
| 10 | 省份集中度 | 某省人力占总人力 | >50% | `某省人力 / 总人力` |
