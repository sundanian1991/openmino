VIS-01 / DATA SHOCK
一个开发者 8 个月消耗 100 亿 Token
订阅费 ¥40/月 vs API 实际价值 $15,000+ | 2026
目的
用单一数字建立认知锚点，让读者感受到"100 亿"这个量级的荒谬感
表达
等宽字体大字 + 黑色背景 + 底部微型标注 + 右侧 ¥40 vs $15,000 对比 badge
结论
¥40 的订阅背后，是 $15,000 的真实消耗。订阅不是便宜的替代方案，是唯一可行的方案。
思路
不画图。用排版和字号层级做视觉冲击。第一眼看数字，第二眼看底部标注，第三眼看右侧对比。
VIS-02 / KEY METRICS
18 个月的价格战，四个关键数字
参战平台多、计量混乱、浪费严重，但订阅能省大钱
目的
用四个指标快速建立对整个价格战的全景认知
表达
四宫格独立卡片，70% 红色最大号（浪费触目惊心），93% 绿色（订阅是唯一活路）
结论
参战平台多、计量方式乱、浪费高达 70%，但订阅能省下 93% 的费用
思路
不是图表，是指标阵列。数字大小差异本身就是叙事——70% 最大，因为浪费是最刺痛的数字。
VIS-03 / TIMELINE
18 个月的价格战争地图
2024 Q4 → 2026 Q4 | 国内 AI 大模型定价演变
目的
展现这不是线性演进，是 Agent 时代催生的必然转向
表达
水平时间线，事件卡片沿轴排列。高亮 OpenClaw（催化剂）和 DeepSeek V4（终态信号），其余灰化
结论
OpenClaw 爆火是转折点，之后的 Token Plan 更名是终态信号——这是 Agent 时代的定价重构
思路
时间线不是事件罗列，是因果关系。OpenClaw → Coding Plan 承压 → Token Plan，用因果箭头串联。
VIS-04 / TRANSLATION BOARD
五种计量单位 = 五种语言
90% 开发者在 ¥40-50 区间踩坑 | 直接比数字毫无意义
目的
揭示"比数字之前先统一计量单位"——五种计量就是五种语言
表达
机场汇率牌风格，五列面板。API 请求最红（信息最不透明），Token 最绿（完全透明）
结论
API 请求看起来最多（18000 次），实际最少。信息不透明是最大的坑
思路
类比机场汇率牌——不同国家用不同货币，直接比较面值没有意义。必须先换算。
VIS-05 / PACKAGE COMPARISON
同样 ¥40/月，实际可用量差距最大 15 倍
入门档套餐对比 | 2026-04 价格
目的
用水平条形图直观展示"同样月费，实际购买力天差地别"
表达
条子长短不一，阿里百炼用红色虚线框标注（范围最宽 = 信息最不透明），Kimi 标"不确定"
结论
看起来差不多的月费（¥39-49），实际购买力差 15 倍。Cursor Pro 最划算，阿里百炼最不透明
思路
水平条形图比垂直柱状图更适合——平台名可以完整显示，条子长度差异一目了然。
VIS-06 / TOKEN COMPOSITION
Agent 消耗的 Token 里，只有 5-15% 是代码生成
一个开发者的 Token 消耗构成 | 2026
目的
揭示 Agent 消耗的 Token 中，代码生成只占极小部分，大部分是 Agent 自身开销
表达
单根 100% 堆叠条。代码生成（Teal）最短，上下文重发（Coral）标注"纯浪费"
结论
70% 的 Token 是 Agent 自身的开销，不是你的需求。上下文重发是纯浪费
思路
第一眼看到代码生成那小段绿色——占比小得可怜。第二眼看到上下文重发的红色——纯浪费。结论：你需要的是理解 Agent 的开销结构，而不是只看总量。
VIS-07 / CHIP SUPPLY
华为昇腾每多出货一片，Token 就便宜一分
2024-2028 芯片市占率演变 | 国产替代 = 成本下降
目的
建立芯片供给与 Token 定价的因果关系——芯片多了，价格就降
表达
双线对比：英伟达（灰虚线下降）vs 华为（Warm 实线上升），2026E 交叉点标注
结论
芯片国产替代直接打开定价天花板。昇腾 950PR 量产 = 价格大幅下调窗口打开
思路
不是看芯片，是看供给。供给增加 → 成本下降 → 价格下调。这条线是定价的天花板。
VIS-08 / THREE-STAGE PRICING
所有互联网赛道的定价，都走这三步
AI 也不例外 | 进入新市场：先跑量 → 测弹性 → 精分层
目的
提取可复用的定价节奏——所有互联网赛道都走这三步
表达
三段宽度递减的卡片（感受"收紧"），箭头连接。补贴→收紧→分层
结论
这不是 AI 定价独有的，是所有互联网赛道的定价节奏。别一开始就想好最终定价
思路
漏斗变体。宽度递减本身就是叙事——从"大而便宜"到"小而贵"。阶段 3 最窄但价值最高。
VIS-09 / DOMESTIC vs OVERSEAS
同一产品，两套价格体系
国内 7 家抢池子 vs 海外 3 巨头护城河
目的
揭示国内低价的真相——不是成本低，是 7 家竞争逼迫
表达
左右两栏，中间虚线分隔。左侧 Coral（竞争激烈），右侧 Warm（稳定护城河）
结论
国内便宜是因为"卷"，海外贵是因为"生态"。同一产品，两套逻辑，5 倍价格差
思路
第一眼是左右价格的巨大差距（¥29-49 vs $20-200），第二眼是竞争环境的根本差异，结论：国内低价不可持续。
