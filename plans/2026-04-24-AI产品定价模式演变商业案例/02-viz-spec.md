**叙事主线**：从 ¥40 的幻觉到 100 亿的真相——一场 18 个月的价格战调查。

**视觉基调**：调查档案风。等宽字体数据 + 衬线体结论 + 红绿信号色。不是报告，是卷宗。

**色系选择**：

|------|------|------|
| 主数据 | Warm 500/800 | 主叙事线 |
| 正面/利好 | Teal 500 | 成本下降、效率提升 |
| 负面/预警 | Coral 500 | 浪费、涨价、收紧 |
| 基准/灰化 | Stone 300/500 | 背景、对比线 |

## VIS-01 | 数据冲击开场

> **对应 storyboard**: Page 01 Scene 1 (B5 Cold Open)

### 第一部分：叙事意图

**【可视化目的】**
读者看到"¥40/月订阅"这个数字时，觉得"好便宜"。这张图要用一个巨大的、违反直觉的数字建立认知锚点——一个开发者 8 个月消耗了 100 亿 Token。认知缺口：从"便宜"到"荒谬"。

**【想传达什么】**
核心信息：¥40 的订阅背后，是 $15,000+ 的真实消耗。订阅不是便宜的替代方案，是唯一可行的方案。这条信息是整个调查的起点——没有这个锚点，后续所有对比都站不住。

**【结论】**
如果一个开发者按 API 价格付费，8 个月要花 $15,000+。¥40 订阅是价格战的产物，不是定价的真相。

**【思路】**
不画图。用排版和字号层级做视觉冲击。第一眼看到巨大的 10,000,000,000（等宽字体增加"数据感"），第二眼看底部微型标注理解这个数字的含义，第三眼看右侧 ¥40 vs $15,000 的对比 badge。如果画成柱状图或其他图表，冲击力反而被分散。

### 第二部分：视觉执行

**【模式选择】**
- **模式**：69-成套工具 → 变形为单数字冲击
- **理由**：1-3 个关键数字直接写大字，不画图（13-VISUALIZATION.md 铁律）

**【标题】**
- **主标题**：一个开发者 8 个月消耗 100 亿 Token
- **副标题**：订阅费 ¥40/月 vs API 实际价值 $15,000+ | 2026

**【视觉编码】**
- **单一巨大数字**：`10,000,000,000`（JetBrains Mono，60px+，Warm 800）
- **底部微型标注**：`一个开发者 × 8 个月 × Claude Code`（Stone 500，12px）
- **右侧浮动 badge**：`¥40 订阅`（Teal 500 描边）对比 `API 价 $15,000+`（Coral 500 描边）

**【数据组织】**
- 3 个关键数字：100 亿（Token 消耗）、¥40（订阅费）、$15,000（API 等价）
- 无需聚合

**【标注策略】**
- **高亮**：无额外标注，数字本身即信息
- **标注**：底部脚注标注数据来源

**【图例与辅助】**
- **图例**：不需要
- **脚注**：`数据：2026 | 一个开发者 8 个月 Token 消耗估算`

**【布局】**
- **画布**：100vw × 100vh（全屏）
- **数字**：绝对居中
- **标注**：数字下方 40px
- **右侧 badge**：距右边 24px
- **背景**：#0a0a0f（纯黑）

## VIS-02 | 关键指标四宫格

> **对应 storyboard**: Page 01 Scene 2 (B9 Data Bombardment)

### 第一部分：叙事意图

**【可视化目的】**
冲击之后需要全景。读者需要 10 秒内建立对整个价格战的四维认知：多少人打、用什么打、浪费多少、怎么解。认知缺口：从"一个数字的震撼"到"整个战场的轮廓"。

**【想传达什么】**
核心信息：战场混乱（7+ 平台 × 5 种计量）、浪费严重（70%）、但订阅是唯一活路（93% 节省）。这四个数字组合在一起，建立"问题严重但解法明确"的判断。

**【结论】**
参战平台多、计量方式乱、浪费高达 70%，但订阅能省下 93% 的费用。这是订阅制必须被选择的理由。

**【思路】**
不用图表，用指标阵列。数字大小本身就是叙事——70% 最大最刺眼（最痛），93% 次之（最强的解法信号）。第一眼被红色 70% 抓住（痛），第二眼看到绿色 93%（解）。如果用柱状图，四个柱子一样高，丢失了"痛感 vs 解法"的对比。

### 第二部分：视觉执行

**【模式选择】**
- **模式**：69-成套工具 → 变形为指标卡
- **理由**：四宫格独立卡片，每个数字独立表达一个维度

**【标题】**
- **主标题**：18 个月的价格战，四个关键数字
- **副标题**：2024 Q4 → 2026 Q4 | AI 定价演变

**【视觉编码】**
- **四宫格**：2×2 网格，每个指标独立卡片
- **颜色**：70% 用 Coral（负面痛点），93% 用 Teal（正面解法），其余 Stone
- **字号**：70% 最大（52px），其余 42px

**【数据组织】**

| 指标 | 数值 | 色系 |
|------|------|------|
| 参战平台 | 7+ | Warm 500 |
| 计量方式 | 5 种 | Stone 500 |
| Token 浪费率 | 70% | Coral 500 |
| 订阅节省比例 | 93% | Teal 500 |

**【标注策略】**
- **高亮**：70%（Coral 红色描边，最大字号），93%（Teal 绿色描边）
- **标注**：无额外标注，数字本身即信息
- **底部**：标注核心结论"参战平台多、计量混乱、浪费严重，但订阅能省大钱"

**【图例与辅助】**
- **图例**：不需要
- **脚注**：底部标注核心结论

**【布局】**
- **画布**：840 × 320
- **卡片**：2×2 网格，间距 16px，内边距 16px，圆角 10px
- **数字**：卡片垂直居中

## VIS-03 | 价格战时间线

> **对应 storyboard**: Page 02 Scene 1 (B13 Flashback)

### 第一部分：叙事意图

**【可视化目的】**
读者认为价格变化是"线性演进"的。这张图要展示：这不是自然演进，是 Agent 时代催生的必然转向。认知缺口：从"慢慢变"到"被推着走"。

**【想传达什么】**
核心信息：OpenClaw 爆火（2026-02）是转折点，之后的 Token Plan 更名是终态信号。这不是计划内的调整，是 Agent 工具倒逼定价模式重构。

**【结论】**
18 个月的价格战，本质是 Agent 时代对传统订阅制的冲击。OpenClaw 是催化剂，Token Plan 是终态，这是必然转向，不是线性演进。

**【思路】**
选时间线而不是折线图，因为要表达的是"事件"而非"数值趋势"。地铁路线图的因果连线表达"不是独立事件，是连锁反应"。第一眼落在中间位置的 OpenClaw（催化剂），第二眼看到前后收紧趋势，最终理解"这不是自然演进"。如果用折线图，丢失了事件因果关系。

### 第二部分：视觉执行

**【模式选择】**
- **模式**：25-时间线 + 37-地铁路线图
- **理由**：事件序列 + 因果连线，表达"不是独立事件，是连锁反应"

**【标题】**
- **主标题**：18 个月的价格战争地图
- **副标题**：2024 Q4 → 2026 Q4 | 国内 AI 大模型定价演变

**【视觉编码】**
- **X 轴**：时间（2024 Q4 → 2026 Q4）
- **事件卡片**：沿主轴上下排列
- **颜色编码事件类型**：Coral=收紧/催化，Teal=新形态/终态，Sky=切换
- **因果箭头**：虚线连接关联事件

**【数据组织】**

| 时间 | 事件 | 类别色 |
|------|------|--------|
| 2024 Q4 | DeepSeek 带头降价，百万 Token 几十元→几毛钱 | Coral（开战） |
| 2025 Q4 | 智谱 GLM 首推 Coding Plan | Teal（新形态） |
| 2025 H2 | 首购优惠取消、低档停售、限时抢购 | Coral（收紧） |
| 2026-01 | Kimi 切换 Token 计量 | Sky（切换） |
| 2026-02 | OpenClaw 爆火，Token 消耗 6-10 倍 | Coral（催化剂） |
| 2026-03 | MiniMax 更名 Token Plan | Teal（终态信号） |
| 2026-03-05 | 腾讯云入局，六大全上线 | Sky（全面） |
| 2026-04 | 阿里百炼 Lite 下架 / 企业版 Token Plan | Coral + Teal |
| 2026-04 | 小米 MiMo $1/$3 极低价入局 | Coral（冲击） |
| 2026-04 | Anthropic 移除 OpenClaw 访问 | Coral（收缩） |
| 2026-04-24 | DeepSeek V4 开源，成本降 73% | Teal（革命） |

**【标注策略】**
- **高亮 ≤10%**：只高亮 OpenClaw 爆火（催化剂，加框放大）和 Token Plan 更名（终态信号，加粗）
- **因果连线**：OpenClaw → Token Plan 之间画虚线箭头，标"催化转向"
- **其余事件**：Stone 300 描边，灰化处理

**【图例与辅助】**
- **图例**：底部三色列举（Coral=收紧、Teal=新形态、Sky=切换）
- **脚注**：`数据来源：公开报道整理，截至 2026-04-24`

**【布局】**
- **画布**：960 × 440
- **主轴**：水平居中，最大宽度 900px
- **事件卡片**：主轴上下交错排列，重要事件间距更大
- **因果箭头**：主轴上方，不穿越卡片
