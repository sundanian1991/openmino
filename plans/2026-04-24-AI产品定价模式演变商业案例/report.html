01 · Executive Summary

执行摘要

AI 大模型定价正从「卖工具订阅」演进为「卖算力资源包」，这个过程精确重演了电信行业从话费套餐到流量套餐的 20 年历史。Coding Plan 是过渡产品，Token Plan 是终态。能从中学到的不是具体的定价数字，而是三个底层逻辑。

核心 Takeaways

  • 计量单位即定价权：谁定义计量单位（Token / Prompt / 请求 / Credit），谁就掌握定价权。MiniMax 从 Prompt 转 Token、Kimi 从 Prompt 转 Token，都是在争夺这个定义权。
  • 补贴 → 收紧 → 分层的三段式：和互联网所有赛道一样，烧钱圈用户 → 市场份额稳定 → 精细分层收割。2024-2025 价格战 → 2025 H2 收紧 → 2026 分层，节奏清晰。
  • Agent 时代的成本指数级增长不可逆：OpenClaw 爆火（GitHub 355K stars）直接催热 Coding Plan 需求，一个 Agent 任务 = 几十次模型调用，Coding Plan 的「不限量」承诺在 Agent 时代不可持续。
本文档回答的问题
Coding Plan 到 Token Plan 的转变，本质是什么商业逻辑？国内七家厂商的定价策略有哪些共性和差异？这个案例对未来的产品定价有什么可复用的模式？

2.2 参与者格局

国内 AI 定价市场形成两条并行路线，外加海外对标参照：

路线代表厂商特征定价逻辑
专卖店 模型厂商智谱 GLM、Kimi、MiniMax、小米 MiMo只卖自家模型，深度优化低价抢量 → 收紧分层

2.3 计量体系：最大的信息壁垒

五种计量单位并存，直接比数字毫无意义，90% 开发者容易踩坑。计量单位的选择不仅是技术决策，更是定价权的争夺。

计量方式采用者换算关系透明度
API 请求次数阿里百炼、火山方舟、腾讯云、百度千帆1 次提问 = 5-30 次 API 请求低
Prompt 次数智谱 GLM、MiniMax（早期）1 Prompt ≈ 15-20 次模型调用 = 1200-1600 API 请求中

演变趋势：API 次数 → Prompt 次数 → Token/Credit → 统一 Credits。方向是越来越透明、越来越可预测。

2.5 OpenClaw：Token 消耗的催化剂

OpenClaw（GitHub 355K stars）作为开源 AI Agent 框架，是整个定价模式转变的催化剂。它的影响链路清晰：

Agent 自动循环调用 → Token 消耗 6-10 倍增长 → Coding Plan 承压 → 算力告急 → 限量供应 → Lite 档停售 → Token 精细计量成为必然

MiniMax M2 token 用量 2 月 vs 12 月增长 6 倍，Coding Plan 消耗增长 10 倍+。阿里云算力告急，Lite 档 4-13 停售。Anthropic 2026-04 从标准订阅移除 OpenClaw 访问——成本撑不住了。

2.6 模型进步与芯片供给的底层变量

定价模式演变不只是商业策略问题，还有两个底层变量在推动：

模型能力跃升

2026 年密集发布：Kimi K2.5（1T 参数）、智谱 GLM-5（744B）、MiniMax M2.5（229B）、MiMo-V2.5-Pro（对标 Opus 4.6）、DeepSeek V4（1.6T，成本暴降 73%）。模型能力跃升 → Agent 可用性提高 → Token 消耗指数级增长 → Coding Plan 承压 → Token Plan 成为必然。

芯片供给格局剧变

英伟达中国市占率从 66%（2024）跌至 54%（2025），2025-04 全面禁止 AI GPU 对华出口。华为昇腾崛起：2025 年交付 81.2 万片，950PR 算力 2.87 倍于 H20、价格仅 1/3，4 月量产。DeepSeek V4 同时适配华为+寒武纪，明确说「受限于高端算力，预计下半年昇腾 950 批量上市后价格大幅下调」。芯片供给直接绑定定价天花板。

2.7 IPO/资本化进程

智谱 GLM（2513.HK）2026-01-08 港交所上市，市值 511 亿港元，Coding Plan 两个月付费开发者超 15 万，ARR 快速破亿。MiniMax 2026 年 1 月上市，营收 7904 万美元 +159%，海外收入 73%。上市公司必须证明商业可持续性——Coding Plan / Token Plan 是可见的收入增长引擎。

03 · Framework

框架层：为什么这么演变

理解「发生了什么」之后，需要一个框架来解释「为什么」。电信套餐演进史提供了最精确的类比。

3.1 类比框架：电信套餐 20 年

一位日耗 10 亿 Token 的工程师的观察：2009 年充 100 块话费用不完，2013 年 9.9 块买几 G，2017 年「不限量」达量降速，2020 年真正无限流量消失。Token 正在走一模一样的路。

电信阶段AI 对应核心逻辑
2009 话费套餐（充 100 用不完）2024 API 按量计费（百万 Token 几十元）供给过剩，用户感知不到成本
2013 流量套餐（9.9 几 G）2025 Coding Plan（¥29-200/月）用量激增，包月降低感知
2017「不限量」达量降速2026 分层套餐（极速版/普通版）精细化分层，按速度/质量定价
2020 按 GB 精打细算2026+ Token 精细计量用户开始算账，厂商利润回归
关键类比：Coding Plan ≈ 手机合约机（绑定特定工具/场景），Token Plan ≈ 手机流量包（通用算力资源，不限场景）。MiniMax 改名 Token Plan 的逻辑：不只卖编程能力，卖的是全模态 AI 能力的「算力通票」。

3.2 五力分析

力量分析强度
供应商议价力算力（GPU）是刚性成本。NVIDIA 占 85-90% 市场，但华为 950 正在打破垄断高 → 正在降低
买方议价力开发者选择多（7+ 平台），切换成本低（API Key 一换就行）高
替代品威胁开源模型（DeepSeek、Qwen）+ 本地部署 = 潜在替代，压制定价上限高
新进入者威胁需要百亿级融资 + 算力，但小米 MiMo 等新玩家仍在进入中
行业内竞争7 家在一个池子里打价格战，首月优惠从 7.9 元起极高

3.3 定价模式演进的三层逻辑

定价模式的演变不是随机的，而是沿着三层逻辑递进：

第一层：成本驱动 — 算力成本 → Token 成本 → 套餐定价。这一层决定了价格的下限。国产芯片放量（华为 950 年内 75 万片）+ DeepSeek V4 开源 MoE 效率革命，正在持续压低这个下限。

第二层：竞争驱动 — 价格战 → 补贴圈用户 → 收紧利润。这一层决定了价格战的节奏。七家厂商在同一池子里竞争，首月优惠从 7.9 元到免费，然后逐步收紧——和外卖、网约车的补贴战逻辑完全一致。

第三层：价值驱动 — 从「卖 Token」到「卖工作流」到「卖生态」。这一层决定了价格的上限。MiniMax 的 Token Plan 不只卖编程，打包了图片、语音、音乐、视频——从编程工具升级为 AI 多媒体工作室。

04 · Insights

洞察层：能提炼什么

事实和框架是基础，但真正有价值的是从这个案例中提炼出的可复用模式。以下六个洞察，每个都超越了 AI 定价本身。

4.1 计量单位即定价权

谁定义计量单位，谁就掌握定价权。

MiniMax 从 Prompt → Token：从「模糊打包」到「透明计费」，争夺用户信任。Kimi 从 Prompt → Token：同理。云厂商用「API 请求次数」：数字看起来大，但实际消耗多，信息不对称有利于厂商。

可复用模式：当你推出新产品时，计量单位的选择决定了用户感知和你的利润空间。模糊计量有利于早期获客（数字大，用户觉得值），透明计量有利于长期信任（用户能算清楚，愿意长期付费）。

4.2 补贴 → 收紧 → 分层的三段式

所有互联网赛道的定价都走这三步，AI 也不例外。
阶段目的AI 案例时间
补贴抢用户规模，建立习惯百万 Token 几毛钱，首月 7.9 元2024-2025
收紧测试价格弹性，筛选付费用户限购、停售低档、限时抢购2025 H2
分层利润最大化，匹配不同支付意愿极速版 ¥98-899，Kimi Allegro ¥5592026

可复用模式：进入新市场时，不要一开始就想好定价。先补贴跑量，再收紧测试弹性，最后精细分层。

4.3 从「入口」到「生态」的升级路径

Coding 是获客入口，Token 是生态载体。

MiniMax 的 Token Plan 不只卖编程，打包了图片、语音、音乐、视频。从「编程工具」升级为「AI 多媒体工作室」。¥29 买到的不只是代码补全，是全模态 AI 能力。

可复用模式：用高频刚需场景（编程）获客，用低频高价值场景（视频/音乐）提高 ARPU。

4.4 国内外定价逻辑的根本差异

维度国内海外
竞争环境7 家在同一池子打价格战面对 Claude/GPT/Gemini 三巨头
定价策略低价抢量，比谁更便宜证明「我不比 Claude 差」，比谁更强
用户预期「你在国内买到的便宜，是因为有人在替你卷」为质量和生态付费

4.5 Agent 时代的成本结构剧变

70% coding agent token 是浪费（DEV Community 研究）。一个开发者 8 个月 Claude Code 消耗 100 亿 Token，API 价 $15,000+，Max plan 只花 $800——省 93%。Cursor 2025-06 转 credit-based，重度用户一周超支 $350，引发退款风波。

Token 构成：文件读取 35-45%，工具输出 15-25%，上下文重发 15-20%，推理 10-15%，代码生成仅 5-15%。订阅费是地板不是天花板，实际账单是标价的 2-5 倍。

4.6 新入局者用低价破局，但终局必然是收紧

小米 MiMo V2-Pro 定价 $1/$3（Claude 的 1/5），但官方同时说「低价是陷阱」。这和 DeepSeek 当年带头降价后各家跟进再收紧的节奏完全一致。低价是手段，不是目的。入局需要低价，活下去需要价值。

05 · Forecast

未来预判

基于事实和框架，四个趋势正在同时发生。

趋势 1: Token 成本将持续下降

驱动力三重叠加：国产芯片放量（华为 950 年内 75 万片）+ DeepSeek V4 开源 MoE 效率革命（1.6T 参数只激活 49B）+ 推理专用芯片竞争（英伟达 Rubin CPX / 谷歌 TPU v8 / 华为 950PR）。结果：Token Plan 的价格战会继续，但利润空间来自「服务溢价」而非「算力差价」。

趋势 2: 国产芯片替代加速改变供给格局

英伟达市占率持续下降（66% → 54% → 2026 年可能低于 50%）。华为昇腾 + 寒武纪 + 阿里平头哥形成国产算力三角。软件生态从 CUDA 迁移到 CANN，效率已超 90%。算力供给不再被英伟达卡脖子 → 国内厂商成本结构和海外产生分化 → 定价策略可以更激进。

趋势 3: 开源模型重塑定价权

DeepSeek 模式：开源 + 极致低价 + 不卖订阅。V3.2 在 OpenRouter 月调用 5.35 万亿 token，全球第三。但 DeepSeek 没有 Coding Plan / Token Plan，只卖 API。冲击：开源模型的免费部署能力持续压制定价上限。厂商定价权来自「服务便利性」而非「模型独占性」。

趋势 4: Agent 时代的成本指数级增长不可逆

推理需求 2026 年已达训练的 4-5 倍，IDC 预测 2028 年推理占比 73%。Coding Plan「不限量」模式不可持续。所有厂商最终转向 Token 精细计量——谁先转谁主动，谁后转谁被动（Cursor 2025-06 的教训）。

判断：Coding Plan 是过渡产品，Token Plan 是终态。这个转变不是可选的，而是 Agent 时代的必然。2026 年底前，所有主要平台都会完成计量方式的统一化。
06 · Checklist

实战清单

从这个案例中提炼的决策检查项，可直接用于产品定价和商业模式设计。

定价策略决策清单

  • 定义计量单位：用什么计费？（Token / 请求 / 次数 / Credit）——计量单位的选择决定了用户感知和利润空间
  • 选择获客策略：补贴还是差异化？——补贴期亏损换规模，差异化期溢价换利润
  • 设计分层结构：几档？价差多少？——入门档获客，Pro 档盈利，企业档溢价
  • 确定透明度：额度描述要多透明？——模糊有利于早期获客，透明有利于长期信任
  • 规划演进路径：从什么到什么？——从「卖工具」到「卖算力」到「卖生态」

商业模式转型检查点

  • 用户成本可预测性：用户能不能算清楚自己要花多少？——不可预测 = 不信任 = 流失
  • 利润弹性：收紧定价后用户留存如何？——如果收紧就流失，说明价值不够
  • 生态粘性：从单一场景到多场景的升级是否自然？——Token Plan 打包全模态是正确方向
  • 竞争护城河：计量单位/定价结构是否容易被复制？——计量创新容易被跟进，服务创新不易

反模式（踩过的坑）

  • Cursor 2025-06 退款风波：突然改变计费模式，用户反弹。教训：计量方式变更需要过渡期和用户沟通。
  • 计量单位混乱：90% 用户踩坑，信息不对称最终伤害信任。教训：短期获利，长期失血。
  • 饥饿营销反噬：每天 10:30 抢购，短期有效，长期伤品牌。教训：稀缺感制造焦虑，不是忠诚。
  • 低价陷阱：MiMo 官方自己说「低价竞争是陷阱」——低价入局容易，活下去需要价值支撑。
  • 不限量承诺：Agent 时代一个任务 = 几十次调用，「不限量」不可持续。教训：承诺要留余量。
