# Checklist — 验收清单

## 叙事验收

- [ ] 标题是结论性（"毅航领跑"），不是描述性（"供应商业绩"）
- [ ] 一眼能看出谁最高谁最低
- [ ] 基准线清晰区分安全/危险区

## 视觉验收

- [ ] 毅航 Warm 500 (#c26d3a) 高亮
- [ ] 其余柱体统一 Stone 300 (#ada599) 灰化
- [ ] 高亮比例 ≤10%（仅毅航 1 根柱体）
- [ ] 基准线虚线样式
- [ ] Y 轴 min = 0
- [ ] 网格线虚线（不是实线）

## 标注验收

- [ ] 毅航柱顶有数值标注
- [ ] 赛维斯有"低于清退阈值"标注
- [ ] 标注写原因/幅度，不只写数字

## 排版验收

- [ ] 字号层级：标题 16px > 标注 11px > 轴标签 11px
- [ ] 四周留白 ≥20px
- [ ] 无渐变/无阴影/无发光
- [ ] 数据真实，无编造值
