# 金条人力供给 — 数字策展方案

## 阶段1：选题界定

**策展对象**：金条业务线 BPO 供应商人力供给体系
**进入尺度**：数字（5,174 / 746 / 130）→ 观众从具体数字进入
**真正主体**：人力供给的结构性分化——基本盘已成型，长尾在分化，脉冲效应掩盖了衰减趋势
**目标观众**：数据科技业务部管理层（供应商管理决策）
**材料边界**：17个月首呼数据 · 11个职场 · 同比 · 异常检测
**排除**：其他业务线 · 财务 · 质检 · 当月外呼

**推荐 framing**：职场剖面 × 结构分化

## 阶段2：策展文本

### 展览标题
**5,174 人——金条人力供给的结构性分化**

### 导言
> 2025年1月至2026年5月，11家 BPO 职场累计向金条业务线输送 5,174 名首次外呼人力。数字背后不是均匀的供给曲线，而是一场正在发生的结构性分化：三家职场已经占据总量的 41%，两家老牌职场进入下降通道，一家新入场者成为当月第一，一家几乎归零。

### 单元一：17 个月的脉搏
- **标题**：从 59 到 746，再回到 147
- **定位句**：三次脉冲（2025-02 节后、2025-09 旺季、2026-03 春节后），每次高度递增，但回落越来越深
- **策展主题**：季节性脉冲 vs 趋势性衰减——Q1 同比扩张 +35%，Q2 已转负
- **正文**：（趋势分析，见原始数据第一节）

- **标题**：前三名占据四成
- **定位句**：伽玛 + 毅航 + 毛毛虫 = 2,126 人 = 41.1%
- **策展主题**：稳定高产型 vs 爆发型 vs 可靠型——三种供给模式各有价值

## 阶段4：视觉语言

### 参照系
**分析型仪表册**——不是艺术品，是决策工具。参照 McKinsey 季度报告的信息密度和瑞士表格的排版精度。

### 材料规则
- 证据材料：数值、百分比、排名 → Space Mono
- 定位材料：月份标签、职场名称 → Space Grotesk
- 分析材料：趋势线、进度条、同比条 → Canvas / CSS
- 无氛围材料（纯数据展览，不加纹理/声音/装饰图像）

### 字级层级
- Hero 数字：Doto 72px（5,174）
- 章节标题：Space Grotesk 24px Medium
- 定位句：Space Grotesk 18px
- 正文：Space Grotesk 16px
- 数据标签：Space Mono 11px ALL CAPS
- 图注/来源：Space Mono 12px

### 色彩系统
- 亮色模式：#F5F5F5 底 · #FFF 卡片 · #000 主文字
- 数据状态：绿 #3B8A4A（增长/正）· 红 #D71921（下降/负/异常）· 琥珀 #B8922E（注意）
- 趋势标签：与数据状态色对齐
- 无装饰色

### 交互语法
- 纵向滚动为主
- 无 hover 动效（报告页不是交互玩具）
- 条形图入场动画（加载时从 0 展开）
- 编辑器：可修改文字和数字

### 排除方向
- 禁止暗色主题
- 禁止 emoji
- 禁止渐变、阴影、圆角 >16px
- 禁止插画/图标装饰
