## Big Idea

首贷与复贷绩效表现呈双极化格局：**S 级人均 GMV 约 364.09 万，是 C 级 61.95 万的 5.9 倍**。核心矛盾在"有效→首贷转化"环节（S 级 3.49% vs C 级 1.24%，差 2.8 倍），以及 管户金条-翰锐曲靖职场（C 级率 42.1%）等职场的系统性 C 级集中。建议：①C 级高发职场启动专项诊断；②新人前置培训体系降低 MOB1-3 C 级率（56人，C 级率 37.6%）；③头部经验萃取复制。

置信度：高

## 反对证据与局限性

- 单期截面数据（近 30 日），无法判断季节性波动和长期趋势
- 转化率分母为接通用户数而非名单下发数（已在备注中说明待改）
- 缺少质检合规数据，无法评估服务质量维度

## 底部总结

首贷大盘 GMV 99,663 万、复贷 375 万。核心矛盾不在接通量而在转化能力。管户金条-翰锐曲靖职场/管户金条-伽玛职场/管户金条-汇讯职场 三个职场需立即诊断，且建议将"有效→首贷"转化率作为坐席培训的核心 KPI。
