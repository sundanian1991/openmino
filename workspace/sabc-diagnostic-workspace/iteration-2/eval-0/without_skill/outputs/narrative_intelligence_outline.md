# 金条坐席 SABC 评级 — 叙事情报大纲

## 背景

基于近 30 日（截止 2026-04-19）CRM 系统数据，对 12 个职场 1,032 名坐席进行 SABC 评级诊断。首贷参评 531 人，复贷参评 593 人。

---

## 一、全景素描

- **总量**：首贷总坐席 1,032 人，参评 531 人（51.4%），非参评 501 人（48.6%）
- **非参评原因**：超 7 天未上线 40 人，外呼天数不足 22 天 502 人（部分重叠）
- **评级结构**：S 级 106（20.0%）、A 级 159（29.9%）、B 级 159（29.9%）、C 级 107（20.2%）
- **复贷对照**：S 级 118（19.9%）、A 级 178（30.0%）、B 级 178（30.0%）、C 级 119（20.1%）

**核心叙事**：首贷和复贷的等级结构几乎完全相同，均为对称的"橄榄型"。但 C 级体量与 S 级相当，说明系统同时面临顶部人才储备不足和底部人员积压的双重挑战。

## 三、帕累托分析

- **S 级集中度**：毅航合肥占全部 S 级的 42.5%，前 3 职场（毅航+毛毛虫+博岳）占 73.6%
- **C 级集中度**：毅航合肥占全部 C 级的 28.0%，前 3 职场（毅航+伽玛+汇讯）占 62.6%
- **帕累托矩阵**：伽玛、翰锐曲靖、汇讯落在"低 S + 高 C"象限，为系统性风险点

---

## 四、司龄效应

| 司龄段 | S | A | B | C | C 级占比 |
|--------|---|---|---|---|---------|
| MOB1-（<1月） | 0 | 1 | 1 | 2 | 50.0% |
| MOB1-3（1~3月） | 15 | 31 | 47 | 56 | **37.6%** |
| MOB3-6（3~6月） | 8 | 13 | 16 | 6 | 14.0% |
| MOB6-12（6~12月） | 14 | 23 | 14 | 7 | 12.1% |
| MOB12+（>1年） | 69 | 91 | 81 | 36 | 13.0% |

**关键洞察**：
- S 级中 65.1% 来自 MOB12+，说明经验积累是 S 级的核心来源
- C 级中 **52.3%** 来自 MOB1-3，新人保护机制不足
- MOB3-6 是评级跃升的关键窗口——C 级占比从 37.6% 骤降至 14.0%
- MOB12+ 仍有 36 名"老 C 级"（13.0%），需个别诊断

## 五、GMV 驱动因素

**GMV 排序**（首贷 T0）：
1. 毅航合肥：29,131 万（38.0%）
2. 毛毛虫：18,587 万（24.2%）
3. 汇讯：6,105 万（8.0%）
4. 伽玛：5,469 万（7.1%）
5. 赛维斯：5,337 万（7.0%）

**驱动因素对比**：
- **规模为王**：毅航合肥日均通时 123 分钟、件均 39,647 元、转化率 0.92%，三项指标均非第一，但人员规模驱动总 GMV 遥遥领先
- **高转化≠高 GMV**：赛维斯转化率 1.02% 为最高，但因规模小，GMV 仅排第 5
- **件均策略**：毛毛虫件均 44,725 元为最高，转化率 0.93% 中等，走"高客单"路线
- **人和模式**：人和职场 S+A 占比最高、转化率 1.19%、无 C 级，为标杆模型

## 七、C 级人员全景诊断

### C 级分布三维矩阵

**按职场**：毅航合肥（30人）、伽玛（22人）、汇讯（15人）为 C 级最多的 3 个职场

**按司龄**：MOB1-3（56人）= 52.3%、MOB12+（36人）= 33.6%、MOB3-6（6人）= 5.6%

**按密集度**：
- 翰锐曲靖：42.1% → 每 2.4 人就有 1 个 C 级
- 伽玛：39.3% → 每 2.5 人就有 1 个 C 级
- 汇讯：31.9% → 每 3.1 人就有 1 个 C 级

### C 级画像

C 级典型画像包括两类：
1. **新人C级**（52.3%）：入职 1-3 个月，转化率低、通时不足，培训转化失效
2. **老员工C级**（33.6%）：入职超过 1 年仍为 C 级，可能存在技能瓶颈或态度问题

---

## 八、核心结论与建议

### 结论

1. **结构平衡但底部风险不容忽视** — S（20%）与 C（20%）体量相当，系统存在"低绩效滞留"问题
2. **职场分化严重** — 毅航合肥一家独大，但翰锐曲靖、伽玛结构严重失衡
3. **司龄是关键杠杆** — MOB1-3 到 MOB3-6 的过渡期是决定坐席命运的关键窗口
4. **规模效应主导 GMV** — 转化率领先不等于 GMV 领先，人员规模和通时覆盖率才是决定性因素

### 建议

**近期动作**：
- 对翰锐曲靖、伽玛、汇讯三个高风险职场启动专项干预
- 对 MOB12+ 的 36 名"老 C 级"开展一对一诊断面谈
- 建立新人保护期（0-3 月）的渐进式考核机制

**中期优化**：
- 萃取人和职场的"小规模高绩效"模式，向伽玛、赛维斯等中等规模职场推广
- 优化 MOB1-3 到 MOB3-6 的培训衔接，将新人 C 级转化率降低 50%
- 建立 C 级月度跟踪机制，防止 C 级人员长期滞留

**长期建设**：
- 毅航合肥模式（大规模+高件均+中转化）成本可控性验证
- 探索「转化率 × 规模」双维度人才选拔通道
