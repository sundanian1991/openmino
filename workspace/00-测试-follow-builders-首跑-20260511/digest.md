### Aaron Levie — Box CEO

Aaron Levie 发表长文分析 AI agent 对就业市场的影响。核心观点：agent 确实让入门复杂领域变得更容易，但有经验的人借助 agent 的输出质量远超新手。他们知道何时 agent 犯了致命错误，能给 agent 提供正确的上下文。结果和所有技术革命一样——更大众化的同时，专家产出更高，需求不会消失。"你会更信任用 agent 的律师，而不是从未见过合同实际执行效果的人。"

### Training Data: ElevenLabs 联合创始人 Mati Staniszewski — 语音如何成为万物的接口

**一句话核心**：ElevenLabs 在 2022 年加密货币和元宇宙热潮中选择押注音频 AI，如今已突破 4 亿美元 ARR，Q1 新增净收入超 1 亿美元——而这一切全靠"选了一条没人走的路"。

ElevenLabs 的故事从波兰华沙郊区的两个好朋友开始。波兰有个奇特的现象：所有外语电影的配音都由**一个**单调的旁白角色完成，男女角色不分。这种体验让 Mati 和联合创始人 Piotr 意识到音频领域的巨大缺口——未来，语音将是人机交互的主要界面。

2022 年，当所有人都在炒加密货币和元宇宙时，音频 AI 还是一个被忽视的利基市场。音频模型比文本和视觉模型小，计算需求更低，数据标注虽然量大但可控。他们选择完全远程组建团队，在 GitHub 上挖研究人员，快速上线产品并迅速变现，用收入反哺模型研发。

如今 ElevenLabs 覆盖了音频全栈：文本转语音、语音转文本、实时流式对话、语音 agent 引擎，甚至音乐生成。Mati 分享了一个标志性时刻——他们让 AI 学会了"笑"，冲上了 Hacker News 头条。而 Matthew McConaughey 用 AI 克隆自己的声音说西班牙语和葡萄牙语，家人第一次能"听到他用这些语言说话"，是他们近期最兴奋的成果。

**语音 agent 的下一个前沿**：情感智能。当用户焦虑时，agent 能给出安抚的情绪；当用户兴奋时，匹配那种情绪；当用户说话慢下来，agent 也慢下来。"我们终于看到了一条解决这条路。"此外，将叙事、停顿、同一声音的歌唱融合到一个连续流中——"音频通用智能"——也即将到来。

**创业建议方面**，Mati 分享了几个反直觉的做法：400 人、4 亿多美元营收的团队，每个小组不超过 10 人，包括法务和市场团队都配了工程师；全公司没有职级头衔；非技术团队嵌入技术资源帮助自动化。

关于 agent 之间的通信，ElevenLabs 做过一个 hackathon：两个 voice agent 对话时检测到对方都是 agent，直接切换到了一种更高效的传输语言。"100% 会发生这种事。"
