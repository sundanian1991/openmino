# C2电销BPO质量管理规范

**文件编号**：SM-2026-005
**版本**：V1.0
**生效日期**：2026年XX月XX日
**发文单位**：C2电销供管岗

---

## 第一章 总则

### 第一条 目的

明确供管岗在质量管理中的角色定位，规范质检数据产出、传递、应用的全流程，确保质检数据客观、公正、有效，为供应商评估提供可靠依据。

### 第二条 角色定位原则

**核心原则**：不替代质检、不绕过质检

| 角色 | 职责边界 |
|------|----------|
| 质检团队 | 独立执行质检，产出质检数据，对数据质量负责 |
| 供管岗 | 管理质检数据产出和应用，不参与质检执行，不替代质检判断 |
| 供应商 | 接受质检，建立内部质量机制，配合质检工作 |

### 第三条 适用范围

全体电销BPO供应商及供管岗、质检团队。

---

## 第二章 对质检团队的要求

### 第四条 质检月报要求

**报告内容**：

| 内容项 | 说明 |
|--------|------|
| 质检概况 | 质检坐席数、抽检录音数、抽检覆盖率 |
| 话术合规得分 | 各供应商话术合规率 |
| 录音完整性 | 各供应商录音留存情况 |
| 问题分类统计 | 问题类型分布、问题严重程度分布 |
| 典型问题案例 | 问题录音编号、脱敏后的问题描述 |
| 趋势分析 | 与上月对比、问题趋势预判 |

**报告提交**：

| 要求 | 说明 |
|------|------|
| 提交时间 | 每月第3个工作日18:00前 |
| 提交形式 | Excel + Word双版本 |
| 报告命名 | `质检月报_YYYYMM_版本号.xlsx` |
| 确认签收 | 供管岗回复确认 |

### 第五条 质检覆盖率要求

**覆盖率标准**：

| 坐席规模 | 月度抽检覆盖率 | 说明 |
|----------|----------------|------|
| ≤50人 | 100% | 全员覆盖 |
| 51-100人 | ≥50% | 不少于50人 |
| 101-300人 | ≥30% | 不少于30人 |
| >300人 | ≥20% | 不少于60人 |

**抽检录音数量**：

```
最低抽检数 = MIN(在岗坐席数 × 覆盖率, 月度有效录音总数)
```

### 第六条 质检校准机制

**校准周期**：每季度一次

**校准内容**：

1. 质检标准一致性校准（不同质检员对同一录音的评价一致性）
2. 评分标准更新（如有话术更新，及时更新质检标准）
3. 特殊场景处理方式统一

**校准流程**：

```
校准会议 → 抽取样本录音 → 盲测评分 → 评分差异讨论 → 标准统一 → 形成校准纪要
```

**校准记录**：

| 记录项 | 说明 |
|--------|------|
| 校准日期 | 校准会议日期 |
| 参与人员 | 全体质检员 |
| 样本数量 | 抽取评分的录音数量 |
| 一致性指标 | Kappa系数或类似指标 |
| 差异处理 | 评分差异的处理结果 |

## 第三章 对供应商的要求

### 第七条 接受质检义务

**配合义务**：

1. 不得拒绝、干扰、阻碍正常质检工作
2. 不得对质检员进行威胁、贿赂、打击报复
3. 不得伪造、篡改质检对象信息

**违规处理**：

| 违规情形 | 处理措施 |
|----------|----------|
| 拒绝配合质检 | 扣10分/次，暂停新产能分配 |
| 干扰质检工作 | 扣15分/次，约谈负责人 |
| 威胁/报复质检员 | 直接清退 |

### 第八条 内部质量机制要求

**最低要求**：

| 要求项 | 标准 |
|--------|------|
| 内部质检团队 | 50人以下供应商须有兼职质检员；50人以上须有专职质检团队 |
| 自检覆盖率 | 月度自检覆盖率≥20% |
| 问题整改率 | 自检发现问题整改率≥95% |
| 培训计划 | 每月至少2学时合规培训 |

**机制建设**：

| 机制 | 要求 |
|------|------|
| 每日自检 | 每日抽取当天录音进行内部自检 |
| 周度复盘 | 每周汇总自检问题，分析改进 |
| 月度培训 | 每月至少2学时合规培训，覆盖全员 |
| 问题库更新 | 质检反馈问题24小时内更新至内部问题库 |

### 第九条 录音管理要求

**录音留存**：

1. 留存时长：不少于6个月
2. 留存格式：标准WAV/MP3格式
3. 存储位置：公司授权系统
4. 备份要求：本地+云端双重备份

**录音调取**：

| 情形 | 提供时限 | 申请流程 |
|------|----------|----------|
| 常规质检 | 实时 | 质检系统自动调取 |
| 专项检查 | 2小时内 | 书面申请 → 审批 → 提供 |
| 客户投诉核查 | 1小时内 | 电话/邮件通知 → 即时提供 |

## 第四章 质量数据在评估中的应用

### 第十条 数据引用原则

**数据流**：

```
质检团队 → 质检月报 → 供管岗 → 评估数据 → 月度评估
```

**引用原则**：

1. 以质检月报数据为准
2. 数据须完整、可追溯
3. 数据争议时启动复核流程

### 第十一条 评估数据应用

**话术合规率应用**：

```
话术合规率 = 合格录音数 / 抽检录音总数 × 100%
得分 = 按SM-2026-004第二章第五条执行
```

**录音完整性应用**：

```
录音完整率 = 完整录音数 / 应留存录音总数 × 100%
得分 = 按SM-2026-004第二章第五条执行
```

**客户投诉率应用**：

```
投诉率 = 投诉单数 / 总外呼量 × 100%
得分 = 按SM-2026-004第二章第五条执行
```

### 第十二条 数据争议处理

**争议触发**：供应商对质检数据有异议

**争议时效**：公示后3个工作日内提出

**争议流程**：

```
提出争议（书面） → 质检团队核查 → 给出核查意见 → 供管岗裁决 → 执行
```

**争议处理原则**：

1. 以录音原始记录为准
2. 评分差异由质检团队复核确认
3. 供管岗不替代质检判断

## 第五章 兜底机制

### 第十三条 质检数据缺位处理

**缺位情形**：

| 情形 | 说明 |
|------|------|
| 质检报告延迟 | 质检月报未在规定时间提交 |
| 覆盖率不足 | 实际抽检覆盖率低于规定标准 |
| 数据明显异常 | 数据与实际情况严重不符 |
| 供应商不配合 | 供应商拒绝/干扰质检 |

**缺位时的处理原则**：

1. **不替代**：供管岗不自行开展质检
2. **不搁置**：不影响月度评估按时进行
3. **预估分**：缺位指标按预估分计算（取该项满分的80%）

### 第十四条 预估分机制

**预估分触发条件**：质检报告延迟提交超过5个工作日，或覆盖率低于标准的50%

**预估分计算**：

| 指标 | 预估分计算 |
|------|------------|
| 话术合规率 | 按该项满分×80%计算 |
| 录音完整性 | 按该项满分×80%计算 |
| 客户满意度 | 按该项满分×80%计算 |
| 投诉率 | 按该项满分×80%计算 |

**预估分修正**：

1. 质检数据补齐后，按实际数据重新计算
2. 重新计算结果覆盖预估分
3. 重新计算的得分与预估分差异不追溯调整

### 第十五条 供应商不配合质检的处理

**处理措施**：

| 不配合情形 | 第一次 | 第二次 | 第三次 |
|------------|--------|--------|--------|
| 拒绝抽检 | 扣10分，书面警告 | 扣20分，暂停新产能 | 清退评估 |
| 不提供录音 | 扣10分，限期提供 | 扣20分，暂停业务 | 清退评估 |
| 干扰质检 | 扣20分，约谈负责人 | 清退评估 | - |

## 第六章 质检与评估联动

### 第十六条 联动机制

**月度联动流程**：

```
T+3日：质检提交月报
     ↓
T+4日：供管岗核对数据
     ↓
T+5日：月度评估结果公示
     ↓
T+7日：评估结果生效应用
```

### 第十七条 重大问题升级

**升级条件**：

1. 单月话术合规率低于80%
2. 发现重大违规行为
3. 同一问题重复出现3次以上

**升级处置**：

```
升级报告 → C2电销供管岗负责人确认 → 通知供应商 → 启动专项整改 → 整改验收
```
