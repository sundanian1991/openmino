# C2电销BPO供应商月度综合评估方案

**文件编号**：SM-2026-004
**版本**：V1.0
**生效日期**：2026年XX月XX日
**发文单位**：C2电销供管岗

---

## 第一章 总则

### 第一条 目的

建立统一、公平、量化的供应商月度评估体系，为供应商分级管理、奖惩决策提供客观依据，促进供应商持续提升服务质量。

### 第二条 评估周期

每月一次，评估周期为自然月。评估结果于次月第5个工作日前完成公示。

### 第三条 评估维度与分值

| 评估维度 | 满分 | 说明 |
|----------|------|------|
| 交付质量 | 40分 | 引用质检数据 |
| 业务达成 | 30分 | 业绩完成情况 |
| 合规表现 | 20分 | 合规指标达标情况 |
| 管理配合 | 10分 | 信息报送、异常响应等 |
| **合计** | **100分** | |

---

## 第二章 交付质量评分规则（40分）

### 第四条 质检数据引用

质检数据由质检团队提供，供管岗引用后纳入评估。

**数据来源**：

1. 月度质检报告（主数据）
2. 专项质检数据（补充数据）
3. 客户投诉数据（参考数据）

### 第五条 交付质量评分细则

| 评分项 | 维度 | 分值 | 评分规则 |
|--------|------|------|----------|
| 话术合规率 | 交付质量 | 15分 | 达标线≥98%：15分；95%≤x<98%：12分；90%≤x<95%：8分；<90%：0分 |
| 录音完整性 | 交付质量 | 10分 | 达标线100%：10分；98%≤x<100%：8分；95%≤x<98%：5分；<95%：0分 |
| 客户满意度 | 交付质量 | 8分 | 达标线≥90%：8分；85%≤x<90%：6分；80%≤x<85%：3分；<80%：0分 |
| 投诉率 | 交付质量 | 7分 | 达标线≤1%：7分；1%<x≤2%：5分；2%<x≤3%：2分；>3%：0分 |

**数据获取要求**：

| 数据类型 | 提供方 | 提供时间 | 覆盖率要求 |
|----------|--------|----------|------------|
| 话术合规率 | 质检团队 | 每月第3个工作日 | ≥30%坐席 |
| 录音完整性 | 质检团队 | 每月第3个工作日 | 100% |
| 客户满意度 | 客服/业务方 | 每月第3个工作日 | - |
| 投诉率 | 客服/业务方 | 每月第3个工作日 | - |

## 第三章 业务达成评分规则（30分）

### 第六条 业绩数据评分

| 评分项 | 维度 | 分值 | 评分规则 |
|--------|------|------|----------|
| 目标达成率 | 业务达成 | 15分 | ≥100%：15分；95%≤x<100%：12分；90%≤x<95%：8分；80%≤x<90%：4分；<80%：0分 |
| 人均产能 | 业务达成 | 8分 | ≥目标值110%：8分；100%≤x<110%：6分；90%≤x<100%：4分；<90%：0分 |
| 产能稳定性 | 业务达成 | 7分 | 周间波动≤10%：7分；10%<x≤20%：5分；20%<x≤30%：2分；>30%：0分 |

**数据来源**：业务方提供的月度业绩数据

### 第七条 产能稳定性计算

```
周间波动率 = MAX(周转化量) / MIN(周转化量) - 1
产能稳定性 = 1 - 周间波动率
```

## 第四章 合规表现评分规则（20分）

### 第八条 合规指标评分

| 评分项 | 维度 | 分值 | 评分规则 |
|--------|------|------|----------|
| 人员持证率 | 合规表现 | 8分 | 100%：8分；98%≤x<100%：6分；95%≤x<98%：3分；<95%：0分 |
| 数据准确率 | 合规表现 | 6分 | ≥99%：6分；97%≤x<99%：4分；95%≤x<97%：2分；<95%：0分 |
| 安全生产 | 合规表现 | 6分 | 无安全事故：6分；一般事故：3分；较大事故：0分 |

### 第九条 合规加分项

| 加分项 | 分值 | 说明 |
|--------|------|------|
| 主动发现并报告安全隐患 | +2分/次 | 最高加4分 |
| 配合监管检查 | +2分/次 | 最高加4分 |
| 合规培训覆盖率100% | +2分 | 须提供培训记录 |

### 第十条 合规扣分项

| 扣分项 | 分值 | 说明 |
|--------|------|------|
| 触碰信息安全红线 | -20分 | 直接清退 |
| 触碰数据造假红线 | -20分 | 直接清退 |
| 触碰录音篡改红线 | -20分 | 直接清退 |
| 无证人员上岗 | -5分/人 | 最高扣10分 |
| 数据报送延迟 | -2分/次 | 最高扣10分 |
| 异常未按时报备 | -3分/次 | 最高扣10分 |

## 第五章 管理配合评分规则（10分）

### 第十一条 管理配合评分

| 评分项 | 维度 | 分值 | 评分规则 |
|--------|------|------|----------|
| 信息报送及时性 | 管理配合 | 4分 | 100%及时：4分；98%≤x<100%：3分；95%≤x<98%：2分；<95%：0分 |
| 会议参与度 | 管理配合 | 3分 | 全勤：3分；请假1次：2分；缺席1次：1分；缺席2次及以上：0分 |
| 整改完成率 | 管理配合 | 3分 | 100%完成：3分；80%≤x<100%：2分；60%≤x<80%：1分；<60%：0分 |

## 第六章 评级标准

### 第十二条 五级评级

| 评级 | 分值区间 | 说明 |
|------|----------|------|
| A级 | 90分≤x≤100分 | 优秀 |
| B级 | 80分≤x<90分 | 良好 |
| C级 | 70分≤x<80分 | 合格 |
| D级 | 60分≤x<70分 | 需改进 |
| E级 | x<60分 | 不合格 |

### 第十三条 一票否决规则

**以下情形直接判定为E级，并启动清退评估**：

| 否决项 | 说明 |
|--------|------|
| 信息安全红线 | 发生信息泄露事件 |
| 数据造假红线 | 确认存在数据造假行为 |
| 录音篡改红线 | 存在录音篡改或删除行为 |
| 重大安全事故 | 发生人员伤亡或重大财产损失 |
| 违法经营 | 无证经营或超范围经营 |
| 拒绝整改 | 拒绝执行整改要求 |

## 第七章 评估结果应用

### 第十四条 评估结果公示

**公示内容**：

1. 各供应商综合得分及评级
2. 各维度得分明细
3. 排名情况
4. 环比变化

**公示时间**：每月第5个工作日

**公示方式**：邮件发送 + 公告系统发布

### 第十五条 A级供应商激励

| 激励措施 | 说明 |
|----------|------|
| 优先分配新业务 | 在同等条件下优先获得新业务 |
| 产能增加 | 可申请增加坐席规模 |
| 续约优先 | 合同到期时优先续约 |
| 公开表彰 | 季度会议上公开表扬 |

### 第十六条 B/C级供应商管控

**B级供应商**：

1. 正常运营
2. 鼓励向A级努力

**C级供应商**：

1. 要求提交改进计划
2. 加大质检比例
3. 逐步缩减不达标业务

### 第十七条 D级供应商整改

**触发条件**：月度评估为D级

**整改要求**：

1. 5个工作日内提交整改计划
2. 整改期限30天
3. 每两周汇报整改进度
4. 整改期间暂停新产能分配

**整改验收**：

| 整改结果 | 后续措施 |
|----------|----------|
| 整改通过 | 恢复常态管理，下月评估若达C级则解除整改 |
| 整改未通过 | 继续整改或启动清退评估 |

### 第十八条 E级供应商处置

**触发条件**：月度评估为E级

**立即措施**：

1. 暂停所有新产能分配
2. 约谈供应商负责人
3. 制定紧急整改计划

**整改期限**：最长60天

**整改期间要求**：

1. 每周评估一次
2. 产能逐步缩减
3. 做好业务交接准备

**整改结果处置**：

| 整改结果 | 后续措施 |
|----------|----------|
| 连续2个月E级 | 启动清退流程 |
| 触碰一票否决项 | 立即启动清退流程 |

## 第八章 申诉机制

### 第十九条 申诉条件

供应商对评估结果有异议的，可在公示后5个工作日内提出申诉。

**申诉范围**：

1. 评估数据计算错误
2. 评估依据不充分
3. 评分标准执行有误

**不受理申诉**：

1. 评级结果符合一票否决条件
2. 申诉超过规定时限
3. 申诉理由不属于本规定范围

### 第二十条 申诉流程

```
提交申诉表 → 受理审核（3个工作日）→ 独立复核（5个工作日）→ 给出复核意见 → 执行复核结果
```

### 第二十一条 独立复核人

**复核人员组成**：

| 角色 | 人数 | 说明 |
|------|------|------|
| C2电销供管岗 | 1人 | 原始评分方 |
| 业管代表 | 1人 | 业务方代表 |
| 质检代表 | 1人 | 质检数据提供方 |
| 纪检/合规 | 1人 | 独立第三方 |

**复核原则**：

1. 独立、客观、公正
2. 仅复核评估过程，不重新评估
3. 复核结果为最终结果

### 第二十二条 申诉表格

统一使用《申诉表》（模板SM-TPL-010）

## 第九章 长期应用

### 第二十三条 续约决策

| 评估结果 | 续约建议 |
|----------|----------|
| 年度平均评级A级 | 优先续约，可签长期合同 |
| 年度平均评级B级 | 正常续约 |
| 年度平均评级C级 | 谨慎续约，加强管控 |
| 年度有D级或E级 | 不续约或仅短期续约 |

### 第二十四条 淘汰决策

**淘汰条件**：

1. 连续2个月E级
2. 触碰一票否决项
3. 整改期满未达到整改目标
4. 年度平均评级为E级

**淘汰流程**：

```
淘汰决策 → 提前30天通知 → 业务交接 → 数据归档 → 关系终止 → 1年内禁止准入
```
