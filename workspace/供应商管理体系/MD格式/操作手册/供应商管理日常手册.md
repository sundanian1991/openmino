# C2电销BPO供应商管理日常手册

**文件编号**：SM-2026-002
**版本**：V1.0
**生效日期**：2026年XX月XX日
**发文单位**：C2电销供管岗

---

## 第一章 总则

### 第一条 目的

本手册明确供应商日常管理中各角色的具体职责、操作规范、时间节点，确保管理动作标准化、执行有据可查。

### 第二条 适用范围

全体电销BPO供应商及供管岗、业务方、质检团队、站点管理人员。

## 第二章 信息报送规范

### 第三条 报送分类与时间节点

| 报送类型 | 报送频率 | 截止时间 | 报送方 | 接收方 |
|----------|----------|----------|--------|--------|
| 每日出勤 | 每日 | 10:00前 | 供应商 | 供管岗 |
| 每日业绩 | 每日 | 20:00前 | 供应商 | 供管岗 |
| 周报 | 每周一 | 12:00前 | 供应商 | 供管岗 |
| 月度总结 | 每月第2个工作日 | 18:00前 | 供应商 | 供管岗 |
| 花名册 | 实时+月度汇总 | 变动24h内/每月末 | 供应商 | 供管岗 |

### 第四条 每日出勤报送规范

**报送内容**：

| 字段 | 说明 |
|------|------|
| 日期 | 统计日期（YYYY-MM-DD） |
| 应到人数 | 当日应出勤坐席数 |
| 实到人数 | 当日实际出勤坐席数 |
| 出勤率 | 实到/应到×100% |
| 请假人数 | 已请假人数 |
| 旷工人数 | 无故缺勤人数 |
| 在岗培训人数 | 参加培训人数 |

**格式要求**：

1. 统一使用《每日出勤报送表》（模板SM-TPL-001）
2. 以Excel格式提交，文件名：`日期_供应商名称_出勤.xlsx`
3. 数据须与站点打卡记录一致

**异常判定**：

| 情形 | 判定 | 处置 |
|------|------|------|
| 出勤率≥95% | 正常 | 归档 |
| 90%≤出勤率<95% | 关注 | 供应商说明原因 |
| 出勤率<90% | 异常 | 启动异常处理流程 |

### 第五条 每日业绩复盘规范

**报送内容**：

| 字段 | 说明 |
|------|------|
| 日期 | 统计日期 |
| 业务产线 | 金条/信用卡/财富等 |
| 外呼量 | 当日总外呼数量 |
| 接通量 | 实际接通数量 |
| 转化量 | 成功转化数量 |
| 转化率 | 转化量/接通量×100% |
| 人均产能 | 转化量/实到人数 |

**格式要求**：

1. 统一使用《每日业绩报送表》（模板SM-TPL-002）
2. 以Excel格式提交，文件名：`日期_供应商名称_业绩.xlsx`
3. 须与业务方系统数据一致

### 第六条 周报规范

**报送内容**：

1. **本周工作概述**（200字以内）
2. **业绩数据汇总**：每日数据汇总表 + 趋势分析
3. **人员变动情况**：本周入职、转正、离职人数
4. **合规执行情况**：培训完成率、证书持有情况
5. **问题与改进**：本周遇到的问题及下步改进措施
6. **下周工作计划**：重点工作和预期目标

**格式要求**：

1. 统一使用《周报模板》（模板SM-TPL-003）
2. 以Word格式提交，文件名：`周次_供应商名称_周报.docx`
3. 须加盖供应商公章或签字

### 第七条 月度总结规范

**报送内容**：

1. **月度业绩总结**：业绩数据、目标完成率、同比/环比分析
2. **人员管理总结**：团队规模变化、持证情况、培训情况
3. **合规执行总结**：质检得分、问题整改、风险事件
4. **供应商自评**：自我评价及改进计划
5. **下月工作计划**：重点工作和目标承诺

**格式要求**：

1. 统一使用《月度总结模板》（模板SM-TPL-004）
2. 以Word格式提交，文件名：`月份_供应商名称_月度总结.docx`
3. 须加盖供应商公章

### 第八条 花名册管理规范

**花名册内容**：

| 字段 | 说明 |
|------|------|
| 员工编号 | 唯一编号 |
| 姓名 | 真实姓名 |
| 身份证号 | 18位身份证号 |
| 岗位 | 坐席/组长/主管等 |
| 入职日期 | 入职时间 |
| 资质证书 | 证书名称+到期日期 |
| 合同到期日 | 劳动合同到期时间 |
| 状态 | 正常/离职/停薪等 |

**实时变动报备**：

| 变动类型 | 报备时限 | 证明材料 |
|----------|----------|----------|
| 新入职 | 入职当日 | 身份证、资质证书复印件 |
| 离职 | 离职当日 | 离职申请表、工作交接单 |
| 岗位调动 | 调动生效日 | 调动通知 |
| 资质证书到期 | 到期前30天 | 新证书或续期证明 |

**格式要求**：

1. 统一使用《花名册模板》（模板SM-TPL-005）
2. 以Excel格式提交，实时变动以《人员变动报备表》（模板SM-TPL-006）单独报送
3. 每月末提交完整花名册汇总

## 第三章 人员管理全流程

### 第九条 人员准入管理

**准入条件**：

1. 年龄18周岁以上
2. 持有有效身份证
3. 具备金融相关资质证书
4. 无犯罪记录
5. 通过供应商内部面试

**准入流程**：

| 步骤 | 操作 | 责任方 | 时限 |
|------|------|--------|------|
| 1 | 提交入职申请及资质证明 | 供应商 | 入职前3天 |
| 2 | 资质审核 | 供管岗 | 2个工作日 |
| 3 | 岗前培训（合规、业务） | 供应商 | 不少于20学时 |
| 4 | 培训考核 | 供应商+业务方 | 培训结束后 |
| 5 | 考核通过后上岗 | 供应商 | 考核通过当日 |

**禁止行为**：

1. 无证或持假证人员办理入职
2. 未经培训或考核未通过人员上岗
3. 冒用他人资质证书

### 第十条 试用期管理

**试用期时长**：3个月

**转正条件**：

| 条件 | 要求 |
|------|------|
| 培训完成率 | 100% |
| 考核合格率 | ≥90% |
| 出勤率 | ≥95% |
| 产能达标率 | ≥80% |

**试用期评估**：

1. 第1个月：培训完成情况 + 基本合规意识
2. 第2个月：产能达成 + 话术规范
3. 第3个月：综合评定 + 转正决策

**转正流程**：

```
试用到期评估 → 评估合格公示 → 转正审批 → 正式上岗
```

### 第十一条 人员变动管理

**变动类型**：

| 变动类型 | 说明 | 报备时限 |
|----------|------|----------|
| 入职 | 新员工入职 | 入职当日 |
| 转正 | 试用期转正 | 转正当日 |
| 调岗 | 岗位调整 | 生效当日 |
| 离职 | 主动或被动离职 | 离职当日 |
| 辞退 | 违反规定被辞退 | 决定当日 |

**离职交接规范**：

| 交接内容 | 操作要求 | 完成时限 |
|----------|----------|----------|
| 工作文档 | 整理归档，列出清单 | 离职前3天 |
| 客户信息 | 全部移交，不得带走 | 离职当日 |
| 系统账号 | 提交注销申请 | 离职当日 |
| 资产归还 | 归还工牌、电脑等 | 离职当日 |
| 培训费用 | 确认是否需要赔偿 | 离职当日 |

### 第十二条 持证上岗管理

**资质要求**：

1. 上岗30日内须取得金融相关资质证书
2. 证书须在有效期内
3. 证书信息与实际作业人员一致

**证书管理流程**：

```
入职登记 → 证书考取 → 证书备案 → 到期提醒 → 续期处理
```

**到期提醒机制**：

| 提醒时点 | 提醒对象 | 处理要求 |
|----------|----------|----------|
| 到期前60天 | 供应商+个人 | 提交续期计划 |
| 到期前30天 | 供应商+个人 | 确认续期进度 |
| 到期前7天 | 供管岗 | 最后通牒 |
| 到期当日 | 供管岗 | 无证人员下岗 |

## 第四章 异常处理机制

### 第十三条 异常分类

| 等级 | 分类标准 | 响应时限 | 处理时限 |
|------|----------|----------|----------|
| 一般异常 | 单点问题，影响有限 | 4小时 | 24小时 |
| 重要异常 | 较大范围影响，需协调处理 | 2小时 | 12小时 |
| 紧急异常 | 重大违规或业务中断风险 | 即时 | 4小时 |

### 第十四条 一般异常处理流程

**适用情形**：

- 单个坐席迟到、早退
- 单日出勤率低于90%
- 偶发数据报送延迟
- 轻微话术违规

**处理流程**：

```
发现问题 → 记录台账 → 通知供应商 → 供应商整改 → 整改验收 → 结案
```

**记录要求**：

1. 填写《异常处理记录》（模板SM-TPL-007）
2. 记录问题描述、发生时间、影响范围
3. 记录整改措施和结果

### 第十五条 重要异常处理流程

**适用情形**：

- 连续3天以上出勤率低于90%
- 单日超过5%人员异常缺勤
- 数据造假或疑似造假
- 重要岗位人员批量流动
- 资质证书集中到期

**处理流程**：

```
发现/通报 → 2小时响应 → 现场/会议沟通 → 整改方案提交 → 执行监督 → 验收结案
```

**升级机制**：

1. 供管岗直接约谈供应商负责人
2. 要求供应商3个工作日内提交整改方案
3. 暂停新产能分配，直至整改通过

### 第十六条 紧急异常处理流程

**适用情形**：

- 触碰合规红线（信息泄露、数据造假、录音篡改）
- 群体性事件（罢工、聚众闹事）
- 重大业务事故（系统崩溃、数据丢失）
- 突发法律诉讼或监管调查

**处理流程**：

```
即时响应 → 成立专项小组 → 48小时内形成处置方案 → 执行 → 善后 → 复盘
```

**即时措施**：

1. 立即暂停相关业务
2. 保护现场证据
3. 通知相关方
4. 启动清退评估

### 第十七条 异常升级路径

```
一般异常 → 供管岗 → 供应商驻场负责人
     ↓
重要异常 → 供管岗 → 供应商负责人 + 业管负责人
     ↓
紧急异常 → 供管岗 → C2电销供管岗负责人 → 公司合规 + 法务
```

## 第五章 沟通与会议机制

### 第十八条 日常沟通规范

**响应时限**：

| 消息类型 | 响应时限 | 超时处置 |
|----------|----------|----------|
| 普通咨询 | 4小时 | 催促一次 |
| 紧急咨询 | 1小时 | 升级处理 |
| 数据核对 | 2小时 | 催促一次 |
| 问题反馈 | 4小时 | 升级处理 |

**沟通渠道优先级**：

1. 正式公函（最优先，用于重要决策、正式通知）
2. 邮件（用于日常报送、记录备查）
3. 即时通讯（用于一般沟通、紧急联络）
4. 电话（用于紧急事项，需事后补发书面记录）

### 第十九条 周度复盘会议

**参会人员**：

| 参会方 | 参会人员 |
|--------|----------|
| 供管岗 | 主持 |
| 供应商 | 驻场负责人、业绩负责人 |
| 业务方 | 业管代表（必要时） |

**会议频次**：每周一次，周五14:00

**会议内容**：

1. 本周业绩数据复盘
2. 问题回顾与整改跟踪
3. 下周重点工作沟通
4. 异常事项通报

**会议输出**：

1. 会议纪要（24小时内发出）
2. 行动事项清单（明确责任人和完成时限）

### 第二十条 月度评估会议

**参会人员**：

| 参会方 | 参会人员 |
|--------|----------|
| 供管岗 | 主持 |
| 供应商 | 总经理或负责人 |
| 业务方 | 业管负责人（必要时） |
| 质检 | 质检负责人（报告环节） |

**会议频次**：每月第5个工作日

**会议内容**：

1. 上月评估结果通报
2. 质检数据汇报
3. 问题整改跟踪
4. 整改期供应商整改进度汇报
5. 下月工作重点与目标确认

**会议输出**：

1. 月度评估报告
2. 评估结果公示（排名公告）
3. 需整改供应商的整改计划确认

## 第六章 台账与档案管理

### 第二十一条 台账体系

**台账类型**：

| 台账类型 | 内容 | 更新频率 | 保管期限 |
|----------|------|----------|----------|
| 供应商基本信息台账 | 营业执照、联系人、资质等 | 实时更新 | 合作终止后5年 |
| 人员花名册台账 | 全体人员信息 | 实时更新 | 合作终止后5年 |
| 业绩数据台账 | 日/周/月业绩数据 | 每日更新 | 5年 |
| 异常处理台账 | 所有异常记录 | 实时更新 | 5年 |
| 评估结果台账 | 月度评估原始数据及结果 | 每月更新 | 5年 |

### 第二十二条 档案管理规范

**档案分类**：

| 档案类别 | 内容 |
|----------|------|
| 供应商档案 | 准入资料、合同、资质证明、评估记录 |
| 人员档案 | 花名册、资质证书、入离职记录 |
| 业务档案 | 业绩数据、复盘记录、会议纪要 |
| 合规档案 | 质检报告、整改记录、处罚记录 |

**保管要求**：

1. 电子档案与纸质档案并行保管
2. 电子档案存储于公司授权系统
3. 纸质档案存放于档案室
4. 涉及商业秘密的档案须加密

**借阅权限**：

| 人员 | 借阅权限 |
|------|----------|
| 供管岗 | 全部档案 |
| 业管 | 业务相关档案 |
| 质检 | 质检相关档案 |
| 供应商 | 仅可查阅自身相关档案 |
