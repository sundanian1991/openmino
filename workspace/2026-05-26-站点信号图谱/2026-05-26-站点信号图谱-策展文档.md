# Q1 站点信号图谱 — 研究型数字策展文档

> 数据源：站点2026.xlsx | 108站 28家供应商 15个项目 | 2026年1-3月在册人力
> 策展工作流：research-digital-curation-workflow 6阶段完整产出

---

## 一、选题与对象界定（curatorial-topic-framing）

### 策展对象

**2026 Q1 供应商站点的变化信号图谱**

### 进入尺度

变化率 × 供应商 × 项目

### 真实主体

人力规模的变化模式及其管理含义

### 目标观众

- 年老师（日常管理）
- 上级军哥/王易人（月度汇报）

### 材料边界

- 站点2026.xlsx，1-3月在册人力
- 不含业绩数据
- 不含薪酬、合规、质量数据

### 排除方向

- 不展开单站点运营细节
- 不做业绩归因
- 不进入薪酬或合规领域

### 选定 framing

**"增长与收缩的信号"作为主线，"供应商生命体征"作为底图。**

理由：年老师的核心工作是主动管理——不需要静态全景图（ERP能做），需要的是"哪里需要我去看"。信号导向让策展有叙事张力：不是描述"是什么"，而是追问"为什么在变"。

## 二、策展文本（research-to-curatorial-copy）

### 单元1：全景底图

**标题**：Q1 站点版图 — 谁在哪里，有多大

**定位句**：108个站点，28家供应商，3,340人。两张底图——按供应商看势力，按项目看战线。

**导言**：
这不是一张要读完的报表，是一张底图。后续所有信号判断都以此为参照系。记住三个数字：总量3,340人微增0.9%，金条占51%是基本盘，安徽+河北占61%是地理重心。

**观众进入层**：
- 毅航埇桥金条：190人，单体最大站点
- 毅航蜀山财富固收：98人，Q1净增30人
- 速讯达沈阳和平菁卡拉新：34人，从59人跌落

**策展主题层**：
规模的集中度与增长的分散性——少数供应商掌握多数人力，但增长信号来自边缘。

**正文**：
供应商人力集中度极高。前5家（毅航、毛毛虫、伽玛、汇讯、赛维斯）合计1,608人，占总量48%。但这5家Q1平均增速仅+7%，远低于岐力（+84%）、新动力（+43.8%）、华啸（+26.7%）等中小供应商。

地理上，安徽（1,140人）和河北（908人）是双中心。但Q1增量的主力不是这两个省份——江西净增63人（+84%），山东净增25人（+55.6%），湖南净增7人（+33.3%）。增长发生在边缘地带。

**证据层**：
- 数据源：站点2026.xlsx，2026年1-3月在册人力
- 4月数据仅毅航（527人）和华啸（90人）有值，未纳入同比
- 28家供应商中，阅文3月从0到16人（新开站点），统计增长率标为"新开"而非百分比

### 单元2：扩张信号

**标题**：正在变大的站点 — 谁在快速膨胀

**定位句**：5家供应商增速超过30%。有的是项目红利，有的是人力堆积。管理介入的窗口在信号出现时，不在结果确认后。

**导言**：
以下站点1-3月人力增长率超过40%（基准：1月在册10人以上）。增长本身不是问题，问题是增长是否可持续、是否有配套管理跟上。

**观众进入层**：
- 秦冀德辰辛集企业主贷：10→25人（+150%）——单项目最大增幅
- 岐力红谷滩金企融合：20→45人（+125%）——新产线爬坡
- 毅航蜀山财富固收：68→98人（+44%）——绝对增量最大
- 承德新动力兴隆企业主贷：36→53人（+47%）——从64人的小供应商里冒出来的

**策展主题层**：
快速扩张的三种模式——新产线爬坡（岐力金企融合）、老站点放量（毅航蜀山）、小供应商爆发（秦冀德辰辛集）。不同模式需要不同的管理关注点。

**正文**：

**新产线爬坡型**：岐力红谷滩金企融合从20人到45人，南昌县站点3月新开24人。金企融合是岐力的新方向，Q1总人力从75增至138人（+84%）。风险点：新团队的质量管控和培训是否跟上节奏。

**老站点放量型**：毅航蜀山财富固收从68人到98人，埇桥金条从170人到190人。毅航是最大供应商（3月481人，4月已到527人），整体增速+15.3%。风险点：大规模快速扩张时的管理半径。

**小供应商爆发型**：秦冀德辰辛集企业主贷+150%，但同站点金企融合-36%。这不是整体扩张，而是内部项目间的人力调配。锦瑞裕华企业主贷+91%同理。风险点：增长是真实的业务需求还是人力腾挪。

**证据层**：
- 增长率 = (3月人力 - 1月人力) / 1月人力 × 100%
- 秦冀德辰辛集同时运营企业主贷（+150%）、金企融合（-36%）、企业主贷另一线（-33%）三个项目，总站点人力从137降至108人（-21%）。所谓"增长"是项目间调配。
- 毅航4月数据527人，仍在扩张通道中

### 单元3：收缩信号

**标题**：正在缩小的站点 — 哪些需要马上看

**定位句**：菁卡拉新全线收缩是Q1最明确的系统性信号。其他收缩分散在不同供应商和项目，需要逐个判断是业务调整还是问题暴露。

**导言**：
以下站点1-3月人力下降超过25%（基准：1月在册10人以上）。收缩比扩张更需要管理介入——人力流失往往先于业绩恶化。

**观众进入层**：
- 汇讯合肥金条：35→16人（-54%）——同一供应商，洪山站点在增长，合肥在崩
- 汇讯陇西菁卡拉新：32→15人（-53%）——菁卡拉新下滑最深的站点
- 速讯达和平菁卡拉新：59→34人（-42%）——菁卡拉新最大单站跌幅
- 创世伙伴盐湖财富固收：20→12人（-40%）——全公司下行趋势的一部分

**策展主题层**：
收缩的三种性质——项目系统性下滑（菁卡拉新）、供应商内部失衡（汇讯）、整体衰退（创世伙伴）。

**正文**：

**项目系统性下滑**：菁卡拉新15站，Q1从531人降至430人（-19%），是唯一全线收缩的主力项目。涉及5家供应商：速讯达-27.6%、艾斯欧艾斯-23.1%、中弘智享-12%、靖涵-9.8%、汇讯陇西-53%。只有拓银长沙一枝独秀+33.3%。这不是单点问题，是产品线层面的信号。

**供应商内部失衡**：汇讯9站280人（+1.8%），表面平稳，但内部两极分化：洪山企业主贷+67%，合肥金条-54%，陇西菁卡拉新-53%。总量持平掩盖了站点级的剧烈变化。合肥金条从35人降至16人，需要确认是项目迁移还是团队流失。

**整体衰退**：创世伙伴2站，西安高新60→52人（-13%），盐湖职场20→12人（-40%），全公司-20%。博岳2站持平，人和2站-14.3%，均为负增长。这类小型供应商的收缩更难逆转。

**证据层**：
- 菁卡拉新数据：1月531人 → 2月477人 → 3月430人，连续两月下滑
- 汇讯合肥站点：2月已降至28人（1月35人），3月继续降至16人，非单月异动
- 创世伙伴两站均为财富固收项目，该项目整体Q1增长+7.2%，说明收缩是供应商自身问题而非项目萎缩

### 单元4：地理位移

**标题**：人力正在往哪里走

**定位句**：Q1的人力增长不是均匀的。江西翻倍，重庆从零出现，辽宁和甘肃在消失。

**导言**：
省份维度的变化揭示了比供应商视角更大的图景——人力正在从北方向南方、从东北向中部迁移。

**观众进入层**：
- 江西：75→138人（+84%），全部来自岐力南昌
- 山东：45→70人（+55.6%），华啸沂水+速腾泰安
- 重庆：3月0人，4月华啸永川出现40人
- 辽宁：165→132人（-20%），速讯达+中弘智享全线萎缩
- 甘肃：42→24人（-43%），汇讯陇西两项目同时缩水

**策展主题层**：
地理位移反映的是供应商策略选择——在哪里开站点、在哪里收缩。这不是被动的人口流动，是主动的资源配置。

**正文**：
安徽和河北作为双中心的地位没有改变，合计2,048人。但增量来自边缘：江西+63人、山东+25人、重庆+40人（4月）。这些新增长点的供应商都是中小型——岐力、华啸、速腾，而非头部5家。

收缩端，东北三省（辽宁-20%、黑龙江-16%）和西北（甘肃-43%）是最明确的退缩区域。供应商在这些区域的人力减少，可能反映的是当地招聘难度和运营成本。

**证据层**：
- 重庆华啸永川：1-3月人力为0，4月突增至40人，疑似新开站点
- 辽宁5站全部分属速讯达（2站菁卡拉新）和中弘智享（3站菁卡拉新），5站合计从165降至132人，全部为菁卡拉新项目。辽宁的萎缩本质是菁卡拉新项目的地理表现。

## 四、视觉语言（curatorial-visual-language）

### 设计参照系

**分析型简报板（Analytical Briefing Board）**

高级管理层桌面上的"一页纸简报"——信息密度高、判断前置、视觉安静。文字驱动页面，图表服务判断，交互服务效率。

### 材料身份

| 材料类型 | 本项目形态 | 页面角色 |
|----------|-----------|----------|
| 证据材料 | 站点人力数据表、变化率数值 | 卡片底部折叠区域 |
| 定位材料 | 供应商排名索引、省份列表、项目标签 | P1索引页、侧栏导航 |
| 分析材料 | 四象限气泡图、趋势折线图、排名条形图、省份柱状图 | P0-P4核心视觉模块 |
| 氛围材料 | 无 | 不使用 |

### 字体层级

| 角色 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 18px | 600 | 每页只有一个 |
| 单元标题 | 15px | 600 | 模块标题 |
| 定位句 | 12px | 400 | 标题下方一句 |
| 正文 | 13px | 400 | 判断文本 |
| 指标值 | 24px | 600 | 核心数字 |
| 指标标签 | 11px | 400 | 数字上方说明 |
| 卡片标题 | 12px | 600 | 站点卡片 |
| 卡片详情 | 11px | 400 | 项目、省份、变化明细 |
| 图注/证据 | 10px | 400 | 数据来源、编辑说明 |
| 导航标签 | 11px | 600 | 页面跳转、标签切换 |

字体族：`system-ui, -apple-system, 'PingFang SC', 'Microsoft YaHei', sans-serif`。只用两个字重：400 和 600。

色彩规则：单页不超过3个色系；色彩编码含义全局一致；Chart.js 用 hex 不用 CSS 变量。

### 交互语法

| 交互 | 行为 | 策展关系 |
|------|------|----------|
| 标签切换 | 点击标签，内容替换 | 改变观察角度 |
| 卡片展开 | 点击卡片，展开证据层 | 揭示判断依据 |
| 气泡点击 | 点击四象限气泡 | 从全景进入个体 |
| 行点击 | 点击表格行 | 从索引进入详情 |
| hover数值 | 悬停数据点 | 临时查看精确值 |

### 排除方向

- 不做渐变、阴影、毛玻璃
- 不做深色主题
- 不做 emoji 或图标装饰
- 不做装饰性动画
- 不做地图引擎
- 不做实时刷新

## 六、方法回收（curatorial-log-method-recovery）

### 可复用方法

**方法1：信号导向策展法**
> 从数据中提炼"哪里需要管"而非"有什么"。用扩张/收缩/位移三类信号组织叙事，全景底图只做参照系不做主角。适用：任何有周期性变化数据的管理报告。

**方法2：标准物件页模式**
> P5供应商穿透页是可复用模板——传入实体名，自动渲染其所有子单元、趋势和信号标记。适用：任何有"实体→子单元"层级结构的数据。

**方法3：三级文本模型**
> 观众进入层（具体人/站点/数字）→ 策展主题层（这一单元真正处理的问题）→ 证据层（数据和来源，折叠）。适用：所有需要"扫一眼→深读→验证"的管理报告。

**方法4：视觉语言四色系**
> Teal=好/增长、Coral=注意/收缩、Warm=重点对象、Stone=中性/其他。全局语义固定，跨页一致。适用：所有有正负向判断的数据可视化。

### 关键判断

1. 供应商管理场景的策展主线应是"变化信号"，不是"状态全景"
2. 四象限气泡图（规模×增长率）是供应商管理最有效的入口视图
3. 站点级分析是信号发现的关键粒度，公司级聚合会掩盖分化
4. 菁卡拉新Q1全线收缩是需要持续跟踪的系统性信号
5. "分析型简报板"视觉语言适合B端管理场景

### 余留问题

1. P2/P3的模式标签分类标准需公式化（当前为人工判断）
2. P5供应商穿透页未实际渲染，需用毅航数据验证
3. 4月数据完整后需更新趋势判断
4. 无业绩数据配套，信号解读仅基于人力变化
