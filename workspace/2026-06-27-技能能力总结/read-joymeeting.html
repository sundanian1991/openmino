read-joymeeting

慧记 JoyMinutes 会议记录提取 · v1.0.0
完整转写 Headless CDP

概述

核心突破：突破"只能看摘要"限制，通过 Headless 浏览器 + 原生 CDP 协议直接调用 JoyMinutes API，获取完整的会议转写文本（逐字稿）——包括每段发言的说话人、内容、起止时间。

一个脚本完成全部操作：启动浏览器 → 继承登录态 → 调用 API → 返回数据 → 清理浏览器。无需 Playwright MCP 或额外依赖。通过复制本机浏览器 profile 自动继承 ERP SSO 登录态。

前置条件

本机需安装以下浏览器之一并已登录 JoyMinutes / ERP：

Google Chrome
优先
Tabbit
京东内部浏览器
Microsoft Edge
备选
Chrome 优先 → Tabbit fallback → Edge 备选 → 自动打开登录页

使用方法

日期范围查询

# 今天
node "$SKILL_DIR/scripts/fetch-meetings.js"

# 昨天
node "$SKILL_DIR/scripts/fetch-meetings.js" --date yesterday

# 本周
node "$SKILL_DIR/scripts/fetch-meetings.js" --date thisweek

# 上周
node "$SKILL_DIR/scripts/fetch-meetings.js" --date lastweek

# 指定日期
node "$SKILL_DIR/scripts/fetch-meetings.js" --date 2026-06-08

# 近7天
node "$SKILL_DIR/scripts/fetch-meetings.js" --date last7days

# 日期区间 + 详情
node "$SKILL_DIR/scripts/fetch-meetings.js" --start 2026-06-03 --end 2026-06-09 --detail

指定会议ID查询

node "$SKILL_DIR/scripts/fetch-meetings.js" --id 21ac80c26e507f76 --detail

日期解析规则

用户说法参数说明
"今天" / 未指定today（默认）仅今天
"昨天"yesterday仅昨天
"本周"thisweek / week本周一到今天
"上周"lastweek上周一到上周日
"近 N 天"lastNdays（如 last7days）从 N 天前到今天
"YYYY-MM-DD"--date YYYY-MM-DD指定单日
"X月X日到X月X日"--start / --end指定区间

输出格式

会议列表项

字段来源说明
titleminutes.detail会议标题
startTimeminutes.detail / list开始时间（毫秒时间戳）
durationlist录制时长（毫秒）
creatorlist.userInfo创建者姓名
membersminutes.detail参会人列表（姓名+ERP）
speakersminutes.speakers.timelines发言人列表（姓名+占比+时长）
transcriptminutes.meetingrecord.query文字记录（需 --detail；数组格式；null=无转写）
attachmentsminutes.detail附件（JoySpace 文档等）
url构建会议详情页链接

transcript 字段结构（关键）

"transcript": [ { "speaker": "发言人姓名", "text": "发言内容（该发言人该时段的所有句子已合并）", "beginMs": 1234567, "endMs": 2345678 }, { "speaker": "另一位发言人", "text": "发言内容...", "beginMs": 2345678, "endMs": 3456789 } ]

总结输出规范

单日查询
  • 总览（会议数、主要议题）
  • 每场：标题、时间、链接、参会人、内容概述、关键讨论、决策结论、待办
  • 综合要点
多日/区间查询
  • 总览（区间、会议总数、主要主题）
  • 按日分组，每日下列出各场详情
  • 综合要点（跨天主题归纳、持续跟进）

质量要求：每条会议含标题与链接可追溯；总结基于详细文字记录不编造；缺失信息明确标注；不输出 Cookie/Token/登录信息。

常见问题

SSO_REDIRECT 错误

ERP SSO cookie 过期。自动尝试：当前活跃浏览器 → Chrome → Tabbit → Edge → 自动打开登录页。

会议数据为空但 totalInList > 0

日期过滤未匹配到会议。检查日期或扩大查询范围。

浏览器相关错误

"未找到浏览器" → 需安装 Chrome/Tabbit/Edge。"CDP not ready" → 端口占用，自动在 9333-9350 范围查找可用端口。
