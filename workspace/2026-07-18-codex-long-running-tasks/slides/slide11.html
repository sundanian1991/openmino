# 习惯 操作要点 核心价值
01 自主规划
路线图		 对话式生成计划 → 设定持久 Goal → 写入 GOALS.md 持续迭代 保持早期灵活性，用文件锚点对抗漂移
02 主线程
全局协调		 只聚焦目标 / 约束 / 决策 / 状态 4 件事，细节委派 subagent 统筹全局，多任务并行可追溯
03 里程碑
审计 + /review		 5 项里程碑审计 + 代码 /review，所有操作留存记录 提前发现偏差，避免累积返工
04 本地线程
专属校验		 登录态 / 凭据 / Xcode / iOS 模拟器等委派本地线程 远程持续运行 + 本地专属校验协同
05 简报 +
仪表盘		 三段式汇报 + /side 探查 + dashboard.html 可视化 状态外置，降低回归成本
