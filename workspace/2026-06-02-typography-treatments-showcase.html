Typography Treatments 建筑 特效 装饰
Nian Design System / Typography Treatments

文字不只是"放大"。
文字是建筑、是特效、是装饰。

这个页面展示 25 种文字处理技法。每种技法都是一条 CSS 规则——不依赖 JS 库、不依赖图片，只用字体属性、transform、opacity、clip-path 做出有表现力的排版。

Category 1 / Text as Architecture

文字即建筑

不依赖图片、不依赖动效。纯粹的字体属性——size、weight、spacing、line-height、alignment——构成视觉结构。文字本身就是页面布局。

Category 2 / Text Effects

文字特效

用 transform、opacity、clip-path 给文字加入场、出场和交互效果。不依赖 JS 库，纯 CSS 为主，少量需要 IntersectionObserver 触发 class 切换。

3.2 Text Reveal 滚入揭示
文字 从下方 滚入 视野
.t-reveal span { display: inline-block; transform: translateY(110%); transition: transform .8s cubic-bezier(.16,1,.3,1); } .t-reveal.visible span { transform: translateY(0); } span:nth-child(2) { transition-delay: .1s; } /* 逐词延迟 */ /* 适用：Section 标题入场、Hero 文字动画 */
3.6 Outline Text 描边/悬停填充
OUTLINE
color: transparent; -webkit-text-stroke: 2px var(--td); transition: all .5s ease; /* hover → */ color: var(--td); -webkit-text-stroke: 0; /* 适用：Ghost 装饰文字、悬停交互 */
3.21 Clip Path Reveal 幕布展开
幕布从左到右展开
clip-path: inset(0 100% 0 0); transition: clip-path 1s cubic-bezier(.16,1,.3,1); /* visible → */ clip-path: inset(0 0 0 0); /* 适用：章节标题入场、方向性揭示 */
3.23 Rotation Entrance 翻板入场
信息到达
opacity: 0; transform: rotateX(90deg); transform-origin: bottom center; transition: all .8s cubic-bezier(.16,1,.3,1); /* visible → */ opacity: 1; transform: rotateX(0); /* 适用：单行标题入场、翻板式信息更新 */
3.19 Letter Spacing Expand 字距扩张
EXPAND INTO VOID
letter-spacing: -.04em; transition: letter-spacing 1s cubic-bezier(.16,1,.3,1); /* visible → */ letter-spacing: .5em; /* 适用：奢侈品牌、空间主题、戏剧性揭示 */
3.18 Strikethrough Reveal 删除线消去
错误的做法
::after { content: ''; position: absolute; top: 50%; width: 104%; height: 3px; background: var(--scene); transform: scaleX(1); transition: transform .6s; } .revealed::after { transform: scaleX(0); transform-origin: right; } /* 适用：对比（旧方法→新方法）、价格删除 */
3.16 Hand-Drawn Underline 手绘底线
关键要点强调
::after { background: url("SVG 手绘波浪线"); transform: scaleX(0); transform-origin: left; transition: transform .6s; } .visible::after { transform: scaleX(1); } /* 适用：行内强调、编辑式高亮 */
Category 3 / Text as Decoration

文字作为装饰

文字不在第一阅读层。它作为背景、边缘、纹理存在——用户感觉到它但不必须读它。这是"氛围层"。

4.1 Watermark 背景大字
TYPO
前景内容
透明度 3% 的大字在背景层，营造品牌感但不干扰阅读。nian v2/v3/v4 的 ghost "12" 装饰就是这个技法。
font-size: 25vw; font-weight: 900; color: var(--td); opacity: .03; position: absolute; white-space: nowrap; pointer-events: none; /* 适用：Hero 背景、章节大序号、品牌名水印 */
4.7 Background Numbers 描边数字
03
深色底 + 描边数字
text-stroke 描边数字比实心数字更含蓄。在深色背景上，5% 透明度的描边产生"刻痕"效果。
font-size: 30vw; font-weight: 900; color: transparent; -webkit-text-stroke: 1px rgba(255,255,255,.05); position: absolute; pointer-events: none; /* 适用：步骤序号、数据报告的章节标记 */
4.3 Vertical Text / 4.4 Rotated Label 边缘标注
SECTION LABEL / VERTICAL TEXT / DECORATIVE
竖排 + 旋转标注
writing-mode: vertical-rl 在左侧做装饰标签。transform: rotate(-90deg) 在边缘做水平标注。两者都不在主阅读路径上。
/* Vertical */ writing-mode: vertical-rl; font-size: .7rem; letter-spacing: .3em; text-transform: uppercase; color: rgba(255,255,255,.35); /* Rotated */ transform: rotate(-90deg); transform-origin: left center; position: absolute; left: 20px; /* 适用：章节标签、元数据、装饰性边栏文字 */
