## Big Idea

金条坐席首贷与复贷绩效分化呈现"双极化"格局：毅航合肥复贷 S 级独占大盘 62%（67/108），而翰锐/赛维斯/伽玛首贷 C 级率超 34%（合计 C 级 37 人占首贷 C 级 39.8%）。核心矛盾是 **MOB1-3 新手活化陷阱**（C 级中 33% 为入职 30-89 天新人）和 **三个职场的系统性绩效危机**（翰锐 40%、伽玛 36.4%、赛维斯 34.1% 首贷 C 级率）。建议立即启动翰锐/赛维斯/伽玛专项改善、MOB1-3 新人前置培训体系、毅航合肥经验萃取复制。
置信度：高

## 数据推理

| 推理类型 | 分析 | 置信度 |
|---------|------|--------|
| **趋势推算** | 翰锐首贷 C 级率 40%，若保持当前管理方式，下期 C 级占比将进一步恶化至接近 50%（新人持续加入 + 缺乏改善机制） | 中 |
| **相对量级** | 毅航合肥复贷 S 级占比 57.8%，是均值(20%)的 2.9 倍；翰锐首贷 C 级率 40%，是均值(20%)的 2 倍 | 高 |
| **排名变化** | 首贷 10 个参评职场中翰锐 ATT 最低(73.1s)、C 级率最高(40%)；博岳石家庄转化率最高(116.3%)但 C 级率仍达 17.9% | 高 |
| **业务后果** | C 级坐席人均首贷 GMV 仅 96.4 万 vs S 级 393.9 万 = 4.1 倍差距。若将 93 名 C 级提升至 B 级中位水平(177 万)，可释放约 7,500 万增量 GMV | 中 |
| **分母陷阱** | 转化率分母当前为接通用户数，未使用名单下发数。若改用名单分母，转化率绝对值会下降，但等级间相对差距趋势不变 | 探索性 |
| **基线校准** | 首贷 ATT 整体均值 89.5s，低于部分头部电销中心标杆（100-110s），说明通话质量仍存提升空间 | 中 |
| **季节性因素** | 数据为单期近 30 日截面，无法判断季节性波动。建议连续 3 期数据对比后再纳入考虑 | 探索性 |

## 反对证据与局限性

- 数据为单期截面（近 30 日），无法判断绩效的季节性波动和长期趋势
- 转化率分母为接通用户数而非名单下发数，目前的转化率可能偏高
- 复贷业务缺少 T0 GMV 和件均跨等级详细对比数据
- 缺少质检合规数据（暂未纳入评级），可能低估部分坐席的真实问题
- 帕累托曲线使用模拟拟合数据，实际个体数据在座席明细中待逐条验证

---

## 底部总结

首贷大盘 GMV 10.4 亿、复贷 46.2 亿。S 级人均 GMV 是 C 级的 4.1 倍（首贷），3.9 倍（复贷）。核心矛盾不在"接通量"而在"转化能力"——有效→首贷环节 S 级转化率 3.8% 是 C 级 1.4% 的 2.7 倍。翰锐/赛维斯/伽玛三个职场首贷 C 级率超 34% 需立即诊断，MOB1-3 新人 C 级率 33% 需前置培训体系，毅航合肥经验值得萃取复制。
