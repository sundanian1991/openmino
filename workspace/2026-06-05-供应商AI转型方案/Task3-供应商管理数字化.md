## 一、分工与核心任务

**任务三：供应商管理数字化 @孙大年**
构建完整的供应商管理数字人体系，向职场提供可咨询、可记录、可提醒的供应商管理 Agent
1）通过 Agent 实现供应商档案数字化，搭建完整的 5 库知识库体系，支持随时咨询查询和自动记录更新
2）通过 Agent 执行标准化管理规则（5 维考核评分 / 分量建议 / 异常告警 / 合规处理），自动推送管理举措建议
3）通过 Agent 贯穿供应商全生命周期（引入评估 → 日常监控 → 清退复盘），实现关键节点的自动推送与闭环跟踪

| 任务 | 维度 | 包含内容 | 维护人 | 6月 |
|------|------|---------|--------|-----|
| **任务三** | **1️⃣ 供应商档案数字化** | 32+家供应商全景信息入库；信息收集模板（详见附件）落地；标签体系定义；供应商老板画像收集 | 孙大年 | 第2周 |
| **任务三** | **2️⃣ 考核模型与规则体系设计** | 5维评分权重确认、各维度评分细则、ABCD评级标准、红黄牌判定规则、分量规则框架 | 孙大年 | 第3周 |
| **任务三** | **3️⃣ 核心Skill原型 + 知识库框架落地** | 档案查询Skill + 日简报Skill 原型可跑通；5个知识库结构搭建完成并录入首批内容 | 孙大年 | 第4周 |

### 知识库使用方式

- 5 个知识库独立 topic，数字人按场景自动选择检索范围
- 查 SOP → 库1，查档案 → 库2，查先例 → 库3，查规则 → 库4，查商务 → 库5
- 新增信息后自动归入对应库
