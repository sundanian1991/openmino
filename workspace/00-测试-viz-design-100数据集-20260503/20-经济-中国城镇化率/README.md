# 中国城镇化率

- **来源**: World Bank (https://data.worldbank.org/indicator/SP.URB.TOTL.IN.ZS?locations=CN)
- **数据范围**: 2015-2024
- **说明**: 城镇化率及人口结构变化

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| urban_population_pct | 城镇化率(%) |
| urban_pop_million | 城镇人口（百万人） |
| rural_pop_million | 农村人口（百万人） |
| urbanization_change_pct | 城镇化率变化（百分点） |
