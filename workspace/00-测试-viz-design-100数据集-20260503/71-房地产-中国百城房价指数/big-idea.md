# Phase 0 — Big Idea

## 数据集 71：中国百城房价指数（2023-2025）

**Big Idea**：百城房价两年"心电图式"横盘——政策密集但市场无感

**分析角度**（模式 B，AI 自主推荐）：
1. **趋势线**：2023-2025 样本均价走势（微涨 0.65%，几乎持平）
2. **涨跌城市博弈**：上涨 vs 下跌城市数量的时间演变（上涨始终 < 下跌）
3. **同比拐点**：环比/同比增速变化（2024-03 首次转正后的脆弱企稳）

**叙事逻辑**：先给全景（均价几乎没变）→ 再看内部结构（涨少跌多）→ 最后看政策效果（密集但微弱）。这是一张"政策与市场脱节"的图。

## 数据集 72：全球城市房价收入比（2024）

**Big Idea**：中国 3 城霸占全球最难买房 TOP3——深圳北京上海的购房难度是纽约东京的 3 倍

**分析角度**（模式 B，AI 自主推荐）：
1. **排名全景**：20 城市房价收入比横向对比（TOP3 中国城市断层领先）
2. **梯队分化**：超高负担（>25）、高负担（15-25）、中等负担（10-15）、相对合理（<10）

**叙事逻辑**：排名冲击开场（TOP3 全部中国）→ 梯队分化展示（亚洲城市占顶部，欧美在底部）。这是一张"购房负担全球失衡"的图。
