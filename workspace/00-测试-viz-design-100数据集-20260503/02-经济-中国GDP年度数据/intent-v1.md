# Intent — 中国 GDP 年度数据 2015-2024

## 意图决策

**可视化目的**：展示中国 10 年经济转型的核心叙事——总量翻倍但增速放缓，经济结构从工业转向服务业。

**信息类型**：趋势变化 + 结构转型

**核心信息（一句话）**：中国用 10 年完成从工业驱动到服务驱动的结构转型，但增速从 7% 降到 5%——新引擎能维持多高增速？

**结论**：增速放缓不是周期性波动，是结构转型的代价。

**为什么选双轴图**：增速（右轴 %）和经济结构（左轴 占比%）是同一张图里的两条线。一条下降（增速），一条上升（服务业占比），天然形成对比。

**备选模式**：
- 纯增速折线 → 太单薄，缺少"为什么放缓"的结构性解释
- 堆叠面积图 → 展示结构变化但看不到增速
- 双轴折线 → 同时看到增速放缓 + 结构转型 → 选中

## 减法迭代预判

1. GDP 总量柱状图 → 占空间且不是核心叙事 → 删
2. 人均 GDP → 和总量高度相关，信息冗余 → 删
3. 工业占比 → 是服务业占比的镜像，选一个就够了 → 只留服务业
4. 网格线 → 淡化
5. 标注 → 只标 3 个关键点：2020 疫情、50% 服务业拐点、2024 最新
