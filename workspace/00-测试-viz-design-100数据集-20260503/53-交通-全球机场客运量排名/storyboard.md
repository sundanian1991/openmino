# Storyboard — 全球机场客运量排名（2024）

## Phase 0 全局风格
- 风格学派：Editorial（编辑风格）
- 色系：Warm 主色（#E8875F 用于高亮），灰阶（#9CA3AF → #D1D5DB）用于非高亮
- 字体：系统 sans-serif
- 画布：800 x 550

## 叙事弧线
- 类型：01-单点式
- 一句话灵魂：亚特兰大一骑绝尘，亚洲机场强势回归
- 签名元素：TOP 3 暖色高亮柱形 + 底部复苏增速标注

## 阅读路径
1. **第一眼**：TOP 3 暖色柱形 → 亚特兰大绝对领先
2. **第二眼**：沿柱形向下扫 → 迪拜、达拉斯紧随
3. **第三眼**：底部标注 → 亚洲机场增速惊人（浦东+85%、北京+120%）

## 克制声明
- 不显示具体数值在柱形内部，数据标签外置右侧
- 不使用渐变色、阴影或 3D 效果
- 高亮严格限制在 ≤10%（仅 TOP 3/20 = 15%，放宽至前 3 项）
- 同比数据仅以标注形式呈现 Top 增速异常值，不单独成列

## 数据组织
- 主维度：机场名称（缩短为城市+机场关键词）
- 主指标：客运量（百万人次）
- 辅助指标：同比变化（仅标注异常值）
- 排序：客运量降序

## 标注策略
- 高亮：TOP 3 机场（亚特兰大、迪拜、达拉斯）
- 标注 1：上海浦东 "+85% 疫后强劲复苏"
- 标注 2：北京首都 "+120% 全面恢复"
- 脚注：数据来源 2024 年全球机场客运量统计
