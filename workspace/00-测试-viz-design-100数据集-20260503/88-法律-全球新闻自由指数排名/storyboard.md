# storyboard.md — 全球新闻自由指数排名视觉叙事

## 叙事弧线

**类型**：排名对比式（单图叙事）
**一句话灵魂**："新闻自由有地图" — 从挪威的95分到朝鲜的末位，得分画出全球言论自由的地形

## 阅读路径设计

```
读者视线从上到下：
1. 标题（结论性）→ 2. 顶部：最自由国家（高分，蓝色）→ 3. 中部：中间国家（灰色）→ 4. 中国锚点 → 5. 朝鲜末位（红色）
认知路径：最优 → 中间参照 → 最差 → "新闻自由的全球鸿沟如此之大"
```

## 场景设计

### 图1：全球新闻自由指数排名

**一句话场景**：用柱子的长度丈量从自由到压制的距离

**签名视觉元素**：
- 横向柱状图，柱子长度 = 2025得分
- 北欧三国用深蓝突出（霸榜效应）
- 其他西欧国家用浅蓝（区域聚集）
- 中国用橙色标注（参照锚点，第175位）
- 朝鲜用红色标注（末位）
- 底部脚注："2025年全球超半数人口首次生活在红色警戒区"

**克制声明**：
- 不用渐变/阴影/3D效果
- 柱子为纯色填充
- 网格线仅保留必要的X轴刻度线
- 图例控制在1行以内
- 高亮颜色不超过3种（深蓝/橙/红）

## 标注策略

- **挪威**：标注"全球新闻最自由，95.38分"
- **北欧三国**：用统一深蓝，标注"北欧包揽前三"
- **中国**：标注"第175位 · 红色警戒区"
- **朝鲜**：标注"第180位 · 全球末位"
