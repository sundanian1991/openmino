## 图表 A：灾害趋势双轴图

### 一句话灵魂
> "灾害数量在下降，但代价在上升" — 频率与成本的背离是最核心的故事

### 叙事弧线
**02-对比式弧线**：先建立"灾害频率趋稳"的预期，再用"损失持续攀升"打破预期，制造认知张力

### 阅读路径
```
1. 标题区（顶部）→ "极端天气灾害5年趋势：数量趋稳但经济损失持续攀升"
2. 图表主体 → 双线走势，灰线=事件数（左轴），红线=十亿美元灾害（右轴）
3. 2024数据点 → 红色圆圈高亮，标注"28起，5年最高"
4. 底部脚注 → 数据来源：EM-DAT + NOAA
```

### 场景设计
- **视觉节拍**：双Y轴对比，左轴范围350-450（事件数），右轴范围15-30（十亿美元灾害）
- **克制声明**：
  - 只用两条线，不添加面积填充
  - 除2024高亮点外全部灰色
  - 不添加网格线或仅保留极淡水平线
  - 标注写原因（"损失创新高"），不堆数字
- **配色**：
  - 事件数线：#8E8E93（中性灰）
  - 十亿美元灾害线：#E8875F（暖色警报）
  - 2024高亮点：#C0392B（红色圆圈）
  - 背景：纯白 #FFFFFF

### 数据组织
- X轴标签：2020、2021、2022、2023、2024
- 数据标签仅标注关键拐点（2021峰值432、2024损失28起）
- 2022死亡人数30,700异常值可在脚注中提及

## 图表 B：2024极端天气快照

### 一句话灵魂
> "2024年：1.55°C突破巴黎协定红线，152起前所未有" — 用具体数据锚定"前所未有"的规模

### 叙事弧线
**01-开门式**：单图即结论，不需要前置铺垫

### 阅读路径
```
1. 标题区 → "2024年全球极端天气快照：多项指标突破历史纪录"
2. 柱状图主体 → 5个关键指标垂直排列
3. 红色警报柱 → 温度距平+1.55°C标红（唯一超过阈值的指标）
4. 底部脚注 → "前所未有"级别事件152起，气候变化致因死亡≥3,700人
```

### 场景设计
- **视觉节拍**：5个柱体，按重要性从左到右排列
- **克制声明**：
  - 只高亮最关键的1-2个柱体（温度距平、前所未有事件）
  - 其余柱体灰色
  - 柱顶显示简洁数据标签
- **配色**：
  - 普通柱体：#8E8E93（灰色）
  - 高亮柱体：#E8875F（暖色）
  - 温度距平柱：#C0392B（红色，超越巴黎协定红线）
  - 背景：纯白

### 数据组织
- 5个指标：极端天气事件617起、前所未有事件152起、致因死亡3,700人、温度距平+1.55°C、灾害损失1,820亿美元
- 注意量级差异：温度距平用独立刻度或标注方式
