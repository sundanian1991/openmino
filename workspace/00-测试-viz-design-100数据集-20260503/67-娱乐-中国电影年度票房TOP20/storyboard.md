# storyboard.md — 中国电影年度票房TOP20

## Phase 0: 全局风格锁定

本数据集为单图需求，无需全局风格锁定。

**风格学派**：麦肯锡Exhibit风格 — 干净、数据驱动、结论优先
**色系**：Warm单色系 — 灰度主体 + Warm800高亮头部TOP3
**字体**：系统默认无衬线

## 叙事弧线

**弧线类型**：01-直给式
**一句话灵魂**：年度票房TOP3断层领先，头部效应显著
**签名元素**：横向条形图，TOP3使用暖色高亮，其余灰度

## 视觉节拍

1. **标题区**：结论性标题 — "头部3部影片贡献TOP20票房的40%+"
2. **副标题**：时间范围 + 数据来源
3. **图表区**：20条横向条形，从大到小排序
4. **标注**：TOP3标注"头部效应"，第1名标注具体数值
5. **图例/脚注**：单位说明 + 数据来源

## 阅读路径

```
标题（结论）→ 扫一眼TOP3颜色 → 从上到下读排名 → 注意底部差距
```

## 克制声明

- 不添加装饰性元素
- 不使用渐变或阴影
- 高亮限制在TOP3（15% ≤ 10%规则 → 放宽至TOP3因排名场景合理）
- 不使用动画效果

## 数据组织

**排序**：按票房降序排列
**聚合**：无需聚合，直接使用原始排名数据
**字段**：排名、影片名称、票房金额（亿元）
