# Spec — 上证指数月度数据 2023.01-2025.04

---

## 叙事意图

**可视化目的**：展示上证指数 28 个月从 3255 回到 3258 的荒诞——不是没动，是坐了过山车。

**信息类型**：趋势 + 异常值

**想传达什么**：28 个月的剧烈波动最终回到原点，市场在等真正的增长叙事。

**结论**：回到原点不是平静，是经历了极端波动后的无奈。

**思路**：折线图 + 面积渐变（增加波动厚度感）+ 4 个关键事件标注。

## 视觉执行

**模式选择**：折线图 + 面积渐变

**标题**：上证指数 28 个月回到原点

**副标题**：2023.01 — 2025.04 月度收盘价

**视觉编码**：
- X 轴：28 个月（2023-01 到 2025-04）
- Y 轴：指数点位（2700-3500，不从零开始，放大波动感）
- 面积渐变：深灰 → 透明，增加视觉厚度

**标注策略**（4 个关键点）：
1. 起点 3255（灰）
2. 最低 2842（红）
3. 反弹 3336（绿）
4. 终点 3258（橙）

**布局**：
- 画布：900×420
- grid：left 65, right 55, top 70, bottom 50
- 提要文字在图下方左对齐
