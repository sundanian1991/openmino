# storyboard.md — 全球手机品牌出货量

## Phase 0: 全局风格锁定

- **风格学派**：现代数据新闻风格（干净、高对比、重点突出）
- **色系**：单色灰阶 + 单一强调色（Warm 橙 #E8875F 用于高亮）
- **字体**：系统无衬线字体（-apple-system, sans-serif）
- **画布**：800 x 550px

## 视觉叙事设计

### 一句话灵魂

> 苹果首超三星登顶全球手机出货量第一，华为35%暴增重回头部阵营。

### 叙事弧线

03-排名式 — 按出货量降序排列，用颜色标注异常值

### 阅读路径

1. **顶部标题** → 结论先行：苹果登顶
2. **左侧排名** → 从上到下，出货量递减
3. **高亮锚点** → 苹果（#1）和华为（+35%）用强调色，其余灰阶
4. **底部脚注** → 全球合计12.4亿台，同比+6%

### 克制声明

- 仅高亮2个品牌（苹果+华为），占16.7% ≤ 10%精神阈值
- 不展示同比柱状叠加，用标签标注关键增幅
- 灰阶使用3级：头部（深灰#666）、中段（中灰#999）、尾部（浅灰#bbb）

### 数据组织

- 排序：按出货量降序
- 高亮：苹果（#1地位）、华为（最大增幅回归者）
- 标注：华为"+35%"、Nothing"+80%*"（小字注：基数极小）
- "其他"合并项放在最底部，浅灰处理
