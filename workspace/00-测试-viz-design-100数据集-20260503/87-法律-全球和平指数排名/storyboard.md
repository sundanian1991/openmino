# storyboard.md — 全球和平指数排名视觉叙事

## 叙事弧线

**类型**：排名对比式（单图叙事）
**一句话灵魂**："和平是有地图的" — 从冰岛的宁静到阿富汗的烽火，GPI得分画出全球安全的地形

## 阅读路径设计

```
读者视线从上到下：
1. 标题（结论性）→ 2. 顶部：冰岛（最短柱，蓝色高亮）→ 3. 中段：其余国家（灰色，密集排列）→ 4. 末尾：三国（红色，冲突锚点）
认知路径：最优 → 差距感知 → 最差 → "和平鸿沟如此之大"
```

## 场景设计

### 图1：全球和平指数排名

**一句话场景**：用柱子长度丈量从和平到冲突的距离

**签名视觉元素**：
- 横向柱状图，柱长 = GPI得分
- 冰岛唯一蓝色高亮（#2563EB）
- 其余Top15 + 中国 = 灰色（#94A3B8）
- 末尾三国 = 红色（#DC2626）
- GPI=2.0处虚线分隔"和平/动荡"区域

**克制声明**：
- 不用渐变/阴影/3D效果
- 柱子纯色填充
- 网格线仅保留X轴刻度线
- 高亮颜色不超过2种（蓝/红）

## 标注策略

- **冰岛**：标注"全球最和平"
- **南苏丹**：标注"冲突泥潭"
- **参考线**：GPI=2.0虚线
