# storyboard.md — 中国高考报名人数变化

## 叙事弧线：趋势+拐点叙事（C07）

### 阅读路径
1. **第一眼**：两条线（报名/录取人数）的持续上升 → "越来越多人在参加高考"
2. **第二眼**：录取率线先升后降 → "但考上大学的难度在变化"
3. **第三眼**：2021年后报名人数陡峭上升 + 录取率下降 → "竞争正在加剧"
4. **关键标注**：标注三个拐点（2008年峰值、2012年谷底、2024年新高）+ 录取率拐点（2021）

### 一句话灵魂
"高考规模膨胀了24年，但竞争的钟摆正在回摆"

### 视觉编码
- **X轴**：年份（2000-2024），等间距
- **左Y轴**：人数（万人），0-1400
- **右Y轴**：录取率（%），40-90
- **线条1**：报名人数 — 粗线，主色，关键节点标注
- **线条2**：录取人数 — 细线，灰色调
- **线条3**：录取率 — 虚线，辅助色，右轴

### 高亮策略
- 高亮 2024年报名人数（1342万，历史新高）
- 高亮 2021年录取率拐点（84.3%，之后开始下降）
- 高亮区域 ≤ 10%，仅突出最关键的信号

### 标注内容
- "2008年首破千万"（报名人数拐点）
- "2021年录取率见顶 84.3%"（趋势逆转点）
- "2024年 1342万，历史新高"（最新数据点）
- "录取率回落至74.7%"（2023年数据）

### 克制声明
- 不使用渐变/阴影/3D效果
- 颜色不超过2个ramp
- 标注不写具体数字（原因+幅度），数字留给坐标轴
- 脚注说明：2024年录取数据待公布；年份非等间隔采样

### 全局风格（单图，无需全局锁定）
- 风格：线性克制
- 色系：Warm 主色 + 辅助色
- 画布：800x550
