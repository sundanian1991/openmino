# checklist.md — 金融电销 Q1 验收清单

> 出图后逐项对照验收

## 通用项

- [ ] 每张图标题是结论性的（不是描述性的）
- [ ] 全局色系统一（#0066FF / #999999 / #E74C3C），无额外颜色
- [ ] 每张图 ≤2 个标注
- [ ] 高亮 ≤10%（除图1排名对比需标注头尾外）
- [ ] 字体统一 Noto Sans

## 图1：团队业绩总览

- [ ] 水平条形图，降序排列
- [ ] 均值参考线（红色虚线，9,554万）
- [ ] 达标组蓝色，未达标组灰色
- [ ] B/F 标注差距幅度（62%）

## 图2：月度转化率趋势

- [ ] 6 条折线，X 轴 3 个月
- [ ] B/D 蓝系粗线，C/F 灰系细线
- [ ] B 组 2.8% 峰值标注
- [ ] 头尾差距 0.9pp 标注

## 图3：转化漏斗

- [ ] 4 步漏斗：外呼→接通→意向→成交
- [ ] 接通率步骤红色（#E74C3C），其余蓝色
- [ ] 每步标注转化率百分比
- [ ] 接通步骤旁标注"71.4% 未接通"

## 图4：散点+象限

- [ ] 18 个散点（6 团队 × 3 月）
- [ ] 均值十字线（虚线）
- [ ] 四象限标签
- [ ] B/D 大蓝点带标签，C 标注"无效忙碌"

## 叙事验证

- [ ] 4 张图按"总览→趋势→诊断→根因"顺序排列
- [ ] 管理层看完 4 张图能直接得出行动建议
- [ ] 无冗余图表（每张图传递独特信息）
