# Design Intent + Storyboard — 金融电销 Q1 运营数据

> 合并版：意图决策 + 叙事设计。一张文件，不再分写。

---

## 六步法第1步：理解语境

**对象**：管理层/运营总监，需要决定 Q2 资源分配和团队优化

**内容**：6 个团队 Q1 业绩对比 → 找出差距根因 → 制定针对性行动

**方式**：4 张图串联叙事，起承转合

**中心思想**：
> B/D 组凭借"长通话+精拨打"策略领先（转化率 2.3-2.8%），C/F 组因追求拨打量导致转化垫底（1.7-1.8%）。建议 Q2 对 C/F 组实施通话质量培训，控制日均拨打量，预计整体转化率可从 2.1% 提升至 2.5%。

## 叙事弧线：起承转合（4 幕）

> 每张图对应一幕，故事随进展推进，图表呈现方式不同。
> checklist 随叙事节拍定制，不用通用模板。

### 第一幕 · 起（全景）：谁在领跑？

**角色**：开场定调，建立参照系

**图表**：水平条形图 — 6 团队 Q1 总成交金额排名

**视觉设计**：
- 所有团队默认灰色（#C1C0B8）
- B 组暖橙高亮（#D97757），F 组红色问题（#c25030）
- 均值参考线（虚线 #E6E3DA）
- 标注："B 组领先 F 组 62%"

**标题**：B 组 Q1 成交 1.14 亿领跑，F 组仅 6993 万

**checklist 定制节拍**：
- [ ] 标题包含量化对比（62%差距）
- [ ] B 组/F 组颜色正确区分
- [ ] 均值参考线可见
- [ ] 条形降序排列

### 第二幕 · 承（动态）：谁在进步？

**角色**：展开证据，揭示趋势

**图表**：折线图 — 3 个月转化率走势

**视觉设计**：
- B 组深灰实线（3px），D 组中灰实线（2px）→ 上升趋势
- C 组红色虚线（2px），F 组红色虚线（2px）→ 停滞/下降
- A/E 浅灰虚线 → 中间状态
- 标题：B 组从 2.0% 升至 2.6%，C/F 三个月原地踏步

**checklist 定制节拍**：
- [ ] 趋势方向明确（上升/下降/停滞）
- [ ] B/D 线比 C/F 线更醒目
- [ ] 月份标签清晰可读

### 第三幕 · 转（根因）：为什么差距这么大？

**角色**：转折，揭示深层原因

**图表**：散点图 — X=日均拨打, Y=转化率, 标注通话时长

**视觉设计**：
- 所有点灰色，B 组暖橙大圆，C 组红色小圆
- 拟合回归线（虚线，负相关）
- 标注："日均拨打越少，转化越高"
- 标题：日均拨打 270 通的 B 组转化 2.6%，316 通的 C 组仅 1.7%

**checklist 定制节拍**：
- [ ] 回归线方向正确（负相关）
- [ ] B/C 点位置准确
- [ ] 标注写原因不写数字

### 第四幕 · 合（行动）：下一步怎么做？

**角色**：总结收束，给出行动建议

**图表**：桥接图（水fall 变体）— 当前均值 → 改进贡献 → 目标

**视觉设计**：
- 起点柱（灰色）：当前均值 2.1%
- 增量柱（绿色）：C/F 培训 +0.2%、B/D 经验推广 +0.1%、号码优化 +0.1%
- 终点柱（暖橙）：目标 2.5%
- 标题：三项改进预计提升转化率 0.4 个百分点至 2.5%

**checklist 定制节拍**：
- [ ] 增量之和 = 总增量（0.4%）
- [ ] 终点 = 起点 + 增量
- [ ] 每项改进可执行

## 全局风格锁定

## 克制声明

- 4 张图共享同一色系，不得各自选色
- 每张图只讲一个判断，不堆叠信息
