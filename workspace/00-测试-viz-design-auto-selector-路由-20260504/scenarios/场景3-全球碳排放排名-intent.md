# Design Intent — 全球碳排放排名

> 测试场景 3：单图 + 排名对比

---

## 基本信息

- **任务**：全球碳排放排名可视化
- **输入类型**：数据
- **数据状态**：有精确数据
- **受众**：通用
- **使用场景**：教育

## 意图识别

### 你要表达什么？

| 维度 | 判断 | 理由 |
|------|------|------|
| 核心关系 | 对比（排名） | 各国碳排放量的排名对比 |
| 叙事目的 | 展示各国碳排放的排名差距，突出头部排放国 | |
| 认知缺口 | 读者现在不知道各国碳排放排名，看完应知道谁排放最多、差距多大 | |

### 模式匹配

- **主选模式**：13-柱状图（分类对比）
- **为什么选这个**：排名对比 = 分类数据按数量排序 → 柱状图天然支持排名感知
- **备选模式**：01-饼图
- **为什么没选备选**：饼图适用于占比（≤3 类），碳排放排名涉及多国，饼图不适用

### 多图拆分（不适用）

> 单图场景，不拆分。

## 自动选择记录

- **输入**：全球碳排放排名
- **Hash**：H = hash("全球碳排放排名")（UTF-8 字节累加和 mod 1000000）
- **L1 内容锁死**：
  - 模式：13-柱状图（意图匹配：排名对比 = 分类数据 → 柱状图，**不走 hash**）
  - 弧线：无（单图不走弧线，**不走 hash**）
  - 构图：01 单中心聚焦（1 张图 = 单中心，**不走 hash**）
- **L2 场景约束**：
  - 场景：数据新闻（教育场景的数据展示）→ Tier 1 = [05 FT, 06 Economist, 07 NYT]，Tier 2 = [10 OWID, 17 Cairo]
  - 风格学派：H % 5 = 余数 → 从 5 个候选中选择（hash 选择）
  - 开场骨架：震惊/冲击（排名数据常有冲击感）→ Tier 1+2 = [01 数据冲击, 04 对比开场]
    - (H/10) % 2 = 余数 → 选择（hash 选择）
- **L3 风格自由**：
  - 配色：hash 选择（详见 auto-selector §六）
  - 字体：hash 选择（详见 auto-selector §七）
  - 节拍：H % 20 → 2-3 个节拍组合。预期节拍：C-10 排序阶梯 + C-02 色彩弹出（排名数据的典型节拍）

## 决策边界

- **必须包含**：国家名称、排放量数据、排名标注
- **不需要**：历史变化、人均数据（除非特别要求）
- **约束**：排放国数量可能较多，需控制标签密度
