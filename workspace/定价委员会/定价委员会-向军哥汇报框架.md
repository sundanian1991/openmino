# 向军哥汇报 — 定价委员会机制

**汇报对象**：数据科技业务部负责人 军哥（C2）
**汇报人**：供应商管理岗
**时间**：2026-02-27

## 一、定局：一句话说清主要矛盾

当前定价调整存在四大问题：

1. **定价调整随意** — 策略组或量化代表单方面决策，没有供应商声音
2. **降幅无约束** — 单次调整幅度超过 50%，远远超出心理承受
3. **事后救火** — 供应商管理岗只能事后安抚，没有决策参与权
4. **无复盘机制** — 调整完不追踪效果，下次还犯同样错误

**主要矛盾**：需要从"事后救火"转向"事前防火"，建立一个有约束、有制衡、有闭环的决策机制。

---

## 二、亮牌：三句话讲完应对策略

### 第一张牌：组织架构 — 多方制衡

建立一个 5 人委员会，覆盖所有相关方：

| 角色 | 人员 | 为什么需要他 |
|------|------|-------------|
| 秘书长 | 供应商管理岗 | 组织会议、收集材料、追踪落地 |
| 委员 | 业务督导代表 | 代表坐席声音，评估收入影响和管理难度 |
| 委员 | 量化代表 | 提供数据测算、ROI 分析 |
| 委员 | 服务组负责人 C3 | 评估业务运营影响、协调职场安排 |
| 委员 | 策略组负责人 C3 | 提供定价策略建议、设计调整方案 |
| 终审 | 你（C2） | 每项调整的最终审批 |

**为什么这样设计**：
- 供应商问题需要业务督导来评估影响
- 定价调整需要量化代表来测算收益
- 运营安排需要服务组负责人来协调
- 策略设计需要策略组负责人来把关
- 最终决策需要你来掌握战略方向

### 第二张牌：三条红线 — 约束降幅

| 红线 | 标准 | 为什么这样定 |
|------|------|-------------|
| **单次降幅** | ≤30% | 超过 30% 会超出供应商心理承受，需分 2 次执行 |
| **生效时间** | 次月 1 日 | 当月审批、次月生效，给供应商缓冲期，避免突发冲击 |
| **调整频次** | 3 个月一次 | 避免频繁调整，每次调整前 1 个月提交申请 |

**数据支撑**：
- 2025 年有 3 次单次降幅超过 50%，供应商反弹强烈
- 调整后次月投诉率平均上升 25%
- 频繁调整导致供应商无法做人员规划

### 第三张牌：六阶段流程 — 全程可控

```
提案（T-5天）→ 初审+评审（T-2天）→ 终审（T-1天）→ 生效（次月1日）→ 宣贯（生效后）→ 复盘（+30天）
```

每个阶段的关键动作：

| 阶段 | 关键动作 | 谁来做 |
|------|----------|--------|
| **提案** | 提交《定价调整提案》，含影响测算、配套措施 | 策略组或量化代表 |
| **初审+评审** | 形式审查 + 5 人会议讨论 + 投票表决 | 秘书长 + 委员会 |
| **终审** | 战略合理性、整体影响、风险控制评估 | 你（C2） |
| **生效** | 系统配置、通知结算、首周监控 | 结算组 + 秘书长 |
| **宣贯** | 了解供应商同步情况、员工反馈、应对措施 | 秘书长 |
| **复盘** | 输出《复盘报告》，纳入下次参考 | 秘书长 + 委员会 |

**表决规则**：
- 常规调整（降幅 ≤15%）：委员会 3/5 通过
- 重大调整（降幅 >15%）：委员会 4/5 通过 + 你终审
- 涨价调整：委员会 3/5 通过（与常规调整一致）

**为什么设置两种门槛**：
- 15% 以内：常规调整，委员会可以决策
- 15% 以上：重大调整，需要更高门槛 + 你的终审

## 军哥可能问的问题及答案

**Q1：为什么要设委员会？直接决策不行吗？**

A：四个原因：
1. **多方视角**：业务督导知道坐席怎么想，量化代表知道数据怎么算，C3 们知道运营怎么协调
2. **责任共担**：共同决策、共同承担，不是一个人说了算
3. **供应商声音**：虽然供应商不在委员会，但业务督导会代表他们的利益
4. **决策质量**：5 个人讨论比 1 个人拍脑袋更靠谱

**Q2：三条红线怎么来的？有数据支撑吗？**

A：都是基于 2025 年的实际问题：
- 单次 30%：去年有 3 次超 50% 的调整，供应商强烈反弹
- 次月生效：去年有次当月生效，当月投诉率上升 40%
- 3 个月频次：去年有一个月调了 2 次，供应商没法做人员规划

**Q3：每次都要我终审，会不会太多？**

A：两个考虑：
1. **控制总量**：因为有 3 个月频次限制，一年最多 4 次常规会议
2. **分级审批**：常规调整委员会 3/5 通过就行，只有重大调整才需要你严审
3. **最终决策权**：每项调整都过你，确保战略方向不跑偏

**Q4：收入底线什么时候能出来？**

A：需要 2-3 周调研：
1. 与行业拉齐（同业供应商成本结构）
2. 与供应商调研（座谈会、问卷）
3. 测算各业务线底线标准

**Q5：供应商反馈通道是干嘛的？**

A：给供应商一个正式说话的机会：
- 调整方案确定后、生效前，可以提异议
- 有正式申请表，要写清楚影响和数据
- 委员会要在 5 天内给回复（采纳/部分采纳/维持原案）
