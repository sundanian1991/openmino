Nian Design System / Textures

15 种 CSS 材质。

每条 CSS 生成一种背景纹理——不需要图片文件。展示时 opacity 已放大到可见级别，代码块里标注的是实际使用的低 opacity 值。

Noise / 噪点

噪点

SVG feTurbulence 生成的随机颗粒。胶片/纸张/混凝土三种质感。

#1 Film Grain Fine 浅色底 · 35mm 胶片

前景标题

细颗粒噪点在浅色底上产生微妙的颗粒感，像未冲洗的胶片。

展示 opacity .15 → 实际使用 .03
background-image: url("data:image/svg+xml,…feTurbulence baseFrequency='0.9' numOctaves='4'…"); opacity: 0.03; /* 实际值 */ /* 适合：全局背景颗粒感 */
#4 Paper Aged 暖色底 · 老化纸张

前景标题

温暖的手稿质感。底色 #F5E6D3 加 SVG noise。

展示 opacity .9 → 实际使用 .04
background: #f5e6d3; background-image: url("data:image/svg+xml,…feTurbulence baseFrequency='0.4'…"); opacity: 0.04; /* 实际值 */ /* 适合：古老手稿感、温暖氛围 */
#6 Concrete Rough 深色底 · 混凝土

前景标题

粗野主义建筑感。粗糙颗粒覆盖深色表面。

展示 opacity .2 → 实际使用 .06
background-image: url("data:image/svg+xml,…feTurbulence type='turbulence' baseFrequency='0.6'…"); opacity: 0.06; /* 实际值 */ /* 适合：粗野主义、工业质感 */
Weave / 编织

编织

repeating-linear-gradient 生成的织物纹理。十字交叉线纹和丝绸微光。

#5 Linen Weave 浅色底 · 亚麻编织

前景标题

横纵双向交叉线纹，像织物的经纬线。

展示 opacity .12 → 实际使用 .03
background-image: repeating-linear-gradient(0deg, transparent, transparent 2px, rgba(0,0,0,0.03) 2px, rgba(0,0,0,0.03) 4px), repeating-linear-gradient(90deg, transparent, transparent 2px, rgba(0,0,0,0.03) 2px, rgba(0,0,0,0.03) 4px); /* 实际 opacity .03，展示放大到 .12 */
#25 Silk Sheen 浅色底 · 丝绸光泽

前景标题

一条对角线方向的微光带。极简的奢侈感。

展示 opacity .06 → 实际使用 .03
background: linear-gradient(110deg, transparent 30%, rgba(255,255,255,0.03) 45%, transparent 55%); /* 极低 opacity 单色渐变，丝绸般光泽 */
Dot / 点阵

点阵

radial-gradient 生成的均匀点阵。印刷复古感和高密度半色调。

#8 Dot Matrix 浅色底 · 12px 点阵

前景标题

均匀分布的圆点，像印刷品的网点。nian v4 Hero 用的就是这种。

展示 opacity .25 → 实际使用 .1
background-image: radial-gradient(circle, rgba(0,0,0,0.1) 1px, transparent 1px); background-size: 12px 12px; /* nian 最常用的背景纹理 */
#24 Halftone 浅色底 · 6px 半色调

前景标题

比 Dot Matrix 更密更小的点。波普艺术/漫画质感。

展示 opacity .2 → 实际使用 .04
background-image: radial-gradient(circle, currentColor 1px, transparent 1px); background-size: 6px 6px; opacity: 0.04; /* 实际值 */
Line / 线条

线条

repeating-linear-gradient 生成的各种线条纹理。对角线、交叉线、网格。深色底上用白色线条。

#9 Diagonal Lines 浅色底 · 45度对角线

前景标题

蓝图/技术感的 45 度斜线。

展示 opacity .08 → 实际使用 .03
background-image: repeating-linear-gradient(45deg, transparent, transparent 10px, rgba(0,150,200,0.03) 10px, rgba(0,150,200,0.03) 11px);
#10 Cross-Hatch 浅色底 · 交叉线

前景标题

双向交叉细线。铅笔素描质感。

展示 opacity .08 → 实际使用 .02
background-image: repeating-linear-gradient(45deg, transparent, transparent 5px, rgba(0,0,0,0.02) 5px, rgba(0,0,0,0.02) 6px), repeating-linear-gradient(-45deg, transparent, transparent 5px, rgba(0,0,0,0.02) 5px, rgba(0,0,0,0.02) 6px);
#19 Grid Fine 深色底 · 20px 细网格

前景标题

白色线条在深色底上产生技术/UI 背景网格感。

展示 opacity .08 → 实际使用 .03
background-image: linear-gradient(rgba(255,255,255,0.03) 1px, transparent 1px), linear-gradient(90deg, rgba(255,255,255,0.03) 1px, transparent 1px); background-size: 20px 20px; /* 注意：深色底才能看见白色网格线 */
#20 Grid Wide 深色底 · 60px 宽网格

前景标题

更大的格子间距，建筑/大尺度结构感。

展示 opacity .06 → 实际使用 .03
/* 同 Grid Fine，background-size: 60px 60px */ /* 适合：深色 section 背景的网格纹理 */
#29 Pixel Grid 深色底 · 4px 像素网格

前景标题

极细密的绿色网格。数字/8-bit 感。

展示 opacity .06 → 实际使用 .02
background-image: linear-gradient(rgba(0,255,0,0.02) 1px, transparent 1px), linear-gradient(90deg, rgba(0,255,0,0.02) 1px, transparent 1px); background-size: 4px 4px;
Particle / 粒子

粒子

#23 Dust Particles 深色底 · 尘埃

前景标题

#27 Water Ripple 浅色底 · 水波纹

前景标题

同心圆波纹。神秘/梦幻氛围。

background: repeating-radial-gradient(circle, transparent, transparent 10px, rgba(100,200,255,0.02) 10px, transparent 20px);
