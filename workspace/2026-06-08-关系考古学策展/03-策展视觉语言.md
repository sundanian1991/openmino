# 策展视觉语言

> 阶段 3 产出 | 2026-06-08

---

## 设计参照系：田野笔记 / 研究笔记本

不是博物馆、不是画廊、不是机构网站。是翻开一个人的笔记本——里面夹着对话记录、手写批注、剪报、时间线。

功能性定义：
- 文字是主体，不是图片的说明
- 材料以"原始状态"呈现（对话就是对话，不美化）
- 时间感通过排版节奏传递，不靠动画
- 导航像翻笔记本的页签，不像网站菜单

## 材料身份规则

| 材料类型 | 内容 | 处理方式 |
|----------|------|----------|
| **证据** | 对话原文、思考日志原文、场景手册原文 | 原样呈现，不改写，保留时间戳和格式痕迹 |
| **定位** | 时间线、章节标记、日期标签 | 始终可见，标记当前位置 |
| **分析** | 维度对比表、进化路径图、关系图 | 从证据中提炼，与证据保持视觉距离 |
| **氛围** | 无 | 不需要。材料本身足够 |

关键判断：这个项目不需要氛围材料。对话和思考日志本身就带有情感密度，加纹理、加动效反而稀释。

## 字体层级

| 层级 | 角色 | 方向 |
|------|------|------|
| **标题** | 对话的核心问题 | 大号、紧凑、不装饰 |
| **日期标签** | 时间锚点 | 小号、等宽、弱化 |
| **对话正文** | 年老师 / Mino 的原话 | 正文号，保持对话格式（缩进或引号区分角色） |
| **分析文本** | 策展者的判断和解读 | 正文号，与对话正文有视觉区分（如不同字重或颜色） |
| **图注/来源** | 材料出处 | 最小号，弱化 |

## 色彩系统

基于 Nian 14 色，克制使用：

| 用途 | 色彩 |
|------|------|
| 背景 | 白/近白 |
| 正文 | 深灰（非纯黑） |
| 强调 | 1 个主色（用于标题、关键转折标记） |
| 弱化 | 浅灰（日期、来源、图注） |
| 对话角色区分 | 不用颜色区分——用排版（缩进/位置） |

排除：深色背景、渐变、多色系统。气质是"翻开笔记本"，不是"走进展厅"。

## 版式节奏

- 单列阅读，不并排
- 阅读列宽适中（600-700px），不撑满屏幕
- 时间线作为常驻侧栏或顶部导航，不占主阅读区
- 对话块之间留白充分，每个转折点之间有呼吸感
- 来源/注释区域在页面底部或折叠

## 交互语法

| 交互 | 作用 | 方式 |
|------|------|------|
| **进入** | 从时间线进入某个对话 | 点击时间线节点，页面滚动到对应位置 |
| **关联** | 看到对话之间的因果 | 对话页内"关联"链接，跳转到相关对话 |
| **揭示** | 查看原始材料来源 | 折叠/展开原始文件路径 |
| **返回** | 回到全局地图 | 顶部或侧栏的时间线始终可回 |
| **切换** | 在对话和分析之间跳转 | Tab 或锚点切换，不离开当前时间点 |

## 排除方向

| 排除 | 原因 |
|------|------|
| 深色背景 | 与田野笔记气质冲突 |
| 大面积图像 | 材料是文字，不是图像 |
| 时间线动画 | 时间感靠排版，不靠运动 |
| 卡片式布局 | 每个对话是连续叙事，不是独立卡片 |
| AI 主题的科技感视觉 | 避免"AI 展览"的刻板印象 |
