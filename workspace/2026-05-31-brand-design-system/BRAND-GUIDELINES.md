# Nian's Brand Guidelines

> 品牌规范：视觉核心、主品牌、备选品牌的使用场景和变体

---

## 1. 视觉核心：S

### 1.1 使用场景

| 场景 | 说明 |
|------|------|
| Logo 标志 | 作为品牌标志的核心图形 |
| 点阵图案 | 装饰性背景图案 |
| 水印 | 透明叠加在图片或背景上 |
| 图标 | App 图标、Favicon |
| 社交媒体头像 | Instagram、Twitter 等 |

### 1.2 变体

| 变体 | 说明 | 使用场景 |
|------|------|---------|
| **几何 S** | 标准字体，简洁现代 | Logo、正式场合 |
| **点阵 S** | 5×7 网格，点阵风格 | 装饰、品牌标识 |
| **简化 S** | 极简线条 | 小尺寸、图标 |
| **圆形 S** | 圆形背景 | 头像、徽章 |

### 1.3 尺寸规范

| 用途 | 最小尺寸 | 推荐尺寸 |
|------|---------|---------|
| Logo | 24px | 48-96px |
| 点阵 | 32px | 64-128px |
| 图标 | 16px | 24-32px |
| 头像 | 32px | 48-72px |

### 1.4 最小空间

- **Logo**：四周留白等于 S 高度的 50%
- **点阵**：四周留白等于点阵宽度的 25%
- **图标**：四周留白等于图标尺寸的 25%

---

## 2. 主品牌：Nian's

### 2.1 使用场景

| 场景 | 说明 |
|------|------|
| 官方网站 | 首页、产品页、关于页 |
| 社交媒体 | Instagram、Twitter、LinkedIn |
| 产品包装 | 标签、吊牌、包装盒 |
| 营销材料 | 海报、传单、广告 |
| 正式文档 | 合同、提案、报告 |

### 2.2 规范

**字体：** Georgia（衬线）
**颜色：** 
- 浅色背景：var(--text-display) #2C2C2C
- 深色背景：var(--bg) #FAFAF8

**间距：**
- Logo 最小空间：S 高度的 50%
- 文字与 Logo 间距：16px

### 2.3 变体

| 变体 | 说明 | 使用场景 |
|------|------|---------|
| **标准** | Nian's（衬线） | 官方场合 |
| **大写** | NIAN'S | 强调场合 |
| **小写** | nian's | 休闲场合 |

### 4.2 视觉核心使用

| 场景 | 变体 |
|------|------|

| 头像 | 圆形 S |

| 背景 | Logo 颜色 |
|------|----------|
| 浅色 | var(--text-display) #2C2C2C |
| 深色 | var(--bg) #FAFAF8 |

---

## 5. 禁止用法

- ❌ 不要拉伸或变形 S
- ❌ 不要旋转 S（除非是点阵装饰）
- ❌ 不要给 S 添加阴影或渐变
- ❌ 不要在复杂背景上使用 S
- ❌ 不要混合使用不同品牌的元素
