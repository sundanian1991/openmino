# storyboard.md — 视觉叙事设计

**数据集**：80-制造-中国新能源汽车销量
**日期**：2026-05-03

---

## 叙事弧线

**单图场景** — 采用"断层展示"的 McKinsey Exhibit 风格。

### 一句话灵魂

> "比亚迪独占 40% 份额，小米首年交付即入局，新势力加速分化"

### 签名视觉元素

- **横向柱状图**：15 个品牌从上到下按排名排列
- **双重高亮**：比亚迪（统治者，暖橙色）+ 小米（新入局者，次强调色）
- **断层可视化**：比亚迪柱子长度是第二名的近 5 倍，视觉上形成"截断感"
- **增速标签**：每个品牌右侧标注同比变化，正增长用黑色，负增长用灰色斜体

### 阅读路径

```
1. 标题：结论性标题 → "比亚迪独占中国新能源车市 40% 份额"
2. 第一眼：超长柱子（比亚迪）→ 断层式差距
3. 第二层：第二梯队（吉利/特斯拉/五菱/理想）→ 20 万辆内的缠斗
4. 细节：小米"首年"标注 → 新玩家的冲击力
5. 补充：哪吒/埃安的负增长 → 淘汰赛已经开始
```

### 克制声明

- 不做堆叠或分组（保持单维度的纯粹性）
- 不用饼图（15 个品类太多，饼图会混乱）
- 不超过 2 种强调色（比亚迪暖橙 + 小米次强调）
- 不在柱子上标注绝对值数字（长度本身即信息）
