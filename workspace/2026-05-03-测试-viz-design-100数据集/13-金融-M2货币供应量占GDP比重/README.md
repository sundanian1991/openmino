# M2货币供应量占GDP比重

- **来源**: World Bank (https://data.worldbank.org/indicator/FM.LBL.BMNY.GD.ZS?locations=CN)
- **数据范围**: 2015-2024
- **说明**: 广义货币(M2)占GDP百分比及相关增速

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| broad_money_pct_gdp | M2占GDP比重(%) |
| m2_growth_pct | M2同比增速(%) |
| gdp_growth_pct | GDP实际增速(%) |
| m1_growth_pct | M1同比增速(%) |
