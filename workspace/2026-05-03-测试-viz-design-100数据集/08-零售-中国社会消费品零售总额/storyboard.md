# storyboard.md — 中国社会消费品零售总额视觉叙事设计

## Phase 0 全局风格锁定（单图，简化）

- **风格学派**：经济报告派（Economic Report）— 克制、数据密度高、用面积传递量感
- **色系**：Warm 主色（网上零售）+ Stone 辅色（线下零售）
- **字体**：系统默认无衬线

## 叙事弧线

**弧线类型**：渐变重构式（Structure Shift）

```
起点：2015年，线上占比13%（线下主导时代）
  ↓
缓慢渗透：2016-2019，线上占比稳步提升（18%→26%）
  ↓
疫情催化：2020年，线上逆势增长，线下萎缩（渗透率跃升）
  ↓
韧性恢复：2021-2024，总量持续增长，线上占比稳定在32%
  ↓
终局：2024年，线上占比32%，十年翻了2.5倍
```

## 视觉节拍

### 节拍1：初始状态（2015）
- 视觉处理：面积图底部薄层（网上零售 3.88万亿）
- 含义：线上零售刚刚起步

### 节拍2：渗透加速（2016-2019）
- 视觉处理：Warm 500 面积逐渐变厚
- 含义：线上零售在总量增长的同时提升渗透率

### 节拍3：疫情冲击（2020）
- 视觉处理：总面积出现凹陷（负增长-3.9%），但线上面积反增
- 标注文字："2020年疫情冲击 -3.9%，线上逆势增长"
- 含义：疫情是线上渗透的催化剂

### 节拍4：总量新高（2024）
- 视觉处理：面积图达到最高点
- 标注文字："2024年总量48.79万亿，线上占比32%"
- 含义：十年增长+结构重构的结果

## 阅读路径

1. **第一眼**：标题"线上渗透率翻倍"→ 建立关注点
2. **第二眼**：面积图的整体轮廓 → 感受总量增长
3. **第三眼**：Warm色层的厚度变化 → 理解线上渗透率跃迁
4. **第四眼**：两个标注 → 疫情冲击 + 终局数据
5. **第五眼**：脚注中的增速数据 → 理解增速换挡

## 克制声明

- 不做：饼图（>3分类禁止）、双Y轴、3D效果
- 不做：显示全部10年的数值标签
- 不做：预测线（数据截止到2024）
- 只做：2层堆叠面积 + 2个标注 + 增速脚注

## 布局方案

**单图布局**：
- 面积图占据主要区域
- 标注在图内叠加
- 总画布：800 x 550px

```
┌─────────────────────────────────────────┐
│  标题（结论性）                          │
│  副标题（时间范围+对象）                  │
├─────────────────────────────────────────┤
│                                         │
│  堆叠面积图                              │
│  ├── 底层：网上零售（Warm 500）          │
│  ├── 中层：餐饮收入（Warm 100）          │
│  └── 上层：线下零售（Stone 200）          │
│  标注 x2                                 │
│                                         │
├─────────────────────────────────────────┤
│  脚注：增速变化 + 数据来源                │
└─────────────────────────────────────────┘
```

**13-VISUALIZATION 合规确认**：
- 禁止双Y轴 → 增速用脚注传达
- 高亮 <=10% → 网上零售层高亮
- 颜色 <=2 ramp → Warm + Stone
- 标题结论性
- 无渐变/阴影/emoji
