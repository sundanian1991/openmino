# storyboard.md — 全球音乐流媒体用户数

## Phase 0: 全局风格锁定

本数据集为单图需求，无需全局风格锁定。

**风格学派**：麦肯锡Exhibit风格 — 干净、数据驱动、结论优先
**色系**：Warm单色系 — 灰度主体 + Warm800高亮领导者
**字体**：系统默认无衬线

## 叙事弧线

**弧线类型**：01-直给式
**一句话灵魂**：Spotify以绝对优势领跑全球音乐流媒体，用户数是第二名的近两倍
**签名元素**：横向条形图，第一名暖色高亮，其余灰度

## 视觉节拍

1. **标题区**：结论性标题 — "Spotify用户数碾压第二名，一超格局稳固"
2. **副标题**：时间范围 + 数据来源
3. **图表区**：横向条形图，按用户数降序排列
4. **标注**：Spotify标注"行业领导者"，第二名标注差距倍数
5. **图例/脚注**：单位说明 + 数据来源

## 阅读路径

```
标题（结论）→ 注意最长条形（Spotify）→ 对比第二名差距 → 扫其余平台
```

## 克制声明

- 不添加装饰性元素
- 不使用渐变或阴影
- 高亮限制在Spotify一家（1/8 < 10%）
- 不使用动画效果

## 数据组织

**排序**：按用户数降序排列
**聚合**：无需聚合
**字段**：排名、平台名称、用户数（亿）
