# Global Style — 100 数据集测试

> Phase 0 全局风格锁定。后续每张图从此继承，不得各自选色。

---

## 风格学派

- **hash seed**: viz-design-test-100
- **风格**: 现代极简（数据驱动，consulting-grade）
- **字体**: 标题 Georgia serif（如有），正文 -apple-system / PingFang SC

## 全局色系

| 角色 | 色值 | 用途 |
|------|------|------|
| **主色** | #3D3D3A | 主线/主折线/主柱 |
| **暖橙** | #D97757 | 高亮/重点品牌/关键省份 |
| **绿** | #788C5D | 增长/正面异常值 |
| **红** | #c25030 | 下跌/负面异常值 |
| **灰-深** | #87867F | 次级元素/标注文字 |
| **灰-中** | #A3A29B | 脚注/来源 |
| **灰-浅** | #C1C0B8 | 非高亮柱/条 |
| **灰-极浅** | #D1CFC5 / #E3DACC | 堆叠底部/背景 |
| **背景** | #FAF9F5 | 页面背景 |
| **卡片** | #fff | 图表卡片 |

## 全局字体

- 标题: 17px, bold 700
- 副标题: 11px, 灰 #A3A29B
- 提要: 13px, 行高 1.7
- 轴标签: 10-12px
- 标注: 10px, bold 600-700

## 通用布局

- 画布宽度: 800-900px
- 标题左对齐
- 提要左对齐在图下方
- 来源左对齐或左下

## 通用规则

- 网格线: 淡化或去掉
- 高亮 ≤10%
- 颜色 ≤5 种/图
- 标注只指数据点，不写解释
