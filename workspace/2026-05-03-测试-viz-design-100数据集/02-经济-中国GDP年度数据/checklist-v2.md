# Checklist — 中国 GDP 年度数据

## Loop 2 减法迭代记录

| 操作 | 理由 |
|------|------|
| 删除 GDP 总量柱状图 | 核心叙事是增速+结构，不是总量。柱状图占空间且无结构价值 |
| 删除 legend | 双轴+两条线已自解释，legend 冗余 |
| 4 个标注减到 2 个 | 2020 疫情和 2021 反弹是同一件事。只留疫情异常点和 2024 最新 |
| 删除"疫后反弹 8.6%"标注 | 是低基数技术性反弹，不是真正的增长叙事，容易误导 |

## Loop 3 验收

**人类视角读图**：
- 第一眼看到什么？→ 两条线，一条下降（增速），一条上升（服务业）
- 核心信息传达到位了吗？→ 标题"增速从 7% 降到 5%，服务业突破 57%"配合两条交叉线，一目了然
- 右侧注释有价值吗？→ 有，总结了"一降一升"的张力，并抛出问题
- 有没有不对劲的地方？→ 右侧注释区在 plot area 外面，不和数据打架

**机械检查**：
| 检查项 | 状态 |
|--------|------|

| 高亮 ≤10% | ✓ 2 个圆点 + 右侧注释 |
| 标题结论性 | ✓ |
| 标注对齐 | ✓ graphic 精确定位，不偏移 |
