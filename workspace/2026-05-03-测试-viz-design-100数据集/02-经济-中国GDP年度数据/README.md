# 中国GDP年度数据

- **来源**: World Bank (https://data.worldbank.org/indicator/NY.GDP.MKTP.CN?locations=CN)
- **数据范围**: 2015-2024
- **单位**: 亿元人民币 / 美元 / 百分比

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| gdp_cny_billion | GDP（亿元人民币） |
| gdp_per_capita_usd | 人均GDP（美元） |
| gdp_growth_pct | GDP实际增长率(%) |
| services_pct_gdp | 服务业占GDP比重(%) |
| industry_pct_gdp | 工业占GDP比重(%) |
