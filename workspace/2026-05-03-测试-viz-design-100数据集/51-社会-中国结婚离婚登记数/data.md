# 中国结婚与离婚登记数（2010-2023）

| 年份 | 结婚登记（万对） | 离婚登记（万对） | 离结比（%） |
|------|----------------|----------------|----------|
| 2010 | 1,241 | 268 | 21.6 |
| 2011 | 1,302 | 287 | 22.0 |
| 2012 | 1,324 | 310 | 23.4 |
| 2013 | 1,347 | 350 | 26.0 |
| 2014 | 1,307 | 364 | 27.8 |
| 2015 | 1,224 | 384 | 31.4 |
| 2016 | 1,143 | 416 | 36.4 |
| 2017 | 1,063 | 437 | 41.1 |
| 2018 | 1,014 | 446 | 44.0 |
| 2019 | 927 | 470 | 50.7 |
| 2020 | 814 | 373 | 45.8 |
| 2021 | 764 | 284 | 37.2 |
| 2022 | 683 | 210 | 30.7 |
| 2023 | 768 | 259 | 33.7 |

## 说明

- 2020-2021 年受疫情影响，结婚登记大幅下降
- 2021 年起实行离婚冷静期（30 天），离婚登记明显减少
- 结婚人数自 2013 年峰值后连续下降
- 2023 年结婚登记有所回升，主要因疫情后补偿性结婚
