# 中国进出口贸易额

- **来源**: World Bank
  - 出口: https://data.worldbank.org/indicator/NE.EXP.GNFS.CD
  - 进口: https://data.worldbank.org/indicator/NE.IMP.GNFS.CD
- **数据范围**: 2015-2024
- **单位**: 十亿美元

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| exports_usd_billion | 出口总额（十亿美元） |
| imports_usd_billion | 进口总额（十亿美元） |
| trade_balance_usd_billion | 贸易差额（十亿美元） |
