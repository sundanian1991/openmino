# 中国固定资产投资总额

- **来源**: 国家统计局年度统计公报
- **数据范围**: 2015-2024
- **单位**: 万亿元人民币 / 百分比

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| total_investment_trillion_cny | 固定资产投资总额（万亿元） |
| manufacturing_growth_pct | 制造业投资增速(%) |
| infrastructure_growth_pct | 基建投资增速(%) |
| real_estate_investment_trillion_cny | 房地产开发投资（万亿元） |
