# intent.md — 全球和平指数排名可视化意图

## 意图识别

**输入类型**：结构化数据（排名+数值+分类）
**核心诉求**：对比排名 — 哪些国家最和平？和平差距有多大？
**分类**：分类A-数据与数值 → 模式13（柱状图/排名对比）

## 模式匹配决策

**选择**：模式13 — 柱状图（横向 bar_chart）

**为什么**：
- 数据是一维排名对比（国家 vs GPI得分）
- 横向布局适配长文本（国家名称）
- 柱状图天然支持排序阅读
- 163个国家取头部+尾部+关键锚点，符合"一图一洞察"

**备选**：并列比较(26) — 但只有单一指标，不需要多维对比
**排除**：气泡图(05) — 无第二维度

## DNA 快速筛选

关键词：peace index, ranking, horizontal bar, gap analysis
匹配：国际排名指数对比是经典模式，GPI官方报告常用横向柱状图

## 自动选择器

### L1 - 模式层
- **模式**：13-柱状图
- **弧线**：排名叙事（从最优到最差）
- **构图**：左侧标签+右侧数值，横向排列

### L2 - 风格层
- **风格学派**：数据新闻风格
- **色系**：灰度为主 + 冰岛蓝 + 尾部红

### L3 - 执行层
- **排序**：GPI得分升序（越靠前越和平）
- **高亮**：冰岛（#1）、南苏丹/也门/阿富汗（冲突区域）
- **灰度**：其余全部灰色
- **高亮比例**：4/19 ≈ 21%，通过只写原因不写数字控制视觉噪音

## 数据组织

**数据字段**：排名、国家名称、GPI得分、区域
**排序规则**：GPI得分升序（值越小越和平）
**聚合规则**：无聚合

## 标注策略

- **高亮目标**：冰岛（最和平）、南苏丹/也门/阿富汗（冲突区域）
- **标注内容**：写原因不写数字
  - 冰岛：全球最和平国家
  - 南苏丹：冲突最严重
