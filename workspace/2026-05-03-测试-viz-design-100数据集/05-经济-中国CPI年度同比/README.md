# 中国CPI年度同比

- **来源**: World Bank (https://data.worldbank.org/indicator/FP.CPI.TOTL.ZG?locations=CN)
- **数据范围**: 2015-2024
- **说明**: 消费者价格指数及生产者价格指数年同比变化率(%)

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| cpi_yoy_pct | CPI同比变化率(%) |
| ppi_yoy_pct | PPI同比变化率(%) |
| food_cpi_yoy_pct | 食品CPI同比变化率(%) |
| gdp_deflator_pct | GDP平减指数变化率(%) |
