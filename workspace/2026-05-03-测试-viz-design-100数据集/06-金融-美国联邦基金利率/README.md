# 美国联邦基金利率

- **来源**: FRED (https://fred.stlouisfed.org/series/FEDFUNDS)
- **数据范围**: 2015-2025（月度关键采样点）
- **说明**: Federal Funds Effective Rate

## 字段说明
| 字段 | 说明 |
|------|------|
| date | 日期 |
| fed_funds_rate_pct | 联邦基金有效利率(%) |
| cpi_yoy_pct | 美国CPI同比变化率(%) |
| unemployment_pct | 美国失业率(%) |
| gdp_growth_pct | 美国GDP增长率(%) |
