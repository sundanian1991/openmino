# Phase 3: SPEC — 数据集17 全球主要国家股市市值

## 叙事意图

**可视化目的**：让读者一眼看出各国股市规模的量级差距和中国股市的周期性波动。

**想传达什么**：中国股市市值远超日德，但2021年后经历了显著回调。

**结论**：中国股市在全球主要经济体中市值最大但波动最剧烈，日德市场相对稳定。

**思路**：折线图天然表达时间序列趋势，4条线并置呈现对比关系。

---

## 视觉执行

### 模式选择

- **模式**：07-折线图
- **ECharts chartType**：multi_line
- **理由**：时序+多对象 = 折线图最匹配

### 标题

- **主标题**：中国股市市值2021年见顶后回落25%，日德窄幅震荡
- **副标题**：2015-2024 主要国家股市总市值走势（十亿美元）

### 视觉编码

| 编码 | 映射 |
|------|------|
| X轴 | 年份（2015-2024） |
| Y轴 | 股市总市值（十亿美元） |
| 颜色 | 国家（4色区分） |
| 线型 | 实线，粗细一致 |

### 数据组织

使用中国、日本、德国三国完整数据（10年连续）。英国和法国数据不连续且年份不完整，加入会导致折线断裂，仅作为参考信息在脚注中说明。

### 标注策略

- 高亮中国2021年峰值（唯一标注点）
- 标注内容："峰值 14,437B → 回调25%"
- 高亮比例：1/40 = 2.5%，远低于10%阈值

### 图例/辅助

- 图例位置：右下角
- 脚注：英国（仅2021-2022）、法国（仅2015-2018）数据不完整，未纳入主线；美国数据缺失

### 布局

- 画布：800 x 550
- 配色：Warm 色系，中国线用强调色，其余灰调
- Y轴起始：0（展示绝对规模差距）
