# Visual Storyboard — 视觉叙事设计

## Phase 0：全局风格锁定

- **整体叙事弧线**：02-对比式 — TOP3 冲击 → 梯队分化 → 全球失衡
- **风格学派**：restrained-warm
- **全局色阶**：
  - 主 ramp：Warm（`#c26d3a` 主色 / `#6b3410` 标题）
  - 辅助 ramp：Stone（`#857d74` 基准 / `#ada599` 网格）
  - 语义色：Warm（中国城市高亮）/ Stone（其他城市）
- **字体家族**：system-ui, -apple-system, sans-serif
- **字号层级**：标题 16/600、副标题 11/400、轴标签 11/400、标注 11/600、脚注 10/400
- **圆角/间距**：卡片圆角 10px、卡片间距 12px、卡片内边距 16px
- **什么必须统一**：色系、字号、圆角、间距、高亮规则
- **什么必须变化**：标题、数据、高亮目标
- **什么可以复用**：基准线样式、脚注格式

### Scene 1: 全球最难买房城市 TOP20

- **场景论文**：用横向柱状图揭示"中国 3 城霸占 TOP3，深圳是纽约的 3 倍"
- **在叙事中的角色**：开场定调 — 第一眼看到 TOP3 全部是中国城市
- **叙事节拍**：
  - 第一眼看到什么 → TOP3 中国城市的超长条（视觉冲击：断层领先）
  - 第二眼看到什么 → 梯队标注：超高负担（>25）/ 高负担（15-25）/ 中等负担（10-15）/ 相对合理（<10）
  - 最终理解什么 → 全球购房负担极度失衡，亚洲城市占据不负担顶部
- **签名视觉元素**：TOP3 中国城市 Warm 色高亮，其余 Stone 色，梯队间以虚线分隔
- **克制声明**：不显示具体数值标签（区间值不适合精确标注）、不画 3D、不加动画
