# 黄金价格年度数据

- **来源**: LBMA Gold Price / World Bank Commodity Markets (https://www.lbma.org.uk/prices-and-data/precious-metal-prices)
- **数据范围**: 2015-2024
- **单位**: 美元/盎司 / 百分比 / 吨

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| gold_price_usd_per_oz | 黄金年度均价（美元/盎司） |
| usd_index_avg | 美元指数年度均值 |
| us_10yr_yield_pct | 美国10年期国债收益率(%) |
| gold_etf_tonnes | 全球黄金ETF持仓量（吨） |
