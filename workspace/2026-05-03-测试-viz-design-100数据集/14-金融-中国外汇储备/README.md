# 中国外汇储备

- **来源**: World Bank (https://data.worldbank.org/indicator/FI.RES.TOTL.CD?locations=CN)
- **数据范围**: 2015-2024
- **单位**: 十亿美元 / 吨

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| total_reserves_usd_billion | 外汇储备总额（十亿美元） |
| gold_reserves_tonnes | 黄金储备（吨） |
| trade_balance_usd_billion | 贸易顺差（十亿美元） |
| usd_cny_avg | 年均汇率（1美元兑人民币） |
