# Phase 0 + Phase 1 — 意图决策

## Phase 0: Big Idea

**数据本质**：十年时间序列（餐饮收入 + 增长率 + 占社零比例）+ 2023 年细分市场构成。

**Big Idea**：中国餐饮市场经历疫情"深V"反弹后于2023年创新高至5.29万亿，但正餐一家独大（45%）的格局暴露了结构单一的风险。

**全局风格锁定**（单图场景，轻量）：
- 风格学派：McKinsey Exhibit
- 色系：Warm，高亮 2020 年（疫情低点）和 2023 年（新高拐点）
- 字体：系统无衬体

## Phase 1: 意图识别

### 你要表达什么？

**趋势拐点 + 波动叙事**。不是简单画一条线，而是让读者感受到"疫情打击 → 快速恢复 → 创新高"的戏剧性。

### 模式匹配

**主图模式选择：dual_axis（双轴图）**

匹配理由：
- 有时间序列（年份）+ 两个不同量纲的指标（营收绝对值 + 增长率%）
- 双轴图将趋势和增速放在同一视觉空间，拐点自然浮现
- 2020 年的深V 和 2023 年的新高在同一图中形成强烈对比

**备选**：
- line_chart：只显示营收，缺少增长率视角 → 不够丰富
- 面积图(08)：适合累积量，这里不是 → 不选

### 数据全景分析（模式 B：AI 自主决策）

从数据中我选了 2 个分析角度：
1. **趋势拐点（主图）**：2015-2024 营收 + 增长率双轴图
2. ~~细分市场构成（辅图）~~ → 2023 年细分市场构成饼图/柱状图，信息量有限（正餐一家独大），融入主图脚注
