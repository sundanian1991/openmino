# storyboard.md — 视觉叙事设计

**数据集**：79-制造-全球汽车品牌销量排名
**日期**：2026-05-03

---

## 叙事弧线

**单图场景** — 采用"结论先行"的 McKinsey Exhibit 风格。

### 一句话灵魂

> "丰田守擂但收缩，比亚迪以 41% 增速冲进全球第六"

### 签名视觉元素

- **横向柱状图**：15 家集团从左到右按排名排列，柱子长度 = 销量
- **单一高亮**：比亚迪用暖橙色（#E8875F），其余全部灰度
- **增速标注**：每根柱子右侧标注同比变化（正绿负红或仅文字）
- **Top 3 标记**：丰田/大众/现代-起亚用稍深灰色区分梯队

### 阅读路径

```
1. 标题：结论性标题 → "丰田守住全球第一，比亚迪以 41% 增速冲入 Top 6"
2. 第一眼：最长柱子（丰田）+ 唯一彩色柱子（比亚迪）→ 两个锚点
3. 第二层：柱子长度递减 → 排名梯度一目了然
4. 细节：增速标注 → 发现谁在增长谁在收缩
```

### 克制声明

- 不做双轴图（避免混乱）
- 不分组标（不区分国家/地区，让排名本身说话）
- 不超过 2 种颜色（灰度 + 一个强调色）
- 不在柱子上写具体数字（标注仅写增速，绝对值通过柱子长度感知）
