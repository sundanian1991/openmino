# storyboard.md — 中国移动通信基站数量

## Phase 0: 全局风格锁定

- **风格学派**：现代数据新闻风格（干净、高对比、重点突出）
- **色系**：双色 — 4G用冷灰蓝，5G用Warm橙 #E8875F（突出主角）
- **字体**：系统无衬线字体
- **画布**：800 x 550px

## 视觉叙事设计

### 一句话灵魂

> 5年，5G从零到425万个基站、10亿用户——中国速度。

### 叙事弧线

07-趋势叙事 — 用时间线展示5G的爆炸式增长

### 阅读路径

1. **顶部标题** → 结论先行：5G爆发
2. **左侧Y轴（基站数）** → 堆叠面积，下方4G存量，上方5G增量
3. **右侧Y轴（用户数）** → 折线叠加，展示5G用户从0到10亿
4. **关键标注** → 2019年"5G元年"、2024年"5G用户破10亿"
5. **底部X轴** → 2015-2024时间线

### 克制声明

- 仅5G相关元素使用强调色（橙色）
- 4G基站用冷灰蓝，总数线用浅灰
- 标注不超过3个：5G元年、用户破亿、用户破10亿
- 堆叠面积填充透明度40%，避免过重

### 数据组织

- 左轴：堆叠面积（4G基站 + 5G基站）
- 右轴：折线（5G用户数）
- 时间：2015-2024（10年）
- 2015-2018年5G数据为空，从2019年开始
