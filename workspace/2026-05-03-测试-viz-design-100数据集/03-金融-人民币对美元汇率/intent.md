# Intent — 人民币对美元汇率

## Phase 0：数据解读

### 数据特征

| 维度 | 描述 |
|------|------|
| **规模** | 28条月度数据，2023-01至2025-04 |
| **字段** | date（月份）、usd_cny（汇率）、cny_neer_index（名义有效汇率指数）、trade_balance_usd_billion（贸易顺差） |
| **usd_cny 范围** | 6.75 ~ 7.32 |
| **neer 指数范围** | 126.5 ~ 106.8（持续下行） |
| **贸易顺差范围** | 16.9 ~ 104.8 十亿美元（高波动） |

### 关键发现

1. **W型走势**：usd_cny 先贬（6.75→7.32）→反弹（7.32→7.02）→再贬（7.02→7.30）
2. **名义有效汇率单边下行**：从126.5跌至106.8，30个月无有效反弹，说明对一篮子货币持续走弱
3. **贸易顺差与汇率背离**：2024-12顺差创纪录104.8十亿美元，但同期汇率7.30接近高点
4. **"稳"的代价**：对美元汇率的相对稳定，是以对一篮子货币持续贬值为代价

### Big Idea

> 人民币对美元的"稳"是假象——名义有效汇率指数30个月单边下行近20点揭示的真相是：对美元的锚定靠的是对一篮子货币的持续让步。

## Phase 1：意图锁定

### 输入类型

时序多变量数据（3个连续指标 + 时间维度）

### 数据状态

- 结构化、无缺失
- 月度频率，28个时间点
- 三条线的量纲不同（汇率~7、指数~110、顺差~70），需要双轴或标准化

### 核心关系

**主关系**：usd_cny 与 cny_neer_index 的分化趋势——对美元"稳中有升"vs对一篮子"持续走弱"
**辅关系**：贸易顺差作为背景语境（解释"为什么汇率还能撑住"）

### 模式选择

**主模式：dual_axis（双轴图）**

- 编号：viz-echarts 内置类型
- 理由：三条线量纲完全不同（汇率~7、指数~110、顺差~70十亿美元），必须双轴才能同框比较。左轴放 usd_cny + neer_index（都是汇率相关），右轴放贸易顺差。核心洞察在于 usd_cny 与 neer_index 的方向分化——usd_cny 在7.0-7.3区间波动，neer_index 却从126.5单边跌到106.8，这个"剪刀差"只有同框才能看见。

**备选模式 1：multi_line（多折线图）**
- 放弃原因：三条线量纲差距太大，单轴会导致 neer_index（~110）压平 usd_cny（~7），视觉上无法读出汇率变化。

**备选模式 2：area_chart（面积图）**
- 放弃原因：面积图强调累积量感，不适合这里的"趋势分化"叙事。neer_index 下行叠加后会产生误导性的"总量缩减"视觉。

### 输出规格

- 单图
- 目标：揭示"对美元稳 vs 对篮子贬"的结构性矛盾
