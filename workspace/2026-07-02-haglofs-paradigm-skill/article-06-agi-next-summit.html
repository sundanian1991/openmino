核心观点

四位领袖
四个关键判断

每位嘉宾的演讲都像在做技术报告，涵盖模型架构、应用落地、中美竞争等关键议题。

唐杰
智谱 AI · 从 Chat 到 Agent
Chat 时代已近尾声，AI 需从对话转向任务执行。模型进化路径：知识记忆（2020-2022）→ 复杂推理（2023-2024）→ 人类终极测试（2025+）。RLVR（可验证强化学习）是关键突破点。
杨植麟
Kimi · 世界观创造
做模型本质上是在创造一种世界观。Agent 应该是由大模型内生的 native 系统——目标、规划都可以由大模型定义。基座应用永远存在矛盾，应用速度是拉开差距的关键。
林俊旸
阿里千问 · 竞争格局
中国想在 AI 赛道反超，很难。20% 这个数字已经很乐观。基座的应用永远是矛盾——调用 API 就能解决的事情，基座公司也想做，这是不可避免的冲突。
姚顺雨
腾讯 · 需求判断
toC 的话，大部分人其实用不着那么强的智能。Agent 本身的价值有多大、是不是真的能帮到人——这是决定未来走势的第一个因素。做应用的速度是第三个关键。

技术共识

共识 01
Agent 是未来方向
从 Chat 到 Agent 是确定性趋势。多位嘉宾明确表示：Chat 时代已近尾声，下一步是走向做事。
共识 02
Agent 需要原生设计
Agent 应该由大模型内生——目标、规划都可以由大模型定义。不是外挂一个工作流框架。
共识 03
做应用速度很关键
谁能拉开半年时间窗，迅速满足应用需求，谁就能占得先机。速度本身就是护城河。
结语

同一个未来
四种视角

这届峰会的信息密度极高。四位嘉宾的性格和观点各不相同——唐杰的技术深度、杨植麟的哲学思考、林俊旸的冷静判断、姚顺雨的需求导向——但指向同一个未来。

Agent 是中国 AI 的下一个战场。谁能把 Agent 做好、做快、做到用户手里，谁就能在下一个时代占据先机。
