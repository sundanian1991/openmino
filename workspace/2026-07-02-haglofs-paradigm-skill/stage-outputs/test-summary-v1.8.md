## v1.9 迭代建议（基于第一批）

### P1（应修）
1. **补浅底品牌强调色梯队**（NC-2）：新增 `--color-forest-soft` / `--color-moss-soft` 高明度低饱和绿系浅底专用档，解决 T3 调性品牌失去浅底绿色强调能力
2. **建品牌色映射协议**（NC-3）：新建 `rules-brand-color-mapping.md`，外部品牌色→范式 token 就近映射 + 受限 `--brand-accent` token
3. **Data Table 移动端卡片化**（NC-5）：给 Data Table 加移动端塌缩方案

### P2（可修）
4. 组件骨架固定 px 全 clamp 化（NC-1）
5. Tension Grid --light 张力补偿（NC-4）
6. Pipeline --connected 连接线（NC-6）
7. V3 Split 深段归属明文化（NC-7）
