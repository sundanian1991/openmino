### 2.2 决策区（agent 推导 · 三大关键决策）

#### 🔑 关键决策 1：页面类型 → **品牌展示页**（非数据/文档/技术）

**判定依据**（intake-rules 推导规则 + paradigm-boundary 五类预设）：
- 纯品牌叙事 **无数据指标** → 排除品牌数据简报（混合页）
- 核心叙事点 **5 个 ≥4** → **品牌展示页**（不是宣言页 ≤3）
- 非设计系统本身 → 排除文档页
- 非翻页教学 → 排除知识模板

**联动组**（B 表预设 · 品牌叙事/展示页行）：
| padding | 分隔 | 宽度 | 动画 | 网格 | .cmp |
|---------|------|------|------|------|------|
| 96-120px（取 120px） | 背景色切换 | 单一 1120px | 无 | 大间距留白 | 可省 |

✅ **判定正确**：手工咖啡小品牌是典型的品牌展示页，不是数据看板/文档/技术页。

#### 🔑 关键决策 2：Hero → **V3 Split**（非 Pulse，非 Statement）

**选型推理**（rules-hero 决策树 v2 · 偏好优先 + 内容主轴首命中标）：
- 内容主轴 = **工艺哲学主张**（"慢哲学"/"不拼配"/"产地直采"是强品牌主张）→ 命中"宣言/主张/张力"分支
- 口诀：**"有宣言/主张 → Split"** → V3 Split
- **v1.7 关键裁决**：声明/宣言类默认 **V3 Split**（非 V4 Statement）；**V4 Statement 仅用于设计系统文档首页**
- 手工咖啡不是设计系统文档 → **排除 V4 Statement** ✓
- 无 KPI 数据 → **排除 V5 Pulse** ✓
- 任务指引列"Statement 或 Split"，但按 v1.7 规则，Statement(V4) 是文档页专用——Split 是声明类品牌页的范式正确默认。

✅ **判定正确**：主张驱动的品牌页 + 非 docs = V3 Split（斜切深块 + 点阵 + 年份水印）。

#### 🔑 关键决策 3：调性 → **暖调手工质感** + 咖啡棕褐就近映射 forest 槽位

**色彩映射**（rules-brand-color-mapping.md · v1.9）：
| 品牌色系 | 范式槽位 | 浅底文字 | 浅底装饰 | 深底 |
|---------|---------|---------|---------|------|
| 棕褐/大地/赭石（咖啡） | forest（暖调） | **forest-soft #547444** (4.79:1) | moss-soft / sand | forest(提亮) |

- 咖啡棕褐虽是暖色，但**就近映射 forest 槽位**（暖调绿，色相接近烘焙咖啡的焦糖褐），走 forest-soft 浅底软档可保识别又不显重
- **未用 --brand-accent**（咖啡棕与范式 forest 色相接近，无需破例口）
- 暖米白底 `#F5F3EF`（温润有人味）+ 暖灰 charcoal（泥土感）= 天然手工温度，与"暖调手工质感"一致

✅ **判定正确且一致**：调性判断（暖调手工）↔ 品牌色映射（咖啡棕→forest 暖调）↔ 范式底色（暖米白/暖灰）三者贯通。
