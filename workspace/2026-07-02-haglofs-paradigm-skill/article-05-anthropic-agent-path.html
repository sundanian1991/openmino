关键洞察

五个
反直觉真相

读完 15 篇后，你会发现 Anthropic 的 Agent 开发方法论有几个反直觉的核心理念：

洞察 01
Start Simple
多数场景不需要复杂 Agent，定义良好的工作流更高效。
洞察 02
工具设计决定上限
工具描述（Description）的质量直接影响 Agent 能力上限。
洞察 03
上下文检索创新
Contextual Retrieval 通过生成解释性上下文，解决传统 RAG 的信息碎片化问题。
洞察 04
工程化优先
从设计初期就需考虑 Harness 架构、状态持久化和错误恢复机制。
洞察 05
失败案例的价值：三篇真实故障复盘提供了宝贵的实战经验。Anthropic 公开分享失败案例，这在行业内非常罕见——大多数公司只展示成功。
