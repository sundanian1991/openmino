Pre-train · Post-train · RL Scaling · Agent 框架

"现在基本上大家在同一水平线。"罗福莉的这句话，既是对现状的判断，也是对未来的预言。1T 基座模型，是实现接近 Claude Opus 4.6 水准模型的重要入场券。全球大模型军备竞赛的第二场战役，已经打响。

这场战役的方向变了：从 Pre-train（预训练）主导的 Chat 时代，转向 Post-train（后训练）主导的 Agent 时代。不是渐进改良，是范式跃迁。

"我之前认为我们自己做的工作已经足够有创造力、足够不会被 Skill 化、不会被 Workflow 化。但我现在发现，它竟然也能！那它可不可以训出更强的模型？自己左脚踩右脚就提升了？——这是这一两年会发生的事情。"
—— 罗福莉，小米大模型团队负责人

关键判断
Anthropic 的路径是正确的——这是当下共识。在路径更清晰的情况下，国内大模型团队进入加速追赶状态。现在大家在 Pre-train 上的代差基本没有，或者说非常接近。真正的战场转移到了 Post-train 和 RL Scaling。
OpenClaw 时刻

Agent 框架的
革命性突破

罗福莉将 OpenClaw 视为"划时代的 Agent 框架"。核心价值不在模型本身，而在精细的 Context 编排（时间感知、持久化分层记忆）和多模型协同调度（自动弥补单一模型短板）。

突破 01 · 自主性与温度
通过上下文时间戳等机制实现类人交互（如提醒用户休息），不再是冷冰冰的问答机器。
突破 02 · 复杂任务处理
两小时内完成 User Agent 构建，支持 Post-train 数据生成。任务复杂度远超传统 Chatbot。
突破 03 · 中层模型激发
3B 端侧模型在框架加持下实现接近顶尖模型的效果。小模型 + 好框架 > 大模型 + 差框架。
"OpenClaw 让 3B 端侧模型在框架加持下实现接近顶尖模型的效果。小模型 + 好框架 > 大模型 + 差框架。"
—— 罗福莉谈 Agent 框架的价值
技术路径

模型进化四阶段

从知识记忆到人类终极测试，每个阶段的能力要求和训练方法完全不同。

2020–2022
知识记忆阶段
模型学会存储和检索海量知识。GPT-3 到 GPT-4 的跃迁，本质是从"记住"到"理解"。
Pre-train · Next Token Prediction
2023–2024
复杂推理阶段
Chain-of-Thought、Tree-of-Thought 等技术让模型学会"思考"。推理能力成为新的竞争焦点。
CoT · ToT · Self-Consistency
2025
Agent 执行阶段
从"回答问题"到"完成任务"。RLVR（可验证强化学习）通过代码、数学等可验证环境实现自主反馈。
RLVR · Tool Use · Planning
2025+
人类终极测试（HLE）
多模态感统、记忆与持续学习、反思与自我认知。模型开始具备类人的完整认知能力。
Multimodal · Memory · Self-Reflection

组织挑战

范式跃迁的
真正难点不在技术

系统从"以 Rollout 推理引擎为核心"转变为"以 Agent 为核心"的复杂系统。这对团队提出了更高的要求。

挑战 01
预训练的前置战略
需要提前半年到一年想清楚：这一代模型结构在未来很长一段时间里要承担什么角色。平庸的结构不一定带来平庸的效果，但一定会带来成本和效率上的劣势。
挑战 02
后训练的敏捷性
后训练与 Agent 耦合迭代，很多事情无法提前规划。需要基于当前模型能力和 Agent 范式产生"化学反应"，快速设计新的 RL Infra 架构。
挑战 03
文化与价值观
应对巨变的根基不是技术，而是文化。"像喝咖啡一样做研究"——好奇心和热爱，是穿越范式变迁的底层支撑。
"应对巨变的根基不是技术，而是文化。好奇心和热爱，是穿越范式变迁的底层支撑。"
—— 罗福莉谈组织变革
结语

范式跃迁的窗口期
以月计算，不以年计算

罗福莉的访谈信息密度极大。从技术路径到资源调配，从组织变革到文化根基——这是一次范式跃迁的全景图。

核心判断清晰且坚定：Anthropic 路径正确，国内团队加速追赶。Pre-train 差消失，真正的战场在 Post-train 和 RL Scaling。Agent 框架（OpenClaw）的出现正在显著加速整个过程。

"接下来两三个月，大家怎么发生变化，是考验团队整体研究水平、技术敏捷程度，以及怎么拥抱新的范式来做研究的关键。"
—— 罗福莉
窗口期判断 上半年到一年。 不是以年计算，是以月计算。
