> 数据源：8 个职场 GMV 目标达成率（2026 年 3 月）

### SPEC — 组合 A

**【可视化目的】**
高管只有 30 秒。这张图要让他 10 秒内知道"总体达标但分化严重"，20 秒内理解"为什么分化"，30 秒内记住"汇讯和九通是两个极端"。认知缺口：不是"达成率是多少"，而是"分化程度有多大"。

**【想传达什么】**
3 家达标、5 家未达标，极差 64 个百分点——这不是正常波动，是系统性分化。

**【结论】**
职场间达成率分化严重，需针对末位 3 家启动根因分析。

**【思路】**
金字塔式先亮结论（大数字+红绿灯判断），第二层给出 3 个支撑论据（达标组/未达标组/极差），第三层展示 8 家明细。第一眼落在顶部的红绿灯判断，第二眼扫描 3 个论据卡片，最后下钻到明细条。

## 组合 B：对比式 + FT 风格 — "公平感"

### 自动选择记录

- **输入**: "竞品分析：3月各职场GMV达成对比"
- **场景意图**: A/B 对比 → 建立基准，量化差异
- **L1 内容锁死**:
  - 模式: 26-并列比较（弧线 02）
  - 弧线: 02 对比式（图1:建立基准 → 图2:呈现差异 → 图3:量化影响 → 结论:选择建议）
  - 构图: 02 左右对比（2 图对比）→ 扩展为 3 图递进
- **L2 场景约束**:
  - 风格: 05 FT（数据新闻 Tier 1）
  - 开场: 04 对比开场（对比式天然需要对比开场）
- **L3 风格自由**:
  - 配色: 鲑鱼粉底+深蓝（FT 色系 0）
  - 字体: Newsreader + Newsreader
  - 节拍: C-05 对比揭示 + C-03 排序阶梯

### SPEC — 组合 B

**【可视化目的】**
读者以为"各家表现差不多"。这张图要让他看到达标组和未达标组之间的鸿沟——不是"差几个百分点"，是"完全不同的两个世界"。认知缺口：把"连续的达成率"重新定义为"两个阵营"。

**【想传达什么】**
达标组均值 114% vs 未达标组均值 81%，两组之间存在 33 个百分点的结构性差距，不是随机波动。

**【结论】**
职场已分化为两个阵营，需分别制定策略——达标组复制经验，未达标组诊断根因。

**【思路】**
对比式弧线强调"公平感"。第一张图用蝴蝶图让双方在同一个视觉平面内对峙——左边达标组、右边未达标组，以 100% 为中轴。第一眼看到左右不对称，第二眼读差距数字，第三张图将差距翻译为"业务影响"。

---

**【模式选择】**
- **模式**: 26-并列比较
- **理由**: 核心诉求是"达标组 vs 未达标组"的阵营对比，并列比较天然表达对立

**【标题】**
- **主标题**: 达标组 vs 未达标组——两个阵营，33 个百分点的结构性差距
- **副标题**: 2026 年 3 月 · 8 个职场 · 新客-未授信

**【视觉编码】**
- **图 1（基准全览）**: 8 个职场横向条形图，按达成率降序，100% 线居中
- **图 2（阵营对比）**: 蝴蝶图——左半达标 3 家（向右延伸），右半未达标 5 家（向左延伸），中轴 = 100%
- **图 3（影响量化）**: 两组均值对比柱 + 差距标注，33pp 用大字号
- **颜色**: 达标组用 FT 深蓝 #1A3C6D，未达标组用 FT 鲑鱼粉 #FF7F50，中轴浅灰 #E0E0E0

**【数据组织】**
- **字段**: workplace_name, achievement_rate, group(达标组/未达标组), group_mean
- **排序**: 图 1/3 降序，图 2 按组内降序
- **聚合**: 达标组均值 = (132.5+105.8+103.9)/3 = 114.1%；未达标组均值 = (93.9+85.3+80.3+76.0+68.2)/5 = 80.7%

**【标注策略】**
- **高亮**: 图 3 中 "33pp 差距" 加粗放大
- **标注**: 图 2 蝴蝶图左侧标"均值 114%"，右侧标"均值 81%"
- **基准线**: 所有图 100% 位置浅灰虚线

**【图例与辅助】**

- **脚注**: 数据来源：业务系统 | 分组标准：达成率 ≥100% 为达标组

## 组合 C：诊断式 + Excalidraw 手绘风格 — "从症状到处方"

### 自动选择记录

- **输入**: "问题诊断：3月GMV达成率下滑根因分析"
- **场景意图**: 问题诊断 → 从症状到根因到处方
- **L1 内容锁死**:
  - 模式: 04-诊断式（弧线 04）
  - 弧线: 04 诊断式（图1:症状呈现 → 图2:逐层拆解 → 图3:处方 → 结论:优先级排序）
  - 构图: 09 递进阶梯（3 图递进，从问题到方案）
- **L2 场景约束**:
  - 风格: 20 Excalidraw（脑暴/草图 Tier 1 — 诊断场景需要"草稿感"来降低防御心理）
  - 开场: 05 漏斗开场（诊断式从症状开始）
- **L3 风格自由**:
  - 配色: 纸张色+柔和彩（Excalidraw 色系 0）
  - 字体: Amatic SC + Lato
  - 节拍: C-06 异常闪烁 + C-07 标注箭头

### SPEC — 组合 C

**【可视化目的】**
读者看到数字会觉得"未达标就是能力差"。这张图要让他看到结构性的问题——5 家未达标不是"5 个独立的失败"，而是"同一个系统性问题的 5 个表现"。认知缺口：把"谁没达标"升级为"为什么都没达标"。

**【想传达什么】**
未达标 5 家的平均缺口 21%，其中 3 家（中乾/速腾/九通）缺口超过 20%，占了总缺口的 55%，指向同一个系统性瓶颈。

**【结论】**
优先诊断末位 3 家的共性问题（占 55% 的缺口），而非逐家解决。

**【思路】**
诊断式弧线走"症状→归因→处方"路径。第一张图用热力图式的网格展示 8 家的红绿灯状态，制造"一片红"的冲击感。第二张图按缺口大小拆解，发现末位 3 家占了总缺口的 55%。第三张图给出优先级排序的行动建议。手绘风格降低"审判感"，让未达标方更容易接受。

---

**【模式选择】**
- **模式**: 诊断式（叙事弧线），配合 C09-差距到目标（单图模式）
- **理由**: 需要从"表面症状"走到"根因处方"的完整诊断路径

**【标题】**
- **主标题**: 未达标 5 家不是 5 个独立问题——末位 3 家共占 55% 缺口
- **副标题**: 2026 年 3 月 · 达成率诊断 · 手绘草稿版

**【视觉编码】**
- **图 1（症状呈现）**: 8 个职场的红绿灯矩阵——绿色（达标）/黄色（90-99%）/红色（<90%），一眼看出"红多绿少"
- **图 2（逐层拆解）**: 横向瀑布图展示缺口构成——从 100% 基准线出发，每家偏离量用不同方向/长度的柱表示
- **图 3（处方）**: 优先级排序卡片——高（末位 3 家共诊）、中（德辰/广达个别辅导）、低（达标组经验分享）
- **颜色**: 绿色 #4CAF50（达标）、黄色 #FFC107（接近）、红色 #E53935（危险），手绘风格用蜡笔质感

**【数据组织】**
- **字段**: workplace_name, achievement_rate, gap_from_100, severity_level(高/中/低), gap_share
- **排序**: 图 2 按 gap 绝对值降序
- **聚合**: 末位 3 家 gap 总和占比 = (19.7+24.0+31.8)/138.5 = 75.5/138.5 = 54.5%

**【标注策略】**
- **高亮**: 图 2 中末位 3 家用红色加粗，其余灰化
- **标注**: 图 1 红色格子旁标"需诊断"，绿色格子旁标"可复制"
- **箭头**: 图 3 中用箭头连接"共性问题"→"统一干预方案"

**【图例与辅助】**
- **图例**: 图 1 需要红绿灯图例，放右上角
- **脚注**: 手绘草稿版 · 数据仅供参考方向性 · 来源：业务系统 2026-03
- **特殊说明**: 手绘风格——线条略有不规则，填充色有蜡笔纹理感，降低正式感

**【布局】**
- **画布**: 800 × 700（3 图纵向递进）
- **图 1**: 顶部，高度 180px，8 宫格红绿灯
- **图 2**: 中间，高度 250px，横向瀑布/偏离图
- **图 3**: 底部，高度 200px，3 张优先级卡片
- **留白**: 图间距 20px，四周 30px，手绘风格留白更大

## 差异化对比总结

| 维度 | 组合 A：金字塔式 | 组合 B：对比式 | 组合 C：诊断式 |
|------|-----------------|---------------|---------------|
| **叙事意图** | 高管汇报：先说结论 | 阵营对比：公平感 | 问题诊断：症状→处方 |
| **弧线** | 07 金字塔式（结论→论据→数据） | 02 对比式（基准→差异→影响） | 04 诊断式（症状→拆解→处方） |
| **风格学派** | 01 麦肯锡（深蓝+灰+红，Inter Tight） | 05 FT（鲑鱼粉+深蓝，Newsreader） | 20 Excalidraw（纸张色+柔和彩，Amatic SC） |
| **核心图表** | 红绿灯大数字 + 卡片 + 条形图 | 蝴蝶图（左右对称） | 红绿灯矩阵 + 瀑布偏离图 + 优先级卡片 |
| **第一眼落在** | "5/8 未达标" 红色大数字 | 蝴蝶图左右不对称 | "一片红"的热力矩阵 |
| **核心洞察** | 分化程度（64pp 极差） | 阵营差距（38pp 结构性差距） | 共性问题（末位 3 家占 54% 缺口） |
| **行动建议** | 针对末位启动根因分析 | 两组分别制定策略 | 末位 3 家共诊，非逐家解决 |
| **画布** | 800×900（3 图堆叠） | 800×700（蝴蝶图为主） | 800×700（递进阶梯） |
| **情绪** | 专业、冷静、决策导向 | 新闻感、客观、数据驱动 | 草稿感、探索性、降低防御 |

**结论**：3 种组合产出的 SPEC 在叙事意图、图表选择、视觉编码、行动建议上均有实质性差异，不仅仅是配色不同。每张图的"第一眼"和"核心洞察"都不同，验证了 viz-pattern 的 combinatorial architecture 能产出真正差异化的可视化方案。
