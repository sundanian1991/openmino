## 评估标准（Binary 清单）

```
□ 双层分离（填充层 + 描边层）
□ stroke-linecap="round" 全局应用
□ 色彩系统完整（陶土色 + 墨黑 + 灰度层次）
□ path 数量合理（简单 10-30，复杂 60-80）
□ 结构清晰（头部/身体/四肢分组）
□ 中文字符清晰可读（如果需要）
□ 视觉平衡（左右对称、重心稳定）

通过率 = 通过项 / 7
目标：迭代 3 达到 7/7
```

## 迭代流程

### 每次迭代的步骤

1. **设计** — 根据哲学确定参数
2. **实施** — 编写 SVG 代码
3. **评估** — 用 binary 清单打分
4. **记录** — 写评估报告 + 问题清单
5. **修正** — 根据问题进入下一次迭代

## 成功标准

**达成条件**：
- 模板 1（简单）在 3 次迭代内达到 90+ 分
- 模板 2（中等）在 3 次迭代内达到 90+ 分
- 模板 3（复杂）在 3 次迭代内达到 90+ 分

**如果失败**：
- 分析哲学哪里不适用
- 调整哲学或执行方法
- 重新验证
