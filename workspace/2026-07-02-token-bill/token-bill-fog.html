Token 账单
迷雾

当每百万 Token 多少钱变成「比价陷阱」——单位统一了，算法却没有统一。看似清晰的一道乘法，实则是一张谁都读不透的网。

作者 杨依婷
两年 · 1000 倍

Token 不再只是
工程师终端里的日志

它正在印入预算表、损益表和季度财报。AI 支出从"研发预算"变成了"运营成本"，而且后者还在以季度为单位陡增。

1000亿 2024 日均调用
140万亿 2026.03 日均调用
?
核心主张

度量衡的意义，本应是放之四海而皆准。但当下的 Token，看起来是一个统一计价的基础单位，实际上更像是每家公司自己定义、自己解释、外部无法验证的内部货币。

Token 的出现，第一次让模型生成能力被压缩成一个可计量、可交易、可结算的单位。它像极了电力时代的"度"——把复杂的发电、输电和配电过程，压缩成一个所有人都能理解的计量单位。

但计量单位统一之后，一个隐秘的裂缝也随之裂开：单位统一了，算法却没有统一。表面上看，账单只有两个变量：价格和用量。但这两个数字都不是常量——模型怎么定价、缓存命中有没有折扣、长思考模式会不会额外消耗 Token，甚至数据中心建在哪个省份，都在左右最终的数字。

"野蛮生长，是这个阶段的必然。"

罗忆 · 云天励飞副总裁

01
第一章定价

为什么同样是 Token
价格差距越来越大？

大模型 API 的价格战，让 Token 的单价看起来前所未有的透明。但恰恰是这种"一分钱一分货"的明码标价，将真正的成本结构藏进了黑箱。同一个数字背后，可能是技术效率的极致兑现，也可能是资本补贴的饮鸩止渴。

从事芯片行业的吴昊把这种成本的不可比，归结为三个互相纠缠的变量：设备成本、运营成本、合同周期。

A
设备成本
GPU 型号、采购量、是否搭售软件、配没配外部存储——每一项都以不同方式影响初始成本分摊。顶尖卡几乎不单独租赁，以集群出货，无法被充分利用的顶级算力最终摊进每个 Token 定价。
B
运营电价
西北电价两三毛，东部六七毛——整整差出三倍。大量"低价 Token"未必是效率提升，而是一场仍在持续的补贴战争的产物。一旦补贴退场，账单会反弹多少仍是未知数。
C
合同周期
算力正从整租变成散租。1 年和 5 年的长期锁定，与按 Token 量结算的散租模式，背后是两套完全不同的运营逻辑和成本结构。

太初元碁首席产品官洪源把视线投向了"未来的账单"：当行业真正进入 Token 经济时代，最重要的衡量指标可能会从单纯的算力速度，转向能源转化效率——每瓦电能能够产生多少 Token。

02
第二章比较

「每百万 Token 多少钱」
不能直接比较

Token 的计费规则本身就没有统一标准。不同模型对同一段文本的切分方式不同——GPT 系列用 BPE，Claude 用 BPE 变体，Llama 和 DeepSeek 用 SentencePiece，各家的词表大小、中文覆盖度、子词策略都不一样。

当计费口径本身就不统一，"每百万 Token X 元"的比价，比的可能就不是同一把尺子。

模型输出价格（每百万 Token）备注
OpenAI GPT-5.5 Pro上百美元高性能版本
Anthropic Claude Opus 4.8二三十美元标准模式
DeepSeek 最新版6 元人民币国内厂商普遍个位数

云天励飞副总裁罗忆坦言，当前 Token 定价体系尚处混沌期。现在的定价好比卖牛肉——不同部位、不同品质对应不同价格，但最终仍以重量作为基本的计量单位。

所谓标准化，标准的只是这个名字，而不是它的质量与价值。

03
第三章消耗

Token 越用越贵
问题不在模型？

算力中心在建，模型在跑，Token 在烧。但烧掉的那些 Token，到底换回了什么——这个问题，正在成为每家企业 IT 负责人的日常拷问。

以搭建一个股票分析 workflow 为例：前期调试消耗上千万 Token，流程跑通后每次运行再耗百万 Token；一旦从单只扩展到批量分析，日总消耗立即骤增百倍。Agent 的调用逻辑不是线性的——它会反复迭代、自我对话、调用上下文。

"好的设计能把 Token 变成智慧，
差的设计只能把 Token 变成成本。"

04
第四章标尺

当「烧多少」不能代表「值多少」
Token 需要新标尺

整个行业的能力边界还在不断延伸，远未触达上限——技术会催生新的服务形态，服务落到场景里跑通了，质量体系才会跟着长出来。共识在聚拢，但标准还没来。

云天励飞联合三十多家产业链企业签署"1001 计划"，先把各方拉到一张桌子上，以行业共识为起点推动标准制定。

A
按度计量
九章云极提出算力端统一标尺，让不同芯片、不同集群产出的 Token 可在同一尺度下可比。
B
场景 Token
迈富时区分"大模型 Token"与"场景 Token"，按任务效果计价，将价值度量从"消耗端"迁移到"结果端"。
C
能耗标尺
太初元碁发布高密液冷集群，在性能、功耗与建设成本间寻求最优平衡，为生产成本划出参照。

Token 未必会成为 AI 时代
最终的"度电"

但围绕它所构建的新价值尺度，终将左右下一代 AI 基础设施的话语权。

从共识到标准，仍有距离。但标尺已在打磨，刻度隐约浮现。
