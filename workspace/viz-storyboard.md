## Phase 0：全局风格锁定

- **整体叙事弧线**：04 诊断式 — 症状（KPI）→ 构成（在职结构）→ 趋势（月度变化）→ 诊断（流失拆解）→ 明细
- **风格学派**：restrained-warm（暖色克制）
- **全局色阶**：
  - 主 ramp：Warm（`#d4a574` 主色 / `#c0392b` 警示 / `#e8d5c0` 浅暖）
  - 辅 ramp：Stone（`#888888` 基准 / `#e0e0e0` 网格）
  - 语义色：`#2e7d32`（正面/留存）/ `#c0392b`（负面/流失）/ `#d4a574`（中性/常态）
  - **色阶锁定**：后续所有图不得引入新 ramp，只从上述色阶中选取
- **字体家族**：-apple-system, system-ui, sans-serif
- **字号层级**：标题 18/700、图表标题 15/600、副标题 13/400、轴标签 11/400、标注 11/600、脚注 10/400
- **圆角/间距**：卡片圆角 12px、卡片间距 24px、卡片内边距 24px
- **什么必须统一**：色系、字号、圆角、间距、高亮规则（≤10%元素用警示色）
- **什么必须变化**：每张图的标题、数据、高亮目标
- **什么可以复用**：基准线样式、脚注格式、卡片容器

## 单图场景设计

### Scene 1: KPI 总览卡片组

- **场景论文**：5个数字定全局基调——117人中63人在岗、54人流失、46.2%流失率、402万人均GMV
- **在叙事中的角色**：开场定调
- **叙事节拍**：
  - 第一眼看到"46.2% 流失率"（红色） → 触发警觉
  - 第二眼看到"63人在职 vs 54人流失" → 理解规模
  - 最终理解"人均402万GMV" → 产出价值锚定
- **签名视觉元素**：流失率卡片用 #c0392b 红色高亮，其余4张用 #d4a574 暖棕
- **克制声明**：不加趋势箭头（KPI卡片不需要）、不加环比（数据不支持）、不用图标装饰

### Scene 2: 在职人员司龄结构（环形图）

- **场景论文**：65%的在职人员入职不超过60天——团队是新人堆出来的
- **在叙事中的角色**：全景展示（从KPI转入结构分析）
- **叙事节拍**：
  - 第一眼看到0-30天和31-60天两个大段 → 新人为主
  - 第二眼看到91-180天和181-365天各仅6人 → 老人稀缺
  - 最终理解"新人占65%" → 稳定性隐患
- **签名视觉元素**：环形图中心显示"63人在职"总数，0-30天用主色 #d4a574，其余递减
- **克制声明**：不加百分比标签在每段上（中心总数+段数足够）、不用3D效果、不加图例（直接标注）

### Scene 3: 在职人员按入职月人均GMV（柱状+折线双轴）

- **场景论文**：3月入职的23人在职，人均568万GMV——这批人是当前产出主力
- **在叙事中的角色**：证据支撑（产出维度补充司龄结构）
- **叙事节拍**：
  - 第一眼看到3月的高柱+高线 → 产出集中
  - 第二眼看到5月的矮柱+高线 → 新人产出低（正常）
  - 最终理解"产出靠3月这批人撑着" → 留存这批人是关键
- **签名视觉元素**：柱状颜色深=高产出（>700万用 #c0392b，400-700用 #d4a574，<400用 #e8d5c0），折线用 #1a1a1a
- **克制声明**：不加网格线、Y轴从0开始、折线不加面积填充

### Scene 4: 按月入职人数（柱状图）

- **场景论文**：3月集中入职46人，占总入职量的39%——招聘脉冲式爆发
- **在叙事中的角色**：趋势（解释为什么3月是关键月份）
- **叙事节拍**：
  - 第一眼看到3月的高柱（46人） → 招聘爆发
  - 第二眼看到其他月份平稳（2-6人） → 非常态
  - 最终理解"3月一次性涌入大量人" → 后续流失集中在这批人
- **签名视觉元素**：3月柱子用 #c0392b 高亮，其余用 #d4a574，标注"46人"
- **克制声明**：不加趋势线、不加图例、标签只在柱顶显示人数

### Scene 5: 月度在职人数趋势（面积折线图）

- **场景论文**：4月峰值76人后回落至65人——增长动能衰减
- **在叙事中的角色**：趋势（从入职转入留存视角）
- **叙事节拍**：
  - 第一眼看到4月峰值76人 → 最高点
  - 第二眼看到5月下降到65人 → 拐点
  - 最终理解"新入职补不上流失" → 需要干预
- **签名视觉元素**：面积填充从 #d4a574 到透明的渐变，峰值点标注"76人"，5月点标注"65人"
- **克制声明**：不加预测虚线、不加网格线、Y轴从0开始

### Scene 6: 各司龄段流失率（水平柱状图）

- **场景论文**：61-90天流失率65.5%，是所有阶段中最高的——入职第二个月是生死关
- **在叙事中的角色**：诊断核心（这是全报告最关键的发现）
- **叙事节拍**：
  - 第一眼看到61-90天的长柱（65.5%） → 最严重
  - 第二眼看到0-30天反而最低（36.1%） → 不是"留不住"，是"留到60天后留不住"
  - 最终理解"干预窗口在60天左右" → 有时间窗口
- **签名视觉元素**：61-90天柱子用 #c0392b 高亮，其余用 #d4a574，标注"65.5%"加粗
- **克制声明**：不加人数标注（流失率本身就够说明问题）、不加网格线

### Scene 7: 流失人员在职天数分布（柱状图）

- **场景论文**：19人流失在61-90天——与流失率图互为印证
- **在叙事中的角色**：证据支撑（从"率"到"人"，让数字有实感）
- **叙事节拍**：
  - 第一眼看到61-90天的高柱（19人） → 绝对数也最多
  - 第二眼看到0-30天也有13人 → 早期流失也不少
  - 最终理解"两波流失：早期放弃 + 中期倦怠" → 干预策略不同
- **签名视觉元素**：61-90天柱子用 #c0392b 高亮，其余用 #d4a574
- **克制声明**：不加累计线、不加图例、标签只在柱顶

### Scene 8: 流失人员 GMV vs 在职天数（散点图）

- **场景论文**：高GMV人员也在流失（吴才亮1430万、郑民811万）——不是只有低产出的人走
- **在叙事中的角色**：诊断深化（打破"低产出=流失"的假设）
- **叙事节拍**：
  - 第一眼看到右上角的高GMV散点 → 高产出也在流失
  - 第二眼看到左下角的零GMV散点 → 也有完全没产出就走的
  - 最终理解"流失不是纯淘汰，也有被动损失" → 需要区分对待
- **签名视觉元素**：GMV>500万的点标注姓名，其余不标注；所有点用 #c0392b 半透明
- **克制声明**：不加趋势线（没有因果关系）、不加回归分析、最多标注8个姓名

### Scene 9: 流失人员明细表

- **场景论文**：54人逐一列出——让每个流失可追溯、可复盘
- **在叙事中的角色**：收束（从宏观分析落到个体）
- **叙事节拍**：
  - 第一眼看到按最后外呼时间倒序排列 → 最近流失的排最前
  - 第二眼看到离线天数、GMV、等级 → 判断是否值得挽留
  - 最终理解"每个人都有数据支撑" → 可以逐一跟进
- **签名视觉元素**：等级列用标签色（A绿/B蓝/C橙/非参评灰），表格支持滚动
- **克制声明**：不加排序功能（已按最后外呼倒序）、不加筛选（保持简洁）

## 为什么不是别的视觉方案

- **为什么选环形图不选饼图/树状图？** 饼图太"重"，环形图中心可放总数，信息密度更高。树状图不适合≤6类的简单构成。
- **为什么选面积折线不选纯折线/柱状？** 面积填充强调"量感"，让76→65的下降更有视觉冲击。纯折线太平淡，柱状图在11个时间点上太密。
- **为什么选水平柱状不选垂直柱状？** 司龄段名称长（"181-365天"），水平布局标签可读性好10倍。
- **为什么暖棕色系不用蓝色系？** 年老师偏好亮色暖色系，且数据是"人力"而非"财务"，暖色更有人情味。
