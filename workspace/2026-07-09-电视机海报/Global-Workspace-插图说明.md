# Global Workspace Theory 文章插图说明

> 基于 Anthropic 文章《A global workspace in language models》生成的贵藏风格插图

---

## 文章核心概念

**J-space（Jacobian 空间）**：Claude 内部的一个小型神经活动模式集合，类似人类意识中的"全局工作空间"。它支持以下功能：

1. **可报告性**：Claude 能说出 J-space 里的内容
2. **可调控性**：Claude 能按要求在 J-space 里保持特定概念
3. **用于推理**：多步推理的中间步骤出现在 J-space
4. **灵活性**：同一个 J-space 表示可以被多个任务使用
5. **非自动处理**：大部分处理不通过 J-space，删除 J-space 后 Claude 仍能流利说话但失去高阶推理
