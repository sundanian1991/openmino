# 供应商知识库建设 - 需求文档

## 目标

将 `docs/知识资源/供应商人力&站点盘点2026.xlsx` 转化为结构化的供应商知识库系统，支持快速检索、关联分析和智能识别。

## 排除项

- 不修改原始 Excel 文件
- 不删除历史供应商数据（仅做标记）
- 不涉及敏感财务数据（结算单价等）

## 交付物

1. **供应商画像文档**（17个）：`docs/supplier-kb/profiles/[供应商简称].md`
   - 基础信息（全称、简称、别名）
   - 站点分布（省/市/区/详细地址）
   - 人力规模与趋势
   - 关联人物（从会议纪要等提取）

2. **供应商索引**：`docs/supplier-kb/index/supplier-index.md`
   - 简称 → 全称映射表
   - 别名/关键词检索表
   - 按省份/职场分类索引

3. **实时画像更新机制**：`insights/suppliers/`
   - 与 `insights/年老师-实时画像.md` 打通
   - 支持会议中自动识别供应商

## 完成标准

- [ ] 17 家供应商全部生成独立 MD 画像
- [ ] 索引支持简称、别名、省份等多维度检索
- [ ] 画像模板标准化，便于后续维护
- [ ] 与会议纪要分析器打通（自动识别供应商）

## 数据范围

**来源文件**：`docs/知识资源/供应商人力&站点盘点2026.xlsx`

**已提取数据**：
- 17 家供应商
- 76 个站点
- 覆盖省份：河南、河北、陕西、安徽、甘肃、湖北、黑龙江、山东、四川、江西、江苏、北京、广东

**关键字段**：
- 公司全称、项目（职场）
- 站点所在省/市/区/详细地址
- 11-12月人力数据
