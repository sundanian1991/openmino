# 供应商人力看板 — BI 需求文档

> 版本：V1.0 | 日期：2026-03-20
> 数据源：供应商人力盘点表

---

## 一、数据源说明

### 1.1 当前数据结构

| 字段 | 说明 | 示例 |
|------|------|------|
| 项目 | 产品线+模式 | 企业主贷、金条、金条人机、固收、菁卡拉新 |
| 公司全称 | 供应商名称 | 中乾（深圳）投资控股有限公司 |
| 职场 | 站点名称 | 信安大厦职场、通达花园职场 |
| 省/市/区/地址 | 四级地理信息 | 河南省 / 洛阳市 / 西工区 / ... |
| 月度人力列 | 每月在职人力数 | 1月站点在职人力 = 30 |

**当前覆盖**：20+ 产品线、30+ 供应商、70+ 职场站点、12 个月度数据点

### 1.2 数据改进建议

**目标粒度**：按日更新

**建议增加的派生字段**：

| 字段 | 计算方式 | 用途 |
|------|----------|------|
| 业态 | 从项目名归类 | 金条/企金/信用卡/财富 四大业态 |
| 作业模式 | 从项目名提取 | 纯人 / 纯机 / 人机混合 |
| 环比变化量 | 当日人力 - 前日人力 | 判断增减 |
| 环比变化率 | (当日-前日)/前日 | 判断变化幅度 |
| 月累计均值 | 月内日人力均值 | 月度趋势对比 |

## 二、看板核心视角

### 视角 1：按职场看趋势

**目标**：一眼看出每个职场的人力变化，识别扩张/收缩

| 可视化 | 说明 |
|--------|------|
| **折线图** | X轴=时间，Y轴=人力，每条线=一个职场 |
| **热力图** | 行=职场，列=月份，颜色深浅=人力多少 |
| **排名表** | 按当前人力排序的职场 Top 20 |
| **变化卡片** | 展示环比变化最大（增长/下降）的 Top 5 职场 |

**交互**：
- 支持按省/市筛选
- 支持按产品线筛选
- 点击单个职场可下钻到供应商+项目明细

### 视角 2：按供应商看趋势

**目标**：监控每家供应商的人力投入变化，评估稳定性

| 可视化 | 说明 |
|--------|------|
| **面积图/堆叠柱状图** | X轴=时间，Y轴=总人力，按供应商堆叠，看占比变化 |
| **供应商矩阵** | X轴=当前人力规模，Y轴=环比增长率，气泡大小=覆盖职场数 |
| **趋势折线** | 每条线=一个供应商的人力趋势 |
| **供应商-产品线桑基图** | 看每家供应商的人力如何分布在各产品线 |

**交互**：
- 支持按产品线筛选
- 点击供应商可下钻到职场级明细

### 视角 3：按产品线看趋势

**目标**：看各产品线的人力总量变化

| 可视化 | 说明 |
|--------|------|
| **分组柱状图** | 每组=一个时间点，每根柱子=一个产品线 |
| **百分比堆叠图** | 看各产品线人力占比变化趋势 |
| **环形图** | 当前各产品线人力占比快照 |

**交互**：
- 支持按业态（金条/企金/信用卡/财富）
- 支持按作业模式（人/机/人机）筛选

## 三、对比分析模块

### 3.1 供应商横向对比

| 分析项 | 说明 |
|--------|------|
| 人力规模排名 | 当前各供应商总人力排名 |
| 增长率排名 | 环比增长率排序 |
| 覆盖度 | 每家供应商覆盖多少产品线、多少职场 |
| 趋势对比 | 选择 2-5 家供应商，叠加对比趋势线 |

### 3.2 同产品线下供应商对比

| 分析项 | 说明 |
|------|------|
| 产品线内份额 | 选择某产品线，看各供应商占比 |
| 产品线内排名 | 各供应商在该产品线的人力排名 |
| 替换跟踪 | 某供应商人力下降时，谁在接替 |

### 3.3 同供应商跨产品线

| 分析项 | 说明 |
|------|------|
| 业务集中度 | 某供应商的人力集中在哪些产品线 |
| 扩项跟踪 | 供应商是否在新增产品线 |
| 鸡蛋篮子 | 一家供应商超过3个产品线的风险 |

## 四、预警与异常检测

### 4.1 供应商集中度预警

| 指标 | 规则 | 展示 |
|------|------|------|
| 单供应商占比过高 | 某供应商在某产品线人力占比 > 40% | 红色标记 |
| 单职场依赖度过高 | 某产品线 > 50% 人力集中在一个职场 | 黄色标记 |
| 区域集中度 | 某产品线 > 60% 人力集中在同一省份 | 黄色标记 |

### 4.2 人力波动预警

| 指标 | 规则 | 展示 |
|------|------|------|
| 单日骤降 | 职场/供应商人力日环比下降 > 20% | 红色告警 |
| 持续收缩 | 连续 7 天人力下降 | 橙色告警 |
| 异常增长 | 单日增长 > 50%（可能数据异常） | 黄色提示 |

## 六、技术要求

### 6.1 数据接入

| 项目 | 要求 |
|------|------|
| 数据格式 | Excel (.xlsx) 或 CSV |
| 更新频率 | **按日更新** |
| 历史数据 |  2026.01 至今所有历史数据 |
| 增量更新 | 支持追加新日期数据，不覆盖历史 |

### 6.2 交互要求

| 功能 | 说明 |
|------|------|
| 时间范围选择 | 可选择查看任意时间段 |
| 维度联动筛选 | 选产品线→自动联动供应商/职场 |
| 下钻 | 全局 → 产品线 → 供应商 → 职场 |
| 悬浮提示 | 鼠标悬浮显示具体数值和环比变化 |
| 导出 | 支持导出当前视图为图片或 Excel |

### 6.3 性能

| 指标 | 要求 |
|------|------|
| 首页加载 | < 3 秒 |
| 筛选响应 | < 1 秒 |
| 数据量 | 支持 70+ 职场 × 365 天 × 多维分析 |

## 七、数据字典（原始字段映射）

| 原始字段 | BI 字段名 | 类型 | 说明 |

## 八、优先级建议

建议分两期实现：

| 期数 | 内容 | 优先级 |
|------|------|--------|
| **P0 — 一期** | 全局概览 + 按职场趋势 + 按供应商趋势 + 数据接入 | 最高 |
| **P1 — 二期** | 产品线视角 + 对比分析 + 预警中心 | 高 |
| **P2 — 后续** | 下钻交互 + 导出 + 职场生命周期 | 中 |
