viz-design Skill

可视化设计器完整工作流

三循环工作流：初稿 → 减法迭代 → 验收交付 — 从需求入口到渲染出图

CYCLE 1

初稿 — 先有

  • DNA 意图匹配（9509 条模式库）
  • 命中施工图模板 → 直接套用
  • 未命中 → fallback 链 → 自主推理
  • 版式策略 + 写中间文件
  • 出第一版图表

CYCLE 2

减法迭代 — 再做精

  • 7 问逐项追每个元素价值
  • 标题→结论性 / 图例→可省略
  • 网格/边框/色块→能去则去
  • 标注写原因+幅度，不写纯数字
  • 最多 3 轮，用户可随时终止

CYCLE 3

验收 — 最后检查

  • 8 项检查：截断/泄露/乱码/重叠
  • 色盲友好（禁用红绿对比）
  • 高亮 ≤10% / 标题结论性
  • 4+ 项不通过 → 回退循环 1
  • 通过 → 渲染路由 → 交付
