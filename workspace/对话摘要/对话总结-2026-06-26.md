# Errors & Corrections
1. **shell `IFS=$"\t"` bug**：误用本地化字符串引号，应为 `$'\t'`。导致字段未拆分，651次请求未执行。→ 改用 while read 循环 + 后台并发。
2. **xargs 命令行长度超限**：书名含超长字符。→ 去掉 title 参数，只传 bookId。
3. **时间戳单位误判**：划线 createTime 是秒级（10位），初版误除以1000导致全部显示1970。→ 秒级直接 strftime。
4. **jq 变量作用域**：同一 shell 里复用 `--arg bid` 会冲突，应每次重新定义。→ 用 for 循环 + 唯一变量名。
5. **macOS bc 对空输入报错**：主题统计时部分关键词匹配为空。→ 改用 jq `add // 0`。
