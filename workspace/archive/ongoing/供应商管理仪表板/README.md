# 供应商管理仪表板 - 演示原型

## 这是什么

供应商全生命周期管理平台的可点击 HTML 原型，用于演示销售、利益相关者评审。

## 功能特性

- **三屏结构**：概览仪表板、供应商列表、（预留合规/绩效页）
- **三主题切换**：专业蓝、深邃夜、翡翠绿
- **三密度布局**：紧凑、舒适、宽松
- **三角色视角**：运营（normal）、空状态（empty）、高管（power）
- **实时数据渲染**：供应商列表、统计数据、通知提醒
- **交互反馈**：Toast 提示、按钮响应
- **分享链接**：URL 保存当前主题/布局/角色状态

## 技术栈

- HTML5 + Tailwind CSS (CDN)
- Vanilla JS (无框架依赖)
- 本地状态管理 (URL + localStorage)

## 如何运行

```bash
cd workspace/00-原型-供应商管理仪表板-20250519
python3 serve.py
# 打开 http://localhost:8000
```

或直接用浏览器打开 `index.html`。

## 数据来源

模拟数据基于你的实际工作场景：
- 供应商名称参考真实合作方（毅航、毛毛虫、伽玛等）
- 合规率、支出、人数等数据虚构但符合业务逻辑

## 使用说明

1. **切换主题**：底部控制栏点击主题按钮
2. **切换布局**：紧凑/舒适/宽松三档
3. **切换角色**：
   - 运营：6 家供应商，正常数据
   - 空：无数据状态演示
   - 高管：8 家供应商，含预警和停用
4. **分享状态**：点击 🔗 Share 复制带状态 URL
5. **反馈标注**：点击 💬 Feedback 开启标注模式

## 浏览器支持

Chrome、Safari、Firefox 最新版本。

## 限制说明

- 数据为静态模拟，无后端 API
- 部分功能仅显示 Toast 提示（演示用）
