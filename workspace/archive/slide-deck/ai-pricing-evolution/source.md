# AI产品定价模式演变：从Coding Plan到Token Plan

## 核心结论
AI 大模型定价正从"卖工具订阅"演进为"卖算力资源包"，这个过程精确重演了电信行业从话费套餐到流量套餐的 20 年历史。Coding Plan 是过渡产品，Token Plan 是终态。

## 第一幕：时间线

### 2024 Q4 - 2025 Q2：DeepSeek 带头降价
百万 Token 从几十元降到几毛钱，价格战启动，抢用户规模。

### 2025 Q4：智谱 GLM 率先推出 Coding Plan
固定月费模式出现，从按量付费转向订阅制。

### 2026-01-28：Kimi 切换 Token 计量
从 Prompt 计量切换为 Token 计量，计量单位统一化趋势。

### 2026-02：OpenClaw 爆火
GitHub 355K stars，Agent 自动循环调用，Token 消耗 6-10 倍增长。

### 2026-03：MiniMax 更名为 Token Plan
从 Coding Plan 升级为 Token Plan，从编程工具到全模态 AI 能力。

### 2026-04：DeepSeek V4 开源 / 小米 MiMo 入局
DeepSeek V4 成本暴降 73%，小米 MiMo V2-Pro $1/$3 定价入局。

## 第三幕：计量体系

### 五种计量方式并存
1. API 请求次数 — 阿里百炼、火山方舟、腾讯云。透明度低
2. Prompt 次数 — 智谱 GLM、MiniMax 早期。中等透明
3. Token 计量 — Kimi、MiniMax Token Plan。高透明度
4. Credit 计量 — Cursor、小米 MiMo、阿里百炼。中等透明
5. 时间窗口配额 — 智谱 GLM（5h 窗口）。中等透明

### OpenClaw 效应
- GitHub 355K stars
- Token 消耗增长 6-10 倍
- Coding Plan 消耗增长 10 倍+
- 一个 Agent 任务 = 几十次模型调用

## 第四幕：核心洞察

### 洞察 1：电信套餐的精确重演
2009 话费套餐 → 2013 流量套餐 → 2017 "不限量" → 2020 按 GB 精打细算

AI 对应：
2024 API 按量计费 → 2025 Coding Plan → 2026 分层套餐 → 2026+ Token 精细计量

### 洞察 2：补贴-收紧-分层三段式
1. 补贴期：抢用户规模，建立使用习惯。2024-2025 百万 Token 几毛钱
2. 收紧期：测试价格弹性，筛选付费用户。2025 H2 限购、限时抢购
3. 分层期：利润最大化。2026 MiniMax 极速版 ¥98-899

### 洞察 3：从入口到生态
Coding 是获客入口，Token 是生态载体。MiniMax Token Plan 打包了图片、语音、音乐、视频。

## 未来预判

### Token 成本将持续下降
国产芯片放量（华为 950 年内 75 万片）+ DeepSeek V4 开源 MoE 效率革命

### 国产替代加速
英伟达市占率 66% → 54% → 2026 可能低于 50%。华为昇腾 + 寒武纪 + 阿里平头哥形成国产算力三角

### 开源模型重塑定价权
DeepSeek 模式：开源 + 极致低价 + 不卖订阅。厂商定价权来自"服务便利性"而非"模型独占性"

### Agent 时代不可逆
推理需求 2026 年已达训练的 4-5 倍。Coding Plan "不限量"不可持续，谁先转 Token 精细计量谁主动

## 结语
Coding Plan 是过渡产品，Token Plan 是终态。

能从中学到的不是具体的定价数字，而是三件事：计量单位即定价权、补贴-收紧-分层的三段式节奏、以及"入口产品"到"生态产品"的升级路径。
