deep-dive · 2 / 6

五种计量单位并存，
直接比数字没有意义。

API 请求低透明

阿里/火山/腾讯 — 1 次提问 = 5-30 次 API 调用，数字大但不透明

Prompt 次中透明

智谱/MiniMax 早期 — 1 Prompt ≈ 15-20 次调用 ≈ 1200-1600 API 请求

Token高透明

Kimi/MiniMax — 直接按输入输出 Token 计费，仅统计未缓存

Credit中透明

Cursor/小米/百炼 — 统一 Credits，不同模型不同倍率

时间窗口限节奏

智谱 5h / Claude Code 5h+7d — 限制节奏而非总量

演变方向：API 次数 → Prompt → Token/Credit → 统一 Credits。越来越透明、越来越可预测。

SECTION · 03

洞察层

能提炼什么？计量即权力、三段式节奏、入口到生态、Agent 成本革命。

insight · 1 / 4

谁定义计量单位，
谁就掌握定价权。

"计量单位的选择，决定了用户感知和你的利润空间。"

模糊计量 → 数字大、用户难换算、厂商掌握定价空间，适合早期获客。

透明计量 → 用户能算清成本、建立长期信任，适合留存和生态。

可复用模式：推出新产品时，计量单位的选择决定了用户感知和利润空间。MiniMax 从 Prompt 转向 Token、Kimi 也做了同样的切换，都在争夺定义权。

SECTION · 04

未来预判

终态是什么？芯片替代、开源冲击、Agent 成本不可逆。

// forecast

四条确定性趋势。

01 · Token 成本持续下降

国产芯片放量 + DeepSeek V4 MoE 效率革命 + 推理芯片竞争。利润空间来自"服务溢价"而非"算力差价"。

02 · 国产芯片替代加速

华为昇腾 950 年内 75 万片，自研 HBM 绕开限制。算力不再被英伟达卡脖子，定价策略可更激进。

03 · 开源重塑定价权

DeepSeek V3.2 月调用 5.35 万亿 token 全球第三。开源的免费部署能力持续压制定价上限，定价权来自服务便利性。

04 · Agent 成本不可逆

推理需求 2026 年已达训练 4-5 倍，IDC 预测 2028 年推理占 73%。所有厂商最终转向 Token 精细计量。

// takeaways

三件事带回去。

1 · 计量单位即定价权

推出新产品时，计量单位的选择决定了用户感知和利润空间。MiniMax 从 Prompt 转向 Token、Kimi 也做了同样的切换。

2 · 补贴 → 收紧 → 分层

不要一开始就想好定价。先补贴跑量，再收紧测试弹性，最后精细分层。可复用的互联网产品定价节奏。

3 · 从入口到生态

Coding Plan 是获客工具，Token Plan 是生态载体。从编程工具升级为 AI 多媒体工作室，月费跨度扩大 22 倍。
