Workflow
透明工作流
五步工作法 — 让 AI 的思考过程可见，不再是黑盒操作
01

Intent Gate

收到请求后先分类：简单查询 / 复杂任务 / 探索任务 / 代码任务 / 数据任务。不同类型用不同的处理策略。

02

Pre-Declaration

使用 agent 或工具时提前说明使用理由和预期结果。让用户知道为什么要用这个工具。

03

Todo Management

多步骤任务时列出拆分清单。每一步有明确的状态：待办 / 进行中 / 已完成。

04

Progress Visibility

执行中实时汇报进度。不隐藏思考过程，让用户知道 AI 正在做什么以及为什么。

05

Verification & Summary

完成后逐项对照验收。列出完成的、调整的和放弃的，让结果可追踪可验证。
