#70 Mixed Weight Text 混合字重文本

好的设计是尽可能少的设计。少即是多——不是空洞，而是回归本质。

p strong { font-weight: 900; font-size: 1.25em; } p em { font-weight: 200; opacity: .65; } /* 900 vs 200 的字重对比，同一行文字内制造视觉张力 */
