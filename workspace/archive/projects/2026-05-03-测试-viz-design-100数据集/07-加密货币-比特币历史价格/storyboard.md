# storyboard.md — 比特币历史价格视觉叙事设计

## Phase 0 全局风格锁定（单图，简化）

- **风格学派**：数据新闻派（Data Journalism）— 用数据讲故事的克制风格
- **色系**：Warm 主色 + Stone 辅色
- **字体**：系统默认无衬线（widget/ECharts 标准）

## 叙事弧线

**弧线类型**：过山车式（Rise-Fall-Rise-Fall）

```
起点：6.3万盘整期（平静）
  ↓
暴跌：8月闪崩至4.9万（冲击）← 拐点1
  ↓
反弹：9-12月V型反转至10万+（狂热）
  ↓
冲顶：12月突破10万、1月冲至10.75万（高潮）← 拐点2
  ↓
回调：2-4月跌至8.3万（冷却）← 拐点3
  ↓
企稳：5月回升至9.4万（悬念）
```

## 视觉节拍

### 节拍1：背景基调（5-7月）
- 视觉处理：全线 Stone 300 灰色，线宽1.5px
- 含义：盘整期，不值得关注

### 节拍2：闪崩（8月5日）
- 视觉处理：Coral 色标注框，箭头指向暴跌点
- 标注文字："8月闪崩 -23%，单日交易量暴增64%"
- 含义：一年中最剧烈的单日事件

### 节拍3：狂飙（9月-12月）
- 视觉处理：Warm 500 (#c26d3a)，线宽3px
- 含义：核心叙事段——翻倍之旅

### 节拍4：冲顶（12月-1月）
- 视觉处理：Warm 800 (#6b3410) 高亮区域
- 标注文字："首破10万 → 冲顶10.75万"
- 含义：这一轮的高潮

### 节拍5：回调（2月-4月）
- 视觉处理：Stone 500 灰色虚线
- 标注文字："3个月回吐43%涨幅"
- 含义：牛市的代价

### 节拍6：企稳（5月）
- 视觉处理：Warm 300 淡色
- 含义：市场在寻找方向

## 阅读路径

1. **第一眼**：标题"4个月翻倍，3个月腰斩"→ 建立预期
2. **第二眼**：折线的整体形状 → 感受波动幅度
3. **第三眼**：3个标注 → 理解每个拐点的含义
4. **第四眼**：底部交易量 → 验证拐点可信度

## 克制声明

- 不做：3D效果、渐变填充、动画
- 不做：显示全部30个数据点标签
- 不做：预测线（数据截止到2025-05）
- 只做：1条价格线 + 3个拐点标注 + 交易量辅助

## 布局方案

**单图双区布局（上下分图，共享X轴）**：
- 上区（70%高度）：价格折线图
- 下区（30%高度）：交易量柱状图
- X轴共享时间刻度
- 总画布：800 x 550px

```
┌─────────────────────────────────────────┐
│  标题（结论性）                          │
│  副标题（时间范围+对象）                  │
├─────────────────────────────────────────┤
│                                         │
│  价格折线（主区 70%）                    │
│  拐点标注 x3                             │
│                                         │
├─────────────────────────────────────────┤
│  交易量柱状（辅区 30%）                  │
├─────────────────────────────────────────┤
│  脚注：数据来源                          │
└─────────────────────────────────────────┘
```

**13-VISUALIZATION 合规确认**：
- 禁止双Y轴 → 用上下分图替代
