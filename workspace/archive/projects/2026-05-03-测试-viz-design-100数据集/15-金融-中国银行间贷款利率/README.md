# 中国银行间贷款利率

- **来源**: World Bank (https://data.worldbank.org/indicator/FR.INR.LEND?locations=CN)
- **数据范围**: 2015-2024
- **说明**: 银行间市场关键利率(%)

## 字段说明
| 字段 | 说明 |
|------|------|
| year | 年份 |
| lending_rate_pct | 1年期贷款基准利率(%) |
| lpr_1yr_pct | 贷款市场报价利率LPR(%) |
| shibor_overnight_pct | 隔夜Shibor(%) |
| mlf_rate_pct | 中期借贷便利MLF利率(%) |
