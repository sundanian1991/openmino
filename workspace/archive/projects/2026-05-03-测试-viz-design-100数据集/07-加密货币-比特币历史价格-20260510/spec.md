# Spec — 比特币价格演进图

## 叙事意图
【可视化目的】展示比特币 2024-05 至 2025-05 价格波动的完整路径
【信息类型】趋势 — 时间序列 + 关键拐点
【想传达什么】"10 万大关的突破与失守，波动超 50%"
【结论】比特币不是线性增长，是暴涨崩盘新高回调的过山车
【叙事弧线】#03 演进式
【思路】折线图突出路径波动，三标注（峰值/谷值/终点）锚定关键数字

## 视觉执行
【模式选择】折线图 — 时间序列标准图表
【标题】主标题：比特币突破 10 万后失守，年末收于 94.2K
        副标题：2024年5月 — 2025年5月 月度收盘价
【视觉编码】X=时间 / Y=价格(USD) / 颜色=阶段标识（clay 峰值/olive 回升/clay-d 回调）
【数据组织】30 个数据点，按月，price_usd 字段
【标注策略】高亮 3 点 ≤10%：107.5K 峰值 / 49K 谷值 / 94.2K 终点
【布局】880×420 / Birchline 色系
