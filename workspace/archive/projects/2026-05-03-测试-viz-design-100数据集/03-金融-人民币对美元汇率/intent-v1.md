# Intent — 人民币对美元汇率 2023-2025

## 意图决策

**可视化目的**：展示人民币贬值但贸易顺差未扩大的矛盾——贬值未能带来贸易改善。

**核心张力**：USD/CNY 从 6.75 → 7.25（贬值 ~7%），但贸易顺差没有系统性增长——经典经济学理论预期贬值→出口竞争力增强→顺差扩大，数据不支持。

**标题（结论性）**：人民币贬值 7%，但贸易顺差并未扩大

**图表选择**：双轴——汇率（主轴折线）+ 贸易顺差（次轴柱状）。汇率线持续向右下走（贬值），顺差柱上下波动但无明显趋势——张力一目了然。

**减法迭代预判**：
- 有效汇率指数（NEER）→ 和 USD/CNY 高度同向，信息冗余 → 删
- 贸易顺差用柱状图 → 和折线叠加容易乱 → 考虑放在底部小图
- 网格线 → 淡化
- 标注 → 只标关键：6.75 起点、7.32 最高贬值点、104.8B 顺差峰值
