# Phase 2 — 视觉叙事设计

## 叙事弧线

**单图场景，采用"起点 → 低谷 → 新高"的三幕叙事弧线。**

## 一句话灵魂

"2020 年疫情打掉了 15% 的餐饮收入，但 2023 年不仅恢复还创了新高"——这张图要让人看到韧性和反弹速度。

## 签名视觉元素

- 主轴线（柱状）：餐饮收入（万亿），浅灰色柱
- 副轴线（折线）：同比增长率（%），Warm 橙色线
- 2020 年柱子用深色标注（最低点）
- 2023 年柱子用 Warm 橙色标注（新高）
- 在 2020 年标注"-15.4%"，在 2023 年标注"+20.4%"

## 阅读路径

```
1. 标题（结论性）→ "V型反弹创新高"
2. 2020 年深色柱子 → 视觉上最低点
3. 增长率折线 → 看到 2021 和 2023 两个峰值
4. 2023/2024 最新柱子 → 历史新高
5. 底部脚注 → 2023 细分市场概况
```

## 克制声明

- 不在每个柱子上显示具体数值（10 年数据太密）
- 不在折线上每个点都标数值（只标 2020 和 2023）
- 社零占比数据不单独呈现，融入脚注
- 高亮比例：2/10 = 20% 略高，但 2020 和 2023 是关键拐点，合理

## 布局

- 画布：800×550
- 标题位置：顶部居中
- 主图区：双轴图，左Y轴=营收（万亿），右Y轴=增长率（%），X轴=年份
- 脚注：底部小字 — 2023 正餐占比 45%，快餐 25%，火锅 11%
- 内边距：上 60 / 右 60（右轴空间）/ 下 80 / 左 60
