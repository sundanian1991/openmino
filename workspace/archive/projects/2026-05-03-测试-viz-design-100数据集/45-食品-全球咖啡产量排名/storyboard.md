# Phase 2 — 视觉叙事设计

## 叙事弧线

**单图场景，采用"结论先行 → 数据支撑"的微弧线。**

## 一句话灵魂

"巴西的咖啡产量是越南的 2.3 倍，超过第 2-3 名之和"——这张图要让人不看数字也能感受到差距。

## 签名视觉元素

- 水平柱状图，从左到右降序
- 巴西的柱子用 Warm 橙色（#E8875F）高亮
- 其余 14 国统一浅灰（#D3D3D3）
- 在巴西柱子末端标注"超 2-3 名之和"

## 阅读路径

```
1. 标题（结论性）→ 第一眼看到"巴西主导"
2. 巴西橙色柱子 → 视觉上最长的 + 唯一的彩色
3. 其余灰色柱子 → 自然形成对比
4. 底部脚注 → 全球供需背景
```

## 克制声明

- 不显示具体数值标签在每个柱子上（15 个数字太密，只看趋势）
- 不在图中标注每个国家的品种（阿拉比卡/罗布斯塔）
- 不做装饰性元素，线性克制风格
- 高亮比例：1/15 = 6.7%，符合 ≤10% 规则

## 布局

- 画布：800×550
- 标题位置：顶部居中，结论性标题
- 主图区：水平柱状图，Y 轴国家名，X 轴产量（千袋）
- 脚注：底部小字 — 全球总产量 1.78 亿袋，消费量 1.72 亿袋
- 内边距：上 60 / 右 40 / 下 60 / 左 120（Y 轴国家名需要空间）
