## 图表 A：平台排名水平柱状图

### 一句话灵魂
> "Meta三平台霸占前三，TikTok是最大挑战者" — 排名本身即结论

### 叙事弧线
**01-开门式**：排名数据开门见山，不需要铺垫

### 阅读路径
```
1. 标题区（顶部）→ "全球社交媒体平台月活跃用户排名：Meta三平台包揽前三"
2. 水平柱状图主体 → 15个平台按MAU降序排列
3. 头部区域 → Top 3用品牌色（蓝色系）高亮
4. 挑战者标注 → TikTok用强调色标出，标注"最快增长"
5. 底部脚注 → 数据截至2025Q1，全球社交媒体用户约52-57亿
```

### 场景设计
- **视觉节拍**：水平柱状图，Y轴从上到下MAU递减
- **克制声明**：
  - 只高亮Top 3和TikTok，其余灰色
  - 高亮比例 = 4/15 ≈ 27%，控制在合理范围（头部聚焦）
  - 柱体末端显示数值标签
  - 按公司分组用微妙色条区分
- **配色**：
  - Meta平台：#4A90D9（蓝色系，品牌关联）
  - TikTok/抖音：#E8875F（暖色强调，挑战者）
  - 腾讯平台：#6C5CE7（紫色系，区分）
  - 其余平台：#8E8E93（灰色）
  - 背景：纯白 #FFFFFF

### 数据组织
- 排序：MAU降序（Facebook 30.7亿 → QQ 5.5亿）
- 平台名称左侧对齐，可读性优先
- X轴：0-35亿范围，等间距网格

## 图表 B：公司阵营矩形树图

### 一句话灵魂
> "Meta一家独大，中美三巨头控制全球社交流量" — 集中度是核心信息

### 叙事弧线
**01-开门式**：份额分布开门见山

### 阅读路径
```
1. 标题区 → "社交媒体平台公司阵营：Meta系占绝对主导"
2. 矩形树图主体 → 按公司聚合的MAU份额
3. Meta最大色块 → 蓝色系，占据最大面积
4. 字节/腾讯 → 暖色/紫色，中等面积
5. 底部脚注 → 合计约52-57亿全球用户，占比69.9%
```

### 场景设计
- **视觉节拍**：一级=公司，二级=旗下平台
- **克制声明**：
  - 只高亮Meta区域（最大色块），其余用不同灰度
  - 标签只显示公司和MAU总量，不堆叠平台细节
  - 面积编码精确对应数值比例
- **配色**：
  - Meta：#4A90D9（蓝色系主色块）
  - 字节跳动：#E8875F（暖色）
  - 腾讯：#6C5CE7（紫色）
  - 其他（Snap/Telegram/新浪/微软LinkedIn）：#8E8E93（灰色聚合）
  - 背景：纯白

### 数据组织
- 公司聚合数据：
  - Meta：71.0亿（FB 30.7 + WA 30.0 + IG 20.0 + Messenger 10.3）*注：有用户重叠*
  - 字节跳动：24.5亿（TikTok 17.0 + 抖音 7.5）
  - 腾讯：19.1亿（微信 13.6 + QQ 5.5）
  - 其他：38.0亿（YouTube 26.5 + Snapchat 8.5 + X 6.1 + LinkedIn 10.0 + 快手 7.0 + 微博 5.9 + Telegram 9.5）*注：YouTube/LinkedIn归Google/微软*

- 实际公司分组修正：
  - Google/Alphabet：26.5亿（YouTube）
  - 微软：10.0亿（LinkedIn）
  - Snap：8.5亿
  - Telegram：9.5亿
  - 新浪：5.9亿
  - 快手：7.0亿
