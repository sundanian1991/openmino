# 供应商ABC分层分级管理方案

> 版本：V1.0 | 日期：2026-04-13 | 编制：Mino

---

## 一、方案概述

### 1.1 背景

原管理规则为"一个季度有2次及以上在综合赛马排名后30%，进入PIP阶段，不能改善则淘汰"。该规则存在以下问题：
- 仅关注尾部供应商，对头部和腰部供应商缺乏差异化管理
- 排名后30%即触发PIP，未考虑绝对表现水平
- 缺少正向激励，供应商缺乏向上动力

### 1.2 目标

建立ABC三级分层管理体系，实现：
- **精细化管理**：按表现分层，差异化配置资源和关注度
- **正向激励**：A级获得资源倾斜，激发向上动力
- **精准淘汰**：C级自动进PIP，连续不改善则淘汰，确保队伍健康

### 1.3 适用范围

全体在库供应商（30+），覆盖金条、企金、信用卡、财富四条业务线。

### 1.4 核心原则

| 原则 | 说明 |
|------|------|
| 定量为主，定性为辅 | 业务督导和质检定量打分占70%，供应商管理定性评价占30% |
| 绝对+相对结合 | 既看绝对分数，也看相对排名 |
| 动态调整 | 按季度评估，可升可降 |
| 透明公开 | 评分标准、计算方式、分级规则对供应商公开 |

## 二、打分模型

### 2.1 评分维度与权重

| 维度 | 权重 | 性质 | 打分主体 |
|------|------|------|----------|
| 业务督导 | 60% | 定量 | 业务督导（日常业务运营团队） |
| 供应商管理 | 30% | 定性 | 供应商管理团队 |
| 质检 | 10% | 定量 | 质检团队 |

**综合得分 = 业务督导得分 × 60% + 供应商管理得分 × 30% + 质检得分 × 10%**

### 2.2 细项指标定义

#### （一）业务督导指标（60分）

| 序号 | 指标 | 满分 | 数据来源 | 计算公式 | 满分条件 |
|------|------|------|----------|----------|----------|
| 1 | 产能达成率 | 25分 | 业务系统 | 实际产能÷目标产能×100% | ≥100%得满分 |
| 2 | 人均产出 | 15分 | 业务系统 | 总产能÷在岗人数 | 达到目标人均值得满分 |
| 3 | 交付时效率 | 10分 | 任务管理系统 | 按时交付工单数÷总工单数×100% | ≥95%得满分 |
| 4 | 任务完成率 | 5分 | 任务管理系统 | 已完成任务数÷下发任务总数×100% | 100%得满分 |
| 5 | 业务增长率 | 5分 | 业务系统 | (本期产能-上期产能)÷上期产能×100% | 环比增长≥10%得满分 |

**各项计分规则：**

**1. 产能达成率（25分）**

| 达成率 | 得分 |
|--------|------|
| ≥100% | 25分 |
| 90%-99% | 25 × 实际达成率 |
| 80%-89% | 20分 |
| 70%-79% | 15分 |
| <70% | 10分 |

**2. 人均产出（15分）**

| 达成率 | 得分 |
|--------|------|
| ≥目标值 | 15分 |
| 目标值90%-99% | 15 × 实际达成率 |
| 目标值80%-89% | 12分 |
| 目标值70%-79% | 9分 |
| <70% | 6分 |

**3. 交付时效率（10分）**

| 时效率 | 得分 |
|--------|------|
| ≥95% | 10分 |
| 90%-94% | 8分 |
| 85%-89% | 6分 |
| 80%-84% | 4分 |
| <80% | 2分 |

**4. 任务完成率（5分）**

| 完成率 | 得分 |
|--------|------|
| 100% | 5分 |
| 95%-99% | 4分 |
| 90%-94% | 3分 |
| 85%-89% | 2分 |
| <85% | 1分 |

**5. 业务增长率（5分）**

| 环比增长 | 得分 |
|----------|------|
| ≥10% | 5分 |
| 5%-9% | 4分 |
| 0%-4% | 3分 |
| 下滑0%-5% | 2分 |
| 下滑>5% | 1分 |

#### （二）供应商管理指标（30分）

| 序号 | 指标 | 满分 | 评估方式 | 评分标准 |
|------|------|------|----------|----------|
| 1 | 响应速度 | 8分 | 主观评价（1-5档） | 5档=8分, 4档=6分, 3档=4分, 2档=2分, 1档=0分 |
| 2 | 配合度 | 8分 | 主观评价（1-5档） | 同上 |
| 3 | 团队稳定性 | 6分 | 流失率数据 | 流失率<5%得6分, 5%-10%得4分, 10%-15%得3分, >15%得1分 |
| 4 | 整改执行力 | 5分 | 整改闭环率 | 闭环率100%得5分, 90%-99%得4分, 80%-89%得3分, <80%得1分 |
| 5 | 管理规范性 | 3分 | 日常合规检查 | 无违规得3分, 轻微违规得2分, 一般违规得1分, 严重违规得0分 |

**评分细则：**

**1. 响应速度（8分）**

| 档位 | 描述 | 得分 |
|------|------|------|
| 5档（优秀） | 紧急需求30分钟内响应，日常需求2小时内响应 | 8分 |
| 4档（良好） | 紧急需求1小时内响应，日常需求4小时内响应 | 6分 |
| 3档（一般） | 紧急需求2小时内响应，日常需求当日内响应 | 4分 |
| 2档（较差） | 紧急需求超2小时响应，或经常性延迟 | 2分 |
| 1档（差） | 经常失联，需要多次催促 | 0分 |

**2. 配合度（8分）**

| 档位 | 描述 | 得分 |
|------|------|------|
| 5档（优秀） | 主动配合临时任务、紧急需求，无条件支持业务调度 | 8分 |
| 4档（良好） | 基本配合各类需求，偶有讨价还价但能执行 | 6分 |
| 3档（一般） | 仅完成分内工作，对额外需求消极应对 | 4分 |
| 2档（较差） | 经常推诿，需要多次沟通才配合 | 2分 |
| 1档（差） | 明确拒绝配合业务调度 | 0分 |

**3. 团队稳定性（6分）**

| 月均流失率 | 得分 |
|------------|------|
| <5% | 6分 |
| 5%-10% | 4分 |
| 10%-15% | 3分 |
| >15% | 1分 |

**4. 整改执行力（5分）**

| 整改闭环率 | 得分 |
|------------|------|
| 100% | 5分 |
| 90%-99% | 4分 |
| 80%-89% | 3分 |
| <80% | 1分 |

**5. 管理规范性（3分）**

| 情况 | 得分 |
|------|------|
| 无任何违规记录 | 3分 |
| 轻微违规（如报表迟交），已整改 | 2分 |
| 一般违规（如流程执行不到位），整改中 | 1分 |
| 严重违规（如数据造假、重大合规问题） | 0分 |

#### （三）质检指标（10分）

| 序号 | 指标 | 满分 | 数据来源 | 计算公式 | 满分条件 |
|------|------|------|----------|----------|----------|
| 1 | 质检合格率 | 5分 | 质检系统 | 合格工单数÷抽检总数×100% | ≥98%得满分 |
| 2 | 客户投诉率 | 3分 | 客诉系统 | 投诉工单数÷总工单数×100% | ≤0.1%得满分 |
| 3 | 红线事件 | 2分 | 质检/合规 | 发生次数 | 0次得满分 |

**计分规则：**

**1. 质检合格率（5分）**

| 合格率 | 得分 |
|--------|------|
| ≥98% | 5分 |
| 95%-97% | 4分 |
| 90%-94% | 3分 |
| 85%-89% | 2分 |
| <85% | 1分 |

**2. 客户投诉率（3分）**

| 投诉率 | 得分 |
|--------|------|
| ≤0.1% | 3分 |
| 0.1%-0.3% | 2分 |
| 0.3%-0.5% | 1分 |
| >0.5% | 0分 |

**3. 红线事件（2分）**

| 次数 | 得分 |
|------|------|
| 0次 | 2分 |
| 1次 | 0分 |
| ≥2次 | 0分 + 直接降为C级 |

### 2.3 综合得分计算

```
综合得分 = 业务督导实际得分 + 供应商管理实际得分 + 质检实际得分
```

**计算示例：**

| 维度 | 满分 | 供应商甲得分 |
|------|------|-------------|
| 业务督导 | 60 | 52 |
| 供应商管理 | 30 | 24 |
| 质检 | 10 | 8 |
| **综合得分** | **100** | **84** |

## 三、ABC分级规则

### 3.1 分数→等级转换表

| 等级 | 分数区间 | 参考占比 | 定位 |
|------|----------|----------|------|
| **A级（优秀）** | ≥85分 | 30%-40% | 核心供应商，重点维护 |
| **B级（合格）** | 70-84分 | 30%-40% | 中坚力量，持续提升 |
| **C级（待改善）** | <70分 | 20%-30% | 重点整改，PIP候选 |

### 3.2 排名校正规则

仅看绝对分数不够，需结合相对排名进行校正：

| 场景 | 规则 |
|------|------|
| 分数≥85且排名前70% | 维持A级 |
| 分数≥85但排名后30% | 降为B级（绝对表现好但相对落后） |
| 分数70-84且排名前70% | 维持B级 |
| 分数70-84但排名后30% | 降为C级 |
| 分数<70 | 维持C级（分数不够就是不够） |

**简单总结：分数决定基准等级，排名后30%最多降一级。**

### 3.3 连续表现调整规则

| 连续季度表现 | 调整规则 |
|--------------|----------|
| 连续2个季度A级 | 授予"战略合作供应商"称号，额外资源倾斜 |
| 连续2个季度B级 | 保持稳定，关注提升空间 |
| 连续2个季度C级 | **启动淘汰评估** |
| C→B→C（反复） | 视为整改无效，启动淘汰评估 |
| B→A（升级） | 正常升级 |
| A→B（降级） | 预警提示，关注下滑原因 |
| B→C（降级） | 直接进入PIP |
| A→C（跳级降级） | 直接进入PIP + 专项调查 |

### 3.4 特殊情况处理

| 情况 | 处理方式 |
|------|----------|
| **新供应商**（入库<1个季度） | 首个季度不参与评级，给予观察期，次季度起正常参评 |
| **红线事件** | 质检红线事件≥2次，直接降为C级，无论分数 |
| **重大贡献** | 为业务带来显著增量（>20%增长），可破格升一级 |
| **业务线调整** | 因业务策略调整导致产能变化的，经业务负责人审批后可豁免当季评级 |
| **不可抗力** | 自然灾害、政策变化等不可抗力因素，经审批后可豁免 |

## 四、分级管理策略

### 4.1 A级供应商管理

**管理策略：资源倾斜 + 深度合作 + 免检优先**

| 维度 | 管理动作 | 执行频率 |
|------|----------|----------|
| 资源分配 | 优先分配优质业务线、高价值项目 | 月度 |
| 业务倾斜 | 在总量目标内优先保障A级供应商产能 | 季度 |
| 免检/抽检 | 质检抽检率降低50% | 季度 |
| 沟通机制 | 季度战略对话（管理层对管理层） | 季度 |
| 结算 | 优先结算，缩短账期 | 月度 |
| 激励 | 年度评选"优秀供应商"，公开表彰 | 年度 |
| 合作深化 | 优先纳入新业务试点 | 按需 |

**激励措施：**
1. 业务量优先分配（在总目标内优先保障）
2. 质检抽检率减半
3. 优先结算（账期缩短5-10个工作日）
4. 新业务试点优先权
5. 年度"优秀供应商"称号
6. 季度管理层面对面沟通

### 4.2 B级供应商管理

**管理策略：常规监控 + 针对性提升**

| 维度 | 管理动作 | 执行频率 |
|------|----------|----------|
| 资源分配 | 正常分配，不优先不限制 | 月度 |
| 质检 | 标准抽检率 | 月度 |
| 沟通机制 | 月度业务复盘会 | 月度 |
| 提升计划 | 针对薄弱指标制定提升方案 | 季度 |
| 整改跟踪 | 对发现的问题限期整改 | 按需 |
| 预警 | 出现下滑趋势时预警 | 实时 |

**提升方向：**
1. 分析失分项，针对性改善
2. 对标A级供应商，找出差距
3. 制定季度提升目标，争取升级
4. 关注团队稳定性和响应速度

### 4.3 C级供应商管理（PIP流程）

**管理策略：限期整改 + 高频监控 + 淘汰退出**

#### PIP（Performance Improvement Plan）流程

| 阶段 | 时间 | 动作 | 负责人 | 输出物 |
|------|------|------|--------|--------|
| **预警** | 评级确定后3个工作日内 | 发送《整改通知书》，明确整改目标和期限 | 供应商管理员 | 《整改通知书》 |
| **沟通** | 预警后5个工作日内 | 面对面沟通，分析原因，制定整改计划 | 供应商管理员+业务督导 | 《整改计划书》 |
| **执行** | 整改期60天 | 供应商执行整改，高频监控 | 供应商管理员 | 周度监控报告 |
| **中期评估** | 整改第30天 | 中期评估，检查进度 | 供应商管理员+业务督导 | 《中期评估报告》 |
| **终期评估** | 整改第60天 | 终期评估，判定结果 | 供应商管理员+业务督导+质检 | 《终期评估报告》 |

#### PIP期间管理动作

| 维度 | 动作 | 频率 |
|------|------|------|
| 业务量 | 缩减30%-50%业务量 | 立即执行 |
| 质检 | 提高抽检率至200% | 持续 |
| 监控 | 周度业务数据跟踪 | 每周 |
| 沟通 | 周度整改进度沟通 | 每周 |
| 报告 | 整改进度报告 | 每周 |

#### 淘汰触发条件

| 条件 | 说明 |
|------|------|
| 连续2个季度C级 | PIP整改后仍为C级 |
| PIP终期评估不达标 | 整改目标未达成 |
| C→B→C反复 | 整改后短期改善但无法维持 |
| 重大红线事件 | 合规红线事件≥2次 |
| 主动退出 | 供应商因自身原因无法继续合作 |

#### 淘汰流程

| 阶段 | 时间 | 动作 | 负责人 |
|------|------|------|--------|
| 淘汰决策 | 终期评估后5个工作日内 | 出具《淘汰评估报告》，报服务组负责人审批 | 供应商管理员 |
| 审批 | 3个工作日内 | 服务组负责人审批 | 王易人 |
| 通知 | 审批后3个工作日内 | 发送《清退通知书》 | 供应商管理员 |
| 交接 | 30天内 | 业务平稳交接，人员妥善安排 | 业务督导 |
| 归档 | 交接完成后 | 供应商档案归档 | 供应商管理员 |

## 六、附录

### 6.1 Excel打分模板字段说明

Excel模板包含以下工作表：

**Sheet 1：供应商基础信息**
- 供应商名称、供应商编码、业务线、合作起始日期、当前等级

**Sheet 2：业务督导评分（60分）**
- 供应商名称、评估季度
- 产能达成率→产能得分（25分，自动计算）
- 人均产出→人均得分（15分，自动计算）
- 交付时效率→时效得分（10分，自动计算）
- 任务完成率→任务得分（5分，自动计算）
- 业务增长率→增长得分（5分，自动计算）
- 业务督导总分（自动求和）

**Sheet 3：供应商管理评分（30分）**
- 供应商名称、评估季度

### 6.2 与原规则对比

| 维度 | 原规则 | 新规则 |
|------|--------|--------|
| 分级维度 | 单一维度（后30%） | 三维评分（业务60%+管理30%+质检10%） |
| 评价方式 | 纯排名 | 绝对分数+相对排名 |
| 激励措施 | 无 | A级资源倾斜 |
| PIP触发 | 季度2次后30% | C级直接PIP |
| 淘汰标准 | 不能改善 | 连续2季度C级或PIP不达标 |
| 管理颗粒度 | 粗放 | 精细化分级管理 |
| 透明度 | 低 | 评分标准公开透明 |

### 6.3 实施建议

1. **首月试运行**：用历史数据回测，验证评分模型的合理性
2. **首季度正式运行**：首次评级后观察分级分布是否合理（A级30-40%，B级30-40%，C级20-30%）
3. **持续优化**：每季度末复盘评分指标的有效性，必要时微调权重
4. **系统支持**：建议将评分模型固化到系统中，自动计算，减少人工干预
