## 一、核心讲师评分

### 1.1 孙总（孙弦）——经营体系全景图 + 客户识别与转化

**评级**：**S（联盟标杆讲师）**

**核心依据**：
1. **内容完整度（S）**：31页PPT覆盖品牌定位→创业历程→供应商价值→赛马机制→产品认知→管理理念→核心指标→效率与长期价值→AI工具→BPO价值顺序，逻辑闭环完整，是联盟培训的标准模板
2. **实战深度（S）**：从「30%数据创造50%GMV」到「打开APP指标系统未统计需手工追踪」到「名产=总放款金额/下发名单量」——每个观点都有数据和自身实战案例支撑，不是概念堆砌，是经验提炼
3. **现场感染力（S）**：5个关键时刻（活下来/守承诺/亲上场/看数据/提效率）的个人故事线，配合「创业失败是必然的，成功才是偶然的」「今天赚的钱未必属于未来」等金句，现场多次引发供应商自发笔记
4. **金句产出（S）**：本场培训金句最多、传播力最强——「数据是资源，经营是能力，结果是创造价值」「老板是给一线做服务的」「只要京东允许做，就要做到第一」等10+条
5. **互动引导（A）**：现场提问环节主动引导供应商思考，将培训从单向输出转向双向对话，但答疑环节时间略短

**综合建议**：联盟标杆讲师，建议每期培训做开场锚定（30-40分钟），为整场培训定调。「第一课」可作为联盟供应商准入标准培训材料。

### 1.2 章志鹏——大额拉新业务破局（ToB vs ToC / 招聘 / 薪酬 / 培训 / 经营）

**评级**：**A（优秀，可继续邀请）**

**核心依据**：
1. **ToB vs ToC 差异认知（A）**：从「ToC借款是偶然」→「ToB借款是必然」→「从电销专员转向京东企业金融服务员」的认知转变线清晰，是培训核心底层逻辑，但案例和数据的支撑密度稍弱于孙总
2. **招聘/薪酬/培训/经营四模块实操（S）**：「30岁以上+有房贷车贷+自主经营意识强」的招聘画像、「前两月底薪+提成保底，两月后自主选择合伙人模式」的合伙人薪酬、「新人培训重心从话术转认知塑造」的培训体系迭代、「从粗放式KPI到精细化客户经营」的经营思路——四模块全部贴近一线可落地
3. **金句（A）**：4条金句有穿透力但偏少——「真诚才是必杀技」「最难的不是设计话术，而是我们知不知道在经营什么客群」「业绩是主管经营出来的，不是员工打出来的」「不同产品，整个成交链路完全不一样」
4. **现场互动（A）**：主动回应参会供应商提问，对小团队主管培训/业绩占比分配/合伙人薪酬落地条件等实操问题给了明确回答，但整体分享节奏稍快

**综合建议**：章志鹏的招聘/薪酬/培训/经营四模块是联盟培训中「落地性最强」的部分。建议下次培训将这四个模块拆为独立专题，每个模块30-40分钟深度展开，配合更多现场案例和数据。

### 1.3 郑畅——数据追踪与名单管理

**评级**：**A（优秀，可继续邀请）**

**核心依据**：
1. **数据驱动思维（S）**：从「5月汇讯人均月授信成功77个，日均3个」→「ATP平均38秒/通」→「日均通时190分钟」→「通过收入倒推动作指标」的完整数据链条是培训中最具操作性的部分。指标宣导逻辑（给员工讲清楚通时背后的收入逻辑，而非单纯强制通时）是核心亮点
2. **名单管理实操（S）**：名单批次差异化分配/预测拨打规则（每日两次，每人100条上限）/每日拨打节奏（8个环节分时段安排）/授信成功轮转机制（共享台账/3天未跟进强制轮转/业绩分成）——全流程可抄可落地
3. **数据案例（A）**：5月份团队数据盘点完整，但缺少同比/环比数据和更长时间维度的趋势分析
4. **现场互动（A）**：主动引导提问和讨论，对「小团队预测落地」「拨打次数对成交率的影响」等新问题的回答偏「后续会补充统计」

**综合建议**：郑畅的数据追踪和名单管理部分是供应商最直接可落地的模块。建议下次培训补充更多可视化数据图表和对比分析，将「数据追踪」和「名单管理」拆为两个独立专题。

### 1.4 杨莹莹——稳单培训与培训体系

**评级**：**A（优秀，可继续邀请）**

**核心依据**：
1. **培训体系系统化（S）**：从「新人班岗前培训六板块」→「新人衔接训五板块」→「高阶培训适配规则」的三层培训体系逻辑清晰。核心亮点：「话术只是表皮，最关键的是讲销售流程的底层逻辑」「为什么员工会瞎想？因为他脑袋太空了，当把脑袋装满就不会瞎想了」——从认知层面拆解培训思路
2. **稳单与AI工具实操（S）**：以一线实战录音为培训素材/高阶培训只针对达到业务水平的员工/AI工具新老员工分级使用规则——全部可抄可落地。特别是「豆包生成话术需截图发群，由管理者结合质检要求修改为合规话术后方可使用」的AI工具使用规则，为行业首创
3. **金句（S）**：「话术只是表皮」「卖点如果没有客户利益就不叫卖点」「员工脑袋太空了才会瞎想」「培训不是终点而是起点」——金句密度高，穿透力强
4. **现场互动（A）**：培训内容偏多，留给现场互动的时间偏少。部分模块（如衔接训五板块）展开深度不够

**综合建议**：杨莹莹的培训体系是供应商联盟培训中最具「方法论沉淀」价值的模块。建议将「新人培训六板块」和「衔接训五板块」分别做成标准化教材（含具体话术/录音样本/考核标准），供联盟供应商直接使用。

### 1.5 新动力团队——企微客户运营SOP

**评级**：**A（优秀，可继续邀请）**

**核心依据**：
1. **企微SOP系统化（S）**：从「加微目标/范围/话术」→「客户分层/跟进节奏/激活规则」→「软文/合规/朋友圈维护」的全链路SOP，已迭代至3.0版本可对外分享
2. **数据验证（S）**：「加微用户余额留存比未加微用户高10个百分点」「电销窗口期外通过企微可带来40%业绩增量」——用数据说话，效果可量化
3. **现场互动（A）**：金句「一天能加十来个企微，只有真正吃过私域的人才知道这个螃蟹的味道多肥美」「三块钱一个五块钱一个搞起来」——生动接地气，但整体内容展开深度稍弱于其他讲师

**综合建议**：新动力企微SOP是供应商联盟培训中最「可抄」的模块之一。建议后续培训将3.0版SOP直接发放给所有供应商，并安排新动力团队做一次专题线上分享，针对不同规模和不同阶段的供应商做差异化的落地指导。
