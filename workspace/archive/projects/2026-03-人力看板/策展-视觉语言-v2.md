# 视觉语言：2026 供应商站点盘点

> 产出自 curatorial-visual-language

## 设计参照系

**分析型地图册（Analytical Atlas）**

参照军用地图和商业分析报告的视觉契约：地图是第一阅读对象，所有元素服务于"一眼看清分布"。核心原则：

- **地图主导**：页面中心永远是地图，筛选器和数据卡围绕地图布局
- **点标记优先**：站点是核心实体，每个站点用一个明确的圆点表示，不重叠
- **数据层级**：概览（地图）→ 筛选（缩小范围）→ 详情（浮层揭示），每层只展示当前需要的信息

## 材料身份分类

| 类别 | 内容 | 视觉处理 |
|------|------|---------|
| **定位材料** | 地图、省份轮廓、站点坐标 | 页面主体，占据60%+面积 |
| **索引材料** | 筛选器、省份列表、供应商列表 | 侧栏/顶部，紧凑布局 |
| **证据材料** | 站点数据卡片、人力数字、表格 | 浮层/抽屉，按需弹出 |
| **状态材料** | 数据填充率标签、规范性问题标注 | 底部/角落，最小字号 |

## 组件规范

### 地图
- 居中放置，宽度100%，高度至少360px
- 省份用浅色填充 + 深色边框（1px, 15%不透明度）
- 站点用直径10px的圆形标记，中心带2px白底
- 标记悬停：直径扩大至14px，显示名称tooltip
- 标记点击：保持放大状态，打开详情浮层

### 筛选器
- 水平排列在地图上方或左上方
- 三组筛选：供应商（下拉）、项目（下拉或标签组）、省份（下拉）
- 选中态：accent底色 + white文字
- 全部选项不选中时 = 显示全部站点
- 多选场景下显示"已选N项"标签

### 站点详情浮层（Drawer）
- 从右侧滑入，宽度≤360px
- 头部：站点名（14px, 600），右上角关闭按钮
- 字段行：供应商 | 项目 | 省份-城市-区县 | 地址 | 人力数据（如果有）
- 底部：链接到供应商页/项目页（如有对应页面）

### 统计卡
- 横排4卡，在筛选器下方
- 数字 + 标签 格式
- 筛选范围变化时数值联动更新

### 数据状态标签
- 完整：绿色圆点 + "有月度数据"
- 部分：黄色圆点 + "部分数据"
- 缺失：灰色圆点 + "地址仅"

## 交互状态

| 交互 | 状态变化 |
|------|---------|
| 地图站点悬停 | 圆点放大 + tooltip显示名称 |
| 站点点击 | 圆点保持放大 + 右侧抽屉滑入 |
| 筛选器选择 | 非匹配站点淡化至20%不透明度 |
| 抽屉打开 | 地图区域宽度缩小（flex调整） |
| 抽屉关闭 | 恢复地图全宽 |
| 筛选重置 | 所有站点恢复100%不透明度 |
| 数据状态切换 | 点击"仅显示有数据的站点"开关 |

### 排除项

| 行为 | 原因 |
|------|------|
| 地图动效/过渡动画 | 信息工具不需要 |
| 3D地图 | 扭曲空间感知，增加渲染负担 |
| 实时定位 | 数据是静态的，不需要实时 |
| 站点间连线 | 人为制造不存在的关联（除非供应商网络图需要） |
| 渐变填充 | 省份色块应均匀，不暗示数据梯度 |
