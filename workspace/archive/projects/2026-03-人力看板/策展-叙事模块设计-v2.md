# 叙事模块设计：2026 供应商站点盘点

> 产出自 curatorial-narrative-modules | 数据：供应商人力&站点盘点2026.md

---

## 页面类型表

| 类型 | 页面角色 | 主角 | 阅读深度 | 导航关系 |
|------|---------|------|---------|---------|
| **入口页** | 地图总览，建立空间感 | 76个站点在全国地图上的分布 | 浅（30秒扫读） | 点击站点→详情抽屉；筛选→各专题 |
| **供应商页** | 展示供应商的站点网络 | 34家供应商 × 各自站点 | 中（2分钟） | 从入口筛选→供应商；供应商→项目/地理 |
| **项目页** | 展示项目的地理分布 | 23个项目 × 站点分布 | 中（2分钟） | 从入口筛选→项目；项目→供应商/地理 |
| **地理页** | 按省份/城市浏览 | 29个省/直辖市 | 中（2分钟） | 从入口→省份；省份→站点清单 |
| **数据状态页** | 盘点完整度说明 | 数据填充率 | 浅（30秒） | 底部入口 |

## 模块拆解

### 1. 入口页（地图总览）

| 模块 | 内容 | 类型 |
|------|------|------|
| 统计条 | 34供应商 / 76职场 / 23项目 / 29省份 | 横排4卡 |
| 地图 | 站点分布（国产地图API或SVG示意图） | 交互地图 |
| 筛选栏 | 多选筛选：供应商、项目、省份 | 筛选项 |
| 站点卡片浮层 | 选中站点后显示：名称、供应商、项目、地址 | 弹出浮层 |
| 数据完整度条 | 显示当前筛选范围内数据填充率 | 进度条 |

**交互**：
- 地图悬停 → 显示站点名和供应商
- 地图点击 → 打开详情浮层
- 筛选 → 地图上对应站点高亮，其余淡化
- 空数据提醒 → 当筛选结果无月度数据时提示

### 2. 供应商页

| 模块 | 内容 | 类型 |
|------|------|------|
| 供应商选择器 | 下拉/搜索选择34家之一 | 选择器 |
| 供应商概要 | 供应商名、项目数、站点数、覆盖省份 | 卡片头 |
| 站点列表 | 该供应商所有站点：职场名、项目、地址、人力数据（如有） | 表格/列表 |
| 省份标注 | 站点所在省份标出，显示跨省运营 | 标签 |

**交互**：
- 供应商切换 → 内容整体刷新
- 点击站点行 → 在地图上定位（跨页跳转到地理页）

### 3. 项目页

| 模块 | 内容 | 类型 |
|------|------|------|
| 项目选择器 | 下拉选择23个项目之一 | 选择器 |
| 项目概要 | 项目名、承接供应商数、站点数 | 卡片头 |
| 供应商列表 | 该项目下所有供应商及其站点数、人力合计 | 排名列表 |
| 站点分布 | 该项目覆盖的省份/城市 | 省份列表+标签 |

### 4. 地理页

| 模块 | 内容 | 类型 |
|------|------|------|
| 省份选择 | 29个省级区域选择 | 列表/下拉 |
| 省份概要 | 该省站点数、供应商数、项目数 | 统计卡 |
| 站点清单 | 该省所有站点：职场名、供应商、项目 | 列表 |
| 数据标记 | 标注该省数据填充状态 | 状态标签 |

### 5. 数据状态页

| 模块 | 内容 | 类型 |
|------|------|------|
| 填充率概览 | 各项目/各供应商的数据填充百分比 | 条形图/热力图 |
| 字段状态表 | 字段维度对照：完整/部分/缺失 | 表格 |
| 质量问题 | 省份名称、项目名称规范性问题 | 列表 |
| 编辑说明 | 数据口径说明 | 文本 |

## 导航图

```
  入口页（地图总览 + 筛选器）
       │
       ├── 点击站点 ──→ 详情浮层（含供应商/项目/地址）
       │
       ├── 筛选供应商 ──→ 供应商页
       │     │
       │     └── 站点列表 ──→ 地理页定位
       │
       ├── 筛选项目 ──→ 项目页
       │     │
       │     └── 供应商/站点 ──→ 供应商页/地理页
       │
       ├── 筛选省份 ──→ 地理页
       │
       └── 底部入口 ──→ 数据状态页
```

## 页面类型优先级

| 优先级 | 页面 | 原因 |
|--------|------|------|
| P0 | 入口页（地图） | 核心交互，建立整体认知 |
| P0 | 供应商页 | 按供应商维度查看的基础功能 |
| P0 | 项目页 | 按项目维度的核心视图 |
| P1 | 地理页 | 基于地图的深入浏览 |
| P2 | 数据状态页 | 辅助理解数据质量 |

## 实现说明

- 地图组件：使用SVG示意图（国产地图API受限），按省份着色，站点用圆点标注
- 所有页面数据一次性加载，无需后端
- 筛选器采用多选下拉，支持交叉筛选（供应商+项目+省份）
- 详情浮层用抽屉（drawer）实现，不另开页面
- 数据状态页为纯文本说明，不需要图表库
