# 视觉语言：2026年度供应商人力看板

> 产出自 curatorial-visual-language

---

## 设计参照系

**分析型仪表盘（Analytical Dashboard）**

参照商业智能工具的视觉契约 —— 信息密度高但不拥挤，数据图表是页面主角，装饰服从刻度。核心原则：

- **先数据，后形式**：每个视觉元素必须帮助读数据，否则删除
- **颜色编码意义**：色板映射管理动作（正常/预警/风险），不用于装饰
- **越低越轻**：二级信息用字重和字号区分，不加额外容器

## 材料身份分类

| 类别 | 包含 | 视觉处理 |
|------|------|---------|
| **证据材料** | 图表、数据表格、具体数值 | 占据内容主体，图表用白色/浅色底，表格用行分割 |
| **定位材料** | 顶部导航tab、页码、当前section标题 | 顶部固定，高亮当前项 |
| **分析材料** | 趋势标注、异常标记、环比标签 | 悬浮于图表之上，用小字号+语义色 |
| **注释材料** | 数据说明、质量问题标注、术语解释 | 底部区域，最小字号，弱化色 |

## 组件规范

### 卡片
- 背景：elevated，1px border，10px radius，12px padding
- 两列布局用 `grid-template-columns: 1fr 1fr; gap:12px`
- 不适用阴影，不使用渐变色

### 图表
- 高度：主图 240-280px，辅助图 200px
- 圆角：bars borderRadius=6
- 线条：tension=0.3，borderWidth=2
- 图例：顶部或右侧，boxWidth=12，fontSize=10

### 表格
- 全局背景透明，仅行间下边框 `rgba(0,0,0,0.06)`
- 数值右对齐，标签左对齐
- 鼠标悬停行：`var(--widget-accent-subtle)` 背景

### 标签/Tag
- 圆角 9999px，`8px` 水平内边距，`2px` 垂直
- 背景 `var(--widget-accent-subtle)`

### 交互筛选器
- 标准 `<select>` 或按钮组
- 选中态：accent 底色 + white 文字
- 非选中态：inset 底色 + 默认文字

## 交互状态

| 状态 | 样式 |
|------|------|
| 图表悬停数据点 | 半径扩大至 5px，边框加深 |
| 选中筛选 | accent 色背景 |
| 图表十字准线 | 无，用 tooltip 替代（避免干扰） |
| 点击跳转 | 无明显点击反馈，直接切页面 |
| 加载过渡 | 无，数据一次性加载 |

### 排除项（不要做）

| 行为 | 原因 |
|------|------|
| 图表3D效果 | 扭曲数据感知 |
| 渐变色填充 | 混淆数值边界 |
| 动画/动效 | 信息型工具不需要 |
| 背景纹理/图案 | 增加视觉噪音 |
| 圆角 < 4px | 与现代 UI 不一致 |
| 图例在图表内部 | 遮挡数据区域 |
