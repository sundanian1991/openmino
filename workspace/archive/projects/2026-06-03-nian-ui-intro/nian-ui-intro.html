02
Capabilities

它能做什么？

Director
导演选择

200+ 导演风格库，每位导演有 signature 技法、构图偏好和叙事节奏。支持多部电影参考。

Narrative
叙事节拍

25 种电影叙事节拍（冷开场、建立镜头、证据墙、深度潜行等），映射到 section 功能。

Composition
构图家族

80 种布局构图模式，从对称到不对称，从全出血到约束网格，每种附带电影参照。

Camera
镜头语言

50 种镜头揭示效果（推进、摇臂、稳定器浮入、幕帘擦拭等），纯 CSS 实现。

Interaction
互动效果

50 种互动效果（焦点光、倾斜、磁性、字幕滚动等），限定的互动预算防止过度动效。

Anti-Converge
反趋同系统

强制每次输出与先前的设计在结构层面不同——shell、hero、节拍、构图全部可审计。

03
Process

如何工作 — 4 Phase 工作流

01
Decisions

选择导演、电影、场景色。研究导演的结构技法。确定构图家族。做 uniqueness audit 和 shell-ban list。

→ decisions.md
02
Storyboard

定义站点级电影语法。写页面场景主题。确定签名构图。选择叙事节拍。应用反趋同检查。

→ storyboard.md
03
Compiled Spec

选择入场效果、互动行为、布局构图。锁定每个 section 的视觉元素。写实施规格。

→ compiled-spec.md
04
Build & Verify

从编译规范构建。应用 reduced-motion 和响应式。通过 nian checklist + screening room 验证。

→ index.html
