# Decisions — 供应商月度通报

## Start Questionnaire
1. **如何开始?** Step-by-step — 供应商管理月报数据展示
2. **图片?** 否

4. **场景色?** **Olive** `#4A5D3A` — 增长、权威、生命力。月报的核心是管理决策——从数据中发现异常，驱动供应商优化。Olive 的"生长"意象匹配供应商体系的健康发展，权威性匹配管理通报的定位。
