Core Thesis

定价模式的根本转变

Coding Plan 时代

以编程场景为入口，固定月费订阅。卖的是工具使用权——按Prompt/请求次数计费，信息不对称有利于厂商。从价格战起步，用补贴抢用户规模。

→

Core Insights

三大商业逻辑

关于定价权与市场节奏

01

计量单位即定价权

谁定义计量单位，谁就掌握定价权。MiniMax 从 Prompt 转 Token、Kimi 从 Prompt 转 Token，都是在争夺这个定义权。云厂商用 API 请求次数——数字看起来大但实际消耗多，信息不对称有利于厂商。可复用：模糊计量利获客，透明计量利信任

02

补贴—收紧—分层三段式

补贴 抢用户规模 → 收紧 测试价格弹性 → 分层 利润最大化。AI 行业用 18 个月走完了电信 20 年的路。不要一开始就想好定价，先跑量再测试最后精细分档。

03

从入口到生态的升级

Coding 是获客入口，Token 是生态载体。MiniMax 的 Token Plan 打包了图片、语音、音乐、视频——29元 买到的不是代码补全，是全模态 AI 能力。用高频刚需获客，用高价值场景提 ARPU。

关于市场格局与成本结构

04

国内外定价逻辑的根本差异

国内 7 家在同一池子打价格战——比谁更便宜。海外面对 Claude/GPT/Gemini 三巨头——证明自己不比他们差。国内低价是因为有人在替你卷，海外用户为质量和生态付费。

05

Agent 时代成本结构剧变

70% 的 coding agent token 是浪费。一个开发者 8 个月消耗 100 亿 Token。Cursor 转 credit-based 后重度用户一周超支 $350。订阅费是地板不是天花板——实际账单是标价的 2-5 倍。

06

OpenClaw：Token 消耗的催化剂

GitHub 355K stars 的开源 Agent 框架直接催热了 Coding Plan。MiniMax M2 用量 2 月比 12 月增长 6 倍，Coding Plan 消耗增长 10 倍+。Anthropic 被迫从标准订阅移除 OpenClaw 访问——成本撑不住了。

Outlook

未来预判

Cost

Token 成本持续下降

国产芯片放量 + DeepSeek V4 MoE 效率革命 + 推理芯片竞争。利润空间从算力差价转向服务溢价。

Supply

国产芯片替代加速

昇腾 + 寒武纪 + 平头哥形成国产算力三角。CANN 效率已达 CUDA 90%。国内成本结构与海外分化。

Power

开源重塑定价权

DeepSeek V3.2 月调用 5.35 万亿 Token（全球第三）。厂商定价权来自服务便利性，而非模型独占性。

Demand

Agent 成本不可逆增长

推理需求已达训练的 4-5 倍，2028 年推理占比预测 73%。不限量订阅不可持续，所有厂商转向精细计量。
