# 组件认知操作映射

> 每个组件对应一个核心认知操作。选组件 = 判断用户在执行什么认知操作 × 数据是什么形态。
>
> 排除组件（标 ✕）不入库。

---

## 常见组合模板

| 场景 | 认知操作序列 | 组件组合 |
|------|------------|---------|
| 报告 | 概览 → 对比 → 解读 | stat-block + comparison + callout |
| 分析 | 排序 → 度量 → 解读 | rank-bar + gauge + annotated-highlight |
| SOP | 流程 → 解读 → 深入 | step-sequence + callout + accordion |
| 品牌 | 氛围 → 背书 → 概览 | Hero + logo-wall + stat-block |
| 监控 | 度量 → 排序 → 对比 | gauge + rank-bar + data-table |
| 诊断 | 度量 → 对比 → 解读 | gauge-arc + comparison + callout |
