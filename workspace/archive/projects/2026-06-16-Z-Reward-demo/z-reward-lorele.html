THE PROBLEM

视觉偏好是主观的，
标量奖励丢掉了不确定性。

同一张图，不同标注者会打出不同分数。传统 reward model 把这种分布压缩成一个数字，丢失了细粒度的质量差异和标注者不确定性。

GDSO LOSS

三路信号联合优化

GDSO 不只用策略梯度，还直接对分数分布和分数间距做监督。

LGDSO · 总损失
LGDSO = LGRPO({rj,i}) + αpt · LptCE + αpw · Lpw
三路信号：策略梯度 + 点态分布监督 + 成对间距监督
rpt · 点态奖励
rpt = 1 - |μ - ŝ| / (Smax - Smin)
预测期望分数与标注分数的接近程度
让绝对打分对齐人类标注，不只是相对排序
rpw · 成对间距奖励
rpw = 1 - avg|Δμ - Δŝ| / range
预测间距与标注间距的一致性
不只学"谁更好"，还学"好多少"。比 BT 更精细
LRISD · 蒸馏损失
L = E[DKL(qT(s|ρT) ∥ qϕ(s))]
Student 直接匹配 Teacher 的分数分布
不学推理过程，只学判断结果

KEY INSIGHTS

三个核心洞察

INSIGHT 01

分布 > 标量

用分数分布而非单一标量来建模视觉偏好，保留标注者不确定性和细粒度质量差异。

INSIGHT 02

判断 ≠ 推理

Reward model 不需要复现 Teacher 的推理过程，只需要复现 Teacher 的判断结果。

INSIGHT 03

间距监督 > BT

Bradley-Terry 只要求赢方更高，不限制间距。GDSO 的 pairwise loss 要求预测间距 = 标注间距。

"
Reward models don't need to reproduce how a teacher reasons; they need to reproduce how a reasoning teacher judges.
