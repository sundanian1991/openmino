用九宫格分级盘点，把 30+ 供应商分成保留、培养、优化三张清单。每月滚动，每季度校准。
P0 项目 月度评估 + 季度盘点
现状
分级靠直觉，管理凭经验
问题 1
30+ 供应商，没有统一标尺
头部 6 家贡献绝大部分产能，但谁是真正的"核心"、谁在"搭车"，缺乏量化依据。
问题 2
赛马只看业绩一个维度
合规问题、交付稳定性、协作意愿被忽略。业绩好不等于"好用"。
问题 3
没有三张清单
保留谁、培养谁、优化谁——每次讨论都从零开始，没有持续的分级记录。

评分细节
基础分 + 合规扣分
赛马基础分
直接采用现有赛马成绩，满分 100。
合规扣分规则
事件扣分说明
黄牌整改-10/次巡检发现并下发整改函
投诉 / 一般违规-5/次质检通报、客户投诉
红线违规当月降级安全、数据泄露等严重事件
基础分 = 赛马分 - 合规扣分
注意
需确认赛马是否已包含合规指标。如已包含，扣分项减半或只计红线事件。
评分细节
稳定性系数 + 供管自评
交付稳定性系数
月底扫三张检查线
A 稳定   x 1.0
人员无大幅变动、产能无异常、SLA 全达成
B 波动   x 0.9
人员流失>10% / 产能波动>20% / SLA 一次未达成
C 失控   x 0.75
核心人员离职 / 产能断崖 / SLA 两次以上未达成
供管自评加分 (0~4)
四项各评高中低，高=1分，中低=0分
维度高=1 的条件
战略协作主动提出建设性建议，与业务方向一致
响应质量24h 内响应，反馈有实质内容
高效执行临时任务按时按质完成，无返工
共同成长产线会议积极贡献，对整体有额外贡献
