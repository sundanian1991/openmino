核心结论

1 双极化严重：毅航合肥复贷独揽 62% S 级席位；翰锐首贷 C 级率 40%、伽玛 36.4%、赛维斯 34.1% — 三个职场系统性弱势，需立即重点诊断。
2 业绩差距在"转化"而非"触达"：S 级有效→首贷转化率 3.8%，C 级仅 1.4%（差 2.7 倍）；而接通→有效环节差距仅 1.1pp。说明问题在谈判能力和客户识别，不在工作量和话术执行。
3 MOB1-3 新人成 C 级重灾区：C 级中 MOB1-3 占比 33%（31/93），司龄分布呈"越短 C 级率越高"趋势。新人标准化培训与带教机制缺失，是 C 级改善的最短路径。

Actions
行动建议决策矩阵
按紧迫度 + 影响面排序的 6 条建议
P0 紧急

翰锐/赛维斯/伽玛 首贷专项诊断

三个职场首贷 C 级率均超 34%，合计 C 级 37 人占首贷大盘 C 级 39.8%。建议立即启动 14 天改善期，暂停优质名单发放，配标准化 SOP+组长每日通关。

P0 紧急

MOB1-3 新人前置培训体系

C 级中 MOB1-3 占比 33%。建议新员工入职前 30 天设"通关门槛"——日均有效通时>60分钟 + 转化率>50% 方可出师。配师傅带教津贴。

P1 重要

毅航合肥复贷 S 级经验萃取

毅航复贷 S 级占大盘 62%（67/108），其 ATT（110.7s）、人均 GMV（1271.7万）均为全盘标杆。建议录制优秀录音 + 编写话术手册，跨职场复制。

P1 重要

转化能力专项提升（有效→首贷环节）

S 级有效→首贷转化率 3.8% 是 C 级 1.4% 的 2.7 倍，而接通→有效环节差距仅 1.1pp。核心提升点不在话术包装，在需求挖掘和客户识别。

P2 常规

广达/华啸 复贷零 S 级问题跟踪

广达陕西复贷 S=0/24, 华啸 S=0/23，且广达 C 级率 54.2%（13/24）全盘最高。建议月度跟踪，若连续 2 月无改善启动供应商约谈。

P2 常规

预警机制强化 — 翰锐/岐力复贷预警率高

翰锐复贷预警率 7.5%（4/53）、岐力 10%（6/60），显著高于整体 3.2%。建议将预警阈值下调至连续 2 天未上线即触发组长关注。
