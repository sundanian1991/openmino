## 七、Section 模块化设计

### 核心理念

**模板 = Section 的有序集合。Section 是独立可复用的 HTML 片段。**

```
模板 T-DATA-L3-ST
  ├── hero-statement (S01)     ← 可被 #13,#14,#25,#26 复用
  ├── data-metrics (S05)       ← 可被 #01,#03,#11 等复用
  ├── data-ranking (S06)       ← 可被 #15,#24 等复用
  ├── data-table (S07)         ← 可被 #16,#20 等复用
  ├── content-statement (S15)  ← 可被 #02,#08 等复用
  └── footer (S21)             ← 全部 36 个模板共用
```

### 复用规则

1. 每个 section 有独立 HTML + CSS，可单独提取
2. section 之间通过 CSS 变量传递主题（palette, accent）
3. 施工时可跨模板引用：`<!-- ref: T-DATA-L3-ST#data-metrics -->`
4. 被引用 section 保持原样，只替换内容数据
