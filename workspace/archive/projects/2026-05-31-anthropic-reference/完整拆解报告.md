## 2. SVG 配图库（已下载 72 个）

### 核心发现：极简双色

**72 个 SVG 中 99% 只用两个颜色：**

| 颜色 | HEX | 出现次数 | 用途 |
|------|-----|---------|------|
| 深色 | `#141413` | 141 次 | 主体形态 |
| 浅色 | `#FAF9F5` | 85 次 | 背景/负空间 |

仅有 3 个 SVG 用了额外颜色（`#D97757` 暖橙、`#C46686` 玫红、`#6A9BCC` 蓝），占比极低。

### 尺寸分布

| 维度 | 数量 |
|------|------|
| 1000x1000（正方形） | 69 个 |
| 1680x1260（4:3） | 2 个 |
| 1681x1261（4:3） | 1 个 |

**绝大多数是 1000x1000 正方形**，适合做文章 hero 配图。

### 复杂度分布

| 级别 | 路径数 | 数量 | 占比 |
|------|--------|------|------|
| 简单 | 2-3 条 | 50 个 | 69% |
| 中等 | 4-8 条 | 21 个 | 29% |
| 复杂 | 9+ 条 | 1 个 | 1% |

**70% 的 SVG 只有 2-3 条路径**，文件大小中位数 13KB。

### 形态特征

- **有机流体形态**（amorphous blobs）：无几何锐角，曲线流动
- **双色正负形**：深色形态 + 浅色背景，互为正负
- **无描边、无渐变、无投影**：纯 fill
- **手绘感**：路径节点密集，像手绘后矢量化
- **部分含散点装饰**：椭圆、小圆点（如最后一个样本中的 `ellipse` 和 `circle` 元素）

### Mino 可用性评估

| 维度 | 评估 | 说明 |
|------|------|------|
| **直接使用** | 可以 | 双色 SVG 零依赖，直接嵌入 HTML |
| **修改颜色** | 极简 | 只需替换 `#141413` 和 `#FAF9F5` 为 Mino 的 `#191919` 和 `#FAFAF7` |
| **复刻难度** | 中等 | 形态是手绘的，难以程序化生成；但原则可学 |
| **设计原则可迁移** | 高 | 双色约束 + 有机形态 + 极简路径 = 可复制的方法论 |

## 3. 字体系统

### Anthropic 字体清单

| 字体 | 类型 | 授权 | Mino 替代 |
|------|------|------|-----------|
| **Anthropic Sans** | 无衬线 | 自研，不可用 | Noto Sans SC |
| **Anthropic Serif** | 衬线 | 自研，不可用 | Noto Serif SC |
| **Anthropic Mono** | 等宽 | 自研，不可用 | Fira Code |
| **Copernicus** | 衬线（Book/Medium） | 商业授权 | Cormorant Garamond（Google Fonts，免费） |
| **Styrene A** | 无衬线（Regular/Medium） | 商业授权 | 无直接替代，风格近似 geometric sans |
| **Styrene B** | 无衬线变体 | 商业授权 | 同上 |
| **Tiempos Text** | 衬线（段落用） | 商业授权 | Spectral 或 Lora（Google Fonts，免费） |
| **JetBrains Mono** | 等宽 | 开源，可用 | Fira Code（已用） |

### 字体使用模式

```
body 默认 → Anthropic Serif（衬线）
标题 → Anthropic Serif / Copernicus（衬线）
UI 元素 → Anthropic Sans（无衬线）
代码 → JetBrains Mono / Anthropic Mono
文章段落 → Tiempos Text（衬线）
```

**关键差异**：Anthropic 的正文全部用衬线，Mino 用无衬线。这是风格选择，不是错误。

### 推荐替代方案

| 如果想对齐 Anthropic | 用 | 来源 |
|---------------------|---|------|
| Copernicus 替代 | **Cormorant Garamond** | Google Fonts（免费） |
| Tiempos Text 替代 | **Spectral** 或 **Lora** | Google Fonts（免费） |
| Styrene A 替代 | 不需要替代 | Mino 的 Noto Sans SC 已足够 |

| 值 | 用途 |

## 5. 配图风格分析与复刻

### Anthropic 的配图三板斧

1. **双色有机 SVG**（hero 配图）：`#141413` + `#FAF9F5`，流体形态
2. **深底信息图**（文章内）：`#141413` 背景 + 彩色色块，概念图解
3. **黑白摄影**（团队页）：低饱和度人物照

### 复刻可行性

| 类型 | 复刻难度 | 方法 |
|------|---------|------|
| 双色 SVG | **中** | 手绘 path 或用 SVG 编辑器（Figma/Inkscape）画有机形态 |
| 深底信息图 | **低** | CSS/SVG 即可，色块+文字 |
| 黑白摄影 | **不适用** | 需要实拍 |

### Mino 可迁移的设计原则

1. **双色约束**：hero 配图只用品牌色+背景色，不用彩色
2. **有机 > 几何**：流体形态比矩形/圆形更有重量感
3. **极简路径**：2-3 条 path 就够，不需要复杂
4. **正负形**：深色形态 + 浅色背景互为正负，自然形成视觉焦点
5. **散点装饰**：小椭圆/圆点做点缀，增加层次但不喧宾夺主
