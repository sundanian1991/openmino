# Anthropic 官网设计拆解

> 日期：2026-05-31 | 来源：anthropic.com/research, /news

---

## 1. 字体系统

| 用途 | Anthropic | Mino 对应 |
|------|-----------|-----------|
| **正文（serif）** | Anthropic Serif（自研） | Noto Serif SC |
| **文章段落** | Tiempos Text（第三方衬线） | — |
| **UI 元素（sans）** | Anthropic Sans（自研） | Noto Sans SC |
| **代码/等宽** | JetBrains Mono / Anthropic Mono | Fira Code |
| **显示/装饰** | Copernicus（衬线）、Styrene A/B（无衬线） | — |

**核心模式**：
- body 默认 serif（`font-family: var(--anthropic-serif), serif`）
- 标题 serif（headline-1~6 都带 `.serif` class）
- UI 元素 sans-serif（labels、buttons、captions、fine-print）
- 段落用 Tiempos Text（`paragraph-s`、`paragraph-m`）

**与 Mino 的匹配度**：标题衬线 + 正文衬线 + UI 无衬线 + 数据等宽。Mino 用衬线标题 + 无衬线正文，方向一致，只是 Anthropic 更彻底——连正文都用衬线。

## 2. 色彩系统

### 主色板（从 CSS 提取）

| 色阶 | Anthropic HEX | Mino 对应 | 匹配度 |
|------|--------------|-----------|--------|
| 最深 | `#0f0f0e` / `#141413` | `#191919`（Slate Dark） | 极近 |
| 深灰 | `#1a1918` / `#1f1e1d` | `#262625`（Slate Medium） | 极近 |
| 中灰 | `#30302e` / `#3d3d3a` | `#40403E`（Slate Light） | 极近 |
| 云灰 | `#73726c` / `#87867f` | `#666663` / `#91918D` | 一致 |
| 暖灰 | `#c2c0b6` / `#d1cfc5` | `#BFBFBA` | 一致 |
| 象牙 | `#f0eee6` / `#f5f4ed` / `#faf9f5` | `#E5E4DF` / `#F0F0EB` / `#FAFAF7` | 完全一致 |

### 强调色

| 用途 | Anthropic | Mino |
|------|-----------|------|
| 主强调 | `#d97757`（暖橙陶） | `#CC785C`（Book Cloth） |
| 辅助 | `#2c84db`（蓝） | `#61AAF2`（Focus） |
| 错误/危险 | `#bf4d43` | `#BF4D43`（Error） |
| 生命力 | `#788c5d`（橄榄绿） | `#2E8B6E`（Signal Good） |
| 粉彩 | `#c46686`（玫红） | — |

### 背景色

| 层级 | Anthropic | Mino |
|------|-----------|------|
| 主背景 | `#faf9f5` | `#FAFAF7`（Ivory Light） |
| 次背景 | `#f0eee6` | `#F0F0EB`（Ivory Medium） |
| 组件背景 | `#fff` | `#fff`（Surface） |

### 结论

**色彩系统几乎完全一致。** Anthropic 的暖灰阶（#0f→#faf9f5）和 Mino 的 Slate→Ivory 链条是同一个设计哲学。强调色 `#d97757` vs `#CC785C` 同属暖陶色系。

## 3. 配图风格

### SVG 抽象插画（hero images）
- **尺寸**：1000x1000px 正方形
- **配色**：仅双色 — `#141413`（深）+ `#FAF9F5`（浅），零彩色
- **风格**：流体有机形态（amorphous blobs），无几何锐角
- **技法**：SVG path fill，无描边，无渐变，无投影
- **情绪**：抽象、有机、安静、有重量感

### 数据图表（article diagrams）
- **风格**：信息图而非统计图
- **配色**：深底 `#141413` + 彩色色块（绿/橙/红/蓝/紫）
- **特点**：概念图解（architecture flow, classification tree）
- **与 Mino 的关系**：Mino 的 inline SVG sparkline/bar 更轻量，但方向一致——都是零依赖

### 摄影
- **风格**：黑白/低饱和度人物照
- **用途**：团队页、about 页
- **不用于**：文章配图

## 4. 排版结构

### 文章页层级

```
ch-number（0.68rem mono, ALL CAPS, 灰色）  ← 我们已有
ch-title（2.5rem+ serif, bold）              ← 我们已有
ch-hook（1rem sans, 灰色副标题）             ← 我们已有
ch-body（serif body text）                    ← 我们用 sans
```

### 字号层级（从 CSS 提取）

| 类名 | 字体 | 特征 |
|------|------|------|
| headline-1 | serif | 最大标题 |
| headline-2~6 | serif | 递减 |
| body-large-1/2 | serif | 大段落 |
| body-1~4 | serif | 正文 |
| paragraph-s/m | Tiempos Text | 文章段落 |
| caption | sans-serif | 说明文字 |
| fine-print | sans-serif | 小字 |
| text-label | sans-serif | 标签 |

### 布局特征
- `reading-column` class — 限制阅读宽度
- `#141413` theme-color — meta 标签定义
- 暖色系全局一致，无冷色干扰

## 5. 与 Mino Design System 匹配度

| 维度 | 匹配 | 说明 |
|------|------|------|
| **色温** | 完全一致 | 暖灰阶 + 暖陶强调 |
| **色阶数量** | 一致 | 6-7 级灰阶 + 1-2 强调色 |
| **背景色** | 完全一致 | `#faf9f5` ≈ `#FAFAF7` |
| **标题字体** | 方向一致 | Anthropic 用自研 serif，Mino 用 Noto Serif SC |
| **正文策略** | 微差 | Anthropic 全 serif，Mino 标题 serif + 正文 sans |
| **等宽字体** | 一致 | JetBrains Mono ↔ Fira Code |
| **配图风格** | 高度一致 | 抽象有机 SVG，双色，零装饰 |
| **层级结构** | 一致 | mono 标号 → serif 标题 → 副标题 → 正文 |
| **禁忌** | 一致 | 无渐变、无 glassmorphism、无 bounce、无 neon |

### Mino 可借鉴的

1. **正文也用 serif** — Anthropic 的文章正文是 Tiempos Text（衬线），阅读体验更沉浸。Mino 可以考虑在长文场景下正文也切 serif
2. **SVG 抽象插画** — 用双色（品牌色 + 背景色）做有机形态 SVG 作为 hero 配图，比纯文字 hero 更有重量感
3. **Copernicus / Styrene 字体** — 如果想进一步对齐 Anthropic 风格，可以引入这两个字体做显示/装饰用途
4. **Stagger 延迟动效** — 文字逐行/逐块 fade-in，营造"打字"感
