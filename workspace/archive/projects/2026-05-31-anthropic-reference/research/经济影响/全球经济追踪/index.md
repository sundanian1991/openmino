title: Anthropic 经济指数：追踪 AI 在美国与全球经济中的角色
date: 2025-09-14

夏威夷的旅行规划、马萨诸塞州的科学研究、印度的网络应用开发。表面上看，这三类活动几乎没有共同点。但它们恰恰是 Claude 在各自地区*最过度集中*的使用场景。

这不意味着这些是最*热门*的任务：软件工程在几乎所有美国州份和全球国家中仍然遥遥领先。它意味着马萨诸塞州的人比其他地方的人更倾向于向 Claude 寻求科研帮助——或者，比如说，巴西的 Claude 用户对语言表现出格外的热情：他们用 Claude 做翻译和语言学习的频率约为全球平均水平的六倍。

这些统计数据来自我们的第三份 [Anthropic 经济指数报告](http://anthropic.com/research/anthropic-economic-index-september-2025-report)。在本期报告中，我们扩展了追踪 AI 采纳早期模式的范围，这些模式正开始重塑工作与经济。我们衡量了 Claude 在以下几个维度上的使用差异：

**美国各州内部**：首次提供了 AI 使用在各州差异的详细评估。我们发现，各州的经济结构决定了哪些州的人均 Claude 使用量最高——令人意外的是，使用量最高的州并*不是*编程主导的州。
**不同国家之间**：新分析发现，各国的 Claude 使用量与收入强相关，且低使用量国家的人用 Claude 自动化工作的频率*高于*高使用量国家。
**时间维度**：我们将最新数据与 [2024 年 12 月—2025 年 1 月](https://www.anthropic.com/news/the-anthropic-economic-index)和 [2025 年 2 月—3 月](https://www.anthropic.com/news/anthropic-economic-index-insights-from-claude-sonnet-3-7)的数据进行比较。发现"指令式"自动化任务的比例从 27% 急剧上升到 39%，表明 AI 承担的责任（以及用户的信任）在快速增加。
**商业用户**：我们首次纳入了 Anthropic 自有 API 客户的匿名数据（在 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 用户的基础上），使我们能够分析企业的交互行为。发现 API 用户比消费者更倾向于用 Claude 自动化任务，这暗示重大的劳动力市场影响可能即将到来。

以下是[报告](http://anthropic.com/research/anthropic-economic-index-september-2025-report)摘要。此外，我们设计了一个[交互式网站](http://anthropic.com/economic-index)，供你自行探索数据。首次支持按美国各州和我们追踪的所有职业搜索 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 使用趋势和结果，了解 AI 在你所在地或同行工作中的使用方式。最后，如果你希望在我们的分析基础上进一步研究，我们已将数据集与之前经济指数报告的数据一同[公开发布](https://huggingface.co/datasets/Anthropic/EconomicIndex)。

**地理分布**

我们将 Anthropic 经济指数扩展到了地理维度。以下是我们对 Claude 在不同国家和美国各州使用模式的发现。

### 跨国比较

美国使用 Claude 的量远远超过其他任何国家。印度位居第二，巴西、日本和韩国紧随其后，各自份额相近。

然而，这些国家的人口规模差异巨大。为修正这一因素，我们按各国的全球劳动人口占比调整了 Claude.ai 使用份额，得出 **Anthropic AI 使用指数**（AUI）。AUI 大于 1 的国家，其 Claude 使用频率高于仅凭劳动人口规模预期的水平，反之亦然。

从 AUI 数据可以看出，一些小规模技术先进国家（如以色列和新加坡）在相对劳动人口的 Claude 采纳率上领先。这在很大程度上可以用收入来解释：我们发现人均 GDP 与 Anthropic AI 使用指数之间存在强相关性（人均 GDP 每高 1%，AUI 约高 0.7%）。这很合理：Claude 使用最多的国家通常拥有健全的互联网连接，以及以知识工作而非制造业为导向的经济结构。但这确实引发了一个经济分化的问题：此前的通用技术，如电气化或内燃机，既带来了巨大的经济增长，也造成了全球生活水平的[大分化](https://www.aeaweb.org/articles?id=10.1257/jep.11.3.3)。如果 AI 的影响在富裕国家最为显著，这项通用技术可能会产生类似的经济后果。

### 美国内部格局

人均 GDP 与人均 Claude 使用量之间的联系同样适用于美国各州之间的比较。事实上，在美国内部，使用量随收入增长的速度比跨国更快：美国人均 GDP 每高 1%，人口调整后的 Claude 使用量约高 1.8%。不过，收入在美国内部的解释力实际上*低于*跨国的解释力，因为整体趋势中存在更大的方差。也就是说，除收入外，其他因素必然解释了人口调整使用量中更多的差异。

还有什么原因能解释这种采纳差距？我们的最佳猜测是各州经济结构的差异。美国 AUI 最高的是哥伦比亚特区（3.82），该地区 Claude 最不成比例频发的用途是编辑文档和搜索信息，以及与 DC 知识工作相关的其他任务。类似地，编码相关任务在加利福尼亚州尤为常见（该州 AUI 总体排名第三），金融相关任务在纽约尤为常见（排名第四）。[^1] 即使在人口调整 Claude 使用量较低的州，如夏威夷，使用也与经济结构紧密相关：夏威夷人使用 Claude 帮助处理旅游相关任务的频率是美国其他地区的两倍。我们的[交互式网站](http://anthropic.com/economic-index)包含了许多其他类似统计。

**Claude 使用趋势**

自 2024 年 12 月以来，我们一直在追踪人们如何使用 Claude。我们采用一种隐私保护的[分类方法](https://www.anthropic.com/research/clio)，将匿名对话记录归类到 O*NET 定义的任务组中。O*NET 是美国政府的工作分类数据库，对职业及其相关任务进行了系统分类。[^2] 借此，我们既可以分析人们交给 Claude 的任务自去年以来发生了怎样的变化，也可以分析人们选择的协作方式——他们对 Claude 工作的监督和参与程度——发生了怎样的变化。

### 任务分布

自 2024 年 12 月以来，计算机和数学类用途在我们的分类中占据主导，约占对话的 37%—40%。

但很多方面发生了变化。在过去九个月中，我们看到"知识密集型"领域的持续增长。例如，教育教学类任务增长了超过 40%（从所有对话的 9% 上升到 13%），物理和社会科学相关任务的份额增加了三分之一（从 6% 上升到 8%）。与此同时，传统商业任务的相对频率有所下降：管理相关任务从所有对话的 5% 下降到 3%，商业和金融运营相关任务的份额减半，从 6% 下降到 3%。（当然，按绝对量计算，每个类别的对话数量仍然显著增长。）

总体趋势虽然存在噪声，但大致规律是：随着一国人均 GDP 的提高，Claude 的使用会*偏离*计算机和数学职业组的任务，转向更多样化的其他活动，如教育、艺术与设计、办公室与行政支持，以及物理和社会科学。

尽管如此，软件开发仍是我们追踪的每一个国家中最常见的用途。美国的情况类似，尽管样本量限制了我们更详细地探索任务组合如何随采纳率变化的可能性。

### 交互模式

如我们[此前讨论](https://www.anthropic.com/news/the-anthropic-economic-index)的，我们大致将任务分为*自动化*（AI 在最少用户输入下直接完成工作）和*增强*（用户与 AI 协作完成工作）两类。我们进一步将自动化分解为*指令式*和*反馈回路*交互——指令式对话涉及最少的人类交互，反馈回路任务中人类将现实世界的结果传回给模型。增强则分解为*学习*（询问信息或解释）、*任务迭代*（与 Claude 协作工作）和*验证*（请求反馈）。

自 2024 年 12 月以来，我们发现指令式对话的占比从 27% 急剧上升到 39%。其他交互模式（特别是学习、任务迭代和反馈回路）的占比因此略有下降。这意味着自动化（49.1%）首次在总体上超过了增强（47%）。一种可能的解释是 AI 正在迅速赢得用户的信任，并越来越多地负责完成复杂工作。

这可能是模型能力提升的结果。（2024 年 12 月我们首次收集经济指数数据时，最新的 Claude 版本是 Sonnet 3.6。）随着模型更好地预测用户需求并产出高质量工作，用户可能更愿意在首次尝试时就信任模型的输出。

令人意外的是，在人均 Claude 使用量较高的国家，Claude 的用途趋向于增强，而低使用量国家的人更倾向于自动化。在控制任务组合的情况下，人口调整后的 Claude 使用量每增加 1%，与自动化约减少 3% 相关。同样，人口调整 Claude 使用量的增加与*远离*自动化的转变相关，而非趋近。

我们尚不确定原因何在。可能是因为各国的早期使用者更放心让 Claude 自动化任务，也可能归结于其他文化和经济因素。

**企业用户**

使用与 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 对话相同的隐私保护[方法](https://www.anthropic.com/research/clio)，我们首次对 Anthropic 自有 API 客户子集的交互进行了抽样分析。[^3] API 客户通常为企业和开发者，他们使用 Claude 的方式与通过 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 访问的用户截然不同：他们按 token 付费而非固定月费，并且可以通过自己的程序发起请求。

这些客户对 Claude 的使用尤其集中在编码和行政任务上：我们样本中 44% 的 API 流量映射到计算机或数学任务，而 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 上这一比例为 36%。（巧合的是，约 5% 的 API 流量专门聚焦于开发和评估 AI 系统。）这一集中度被教育职业（API 中为 4%，而 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 上为 12%）以及艺术与娱乐（5% 对比 8%）相关对话的较低比例所抵消。

我们还发现，我们的 API 客户使用 Claude 自动化任务的频率远高于 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 用户。API 对话中有 77% 显示自动化模式，其中绝大多数是指令式的，仅有 12% 显示增强模式。在 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 上，两者几乎平分秋色。这可能有重大的经济意义：在过去，任务的自动化往往伴随着大规模的经济转型和显著的生产率提升。

最后，考虑到 API 使用的付费方式，我们还可以探索任务成本的差异（由消耗的 token 数量差异导致）是否影响企业选择"购买"哪些任务。我们发现价格与使用量之间存在*正*相关：成本更高的任务类别往往使用更频繁。这表明，对模型基础能力及其创造的经济价值，企业在做决策时看得比任务完成的直接成本更重要。

**结论**

经济指数旨在提供 AI 如何影响人们工作和经济的早期实证评估。到目前为止我们发现了什么？

在本报告涵盖的各项指标中，AI 的采纳呈现出显著的*不均衡性*。高收入国家的人更可能使用 Claude，更倾向于寻求协作而非自动化，更可能在编码之外追求更广泛的使用场景。在美国内部，AI 使用似乎受到当地经济主导产业的强烈影响，从科技到旅游。企业比消费者更愿意赋予 Claude 自主权和代理权。

除了不均衡这一事实之外，过去九个月中指令式自动化在 [Claude.ai](http://claude.ai/redirect/website.v1.33a1fedf-5747-4b68-9502-0e04849443fe) 对话中变得越来越普遍，这一点尤为值得关注。人们使用 Claude 的方式显然仍在被定义中：我们仍在集体决定对 AI 工具有多大信心，以及应该赋予它们多大责任。不过，到目前为止，似乎我们正在越来越适应 AI，并愿意让它为我们工作。我们期待随着时间推移再次审视这一分析，观察用户的选择会落在何处——或者，确切地说，*会不会*真正稳定下来。
