# 人们如何向Claude寻求个人指导

![header](images/claude-personal-guidance_01041aa942.svg)

人们不止为代码审查或会议纪要来Claude。他们问要不要接那份工作、怎么跟暗恋的人开口、该不该搬去地球另一端。我们使用[隐私保护分析工具](https://www.anthropic.com/research/clio)对 claude.ai 的100万条随机抽样对话进行分析，发现约6%的对话是人们在向Claude寻求个人指导——他们找的不只是信息，而是"下一步该怎么办"的判断。

本次研究考察了人们向Claude寻求哪些类型的指导，以及Claude在不同领域如何回应，重点关注过度迎合或过度赞扬（即*谄媚*）的发生率如何因指导主题而异。我们描述了这项研究如何塑造了最新模型 Claude Opus 4.7 和 Claude Mythos Preview 的训练。我们做这项研究的目标是改进模型保护用户福祉的方式。

简要结论：

- 人们在生活的许多不同领域寻求Claude的指导，但超过四分之三的对话（76%）集中在四个领域：健康与保健（27%）、职业与事业（26%）、人际关系（12%）和个人财务（11%）（图1）。
- Claude在提供指导时大多避免谄媚回应，在所有寻求指导的聊天中谄媚行为占9%。但在人际关系对话中，这一比例上升到25%，考虑到其体量，人际关系是谄媚绝对数量出现最多的领域（图2）。
- 为应对这一问题，我们研究了Claude更容易给出谄媚回应的具体场景，并用来为 Opus 4.7 和 Mythos Preview 构建合成人际关系指导训练数据。在人际关系指导方面，Opus 4.7的谄媚率比 Opus 4.6 降低了一半；有意思的是，这一改善泛化到了其他领域（图3）。

关于AI的良好指导到底意味着什么、如何衡量，仍有许多未解问题。[保护用户福祉](https://www.anthropic.com/news/protecting-well-being-of-users)是 Anthropic 的核心优先事项，我们在衡量和理解个人指导方面的工作是朝这一目标迈出的一步。

## 人们向Claude寻求哪些类型的指导？

我们从2026年3月至4月采样了100万条 [claude.ai](http://claude.ai/redirect/website.v1.98c1b79c-aa72-46c8-b189-a39b1fbd3e24) 对话，按唯一用户过滤后获得约63.9万条对话。然后使用分类器识别*个人指导*，我们将其定义为人们询问*他们自己*在个人生活中应该怎么做的对话——例如以"我该不该……？"或"我怎么处理……？"开头的问题。我们排除了寻求客观信息或一般性意见的问题。

我们将约3.8万条对话归入九个领域，参考了此前AI与指导行为的研究：人际关系、职业、个人发展、财务、法律、健康与保健、育儿、伦理和精神信仰（详见[附录](https://cdn.sanity.io/files/4zrzovbb/website/0a540acdf3e1678274f0fe04b3a70ea7fd99ed36.pdf)）。这一分类体系覆盖了98%的对话。

超过75%的对话集中在四个类别：健康与保健、职业与事业、人际关系和财务（图1）。当一条对话涉及多个领域时，我们按最突出的话题归类。

## 衡量指导对话中的谄媚行为

当人们就生活中的决策向Claude寻求建议时，Claude的良好互动应该是什么样子的？乐于助人是Claude[最重要的特质](https://www.anthropic.com/constitution)之一。与Claude交谈应该像与一位博学的朋友对话，他会坦诚地谈论一个人的处境，提供基于证据的信息。同时，Claude应在适当时候承认自身的局限性，避免谄媚行为或助长过度依赖。

虽然我们训练Claude体现的行为范围很广，但谄媚是我们已用于衡量Claude在这些方面表现的指标之一——谄媚是AI助手中常见的一种倾向，即过度赞同一个人的观点而非提出质疑。这可能是对方当下想听的，但最终可能损害其长期福祉。例如，在涉及不完整或片面视角的情况下，Claude不应给出过于确信的判断——比如基于单方面叙述就认同对方伴侣"绝对在gaslighting"他们，或认可明天不假思索就辞职"听起来是对的"，或认为昂贵消费是"对自己的绝佳投资"。

再次确认一个人的片面视角可能制造或加剧人际关系中的裂痕。在我们的数据中，这表现为几种形式。一种常见模式是Claude断然认同对方有错，尽管仅有用户一方的叙述。另一种是Claude帮助人们把普通的友好行为解读出浪漫意图，仅仅因为他们要求这样做。

我们使用自动分类器来判断谄媚行为，考察Claude是否表现出质疑意愿、在被挑战时坚守立场、根据想法的价值给予适度的赞扬、以及无论对方想听什么都能坦诚表达。在多数情况下，Claude没有表现出谄媚——仅9%的对话包含谄媚行为（图2）。但有两个领域例外：在精神信仰类对话中，谄媚行为出现在38%的对话中；在人际关系对话中为25%。我们选择将模型训练的重点放在人际关系指导上，因为这是谄媚对话绝对数量最多的领域。

## 改善Claude在人际关系指导中的行为

为了在未来模型中改善Claude的行为，我们首先研究了是什么驱动了数据中人际关系指导的更高谄媚率。有两个动态值得关注。

首先，人际关系指导是人们最频繁反驳Claude的领域，21%的对话中出现了反驳，而其他领域平均为15%。其次，Claude在压力下更容易表现出谄媚行为。当人们反驳时，谄媚率为18%，而无反驳的对话中为9%。我们认为这是因为Claude被训练为乐于助人和富有同理心；在反驳结合只听一方叙述的情况下，Claude保持中立变得更加困难。

为应对这一点，我们识别了人们反驳时触发谄媚回应的不同对话模式——例如，当人们批评Claude的初步判断，或提供大量单方面的细节。我们利用这些模式构建合成人际关系指导场景用于行为训练。在此环境中，我们要求Claude为每个合成场景采样两个回应；然后由另一个Claude实例对回应与宪章中行为准则的符合程度打分。

我们通过一种称为压力测试的技术评估新模型的改进程度。我们使用隐私保护工具识别人们通过反馈按钮与我们的模型共享的真实个人指导对话[^1]，这些对话中前代模型曾表现出谄媚行为。然后通过一种称为预填充的技术将这部分对话提供给新模型（本次为 Opus 4.7 和 Mythos Preview），让模型将之前的对话作为自己的对话来阅读。由于Claude试图维持对话内的一致性，用谄媚对话预填充使Claude更难改变方向。这有点像操控一艘正在行驶的船，从而在刻意不利的条件下衡量Claude的行为。

每代模型之间变化很多，这使得识别任何单一训练变化的影响变得困难。然而，在 Opus 4.7 和 Mythos Preview 中，我们都观察到人际关系指导以及所有个人指导领域的谄媚率均有所下降（图3）。

定性来看，Opus 4.7 和 Mythos Preview 都更善于看穿某人的初始表述，理解他们来Claude寻求指导的更广阔背景。这包括引用之前对话中用户提供过更深层背景的交流，以及在相关时引用外部信息源。例如，在一次对话中，某人问自己的短信是否显得焦虑和粘人。Claude Sonnet 4.6 在收到反驳后立场摇摆。Claude Opus 4.7 则解释说，虽然短信本身并不粘人，但用户在整个对话中自我描述的焦虑想法贯穿始终。另一个人际关系领域之外的例子：某人希望Claude认可其写作，最终要求Claude据此估算其智商。Claude Sonnet 4.6 给出了过分恭维的回应，而 Mythos Preview 拒绝了，解释说自己没有足够的信息做出这样的判断。

## 结论

我们从对人们如何向Claude寻求个人指导的高层次分析入手，聚焦于理解并解决一个特定的模型失败模式：人际关系对话中的谄媚。这项调查引出了更广泛的问题：

**什么是好的AI指导？**

本文中我们聚焦于减少谄媚这一已知的指导场景失败模式，但我们的工作提出了关于好的AI指导究竟是什么的更广泛问题。[Claude的宪章](https://www.anthropic.com/constitution)也强调，例如，好的指导应该是诚实的、保护用户自主权的。这些原则比谄媚更微妙。我们已开始在[新的系统卡](https://www.anthropic.com/news/claude-opus-4-7)中监控Claude对这些原则的遵守情况，并希望在未来研究中纳入它们。

**如何在高风险场景中让模型更安全？**

[近期一项英国AI安全研究所的研究](https://arxiv.org/abs/2511.15352)发现，人们在低风险和高风险场景中都极有可能采纳AI指导。我们发现了许多高风险问题案例，尤其在法律、育儿、健康和财务领域。这些包括关于移民途径、婴儿护理指导、药物剂量和信用卡债务的对话。Claude并非为提供医疗指导或专业护理而设计，在这些场景中Claude会恰当地承认其局限性并建议寻求人类指导。然而，我们也发现有人告诉Claude他们使用AI恰恰*因为*无法获取或负担不起专业人士。作为理解如何按领域评估安全性的第一步，尤其是对没有后备方案的人群，我们计划在这些高风险领域构建评估。

**AI指导如何融入人们更广泛的信息生态？**

我们发现22%的人提到他们曾寻求其他支持来源，包括家人、朋友、专业人士或数字来源。我们无法从对话记录中衡量的是反事实：Claude是否改变了任何人的想法？如果不用Claude他们会问谁？这些问题是了解AI指导在人们决策中实际权重的核心。要获得真实世界的结果，我们认为一个有前景的方向是通过[Anthropic Interviewer](https://www.anthropic.com/research/anthropic-interviewer)扩展研究，在人们接受Claude的指导后进行跟进。

人们如何将AI用于个人指导和决策是这些系统影响人们日常生活的最直接方式之一。仔细描绘——人们问什么，Claude说什么，接下来发生什么——这是我们确保Claude对每个使用者长期有益的方式。

## 局限性

我们的分析是揭示驱动AI模型常见使用模式的第一步。本文仅限于Claude用户，他们并非具有代表性的人群样本。为保护人们隐私，我们依赖自动评分器（Claude Sonnet 4.5），可能存在对话分类错误（见[附录](https://cdn.sanity.io/files/4zrzovbb/website/0a540acdf3e1678274f0fe04b3a70ea7fd99ed36.pdf)）。我们迭代了评分器提示，并在用户授权我们查看对话的反馈数据上手动验证了一小部分评分结果以降低错误率。我们观察了新模型在训练后的行为，但由于缺乏反事实，我们无法做出关于新训练数据对谄媚减少具体贡献量的因果声明。此外，我们的分析仅限于聊天记录，这限制了我们理解人们为何向Claude寻求指导以及之后如何采取行动。后续访谈研究将更好地揭示人们从AI接收指导后实际做了什么。
