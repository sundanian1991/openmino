04 / Design Principles
设计平衡三原则

在传承与创新之间找到张力，让 111 年的品牌积淀与当代设计语言共存。

历史与未来
111 年品牌积淀 vs 现代几何图形
体现 经典 H + 点阵变体
自然与科技
户外功能属性 vs 技术创新
体现 自然场景 + 数据化图表
传统与现代
北欧设计基因 vs 功能主义当代诠释
体现 极简排版 + 高饱和碰撞
05 / Applications
应用规范

设计系统在品牌触点中的落地方式，覆盖包装、印刷、空间、数字和户外五大场景。

包装
模块化可回收包装。镂空品牌符号作为视觉识别。信息分层：环保材质 > 功能参数 > 品牌。
印刷
极简画册 + 镂空工艺 + 荧光色内页。"透过品牌看自然"。
空间
自然材质 + 工业金属陈列基底。动态光影。产品功能可视化陈列。
数字媒介
响应式全终端。自然场景 x 产品功能交互。地理坐标植入强化户外属性。
户外广告
三联幅：产品特写 + 品牌符号 + 自然场景。
产品标签
产品细节特写与功能说明结合。环保材质属性与功能参数优先展示。
