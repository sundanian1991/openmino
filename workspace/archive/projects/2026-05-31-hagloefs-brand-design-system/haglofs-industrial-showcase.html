PRINCIPLES

8:1 minimum ratio. The gap between the largest and smallest text element on any screen must be at least 8:1. This creates cinematic tension that flat hierarchies cannot achieve.

Three-font stack. Georgia is display authority. Inter is body readability. JetBrains Mono is data precision. Never mix roles.

Gray before color. Four gray levels create visual importance before any color is applied. Color is an event, not a default. The olive accent earns its presence by rarity.

Decorative numbers. 120px Georgia at 5% opacity creates typographic texture without competing with content. Always positioned absolute, always pointer-events none.

THE FORMULA

Heavy. One oversized Georgia number dominates the viewport. It stops the eye and establishes the section's authority. 96px minimum.

Medium. Segmented block bars provide comparative context. They translate the headline number into dimensions the viewer can compare.

Light. Spec tables and stat rows fill the data layer. JetBrains Mono labels, Inter values. Dense but scannable.

Connectors. Nameplate labels mark each layer. Seam dividers separate them. The 2:1 asymmetric grid ensures the data side always has more visual weight than the explanation side.

SECTION 08

Design Principles

The rules that separate industrial-grade design from catalog templates. Each principle is enforced through the component library.

× Avoid
  • Even font sizes creating visual flatness
    16/18/20/22px hierarchy is unreadable at a glance
  • Centered symmetric layouts
    Perfect centering feels static and unconsidered
  • Color for hierarchy instead of gray-scale
    Color should be an event, not the default visual path
  • Shadow and gradient effects
    Depth comes from contrast ratio, not visual decoration
· Prefer
  • 8:1 font-size jumps for cinematic contrast
    120px hero to 14px body creates immediate visual tension
  • Asymmetric unbalanced grids (2:1 ratio)
    Imbalance creates movement and visual hierarchy
  • Gray-scale as primary hierarchy tool
    4 gray levels establish importance before any color
  • Flat surfaces with border separation
    Clean 1px borders create structure without decoration
