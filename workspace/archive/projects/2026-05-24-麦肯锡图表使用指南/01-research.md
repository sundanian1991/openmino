## 触发条件

以下场景打开这个工具：
- 需要为汇报/报告/邮件制作策略级图表
- 有 Excel 数据需要快速可视化
- 需要麦肯锡/咨询风格的标准化图表
- 需要高清图表（4K PNG / SVG）嵌入 PPT 或文档

## 执行步骤

### 第一步：选图表

按业务场景选图，不按个人喜好。

| 你要表达什么 | 选什么图 | 典型场景 |
|-------------|---------|---------|
| 从A到B的过程拆解 | 瀑布图 | 收入桥接、利润分解 |
| 各部分对总体的贡献 | 堆叠柱状图 | 区域贡献、渠道占比 |
| 两组/三组数据的同期对比 | 分组柱形图 | YoY/QoQ 对比 |
| 实际 vs 目标的偏差 | 差值箭头图 | 预算偏差、KPI 达成 |
| 两个群体的对比 | 蝴蝶图 | 性别/竞品/前后对比 |
| 战略四象限 | SWOT | 竞争分析 |
| 趋势+另一指标的叠加 | 折线+柱形 | 销售额+增长率 |
| 横向排名（标签长） | 分组条形图 | 部门/供应商排名 |
| 多维度能力评估 | 雷达图 | 供应商能力对比 |
| 市场定位矩阵 | 气泡矩阵 | 波士顿矩阵 |
| 资金/流量流向 | 桑基图 | 成本结构、流量路径 |
| 单一指标趋势 | 折线图 | 月度走势 |
| 单一指标排名 | 柱状图 | 部门/产品排名 |
| 构成占比 | 饼图 | 业务构成 |
| 两变量相关性 | 散点图 | 客流量 vs 转化率 |
| 三维对比 | 分组气泡图 | GDP+增长率+人口 |
| 层级占比 | 旭日图 | 全球→区域→国家 |
| 占比方块图 | 矩形树图 | 预算分配、品类占比 |
| 渠道贡献趋势 | 堆叠面积图 | 流量来源月度变化 |

### 第二步：准备数据

**格式规则**：
- 第一行是表头，后续是数据行
- 支持 CSV（逗号分隔）和 TSV（Tab 分隔）
- 直接从 Excel 复制粘贴即可（Excel 复制 = Tab 分隔）

**每种图的数据模板**：

**瀑布图**：
```
项目,数值
2023年营收,100
价格效应,20
销量效应,-10
产品结构优化,5
汇率影响,-8
2024年营收,0        ← 末行填 0，自动计算总量
```

**差值箭头图**：
```
项目,实际值,预算值
销售额,120,100
成本,85,80
```
标签自动显示：`120\n(+20.0%)`

**蝴蝶图**：
```
类别,男性用户,女性用户
18-24岁,35,40
```
两侧都填正数，代码自动处理对称。

**SWOT**（唯一纯文本格式）：
```
S: 优势1, 优势2, 优势3
W: 劣势1, 劣势2
O: 机会1, 机会2
T: 威胁1, 威胁2
```

**桑基图**：
```
来源,去向,数值
总收入,主营业务成本,40
总收入,毛利润,60
```
自动提取节点，不需要单独定义。

**气泡矩阵（波士顿）**：
```
产品,市场份额(%),年增长率(%),营收规模
产品A,25,15,100
```

### 第三步：调参到汇报级

默认参数就能用，但汇报级需要调整这些：

| 参数 | 默认值 | 汇报建议 | 调整时机 |
|------|--------|---------|---------|
| 画布尺寸 | 1000×600 | PPT: 1920×1080 或 1280×720 | 嵌入 PPT 前调整 |
| 标题字号 | 20 | 18-22 即可 | 画布小时降到 16 |
| 图例位置 | 底部居中 | 底部居中（最稳） | 数据系列 ≤2 时可选右上 |
| 网格线 | 虚线 | 虚线（专业感） | 极简风格选"无" |
| 数值标签 | 显示 | 关闭（如果数据太密） | >8 个柱体时建议关 |
| 柱体宽度 | 50% | 40-60% | 系列多时缩小 |
| 分类间距 | 20% | 15-30% | 数据密降到 10% |
| X轴旋转 | 0° | 标签长时选 45° | 标签重叠就转 |
| 排序 | 原始 | 降序（排名场景） | 展示 Top N 时 |
| Y轴刻度 | 自动 | 手动设起始=0（柱状图必须） | 避免视觉误导 |

**Y轴起始=0 是柱状图铁律**：不归零会放大差异，汇报时被质疑专业性。

### 第四步：导出

| 输出场景 | 格式 | 画布尺寸 | 操作 |
|---------|------|---------|------|
| PPT 嵌入 | 4K PNG | 1920×1080 | 点"导出 4K PNG" |
| 印刷/放大 | SVG | 任意 | 点"导出 SVG"，矢量无损 |
| 邮件/即时通讯 | 4K PNG | 800×500 | 缩小画布后导出 |
| 报告文档 | SVG | 1000×600 | SVG 嵌入 Word 可无损缩放 |

## 边缘场景

| 情况 | 怎么做 |
|------|--------|
| 数据粘贴后图表空白 | 检查分隔符：确保用逗号或 Tab（不能混用） |
| 标签重叠 | X轴旋转调 45° 或 90° |
| 图例遮住数据 | 图例位置从"右上"改"底部居中" |
| 柱体太窄 | 柱体宽度调到 60-70% |
| 多系列图太挤 | 分类间距降到 5-10%，或减少系列数 |
| Y轴起点不是0 | Y轴"起始"填 0 |
| 导出模糊 | 用 4K PNG（不是普通截图） |
| SWOT 不显示 | 检查是否用了 S:/W:/O:/T: 前缀 |
| 瀑布图总量不对 | 确保末行数值填 0 |
| 桑基图乱线 | 检查来源/去向名称是否拼写一致 |

## 供应商管理场景速查

| 汇报主题 | 推荐图表 | 数据准备 |
|---------|---------|---------|
| 供应商业绩排名 | 柱状图（降序） | 供应商,得分 |
| 业绩同比对比 | 分组柱形图 | 供应商,2024年,2025年 |
| 成本结构拆解 | 瀑布图 | 成本项目,金额 |
| 预算达成率 | 差值箭头图 | 供应商,实际,预算 |
| 供应商能力对比 | 雷达图 | 指标,供应商A,供应商B |
| 市场定位 | 气泡矩阵 | 供应商,份额,增长率,规模 |
| 成本流向 | 桑基图 | 来源,去向,金额 |
| SWOT 分析 | SWOT | S/W/O/T 纯文本 |

## 依赖

- 浏览器（Chrome/Edge 推荐）
- 无需安装，双击 HTML 即可打开
- 需要网络（CDN 加载 Tailwind + ECharts + Google Fonts）

## 成功指标

| 指标 | 基线 | 目标 | 来源 |
|------|------|------|------|
| 单张图表制作时间 | 10-15 分钟（Excel+PPT 手工） | ≤3 分钟 | 实际操作计时 |
| 汇报图表专业度 | 格式不统一 | 麦肯锡风格一致 | 主管反馈 |
| 数据到图表的转化率 | 需要手动调格式 | 粘贴即出图 | 操作体验 |

## 开放问题

- 是否需要增加配色方案切换（非麦肯锡风格）？
- 是否需要批量导出（多图同时生成）？
- 是否需要仪表板模式（多图排版在一页）？
