# Director's Treatment

## Director Brief
- Visual thesis: 一场商业变革的调查档案——数据即证据，时间线即叙事
- Signature technique 1: 数字冲击开场（The Social Network 开场酒吧/数据场景 → 用最大数据抓住注意力）
- Signature technique 2: 证据墙并排（Zodiac 的调查板 → 多个数据源并列呈现，让观众自己发现规律）
- Signature technique 3: 冷峻收尾（Gone Girl 结局 → 不给鸡汤，只给判断）
- Motion rules: Intersection Observer 驱动，每个区块不同入场方式，禁止 fadeUp 超过 2 次
- Typography rules: 标题用大字重无衬体，数据用等宽字体，正文用易读衬底

## Site Cinematic Grammar
- Page-shell logic: 单页长卷，无导航，自然滚动驱动叙事
- Navigation posture: 无顶部导航——叙事本身是唯一的导航
- Framing discipline: 左对齐，max-width 900px，像调查报告
- Density cadence: 数据密集段（时间线/对比表）→ 深度分析段（核心逻辑/洞察）→ 呼吸段（分割线/引用）
- Recurring material layers: 扫描线纹理（全页）、青绿点缀色（仅用于数据/关键判断）
- Allowed composition families: evidence wall、data punch、timeline strip
- What may repeat: 表格结构、等宽数据展示
- What must vary page to page: 入场动画类型、区块视觉重心
- Demo uniqueness guardrail: 不出现圆角卡片堆叠、渐变 hero、通用 SaaS 布局

## Page Arc

### Page: Home
- Page-role scene: 调查档案——你正在翻阅一份关于 AI 定价变革的完整调查报告
- Page scene thesis: 这不是一篇公众号文章，而是一份可以被反复翻阅的商业案例档案
- One big idea: 数字本身会说话——让数据排列揭示规律，而非用形容词告诉观众结论
- Hero dominance statement: 用一个真实的、震撼性的数字对比开场（$15,000 API 费 → $800 套餐费，节省 93%），比任何标题都更有冲击力
- Restraint statement: 不做任何装饰性动画、不用渐变、不用玻璃拟态——只有数据、文字和精确的排版
- Material thesis: 深色背景 + 扫描线纹理 + 青绿数据高亮，像暗房里的调查报告
- Typography thesis: 标题大字重冲击 + 等宽数据精确 + 正文易读，三层字体各司其职
- Narrative arc: B1 Cold Open → B13 Flashback → B8 Evidence Wall → B9 Data Bombardment → B10 Deep Dive → B18 Confrontation → B7 Encounter → B10 Deep Dive → B8 Evidence Wall → B23 Cliffhanger → B24 Loop
- Signature composition: 五计量体系对比——不是普通表格，而是五个并排的"计量面板"，每个面板有自己的颜色标识和实际等效值
- Grid fallback test: 如果简化为通用表格，会失去"计量体系本身就是战场"这个核心叙事
- Shared system holdback: 全局间距和字体系统最后定义，不先于页面合成
- UI exposure guardrail: 无 UI chrome，叙事即界面
- What this page must not inherit from previous demos: 无居中渐变 hero、无圆角 premium cards、无通用 SaaS 三段式
