# AI 产品定价模式演变：从 Coding Plan 到 Token Plan

> 2026-04-24 | 商业案例研究 | 知识沉淀

## 一句话结论

AI 大模型定价正从"卖工具订阅"演进为"卖算力资源包"，精确重演了电信行业从话费套餐到流量套餐的 20 年历史。**Coding Plan 是过渡产品，Token Plan 是终态。**

## 框架层：为什么这么演变

### 类比：电信套餐 20 年演进史

| 电信阶段 | AI 对应 | 核心逻辑 |
|----------|---------|----------|
| 2009 话费套餐（充 100 用不完） | 2024 API 按量（百万 Token 几十元） | 供给过剩，用户感知不到成本 |
| 2013 流量套餐（9.9 几 G） | 2025 Coding Plan（¥29-200/月） | 用量激增，包月降低感知 |
| 2017 "不限量"达量降速 | 2026 分层套餐（极速版/普通版） | 精细化分层，按速度定价 |
| 2020 按 GB 精打细算 | 2026+ Token 精细计量 | 用户算账，厂商利润回归 |

### 五力分析

- **供应商议价力**：GPU 算力刚性成本，NVIDIA 占 85-90%，成本传导到 Token
- **买方议价力**：7+ 平台可选，切换成本低（换 API Key），议价力强
- **替代品**：开源模型（DeepSeek、Qwen）+ 本地部署，压制定价上限
- **新进入者**：门槛高（百亿融资 + 算力），但小米 MiMo 等仍在进入
- **行业竞争**：极高。7 家在同一池子打价格战

### 三层演进逻辑

```
成本驱动：算力成本 → Token 成本 → 套餐定价
竞争驱动：价格战 → 补贴圈用户 → 收紧利润
价值驱动：卖 Token → 卖工作流 → 卖生态
```

## 洞察层：能提炼什么

### 1. 计量单位即定价权

> 谁定义计量单位，谁就掌握定价权。

五种计量体系并存是最大的信息壁垒：

| 计量方式 | 采用者 | 用户感知 | 透明度 |
|----------|--------|----------|--------|
| API 请求次数 | 阿里/火山/腾讯/百度 | 数字大，但 1 次提问 = 5-30 次调用 | 低 |
| Prompt 次数 | 智谱/MiniMax（早期） | 1 Prompt ≈ 15-20 次模型调用 | 中 |
| Token 计量 | Kimi/MiniMax（新版） | 最透明，但用户难预估消耗 | 高 |
| Credit 计量 | Cursor/小米 MiMo/阿里百炼 Token Plan | 统一 Credit 池，不同模型不同倍率 | 中 |
| 时间窗口配额 | 智谱 GLM（5h）/Claude Code（5h+7d） | 限制节奏而非总量 | 中 |

**套餐使用量差异**（入门档 ¥40-50 区间）：

| 平台 | 月费 | 额度 | 约等于 |
|------|------|------|--------|
| 智谱 GLM Lite | ¥49 | ~80 prompts/5h, 2400/月 | 每天 80 次提问 |
| MiniMax Starter | ¥29 | 40 prompts/5h，无周月限 | 连续每 5h 40 次 |
| 阿里百炼 Lite | ¥40 | 18000 API 请求/月 | ~20-120 次提问/天 |
| 腾讯云 Token Lite | ¥39 | 3500 万 Token | ~70 轮深度问答 |
| Cursor Pro | $20 | $20 Credit 池 ≈ 225 Sonnet 请求 | Auto 模式无限 |

同样 ¥40-50/月，不同计量方式下实际可用量差异巨大。Agent 场景消耗爆炸——8 个月 Claude Code 消耗 100 亿 Token（API 价 $15,000+），Max plan 只花 $800（省 93%）。

**可复用模式**：新产品定价时，计量单位选择决定了用户感知和利润空间。模糊计量利于早期获客，透明计量利于长期信任。趋势是从 API 次数 → Prompt → Token/Credit → 统一 Credits，方向是越来越透明。

### 2. 补贴-收紧-分层的三段式节奏

| 阶段 | 目的 | AI 案例 |
|------|------|---------|
| 补贴 | 抢用户规模 | 2024-2025：百万 Token 几毛钱，首月 7.9 元 |
| 收紧 | 测试价格弹性 | 2025 H2：限购、停售低档、限时抢购 |
| 分层 | 利润最大化 | 2026：极速版 ¥98-899，Allegro ¥559 |

**可复用模式**：进入新市场先补贴跑量，再收紧测试弹性，最后精细分层。所有互联网赛道都走这三步。

### 3. 从"入口"到"生态"的升级

MiniMax 的 Token Plan 不只卖编程，打包了图片、语音、音乐、视频。¥29 买到的是"AI 多媒体工作室"。

**可复用模式**：用高频刚需场景（编程）获客，用低频高价值场景（视频/音乐）提高 ARPU。

### 4. 国内外定价逻辑的根本差异

- **国内**：7 家在一个池子打价格战 → "你买到的便宜，是因为有人在替你卷"
- **海外**：面对 Claude/GPT/Gemini → 需要证明"我不比 Claude 差"，比谁更强

### 5. Agent 时代的成本结构剧变

- 70% coding agent token 是浪费（DEV Community）
- 8 个月 Claude Code 消耗 100 亿 Token，API 价 $15,000+，Max plan 只花 $800
- Cursor 2025-06 转 credit-based，重度用户一周超支 $350 → 退款风波
- 订阅费是地板不是天花板，实际账单 2-5 倍

**含义**：Agent 时代用户成本指数级增长，定价模式必须适应这个现实。

### 6. DeepSeek V4 的定价冲击（2026-04-24 发布）

| 模型 | 总参数 | 激活参数 | 输入（缓存） | 输入（未缓存） | 输出 |
|------|--------|----------|-------------|---------------|------|
| V4-Pro | 1.6T | 49B | 1 元/百万 | 12 元/百万 | 24 元/百万 |
| V4-Flash | 284B | 13B | 0.2 元/百万 | 1 元/百万 | 2 元/百万 |
| 对比 GPT-5 | - | - | $5/百万 | - | $15/百万 |

V4-Pro 长上下文场景：单 token 推理计算量仅为 V3.2 的 27%，KV Cache 占用降至 10%。

关键信号：DeepSeek 官方明确说"受限于高端算力，目前吞吐有限，预计下半年昇腾 950 超节点批量上市后价格大幅下调"。**芯片供给直接绑定定价天花板。**

### 7. OpenClaw：Token 消耗的催化剂

OpenClaw（GitHub 355K stars）作为开源 AI Agent 框架，直接催热了 Coding Plan 需求：
- MiniMax M2 token 用量：2 月 vs 12 月增长 6 倍，Coding Plan 消耗增长 10 倍+
- 阿里云算力告急，限量供应 → Lite 档 4-13 停售的直接原因
- Anthropic 2026-04 从标准订阅移除 OpenClaw 访问 —— 成本撑不住了

Agent 自动循环调用让"不限量订阅"不可持续，倒逼厂商转向 Token 精细计量。

### 9. 芯片供给线：算力成本的底层变量

**英伟达在中国的退缩：**
- 市占率：2024 年 66% → 2025 年 54% → 跌破 60% 关口
- 2023 年禁止顶级芯片出口 → 2025-04 全面禁止 AI GPU 对华出口
- 后续松绑 H20/H200，但严苛采购限制，订单落地滞后

**华为昇腾的崛起：**
- 2025 年交付 81.2 万片，市占率 ~20%，国产第一
- 950PR：FP4 算力 1.56 PFLOPS（H20 的 2.87 倍），定价 ~7 万元（H200 的 1/3），4 月量产
- 950DT：2026Q4，面向训练+推理 Decode，144GB HBM
- 路线图：950PR(26Q1) → 950DT(26Q4) → 960(27Q4) → 970(28Q4)
- 2026 目标：AI 芯片市占率 50%，出货 75-100 万片，营收超 800 亿

**国产芯片格局：**
- 2025 年自给率 41%（英伟达 55%，AMD 4%）
- 伯恩斯坦预测 2027 年国产化率 17% → 55%
- 华为自研 HBM（HiBL/HiZQ），绕开供应链限制
- DeepSeek V4 同时适配华为 + 寒武纪，Day 0 支持

**含义：** 算力供给不再被英伟达卡脖子 → 国内厂商成本结构和海外产生分化 → 定价策略可以更激进。DeepSeek V4 官方明确说"下半年昇腾 950 批量上市后价格大幅下调"，芯片供给直接绑定定价天花板。

### 11. 跨越鸿沟：AI 从实验到生产

Gartner：90% 企业推进 AI 实验，仅 41% 成功落地生产环境，预计 2027 年 40% Agentic AI 项目因无法证明 ROI 被终止。真格基金戴雨森判断 Agent 产品仍处 early market，跨越鸿沟"依然需要几年的时间"。

**对定价的含义：** 企业在实验阶段愿意尝试低价 Coding Plan → 进入生产环境后需要 SLA、稳定性、合规性 → 这是 Token Plan 企业版溢价的基础。跨越鸿沟的关键不是模型能力，而是"可信赖度"。

## 未来预判

### 趋势 1: Token 成本将持续下降

驱动力：
- 国产芯片放量（华为 950 年内 75 万片）→ 算力成本下降
- DeepSeek V4 开源 → MoE 架构效率革命（1.6T 参数只激活 49B）
- 推理专用芯片竞争（英伟达 Rubin CPX / 谷歌 TPU v8 / 华为 950PR）→ 推理成本加速下降

结果：Token Plan 的价格战会继续，但利润空间来自"服务溢价"而非"算力差价"。

### 趋势 2: 国产芯片替代加速改变供给格局

- 英伟达市占率持续下降（66% → 54% → 2026 年可能低于 50%）
- 华为昇腾 + 寒武纪 + 阿里平头哥形成国产算力三角
- 软件生态从 CUDA 迁移到 CANN，迁移效率已超 90%
- DeepSeek V4 同时适配华为 + 寒武纪，Day 0 支持

含义：算力供给不再被英伟达卡脖子 → 国内 AI 厂商的成本结构会和海外产生分化 → 定价策略可以更激进。

### 趋势 3: 开源模型重塑定价权

DeepSeek 模式：开源 + 极致低价 + 不卖订阅
- V3.2 在 OpenRouter 月调用 5.35 万亿 token，全球第三
- 开源模型占所有开源 Token 的一半以上
- 但 DeepSeek 没有 Coding Plan/Token Plan，只卖 API

冲击：开源模型的免费部署能力，会持续压制定价上限。厂商的定价权来自"服务便利性"而非"模型独占性"。

### 趋势 4: Agent 时代的成本指数级增长不可逆

- OpenClaw 爆火证明：一个 Agent 任务 = 几十次模型调用
- 推理需求 2026 年已达训练的 4-5 倍，且在加速
- IDC 预测 2028 年推理工作负载占比 73%
- Coding Plan "不限量"模式在 Agent 时代不可持续

结果：所有厂商最终都会转向 Token 精细计量。谁先转谁主动，谁后转谁被动（Cursor 2025-06 的教训）。

## 反模式（踩过的坑）

- **Cursor 2025-07 退款风波**：突然从 request 转 credit-based，用户反弹 → 定价模式切换必须渐进
- **计量单位混乱**：90% 用户踩坑 → 信息不对称短期有利，长期伤害信任
- **饥饿营销反噬**：每天 10:30 抢购 → 短期有效，长期伤品牌
