1.0 启动浏览器并导航到签约页面
Playwright MCP 需要独占启动 Chrome。导航到京东签约平台，确认已登录状态。
// 导航到签约页面
browser_navigate → https://xingkong.jd.com/qu渠道/793559463196729346
browser_snapshot → 确认已登录
注意
如果未登录 → 暂停，等年老师手动登录后继续。
2.0 创建新合同并选择合作意向
点击"新建E签"进入表单。选择合作意向是弹出对话框模式——搜索供应商名称后精准匹配。
// 操作序列
1. 找到"新建E签" → click
2. 等待 2s → snapshot 确认表单已加载
3. 找到"合作意向"输入框 → click
4. 搜索框输入供应商名称
5. 选中匹配行的 radio → 点击"确定"
自动带出字段
伙伴名称、合作模式（渠道代理）、客方签约主体信息。
3.0 选择合同模板
el-select remote 组件，普通点击无法触发搜索。需要通过 Vue 实例设值。
// 已验证方案：通过 Vue 实例设值
1. 找到合同模板的 el-select 组件
2. 获取其 __vue__ 实例
3. 调用 remoteMethod 搜索
4. 等待 options 加载
5. 设置 value 为目标选项的 value
注意
需要先 snapshot 确认 el-select 的 DOM 位置。目标选项：客户服务外包合同-202604
4.0 填写合同信息
6 个字段需要填写。合同期限选"固定期限"后会自动出现"是否宿迁承接对账"字段。
字段 操作方式 目标值
合同类型 el-select → 选"渠道主协议" 渠道主协议
资金流向 el-select → 选"支出" 支出
合同期限 el-select → 选"固定期限" 固定期限
是否宿迁承接对账 radio → 选"是" 是
客方签约方标识 下拉选"乙方" → 点"完成" 乙方
我方签约方标识 下拉选"甲方" → 点"完成" 甲方
5.0 批量填写模板变量
选择合同模板后，页面展开显示所有 ${xxx} 变量字段。通过 evaluate 批量设置 23 个变量。
Excel列 平台变量 取值
供应商名称 ${party_b_name} 直接取
乙方地址/电话/邮箱 ${party_b_address/phone/email} 直接取
测试期相关 ${trial_period} 等7个 统一填 "/"
续约开始 ${renewal_start_year/month/day} 2026 / 6 / 1
续约结束 ${renewal_end_year/month/day} 2027 / 5 / 31
银行信息 ${account_name/number/bank_name} 直接取
统一信用代码 ${uniform_credit_code} 直接取
6.0 填写发票信息
发票类型选择增值税专用发票，开票税率选择 6%。
字段 操作方式 目标值
发票类型 el-select 增值税专用发票
开票税率 el-select（可能被遮挡） 6%
开票税率注意
下拉可能被页面底部遮挡。先 scroll 到发票区域再点击。
7.0 截图确认并保存
截图保存当前表单状态，点击保存按钮，等待 3 秒确认保存成功。
browser_take_screenshot → 保存当前表单状态
browser_click → 点击"保存"按钮
wait → 3s
snapshot → 确认保存成功
