## 一、给所有参会供应商的落地任务（30天改善计划）

> 培训结束 ≠ 结束。每位参会者回去后7天内提交【事后行动回收表】，选定1个动作在30天内优先落地。
> **统一目标：一个月内把人均业绩提升1000元。**
> 参会9家：泉州速腾、中乾、伽玛、广达、德辰、岐力、小蜜蜂、新动力、河北锦瑞

### Step 1：会后7天内 · 提交行动回收表（必做）

每位参会者对照【事中对照记录表】里记下的"可直接抄的动作"，**选定1个**在30天内优先落地：

| 维度 | 你的回答 |
|------|---------|
| 动作名称 | （从下方可抄动作中选1个） |
| 谁来做 | ☐主管 ☐组长 ☐全体坐席 ☐特定人员 |
| 在什么场景做 | （如：每日早会/新人入职第1天/授信成功后） |

### Step 2：会后30天内 · 落地执行 + 数据追踪

| 优先级 | 推荐动作 | 预期效果 | 落地周期 |
|--------|---------|---------|---------|
| P0 | **讲清通时的收入逻辑**——给坐席算清"日均260通≈1万元"的账 | 激发自驱力，通时不用压 | 7天 |

| P1 | **用名产管理每一条数据**——立名产为核心指标，每周复盘 | 数据价值最大化，份额转化提升 | 30天 |
| P1 | **授信成功3天未跟强制轮转**——建共享台账+轮转机制 | 名单利用率提升，业绩不流失 | 14天 |
| P2 | **新人衔接训先于话术**——第一课就用数据拆解1万元动作 | 新人前15天流失率下降 | 30天 |
| P2 | **每日拨打节奏优化**——14-16点只打有接通记录的名单 | 名单不浪费，时段效率提升 | 7天 |

### Step 3：30天后 · 跟进回访

运营官做一次跟进回访，看落地情况+需要补什么支持。

## 二、六大可抄动作Checklist

### 动作1：讲清通时的收入逻辑（P0·7天）

- [ ] 算清自己团队的"动作-收入"等式（参考汇讯：日均260通/通时190min/授信3个≈1万元）
- [ ] 把等式做成1页纸，贴在每个坐席工位/发到工作群
- [ ] 主管每日早会用这个等式做目标拆解
- [ ] 试跑7天，观察通时和授信量变化

### 动作2：接通即前置加微（P0·14天）

- [ ] 召集主管统一认知：加微不止授信成功，接通的意向客户就加
- [ ] 设加微考核指标：每日≥4个/坐席
- [ ] T+0五动作SOP培训：备注+1分钟欢迎语+探需+打标(≥3)+设提醒(T+1/3/7/20)
- [ ] （可选）设3-5元/个通过奖励，当日兑现
- [ ] 试跑14天，统计加微率和企微池增长

### 动作3：用名产管理每一条数据（P1·30天）

- [ ] 统一名产定义：名产=总放款金额÷下发名单量
- [ ] 把名产加入主管周报核心指标
- [ ] 配套统计"打开APP"过程指标（系统未统计，需手工）
- [ ] 评估GMV占比/转化率占比相对数据份额的提升

### 动作4：授信成功3天轮转（P1·14天）

- [ ] 建立授信成功客户共享台账（额度/沟通情况/跟进状态）
- [ ] 制定轮转规则：3天未跟强制轮转；高额度重点客户转绩优坐席
- [ ] 明确分成规则：原坐席七成/跟进三成，或五五分
- [ ] 主管每日核查台账

### 动作5：新人衔接训先于话术（P2·30天）

- [ ] 调整新人培训重心：从话术转为认知（企业文化+合规+京东业务）
- [ ] 衔接训第一课：用数据拆解"1万元对应的每日动作"
- [ ] 主管每日18-19点组织经验分享
- [ ] 选优秀录音：额度大、流程顺、符合培训要求
- [ ] 追踪新人前15天留存率变化

### 动作6：每日拨打节奏优化（P2·7天）

- [ ] 9-10点：拨打存量名单+意向客户+授信成功客户
- [ ] 11:30-12:30：首播高优实时批次
- [ ] 14-16点：只打有接通记录的名单（接通率低时段不浪费新名单）
- [ ] 15:30：回收T+3/T+4未触碰名单，做第二轮预测
- [ ] 17:30后：跟进意向和授信成功客户
- [ ] 18:30后：自愿留下员工发放优质名单
