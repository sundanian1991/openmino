## 给运营官（年老师）的使用说明

### 核心关注指标

| 指标 | 题目 | 警戒线 | 应对 |
|------|------|--------|------|
| 内容实用性 | 第1题均分 | < 3.5 | 主题选错了或主讲人选错了，下期换方向 |
| 内容真实性 | 第2题均分 | < 4.0 | 如果供应商都在讲虚的，检查预审是否太松 |
| 数据颗粒度 | 第3题均分 | < 3.5 | 下次预审加强对「必须有2个具体数字」的要求 |
| 讨论质量 | 第4题均分 | < 3.5 | 讨论冷场——减少分享时间、增加讨论时间 |
| 整体满意度 | 第5题均分 | < 4.0 | 私下问1-2个老实人具体哪里不好 |

### 分模块诊断（第6-11题）

| 均分最高的模块 | → 下一场可深化或做专题 |
| 均分最低的模块 | → 下次备课弱化或调整讲法 |
| 某模块持续低（连续两期同类） | → 该模块讲师或内容有问题，需换人 |

### 本场重点观察项

1. **企微私域模块（第9题）**：新动力的成熟打法是否对其他供应商有实操参考？若均分高，可考虑单独做一期线上专题深化。
2. **培训体系模块（第11题）**：杨莹莹的培训体系是全场最受关注内容之一，若均分高，可考虑做「培训体系搭建」专题工作坊。
3. **时间投入性价比（第5题）**：全天高强度安排（议程08:30-18:20，实际延展至21:14），若均分偏低，下期考虑压缩或拆分。
4. **议题认领意愿（第16题）**：直接影响联盟分享机制能否跑起来。目标：至少3家供应商愿意认领。
