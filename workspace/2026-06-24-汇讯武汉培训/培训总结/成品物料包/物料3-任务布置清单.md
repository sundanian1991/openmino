## 五大可抄动作的落地Checklist（逐项对照）

### 动作1：讲清通时的收入逻辑（P0·7天）

- [ ] 算清自己团队"动作-收入"等式（参考：日均260通/通时190min/授信3个≈1万元）
- [ ] 把等式做成1页纸，贴工位/发工作群
- [ ] 主管每日早会用等式做目标拆解，不同收入对应不同通时
- [ ] 试跑7天，观察通时和授信量变化

### 动作2：接通即前置加微（P0·14天）

- [ ] 召集主管统一认知：接通的意向客户就加
- [ ] 设加微考核：每日≥4个/坐席
- [ ] T+0五动作SOP培训：备注+1分钟欢迎语+探需+打标(≥3)+设提醒
- [ ] （可选）设3-5元/个通过奖励，当日兑现
- [ ] 试跑14天，统计加微率和企微池增长

### 动作3：用名产管理每一条数据（P1·30天）

- [ ] 统一名产定义：名产=总放款金额÷下发名单量
- [ ] 把名产加入主管周报核心指标
- [ ] 配套统计"打开APP"过程指标（手工统计）
- [ ] 评估团队GMV占比/转化率占比相对数据份额的提升

### 动作4：授信成功3天轮转（P1·14天）

- [ ] 建立授信成功客户共享台账（额度/沟通/跟进状态）
- [ ] 制定轮转规则：3天未跟强制轮转；高额度客户转绩优坐席
- [ ] 明确分成：原坐席七成/跟进三成，或五五分
- [ ] 主管每日核查台账登记

### 动作5：新人衔接训先于话术（P2·30天）

- [ ] 调整新人培训重心：从话术转认知塑造
- [ ] 衔接训第一课：用数据拆解"1万元对应的每日动作"
- [ ] 主管每日18-19点组织经验分享
- [ ] 选优秀录音：额度大、流程顺、符合培训要求
- [ ] 追踪新人前15天留存率变化

## 常见问题处理

**Q：供应商说"我们团队太小，这些动作用不上"**
A："动作1（讲收入逻辑）和动作2（接通即加微）对团队规模没要求，1个人也能做。预测/轮转需要一定规模，小团队可以合并团队跑。先从P0动作开始。"

**Q：供应商填了动作但一直没落地**
A：30天中期检查时直接问三个问题——①你选了哪个动作？②做到哪一步了？③卡在哪？卡住不丢人，不落地才丢人。运营官匹配支持帮你过卡点。

**Q：供应商落地有效果，怎么沉淀？**
A：90天复盘会就是干这个的。落地有效果的供应商，在下场联盟分享时上台讲"我怎么把X动作落地的"——既是经验沉淀，也是给他的曝光激励。
