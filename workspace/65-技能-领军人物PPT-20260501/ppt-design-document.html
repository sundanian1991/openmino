01 · Design Decisions

设计决策

这是一份奖项申报材料，需要展现权威感、秩序感、可信度。 所有视觉决策服务于「沉稳、克制、不出错」的目标。

导演参考

David Fincher（精确控制）+ Errol Morris（证据陈列）。 Fincher 的精确网格和层级控制用于数据的权威呈现，Morris 的证据陈列逻辑用于论点→论据→结论的叙事结构。

Shell-Ban List（禁止事项）

  • 禁止深色背景（申报材料需打印和投影，白底最保险）
  • 禁止大面积渐变或炫光效果
  • 禁止过度动画或滚动特效（PPT 静态展示为主）
  • 禁止圆角卡片/气泡等软化设计（保持权威感）
  • 禁止装饰性图标或 emoji
  • 禁止多色混用（仅京东红+黑白灰）
  • 禁止花哨字体（使用系统字体或简洁无衬线体）
Primary Composition Family：结构化证据
不同于以往供应商可视化（深色+多色+动画丰富），也不同于日常报告（随意+模板化）。 本次是正式申报材料，需要「印刷品级别」的精致感。
02 · Color System

色彩体系

京东红 #E2231A 是唯一强调色，每页红色占比不超过 10%。 灰阶系统确保文字层级清晰可读。

角色 色值 用途
背景白 #FFFFFF 所有页面底色
京东红 #E2231A 标题强调、数据高亮、装饰线
深灰 #1A1A1A 正文文字
中灰 #666666 辅助说明、次要文字
浅灰 #F5F5F5 区块底色、分隔背景
边框灰 #E0E0E0 表格线、分隔线

京东红使用规则

  • 仅用于标题首字/词、关键数据、页码、装饰线
  • 每页红色占比不超过 10%
  • 红色背景块仅用于数据数字或重要标签
  • 不使用红色的变体或透明版本

装饰元素规范

允许使用

  • 红色细横线（3px）：标题下方分隔
  • 红色数字高亮：关键数据大字
  • 灰色区块底色：引用或强调区域
  • 细线表格边框：数据表格
  • 红色圆形序号：阶段卡片
  • 红色边框强调：战略目标块
  • 简单几何分割线：区域划分

禁止使用

  • 图标、emoji
  • 渐变、阴影、发光效果
  • 圆角卡片、气泡框
  • 背景纹理或图案

05 · Storyboard

分镜脚本

每页一个核心主张（One Big Idea），配合具体构图策略（Signature Composition）和材质表现（Material）。

P1 · 封面 COLD OPEN
One Big Idea：「2026年度新客服节新客服最佳实践奖」—— 这句话占最大面积。
构图：标题垂直居中偏上，底部京东科技标识，大面积留白制造呼吸感。
材质：白底，标题深灰，「最佳实践」用京东红强调。不做背景图、渐变、装饰图形，纯文字。
P2 · 企业介绍 ESTABLISHING SHOT
One Big Idea：企业名称+SLOGAN+基本信息，像一张名片。
构图：左右分栏（左35%/右65%），左侧品牌信息，右侧6项基本信息纵向排列。
材质：左侧企业名称+SLOGAN+愿景+价值观，右侧基本信息（主营业务/行业/地址/电话/总机/价值观）。
P3 · 呼叫中心现况简介 DATA DUMP
One Big Idea：3个关键数字（3000+人、35家供应商、100%电销渠道）+ 人员结构表。
构图：顶部3列关键数字行 + 下方人员结构表格 + 底部服务渠道说明。
材质：红色大数字 + 灰色标签 + 细线表格。
P4 · 呼叫中心组织架构 POWER STRUCTURE
One Big Idea：层级分明的组织架构图（文字版）。
构图：垂直层级堆叠（上层1个→中层2个→下层3个），形成金字塔视觉，红色连接线。
材质：层级底色（深→浅表示上→下）+ 红色连接短线。
P5 · 转型背景与战略目标 PROBLEM → GOAL
One Big Idea：行业挑战 → 战略目标，因果关系。
构图：左右对比（左挑战灰色底/右目标红色边框）+ 中间红色箭头连接。
材质：挑战灰色引用块，战略目标红色边框强调，底部转型驱动说明。
P6 · 转型实施过程 THE JOURNEY
One Big Idea：4个实施阶段横向排列，每个阶段有目标/措施/挑战。
构图：横向4列等宽卡片，顶部红色圆形序号，卡片内分三段（目标/措施/挑战）。
材质：白底卡片 + 灰色边框 + 红色圆形序号 + 内部灰色标签。
P7 · 转型具体成果 THE VICTORY
One Big Idea：6项核心成果数据 + 三个维度总结。
构图：上部6宫格数据区（2行×3列）+ 下部3列维度总结（灰色底色+红色左边框）。
材质：红色大数字 + 灰色标签 + 灰色引用块（维度总结）。
P8 · 转型的成功关键 THE INSIGHT
One Big Idea：5个成功关键因素，每个配一句话解释。
构图：纵向排列5条，红色序号+标题+解释，底部灰色启发引用块。
材质：红色序号 + 深灰标题 + 中灰解释 + 灰色引用块。
P9 · 组织变革与人才培养 THE PEOPLE
One Big Idea：组织变革 + 人才培养，双主题。
构图：左右双栏对比，左栏灰色标题区/右栏红色标题区，下方内容白底。
材质：栏目标题不同底色区分（左灰/右红），条目纵向排列。
P10 · 下一步计划 THE NEXT CHAPTER
One Big Idea：3个下一步目标 + 持续保障。
构图：2×2网格，3个白底卡片 + 1个红色高亮卡片（持续保障）。
材质：红色序号 + 深灰标题 + 中灰说明，红色高亮卡片反色。
P11 · 结束页 CLOSING
One Big Idea：「THANKS」+ 京东科技标识。
构图：垂直居中，上下大面积留白，与封面呼应首尾闭环。
材质：白底，红色 THANKS + 灰色公司信息。

html2pptx 合规要求

  • 所有文本必须包裹在 <p>、<h1>–<h6>、<ul> 或 <ol> 标签中
  • <p> 元素不能有 border 或 background，需移到 wrapper div
  • 每页内容不能溢出 405pt 高度（底部留 0.5" 安全边距）
  • CSS 渐变和阴影需栅格化为 PNG（本项目未使用）
