# 天普大学博士论文格式指南

> 合并翻译自 G1-G4 四份模板文件

---

## 一、封面页

**论文标题**

标题使用全大写字母。如果标题超过一行，采用倒金字塔格式：

```
    INSERT TITLE HERE IN ALL CAPITALS
         IF THE TITLE IS LONGER
          THAN ONE LINE USE AN
           INVERTED PYRAMID SHAPE
               LIKE THIS
```

**学位信息**

```
博士学位论文

提交给
天普大学研究生委员会

部分满足
博士学位要求

博士学位名称
（或教育学博士、音乐艺术博士）
*从 SSB 系统获取准确的学位名称*

作者：姓名

毕业日期：仅使用 December 20XX、May 20XX 或 August 20XX
（填入相应年份，如 2015）
```

**答辩委员会成员**

```
答辩委员会成员：

导师姓名，答辩主席，所属系
委员姓名，天普大学系别
委员姓名，天普大学系别
委员姓名，天普大学系别
委员姓名，校外成员，所属机构
```

## 二、版权页（可选）

> 如果没有版权页，删除此页至分页符处。

**使用说明**：

1. 将年份中的 XX 替换为当前年份
2. 双击下方"姓名"字样，在公式编辑器窗口输入你的姓名，然后点击文档隐藏工具栏
3. 删除本页上方所有说明文字，使版权符号距上边缘约 4.5 英寸

```
©

版权所有

20XX

作者：姓名
```

## 九、正文章节

```
第 # 章

章节标题

在此输入章节正文。

将上方 # 替换为相应章节编号。
章节编号使用阿拉伯数字，不使用罗马数字。

章节标题超过一行时，按 Shift-Enter（而非 Enter）换行，
以保持标题行距格式正确。
```

---

## 十、参考文献

> 可将标题改为"参考文献"或"引用文献"。

**格式要求**：

- 悬挂缩进 0.5 英寸
- 单条目内单倍行距，条目之间双倍行距（可选）

**或使用段落缩进格式**：

- 首行缩进 0.5 英寸
- 转行文字与左边界对齐
- 单条目内单倍行距，条目之间双倍行距（可选）

---

## 十一、附录

```
附录 #

附录标题

在此插入附录内容。

每个附录单独起页，与前文用分页符隔开（插入 → 分页符）。

附录标题居中排版，将标题中的 # 替换为 A、B 等相应字母。
```
