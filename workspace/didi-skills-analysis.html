重叠分析

与现有技能体系的关系评估

data-visualization vs viz-chart-visualization / viz-data-visualization

Python 可视化路线（matplotlib/plotly）vs 前端渲染路线（ECharts/Chart.js）。互补不冲突：Python 适合论文/研究场景，前端适合汇报/网页场景。

[ COMPLEMENTARY ] 双路线并存，无冲突

word-formatter vs docx 技能

word-formatter 侧重格式排版 + 自动配图决策，docx 侧重文档创建。可以形成协作链路。

[ COMPLEMENTARY ] 可协作

competitive-landscape vs 现有竞品分析能力

didi 版拥有更完整的设计系统（苹果简约风）和更系统的分析框架体系（4 大框架可选），质量超出已有能力。

[ UPGRADE ] 可替代现有方案
