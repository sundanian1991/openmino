# Storyboard — 吴宇森 John Woo / 周润发

## Site Cinematic Grammar

- **Shell logic**: Panoramic slab + symmetrical dual — 宽幅横贯，像宽银幕电影
- **Navigation posture**: 无导航，scroll as dolly
- **Framing rules**: 对称优先，左右镜像——直接翻译"双枪并立"构图
- **Density cadence**: 密集（Hero）→ 对称（双枪宣言）→ 沉浸（叙事）→ 呼吸（静默）→ 收束
- **Recurring materials**: earth-tone 装饰线、点阵纹理（教堂彩窗逻辑）、对称分割
- **Composition family**: Panoramic slab

## Director Brief（nian 颜色修订）

**John Woo 吴宇森**
- **Visual thesis**: "对称中爆发的暴力诗意——教堂里的枪声"
- **3 signature techniques**:
  1. **对称双栏** — 左右镜像布局，像双枪、像教堂的对称拱门
  2. **静默→爆发** — section 前半段缓慢、后半段突然揭示，翻译吴宇森的 slow-motion 枪战
  3. **垂直分层** — 前景/中景/背景的层次感，像教堂祭坛的深度
- **Color tokens**（nian earth 体系）:
  ```
  --scene: #8B7355;  --scene-bg: rgba(139,115,85,0.06);  --scene-border: rgba(139,115,85,0.2);
  ```
- **Typography**: nian 三字体固定
- **Motion rules**: 慢缓过渡（1.5-2.5s），ease-out。无 bounce。优先使用 clip-path 和 opacity 类入场。
