# 策展大纲 · handoff-1

> Step 0a 产出。按 curatorial-topic-framing + curatorial-narrative-modules 流程。
> 素材：中文文章《40 万次 Claude Code 真实对话分析》+ Anthropic 原报告（已读原文核对）。

---

## 对象界定（topic-framing 产出）

### 策展对象
Anthropic《Agentic coding and persistent returns to expertise》报告的一份**中文解读长文**。注意——策展对象不是 Anthropic 报告本身，是这份**带作者视角的解读**。

### 真实主体（关键）
表面在讲"40 万次对话的几个发现"，真实主体是：**在 AI 面前，"专家"的定义被重写了——从资历/职位，变成"具体任务里的指令质量"**。整篇文章所有数据（70%/80% 分工、5 倍差距、翻车分水岭、管理者成功率）都在为这个重写服务。

### 进入尺度
**问题尺度**——通过回答"什么决定了一个人的判断好不好"来进入。不是按报告章节顺序，是按"重新定义专家"这个论证脉络重组。

### 目标受众
已经在用 AI 工具（不限于 Claude Code）的知识工作者——尤其是有资历但不确定自己在 AI 时代位置的人。他们读完应该感到：资历感被挑战 + 但专业能力有出路。

### 材料边界
- **可靠**：40 万/23.5 万/7 个月/70%/80%/5 倍/4%/15%/+27%/职业排名——这些数字中文文章与 Anthropic 原文一致
- **存疑**：中文文章说"管理者成功率比工程师高"暗示管理动作 = 操作 AI 的技术动作。原文更克制——说这可能和管理者更习惯明确确认结果有关，"打个折"。中文文章的延伸比原文更激进
- **缺失**：原文有一个中文文章未充分展开的发现——**"专家和中级者的差距是 modest（适中）的"**。这个细节很重要，它平衡了"5 倍差距"的戏剧性
- **排除**：原文的方法论细节（session 怎么界定、expertise 怎么打分）——技术性太强，长文不需要

### 排除方向（★设计判断的关键输入）
- **排除数据简报形态**——这是带作者声音的解读，不是 KPI 看板
- **排除纯报告体例**——不要 Report Cover/TOC/Chapter Header 大编号
- **排除数据叙事链**（Stat Grid → Elevation → Seam → Matrix）——那是冷数据页的语汇
- **保留作者叙述语气**——「本来没太当回事」「好，接下来的问题就是」「你注意到没？」这些过渡句是文章的签名，不能为了"简洁"删掉

## 叙事结构（narrative-modules 产出）

### 内容形式
**CT-1 长文 / 深度解读文章**。单作者视角，带口语化过渡，金句靠上下文蓄力。不是数据报告，不是品牌展示。

### 叙事弧线
```
引子（不当回事→被打动）
  → 论点（人管判断 AI 管执行 · 70%/80%）
  → 重新定义（专家不看资历，看指令质量）
  → 数据支撑（5 倍差距 · 控制变量 +9%/+13%）
  → 反转（管理者成功率 > 工程师）
  → 转折（翻车才是分水岭 · 4% vs 15%）
  → 收束（三件事检查清单 + 余韵："专业能力没有归零，只是换了接口"）
```

**弧线特点**：归纳式（从具体到一般），有起伏，有两个反转点（管理者 > 工程师、翻车后分水岭）。**不是数据漏斗**（持续加压无呼吸）。

### 页面类型表

| 页面 | 角色 | 主角 | 阅读深度 |
|------|------|------|---------|
| Article Header | 标题区（不占满屏） | 标题主张 + meta | 浅 |
| 引子段 | 钩子 | 作者叙述语气 | 中 |
| 章节 01-06 | 论证段 | 每段一个论点 + 支撑数据 | 深 |
| 收束段 + 三件事清单 | 落地 | 行动清单 + 余韵 | 中 |
| Article Footer | 收束 | 作者位 + 相关 | 浅 |

**只有一种页面类型——长文页**。没有入口页/索引页/物件页（那是展览的模型）。curatorial 的页面类型在这里坍缩成单页长文。

### 交互逻辑
线性阅读 + Reading Progress 进度条。无跳转、无切换、无展开——长文的交互就是滚动。

## 一句话压缩（ljg 式检验）

> 作者借一份 40 万次的真实数据报告，重新定义了"专家"——不是资历职位，是具体任务里的指令质量；人管判断 AI 管执行，资历归零但专业能力换了个接口。

**压得成，不糊弄。** 关键词："借""重新定义""指令质量""换了接口"——每个词都排除了错误的内容形式判断（"借"排除纯数据报告，"重新定义"排除中性总结，"换了接口"排除悲观叙事）。
