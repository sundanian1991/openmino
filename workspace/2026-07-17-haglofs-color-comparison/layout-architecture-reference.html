SYSTEM RULES

Composition Principles

色彩纪律

  • 底色锁定暖白系（#F5F2E8 ± 5%），饱和度 ≤15%
  • 强调色单一（暖橙 #FF6B35），全页占比 5-15%
  • 文字深灰不用纯黑（#333333）
  • 冷调（鼠尾草绿/橄榄）仅做对比段，不做主底
  • 大地色（卡其/暖棕）用于半版分栏和目录页底

布局纪律

  • 双对齐：标题左 + 页码/数据右，中间虚线或留白
  • 半版分栏靠色调对比分隔，不靠实线
  • 信息条最多 1 个橙色强调格
  • 徽章组至少 3 种底色（深/橙/描边）
  • 时间轴关键段用橙，其余冷调

字体角色

  • Playfair Display：标题、数字、年份（有温度的衬线）
  • Inter：正文、描述（功能型无衬线）
  • JetBrains Mono：编号、标签、页码（精度型等宽）
  • 标签全大写 + letter-spacing 0.15-0.2em

禁止事项

  • 不用渐变做视觉主体
  • 不用阴影做层次（靠色块和 1px 线）
  • 不用玻璃拟态 / 模糊效果
  • 不用纯白 #FFFFFF 做页面底
  • 不用纯黑 #000000（用 #333333）
  • 强调色不超过 2 种（橙色 + 可选冷调）
