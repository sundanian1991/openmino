# Current State
项目已启动，完成需求采访和 GOALS.md 路线图制定。Milestone 1（产品设计）待启动。

# Task Specification
基于 github.com/fxy2311-youyou/expression-trainer 做产品重构：
- 融合表达训练 + 语音文档输出
- 引擎层抽象（ASR + 词库 + AI 后端），模式层各自独立
- 浅色温和风设计系统，从零建立
- Electron + Vite + React 技术栈
- MVP 优先：训练模式 + 正式报告模式先跑通

## 关键决策
| 维度 | 决策 |
|------|------|
| 定位 | 自用工具 |
| 技术载体 | Electron + Vite + React |
| 设计风格 | 浅色温和风 |
| AI 后端 | 双模式可切换（云端 + Ollama） |
| 文档类型 | 正式报告、会议纪要、邮件消息、笔记周报 |
| 开发节奏 | MVP 优先（训练 + 报告） |
| 差异化 | 文档模式"转换 + 对比教学"——不只是转换，同时教表达 |
