# 职场生存线调研 - 3天执行检查清单

## Day 1：启动与收集

### 上午（9:00-12:00）

- [ ] **确定抽样名单**
  - [ ] 按业务线整理供应商名单（按结算金额排序）
  - [ ] 确定抽样矩阵：16家（头部+中部+腰部）
  - [ ] 准备供应商编号（SUP-001至SUP-016）

- [ ] **发送调研邀请**
  - [ ] 群发邮件/消息：第一阶段全员风险筛查
  - [ ] 点对点联系：16家抽样供应商，说明深度调研
  - [ ] 同步发送：Excel成本表模板

- [ ] **准备访谈提纲**
  - [ ] 打印访谈问题清单
  - [ ] 预约访谈时间（至少4家）

### 下午（14:00-18:00）

- [ ] **监控问卷回收**
  - [ ] 跟踪问卷填写进度
  - [ ] 提醒未填写供应商（截止前2小时再催一次）

- [ ] **收集成本表**
  - [ ] 跟进16家抽样供应商
  - [ ] 目标：至少收回8份

- [ ] **同步开始访谈**
  - [ ] 完成至少2个深度访谈
  - [ ] 记录关键信息（数据异常点、风险信号）

- [ ] **Day 1 结束检查**
  - [ ] 问卷回收率 ≥ 50%
  - [ ] 成本表收回 ≥ 8份
  - [ ] 访谈完成 ≥ 2个

## Day 2：补充与验证

### 上午（9:00-12:00）

- [ ] **持续收集**
  - [ ] 跟进剩余成本表（目标：≥12份）
  - [ ] 完成所有访谈（4-5个）

- [ ] **问卷数据清洗**
  - [ ] 导出问卷数据
  - [ ] 识别异常值（前后矛盾、极端数据）
  - [ ] 标记待确认项

### 下午（14:00-18:00）

- [ ] **交叉验证**
  - [ ] 同一城市、同一业务线数据对比
  - [ ] 运营成本率合理性检查（15%-30%区间）
  - [ ] 与访谈数据对比验证

- [ ] **异常处理**
  - [ ] 数据明显偏离 → 电话确认
  - [ ] 逻辑矛盾 → 要求补充说明
  - [ ] 拒绝提供 → 记录原因，用同类样本替代

- [ ] **Day 2 结束检查**
  - [ ] 成本表收回 ≥ 12份
  - [ ] 访谈完成 ≥ 4个
  - [ ] 异常数据处理完毕

## 风险应对备忘

| 风险 | 应对措施 |
|------|----------|
| 供应商配合度低 | 强调"双赢"，不是压价，是优化合作 |
| 数据失真 | 交叉验证+访谈对质 |
| 时间不够 | 优先头部+中部，腰部可后续补充 |
| 样本不足 | 用问卷数据补充趋势判断 |
| 敏感信息拒绝 | 承诺保密，可提供脱敏报告 |
