供应商分级盘点

九宫格评估体系与三张清单

周期：月度评估 + 季度盘点 优先级：P0 状态：分析类，直做
目标

对 30+ 供应商进行综合评估

以当月贡献 × 环比趋势 + 供管自评为核心公式，输出分级清单和三张清单（保留 / 培养 / 优化），为资源分配、培养优化、淘汰清退提供决策依据。

对应 HRBP 原版：业务侧人才盘点（九宫格）

现状诊断

六大结构性缺失

  • 缺乏系统性的分级评估机制 — 分级多凭经验和直觉
  • 没有正式的"三张清单" — 保留 / 培养 / 优化无结构化输出
  • 赛马机制只看业绩一个维度 — 有相对排名但过于单一
  • 合规无量化的合规分数 — 有巡检整改记录但无法量化对比
  • 交付稳定性无结构化数据 — 人员流动、产能波动、SLA 未纳入体系
  • 30+ 供应商覆盖四条产线 — 头部 6 家贡献绝大部分产能，但缺乏分级管理
评估体系设计

两套体系，月度评估 + 季度盘点

体系用途频率
月度评估供应商分级 + 三张清单每月
季度盘点战略维度校准 + 格局审视每季度

月度评估：得分计算

最终得分 = [ 赛马基础分（扣分后 × 稳定性系数）] + 供管自评加分（0~4分）
1

赛马基础分

直接采用现有赛马成绩，满分 100。

2

合规扣分

事件扣分说明
黄牌整改-10 分/次巡检发现问题并下发整改函
投诉 / 一般违规-5 分/次质检通报、客户投诉
红线违规当月直接降级安全、数据泄露等严重事件
3

交付稳定性系数

月底扫三张检查线，ABC 三档：

档位系数触发条件（满足任一即降级）
稳定 A1.0人员无大幅变动、产能无异常波动、SLA 全达成
波动 B0.9人员流失 >10%、产能波动 >20%、SLA 一次未达成
失控 C0.75核心人员离职、产能断崖、SLA 两次以上未达成、突发掉链子
4

供管自评加分（0~4 分）

主观评价，四项各评高中低，高 = 1 分，中低 = 0 分：

维度高 = 1 分的触发条件
战略协作主动提出建设性建议 / 参与规划讨论、与业务方向一致
响应质量24h 内响应、反馈有实质内容（非沟通频率）
高效执行临时任务按时按质完成，无返工
共同成长产线会议积极贡献、分享经验、对整体产线有额外贡献

供应商赛马合规扣分稳定性自评最终得分
甲900A (1.0)2/490×1.0 + 2 = 92

九宫格映射（月度）

当月贡献与环比趋势决定供应商位置

轴含义切分方式
X 轴：当月贡献最终得分（赛马 × 稳定性 + 自评）按当月 30+ 家供应商得分的高中低三分位切分
Y 轴：环比趋势本月 vs 上月最终得分变化率上升 ≥5% / -5%~+5% 持平 / 下降 ≥5%

注：第一个月只有贡献无趋势，全部标记为"基线"，从第二个月开始有趋势。

季度盘点

月度评估捕捉变动，季度校准审视格局

维度说明评估方式
战略意义是否在关键产线 / 关键项目有不可替代性管理层评估
长期潜力规模扩张意愿、技术升级能力、团队稳定性访谈 + 观察
风险敞口过度依赖单一大客户、财务状况、舆情公开信息 + 调研

季度校准的四种组合

月度表现好 + 季度校准好 → 保留清单，深度绑定

月度表现好 + 季度校准差 → "高贡献但脆弱"，准备替代方案

月度表现差 + 季度校准好 → "低贡献但值得救"，给资源倾斜

月度表现差 + 季度校准差 → 优化清单，启动清退

输出

三张清单驱动后续行动

保留清单

6-10 家

来源：明星 + 加速器 + 中坚（月度高贡献或中贡献且趋势稳定）+ 季度校准良好

策略：资源倾斜、优先分配、深度绑定

培养清单

12-15 家

来源：成长 + 潜力 + 关注 + 观察（月度中 / 低贡献但有上升趋势）+ 季度校准有潜力

策略：针对性辅导、明确提升方向、月度跟踪进度

优化清单

5-8 家

来源：淘汰 + 预警（趋势下降的高贡献）+ 季度校准无潜力

策略：限期整改、减少份额、准备替代

实施路径

从基线建立到季度校准的三步走

1

基线建立（第 1 个月）

收集当月赛马成绩、合规事件、稳定性评估、供管自评；计算 30+ 家供应商当月得分；按三分位切分高中低，绘制首次九宫格（趋势标记"基线"）

2

月度滚动（第 2 个月起）

每月重复第一步；计算环比趋势，更新九宫格位置；输出三张清单（保留 / 培养 / 优化）

3

季度校准（每季度末）

追加战略意义、长期潜力、风险敞口评估；修正月度评估结果；向易人汇报分级结果，确认分类

风险与应对

合规扣分与赛马成绩中的合规指标可能重复

确认赛马是否已包含合规，如已包含则合规扣分项减半或只计红线事件

供管自评变成"人情分"，全部高分无区分度

每项有明确触发条件，月底过检查表；连续两月无中低分的维度需复核

分级变成"贴标签"，腰部供应商失去积极性

强调分级是动态的，月度调整，每次公布"进步榜"

精力集中在分级本身，后续动作跟不上

分级只是起点，重点是三张清单对应的后续动作
