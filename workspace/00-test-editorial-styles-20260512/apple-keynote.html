月度评估
最终得分怎么算
最终得分 = [ 赛马基础分 × 稳定性系数 ] + 供管自评加分
合规扣分在赛马基础分中扣除
第一步
100
赛马基础分满分
直接采用现有赛马成绩
第二步
合规扣分
黄牌整改 每次扣 10分
投诉 / 一般违规 每次扣 5分
红线违规 当月直接 降级
第三步
交付稳定性系数
A 稳定 系数 1.0 · 人员无大幅变动、产能无异常、SLA全达成
B 波动 系数 0.9 · 人员流失>10% 或 产能波动>20% 或 SLA一次未达成
C 失控 系数 0.75 · 核心人员离职 或 产能断崖 或 SLA两次未达成
第四步
0~4
供管自评加分，四项各评高中低
战略协作 主动提出建议 / 参与规划
响应质量 24h内响应，有实质内容
高效执行 临时任务按时按质，无返工
共同成长 分享经验，对产线有额外贡献

季度校准
三个战略维度
战略意义 关键产线 / 关键项目是否有不可替代性
长期潜力 规模扩张意愿、技术升级能力、团队稳定性

校准结果
月度好 + 季度好 → 保留，深度绑定
月度好 + 季度差 → 高贡献但脆弱，准备替代
月度差 + 季度好 → 低贡献但值得救，给资源
月度差 + 季度差 → 优化，启动清退
