## 🔑 关键信号（5条）

1. **OpenAI 发布 ChatGPT Work：从问答工具到自主工作平台**
   7月9日，OpenAI 发布 ChatGPT Work——一个基于 GPT-5.6 的 Agent，能在云端持续运行数小时，跨应用、文件和邮件自主完成复杂任务。近100%的 OpenAI 员工（不仅是工程师）每周使用 Codex。这标志着 ChatGPT 从"聊天机器人"正式转型为"工作平台"。

2. **Anthropic 的"日发布"模式：PM 工作周期从6个月压缩到1天**
   Cat Wu（Claude Code PM 负责人）透露：2026年Q1，Claude Code 推出了45+个新功能。原型优先替代文档优先，站会变成 Demo 展示。"产品品味"被重新定义为可训练的判断系统——不再是玄学，而是"知道什么值得做"的能力。

3. **Karpathy 加入 Anthropic：vibe coding 发明者的下一站**
   5月，OpenAI 创始成员、"vibe coding"概念发明者 Andrej Karpathy 加入 Anthropic。他提出"Software 3.0"——LLM 通过 prompt/context/tools 编程，上下文窗口成为新的程序。December 2025 被他称为"Agent 拐点"。

4. **NVIDIA 开放模型的"基础设施绑定"战略**
   Bryan Catanzaro（NVIDIA VP）揭示：Nemotron 不是为了在 benchmark 上击败 OpenAI，而是让企业 AI 部署跑在 NVIDIA GPU 上。开放模型降低进入门槛，TensorRT-LM + NIM 微服务创造切换成本。Nemotron 3 Ultra 是当前最强美国开源权重模型之一。

5. **Dario Amodei：我们接近指数终点，政策时间窗口正在关闭**
   6月长文警告："数据中心里的天才之国"即将在1-2年内到来。Mythos Preview 证明前沿模型对网络安全构成真实风险（而非理论风险）。他呼吁建立类似 FAA 的 AI 监管机构——前沿模型上市前必须通过第三方安全审计。
