## 四、供应商管理全生命周期

### 管理模块（六大模块）
1. **储备** - 寻源、储备资源
2. **引入** - 准入、上线
3. **运营** - 日常管理、监控
4. **评估** - 绩效考核、排名
5. **清退** - 风险处置、平稳退出
6. **结算** - 商务、对账

### 管理工具箱（五大类20+项手段）

| 类型 | 具体手段 | 说明 |
|------|---------|------|
| **监控预警类** | 高频问询、绩效评分、现场巡检 | 每日跟踪供应商状态，识别风险信号 |
| **处置干预类** | 暂停分量、恢复分单、要求复盘、约谈管理层 | 倒逼供应商解决交付问题 |
| **准入退出类** | 资质审核、现场考察、述标评估、清退执行 | 严把入口，平稳清理高风险供应商 |
| **赋能协同类** | 培训协助、业务扩增、资源倾斜 | 提升供应商能力 |
| **商务谈判类** | 商务谈判、合同谈判、追款谈判 | 优化成本、挽回损失 |

### 案例一：激活竞争——用"选汰机制"打破供应商舒适区

**核心故事线**：激活竞争

**深层背景**：
- 老供应商依赖严重，头部几家占据主要产能
- 缺乏竞争，部分供应商进入舒适区
- 四个团队合并后，发现老供应商产能还有提升空间，需要通过"换血"验证天花板

**主线逻辑**：
建立"选"和"汰"的完整机制 → 用新供应商搅动竞争格局 → 用清退倒逼现有供应商提升

**我的角色**：
- 体系搭建：统一四个团队标准，流程线上化
- 选的突破：引入8家新供应商，我全流程主导
- 汰的决断：4家合规清退，我发起并设计替代方案
- 0-1突破：金企融合0→400人、企贷大额0→200人，我主导

**核心数据**：
- 8家新供应商，6家进季度前三（75%）
- 4家清退，产能无缝衔接
- 新供应商引入从30天缩短到15天
- 0-1搭建：3个月达成目标

### 案例二：合规风控——用精细化管控守住底线（标题待优化）

**核心故事线**：建立体系并执行监控

**深层背景**：

- 缺乏统一标准和常态化监控机制

**主线逻辑**：
从"事后救火"转向"过程监控" → 建立可执行的监控体系

**我的角色**：
- 体系设计：制定准入门槛、复核机制、退出机制
- 监控机制：准入转正双重把关、过程监控常态化、递进式处置
- 执行管控：全年巡检17次，覆盖20家供应商

**核心数据**：
- 巡检17次，覆盖20家供应商
- 发现问题6家，整改闭环率100%
