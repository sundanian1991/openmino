## 从刘乾坤身上学习的东西

### ✅ 强烈推荐学习
1. **立规矩**：先定规则，再按规则办事
2. **规则具体可考核**：不是"尽快回复"，是"30分钟内"
3. **规则要有后果**：不是"请修改"，是"未修改清除处理"
4. **数据说话**：精确数据，不模糊
5. **要求明确**：具体到面试/培训/入职
6. **三要素法**：让供应商自己承诺（原因+问题+举措）

### ❌ 不需要学
- 过度强势、不给解决方案、只摆问题不给支持

## 供应商管理方法论

> 2026-04-06 通过知识榨取器深挖整理

### 核心判断逻辑：排名思维

> "88分和89分对我来说没有区别，唯一的区别是88分可能是第二名，也可能是倒数第二名。"

- **看相对排名，不看绝对分数**
- **关注后3名**，**连续2个月最后一名触发整改**

### KPI整改沟通框架

| 阶段 | 动作 | 关键话术 |
|------|------|---------|
| 开场 | 问现状+问原因+内部背书+预警 | "业务那边反馈过，再做不好业务难了" |
| 处理解释 | 区分共性问题vs个性问题 | "所有人都面临，你是最差，还是自己的问题" |
| 追问细节 | 不让糊弄，问时间/衔接/数量/原因 | "什么时候离职？培训周期怎么安排？" |
| 支持边界 | 线上指导为主，不派人现场 | "把问题整理好，我们给答复" |
| 目标设定 | 不接受模糊承诺，要明确排名 | "做到第X名" |
| 后续闭环 | 达到表扬+制造竞争，达不到最后一次机会 | "有人在挑战你，能守住吗？" |

### 管理风格特征

| 维度 | 年老师的做法 |
|------|-------------|
| 目标管理 | 不接受"不垫底"，要求明确排名 |
| 支持方式 | 线上指导，不派人现场 |
| 激励方式 | 私下了解经验+公开表扬+制造竞争 |
| 约束方式 | 达不到→最后一次机会→再达不到撤 |

### 对赛马机制的认知

- **人力效应**：老职场有优势，新职场很难超越
- **态度**：接受现状，渐进改善
- **改善路径**：提升尾部培训能力、引入成熟供应商
