## 核心框架

**权力内核**：他想要什么？→ 控制权 + 展示权 + 安全感

**权力来源**：权力从哪来？
1. **规则权** — 立规矩、机制先行
2. **信息权** — 两头堵、阻断信息
3. **关系权** — 拉拢圈人、人情债
4. **注意权** — 群内表演、打鸡血
5. **施压权** — 公开锁定、高压管理

**战术动作**：14种模式归属到5种权力来源

## 工作流程

### 当年老师甩群聊天记录时

**步骤 1：快速分析**

```
【权力来源】XX权 + XX权
【战术动作】模式X + 模式X
【核心逻辑】他在做什么？为什么这么做？
【识别信号】怎么看出他在用这招？
【应对建议】年老师怎么反制？
```

**步骤 2：记录到案例库**

追加到 `_案例库.md`：

```markdown
## 案例XX：事件名称（日期）

【权力来源】XX权 + XX权
【战术动作】模式X + 模式X
【核心逻辑】...
【识别信号】...
【年老师应对】...
【可学习】...
```

**步骤 3：关联到框架**

- 如果是新战术 → 补充到权力来源下的"对应模式"
- 如果是新应对 → 补充到应对策略
- 如果是深层洞察 → 升级权力内核

## 记录规范

### 案例记录格式

```markdown
## 案例33：开门红带节奏（03-29）

### 背景
群内密集同步金条激励问题

### 行为
- @李吉斌质疑方案
- 道德绑架："影响1400个制鞋工人、5000多个家庭"
- 只攻击服务组，不攻击策略组

### 分析
【权力来源】注意权 + 施压权
【战术动作】模式2（群内表演）+ 模式3（公开锁定）+ 模式15（选择性攻击）
【核心逻辑】在上级面前展示"我在解决问题"，同时保护关系网（策略组）

### 年老师应对
与供应商沟通确认"不会出现绩优批量离职"

### 可学习
选择性攻击识别：只攻击一方保护另一方 = 保护关系网
```

## 快速参考

### 5种权力来源速查

| 权力来源 | 识别信号 | 应对方式 |
|----------|----------|----------|
| 规则权 | "先建机制再谈业务" | 确认合理性，不盲目接受 |
| 信息权 | 阻断你和数据源接触 | 建立独立渠道 |
| 关系权 | 私聊帮忙、亲昵称呼 | 接受但不欠人情 |
| 注意权 | 消息多发在有上级的群 | 不抢戏，专注边界 |
| 施压权 | 群里@人、连环追问 | 反问回去，不接任务 |
