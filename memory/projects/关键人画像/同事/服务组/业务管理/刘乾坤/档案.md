## 年老师应对策略

### 核心原则
1. **不主动背锅** — 刘乾坤的问题直接传递，不接"帮我看下"
2. **边界清晰** — 周末不接@，工作日按流程
3. **信息独立** — 建立独立渠道，不依赖他拿数据
4. **学习技巧** — 立规矩、三要素法、数据说话

### 话术参考
| 他的手法 | 年老师回应 |
|----------|-----------|
| "帮我看下，我该如何解答" | "你的想法是什么？我先听你的" |
| 周末@处理供应商问题 | "收到，周一处理" |
| "别找供应商，他们不如我们清楚" | "理解，但我需要一线真实反馈" |
| "辛苦兄弟多费心"（责任转嫁） | "我今天 8 个供应商，担心盯不细"（示弱 + 边界）✅ |

### 04-02 结算预警应对案例（正确）
**刘乾坤话术**：危机渲染（公信力+投诉+人力缺口）+ 责任锁定（全超梳理的机制失效）+ 要机制主导权
**年老师回应**：
- 用事实压危机："预计4月10日完成2月对账" = 有进度，不是失控
- 锁定具体卡点：历史退保追溯流程久 = 技术问题，不是"机制失效"
- 踢球给出方案："坤哥了解吗？看有啥提效的方案"
- 给具体建议：龙虾 = 证明有思考
- 不接他的框架：没回应"协同机制"、"保护机制"
**要点**：用专业语言说话，把"非常困惑"变成"具体流程问题"，不被他带着走
