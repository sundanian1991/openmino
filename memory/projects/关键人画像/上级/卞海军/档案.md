## 核心行为模式

### 1. 穿透式高压管理 — 追问根因，连环施压
**表面行为**：不只看结果，深入追问每一个环节的逻辑、数据、时间线
**潜台词解读**：
- "问题你们早该发现" — 不满下属不及时上报
- "你们逻辑有问题" — 挑战决策依据
- "我给你时间线" — 不是商量，是最后通牒
**典型案例**：2026-03-21 车抵贷复盘，连环追问崔昊田"为什么老人流失还上人？名单不够为什么扩50人？"

### 2. 时间敏感型 — 对进度有严格要求
**表面行为**：明确时间节点，不接受模糊承诺
**潜台词解读**："最大接受4-6周" — 超过就换人
**典型案例**：车抵贷复盘，要求"撤销1-2周追不上就砍人"

### 3. 零容忍态度 — 对拖延和不作为不宽容
**表面行为**：问题严重时情绪激动，定性追责
**潜台词解读**："这属于严重事故" — 问题定性，明确责任
**典型案例**：车抵贷复盘，说"这属于严重事故，你们知道吗？"

### 4. 行动导向 — 要求具体动作，不只是计划
**表面行为**：给出具体指令，要求执行
**潜台词解读**：不只要计划，要具体动作
**典型案例**：车抵贷复盘，要求"约老板来""把人排一遍""刷掉不行的"

## 关键观察记录

### 2026-03-21 — 车抵贷复盘会议（高压场景）

**事件**：车抵贷业绩年后持续下滑，军哥召开复盘会议，情绪激动

**参与者**：
- 军哥（说话人2）：数据科技业务部负责人
- 崔昊田（说话人1）：业管，负责车抵贷业务
- 说话人3：可能是王易人或其他人
- 孙大年（年老师）：供应商管理岗

**军哥提问链条**：
1. 指出问题："业绩掉已经好几周了"
2. 追问根因："为什么老人流失严重，还上那么多人？"
3. 挑战决策："名单不够，为什么扩50人？"
4. 质疑能力："自己一点主见都没有吗？"
5. 定性严重性："这属于严重事故"
6. 设定时间线："最大接受4-6周"
7. 要求行动："约老板来""刷掉不行的"
8. 最后通牒："追不上就砍掉"

**管理风格体现**：
- **穿透**：连环追问，追到根因
- **高压**：情绪激动，问题定性
- **时间敏感**：严格时间线要求
- **行动导向**：要求具体动作
- **零容忍**：对拖延和不作为不宽容

**对崔昊田的施压手法**：
- 先指出问题存在多久
- 再追问决策逻辑矛盾
- 然后质疑下属判断力
- 接着定性问题严重性
- 最后设定时间线和行动要求

**年老师收获**：
1. 军哥是穿透式高压管理者，不只看结果，追到根因
2. 与军哥汇报时：数据+根因+方案+时间线+行动
3. 避免踩雷：不要等问题累积再上报、不要只解释不解决
4. 可学习：穿透式追问、用数据施压、设定时间线、要求具体行动
