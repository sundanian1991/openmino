## 核心行为模式

### 1. 战略型表达 — 宏观把控，不陷细节
**表面行为**：周报偏制度建设、方向规划；数据使用百分比为主；不点名具体问题职场
**潜台词解读**："我在战略层面有贡献，方向清晰，不陷在执行细节里。"
**典型案例**：2026-03-21 周报，王易人点名"毅航、汇讯、博岳"，刘伟佳只说"出勤不稳"

### 2. 服务型姿态 — 强调赋能，不强调权力
**表面行为**："支撑转化效率提升""帮助业管精准定位问题""组织专题会"
**潜台词解读**："策略组是服务业务的角色，不是管控角色。"
**典型案例**：不说"我要求"，而是说"支撑""帮助"

### 3. 模糊化处理 — 问题提及但不具体
**表面行为**：提问题但不点名、不承诺结果、只说方向
**潜台词解读**："问题我知道，但责任不在策略组，我只提供方向。"
**判断**：这是一种**责任边界管理**手法，不把责任压到具体人，不背锅

### 4. 节奏控制 — 有计划但不超前
**表面行为**："预计4月实施""下周组织专题会""详细交流具体落地方案"
**潜台词解读**："我有节奏，但不贸然承诺。细节留到专题会说。"

### 风格相似之处

| 共同特征 | 刘乾坤表现 | 刘伟佳表现 |
|----------|-----------|-----------|
| **不直接担责** | "帮我看下，我该如何解答" | "下周组织专题会" |
| **数据说话** | 精确到小数点，公开晒数据 | 量化成果（580亿、+34%） |
| **观众意识** | 群内表演、CC上级 | 成果前置、制度标签 |
| **责任边界管理** | 追责转嫁、加压链 | 分析外化、模糊化处理 |
| **节奏控制** | 不等结果立刻追加施压 | 不承诺结果，留到专题会说 |

**结论**：核心逻辑一致 — **"不背锅、有贡献、让别人动"**

### 水平差异

| 维度 | 刘乾坤（初级版） | 刘伟佳（高级版） |
|------|-----------------|-----------------|
| **施压方式** | 公开@人、阴阳怪气、连环追问 | 服务型姿态、制度标签、专业分析 |
| **情绪外露** | 激烈、夸张、让人窒息 | 温和、专业、不着痕迹 |
| **观众意识** | 明显的"在表演管理" | 隐性的"在展示贡献" |
| **责任转移** | 直接甩锅，"你负责这个" | 模糊化处理，"问题我分析清楚了" |
| **冲突风险** | 高（阴阳怪气容易得罪人） | 低（服务型姿态不树敌） |

**年老师评价**：
> "刘伟佳的水平更高，看起来没有那么激烈，动作也没有那么夸张"

**本质差异**：
- 刘乾坤：学的是"形" — 手法明显、痕迹重
- 刘伟佳：掌握的是"神" — 手法隐晦、不着痕迹

**刘乾坤的问题**：学到手法，但用力过猛，让人感觉到"在被管理"
**刘伟佳的高明**：掌握本质，润物细无声，让人感觉"在被服务"

### 具体对比案例

| 场景 | 刘乾坤做法 | 刘伟佳做法 |
|------|-----------|-----------|
| **问题出现时** | 公开@供应商："为什么没回复？" | 周报列出"关键因素"，不说谁的锅 |
| **推动事项时** | "您帮我看下"+"3点前回复" | "下周组织专题会，详细交流方案" |
| **展示贡献时** | 群内发数据+@上级看 | 周报成果前置（580亿、+34%） |
| **分配责任时** | "这个你负责"+CC上级 | "问题我分析清楚了，大家一起来" |
| **处理冲突时** | 阴阳怪气、软硬交替 | 劝导和解、调停者角色 |
