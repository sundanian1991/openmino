# 记忆机制 - 事后补记

> 不是每次都记，而是每次结束后自然回顾。

---

## 核心原则

**事后补记 > 实时记录**

- 不在对话中打断你确认"要不要记"
- 会话结束后，我自动回顾
- 挑1-2件最值得记的，写入对应位置

---

## 触发条件

每次会话结束前，我自动做：

| 条件 | 动作 |
|------|------|
| 对话轮数 ≥ 10 轮 | 执行补记 |
| 有重大进展/决策 | 立即记 |
| 你明确说"记一下" | 立即记 |

---

## 记录规则

### 自动归属

| 内容类型 | 写入位置 |
|---------|---------|
| 你的观点/偏好 | CAPTURE.md |
| 具体事件/对话 | daily/今天.md |
| 我的错误/教训 | 05-self-review.md |
| 工作进展/下一步 | 06-NOW.md |

### 数量控制

- 每10轮会话最多记 **2-3条**
- 优先记录：**新洞察 > 重要决策 > 重复模式**

---

## 执行方式

1. 会话即将结束时（我自己判断）
2. 快速扫描对话记录
3. 挑1-2件最值得记的
4. 直接写入对应文件
5. **不需要告诉你**

---

## 为什么这样设计

| 之前 | 现在 |
|------|------|
| 每次都问 | 结束后自动做 |
| 你提醒我记 | 我自己判断 |
| 频繁打断 | 自然完成 |
| 形式化检查 | 有价值才记 |

---

## 效果

- 你不需要任何操作
- 重要的不会漏掉
- 不会过度记录
- 形成"记了→用了→想记更多"的循环
