# Workspace — 工作空间

> **用途**：存放正在编辑的草稿、研究资料、临时文件

---

## 子目录说明

| 目录 | 用途 | 生命周期 |
|------|------|---------|
| **drafts/** | 正在编辑的草稿文件 | 完成后移至 active/ 或删除 |
| **research/** | 研究资料、数据抓取结果 | 提炼后移至 active/ 或 archive/ |
| **temp/** | 临时文件、中间结果 | 随时清理，不保证留存 |

---

## 使用规则

1. **工作文件先放这里** — 正在编辑的文件放在 drafts/
2. **研究资料暂存这里** — 抓取的数据、分析材料放 research/
3. **临时文件随时清理** — temp/ 目录随时可清空
4. **完成后及时归档** — 有价值的内容移至 active/，无价值删除

---

## 与其他目录的区别

| 目录 | 内容 | 典型文件 |
|------|------|---------|
| **workspace/** | 正在工作的 | 未完成的草稿、临时数据 |
| **active/** | 活跃记忆 | 已完成的 daily、thoughts |
| **core/** | 永久核心 | 决策、偏好、身份 |
| **context/** | 项目上下文 | 项目背景、共享知识 |
