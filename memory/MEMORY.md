| 双击操作优先 | 不要命令行，桌面 .command 文件双击启动 | 2026-03-29 |
| 硬件：Mac M4 32GB | 年老师记成 24G，实际 32GB 统一内存 | 2026-03-29 |
| 禁止代码块呈现内容 | MD文档用引用块（>），交互中不用代码块 | 2026-03-30 |
| 可视化替代文本符号 | 流程/关系/对比用 generative-ui-widget，禁止→↓│模拟 | 2026-03-30 |
