# Plan: 供应商结构性困境分析与应对方案

> 创建时间：2026-03-11
> 状态：Draft
> 负责人：年老师（服务组供应商管理岗）

---

## 一、需求重述

### 背景
- **组织结构**：服务组（供应商管理+执行交付）vs 策略组（名单+定价+产品+数据分析）
- **2026年目标**：预算是2025年的60%以上 + 降低费效比
- **策略组选择**：压降价格（而非培训/数据优化）来达成目标

### 核心问题
1. **供应商收益下跌**：人均创收从9,000+跌到7,200-7,800（竞争对手能提供13,000）
2. **数据口径美化**：内部汇报8,000-9,000，实际到手7,000多
3. **结构性依赖**：头部2家占50%份额，完成业绩依赖他们
4. **扶持悖论**：想扶持中尾部但得不到支持（"产能高才值得投数据"），头部又不愿加人
5. **头部供应商困境**：投产比计算精细化，老坐席产能高但新人留存难，整体节奏太快
6. **中尾部供应商痛点**：激励缺乏、培训和能力提升缺乏、数据能力缺失（不会用数据、不会深入挖掘）

### 年老师的诉求
1. **向上汇报**：向总监/VP呈现结构性问题（会带上职场反馈）
2. **应对策略**：在现有框架内逐步改善生态健康
3. **持续讨论**：多轮迭代，深化方案
4. **数据体系**：建立更成体系的供应商实际结算和成本利润状态（让VP真实推动）

---

## 二、问题诊断（三个维度）

### 维度1：目标冲突（组织层面）
- 策略组KPI：预算目标 + 费效比 → 最快路径是压降价格
- 服务组KPI：生态健康 + 交付质量 → 需要供应商有投入意愿
- **矛盾**：短期最优（压降价格）vs 长期最优（生态健康）

### 维度2：激励扭曲（机制层面）
- 供应商感受：增长的产能被抹杀 → 不愿投入培训/质量优化
- 结果：产能不增长 → 质量下降 → 策略组更有理由压降价格
- **恶性循环**：压降 → 不投入 → 产能低 → 压降更多

### 维度3：结构性依赖（执行层面）
- 头部2家占50%份额 → "太大而不能倒"
- 想扶持中尾部 → 但"产能高才值得投数据" → 没法扶持
- **鸡生蛋蛋生鸡**：不投数据 → 产能低 → 不投数据

---

## 三、能力与约束

### 服务组能独立推动的
- ✅ **培训**：服务组能独立培训供应商
- ✅ **试点机制**：可以试着推动试点
- ❌ **数据质量优化**：做不了（策略组负责）

### 现有框架内的调整机制
- **存量调整**：基于赛马做结构化调整
- **增量调整**：基于业绩和所需扶持调整

### 试点机制与框架的连接
- 如何让试点供应商进入"增量调整"的轨道？
- 如何让"扶持→产能提升"的路径与赛马机制兼容？

### 约束
- **组织架构**：服务组 vs 策略组的分工是既定的
- **KPI体系**：预算目标和费效比是硬指标
- **权力结构**：定价权在策略组，数据分配权在策略组
- **汇报对象**：向总监/VP汇报（不是策略组直接）

## 四、实施步骤（待深化）

### Phase 1：问题结构化（向上汇报准备）
1. **数据体系建立**
   - 从"个别职场数据"到"体系化供应商成本利润分析"
   - 真实结算数据 vs 汇报口径数据对比
   - 供应商分层分析（头部/中腰部/尾部）

2. **逻辑框架**
   - 用"目标冲突-激励扭曲-结构性依赖"三层框架
   - 呈现"系统性风险"（不是"管理技巧"问题）

3. **沟通策略**
   - 不指责策略组，而是呈现"长期风险"
   - 带上职场反馈，让问题"可见"
   - 目标：让VP认可问题的结构性，真实推动解决

### Phase 2：试点机制设计
1. **试点供应商选择**
   - 选择1-2家中腰部供应商（有潜力但缺支持）
   - 选择标准：意愿强 + 基础可 + 有成长空间

2. **试点方案设计**
   - **服务组能提供的**：培训、能力提升、数据使用指导
   - **需要策略组配合的**：数据质量优化、增量调整支持
   - **激励机制**：如何让供应商相信"投入有回报"？

3. **与现有框架的连接**
   - 存量部分：如何在赛马机制中保护试点供应商？
   - 增量部分：如何让"扶持→产能提升"的路径进入增量调整？

### Phase 3：分步推进（执行）
1. **短期**（1-2个月）：数据体系建立 + 汇报材料完成 + VP汇报
2. **中期**（3-6个月）：试点供应商选择 + 试点方案执行 + 效果验证
3. **长期**（6-12个月）：机制固化 + 良性循环形成

## 五、成功标准

- **短期**：决策层认可问题的结构性（不是"管理技巧"问题）
- **中期**：1-2家试点供应商产能提升20%+
- **长期**：形成"投入-回报-再投入"的良性循环
