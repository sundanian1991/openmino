# 供应商经营状况分析摘要

## 核心发现

### 1. 整体亏损严重
- 平均利润率: -18.7%（负数=亏损）
- 健康供应商: 12家（57%）
- 亏损供应商: 5家（24%）
- 边缘供应商: 4家（19%）

### 2. 人效数据
- 人均创收: 7197 元
- 人均成本: 6592 元
- 坐席平均薪资: 5222 元
- 薪资占创收比例: 72.6%

### 3. 安全边际窄
- 平均安全边际: 91.5%
- 意味着：结算金额下降8.5%，很多供应商就会亏损

### 4. 成本结构
- 人力成本: 72.6%
- 场地成本: 10.3%
- 运营成本: 7.9%
- 设备成本: 4.5%

### 5. 竞争劣势
- 友商结算单价: 京东的110%-120%
- 友商员工薪资: 7000-9000元 vs 京东5000-7000元
- 我们低约40-50%

## 按规模分组分析

### 头部供应商（>80人）
- 平均利润率: 23.8%
- 人均创收: 10962 元

### 中腰部供应商（30-80人）
- 平均利润率: 8.3%
- 人均创收: 7422 元

### 尾部供应商（<30人）
- 平均利润率: -96.3%
- 人均创收: 4273 元

## 按盈利分组分析

### 健康供应商（利润率≥10%）
- 坐席平均薪资: 5260 元
- 人均创收: 8407 元
- mob3留存率: 53%

### 亏损供应商（利润率<0%）
- 坐席平均薪资: 5439 元
- 人均创收: 5138 元
- mob3留存率: 60%

## 关键洞察

1. **压降价格已到极限**：24%供应商已亏损，继续降价会加速供应商退出
2. **规模效应存在**：头部供应商利润率和人均创收都更高
3. **安全边际窄**：平均只能承受8-9%的结算下降
4. **激励扭曲**：员工薪资占人均创收的73%，供应商没有投入空间

## 建议

1. **停止压降价格**：24%已亏损，继续降价会崩盘
2. **扶持中腰部**：通过培训提升产能，而非压降价格
3. **建立增量收益分享机制**：让供应商相信"投入有回报"
4. **降低对头部依赖**：培养第二梯队
