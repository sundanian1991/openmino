## 任务管理规则

### 创建任务
1. 复杂任务（≥3 步骤）→ 先创建 Plan 文档
2. 在 todo.md 添加条目，链接到 Plan
3. 设置优先级和状态

### 更新进度
1. 每完成一步 → 立即标记 ✅
2. 遇到阻塞 → 更新状态为 `blocked`
3. 需要用户确认 → 在条目中@用户

### 完成任务
1. 所有 Checklist 完成
2. 验证项全部通过
3. 更新 Plan 文档的评审部分
4. 更新 lessons.md（如有经验教训）
5. 标记 todo.md 状态为 `completed`

### 验证规则
**未通过验证的任务，不得标记为完成。**

验证清单：
- [ ] 所有计划步骤完成
- [ ] 验证方案通过
- [ ] 无回归问题
- [ ] 高级工程师会批准吗？
