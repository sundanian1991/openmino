# daily 日志模板

**使用说明**：
1. 按时间顺序记录，精确到小时
2. 每条记录必须打标签
3. 晚上 23:00 前整理归档

---

## 时间轴记录模板

```markdown
# YYYY-MM-DD 周 X | 天气/心情（可选）

## 📍 时间轴

| 时间 | 事项 | 标签 | 简述 |
|------|------|------|------|
| 09:00-10:30 | | | |
| 10:30-12:00 | | | |
| 14:00-15:30 | | | |
| 15:30-17:00 | | | |
| 19:00-21:00 | | | |

---

## 📝 详细记录

### [时间] 事项标题

**标签**：#工作/搭体系 或 #工作/沟通 或 #生活/家庭 或 #娱乐/游戏

**内容**：
- 做了什么
- 和谁
- 结果/产出

**情绪**：🙂/😐/😞（可选）

---
```

## 标签体系

### 一级标签

| 标签 | 说明 | 对应 5311 维度 |
|------|------|--------------|
| `#工作` | 工作相关事项 | 搭体系、沟通对齐 |
| `#生活` | 日常生活事项 | 抽离放松 |
| `#娱乐` | 纯娱乐活动 | 抽离放松 |
| `#学习` | 学习、阅读、课程 | 反思迭代 |
| `#健康` | 运动、医疗、休息 | 抽离放松 |
| `#社交` | 朋友聚会、非工作社交 | 抽离放松 |

### 二级标签（工作）

| 标签 | 说明 | 示例 |
|------|------|------|
| `#工作/搭体系` | SOP、模型、看板、机制搭建 | 写供应商评估模型、优化 Agent 机制 |
| `#工作/沟通` | 会议、汇报、协作沟通 | 周会、1v1、供应商谈判 |
| `#工作/救火` | 临时问题、紧急处理 | 处理供应商突发问题 |
| `#工作/反思` | 复盘、方法论沉淀 | 写周报、周度评估 |

### 二级标签（生活）

| 标签 | 说明 | 示例 |
|------|------|------|
| `#生活/家庭` | 陪伴家人、家务 | 陪老婆、做饭、打扫 |
| `#生活/健康` | 运动、就医、休息 | 健身、体检、午睡 |
| `#生活/日常` | 通勤、购物、琐事 | 买菜、取快递 |

### 二级标签（娱乐）

| 标签 | 说明 | 示例 |
|------|------|------|
| `#娱乐/游戏` | 玩游戏 | Switch、手游 |
| `#娱乐/影视` | 看电影、剧集 | Netflix、B 站 |
| `#娱乐/阅读` | 休闲阅读 | 小说、漫画 |
| `#娱乐/社交` | 朋友聚会 | 吃饭、喝酒、聊天 |

### 二级标签（学习）

| 标签 | 说明 | 示例 |
|------|------|------|
| `#学习/输入` | 听课、读书、看文章 | 看论文、上网课 |
| `#学习/输出` | 写作、分享、教别人 | 写文章、做分享 |
