# 5311 周度评估提醒

> **目的**: 每周日/周一填写 5311 时间分配评估，追踪工作生活平衡状态
> **频率**: 每周一上午提醒填写上周评估
> **模板**: `memory/active/tasks/frameworks/weekly-review-template.md`
> **落点**: `memory/active/weekly/YYYY-Www-review.md`

---

## 触发机制

**自动触发**：
- 每周一会话开始时，Mino 主动询问
- 如果周日/周一用户主动开启对话，触发提醒

**提醒话术**：
```
年老师，周日过完了。该填 5311 周度评估了。

这周时间分配大概啥样？
- 搭体系：~%
- 沟通：~%
- 反思：~%
- 玩：~%

花 5 分钟填一下，我帮你记下来。
```

---

## 评估流程

1. **Mino 提醒** → 周一上午主动询问
2. **用户填写** → 估算各维度时间占比
3. **识别失衡** → 如果有维度严重偏离，提醒调整
4. **记录存档** → 复制到 `memory/active/weekly/`
5. **下周追踪** → 记录上周"需要改进"项，本周检查

---

## 失衡判断标准

| 维度 | 警示阈值 | 含义 |
|------|---------|------|
| 搭体系 | <40% | 陷入日常救火，不是 P7 工作方式 |
| 沟通 | >50% | 时间被会议占据，没有深度工作 |
| 反思 | = 0 | 没有沉淀，机械重复 |
| 玩 | = 0 | burnout 风险，需要强制断开 |

---

## 月度汇总

**每月最后一个周日**：
- 汇总本月 4 周评估
- 识别持续失衡模式
- 填写月度深度评估（见 `5311-framework.md`）

**连续两周失衡** → Mino 主动提醒调整
