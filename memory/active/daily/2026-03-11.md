### Token 优化与 RTK 安装

**标签**：#系统优化 #Token优化 #RTK

**背景**：
- 年老师分享 Get 笔记文章："从Claude Code虎口夺食：两大'邪修'方法实现Token成本优化"
- 核心方法：RTK（命令输出脱水）+ mcp2cli（MCP 按需加载）

**执行内容**：
1. **P0 分析**：
   - 统计命令频率：curl 10次、git 8次、ls 7次（高频）
   - 统计 MCP 使用：tavily_search 5次、tavily_extract 3次（低频）
   - 结论：RTK 收益大，mcp2cli 暂不执行

2. **P1 安装 RTK**：
   - 安装 Rust 1.94.0（~5分钟）
   - 下载 RTK v0.28.2 预编译版本
   - 安装到 ~/.local/bin/rtk
   - 配置全局 hook：`rtk init --global --auto-patch`

3. **测试效果**：
   - git status：原始 180 字符 → RTK 60 字符（节约 65%）
   - ls：原始输出包含详细信息 → RTK 仅文件名（节约 ~70%）
   - curl：预期节约 92-98%

**产出**：
- ~/.local/bin/rtk（RTK 二进制）
- ~/.claude/hooks/rtk-rewrite.sh（自动重定向 hook）
- ~/.claude/RTK.md（10行配置）
- ~/.claude/settings.json（已更新）

**决策**：
- mcp2cli 暂不执行（当前 MCP 数量少，收益有限）
