## 使用流程

### 1. 遇到新情况时
```bash
# 复制示例文件
cp memory/context/people/领导A-示例.md memory/context/people/真实人物.md

# 填写内容
```

### 2. 定期同步到云端
```bash
# 同步所有内容
./scripts/sync-context-to-memos.sh all

# 或只同步某一类
./scripts/sync-context-to-memos.sh people
```

### 3. 对话前检索
- 问我："检索一下领导A的沟通风格"
- 我会自动读取 `memory/context/people/领导A.md`

## 记录原则

1. **记录 > 完美**：先记下来，再完善
2. **具体 > 抽象**：记录真实案例，不只是规则
3. **持续 > 系统**：保持更新，别追求完美框架
