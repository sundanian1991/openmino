# Working Buffer — 实时落盘

> 对话过程中的关键信息暂存区。不等对话结束，发现就写。
> 由每日整理 Cron（10:00）分发到目标目录，清空已处理条目。

---

## 写入规则

**触发**：与 WAL 协议一致——修正、专有名词、偏好、决策、具体值、异常、年老师原话。

**格式**：`- YYYY-MM-DD HH:MM | [类型] 内容`

**类型**：原话 / 异常 / 决策 / 线索 / 偏好

**清空时机**：每日 10:00 整理后，已处理条目移除，只保留头部说明。
