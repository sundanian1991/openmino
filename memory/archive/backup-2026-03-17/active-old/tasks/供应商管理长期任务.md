## 方法论总结

### 六大模块
| 模块 | 目标 | 关键动作 |
|------|------|---------|
| **储备** | 有鱼选 | 市场调研、建立储备池、定期评估 |
| **引入** | 选对人 | 商务谈判、合同签订、准入标准 |
| **运营** | 养好鱼 | 日常监控、巡检抽查、异常预警 |
| **评估** | 知好坏 | 绩效考核、分级调整、诊断分析 |
| **清退** | 敢止损 | 风险供应商清退、零纠纷退出 |
| **结算** | 算清账 | 费用结算、争议处理、合规闭环 |

### 五大支柱
- **标准体系**：《供应商管理手册》作为"宪法"
- **指标体系**：16项KPI、9大诊断场景、50个汇报问题
- **监控体系**：事前监控、风险画像、隐性断裂预警
- **汇报体系**：周报、事件通报、价值备忘录
- **选汰体系**：通过竞争激活，75%新供应商进入前三
