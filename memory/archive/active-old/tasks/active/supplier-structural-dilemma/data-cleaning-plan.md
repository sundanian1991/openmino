# 数据清洗与分析方案

> 供应商经营状况调研数据
> 创建时间：2026-03-11

## 一、数据概况

### 样本量
- **首贷**：11个样本
- **复贷**：10个样本
- **总计**：21个样本

### 核心字段（32列）

| 类别 | 字段 | 说明 |
|------|------|------|
| **基础信息** | 城市、业务类型、坐席规模、合作时长 | 分类维度 |
| **结算数据** | 月度结算金额、账期、员工薪酬占比 | 核心财务数据 |
| **成本结构** | 月度总成本、人力/场地/运营/设备成本占比 | 成本分析 |
| **人力数据** | 平均薪资、TOP薪资、稳定留存薪资、mob3留存率 | 人力分析 |
| **盈利能力** | 结算金额下降多少比例会亏损 | 盈利边界 |
| **运营状态** | 运营状态、降薪/缩减福利、资金周转困难 | 健康度 |
| **对比数据** | 友商结算单价、友商薪资水平 | 竞争对比 |

## 二、数据清洗步骤

### Step 1：数据标准化

#### 1.1 清理选项格式
问题：数据中包含 `□&nbsp;` 等HTML标记

```python
# 清理HTML标记
df['字段'] = df['字段'].str.replace('□&nbsp;', '', regex=False)
df['字段'] = df['字段'].str.strip()
```

#### 1.2 提取数值
问题：部分字段是区间格式（如"40%-50%"），需要提取中间值

```python
# 提取百分比区间中间值
def extract_percentage_range(value):
    if pd.isna(value):
        return None
    if '%' in str(value):
        # 处理"40%-50%"格式
        match = re.findall(r'(\d+)%', str(value))
        if len(match) == 2:
            return (int(match[0]) + int(match[1])) / 2
        elif len(match) == 1:
            return int(match[0])
    return value
```

#### 1.3 统一单位
- 结算金额：统一为"万元/月"
- 薪资：统一为"元/月"
- 成本占比：统一为百分比

---

### Step 2：数据验证

#### 2.1 逻辑一致性检查
| 检查项 | 公式 | 异常示例 |
|--------|------|----------|
| 成本结构 | 人力+场地+运营+设备 ≈ 100% | 总和>110%或<90% |
| 盈利边界 | 结算金额 × (1-亏损比例) ≈ 总成本 | 逻辑矛盾 |
| 薪资合理性 | 平均薪资 ≤ TOP薪资 | 平均>TOP |

#### 2.2 异常值检测
- 结算金额 < 5万元（过小，可能异常）
- 结算金额 > 200万元（过大，可能异常）
- mob3留存率 > 100%或 < 10%

#### 2.3 缺失值处理
| 字段类型 | 处理方式 |
|----------|----------|
| 关键数值（结算、成本） | 剔除样本 |
| 次要数值（TOP薪资、留存率） | 用中位数填充 |
| 文本描述（建议、问题） | 标记为"未填写" |

---

### Step 3：数据补全

#### 3.1 计算派生字段
```python
# 1. 计算人均创收
df['人均创收'] = (df['月度结算金额'] * 10000) / df['坐席规模']

# 2. 计算人均成本
df['人均成本'] = (df['月度总成本'] * 10000) / df['坐席规模']

# 3. 计算利润率
df['利润率'] = (df['月度结算金额'] - df['月度总成本']) / df['月度结算金额']

# 4. 计算单位人力成本
df['单位人力成本'] = df['月度结算金额'] * (df['人力成本占比'] / 100) / df['坐席规模']
```

#### 3.2 分类字段标准化
- 城市分层：一线（北京/上海/广州/深圳）、二线、三线及以下
- 规模分层：小型（<30人）、中型（30-80人）、大型（>80人）
- 合作时长分层：短期（<1年）、中期（1-3年）、长期（>3年）

---

### Step 4：数据分组

#### 4.1 按业务类型分组
- 首贷组
- 复贷组

#### 4.2 按规模分组
- 头部：>80人
- 中腰部：30-80人
- 尾部：<30人

#### 4.3 按盈利能力分组
- 健康：利润率>10%
- 边缘：利润率0-10%
- 亏损：利润率<0%

## 三、分析框架

### 分析维度1：生命线分析

**核心问题**：供应商的盈亏平衡点在哪里？

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| **盈亏平衡点** | 月度总成本 / 月度结算金额 | 了解当前盈利空间 |
| **亏损临界点** | 结算金额下降比例（调研数据） | 直接取自调研 |
| **安全边际** | 1 - 亏损临界点 | 越高越安全 |

**输出**：
- 各供应商盈亏平衡点分布
- 头部/中腰部/尾部供应商安全边际对比
- 首贷 vs 复贷业务盈利能力差异

---

### 分析维度2：成本结构分析

**核心问题**：供应商的钱花在哪了？

| 成本类型 | 分析内容 | 关键洞察 |
|----------|----------|----------|
| **人力成本** | 占比分布、与人均创收关系 | 人力成本是否过高？ |
| **场地成本** | 占比分布、城市差异 | 哪些城市场地成本压力大？ |
| **运营成本** | 占比分布、规模差异 | 规模效应是否存在？ |
| **设备成本** | 占比分布 | 是否有优化空间？ |

**输出**：
- 成本结构占比饼图
- 不同规模供应商成本结构对比
- 不同城市成本结构对比

---

### 分析维度3：利润率分析

**核心问题**：供应商到底赚不赚钱？

| 指标 | 计算方式 | 分析目标 |
|------|----------|----------|
| **毛利率** | (结算-总成本) / 结算 | 整体盈利水平 |
| **人均创收** | 结算金额 / 坐席数 | 人效水平 |
| **人均利润** | 利润 / 坐席数 | 人均贡献 |
| **安全薪资比例** | 平均薪资 / 人均创收 | 薪资占收入比例 |

**输出**：
- 利润率分布直方图
- 人均创收箱线图
- 头部/中腰部/尾部利润率对比

---

### 分析维度4：竞争对比分析

**核心问题**：我们的竞争力如何？

| 指标 | 京东 | 友商 | 差距 |
|------|------|------|------|
| **结算单价** | 基准 | 110%-120% | 我们低10-20% |
| **员工薪资** | 5000-7000 | 7000-9000 | 我们低2000+ |

**输出**：
- 京东 vs 友商结算单价对比
- 京东 vs 友商员工薪资对比
- 竞争劣势量化分析

---

### 分析维度5：风险识别

**核心问题**：哪些供应商有风险？

| 风险类型 | 识别标准 | 应对 |
|----------|----------|------|
| **资金风险** | 出现过资金周转困难 | 关注账期、现金管理 |
| **人员风险** | mob3留存率<40% | 关注培训、激励 |
| **运营风险** | 降薪或缩减福利 | 关注盈利能力 |
| **依赖风险** | 亏损临界点<10% | 抗风险能力弱 |

**输出**：
- 风险供应商名单
- 风险等级分布（高/中/低）
- 风险原因分析
