## 文档处理

| 技能 | 用途 | 格式支持 |
|------|------|---------|
| `minimax-docx` | Word 文档（专业级，.NET OpenXML） | .docx |
| `minimax-pdf` | PDF 处理 | .pdf |
| `minimax-xlsx` | Excel 表格（专业级） | .xlsx, .xlsm, .csv, .tsv |
| `markdown-converter` | 文档转 Markdown | PDF/DOCX/PPTX/XLSX/HTML/图片/音频 |
| `docx` | Word 文档（通用） | .docx |
| `pdf` | PDF 处理（通用） | .pdf |
| `xlsx` | Excel 表格（通用） | .xlsx, .csv |
| `pptx` | PPT（通用） | .pptx |
| `pptx-generator` | PPT 生成 | .pptx |

**建议**：优先用 `minimax-*` 系列（专业级），`mino-pptx` 用于 PPT。

## 冗余技能处理建议

| 类型 | 保留 | 可删除 |
|------|------|--------|
| Word | `minimax-docx` | `docx` |
| Excel | `minimax-xlsx` | `xlsx` |
| PDF | 检查后决定 | `pdf` 或 `minimax-pdf` |
| PPT | `mino-pptx` | `pptx`, `pptx-generator` |
| 前端 | `mino-frontend` | `frontend-design`, `impeccable` |
| 文件整理 | 两者分工不同 | 保留 `file-organizer`（本地）+ `workspace-tidy`（项目） |
