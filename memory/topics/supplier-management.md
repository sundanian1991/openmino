# 供应商管理

> 服务组核心职责，全量管理（合规/准入/清退/分工）

## 基本信息

- **上级**：王易人
- **重点6家**：毅航、毛毛虫、伽玛、赛维斯、岐力、翰锐（金条头部，3亿+职场）
- **四产线**：金条、企金、信用卡、财富
- **辅助**：业务管理（每天与供应商BPO沟通）| 质检培训运营（合规+赋能）

## 目标树设计（2026-07-22）

**三层结构**：

### 战略层（3根柱）
1. **管理闭环体系** — 选-用-育-留-汰全流程管理，执行力（P7）的体现
2. **数字化转型** — AI数字人、数据看板、自动化流程，体系构建力（P8）的体现
3. **生态与赋能** — BPO联盟、活水机制、非货币激励，战略生态力（P9+）的体现

### 战术层（管理闭环下）
- 选 — 引入与梯队建设
- 用 — 运营业绩管控
- 育 — 培训与能力发展
- 留 — 核心保留与激励
- 汰 — 分级清退与对冲

### 执行层
- Issues 挂在战术层下
- 卡点项目（浙江阅文、信用卡清退）挂在"汰"下
- 月度排名、巡检标准挂在"用"下

**设计决策**：
- 三根战略柱对应三个能力水位，也是晋升叙事
- "选用育留汰"是管理学标准框架，放战术层而非战略层
- AI协作设计：战略层（方向锚）→ 战术层（流程触发器）→ 执行层（任务执行单元）

**待办**：用户需在 MyAgents UI 手动创建目标树（CLI 暂不支持 `space goal create`）

## 关键机制

- **业绩排名**：每月5号出排名，异常主动问
- **联盟运营**：线上月度 + 线下季度，朱晓明7月接手
- **电销AI转型**：四阶段推进（能看见→能判断→能行动→自动化）
