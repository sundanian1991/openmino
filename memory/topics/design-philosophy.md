## 第一性原理

| 原理 | 定义 | 反面（AI味） |
|------|------|------------|
| **克制** | 每个元素必有目的，删到不能再删 | 装饰堆砌、为好看而好看 |
| **锐度** | 边界清晰、焦点唯一、有明确态度 | 模糊中庸、试图讨好所有人 |
| **诚实** | 不伪装、不套用、承认材料的本来面目 | 阴影渐变emoji、假装有设计感 |

---

## 信息设计铁律

1. **一页一核心** — 每页只有一个P0，2-4个支撑，其余删掉
2. **空白是设计的** — 不是剩余空间，用进度条/徽章/分割线激活
3. **强调色≤10%** — 唯一温度，一页一处原则
4. **无装饰性元素** — 每个像素服务信息，emoji=信息失败
5. **50%视觉率** — 每屏至少一半视觉元素，否则太干

---

## 反AI味检查清单

- [ ] **无阴影** — 包括卡片阴影、文字阴影、任何投影效果
- [ ] **无渐变** — 包括背景渐变、按钮渐变、任何颜色过渡
- [ ] **无emoji** — 用图标或文字替代，emoji是信息失败的标志
- [ ] **无圆角滥用** — 2px是上限，不是默认值
- [ ] **无居中偏执** — 左对齐是默认，居中需要理由
- [ ] **无装饰性分隔线** — 每个分隔都有信息目的

## 默认决策流程

```
内容 → 情绪 → 阅读 → 密度 → 版式 → 焦点 → 输出
```

| 步骤 | 问题 | 选项 |
|------|------|------|
| 1.内容 | 有什么？ | 数据/流程/清单/洞察 |
| 2.情绪 | 气质是？ | 紧迫/中性/庆祝/沉思 |
| 3.阅读 | 怎么用？ | 扫视/阅读/凝视 |
| 4.密度 | 多密？ | 高/中/低（由情绪×阅读决定）|
| 5.版式 | 什么结构？ | 表格/卡片/对比/时间线 |
| 6.焦点 | 强调色在哪？ | 唯一焦点，≤10%面积 |
