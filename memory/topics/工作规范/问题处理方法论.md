# 问题处理方法论

> 年老师遇到问题时的应对方法

---

## 一、问题诊断三层次

**诊断框架**：
```
现象 → 结构 → 根源
```

找到系统性原因，而非表面问题

**示例**：
新人流失率高的诊断路径：

| 层次 | 分析 |
|------|------|
| 现象 | 新人流失率高 |
| 结构 | 结算低 → 培训投入不足 → 新人体验差 |
| 根源 | 名单结构不合理，不是人力能力问题 |

**避免归因偏差**：
区分能力问题 vs 结构问题，避免简单归因

---

## 二、失败复盘机制

**停止反思铁律**：
修补失败2次 → 立刻停下来反思方向

**架构问题不能靠修补解决，必须重构**

**复盘三步**：
```
记录教训 → 提炼铁律 → 形成方法论
```

---

## 三、风险预防制度化

**提前识别风险场景**：
| 场景 | 风险点 |
|------|------|
| 供应商引入 | 准入标准不明确 |
| 定价调整 | 定价权不在自己 |
| 问题追溯 | 事后规则追溯挖坑 |

**准备标准话术**：
权责不对等保护策略：边界清单 + 留痕模板 + 决策速查表

**建立决策流程**：
关键决策留痕，避免事后追溯无法证明

---

## 四、策略性应对原则

**不只解决问题，而是重新设计机制预防问题复发**

> 示例：供应商议价困境 → 不只谈判，而是培育第二梯队、建立BPO联盟

---

## 五、时序思考原则

**不只看当下，考虑未来趋势**

| 问题 | 时序思考 |
|------|----------|
| 准入评估 | 3年后还需要吗？ |
| 业绩诊断 | 3个月后会怎样？ |
| 问题处理 | 会复发吗？如何预防？ |
