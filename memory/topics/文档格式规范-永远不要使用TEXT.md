# 文档格式规范

---

## 规则：MD文档格式规范

**模板内容**：使用引用块（>）

**永远不要使用**：代码块（``` 或 ```text```）

---

## 原因

代码块格式复制不方便。

---

## 示例

❌ **错误**：
```
收件人：XX
主题：XX
```

✅ **正确**：
> 收件人：XX
> 主题：XX

---

## 适用范围

所有创建的MD文档，包括但不限于：
- 工作文档（workspace/）
- 项目文档（projects/）
- 记忆文档（memory/）
- 任何Write工具创建的.md文件
- 特别是：邮件模板、话术模板、留痕模板
