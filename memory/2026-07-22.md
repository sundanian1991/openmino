## 供应商管理目标树设计

**需求**：用户想在 MyAgents 里为"供应商管理"搭建目标树

**过程**：
1. 我初始设计了6个平铺子目标（准入清退/业绩排名/联盟运营/合规质检/重点供应商/AI转型）
2. 用户给出更完整的三层结构设计：
   - 战略层：管理闭环体系 / 数字化转型 / 生态与赋能
   - 战术层（管理闭环下）：选用育留汰
   - 执行层：Issues
3. 用户的设计有晋升叙事（P7/P8/P9+），我的版本只是工作清单

**CLI 调研**：
- `myagents space goal list` 只读，无法创建子目标
- 缺失 `space goal create/update/delete` 命令
- 已在 MyAgents 社区提 Issue #11 请求支持

**用户设计的三层结构**：
- 战略层三根柱：管理闭环体系（执行力P7）、数字化转型（体系构建力P8）、生态与赋能（战略生态力P9+）
- 战术层用"选用育留汰"——管理学标准框架
- Issues 挂在战术层下，卡点项目挂对位置

**下一步**：用户需要在 UI 里手动创建目标树（CLI 暂不支持）
