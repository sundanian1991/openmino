# 快速捕捉日志

**用途**：随时记录时间和事项，晚上统一整理到 daily

**使用方法**：
1. 做完一件事，立刻记一条
2. 格式：`时间 + 事项 + 标签`
3. 晚上 23:00 前整理到 `memory/active/daily/YYYY-MM-DD.md`
