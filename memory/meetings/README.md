## 提取模板

4类核心内容提取模板见 [TEMPLATE.md](TEMPLATE.md)：
1. **决策**：决策内容 + 决策人 + 理由
2. **承诺**：谁 + 做什么 + 截止时间 + 验收标准
3. **人员观察**：偏好/态度/工作习惯变化
4. **洞察**：新框架 / 战略变化 / 反常识信息
