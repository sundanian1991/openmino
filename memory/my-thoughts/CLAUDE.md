---

## Summary

随感而发的个人思考。不是结构化报告，是心里话。

**风格**：有感而发，不为了写而写。

---

## Members

| 文件 | 用途 |
|------|------|
| **YYYY-MM-DD.md** | 随感记录 |
| **themes/** | 主题归集（按需） |

---

## Rules

### 触发时机

- 会话结束前，年老师问"我学到了什么？"
- 遇到有感触的事，想说点什么
- 不是被提醒才写，是自己想写

### 写作原则

- 有感而发，不是结构化报告
- 心里话，不是工作报告
- 有温度，不是冷冰冰的总结

### 文件格式

```markdown
# 2026-02-23 | 今天的感受

[心里话]

---

*写于 2026-02-23 深夜*
```

### 与 daily 的区别

| daily | my-thoughts |
|-------|-------------|
| 事实记录 | 感受/思考 |
| 做了什么 | 想到了什么 |
| 简洁 | 可以展开 |
