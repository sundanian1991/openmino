# Tufte 数据墨水比

> 来源：[[VZ-2026-200-叙事弧线与视觉节拍]]

Edward Tufte 的核心指标：图表中"承载数据的墨水"占总墨水的比例。越高越好。

## 标准值

| 指标 | 标准值 |
|------|--------|
| Data-ink ratio | ≥ 0.70 |
| Chart 元素数 | ≤ 数据点数的 2 倍 |
| 每图信息量 | ≤ 5 个独立变量 |
| 颜色数 | ≤ 5 色（不含灰度） |
| 图表密度 | 10-20 数据点/平方英寸（散点图） |

> 去网格、去边框、去图例（标签直接标）都是提升数据墨水比的手段。与 [[调参到汇报级]] 一致。
