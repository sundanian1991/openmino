# 框架式温暖设计

> 来源：[[VZ-2026-001-可视化原则总纲]] [[VZ-2026-210-五种视觉样板]]

年老师确认的风格方向：温暖、克制、有品质感。核心是用 [[格式塔七原理]] 构建氛围，而非堆砌装饰。

## 核心元素 × 原理映射

| 元素 | 格式塔原理 |
|------|-----------|
| 暖色渐变背景 | 封闭性建立氛围 |
| 太阳辐射光晕 | 闭合性+对称性 |
| 装饰弧线 | 连续性 |
| 页眉标签（小圆点+大写） | 相似性 |
| 标题局部高亮（`<em>`橙色） | 相似性对比 |
| 数字卡片（白底+淡边框+圆角） | 封闭性+接近性 |

## 五种变种

落地为 5 种样板（详见各 entity 页）：

1. [[光晕样板]] — 单侧放射渐变，单页报告
2. [[网格样板]] — 淡色网格背景，仪表盘
3. [[波浪样板]] — 底部波浪曲线，首页/封面
4. [[几何样板]] — 淡色几何散点，创意报告
5. [[极简样板]] — 圆角+淡边框，正式汇报
