# 图表叙事流

> 来源：[[VZ-2026-520-图表叙事流完整参考]]

数据 → 故事 → 决策的转化方法论（完整 19K 参考见 raw）。

## 核心流程

数据本身不产生价值，必须经过叙事转化才能驱动决策：

```
原始数据 → 可视化（看） → 故事（讲） → 决策（做）
```

> 与 [[叙事弧线]] 互补：叙事弧线讲"多图怎么编排"，本概念讲"单图怎么讲故事"。
