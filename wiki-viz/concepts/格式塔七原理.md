# 格式塔七原理

> 来源：[[VZ-2026-001-可视化原则总纲]]

描述大脑如何自动组织视觉信息的七条法则。与 [[前认知属性]] 互补——前者讲"看到什么"，本原理讲"如何分组"。

## 七原理速查

| 原理 | 一句话 | 典型手法 |
|------|--------|---------|
| 相似性 | 同色/同形 = 同类 | 灰色元素成"背景组"；核心用不同色跳出 |
| 接近性 | 靠近 = 相关 | 核心卡片间距小；用留白代替边框分组 |
| 封闭性 | 被框住 = 一组 | 浅色背景框关键区间 |
| 连续性 | 沿路径 = 相关 | 箭头引导视线 A→B→C |
| 连接性 | 有连线 = 关联 | 点+引线标注 |
| 闭合性 | 大脑补全不完整图形 | 太阳辐射光晕不需完整闭合 |
| 对称性 | 居中/对称 = 重要 | 居中大数字=核心 |
