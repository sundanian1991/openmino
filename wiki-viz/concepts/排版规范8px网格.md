# 排版规范（8px 网格）

> 来源：[[VZ-2026-200-叙事弧线与视觉节拍]]

所有间距基于 8px 倍数，保证视觉节奏统一。

## 基础间距

| 用途 | 间距值 |
|------|--------|
| 元素内紧密 | 4px / 8px |
| 卡片内 | 12px / 16px |
| 卡片/图表间 | 16px / 24px |
| 区块间 | 24px / 32px / 48px |
| 页边距 | 32px / 40px / 48px |
