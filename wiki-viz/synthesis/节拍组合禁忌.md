# 节拍组合禁忌

> 来源：[[VZ-2026-200-叙事弧线与视觉节拍]]

[[视觉节拍]] C 系列的绝对不用组合。

## 5 条禁忌

| 禁止组合 | 原因 |
|----------|------|
| C-02 色彩弹出 ×2（同图多个） | 多个"重点" = 无重点 |
| C-01 锚定数字 >3 个（同区） | 视觉打架，无主次 |
| C-19 异常闪烁 + C-02 色彩弹出 | 冲突：哪个是异常？哪个是重点？ |
| C-08 隔离 + C-12 缩放聚焦（同元素） | 隔离已足够，缩放冗余 |
| C-11 排序阶梯 + C-15 流动引导 | 都暗示序列，冲突 |
