# 图表选型决策树

> 来源：[[VZ-2026-900-图表类型全览]]

按"你想让观众看什么"选图表类型。与 [[60种图表分类总览]]（按数据目的）互补——本树按观众意图。

## 我想对比 A 和 B

| 场景 | 推荐图表 | 备注 |
|------|---------|------|
| 2-3 类别简单对比 | 条形图/柱状图 | 控制条数 |
| 实际vs目标+阈值 | **子弹图** | 含定性范围 |
| 两时期差距 | **哑铃图** | 两点连线 |
| 性别×年龄结构 | 人口金字塔 | 背靠背 |

| 累积+构成 | 堆叠面积图 |
| 两时点变化幅度 | **坡度图** |
| 累加分解 | **瀑布图** |
