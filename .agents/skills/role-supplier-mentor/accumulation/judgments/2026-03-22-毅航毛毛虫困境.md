# 2026-03-22 - 毅航与毛毛虫：被供应商绑架困境

## 背景
金条业务头部两家供应商（毅航、毛毛虫）合计占比50%，面临字节跳动放心贷高薪竞争（结算高40-50%），已发生骨干流失（2-3月30人），年老师陷入"管理收紧 → 供应商反向选择"的困境。

## 年老师的判断
- 他怎么看：从业绩看是核心供应商，几乎没有替代选项
- 为什么：产能不可替代、业务连续性风险
- 有什么顾虑：管理工具失效，任何动作都会反噬自己

## 导师的分析

### 事实梳理
| 维度 | 数据 |
|------|------|
| 头部两家占比 | 50% |
| 第三名占比 | 约10%，且不稳定 |
| 已流失人数 | 30人（骨干） |
| 外部溢价 | 40-50%（字节1.4-1.5万 vs 京东1万） |
| 供应商位置 | 都在合肥，相距20公里 |

### 根本原因（重新诊断）

**不是"供应商绑架"，而是"管理收紧触发反向选择"**

| 内部因素 | 外部因素 | 结果 |
|----------|----------|------|
| 结算要求更严 | 字节结算高40-50% | 供应商有退路 |
| 费率降低 | 字节放心贷扩招 | 供应商选择离开 |
| 出勤要求按人力标准 | 毅航已与字节合作 | 骨干流失 |

**关键洞察**：年老师的管理收紧是正确的，但时机不对——在供应商有明确替代选项时收紧，等于把供应商推向竞争对手。

### 问题诊断

**表层问题**：外部高薪挖人 → 坐席流失
**中层问题**：管理收紧 → 供应商收入下降 → 选择离开
**深层问题**：
1. 供应商结构失衡 → 没有替代选项
2. 新供应商培育失败 → 去年引入两家，管理跟不上
3. 第三名不稳定 → 无法承接头部产能

### 认知模式应用

1. **逆向反射**：不是问"怎么惩罚"，而是问"为什么它们有退路"
2. **时间深度**：管理收紧是长期的正确方向，但短期执行时机不对
3. **战时意识**：现在是"战时"，需要特殊处理，不是常规管理

### 核心洞察

年老师做了一个正确的长期决策（收紧管理），但在错误的时机执行（供应商有替代选项）。

这导致：
- 毅航已经有字节作为退路
- 毛毛虫正在接触字节
- 两者形成默契：京东不划算就走

**这不是"被绑架"，是"你的筹码不够"**

## 状态更新（2026-03-22 后续）

### 当前局面
- **骨干流失已发生**：毅航、毛毛虫的资深员工已转向字节
- **人员置换完成**：我们使用对方招聘的新人，字节吸收了资深员工
- **相对稳定**：人员流动仍在持续，但局面可控
- **关键变量**：新人成长速度

### 结构性问题（未解决）
1. **头部集中度过高**：毅航+毛毛虫占比50%，外部挖人风险持续存在
2. **中腰部承接能力不足**：第三名及以后的供应商无法承担更大业务，供应链弹性不足

### 组织结构约束
- 结算定价权在策略组（刘伟佳）
- 年老师无定价权，需承担供应商稳定责任
- 策略组KPI（费效比）与年老师目标（供应商稳定）存在冲突

### 策略组背景
- 负责人：刘伟佳
- KPI：费效比优化（降成本）
- 可能的协同点：供应商流失 → 产能下降 → 费效比恶化

## 年老师的应对策略

### 已在执行
1. **BPO联盟**：把自营团队成熟的培训体系（AI辅助）交给中腰部供应商
2. **新业务分配**：新业务优先给中腰部供应商
3. **第二梯队培育**：给第三、四、五名设定更高目标，小批量增补

### 新人培训体系
| 维度 | 供应商 | 自营团队 |
|------|--------|----------|
| 3个月流失率 | 30-50% | 10%+ |
| 培训方式 | 自行培训 | AI辅助 + 问答 + 系统培训 |
| 成长周期 | 1个月独立产出，3个月养活自己 | 同 |

**关键洞察**：自营团队流失率是供应商的1/3-1/5，说明培训体系有巨大差异。

### 目标路径
```
短期：BPO联盟推广 → 降低新人流失率 → 中腰部承接能力提升
中期：第二梯队形成 → 第三、四、五名规模扩大 → 抗风险能力增强
长期：第二梯队挑战头部 → 供应商结构健康化 → 降低头部依赖
```

## 与策略组的沟通僵局

### 年老师的做法
- 提出建立定价委员会，要求定价需服务组同意
- 事前与策略组老板、刘伟佳沟通不充分
- 方式太直接，导致对抗情绪

### 刘伟佳的反应（行为模式分析）

**表面言论**：
- "担心供应商不好管理"
- 强调供应商人力不足

**实际行为**：
- 要求头部供应商增人 → 但增人重点是头部职场 → 仍短缺
- 批评中腰部业绩不佳 → 但不解决结构性问题

**潜台词**：用"供应商管理问题"维护自己的定价权，把责任推给年老师。

### 沟通策略建议
- 不要直接要求分权（定价委员会）
- 要用"共同优化费效比"的框架
- 论点：结算太低 → 供应商不投入 → 产出下降 → 费效比恶化

## 京东的供应商培养悖论

```
京东的培养路径：
供应商小 → 京东给业务、给资源 → 供应商做大 → 其他甲方发现 → 供应商向外扩张 → 有了退路 → 开始议价/流失
```

**毅航和毛毛虫就是这个路径的产物**：
- 它们是京东培养出来的头部
- 现在有能力接字节的业务
- 字节给出更高结算，自然选择离开

**这不是背叛，是模式的必然结果。**

## 结果
- 短期：相对稳定，新人成长中，矛盾可能逐步消退
- 中期：BPO联盟推广，第二梯队培育
- 长期：结构性问题待解决，需要与策略组建立共识

## 复盘
- 年老师做得好的：识别到结构性问题，启动BPO联盟，规划第二梯队
- 可以加强的：与策略组的沟通方式，避免直接对抗
- 判断力积累：供应商培养存在"做大就走"的悖论，需要提前布局替代选项
