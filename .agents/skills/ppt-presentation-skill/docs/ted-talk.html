A Final Talk
我的申辩
苏格拉底  |  雅典 · 公元前399年
"一个70岁老人的最后演讲"
今天，我第一次走进法庭。
我70岁了。从未打过官司。
但今天，501个人要决定我的生死。
2
两项指控
① 腐蚀青年
② 不信城邦之神
但真正的原因？我说了太多真话。
24年前，一个喜剧作家把我写成了小丑。
从那以后，谣言从未停止。
今天我不是在和三个人战斗——
我是在和影子战斗。
有一天，我最好的朋友跑去问神：
"有没有人比苏格拉底更聪明？"
神说：没有。
这句话改变了我的一生。
我不相信。
所以我决定证明神是错的。
我去找了最有权势的人、
最有才华的人、最有技术的人。
结果——
政客
自以为无所不知
→ 实际一无所知
诗人
靠灵感写作
→ 解释不了自己的作品
工匠
手艺精湛
→ 但以为自己什么都懂
名声越大，智慧越少。
"我唯一知道的，
就是我一无所知。"
这就是为什么神说我最有智慧。
不是因为我知道得多——
而是因为只有我知道自己不知道。
每次我揭穿一个人——
他就恨我。
然后你们的孩子开始学我。
然后更多人恨我。
美勒托斯说整个雅典都在改善青年，只有我在腐蚀。
我问他：马呢？
是所有人都能训好马，还是只有专业驯马师？
他答不上来了。
一个人腐蚀，全城改善？荒谬。
你的控告书说我"信奉神灵之事"。
同时又说我不信神。
美勒托斯，这就像说——
骡子存在，但马和驴不存在。
自相矛盾。
雅典是一匹高贵的骏马。
体型庞大，行动迟缓。
我是神派来叮它的牛虻。
你们可以拍死我。
但之后——谁来叫醒你们？
两次勇气

民主之下

所有人都说处死将军。
我一个人说不。

暴政之下

命令我去抓人。
我回家了。

无论什么制度，我只听正义的。
我有三个儿子。
但我不会把他们带来哭给你们看。
因为你们宣誓依法审判——
不是施舍同情。
有罪
30票之差。
你们问我该受什么罚？
我的建议：免费吃饭。
在公共食堂。像奥林匹亚冠军一样。
因为冠军给你们幸福的假象——
我给你们真的。
最终罚金：30弥那。朋友们替我出的。

ὁ ἀνεξέταστος βίος οὐ βιωτὸς ἀνθρώπῳ
死刑
我不怕死。我怕的是不义。
不义比死亡跑得更快。
死亡追上了我——不义追上了你们。
我死后，更多人会来质问你们。
他们更年轻，更无情。
杀人堵不住嘴。改善自己吧。
两种可能——

无梦的长眠

永恒不过是一个好觉。

赚了。

灵魂的旅行

遇见荷马！继续提问！

而且——那边不会因为提问就判死刑。

好人无论生死都不会遭受真正的恶。
离别的时刻已经到来。
我去赴死，你们去生活。
哪一个更好，只有神知道。
