## 分类 A：数据与数值（用图表）

> 有明确数据，需要直观呈现数量、趋势、分布、关系。

| 编号 | 模式 | 表达什么 | 典型场景 |
|------|------|---------|---------|
| 01 | 饼图 | 占比（≤3 类） | 资源分配、份额 |
| 02 | 维恩图 | 交集与独有 | 能力重叠、受众交叉 |
| 03 | 同心图 | 由内而外的层级 | 核心→外围、圈层扩散 |
| 04 | 圆形图 | 多维度环形对比 | 多指标环形比较 |
| 05 | 气泡图 | 三维数据（x/y/大小） | 市场定位、多维度对比 |
| 06 | 气泡演变图 | 同一对象多阶段变化 | 竞争格局演变 |
| 07 | 折线图 | 趋势变化 | 业绩、温度、增长率 |
| 08 | 面积图 | 趋势+累积量 | 市场份额累积、流量 |
| 09 | 散点图 | 两变量相关性 | 人效vs成本、投入vs产出 |
| 10 | 旭日图 | 多层级占比分解 | 组织架构、品类嵌套 |
| 11 | 扇形图 | 扇形占比 | 方向性分布 |
| 12 | 风玫瑰图 | 方向+强度 | 风向、客户来源方向 |
| 13 | 柱状图 | 分类对比 | 业绩排行、地区对比 |
| 14 | 磁带图 | 分段占比对比 | 多组分段对比 |
| 15 | 甘特图 | 时间+进度 | 项目计划、排期 |
| 16 | 矩形树图 | 嵌套占比 | 文件占比、预算分配 |
| 17 | 网格图 | 矩阵式密集数据 | 日历、排班表 |
| 18 | 元素周期表 | 分类+属性的二维网格 | 能力矩阵、特性对照 |
| 19 | 弧线图 | 节点间连接关系 | 合作关系、资金流向 |
| 20 | 桑基图 | 流入→流出→转化 | 客户转化、资金流向 |
| 21 | 弦线图 | 双向关系网络 | 跨部门协作、贸易往来 |
| 22 | 雷达图 | 多维能力画像 | 个人/供应商能力评估 |
> **雷达图使用限制**：仅 ≤5 个维度时使用。≥6 维或 ≥4 个对比对象时，雷达图重叠严重、重点不清，改用**水平条形图组**（每维一条，对象分列）或**小多图矩阵**（每对象一张迷你雷达图）。雷达图不适合"一眼看清谁最大"的场景——读者必须先看图例→找颜色→对照形状，认知路径过长。优先用条形图（长度直接映射大小）表达排名/差异。
| 23 | 极坐标网格图 | 极坐标上的分布 | 周期性数据 |
| 24 | 螺旋图 | 螺旋递进 | 成长路径、迭代周期 |
| 25 | 时间线 | 事件序列 | 项目里程碑、发展历程 |
| 26 | 并列比较 | 两组数据逐项对比 | A/B 对比、前后对比 |
| 27 | 抽象树 | 层级分解 | 组织、分类、决策链 |
| 28 | 思维导图 | 放射状关联 | 头脑风暴、知识网络 |
| 29 | 决策树 | 分支决策路径 | 条件判断、策略选择 |
| 30 | 方框图 | 结构化层级流程 | 组织架构、系统模块 |

## 模式匹配决策树

```
有精确数据？
├─ 是 → 数据有几个维度？
│  ├─ 1 维（单指标）→ 柱状图(13)、折线图(07)、面积图(08)
│  ├─ 2 维（x/y 关系）→ 散点图(09)、气泡图(05)
│  ├─ 3 维（x/y/大小）→ 气泡图(05)
│  ├─ 占比 → 饼图(01，≤3 类)、矩形树图(16)、堆叠条(14)
│  ├─ 趋势 → 折线图(07)、面积图(08)
│  ├─ 多层级 → 旭日图(10)、矩形树图(16)、抽象树(27)
│  ├─ 流程/转化 → 桑基图(20)、漏斗(32)、甘特图(15)
│  ├─ 多维对比 → 雷达图(22)、并列比较(26)
│  └─ 网络/关系 → 弦线图(21)、弧线图(19)、桑基图(20)
│
└─ 否 → 要表达的关系本质是什么？
   ├─ 层级/优先级 → 金字塔(31)、同心图(03)、层次(55)
   ├─ 衰减/筛选 → 漏斗(32)、冰山(49)
   ├─ 循环/闭环 → 循环轮(34)、循环系统(60)
   ├─ 联动/依赖 → 齿轮(39)、桑基图(20)
   ├─ 平衡/权衡 → 天平(42)、杠杆(41)
   ├─ 中心辐射 → 辐轮(33)、宇宙(53)
   ├─ 阶梯/进阶 → 楼梯(35)、进化(64)
   ├─ 路径/方向 → 道路(71)、地铁路线图(37)、迷宫(47)
   ├─ 成长/发展 → 树(59)、根源(58)、进化(64)
   ├─ 波动/起伏 → 过山车(56)
   ├─ 集中/分散 → 聚集(66)、气泡演变图(06)
   ├─ 互补/完整 → 拼图(40)、成套工具(69)
   ├─ 隔离/独立 → 岛屿(51)
   ├─ 夹心/压力 → 三明治(52)
   ├─ 假设验证 → 实验(67)
   ├─ 规模化 → 工厂(68)、传送带(70)、机器(72)
   ├─ 竞争/耐力 → 马拉松(63)、食物链(65)
   ├─ 生态/共存 → 城市生活(62)、建筑中的活动(61)
   ├─ 周期/节奏 → 钟面(54)
   ├─ 知识/积累 → 书架(57)
   └─ 情绪/状态 → 切尔诺夫脸谱图(43)

需要表达逻辑关系？
├─ 是 → 分类 E（C01-C16）
│  ├─ 分解/MECE → C01
│  ├─ 变化驱动/同比 → C02/C12
│  ├─ 优先级/定位/四象限 → C03
│  ├─ 评估对比/选择困难 → C05/C13
│  ├─ 瓶颈识别/转化衰减 → C06
│  ├─ 趋势叙事/拐点 → C07
│  ├─ 差距目标/进度 → C09/C14
│  └─ 根因分析/假设验证 → C16
└─ 否 → 继续走分类 A/B/C/D
```
