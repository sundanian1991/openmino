# 使用示例

> viz-design 三种入口模式的典型用法。

---

**示例 1（模式 A 逐步引导）：年老师说"这批数据帮我看看能做几张图"**

→ 0: 数据全景分析 — 列出所有分析角度，等年老师确认
→ Phase 1-4: 按确认后的方案走完整四步流程

**示例 2（模式 B AI 推荐）：年老师说"拿这批数据做最佳方案"**

→ 0: AI 自主决策 — 分析数据后直接出完整图表组合方案
→ Phase 1-4: 直接走完整四步流程，不等待确认

**示例 3（模式 C 自定义）：年老师说"帮我画个柱状图对比 5 家供应商的业绩"**

→ Phase 1+2: 口头确认意图和模式，不写文件
→ Phase 3: spec.md — 完整 SPEC（渲染契约不可省略）→ 委托 viz-echarts → 写入 spec.md
→ Phase 4: checklist.md — 验收 → 写入 checklist.md
