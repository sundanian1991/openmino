# 图表功能变体

> 按功能分类组织图表变体，每个功能类别下提供推荐模式、备选模式、变体说明和选择决策树。对标 fe-cinematic 的 section-archetypes.md + section-functions.md。

---

## 一、趋势

> 数据随时间或有序维度的变化。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 折线图 | SK-07 | 连续时间序列，关注走向和拐点 |
| 面积图 | SK-08 | 趋势+累积量，需要看到总量 |
| 小多图 | SK-04（构图）+ 折线 | 多个对象同一趋势，需要逐一查看 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 柱状图 | SK-13 | 时间点较少（≤8 个），需要精确读数 |
| 面积堆叠 | C08 堆叠构成 | 趋势+构成同时要看 |
| 时序脉冲 | VB-16 视觉节拍 | 有明显周期性波动 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 单线折线 | 一条线，简洁 | 单指标趋势 |
| 多线折线 | 多条线，颜色区分 | 多指标对比趋势 |
| 面积填充 | 线下填充 | 强调量的累积 |
| 拐点标注 | 圆点+注释 | 有关键转折事件 |
| 预测区间 | 后半段半透明带 | 需要表达不确定性 |

### 选择决策树

```
时间点数量？
├─ ≤8 个 → 柱状图（精确读数优先）
├─ 9-20 个 → 折线图
└─ >20 个 → 折线图（简化数据点标签）

需要看累积量？
├─ 是 → 面积图
└─ 否 → 折线图

有多个对象要对比？
├─ ≤4 个 → 多线折线
├─ 5-12 个 → 小 multiples（每个对象一张小图）
└─ >12 个 → 热力图（行=对象，列=时间）

有预测/不确定性？
├─ 是 → 折线+半透明预测区间
└─ 否 → 纯折线
```

## 二、占比

> 整体中各部分的相对大小。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 堆叠条形图 | C08 堆叠构成 | 类别 3-7 个，需要精确比较 |
| 饼图 | SK-01 | 类别 ≤3 个，强调整体感 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 矩形树图 | SK-16 | 类别 >7 个，有层级嵌套 |
| 旭日图 | SK-10 | 多层级占比分解 |
| 百分比堆叠条 | C08 变体 | 对比多个组的构成 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 饼图 | 扇形角度=占比 | ≤3 类，快速感知 |
| 堆叠条 | 分段填充 | 多类别，可精确读数 |
| 百分比堆叠条 | 所有条等长，内部按比例分色 | 对比构成的变化 |
| 矩形树图 | 矩形面积=占比 | 多类别嵌套 |
| 环形图 | 饼图中间挖空 | 中心可放总数值 |

### 选择决策树

```
类别数量？
├─ ≤3 个 → 饼图/环形图
├─ 4-7 个 → 堆叠条形图
├─ 8-20 个 → 矩形树图
└─ >20 个 → 排序条形图（只展示 Top 10+Other）

需要多层级？
├─ 是 → 旭日图/矩形树图
└─ 否 → 堆叠条形图

需要对比多组的构成？
├─ 是 → 百分比堆叠条
└─ 否 → 堆叠条形图
```

## 三、对比

> 两个或多个对象在某个或多个维度上的差异。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 柱状图 | SK-13 | 分类对比，排行或数值比较 |
| 并列比较 | SK-26 | 两组数据逐项对比 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 雷达图 | SK-22 | 多维能力对比 |
| 2x2 矩阵 | C03 | 两维度四象限定位 |
| Harvey Ball 矩阵 | C13 | 定性评估网格 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 简单柱状 | 一组柱子 | 单指标对比 |
| 分组柱状 | 每组多个柱子并排 | 多指标对比 |
| 排序柱状 | 按值降序排列 | 排名场景 |
| 蝴蝶图 | 左右对称反向柱子 | A vs B 直观对比 |
| 棒棒糖图 | 线+圆点替代柱子 | 数据点密集时更清爽 |

### 选择决策树

```
对比维度数？
├─ 1 个维度 → 柱状图/棒棒糖图
├─ 2 个维度 → 分组柱状/2x2 矩阵
└─ >2 个维度 → 雷达图

对比对象数？
├─ 2 个 → 蝴蝶图/并列比较
├─ 3-8 个 → 排序柱状图
├─ 9-20 个 → 水平条形图（标签可读）
└─ >20 个 → Top N + Other

是定性评估？
├─ 是 → Harvey Ball 矩阵
└─ 否 → 定量柱状图
```

## 四、转化

> 从一个阶段/状态到另一个的流动和衰减。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 漏斗图 | SK-32 | 递减转化，关注瓶颈 |
| 桑基图 | SK-20 | 流入→流出→转化，多路径 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 瀑布图 | C02 瀑布分析 | 数值变化分解 |
| 弦线图 | SK-21 | 双向关系网络 |
| 桥接图 | C12 | 阶段间变化拆解 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 标准漏斗 | 上宽下窄梯形分层 | 线性转化 |
| 标注瓶颈 | 关键层右侧标注 | 找到最大流失点 |
| 桑基分流 | 多条分流路径 | 多来源→多去向 |
| 瀑布桥接 | 正负增量柱状 | 从 A 到 B 的驱动因素 |

### 选择决策树

```
转化路径数量？
├─ 单一路径（线性） → 漏斗图
├─ 多路径分流 → 桑基图
└─ 只有起点和终点的差值 → 瀑布图

需要标注瓶颈？
├─ 是 → 漏斗图 + 最大流失率标注
└─ 否 → 标准漏斗/桑基

是双向关系？
├─ 是 → 弦线图
└─ 否 → 桑基图
```

## 五、关系

> 两个或多个变量之间的关联或网络结构。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 散点图 | SK-09 | 两变量相关性 |
| 气泡图 | SK-05 | 三维度关系（x/y/大小） |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 弧线图 | SK-19 | 节点间连接关系 |
| 弦线图 | SK-21 | 双向关系网络 |
| 桑基图 | SK-20 | 流向关系 |
| 热力图 | 密度变换 VB-10 | 两变量交叉密度 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 简单散点 | 点分布 | 两变量 |
| 回归线+散点 | 点+趋势线 | 有相关性假设 |
| 气泡图 | 点大小映射第三维 | 三维度 |
| 气泡演变 | 气泡随时间移动 | 动态三维度 |
| 弧线连接 | 节点间弧线 | 网络/合作关系 |

### 选择决策树

```
变量维度数？
├─ 2 个 → 散点图
├─ 3 个 → 气泡图
├─ 3 个+时间 → 气泡演变图
└─ 网络关系 → 弧线图/弦线图

需要表达相关性？
├─ 是 → 散点图 + 回归线
└─ 否 → 纯散点图

节点数量？
├─ <20 个 → 弧线图
├─ 20-50 个 → 弦线图
└─ >50 个 → 力导向图（需交互）
```

## 六、分布

> 数据在某个空间/维度上的散布模式。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 散点图 | SK-09 | 两变量分布 |
| 热力图 | VB-10 密度变换 | 二维密度分布 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 极坐标网格图 | SK-23 | 极坐标上的分布 |
| 地图 | SK-48 | 地理空间分布 |
| 网格图 | SK-17 | 矩阵式密集数据 |
| 箱线图 | viz-echarts 支持 | 统计分布+异常值 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 散点分布 | 原始数据点 | 精确值可见 |
| 密度热力 | hexbin/kernel density | 点密集时看模式 |
| 极坐标分布 | 角度+半径 | 周期性/方向性 |
| 地图着色 | 地理区域颜色映射 | 空间分布 |

### 选择决策树

```
分布维度？
├─ 两变量 → 散点图
├─ 二维密度 → 热力图
├─ 地理空间 → 地图
└─ 周期性 → 极坐标网格图

数据密度？
├─ 稀疏（<100 点） → 散点图
├─ 中等（100-1000） → 半透明散点
└─ 密集（>1000） → 热力图/hexbin

需要看统计摘要？
├─ 是 → 箱线图
└─ 否 → 散点/热力图
```

## 七、多维

> 同时展示 3 个以上维度的信息。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 雷达图 | SK-22 | 多维能力画像（4-8 维） |
| 气泡图 | SK-05 | 三维度 + 颜色第四维 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 平行坐标 | viz-echarts 支持 | >5 维度，数值型 |
| 元素周期表 | SK-18 | 分类+属性的二维网格 |
| Harvey Ball 矩阵 | C13 | 定性多维评估 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 标准雷达 | 多边形覆盖 | 单对象多维 |
| 多雷达叠加 | 多个半透明多边形 | 多对象对比 |
| 气泡四维 | x/y/大小/颜色 | 四维数据 |
| Harvey Ball | 定性填充网格 | 无法量化的维度 |

### 选择决策树

```
维度数量？
├─ 3-4 个 → 气泡图（x/y/大小/颜色）
├─ 4-8 个，可量化 → 雷达图
├─ >8 个，数值型 → 平行坐标
└─ 定性评估 → Harvey Ball 矩阵

需要多对象对比？
├─ ≤3 个 → 雷达叠加
├─ 4-10 个 → 分组柱状（每个维度一组）
└─ >10 个 → Harvey Ball 矩阵
```

## 八、流程

> 步骤、顺序、时间进度的表达。

### 推荐模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 时间线 | SK-25 | 事件序列，里程碑 |
| 甘特图 | SK-15 | 时间+进度 |

### 备选模式

| 模式 | 编号 | 何时用 |
|------|------|--------|
| 传送带 | SK-70 | 流水线式推进 |
| 齿轮 | SK-39 | 联动/依赖机制 |
| 方框图 | SK-30 | 结构化层级流程 |
| 地铁路线图 | SK-37 | 多线路交叉+换乘节点 |

### 变体说明

| 变体 | 画法差异 | 适用差异 |
|------|---------|---------|
| 水平时间线 | 左→右，节点在线上/下交替 | 事件不多 |
| 甘特条 | 水平条，长度=持续时间 | 项目排期 |
| 依赖箭头 | 条间箭头表示依赖 | 有关联约束 |
| 泳道图 | 多行并行时间线 | 多角色/多业务线 |

### 选择决策树

```
表达什么？
├─ 事件序列 → 时间线
├─ 项目进度 → 甘特图
├─ 流水线流程 → 传送带/方框图
└─ 多线路交叉 → 地铁路线图

节点数量？
├─ <10 个 → 简单时间线
├─ 10-30 个 → 甘特图/泳道图
└─ >30 个 → 可交互时间线

有依赖关系？
├─ 是 → 甘特图 + 依赖箭头
└─ 否 → 简单时间线
```
