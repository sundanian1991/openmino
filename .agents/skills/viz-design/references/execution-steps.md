# Execution Steps · 视觉执行步骤

> 可插拔模块。所有渲染技能（viz-echarts / viz-svg-flow / viz-editorial）共用。
> 目标：不同 AI 执行同一 SPEC 时，产出一致，不靠感觉画。

---

## 六层绘制法

每张图的绘制**严格按以下顺序**，从第 1 层到第 6 层逐层叠加。
禁止跳层、禁止倒序、禁止先画高亮再补灰化。

```
第 1 层 → 画布底色
第 2 层 → 标题 + 副标题
第 3 层 → 图表骨架（坐标轴 / 网格 / 主轴）
第 4 层 → 数据层（全部灰化 — Stone 300 基准色）
第 5 层 → 高亮层（≤10% 元素着色）
第 6 层 → 标注 + 脚注
```

### 第 1 层：画布底色

| 要素 | 值 |
|------|----|
| 背景色 | `#faf9f7`（Warm 50）或透明 |
| 卡片底色 | `#ffffff` |
| 圆角 | 10px |
| 内边距 | 16px |

**为什么先画底色**：底色决定了所有后续元素的对比度基准。后画底色会覆盖内容。

### 第 2 层：标题 + 副标题

| 要素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 主标题 | 15-18px | 600 | Warm 800 `#6b3410` |
| 副标题 | 11-12px | 400 | Stone 500 `#857d74` |

- 标题到内容间距 ≥12px
- 副标题到标题间距 6px
- 标题必须是结论，不是描述

### 第 3 层：图表骨架

根据模式类型加载对应的骨架：

| 模式 | 骨架内容 |
|------|---------|
| 折线图/柱状图/散点图 | X/Y 轴、刻度线、网格线（Stone 300 虚线） |
| 雷达图 | 多边形网格（Stone 300）、轴标签 |
| 漏斗图 | 各阶段梯形轮廓（Stone 300 描边） |
| 时间线 | 主轴直线、刻度点 |
| SVG 架构图 | 节点占位框、连线占位 |

**骨架的颜色一律用 Stone 300**，不得提前用主色。

### 第 4 层：数据层（全部灰化）

**这是关键层。先把所有数据用 Stone 300/500 灰化绘制，不管叙事意图。**

| 元素 | 灰化色 |
|------|--------|
| 柱形/条形 | Stone 300 填充 |
| 折线 | Stone 500 线条，1.5px |
| 气泡 | Stone 300 填充，Stone 500 描边 |
| 雷达图数据 | Stone 300 描边填充 |
| 漏斗层级 | Stone 300 填充 |

**为什么先灰后彩**：灰化是默认态。高亮是偏离默认态的信号。如果先画彩色再"灰化其余"，容易残留不该有的颜色。

### 第 5 层：高亮层（≤10%）

在灰化的数据层上，**只覆盖**需要高亮的元素：

| 高亮方式 | 做法 |
|----------|------|
| 单元素着色 | 将灰化元素替换为 Warm 500/800 或语义色 |
| 加粗线条 | 1.5px → 3px |
| 放大节点 | radius × 1.5 |
| 描边加深 | Stone 300 → Coral 500 |

**约束**：
- 高亮元素 ≤ 总数的 10%
- 每张图只突出 **1 个关键点**
- 跨色必须有语义理由（Warm=主数据, Teal=正面, Coral=负面）

### 第 6 层：标注 + 脚注

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 标注文字 | 11px | 600 | 依语义 |
| 脚注 | 10px | 400 | Stone 400 `#ada599` |
| 数据来源 | 10px | 400 | Stone 400 |

- 标注写**原因+幅度**，不写数字本身
- 脚注包含时间范围和数据来源
- 标注位置不得遮挡数据元素

## 常见图表骨架坐标参考

### 折线图骨架

```
画布: 600×300
标题区: y=0-30
图表区: y=40-260
  X 轴: y=260, x=60→580
  Y 轴: x=60, y=40→260
  网格线: y=80,140,200 (Stone 300, dashed)
脚注区: y=270-290
```

### 水平条形图骨架

```
画布: 700×400
标题区: y=0-30
图表区: y=40-360
  Y 轴(类别): x=60
  条形区: x=140→640
  每条高度: 24px, 间距: 20px
脚注区: y=370-390
```

### 雷达图骨架

```
画布: 800×550
中心: (400, 280)
外圈半径: 180
网格圈数: 4 (25%, 50%, 75%, 100%)
轴数: 依维度数
```

## 渲染技能映射

执行完六层绘制法后，按模式类型委托对应技能：

| 模式范围 | 渲染技能 | 原因 |
|---------|---------|------|
| 01-30（数据图表） | viz-echarts | ECharts 原生支持 |
| 31-48（抽象类比） | viz-svg-flow | SVG 手绘形态 |
| 49-60（模拟类比） | viz-svg-flow | 自然物象需手绘 |
| 61-72（寓言） | viz-svg-flow | 场景类需手绘 |

委托时传入：完整 SPEC + 本文件的六层绘制法。
