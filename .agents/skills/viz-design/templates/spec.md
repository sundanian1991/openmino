## 叙事意图

## 视觉执行

- **模式选择**：编号-名称 + 匹配理由
- **标题**：主标题（结论性）+ 副标题（时间+对象）
- **视觉编码**：X/Y/颜色/大小各编码什么
- **数据组织**：字段 / 排序 / 聚合
- **标注策略**：高亮 ≤10% / 写原因+幅度
- **图例与辅助**：图例 / 脚注 / 特殊说明
- **布局**：画布尺寸 + 版式 + 留白
