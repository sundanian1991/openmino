# The Gap Between 小孩 and 大人

> 同样的组件，不同的流程，完全不同的质量。

---

## 核心问题

为什么同样的 Hallmark 技能，之前做的是"小孩"，这次 Haglöfs 是"大人"？

答案不是组件，是**流程**。

---

## 两种流程

### 小孩的流程（组件驱动）

```
用户要 showcase → 打开组件库 → 选 Hero + Feature Grid + CTA → 填内容 → 完成
```

- 起点：我有什么组件
- 组件选择：选好看的
- 颜色：好看就行
- 布局：标准网格
- 细节：够用就行

**结果：** 模板化，没有灵魂。

### 大人的流程（理念驱动）

```
研究材料 → 发现张力 → 视觉表达 → 组件选择 → 克制使用 → 细节精度
```

- 起点：内容的核心是什么
- 组件选择：选能表达张力的
- 颜色：每个色号都在表达张力
- 布局：打破矩形，创造张力
- 细节：1px 都在思考

**结果：** 有灵魂，有高级感。

---

## 为什么这次 Haglöfs 不同

### 第一步：先研究

我没有直接选组件。我搜索了 Nothing Phone 和 Haglöfs 的设计哲学，理解了它们的核心理念。

**发现：** Haglöfs 的核心张力是 Nature vs Tech。

### 第二步：用视觉表达张力

| 张力 | 视觉表达 |
|------|---------|
| Nature ↔ Tech | 色卡：Core（自然灰调）→ Signal（技术亮色）|
| Heritage ↔ Future | 字体：20世纪 serif 传统 + 现代 mono |
| Organic ↔ Engineered | 符号：有机十字 + 几何精度 |
| Fixed ↔ Fluid | 斜切：稳定矩形被斜线打破 |

### 第三步：组件是"从理念长出来的"

不是"我有这个组件就用"，而是"这个内容需要什么样的表达"。

- 色卡展示 → 不是标准卡片，而是正方形 + 微缩放 hover
- 数字网格 → 不是 Bento 的不规则，而是 4×2 均匀分布
- 张力网格 → 不是 Feature Grid，而是 2×2 + 中间分割线

### 第四步：克制使用

我没有用所有组件。我只用了能表达张力的组件。

- 没有用 Testimonials（品牌设计系统不需要）
- 没有用 CTA（这不是营销页面）
- 没有用标准卡片（色卡需要自己的表达）

### 第五步：细节精度

- 箭头是 1px 线 + 8px 三角，不是粗箭头
- hover 效果是微缩放（scale 1.02），不是放大
- section padding 是 120px，不是 40px

## 对比

| 维度 | 小孩 | 大人 |
|------|------|------|
| **起点** | 我有什么组件 | 内容的核心是什么 |
| **组件选择** | 选好看的 | 选能表达张力的 |
| **颜色** | 好看就行 | 每个色号都在说张力 |
| **布局** | 标准网格 | 打破稳定，创造张力 |
| **节奏** | 随机交替 | 有呼吸的节奏 |
| **细节** | 够用就行 | 1px 都在思考 |

---

## 如何让"大人"成为默认

### 1. 先研究，后设计

在选组件之前，先理解内容的核心张力。

### 2. 组件是结果，不是起点

先有理念，再有组件。组件是从理念"长出来的"，不是"套上去的"。

### 3. 克制使用

不是"能用就用"，而是"只用能表达张力的"。

### 4. 细节精度

1px 都在思考。每个像素都是设计决策，不是意外。

## 检查清单

在交付前，问自己：

- [ ] 我研究过这个材料吗？
- [ ] 我发现了核心张力吗？
- [ ] 我用一句话说清楚了吗？
- [ ] 每个颜色都在表达张力吗？
- [ ] 每个字体都在表达张力吗？
- [ ] 布局在表达张力吗？
- [ ] 我克制使用组件了吗？
- [ ] 细节精度够吗？（1px 都在思考）
- [ ] 留白给内容呼吸了吗？
- [ ] 色彩是"说"而不是"喊"吗？

---

## 反模式

1. **组件驱动** — "我有这个组件，我来填充内容"
2. **跳过研究** — 直接选组件，不理解内容
3. **过度使用** — 能用就用，不克制
4. **纯色呐喊** — 用 #FF0000 而不是 #E8453C
5. **填满空间** — 40px padding，不给呼吸
6. **忽略细节** — "够用就行"
