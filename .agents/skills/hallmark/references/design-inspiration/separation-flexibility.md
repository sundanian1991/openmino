# Separation and Flexibility

> The best brands separate their wordmark from their symbol — and that separation unlocks everything.

---

## Core Idea

A fixed logotype lockup — wordmark and symbol always together, always the same relationship — is the most common brand identity pattern. It's also the most limiting. When wordmark and symbol are locked, every application must accommodate both. The symbol can't stand alone. The wordmark can't breathe.

The best design systems separate them. The symbol works alone (product marks, favicons, social avatars). The wordmark works alone (editorial, typographic contexts). And when they come together, the relationship is flexible — not fixed.

**Haglöfs example:** Separating the wordmark from the symbol "unlocks greater flexibility and versatility in product applications, breaking the static monotony of a fixed logotype lockup and allowing for more dynamic and engaging expressions."

---

## The Separation Spectrum

Identities exist on a spectrum from locked to fully separated:

```
Locked                    Flexible                   Fully Separated
┌─────────┐              ┌─────────┐                ┌─────┐ ┌──────────┐
│ [▲] Name │              │ [▲]     │                │ [▲] │ │ Name     │
└─────────┘              │   Name  │                └─────┘ │          │
                         └─────────┘                          └──────────┘
```

| Level | Description | Example |
|-------|-------------|---------|
| **Locked** | Symbol + wordmark always together, fixed ratio | Most logos |
| **Flexible** | Symbol can appear above/beside/without wordmark | Haglöfs |
| **Separated** | Symbol and wordmark are independent assets | Apple |

---

## Why Separate

### 1. Context Adaptability

Different contexts need different treatments:

| Context | Needs |
|---------|-------|
| **Product** | Symbol only (small, repeated) |
| **Editorial** | Wordmark only (typographic, readable) |
| **Marketing** | Both, flexible relationship |
| **Social** | Symbol (avatar), wordmark (cover) |

A locked logotype forces the same treatment everywhere. Separation lets each context use what fits.

### 2. Scale Flexibility

At small sizes (favicon, app icon), a detailed wordmark is illegible. A symbol works. At large sizes (hero, signage), the wordmark has room to breathe. Separation lets each scale appropriately.

### 3. Dynamic Expression

When wordmark and symbol are separate, designers can create compositions — symbol here, wordmark there, overlapping, adjacent, stacked. The relationship becomes a design variable, not a constraint.

---

## How to Apply

### 1. Design Both, Then Separate

Design the lockup first — the ideal relationship between symbol and wordmark. Then pull them apart. Each must work alone:

- **Symbol alone:** Does it carry the brand? Is it recognizable?
- **Wordmark alone:** Is it legible? Does it have personality?
- **Together:** Does the relationship feel balanced, not forced?

### 2. Define the Rules

Separation isn't chaos. Define the constraints:

```
Symbol:
- Minimum size: 16px
- Clear space: 1x symbol width
- Usage: product marks, favicons, social avatars

Wordmark:
- Minimum size: 12px
- Font: Haglöfs Headline
- Usage: editorial, typographic contexts

Together:
- Relationship: symbol left, wordmark right
- Gap: 1.5x symbol width
- Alignment: centerline
```

### 3. Build the Grammar

The grammar defines when each form appears:

| Context | Form |
|---------|------|
| Product | Symbol only |
| Website header | Wordmark only |
| Campaign hero | Symbol + wordmark, flexible |
| Social profile | Symbol |
| Business card | Wordmark + symbol, locked |
| App icon | Symbol |

## The Numeral Extension

Haglöfs takes separation further with a custom numeral typeface. The numerals are "clearly linked to the Haglöfs symbol" — they carry the brand's technical expression without overusing the symbol itself.

This is a second-order separation: not just wordmark from symbol, but the symbol's DNA distributed into other assets. The numeral typeface IS the symbol, expressed as typography.

---

## Anti-Patterns

1. **Forced lockup** — Symbol and wordmark always together, even when one would suffice
2. **No clear space** — Symbol touching wordmark, no breathing room
3. **Scale mismatch** — Symbol sized for large contexts, forced into small ones
4. **Inconsistent grammar** — No rules for when each form appears
5. **Over-separation** — Symbol and wordmark feel like different brands

---

## Case Study: Haglöfs

**Symbol:** Refined cross-hair mark — organic lines with geometric precision
**Wordmark:** Custom serif typeface — 20th-century tradition
**Numeral:** Custom numeral face — linked to the symbol's DNA

**Separation:** Symbol works alone (product, social). Wordmark works alone (editorial). Numeral extends the symbol's expression into typography.

**Why it works:** Every element carries the brand's DNA, but each has a specific job. The symbol is the visual anchor. The wordmark is the typographic voice. The numeral is the technical expression. Together, they create a system, not a logo.
