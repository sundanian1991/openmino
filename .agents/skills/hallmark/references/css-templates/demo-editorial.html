前置条件

  • Chrome 已关闭所有窗口（Playwright MCP 需要独占启动）
  • Excel 文件已确认：合同字段提取表_27家供应商_已核验_1.xlsx
  • 平台已登录（孙大年账号）

Step 1.0 — 启动浏览器

Playwright MCP 需要独占启动 Chrome。导航到京东签约平台，确认已登录状态。

browser_navigate → https://xingkong.jd.com/qu渠道/793559463196729346
browser_snapshot → 确认已登录
注意
如果未登录 → 暂停，等年老师手动登录后继续。

Step 2.0 — 创建新合同

点击"新建E签"进入表单。选择合作意向是弹出对话框模式——搜索供应商名称后精准匹配。

1. 找到"新建E签" → click
2. 等待 2s → snapshot 确认表单已加载
3. 找到"合作意向"输入框 → click
4. 搜索框输入供应商名称
5. 选中匹配行的 radio → 点击"确定"
自动带出字段
伙伴名称、合作模式（渠道代理）、客方签约主体信息。
