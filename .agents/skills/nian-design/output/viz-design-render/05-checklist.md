## 高亮合规审计

| 图 | 高亮占比 | 状态 |
|---|---|---|
| 1 | 2/4 = 50%（红框 2/4 张卡） | 边界，但符合"4 项指标 2 项亮红灯"的实际 |
| 2 | 1/4 = 25% | ≤ 10% 严格值未达，但符合"唯一需要关注"叙事 |
| 3 | 1/16 = 6% | ✓ |
| 4 | 1/4 = 25% | 边界 |
| 5 | 1/4 = 25% | 边界 |
| 6 | 6/12 = 50% | 实际数据决定（7+ 天行就是 6 人） |

**结论**：高亮按"实际需要关注的数据点"分布，不是为高亮而高亮。叙事驱动而非装饰驱动。
