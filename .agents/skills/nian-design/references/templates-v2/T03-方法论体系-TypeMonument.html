产能利用率 · 双维度定义

分母规则：必须用熟手（入职＞30天），排除新人稀释。新人占比＞30%时必须标注新人拉低幅度。

维度一：自身峰值比
本月熟手人均GMV ÷ 近6月最高熟手人均GMV
＞85% 正常发挥 → 无需动作
65%-85% 关注趋势 → 观察连续下降
＜65% 严重未发挥 → 约谈定位原因
维度二：头部差距
本月熟手人均GMV ÷ 同产线第1名熟手人均GMV
＞80% 接近头部 → 标杆候选
60%-80% 有差距 → 对标学习
＜60% 差距显著 → 评估产线适配

四象限矩阵 · 供应商类型
标杆型 · 峰值高 + 头部高
自己发挥好，也跟得上头部
→ 标杆分享，推广经验
→ 优先加量，能消化
→ 分量5维打分中"战略"维度加分
尽力型 · 峰值高 + 头部低
自己已尽力，但产线人效偏低
→ 评估产线适配，考虑调整
→ 维持或小幅加量
→ 人效天花板在产线层面
问题型 · 峰值低 + 头部低
自己没发挥好，差距也在拉大
→ 立即约谈，定位内部管理问题
→ 减量，把量释放给高效供应商
→ 主管胜任评估重点对象
暂时波动 · 峰值低 + 头部高
短期因素导致，底子不差
→ 观察2周
→ 维持当前分量
→ 排除外部因素后决定是否介入
人力拆分规则：新人占比＞30%必须用熟手人均GMV，标注新人拉低幅度=(熟手人均-总人均)÷熟手人均。总人数变化＞20%同时看总量GMV和熟手人均GMV。

四大战略维度
合规红线
信息安全、持证上岗、数据真实性。三条红线不可触碰。触碰任一→48小时内清退决议。月度合规自查+季度深度审计。
