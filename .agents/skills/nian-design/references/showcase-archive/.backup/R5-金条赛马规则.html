规则调整

四项核心变更：切量基准从职场量级改为项目量级，考核权重按首贷/复贷分化，人机赛马改为架构对决，留存率公式分母调整。

Rule 01 -- 切量规则
旧规则

按照赛马得分结果，切职场自身量级的百分比

新规则

按照赛马得分结果，切项目量级的百分比

核心变化：从"职场自身量级"变为"项目量级"。切量基准不再依赖各职场的历史量级，而是统一按项目总盘子计算。

Rule 04 -- 指标计算公式
旧规则

留存率 = 期末在贷人数 / 期初在贷人数

新规则

留存率 = 期末在贷人数 / (期初在贷人数 + 新增放款人数)

公式详情
旧公式: 留存率 = 期末在贷人数 / 期初在贷人数
新公式: 留存率 = 期末在贷人数 / (期初在贷人数 + 新增放款人数)

变更说明: 分母加入新增放款人数，更准确衡量留存能力
生效时间: 2026年6月
核心变化：分母从单纯的期初在贷人数，扩展为期初在贷人数加上新增放款人数。新公式能更真实地反映用户留存情况，避免因新增放款稀释留存率。
