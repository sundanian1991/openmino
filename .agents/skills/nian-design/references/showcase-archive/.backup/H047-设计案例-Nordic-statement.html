02 / Key Takeaways
四个核心要点
01
品牌设计的核心不是选择一个方向，而是在对立中找到张力。Haglöfs 的四组张力是所有设计决策的出发点。
02
定制字体不只是美学选择，是品牌叙事的载体。5种字体家族各有明确的使用场景。
03
色彩系统的三层结构（Core/Extended/Signal）让品牌在保持一致性的同时拥有表达弹性。
04
标志的"分离"策略是打破静态单调、创造动态表达的关键。

08 / Key Concepts
核心概念
Tension-based Design

基于张力的设计方法论——不选择单一方向，而是在对立元素之间找到动态平衡。

Bespoke Typeface

定制字体——为品牌专门设计的字体家族，包含多种用途的字重和样式。

Brand Architecture

品牌架构——品牌元素的组织方式。Haglöfs 采用分离式架构，增加灵活性。

Tiered Color System

三层色彩系统——Core + Extended + Signal，兼顾一致性和表达力。

09 / Glossary
术语表
Visual Identity
视觉识别。品牌的视觉表达系统，包括标志、字体、色彩等。
Brand Platform
品牌平台。品牌的核心信息框架，包括目的、价值观、个性等。
Design System
设计系统。可复用的组件、规则和标准的集合，确保品牌一致性。
Heritage
品牌传承。品牌的历史、传统和文化积淀。
10 / Connections
关联思考
→
关联：Nothing Phone 设计哲学

Nothing 的透明设计也是"张力"的表达——在隐藏与展示之间找到平衡。两个品牌都用"可见的结构"作为设计语言。

→
应用：竞品分析框架

Tension-based 方法可以用于任何品牌分析——找到品牌的核心对立面，然后看设计系统如何解决这个张力。

→
启发：数据可视化配色

三层色彩系统可以用于数据可视化——基础色做背景，信号色做强调。
