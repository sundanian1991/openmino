## 使用方式

1. 确定场景模板（scene/）后，如需变体 Hero 或色彩灵感，来此查找
2. 按色调分组快速定位：深色系适合权威/深沉，浅色系适合温暖/明亮
3. 取 Hero CSS 片段时，保持该模板的字体+色彩系统完整，不混搭
