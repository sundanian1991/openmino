# AI 绘图 Prompt 关键词速查表

> 每种构图对应 Midjourney / DALL-E / Stable Diffusion 等工具的英文关键词，可直接用于 prompt 编写。

## 使用方法

**Prompt 基础结构：**
```
[主体描述] + [构图关键词] + [风格/色彩] + [光影/氛围] + [质量词]
```

**示例：**
> A majestic lion roaring, **centered composition with symmetrical balance**, golden hour lighting, cinematic atmosphere, hyper-realistic, 8k --ar 3:4

## 通用质量修饰词

在 prompt 末尾添加以提升画质：

**基础质量：** `high quality`, `highly detailed`, `sharp focus`, `professional`

**超高清：** `8k resolution`, `4k`, `ultra detailed`, `photorealistic`

**光影氛围：** `cinematic lighting`, `studio lighting`, `golden hour`, `soft ambient light`, `dramatic shadows`

**艺术风格：** `digital art`, `oil painting`, `watercolor`, `vector illustration`, `flat design`, `3D render`

**镜头语言：** `shot on 85mm lens`, `depth of field`, `bokeh`, `wide angle`, `macro shot`

## 常用比例参数

| 平台 | 参数 | 说明 |
|------|------|------|
| Midjourney | `--ar 16:9` | 横向宽屏 |
| Midjourney | `--ar 9:16` | 竖向手机屏 |
| Midjourney | `--ar 1:1` | 正方形 |
| Midjourney | `--ar 3:4` | 竖版海报 |
| Midjourney | `--ar 4:3` | 传统照片 |
| Stable Diffusion | `宽x高` 如 `1024x768` | 像素尺寸 |
