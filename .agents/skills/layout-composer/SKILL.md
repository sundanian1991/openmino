# Layout Composer — 100 种排版构图技能

## 概述

本技能基于 **100 种经典排版构图法则**，为以下设计场景提供专业布局建议和可执行方案：

- 📸 **AI 图片生成** — 为图像生成模型编写结构化 prompt
- 🎨 **海报设计** — 宣传海报、活动海报、品牌海报
- 🃏 **图文卡片** — 社交媒体卡片、产品介绍卡、知识分享卡
- 📱 **社交媒体封面** — 公众号封面、小红书封面、视频缩略图
- 📊 **信息图 / 知识卡片** — 数据可视化、知识体系图、教程图解
- 💻 **UI 界面布局** — 网页、App 页面、Dashboard 面板

## 快速使用流程

```
1. 明确需求 → 用户描述设计目标和使用场景
2. 场景匹配 → 从下方"场景-布局映射表"筛选适合的构图类型
3. 深入查阅 → 按 layout 编号查阅 references/layouts-full.md 获取完整构图要点
4. 产出方案 → 输出布局建议 + 可视化 Demo + (可选)AI绘图 Prompt
```

## AI 绘图 Prompt 生成指南

### Prompt 结构模板

为 AI 图像生成工具（Midjourney / DALL-E / Stable Diffusion 等）编写 prompt 时，按以下结构组织：

```
[主体描述] + [构图技法] + [色彩风格] + [光影氛围] + [质量修饰词]
```

| 满版构图 (013) | full-bleed, edge-to-edge, immersive frame, no margins |

### Prompt 编写示例

**示例 1：产品宣传海报（使用居中构图 + 色彩对比）**
> A sleek smartphone floating in the center of the frame, surrounded by abstract geometric light trails in deep navy blue and vibrant coral orange, centered composition with bold product showcase, complementary color palette, studio lighting with soft shadows, clean minimal background, commercial photography style, 8k resolution, ultra-detailed --ar 3:4

**示例 2：知识卡片（使用左文右图 + 留白）**
> Educational infographic card about machine learning, split composition with clean typography on left side and simple neural network diagram on right side, generous white space, soft pastel blue and cream color scheme, flat vector illustration style, modern sans-serif font, clean grid alignment, professional design --ar 16:9

**示例 3：社交媒体封面（使用对角线 + 动静对比）**
> Dynamic fitness motivation cover photo, athlete running along a diagonal line from bottom-left to top-right, motion blur on legs contrasting with sharp focused upper body, energetic orange and dark grey palette, diagonal composition creating movement, action shot with frozen moment, high contrast lighting --ar 1:1

## Demo 生成规范

当用户要求生成 Demo（HTML/CSS 可视化示例）时，遵循以下原则：

1. **使用 HTML + CSS 内联样式**，可直接在浏览器预览
2. **尺寸适配场景**：海报 A3(3508×4961)、卡片 1200×630、封面 1080×1080、信息图 1920×1080
3. **配色从布局原图提取**或使用用户指定的品牌色
4. **保留布局的核心结构特征**（如三分线的位置、对称轴等）
5. **使用占位符内容**（Lorem Ipsum 或示意文字），标注各区域用途
6. **响应式设计**：Demo 应支持不同屏幕尺寸查看
