## 六、使用说明
### 沟通规则
1. **跨业务线供应商**：优先使用简称，这些是生态核心
2. **单业务线供应商**：首次用全称，后续可用简称
3. **同音简称区分**：
   - "中乾"（企金）vs "中弘"（菁卡）
   - "赛维斯"（金条人机）vs "锦瑞"（企业主贷）
### 示例
- 您说"汇讯本月业绩" → 我理解为"汇讯（湖北）商务信息咨询有限公司"
- 您说"锦瑞三个点" → 我知道是"河北锦瑞信息科技有限公司-定州/石家庄/邯郸"
- 您说"创博承德" → 我知道是"河北创博信息技术有限公司-承德县/滦平/正定"
