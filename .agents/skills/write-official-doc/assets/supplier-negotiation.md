## 商务谈判邀请（发给供应商）

尊敬的XXX：

为推进双方合作，现就商务条款进行谈判，请贵司派相关负责人参加。

**谈判议题：**

一是价格调整：当前单价为XX元/通，调整方向为下降X%/保持不变/探讨阶梯价格。

二是服务标准（SLA）：接通率目标从XX%提升至XX%，转化率目标从XX%提升至XX%，响应时效要求提升。

三是激励机制：档位奖励新增XX档，季度奖金条件优化，年度返点达成条件调整。

四是特殊条款：独家合作区域为XXX，优先续约权为XXX，数据安全条款加强。

**会议信息：**

时间为X月X日XX:XX，地点为XXX公司XX会议室，时长预计2小时，我方出席人为XXX、XXX，贵司出席人要求为总经理/决策人、商务负责人。

**会议准备：**

请贵司准备以下方案：一是价格调整建议及理由，二是对SLA提升的可行性分析，三是对激励机制的意见和建议，四是其他关注事项。

请于X月X日前确认参会人员。

期待与贵司达成互利共赢的合作方案。

## 激励方案通知（发给供应商）

尊敬的XXX：

为激发合作积极性，现发布新的激励方案如下。

**激励方案：**

档位奖励：钻石档为月通话量≥XX通，奖励幅度+5%；黄金档为月通话量≥XX通，奖励幅度+3%；白银档为月通话量≥XX通，奖励幅度+1%。

质量奖励：转化率为单月≥XX%，奖励金额+XX元；合规率为连续3月100%，奖励金额+XX元；留存率为季度≥XX%，奖励金额+XX元。

冲刺奖励：季度冲刺为最后10天，转化率超目标X%，每超1%+XX元；年度冲刺为12月，年度排名前三奖励XX元。

**参与条件：**

一是投诉率为0，二是无重大违规记录，三是月度考核为B级以上。

**结算方式：**

奖励将在次月结算时一并发放。

如有疑问，随时沟通。

尊敬的领导：

现将X月X日与XXX供应商商务谈判纪要如下。

**谈判概况：**

供应商为XXX，时间为X月X日，地点为XXX，我方出席为XXX、XXX，对方出席为XXX、XXX。

**谈判进展：**

已达成一致：一是价格调整XX元/通自X月生效，二是SLA提升接通率目标XX%→XX%，三是激励机制新增冲刺奖励。

仍在协商：一是独家合作区域对方要求XXX、我方建议XXX，二是数据安全条款需法务进一步审核，三是年度返点条件约定X月X日前再议。

**对方立场：**

期望为价格少降X%、给予较多优惠余地，接受为质量要求提升愿意配合培训，关注为长期合作稳定性倾向签订3年合同。

**我方立场：**

目标为单价下降X%、质量达标，让步为可考虑3年合同锁定价格，红线为合规条款不退让。

**下一步计划：**

一是法务审核数据安全条款（X天内），二是总监决策独家区域问题（X天内），三是我方准备下一轮谈判方案（X天内），四是预计下次谈判为X月X日。

**建议：**

是否接受部分妥协（如区域共营），以快速达成协议？

请批示。

## 合同条款讨论（发给法务/财务）

Hi 法务、财务同事：

附件是XXX供应商的合同草案，针对以下条款望予审核。

**法务需审核：**

一是服务标准（SLA）条款：接通率目标XX是否合理、违约责任是否完整。二是数据安全条款：数据处理权限是否清晰、保密期限是否足够。三是终止条款：单方终止条件是否可接受、清退流程是否完善。四是其他法律风险。

**财务需审核：**

一是结算条款：结算周期是否合理、付款条件是否有风险。二是成本结构：单价是否有竞争力（总成本测算见附件）。三是税务处理：发票要求是否规范、税点是否明确。

**合同基本信息：**

合作期限为X年，服务范围为XXX，预计年结算额为XXX元。

**时间要求：**

请于X月X日（前X天）反馈意见，以便推进合同签署。

如有疑问或需要补充材料，请告知。
