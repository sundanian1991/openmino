# 小黑 IP

## 角色定义

小黑是这套配图的固定 IP，但不是死板的 Logo，也不是吉祥物。它是一个正在真实物件现场里认真工作的荒诞小人。

它可以因为场景不同而有细微形体差异：高一点、矮一点、胖一点、瘦一点、圆一点、歪一点。变化要小，识别必须稳定。

## 必须保留的识别点

- 黑色实心身体。
- 白色小圆点眼睛。
- 细胳膊、细腿。
- 轮廓略微不规则，有手绘感。
- 表情空、呆、冷静、认真。
- 没有嘴或只有极简短线嘴。
- 不穿复杂服装。
- 不做夸张表情。

## 2.0 形体弹性

默认形态：

- 黑豆 / 软胶囊 / 小黑影。
- 高宽比约 1.5:1 到 2.2:1。
- 可以比 1.0 稍胖一点，让它更有生活感。
- 高度通常保持在画面高度的 8%-13%，不要突然变成巨型主角。

允许变化：

- 被拉扯时可以变瘦一点。
- 被压住时可以变扁一点。
- 扛东西时可以更矮胖。
- 钻进物件时可以只露出半个黑影。
- 被筛选/吸入/夹住时可以变形，但仍然能看出是小黑。

禁止变化：

- 不要变成可爱吉祥物。
- 不要变成儿童卡通。
- 不要有夸张大眼睛、笑脸、表情包五官。
- 不要穿西装、帽子、复杂装备来解释身份。
- 不要过度拟人化。
- 不要每张都完全同一个姿势、同一个比例，否则会死板。

## 小黑必须承担核心动作

小黑不能只是站在旁边看。

合格动作：

- 被会议卡片拉进电脑。
- 拿纸板挡消息纸条。
- 解开报警线缆的结。
- 在放大镜下改稿。
- 拉住被贴“自动化”的工牌。
- 把简历纸推向筛子。
- 被下班线缆拽回。
- 把素材塞进机器或从物件里取出。

不合格动作：

- 站在角落里看物件。
- 举牌子解释主题。
- 对着物件摆拍。
- 和真实物件没有物理接触。
- 去掉小黑后图意仍然完整。

判断标准：

```text
如果删掉小黑，隐喻还完全成立，说明小黑失败。
```

## 性格

- 认真，不装可爱。
- 有点笨拙，但不蠢。
- 像一个低调的系统操作员。
- 在荒诞现场里很冷静。
- 它的可爱来自“认真做荒诞事”，不是表情和装饰。
