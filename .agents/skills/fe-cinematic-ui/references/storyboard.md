# Storyboard — 供应商管理全景 × Fincher/Social Network

## Site Cinematic Grammar

- Framing discipline: Asymmetric, edge-anchored. Each section is a "case file" — left-heavy with evidence density
- Density cadence: Dense → breath → dense → breath → resolution
- Material: Matte surfaces, border-defined depth, no shadow/blur/gradient
- Motion: Precise, motivated scroll-reveals. Fincher doesn't animate for decoration

## Compiled Spec

- Signature composition: Full-bleed hero with right-side vertical metrics strip ("evidence sidebar")
- One big idea: Supplier management as a Fincher crime board — data is evidence, process is procedure, compliance is the rule of law
- Restraint: No hero image, no decorative icons, no emoji. Data IS the visual
- Entrance: Sections slide up with Fincher's precise ease. No bounce, no overshoot
