# Premium Calibration（年 UI 版）

导演简报完成后、编译规范之前使用。
目标不是替换导演语言——是阻止设计退化为通用"创意模板"。

## Premium 的含义

Premium = 页面感觉被导演过、编辑过、有意图的。
这**不意味着**添加更多渐变、更多玻璃、更多特效。

年 UI 的 premium 信号优先级：

1. 自信的尺度（8:1 比值）
2. 克制的留白
3. 清晰的层级（三层金字塔）
4. 受控的对比（4 级文本灰度）
5. 有目的的构图（签名构图）
6. 材料深度（border 对比，非阴影）
7. 限定的动效（无 bounce/parallax）
8. 编辑过的颜色（单一场景色）

## Required Outputs

写入 storyboard：

- `Site cinematic grammar`
- `One big idea` — 页面单一主导视觉概念
- `Page scene thesis`
- `Hero dominance statement` — 一句说明 Hero 为何有冲击力（不使用渐变/阴影）
- `Restraint statement` — 故意不做的事
- `Material thesis` — 表面触感
- `Typography thesis` — 文字权威感
- `Signature composition` — 页面独有的不可替代构图
- `Grid fallback test` — 如果缩成通用卡片网格会失去什么
- `Shared system holdback`
- `Uniqueness guardrail`

## Review Questions

移动到实施前问：

1. 3 秒后观众记住的是哪一件事？
2. 什么被故意省略了？
3. 哪个细节让页面感觉被精心设计而非装饰过度？
4. 如果去掉 30% 的效果，页面会更强吗？
5. 这个页面是同一部电影里的独立场景，还是首页的弱化复制品？
6. 页面构图如果去掉颜色仍然感觉有方向吗？
7. 去掉主题颜色，构图本身是否有足够的结构完整性？
8. 场景色是否支持叙事的情感基调？（不是通用的"好看"）
