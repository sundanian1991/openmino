## 核心架构

### 四视角指标体系

| 视角 | 核心问题 | 关键指标 |
|------|---------|---------|
| 人力结构 | 团队底子怎么样？ | 参评率、C级率、S占比、司龄结构 |
| 产能效率 | 产出够不够？ | 转化倍差、GMV驱动结构、ATT×转化 |
| 集中度风险 | 太依赖几个人？ | Top20% GMV占比、单职场偏离度 |
| 问题定位 | 问题在哪？该找谁？ | 职场C级率、预警率、MOB1-3集中度 |

### 三层下钻

```
第一层：健康总览（4卡片 × 10指标）→ 全绿收工，有红黄展开
第二层：视角图表 → 职场选择器切换，发现异常下钻
第三层：坐席级明细 → 定位具体问题点，产出管理动作
```

## 设计系统

### 配色 Token

| Token | 值 | 用途 |
|-------|---|---|
| $bg-page | `#FFF8F0` | 页面背景，温润米白 |
| $bg-card | `#FFFFFF` | 卡片/图表背景 |
| $text-primary | `#2D3748` | 主文字/标题 |
| $text-secondary | `#718096` | 次要文字 |
| $text-muted | `#A0AEC0` | 脚注/来源 |
| $border | `#F0E6D8` | 浅暖灰边框 |
| $accent | `#E8734A` | 珊瑚橙，唯一强调色 |
| $accent-light | `#FFF0EB` | 浅珊瑚（高亮区域） |
| $accent-dark | `#C75A35` | 深珊瑚（P0标签） |
| $grade-S | `#E8734A` | S级（珊瑚橙） |
| $grade-A | `#5A6878` | A级（中灰蓝） |
| $grade-B | `#8B99A8` | B级（浅灰蓝） |
| $grade-C | `#B8C4D0` | C级（极浅灰蓝） |
| $unrated | `#E8E0D6` | 未参评（暖灰） |

### 克制原则

- 全报告仅使用一个彩色：珊瑚橙（#E8734A）
- 珊瑚橙仅用于最需要关注的数据点（S级、关键指标、P0标签）
- 其他所有元素：灰度/灰蓝层次
- 数据不撒谎，颜色不喧宾夺主

## 跨视角联合判断

| 指标组合 | 诊断结论 | 管理动作 |
|---------|---------|---------|
| 参评率低 + C级率高 | 新人招了没带好 | 优化培训 + 师傅带教 |
| 参评率低 + C级率正常 | 团队萎缩，非质量问题 | 扩招为主 |
| 参评率正常 + C级率高 | 老人不行，结构性平庸 | 末位淘汰 + 激励机制 |
| Top20%高 + C级率高 | 头重脚轻，两极分化 | 萃取头部 + 淘汰尾部 |
| Top20%高 + C级率低 | 正常金字塔，健康 | 萃取最佳实践到A级 |
| Top20%低 + C级率高 | 整体平庸，无突出无底线 | 整体能力提升 |

## 每张图 = 决策工具

SABC 方法论的核心铁律：**每张图必须有对应的管理动作，不是风景画**。

- 转化断层图 → 萃取 S 级录音技巧，不是加拨打量
- GMV 驱动拆解 → 转化率偏低→培训 / 件均偏低→话术 / 接通偏低→排班
- 帕累托曲线 → >60% 时紧急扩招 + S 级风险预案
- C 级诊断 → MOB1-3 集中→培训问题 / 某职场高→约谈
