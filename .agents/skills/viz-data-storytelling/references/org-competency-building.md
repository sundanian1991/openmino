# 组织级数据叙事能力建设

> 参考自 Cole Nussbaumer Knaflic | 团队建设参考，非 viz-data-storytelling 技能直接产出

---

## 四支柱框架

### 1. 全员基础培训

- 所有需要数据沟通的岗位参加标准化培训
- 内容：图表选择、消除杂乱、引导注意力、三幕叙事
- 目标：建立共同语言，降低团队内部沟通摩擦

### 2. 内部专家培养

- 从团队中选拔有潜力的人深入学习
- 负责：复杂项目的数据叙事设计、内部培训、质量把关
- 建立内部案例库和最佳实践

### 3. 战略外包

- 复杂或高优先级项目可引入外部专家
- 与内部专家合作，实现知识转移
- 适用于：初期团队能力不足时的杠杆策略

### 4. 监控与调整

- 定期回顾数据叙事的实际效果（受众理解度、行动转化率）
- 根据反馈调整培训内容和工具链
- 建立可持续迭代机制
