# 数据可视化工具学习策略

## 核心理念

数据可视化的工具学习是**操作技能**而非策略知识。策略层（viz-data-storytelling）产出叙事大纲，工具层由下游技能负责。本文件供需要自我提升工具技能的读者参考。

---

## 学习方法

### 专注单一工具

- 选定一个主要工具（Excel / Tableau / Power BI / Python），熟练掌握后再扩展
- 初期同时学习多个工具会分散注意力，减缓进度

### 主动实验代替被动培训

- 培训课程 → 吸收率低。主动尝试 → 持久记忆
- 自己设定数据问题，用工具回答，卡住时针对性搜索
- 效率："需要时学" > "先学完再需要"

### 拥抱挫折

- 工具学习的前 20 小时是最难熬的。这是正常过程
- 每次卡住 = 一次深度学习机会

### 辅助工具

- **PowerPoint**：图表的最终展示仍需 PPT 排版和标注
- **Illustrator/SVG**：精细调整图表元素（标注、对齐、色块）
- 不要为"做一张图"同时打开 3 个工具 — 主线工具 + 1 个辅助足够
