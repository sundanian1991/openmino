# skill-consolidator

审查当前项目的技能目录，输出清理建议。

## 工作流

1. 遍历 `.Codex/skills/` 每个子目录的 `SKILL.md`
2. 按功能领域归类（PPT、设计、数据、搜索、AI、办公、工具类）
3. 识别功能重叠/重复的技能
4. 标记引用破损或疑似废弃的技能
5. 输出分级建议：安全删除 / 需确认 / 保留

## 输出格式

> - 领域一：PPT 类（N 个技能）
>   - skill-a — 保留理由
>   - skill-b → 合并到 skill-a（建议）
> - 领域二：搜索类（N 个技能）
>   - ...

## 注意事项

- 只读，不修改任何文件
- 保留判断留给年老师
