## 赛马机制通知

俊哥：

同步下2026年2月赛马结果及3月安排：
**触发情况：**
- 拓银、华道2月月均人力目标＜20人，触发赛马淘汰机制
- 华道已提出撤场
- 拓银需进入专项提升期
**专项提升期要求（1个月）：**
- 人日均激活 ≥ 大盘均值的70%
- 排名不为倒数第1
**真正淘汰条件（需同时满足）：**
- 进入专项提升期后，排名仍为倒数第1
- 人日均激活 < 大盘均值的70%
**规则说明：**
- 2月赛马标准：人日均核卡
- 3月赛马标准已更新为：人日均激活
- 详细规则参考：[赛马规则文档](https://joyspace.jd.com/pages/iFS1gJe6b8a7kNqGpdYj)

如有疑问，随时沟通。
