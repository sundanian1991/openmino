## 内容

从 Coding Plan 到 Token Plan 的 AI 定价演变分析，25 页完整演示。

## 为什么是标杆

- 全部使用 html-ppt-skill 的 `.deck` / `.slide` 体系 + runtime.js
- 自定义暖色主题 `warm-editorial.css`（非默认主题）
- 卡片无左边框（年老师：左边框"非常 AI"）
- 所有容器严格对齐（`width:100%` 包裹，不用孤立 max-width）
- 密集内容页用 `justify-content:flex-start` 避免顶部裁切
- 间距精确到像素级，不依赖默认值
