# 风格生成系统

> 基于5维特征自动生成设计，非预设风格

---

## 核心原则

1. **无灰阶** — 信息本身足够，不需要灰色引导
2. **图标有用** — 眼前一亮、信息补充、情绪表达
3. **自动生成** — 5维特征输入 → 独特风格输出

---

## 5维特征

| 维度 | 选项 | 决定什么 |
|------|------|---------|
| **信息结构** | 排名 / 对比 / 流程 / 清单 / 洞察 | 版式布局 |
| **情绪气质** | 紧迫 / 中性 / 庆祝 / 权威 / 警示 | Header颜色、强调色 |
| **信息密度** | 高 / 中 / 低 | 留白比例、字号 |
| **视觉元素** | 无图标 / 简单图标 / 复杂可视化 | 图标数量、类型 |
| **色彩倾向** | 冷 / 暖 / 中性 | 整体色调 |

---

## 维度 → 设计决策

### 1. 信息结构 → 版式

| 结构 | 版式 | 示例 |
|------|------|------|
| 排名 | 表格 + TOP3突出 | 供应商排名 |
| 对比 | 左右分栏 | 前后对比、竞品分析 |
| 流程 | 时间线 / 步骤条 | 操作指南、里程碑 |
| 清单 | 卡片网格 / 标签组 | 装备清单、检查项 |
| 洞察 | 中心金句 + 支撑点 | 一句话结论、诊断 |

### 2. 情绪气质 → 颜色

| 情绪 | Header | 强调色 | 图标风格 |
|------|--------|--------|---------|
| 紧迫 | `#8B0000` 深红 | 同Header | 警示图标 ⚠️ |
| 警示 | `#dc2626` 红 | 同Header | 警报图标 🚨 |
| 中性 | `#111` 炭黑 | `#E2725B` 陶土 | 简洁几何 |
| 庆祝 | `#E2725B` 陶土 | `#D4AF37` 金 | 奖杯星星 🏆 |
| 权威 | `#0d7680` 青绿 | 同Header | 正式徽章 |

### 3. 信息密度 → 留白

| 密度 | 留白比例 | 适用场景 |
|------|---------|---------|
| 高 | 10-20% | 数据表、排名、多指标 |
| 中 | 30-40% | 标准内容页 |
| 低 | 50-60% | 金句页、封面、结论 |

### 4. 视觉元素 → 图标

| 级别 | 图标数量 | 作用 | 示例 |
|------|---------|------|------|
| 无 | 0 | 纯文字、数据说话 | 纯数据表 |
| 简单 | 1-3个 | 眼前一亮、情绪点缀 | 奖杯、箭头、警示 |
| 复杂 | 多个 | 信息穿透、可视化 | 流程图、图表 |

**图标来源**：better-icons 搜索 → 改色 → inline嵌入

### 5. 色彩倾向 → 整体色调

| 倾向 | 主色 | 强调色范围 | 适用 |
|------|------|-----------|------|
| 冷 | 青、蓝、绿 | `#0d7680` - `#2563eb` | 金融、战略、信任 |
| 暖 | 红、陶土、金 | `#8B0000` - `#D4AF37` | 激励、庆祝、警示 |
| 中性 | 黑白灰 + 单点缀 | `#111` + 一种强调 | 数据、通用、专业 |

## 风格生成示例

### 示例 1: 供应商产能排名

| 维度 | 选择 |
|------|------|
| 信息结构 | 排名 |
| 情绪气质 | 中性 |
| 信息密度 | 高 |
| 视觉元素 | 简单图标 |
| 色彩倾向 | 中性 |

**生成**:
- 版式：表格 + TOP3色块突出
- Header：`#111` 炭黑
- 强调色：`#E2725B` 陶土（仅TOP1）
- 留白：15%（紧凑）
- 图标：奖杯（TOP1）、箭头（增长）

### 示例 2: 业绩冲刺激励海报

| 维度 | 选择 |
|------|------|
| 信息结构 | 洞察 |
| 情绪气质 | 庆祝 |
| 信息密度 | 低 |
| 视觉元素 | 复杂可视化 |
| 色彩倾向 | 暖 |

**生成**:
- 版式：中心大数字 + 支撑信息环绕
- Header：`#E2725B` 陶土
- 强调色：`#D4AF37` 金
- 留白：55%（大气）
- 图标：奖杯、星星、向上箭头、钱币

### 示例 3: 供应商处罚通报

| 维度 | 选择 |
|------|------|
| 信息结构 | 对比 |
| 情绪气质 | 紧迫 |
| 信息密度 | 中 |
| 视觉元素 | 简单图标 |
| 色彩倾向 | 暖 |

**生成**:
- 版式：左右对比（处罚前/后）
- Header：`#8B0000` 深红
- 强调色：同Header
- 留白：35%
- 图标：⚠️ 警示、禁止符号

## 与旧系统的区别

| 旧系统 | 新系统 |
|--------|--------|
| 14种预设风格 | 5维特征生成 |
| 选择风格 | 描述内容特征 |
| 灰阶层级 | 无灰阶，信息本身说话 |
| 图标可有可无 | 图标是情绪表达必需 |
| 固定配色 | 情绪+色彩倾向动态配色 |
