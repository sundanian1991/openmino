# 刘伟佳

## 基本信息
- 部门：数据科技业务部 - 二级部门
- 角色：策略组负责人
- 职责：定价、费效比优化、策略制定

## 与供应商管理的关系
- 控制结算定价权
- KPI导向：费效比优化（降成本）
- 与年老师的目标存在潜在冲突：
  - 策略组目标：费效比优化 → 收紧结算
  - 年老师目标：供应商稳定 → 需要灵活定价

## 沟通僵局（2026-03）

### 年老师的做法
- 提出建立定价委员会，要求定价需服务组同意
- 事前与策略组老板、刘伟佳沟通不充分
- 方式太直接，导致对抗情绪

### 刘伟佳的反应

**表面言论**：
- "担心供应商不好管理"
- 强调供应商人力不足

**实际行为**：
- 要求头部供应商增人 → 但增人重点是头部职场 → 仍短缺
- 批评中腰部业绩不佳 → 但不解决结构性问题

**潜台词分析**：
- 用"供应商管理问题"维护自己的定价权
- 把责任推给年老师（供应商不好管理）
- 实际在强化头部依赖（要求头部增人）
- 没有意愿解决中腰部承接问题

### 行为模式判断
- **防御性姿态**：维护定价权，拒绝分权
- **责任转移**：用"管理问题"解释结构性问题
- **短期导向**：费效比指标优先，不关注长期结构

## 沟通要点
- 需要协调的事项：费效比考核与供应商稳定的平衡
- 潜在协同点：供应商流失会影响费效比（产能下降）
- **沟通策略调整**：
  - 不要直接要求分权（定价委员会）
  - 要用"共同优化费效比"的框架
  - 论点：结算太低 → 供应商不投入 → 产出下降 → 费效比恶化
