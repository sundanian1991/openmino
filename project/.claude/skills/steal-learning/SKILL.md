# 偷师三行法

极简信息萃取方法论，专治"收藏即学习"。

---

## 核心框架（3分钟上手）

```
【第一行】我看到了什么？
→ 只记录事实：谁+做了什么+结果如何

【第二行】我能用在哪儿？
→ 绑定真实场景：下次遇到XX情况时，我用XX方法

【第三行】我何时试一次？
→ 设定死线：[日期]的[具体事件]，我就试
```

**本质**：事实记录 → 场景绑定 → 限期行动

---

## 一看就会的示例

**场景**：客户说"太贵了"，销售总监的回应

**第一行：我看到了什么？**
客户抱怨贵时，总监没解释成本，而是将价格转为价值："您每月省XX小时，相当于多赚XX钱。"

**第二行：我能用在哪儿？**
下次跟老板申请预算被质疑"太贵"时，不说成本，而是说"投资回报"。

**第三行：我何时试一次？**
本周五周会汇报下季度预算时，就用这个话术。

## 快速开始

**方式1：手动记录**
1. **手机备忘录**创建【偷师萃取本】文件夹
2. **遇到有价值内容**，严格按三行记录
3. **每月回顾**，形成个人武器库

**方式2：AI辅助（推荐）**
1. **找到有价值内容**（文章/视频/书籍）
2. **给我链接**，说"用steal-learning吃透这个"
3. **我自动抓取、萃取、关联**
4. **记录到** `learning-log.md`

**完整流程**：
```
你的输入 → 我抓取 → 三行法萃取 → 知识关联 → 行动计划 → 记录留存
```

## 理论支撑

- **SECI模型**：隐性知识→显性知识的转化
- **70-20-10法则**：70%的学习来自实践
- **认知负荷理论**：碎片时间需要明确的学习单元

**核心洞察**：拉开差距的不是学习量，而是将他人经验转化为个人本能的能力。
