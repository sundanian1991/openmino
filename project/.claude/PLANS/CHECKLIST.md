# Plan模式检查清单

> 每次Plan模式启动前确认这些项

---

## 启动前检查

- [ ] **任务边界明确吗？**
  - 我知道要做什么吗？
  - 知道不要做什么吗？

- [ ] **上下文充分吗？**
  - 相关文件都读了吗？
  - 年老师需求清晰吗？

- [ ] **需要用Agent吗？**
  - 如果需要，用哪个？
  - 需要并行还是串行？

- [ ] **需要用Skill吗？**
  - 查INDEX.md确定用哪个
  - 确认skill可用

---

## 执行中检查

- [ ] **按计划文档执行**
  - 不偏离步骤顺序
  - 完成一步标记一步

- [ ] **遇到障碍立即记录**
  - 放到PLANS/当前任务.md里
  - 不要擅自跳过或绕过

---

## 完成后检查

- [ ] **所有步骤都完成？**
  - 对着任务文档逐项确认

- [ ] **结果可验证？**
  - 年老师能验证我完成了吗？

- [ ] **需要更新知识？**
  - 学到新东西了吗？
  - 需要更新MEMORY.md吗？

---

## 使用示例

**年老师说"进入计划模式，优化述职材料"**

1. ✅ 任务边界：优化三个案例的标题和内容
2. ✅ 上下文：work/供应商管理/06-晋升述职.md已读
3. ✅ 需要Agent：不需要，我来写
4. ✅ 需要Skill：可能需要pptx

→ 可以启动Plan模式
