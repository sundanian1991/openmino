# 巡检报告模板

> 供应商巡检报告的标准模板

## 模板内容

```
【供应商巡检报告】
供应商：XXX
巡检日期：XXXX年XX月XX日
巡检人员：XXX

一、总体评价
[优秀/良好/合格/不合格]

二、逐项检查
1. 人员规模与配置：[达标/不达标] 详情...
2. ...（12项逐项）

三、发现问题
[问题描述 + 严重程度 + 建议措施]

四、整改要求
[整改事项 + 责任人 + 完成时限]

五、下次巡检重点
[基于本次发现的重点关注项]
```
