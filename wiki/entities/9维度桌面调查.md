# 9维度桌面调查

> 新供应商准入的桌面调查框架

## 调查维度

| 维度 | 桌面能查 | 需现场验证 |
|------|---------|-----------|
| 过往业绩 | 行业口碑、客户名单、自述数据 | 实际录音验证 |
| 同类型对标 | 与毅航/伽玛/赛维斯可比性 | 运营细节 |
| 管理方式 | 组织架构图、质检/培训/排班体系 | 管理实效 |
| 人员数量 | 供应商声称规模 | 实际出勤核实 |
| 公司背景 | 天眼查（注册资本、股东、司法风险） | 实际经营状态 |
| 历史交付风格 | 老客户评价、续约情况 | 现场感受 |
| 配合商务 | 合同条款接受度、响应速度 | 正式谈判 |
| 团队架构 | PM/质检/培训三岗是否齐全 | 管理团队实际能力 |
| 资质合规 | 营业执照、增值电信许可证、ISO | 合规执行 |
