# 月度评估流程图

> 月度评估的完整流程

## 流程阶段

```
每月1日   → 提取上月数据
每月5日   → 完成自动评分
每月8日   → 人工复核
每月10日  → 结果确认、通知供应商
每季度    → 综合评级、红黄牌判定
```

## 评估内容

1. **数据提取** — 从业务系统提取上月数据
2. **自动评分** — 按五维考核模型自动计算
3. **人工复核** — 检查异常数据和特殊情况
4. **结果确认** — 与供应商确认评估结果
5. **通知供应商** — 发送评估报告和改进建议
