# 引入流程图

> 新供应商引入的8个阶段

## 流程阶段

```
1. 立项申请
   └─ 供应商管理岗发起，提交《立项申请表》

2. 资质审核
   └─ 审核企业资质、财务状况、合规记录
   └─ 一票否决项检查

3. 现场考察
   └─ 供应商管理岗组织，考察3维度（环境/团队/能力）

4. 综合评估
   └─ 准入诊断6问
   └─ 试单方案设计

5. 审批决策
   └─ 常规：C3审批
   └─ 特殊：C2终审

6. 合同签订
   └─ 法务审核
   └─ 商务条款确认

7. 系统配置
   └─ 账号开通
   └─ 权限配置

8. 试单启动
   └─ 1-3个月观察期
   └─ 密集监控
```
