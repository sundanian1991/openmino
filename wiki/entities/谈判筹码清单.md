# 谈判筹码清单

> 降价谈判的筹码盘点

## 筹码类型

| 筹码类型 | 供应商在乎程度 |
|---------|--------------|
| 分量承诺（保证未来N月量不减少） | 高 |
| 增量机会（降价可以给更多分量） | 高 |
| 新产线机会（优先给新产线份额） | 中 |
| 结算周期（缩短） | 高 |
| 排名数据（暗示不降价可能调分量） | 中 |
| 竞品报价（匿名） | 中 |
