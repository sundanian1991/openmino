# 清退触发路径

> 季度视角的清退触发流程

## 清退触发路径

```
季度评估结果
    │
    ├─ C档 + 趋势下降 → 【当季启动快车道清退】
    │
    ├─ C档 + 趋势持平/上升 → 30天整改期 → 整改未通过 → 启动清退
    │                                           │
    │                                           └─ 整改通过 → 移出C档
    │
    └─ 红线违规 → 直接评级C + 立即启动清退评估
```

## 与月度触发规则的衔接

月度"连续3月后30%"和季度"C档+趋势下降"是**互补而非替代**。月度触发用于及时发现异常并干预，季度九宫格用于综合判定供应商生态位。实践中，月度连续预警的供应商大概率会在季度评估中落入C档。
