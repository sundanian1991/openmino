## 二、准入评估SOP

```
1. 立项申请
   └─ 供应商管理岗发起，提交《立项申请表》

2. 资质审核
   └─ 审核企业资质、财务状况、合规记录
   └─ 一票否决项检查

3. 现场考察
   └─ 供应商管理岗组织，考察3维度（环境/团队/能力）

4. 综合评估
   └─ 准入诊断6问
   └─ 试单方案设计

5. 审批决策
   └─ 常规：C3审批
   └─ 特殊：C2终审

6. 合同签订
   └─ 法务审核
   └─ 商务条款确认

7. 系统配置
   └─ 账号开通
   └─ 权限配置

8. 试单启动
   └─ 1-3个月观察期
   └─ 密集监控
```

## 三、站点变更SOP

### 场景一：同城市迁址

```
1. 供应商提交变更申请
2. 供应商管理岗审核新址合规性
   └─ 商业用途写字楼
   └─ 非禁设区域
   └─ 产权/租赁证明
   └─ 消防验收合格文件
3. 新址现场核查（关键）
4. 决策
   ├─ 审批通过 → 京音平台更新地址
   └─ 审批驳回 → 明确驳回理由
```

### 场景二：新增站点（新城市）

```
1. 供应商提交新增站点申请
2. 供应商管理岗初审
3. 资源规划岗评估需求匹配度
4. 业务运营岗评估业务承接能力
5. C3审批
6. C2终审（涉及特殊资源/大额投入）
7. 通过后系统配置、人员培训
```

### 场景三：站点缩减

```
1. 供应商提交缩减申请（或管理方触发）
2. 业务影响评估
3. 人员安置方案
4. 数据迁移计划
5. C3审批
6. 执行缩减
7. 后续跟踪
```

## 四、清退退出SOP

```
1. 触发评估（72小时内）
   └─ 确认触发条件成立
   └─ 收集证据材料

2. 替代方案（5个工作日内）
   └─ 评估业务影响
   └─ 制定切量方案
   └─ 确认替代供应商

3. 正式通知
   └─ 书面通知供应商
   └─ 明确退出原因和时间表

4. 数据回收
   └─ 客户资料
   └─ 业务数据
   └─ 录音等资料

5. 费用清算
   └─ 结算至退出当日
   └─ 预留金处理

6. 系统关闭
   └─ 账号注销
   └─ 权限回收
   └─ 系统接入关闭

7. 归档
   └─ 退出报告
   └─ 数据归档
   └─ 经验总结
```

## 五、巡检SOP

### 5.1 巡检频次

| 供应商等级 | 频次 | 方式 |
|-----------|------|------|
| S级 | 每半年 | 现场+远程 |
| A级 | 每季度 | 现场为主 |
| B级 | 每月 | 现场为主 |
| C级 | 每两周 | 现场为主 |

### 5.2 巡检12项标准

| # | 检查项 | 关键检查点 |
|---|--------|-----------|
| 1 | 人员规模与配置 | 实际人数 vs 合同人数、人员结构 |
| 2 | 职场环境与安全 | 场地合规、消防、安全设施 |
| 3 | 系统设备与网络 | 系统稳定性、网络带宽、设备配置 |
| 4 | 培训体系与执行 | 培训计划、执行记录、效果评估 |
| 5 | 质量管控流程 | 质检流程、抽检比例、问题处理 |
| 6 | 合规管理制度 | 制度文件、执行记录、红线意识 |
| 7 | 数据安全与保密 | 数据权限、加密措施、泄密预防 |
| 8 | 员工权益保障 | 劳动合同、社保、工资发放 |
| 9 | 业务运营数据 | 业绩达成、效率指标、问题记录 |
| 10 | 客户投诉处理 | 投诉记录、处理时效、闭环情况 |
| 11 | 应急响应机制 | 应急预案、演练记录、响应能力 |
| 12 | 合作配合度 | 沟通响应、问题整改、信息共享 |

### 5.3 巡检报告模板

```
【供应商巡检报告】
供应商：XXX
巡检日期：XXXX年XX月XX日
巡检人员：XXX

一、总体评价
[优秀/良好/合格/不合格]

二、逐项检查
1. 人员规模与配置：[达标/不达标] 详情...
2. ...（12项逐项）

三、发现问题
[问题描述 + 严重程度 + 建议措施]

四、整改要求
[整改事项 + 责任人 + 完成时限]

五、下次巡检重点
[基于本次发现的重点关注项]
```

## 六、考核执行SOP

### 6.1 月度考核流程

```
每月1日   → 提取上月数据
每月5日   → 完成自动评分
每月8日   → 人工复核
每月10日  → 结果确认、通知供应商
每季度    → 综合评级、红黄牌判定
```

### 6.2 考核报告模板

```
【XX月供应商考核报告】

一、总体情况
- 参评供应商数量
- 各等级分布
- 与上月对比

二、各供应商得分
| 供应商 | 产能 | 效率 | 品质 | 配合度 | 合规 | 总分 | 等级 |

三、异常情况
- 排名大幅下降
- 单项指标异常
- 红黄牌触发

四、重点关注
[需重点关注的供应商及原因]

五、下月重点
[管理动作计划]
```
