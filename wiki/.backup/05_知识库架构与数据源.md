## 一、五库知识库体系

### 库1：SOP库（制度与流程）

| 内容 | 说明 | 6月进度 |
|------|------|---------|
| 管理制度 | 供应商全生命周期管理指引、考核制度、分量规则 | 框架搭建 |
| 标准流程 | 准入/巡检/考核/清退SOP | 框架搭建 |
| 操作规范 | 数据填报规范、报告模板、邮件格式 | 首批填充 |
| 决策流程 | 各类审批流的触发条件和权限 | 框架搭建 |

### 库2：供应商档案库

| 模块 | 字段示例 |
|------|---------|
| 基础信息 | 公司全称、统一社会信用代码、法人、注册地址、成立时间 |
| 资质信息 | 经营范围、资质证书、许可范围、有效期 |
| 合同信息 | 签约主体、合同期限、服务范围、结算方式 |
| 人员信息 | 总人数、坐席数、管理层结构、人员流动率 |
| 场地信息 | 职场位置、面积、工位数、设备配置 |
| 业绩信息 | 各业务线产能、达标率、排名趋势 |
| 配合信息 | 响应时效、整改配合度、商务支持态度 |
| 老板画像 | 沟通风格、关注点、历史承诺、谈判偏好 |
| 标签体系 | 业务类型/能力/配合度/稳定性/潜力 |

**6月目标**：12家金条供应商基础信息+模板落地+标签体系定义+老板画像开始收集

## 二、数据来源清单

| 数据类型 | 来源 | 获取方式 | 更新频率 | 口径问题 |
|---------|------|---------|---------|---------|
| 业绩数据（GMV/产能） | 业务系统 | 系统导出/API | 每日 | ⚠️ 需确认口径 |
| 人力数据 | 供应商填报/系统 | 手工/API | 每日/每周 | ⚠️ 供应商口径 vs 系统口径 |
| 质检数据 | 质检系统 | 系统导出 | 每日 | ⚠️ 质检标准是否统一 |
| 出勤数据 | 考勤系统 | 系统导出 | 每日 | ⚠️ 出勤定义（打卡/实际上工） |
| 合规数据 | 违规记录系统 | 手工记录 | 实时 | ⚠️ 违规判定标准 |
| 排名数据 | 各业务线分别输出 | 手工汇总 | 每日 | ⚠️ **各业务线排名口径不一致** |
| 站点信息 | 供应商填报/现场核实 | 手工 | 变动时 | — |
| 合同信息 | 合同管理系统 | 系统导出 | 变动时 | — |
| 商务沟通记录 | 邮件/微信/电话 | 手工记录 | 实时 | — |
| 联盟活动数据 | 活动后手工整理 | 手工 | 活动后 | — |

|------|-----------|--------|

| 汇报是否需要预算估算？ | 你+军哥 | 汇报前确认 |

| 配合度 | 响应速度/信息透明度/整改配合 | 月度统计 |

| 承接质量 | — | — |

### 待统一的口径

| 指标 | 问题 |
|------|------|
| 人力规模 | 供应商填报 vs 系统统计 vs 实际上工，以哪个为准？ |
| 出勤率 | 打卡出勤 vs 实际上工出勤？ |
| 产能口径 | 有效产能 vs 申报产能？ |
| 排名维度 | 各业务线排名口径是否一致？ |
| 质检合格率 | 抽检比例、判定标准是否统一？ |
