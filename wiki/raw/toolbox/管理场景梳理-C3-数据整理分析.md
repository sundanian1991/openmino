# C3 数据整理分析

> 拿到一堆数据后，先看什么、怎么比、怎么从数字里提炼出结论。

## 触发信号

- 月度排名数据出来了，需要分析
- 出勤/质检/GMV数据汇总到手，要看懂
- 领导问"XX供应商最近数据怎么样"
- 准备约谈/汇报，需要数据支撑
- 想比较几家供应商的差距

## 核心问题

> 数据分析不是做表格，是回答问题。先确定要回答什么问题，再去翻数据。

## 思考步骤

### 第1步：确定你要回答的问题

**想什么**：拿这堆数据到底是想搞清楚什么？

| 场景 | 核心问题 | 需要看的数据 |
|------|---------|------------|
| 月度排名分析 | 谁升了谁降了？为什么？ | 排名表+各维度得分 |
| 出勤数据分析 | 哪家有问题？短期还是趋势？ | 周度出勤率+离职率 |
| 质检数据分析 | 扣分集中哪？偶发还是系统性？ | 质检明细+扣分类型 |
| 约谈准备 | 具体哪里不行？差多少？ | 对比头部+自身趋势 |
| 分量评估 | 该给谁加减量？ | 5维打分+产能利用率 |

**判断什么**：
- 问题不明确 → 先用C1模板拆解，再回来分析
- 多个问题 → 按优先级排序，先回答最重要的

**举例**：月度排名出来了。核心问题不是"把排名表整理一遍"，而是"有没有供应商触发预警规则"。

### 第2步：数据质量快速检查

**想什么**：数据有没有缺的、错的、口径不一致的？先确认再分析。

| 检查项 | 怎么查 | 常见问题 |
|--------|--------|---------|
| 完整性 | 有没有缺供应商/产线/时间段 | 某家某条产线漏报 |
| 一致性 | 口径和上月是否一致 | "签到"变"有效出勤" |
| 异常值 | 有没有不合理数据 | 转化率突然120% |
| 时效性 | 数据截至什么时候 | 可能是上月25日非月末 |

**判断什么**：
- 有缺漏 → 标注缺漏项，先分析能分析的部分
- 有口径变化 → 换算成同一口径再对比
- 有异常值 → 关键供应商关键数据先核实
- 数据时效非最新 → 标注"截至X月X日"

**举例**：翰锐企金产线数据缺失 → 标注后先分析其他部分，催补报。

### 第3步：三线对比法

**想什么**：单个数字没有意义，必须放在参照系里。三条线：自身趋势线、同类对比线、目标/阈值线。

**检查什么数据**：

| 对比维度 | 看什么 | 判断标准 |
|---------|--------|---------|
| **自身趋势** | 近3-6个月同一指标 | 持续上升/下降/波动/稳定 |
| **同类对比** | 和同产线其他供应商比 | 排名位置+与头部差距 |
| **阈值对比** | 和预警线/目标线比 | 是否触发预警规则 |

**判断什么**：

三线对比组合判断：

| 自身趋势 | 同类对比 | 阈值对比 | 结论 |
|---------|---------|---------|------|
| 持续上升 | 头部 | 未触发 | 标杆，保持 |
| 持续上升 | 中段 | 未触发 | 进步型，关注能否持续 |
| 波动 | 中段 | 偶尔触及 | 不稳定，找波动原因 |
| 持续下降 | 中段 | 接近 | 预警信号，提前沟通 |
| 持续下降 | 后段 | 已触发 | 需要干预 |
| 稳定 | 后段 | 已触发 | 结构性问题，可能需要约谈或调整分量 |

**举例**：毛毛虫3月排名第4（环比降2位），自身趋势2个月连续下降，离后30%只差1位 → 预警信号，提前沟通。

### 第4步：按场景深挖

**想什么**：三线对比给出了"方向"，现在需要找到"原因"。不同数据场景深挖方法不同。

> **场景A：月度排名数据**

**检查什么**：

1. 排名变化的5维拆解（业绩30%、质检客诉25%、上人15%、配合15%、战略15%）
2. 哪个维度贡献了排名变化？
3. 单维度下钻：业绩掉了，是GMV还是转化率？

**判断什么**：
- 单维度问题 → 聚焦该维度找原因
- 多维度同时下降 → 可能是系统性管理问题
- 业绩维度下降 + 上人维度也下降 → 人不够导致业绩不行，根因在人

**举例**：岐力3月排名从第6跌到第8。5维拆解：业绩得分从3.8降到3.2（权重30%影响最大）。继续钻：金条产线GMV环比降15%，企金持平 → 问题集中在金条产线业绩。

> **场景B：出勤数据**

**检查什么**：

1. 整体出勤率 vs 85%阈值
2. 有效出勤率 vs 名义出勤率（差值大→培训/闲置人员多）
3. 新员工占比 vs 出勤率的相关性
4. 离职率趋势（离职高→出勤低→业绩低）

**判断什么**：
- 出勤率<85% + 新员工占比高 → 扩编期，短期波动，不用过度反应
- 出勤率<85% + 离职率高 → 管理问题，需要关注
- 出勤率正常但业绩低 → 不是人的问题，是人效或业务的问题

**举例**：赛维斯3月有效出勤率78%，新员工占比35%（扩编80人），离职率正常 → 扩编培训期导致，不是管理松散。

> **场景C：质检数据**

**检查什么**：

1. 质检综合得分 vs 85分合格线
2. 扣分类型分布（话术/流程/数据安全/服务态度）
3. 集中在某些质检员还是分散？（标准问题vs执行问题）
4. 是否重复出现？（系统性vs偶发）

**判断什么**：
- 单类型扣分集中 → 针对性整改
- 多类型扣分分散 → 培训体系问题
- 扣分集中在某个产线 → 可能是产线产品复杂度高，需要专项培训
- 扣分持续改善 → 整改有效

**举例**：伽玛3月质检72分。扣分明细：话术不规范12次（占60%，集中在信用卡催收场景）→ 不是整体质量问题，是信用卡催收话术需专项培训。

### 第5步：形成结论和行动建议

**想什么**：分析的目的是输出结论，结论驱动行动。没有行动建议的分析是浪费时间。

**检查什么**：结论是否基于数据？行动建议是否具体到人+时间+交付物？

**判断什么**：每个结论必须附带一个明确的行动建议。

| 结论类型 | 行动模板 |
|---------|---------|
| 正常/稳定 | 持续监控，下次定期复查 |
| 预警信号 | 本周内私下沟通，确认原因 |
| 需干预 | 正式约谈 + 要求整改计划 |
| 需决策 | 准备数据报告，报上级决策 |

## 关键变量

| 变量 | 判断标准 | 影响 |
|------|---------|------|
| **数据完整性** | 缺失<10%可分析/>20%需补充 | 决定结论可靠度 |
| **对比基线** | 有3个月历史/仅1-2个月 | 有历史才能看趋势 |
| **预警阈值** | 后30%/出勤<85%/质检<85分/GMV降>20% | 触发即需行动 |
| **配合度** | 主动报/催了报/有水分 | 配合度低需交叉验证 |
| **时间预算** | 日均0.5h数据整理 | 控制在30分钟内 |

## 常见误区

| 误区 | 后果 | 正确做法 |
|------|------|---------|
| 拿到数据就开始做表 | 做了一堆表但没回答任何问题 | 先确定问题，再做分析 |
| 只看单月不看趋势 | 把正常波动当异常 | 至少看3个月趋势 |
| 只看排名不看维度 | 不知道排名变化的原因 | 排名变化必须做5维拆解 |
| 平均数掩盖极端值 | 3家供应商平均出勤率88%，但1家只有65% | 看分布，不只看均值 |
| 结论太抽象 | "建议加强管理"等于没说 | 每个结论必须带具体动作 |
| 数据不给口径 | 别人拿到你的分析看不懂 | 所有数据标注来源、口径、截止日期 |

## 决策输出

> **数据分析报告模板**

| 维度 | 内容 |
|------|------|
| 分析对象 | （哪些供应商 / 哪条产线 / 哪个时间段） |
| 数据来源 | （排名表 / 出勤系统 / 质检系统，标注截至日期） |
| 核心发现 | （最重要的3条发现，带数据） |
| 预警项 | （触发预警规则的供应商及触发情况） |
| 原因分析 | （每个预警项的可能原因，标注"已确认"或"待验证"） |
| 行动建议 | （具体动作 + 责任人 + 完成时间） |

> **示例**：分析对象：32家供应商3月数据 | 核心发现：伽玛从第5跌至第9（首次后30%）、翰锐连续2个月后30%、毅航连续3月前3 | 预警项：伽玛首次预警、翰锐连续预警 | 行动建议：伽玛本周私下沟通、翰锐正式约谈报备王易人
