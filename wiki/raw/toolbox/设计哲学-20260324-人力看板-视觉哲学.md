# 视觉哲学：校准过的精确 (Calibrated Precision)

> 为 BPO 供应商人力看板定义的视觉世界观

---

## 哲学核心

**这是一个内部管理工具，不是对外展示品。它的视觉语言应该像实验室仪器：精确、克制、可信赖。**

不是"好看"，是"好用"。不是"吸引眼球"，是"降低认知负担"。每一个视觉选择都应该服务于一个目标：让年老师能快速、准确地理解人力数据。

---

## 空间与形式

空间是功能性的，不是装饰性的。留白不是为了"极简美学"，是为了**降低信息密度带来的认知负担**。

形式来自数据本身，不是来自设计系统。一个增长箭头、一个峰值标记、一个异常高亮——这些不是事后添加的装饰，是数据的视觉表达。

网格是隐性的，不是显性的。不像"系统性张力"那样用背景网格强调精确性，这里的网格是**功能性的对齐系统**——元素对齐是为了降低阅读成本，不是为了展示设计感。

---

## 色彩系统

低饱和是必须的。高饱和色彩会降低数据的可信度——这是心理学事实，不是审美偏好。

**色彩的功能是区分，不是装饰。** 每一个颜色都必须回答："这个颜色帮助用户区分了什么？"

- 主色：低饱和蓝灰（冷静、专业、可信赖）
- 强调色：只用于异常/预警（红/橙）
- 背景：暖灰（降低对比度，减少眼疲劳）
- 数据序列：低饱和多色（区分维度，但不抢眼）

**整个页面没有"品牌色"。** 这不是品牌展示，是数据分析工具。

---

## 视觉层级

层级来自数据的重要性，不是设计的戏剧性。

- KPI 数字：最大，因为这是年老师最先看的
- 图表：中等，用于理解趋势
- 标签/说明：最小，但必须清晰可读
- 装饰性元素：无

**字体不是"设计选择"，是功能选择。** 数字用等宽字体（便于对比），文字用无衬线（易读），标签用小字号大写（区隔层级）。

---

## 隐性参考

这个视觉系统的隐性参考是**Bloomberg 终端 + 咨询公司内部工具**。

懂的人能看到：
- 色彩克制来自金融数据终端（数据可信度优先）
- 字体选择来自咨询公司内部系统（长时间阅读不疲劳）
- 布局密度来自专业 SaaS 工具（信息密度高但可读）

不懂的人只看到：这是一个非常专业、非常好用的内部工具，看起来像是某个顶尖团队花了很多时间打磨的。

---

## 工艺标准

最终产出必须看起来：
- 每一个数字都清晰可读（字体大小、字重、颜色都经过校准）
- 每一个颜色都有存在的理由（不是"好看"，是"有用"）
- 没有任何"为了设计而设计"的元素（无阴影、无渐变、无装饰）
- 像是一个专业团队为内部使用打造的工具，不是对外展示的模板

如果有人问"为什么 KPI 数字是 28px 不是 32px"，回答应该是："因为 28px 在保证可读性的同时，让一屏能放下更多 KPI。"

## 应用到 HTML 的技术要点

```css
/* 色彩系统 - 低饱和、可信赖 */
:root {
  --primary: #5b7fa8;        /* 低饱和蓝灰 */
  --primary-light: #8a9eb8;
  --primary-dark: #4a6a8f;

  --text-primary: #2d2a26;   /* 暖灰，不是纯黑 */
  --text-secondary: #6b6560;
  --text-tertiary: #9a948d;

  --bg: #f5f3ef;             /* 暖灰背景 */
  --surface: #ffffff;
  --border: #e8e4df;

  --danger: #b87e7e;         /* 低饱和红，用于预警 */
  --warning: #c4a96a;        /* 低饱和橙 */
  --success: #6d9e84;        /* 低饱和绿 */
}

/* 字体 - 功能优先 */
body {
  font-family: -apple-system, BlinkMacSystemFont, sans-serif;  /* 易读 */
}

/* KPI 数字 - 等宽字体便于对比 */
.kpi-value {
  font-variant-numeric: tabular-nums;  /* 等宽数字 */
}

/* 无阴影、无渐变、无装饰 */
.card {
  background: var(--surface);
  border: 1px solid var(--border);
  /* 无阴影 */
}

/* 圆角克制 - 4-8px，不是 16px */
.card {
  border-radius: 8px;  /* 不是 16px */
}
```

## 与原有设计的对比

| 维度 | 原有设计 | 校准过的精确 |
|------|---------|-------------|
| 圆角 | 16px | 8px |
| 阴影 | 多处使用 | 无 |
| 渐变 | 导航渐变 | 纯色 |
| 饱和度 | 中等 | 低饱和 |
| 装饰 | 有（徽章、图标） | 极少 |
| 字体 | 默认 | 等宽数字 |
| 色彩功能 | 区分 + 装饰 | 只区分 |

**产出**：一个看起来像是专业团队为内部使用打造的工具，而不是对外展示的精美模板。
