# Q7 客诉/合规事件

> 收到投诉或发现违规，快速定级、确认事实、决定反应速度。

## 30秒判断三件事

1. **事实清楚吗？**：投诉方的一面之词不能直接当结论。先确认：有没有录音/截图？质检有没有抓到？供应商自己承认了吗？事实不清的先不下结论。
2. **定级是什么？**：红线（数据造假/飞单/暴力催收，零容忍）/ 橙线（系统性话术违规/大面积质检不合格，48h整改）/ 黄线（个别坐席偏差/偶发投诉，限期整改）。
3. **年老师能自主处理吗？**：黄线年老师自主处理。橙线报王易人知情后年老师执行。红线立即报王易人，暂停业务，走正式流程。

## 一句话口径

- **红线**："这个情况比较严重，我需要先向上汇报，业务暂时先停一下。"（不对供应商解释细节，先冻结）
- **橙线**："这是系统性的问题，需要在48小时内给我一份整改方案。"（明确时限，不给模糊空间）
- **黄线**："这个问题需要关注，你们计划怎么处理？什么时候能给我反馈？"（要求限期，不替他们想方案）
- **事实未确认时**："我收到了一些反馈，先跟你们核实一下当时的情况。"（不预判定性，先听供应商说）

## 需要调的数据

- 投诉原文/录音/截图：客户投诉工单系统
- 涉事坐席质检记录（近30天）：质检系统
- 该供应商历史违规记录：供应商合规台账
- 是否涉及甲方策略变更导致的合规风险：策略组确认（查甲方责任场景表）

## 转给谁

- 红线事件 → 立即报王易人 + 暂停业务 + 留存证据，年老师不自主处理
- 橙线事件 → 报王易人知情 → 48h内出整改通知 → 年老师跟进整改结果
- 黄线事件 → 年老师自主处理 → 整改结果报王易人备案
- 事实不清 → 先核实，不急着定级，核实后再按上述流程走
- 涉及多供应商/多产线的合规风险 → 报王易人 + 军哥，升级处理
