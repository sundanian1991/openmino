# E1 新供应商初筛

> 30分钟内判断一家新供应商值不值得深入了解，不值得就别浪费双方时间。

## 触发信号

- 产线业务负责人推荐了一家供应商
- 现有供应商产能不足，需要储备替代
- 行业会议/同行交流中发现潜在合作方
- 某产线（如财富线）供应商集中度偏高，需要分散风险
- 供应商主动找上门谈合作

## 核心问题

> 这家供应商有没有可能通过准入评审？如果连60分都摸不到，尽早终止，别走完9维度评估再发现硬伤。

## 思考步骤

### 第1步：确认需求背景

**想什么**：为什么现在需要新供应商？是扩容、替代、还是分散风险？

**检查什么数据**：
- 当前该产线的供应商数量和排名（查最近月度排名表）
- 各供应商产能利用率（是否已有供应商还能加量）
- 最近3个月有没有供应商清退或预警记录

**判断什么**：
- 如果现有供应商产能充裕且排名稳定 → 不急于引入，降低优先级
- 如果某产线只剩1-2家供应商 → 分散风险，提高优先级
- 如果是金条线且现有6家头部稳定 → 新供应商很难挤进分量，慎重考虑

**举例**：赛维斯在金条线连续3个月排名前3，产能利用率90%以上 → 不急于在金条线加供应商。但财富线只有2家，其中1家最近质检下滑 → 财富线新供应商优先级高。

### 第2步：快速硬伤筛查（一票否决项）

**想什么**：有没有什么问题是一旦存在就直接淘汰的？

**检查什么**：
- 工商信息：注册时间、注册资本、经营异常记录（天眼查/企查查）
- 是否在同行业竞品（如京东金融之外的其他消费金融平台）做同类业务
- 有没有被监管部门处罚过的记录
- 核心团队是否来自已被清退的供应商（如某供应商因合规问题退出，核心人员换个壳再来）

**判断什么**：
- 注册不满2年 → 高风险，除非有强背书（如上市公司子公司）
- 注册资本低于500万 → 扛不住业务波动，直接淘汰
- 在竞品平台因合规问题被清退 → 一票否决
- 团队来自清退供应商 → 需要额外调查，不直接否决但要警惕

**举例**：某公司注册资本200万，成立1年，核心团队来自之前被我们清退的供应商A → 直接淘汰，不走后续流程。

### 第3步：业务能力初判

**想什么**：这家公司干得了我们要他们干的活吗？

**检查什么**：
- 过往做了什么业务（电销？客服？信审？）和哪些平台合作过
- 团队规模（现有多少人，能扩到多少人）
- 是否有金融行业/消费金融领域的交付经验
- 管理层背景（是否懂电销业务逻辑）

**判断什么**：
- 有金条/信用卡同类业务经验 → 加分，上手快
- 只做过客服没做过电销 → 可以考虑但培训成本高，成长期可能拉长
- 团队规模不到100人 → 只适合小产线试水，不适合金条线
- 管理层不懂业务 → 后续管理成本极高，慎重

**举例**：毛毛虫电销团队800人，做过同类平台催收和电销 → 业务匹配度高。但如果一家公司只做过保险电销，没碰过消费金融 → 需要在引入期额外评估培训投入。

### 第4步：配合度预判

**想什么**：这家公司好不好合作？会不会成为"难搞的供应商"？

**检查什么**：
- 第一次接触时的响应速度（发了信息多久回复）
- 能否按要求提供基本信息（营业执照、团队架构、业绩数据）
- 是否主动提不合理要求（"能不能先给我们保证X万的量"）
- 有没有现成的合规体系（质检流程、数据安全管理）

**判断什么**：
- 响应快、资料齐、态度务实 → 配合度预期好
- 第一次接触就讨价还价 → 后续谈判会很累，标记风险
- 拿不出任何管理文档 → 管理成熟度低，引入后管理成本高
- 过度承诺（"我们什么都能做"）→ 典型的盲目承诺，实际交付大概率拉胯

**举例**：某供应商首次沟通就说"你们给多少量我们都能接" → 过度承诺信号，需要追问具体产能依据和团队储备情况。

### 第5步：性价比初估

**想什么**：引入这家供应商的成本和收益大致是什么水平？

**检查什么**：
- 报价区间（不需要精确，了解大致水平）
- 当前同类型供应商的成本结构（对比现有供应商的单客成本）
- 引入后的管理投入（是否需要派驻质检、培训周期多长）

**判断什么**：
- 报价比现有供应商高20%以上 → 除非有不可替代的优势，否则不值得
- 报价明显低于市场价 → 要追问为什么，可能质量把控有问题
- 管理投入估算超过收益 → 不值得

**举例**：现有金条线供应商单客成本约50元，新供应商报价70元且无独特优势 → 性价比不够，除非金条线急需扩容。

### 第6步：形成初筛结论

**想什么**：综合以上5步，给出明确判断。

**判断什么**：
- **通过初筛**：无硬伤 + 业务能力匹配 + 配合度尚可 → 进入E2准入评审
- **有条件通过**：有小问题但可控 → 列出需要额外验证的点，进入E2但重点核查
- **暂缓**：当前不需要或信息不足 → 加入潜在供应商池，定期跟进
- **淘汰**：有硬伤或综合评估不达标 → 记录原因，归档备查

## 关键变量

| 变量 | 判断标准 | 数据来源 |
|------|---------|---------|
| **当前产能缺口** | 现有供应商利用率是否≥85%且无扩产空间 | 月度产能报表 |
| **硬伤红线** | 注册<2年、资本<500万、合规污点、竞品黑名单 | 天眼查/企查查 |
| **业务匹配度** | 金融行业电销/客服经验 + 团队≥100人 | 供应商自述 + 行业打听 |
| **配合度信号** | 首次接触响应<24h、能提供基本资料、不盲目承诺 | 实际沟通记录 |
| **性价比区间** | 报价在现有供应商均价正负20%以内 | 成本分析表 |

## 常见误区

1. **被"大公司"牌子唬住**：注册资本大不代表电销能力强，要看实际团队规模和业务匹配度
2. **忽略团队来源**：核心团队来自清退供应商是个重要信号，容易被忽略
3. **过度关注报价**：初筛阶段谈价格容易跑偏，重点是"能不能干这个活"
4. **被推荐人背书影响判断**：产线负责人推荐不代表一定合适，仍需独立判断
5. **急扩容时降低标准**：越是缺人越要守住硬伤红线，否则后续管理成本更高
6. **忽视"过度承诺"信号**：说什么都能做的供应商，往往什么都做不好
