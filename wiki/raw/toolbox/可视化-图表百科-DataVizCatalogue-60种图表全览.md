## 比较（14 种）

### 条形图

- **说明**：通过水平或垂直条形对比不同类别的离散数值，聚焦单一类别的数量对比
- **数据结构**：分类变量 + 对应离散数值
- **最佳实践**：控制条形总量以保证标注清晰；条形过多时难以标签
- **关联图表**：直方图、多组条形图、人口金字塔、径向条形图、堆叠式条形图
- **代码工具**：D3.js, Highcharts, Plotly

### 子弹图

- **说明**：显示表现数据，功能类似条形图但加入更多视觉元素（比较度量、定性范围），节省空间
- **数据结构**：功能度量、比较度量、定性范围等级
- **最佳实践**：建议最多使用五个等级；能有效节省空间
- **关联图表**：条形图
- **代码工具**：amCharts, AnyChart, DimpleJS, NVD3.js(D3), Plotly, Vega-Lite

### 多组条形图

- **说明**：在同一轴线上将两个或多个数据集并排绘制和分类，用独立颜色区分
- **数据结构**：共享相同分类维度的 2+ 个数据系列
- **最佳实践**：严格控制每组内条形数量，过多时难以阅读
- **关联图表**：条形图、直方图、堆叠式条形图
- **代码工具**：Chart.js, D3.js, Highcharts, Plotly, ggplot2

### 径向条形图

- **说明**：在极坐标系上绘制的条形图，主要为了美观
- **数据结构**：类别维度 + 数值数据
- **最佳实践**：外围条形显得更长，容易误解；视觉系统不擅解读弧线长度
- **关联图表**：条形图、径向柱图
- **代码工具**：amCharts, AnyChart, Apache ECharts, JSCharting

### 径向柱图

- **说明**：使用同心圆网格绘制条形图，数值由中心向外递增
- **数据结构**：类别维度 + 同心圆刻度的数值
- **最佳实践**：刻度上较低数值由中心开始，外圆可设为零以显示负值
- **关联图表**：条形图、径向条形图、直方图、堆叠式条形图
- **代码工具**：amCharts, D3.js, Bokeh, jChartFX

### 人口金字塔

- **说明**：背靠背的一对直方图，每边代表一个性别，显示年龄组与人口分布
- **数据结构**：性别分组 + 年龄组别 + 人口数值
- **最佳实践**：最适合检测人口模式变化；可并列比较不同群体
- **关联图表**：条形图、直方图
- **代码工具**：amCharts, AnyChart, Vega, Vega-Lite, ZingChart

### 饼形图

- **说明**：将圆圈划分成不同比例分段，展示类别间比例和百分比
- **数据结构**：分类字段 + 数值字段（合计 100%）
- **最佳实践**：不能显示太多项目；占用空间大；不能准确比较不同饼图
- **关联图表**：圆环图、南丁格尔玫瑰图、旭日图
- **代码工具**：Apache ECharts, D3.js, Highcharts, Plotly, Vega-Lite

### 圆环图

- **说明**：中间被切掉的饼形图，促使读者聚焦弧线长度与总体数值变化
- **数据结构**：分类字段 + 数值字段
- **最佳实践**：中央留白可附加额外信息
- **关联图表**：饼形图、旭日图
- **代码工具**：Chart.js, D3.js, Plotly, Vega

### 南丁格尔玫瑰图

- **说明**：又称极面积图，绘制于极坐标系，分段面积代表数值（非半径）
- **数据结构**：数据类别 + 数值
- **最佳实践**：面积较大的外围部分更引人注目，与数值增量成反比
- **关联图表**：饼形图、径向柱图
- **代码工具**：amCharts, AnyChart, Apache ECharts, Chart.js, Vega

### 象形图

- **说明**：借助与主题相符的图标直观呈现数值，每个符号代表固定基数
- **数据结构**：分类明确的少量计数数据
- **最佳实践**：避免用于大型数据集；避免只显示部分图案
- **关联图表**：条形图、点阵图表
- **代码工具**：ECharts, Vega-Lite

### 比例面积图

- **说明**：依靠图形面积传递量级，快速了解数据相对大小，无需使用刻度
- **数据结构**：定量数值
- **最佳实践**：必须计算形状的空间面积而非用长度来确定大小；仅适合传达目的
- **代码工具**：D3.js, Python(circlify), R(ggplot2), Vega

### 点阵图表

- **说明**：以点为单位显示离散数据，每种颜色表示一个特定类别，按矩阵排列
- **数据结构**：分类变量 + 数值（点数）
- **关联图表**：象形图
- **代码工具**：ZingChart, Vega-Lite

### 记数符号图表

- **说明**：利用标记系统来展示数据频率，作为数据记录工具
- **数据结构**：类别/数值 + 计数标记
- **最佳实践**：逐一添加符号，最后加总写在下一列/行
- **关联图表**：直方图

### 不等宽柱状图

- **说明**：显示分类数据中一对变量之间的关系，双向等长条形图，两轴均设刻度
- **数据结构**：两对变量数据（决定宽高）
- **最佳实践**：区块不沿共同基线排列，精确对比困难；分段多时难以阅读
- **关联图表**：堆叠式条形图、树形结构图
- **代码工具**：AnyChart, FusionCharts, D3.js, Vega-Lite

### 雷达图

- **说明**：比较多个定量变量，查看哪些变量具有相似数值或异常值
- **数据结构**：多个定量变量（每个映射到独立轴线）
- **最佳实践**：严格控制变量数量；切勿在同一画面绘制多个多边形
- **关联图表**：平行座标图
- **代码工具**：Apache ECharts, Highcharts, Plotly

### 文氏图

- **说明**：显示集与集之间所有可能的逻辑关系，重迭区域为交集
- **数据结构**：至少 2 个数据集，数据项可归类至一个或多个集合
- **关联图表**：树形结构图、小提琴图
- **代码工具**：AnyChart, D3.js, matplotlib_venn, ZingChart

### 平行座标图

- **说明**：利用独立平行轴线与贯穿连线，同步呈现多个数值变量及其关联
- **数据结构**：多变量数值数据（相同属性的系列记录）
- **最佳实践**：变换轴序揭示模式；启用"刷理"交互技术减少视觉杂乱
- **关联图表**：平行集合图、雷达图
- **代码工具**：Apache ECharts, D3.js, Plotly, Vega

## 趋势/时间（12 种）

### 折线图

- **说明**：在连续间隔或时间跨度上显示定量数值，最常用来显示趋势和关系
- **数据结构**：Y 轴定量数值 + X 轴分类/顺序刻度
- **代码工具**：Chart.js, D3.js, Highcharts, Plotly, matplotlib, ggplot2

### 面积图

- **说明**：折线图变体，线下区域由颜色或纹理填满，显示某时间段内量化数值变化
- **数据结构**：时间序列 + 定量数值
- **最佳实践**：常见分组式和堆叠式两种形式
- **关联图表**：折线图、堆叠式面积图
- **代码工具**：Chart.js, Highcharts, Plotly, matplotlib, D3, Vega

### 堆叠式面积图

- **说明**：每个系列开始点是前一个系列的结束点，展示多组数据累积叠加效果
- **数据结构**：多个可叠加数据系列 + 公共分类轴
- **最佳实践**：数值必须为整数（不适用于负值）；严格避免负数
- **关联图表**：面积图、折线图、量化波形图
- **代码工具**：Chart.js, ECharts, D3.js, Plotly

### 蜡烛图（K 线图）

- **说明**：显示证券/商品随时间的价格变动（开盘、收盘、最高、最低）
- **数据结构**：时间序列：开盘价、收盘价、最高价、最低价
- **最佳实践**：实体越长压力越强烈；42 种图案模式可预测走势
- **关联图表**：卡吉图、美国线
- **代码工具**：amCharts, AnyChart, Google Charts, Plotly, Vega-Lite

### 美国线（OHLC）

- **说明**：追踪证券或商品时段价格波动，垂直线段标示高低区间，短横线标记开盘/收盘
- **数据结构**：时间序列：开盘价、最高价、最低价、收盘价
- **关联图表**：蜡烛图、卡吉图
- **代码工具**：amCharts, AnyChart, Apache ECharts, FusionCharts, Plotly

### 甘特图

- **说明**：项目管理的组织工具，显示活动列表和持续时间，条形长度代表持续时间
- **数据结构**：任务名称、开始/结束时间、分类、依赖关系、完成百分比
- **最佳实践**：用颜色分类活动；箭头显示依赖关系；用符号表示里程碑
- **关联图表**：跨度图
- **代码工具**：amCharts, AnyChart, FusionCharts, Highcharts, Plotly

### 时间线

- **说明**：以时间顺序显示一系列事件的图像化方式，传达时间相关信息
- **数据结构**：事件 + 对应时间点/时间段
- **代码工具**：AnyChart, Google Charts, KronoGraph, Vega

### 螺旋图

- **说明**：沿阿基米德螺旋线绘制基于时间的数据，从中心点往外发展
- **数据结构**：基于时间的序列数据（含周期性）
- **最佳实践**：适合大型数据集和长时间段趋势；可给每个周期段分配颜色
- **代码工具**：D3.js, ggplot(coord_polar)

### 量化波形图

- **说明**：堆叠式面积图变体，数值沿波动中心基线绘制，形成河流状流线
- **数据结构**：时间序列 + 多个分类变量 + 连续定量数值
- **最佳实践**：侧重呈现一般趋势，不适合精确读数；建议以互动形式展示
- **关联图表**：堆叠式面积图
- **代码工具**：Apache ECharts, Vega-Lite, NVD3.js, ZingChart

### 日历

- **说明**：将日期单元顺序排布的可视化工具，显示不同时间段和活动事件组织情况
- **数据结构**：日期 + 事件/活动信息
- **最佳实践**：外观灵活多变，关键确保时间节点按先后次序清晰展示
- **关联图表**：时间表
- **代码工具**：Vega, AnyChart

### 时间表

- **说明**：用作预定事件、任务和行动的引用和管理工具，展示班次时刻
- **数据结构**：时间顺序的事件表格数据
- **最佳实践**：按时间顺序和/或字母顺序排列提升查阅效率
- **关联图表**：日历

## 分布（7 种）

### 箱形图

- **说明**：显示数字数据组的四分位数，晶须展示四分位数之外的分布及异常值
- **数据结构**：连续数值变量（可计算四分位数、均值、异常值）
- **最佳实践**：占用较少空间，适合比较很多组之间的分布
- **关联图表**：直方图、密度图、跨度图、小提琴图
- **代码工具**：AnyChart, Plotly, Seaborn, R, Vega, D3

### 直方图

- **说明**：显示连续间隔或特定时间段内的数据分布，估计集中位置、上下限值
- **数据结构**：数值按连续间隔/时间段分组 + 频率
- **关联图表**：条形图、密度图、人口金字塔
- **代码工具**：D3.js, Plotly, Vega

### 密度图

- **说明**：直方图变体，用平滑曲线绘制数值水平，不受分组数量影响
- **数据结构**：连续数值数据
- **最佳实践**：更好界定分布形状，规避因分箱选择导致的偏差
- **关联图表**：直方图
- **代码工具**：R, D3.js, Python(Seaborn/Matplotlib), Vega

### 小提琴图

- **说明**：显示数据分布及概率密度，结合箱形图和密度图特征
- **数据结构**：连续数值变量
- **最佳实践**：比箱形图显示更多详情，但也可能包含较多干扰信息
- **关联图表**：箱形图
- **代码工具**：D3.js, Highcharts, Python(seaborn), R(ggplot2), Vega

### 茎叶图

- **说明**：按位数组织数据，纵向固定十位作茎，横向罗列个位作叶
- **数据结构**：可拆分为十位与个位的数值型数据
- **最佳实践**：只能处理有限大小数据集；对比两组建议用双重形式
- **关联图表**：记数符号图表

### 点示地图

- **说明**：在地理区域上放置相等大小圆点，检测空间布局或数据分布
- **数据结构**：地理区域 + 数值（单点单值或单点代表单位量）
- **最佳实践**：适合展示空间分布概览，不宜用于读取精确数值
- **关联图表**：气泡地图
- **代码工具**：D3.js, Plotly, Vega-Lite, Python(folium), R(leaflet)

## 地理/位置（4 种）

### 地区分布图（Choropleth）

- **说明**：用不同深浅颜色表示数据变量，显示不同地理分区与数据变量的关系
- **数据结构**：地理区域（分区）+ 归一化数值指标
- **最佳实践**：建议使用归一化值而非原始数据；大面积区域会过度影响视觉感知
- **关联图表**：弦图、其他地理类图表
- **代码工具**：D3.js, Python(geopandas/folium), R(ggplot2+leaflet)

### 连接地图

- **说明**：用直线或曲线连接地图上不同地点，显示地理连接和关系
- **数据结构**：起点/终点坐标 + 连接关系数据
- **关联图表**：流向地图、网络图
- **代码工具**：amCharts, AnyChart, D3.js, yFiles

### 流向地图

- **说明**：在地图上显示信息或物体从一位置到另一位置的移动及数量
- **数据结构**：起点坐标 + 终点坐标 + 流量数值
- **最佳实践**：建议将流向线合并/捆绑，避免彼此交叠
- **关联图表**：连接地图、桑基图
- **代码工具**：AnyChart

### 气泡地图

- **说明**：指定地理区域上方显示圆形，面积与数值成正比
- **数据结构**：地理区域标识 + 对应数值
- **最佳实践**：控制气泡尺寸，避免过大导致重迭遮盖
- **关联图表**：点示地图、比例面积图
- **代码工具**：amCharts, Plotly, Vega

## 关联/关系（6 种）

### 散点图

- **说明**：在笛卡尔坐标上放置数据点，检测两个变量之间的关系或相关性
- **数据结构**：一对数值数据（映射至两坐标轴）
- **最佳实践**：注意"相关性并非因果关系"；留意远离聚集区的异常值
- **关联图表**：气泡图
- **代码工具**：D3.js, Chart.js, Plotly, ECharts, ggplot2

### 气泡图

- **说明**：融合散点图与比例面积图，笛卡尔坐标定位，面积表示第三变量
- **数据结构**：至少 3 个变量（横纵坐标各一，面积对应第三变量）
- **最佳实践**：确保圆圈按面积绘制（非半径/直径）；避免过多气泡
- **关联图表**：散点图、比例面积图
- **代码工具**：Chart.js, ggplot2, Highcharts, Python/R, Vega

### 弦图

- **说明**：显示不同实体之间的相互关系和共通之处，适合比较数据集相似性
- **数据结构**：实体间关系数据 + 连接数值 + 可选分类标签
- **最佳实践**：避免显示过多连接以防混乱；利用颜色区分不同类别
- **关联图表**：非彩带弦图、地区分布图
- **代码工具**：amCharts, D3.js, Highcharts, Plotly, R(circlize)

### 非彩带弦图

- **说明**：弦图的精简版本，仅显示节点和连接线，更强调连接关系
- **数据结构**：节点 + 连接关系数据
- **关联图表**：弦图
- **代码工具**：amCharts, Apache ECharts, D3.js, R(ggraph), Vega

### 网络图

- **说明**：使用节点/顶点和连接线显示事物之间的连接关系，阐明实体关系类型
- **数据结构**：节点/顶点 + 连接线数据
- **最佳实践**：数据容量有限，节点太多时形成"毛球"难以阅读
- **关联图表**：弧线图、脑力激荡图、非彩带弦图、连接地图、树形图
- **代码工具**：amCharts, Apache ECharts, D3.js, Sigma.js, igraph

### 弧线图

- **说明**：节点沿 X 轴放置，用弧线表示节点间连接关系，弧线粗细表示频率
- **数据结构**：一维序列节点清单 + 成对连线数据 + 频率数值
- **最佳实践**：适合查找数据共同出现情况；控制连线密度避免杂乱
- **关联图表**：面积图、二维双轴图表
- **代码工具**：Data to Viz(R), Highcharts, D3

## 占比/部分对整体（6 种）

### 树形结构图（Treemap）

- **说明**：利用嵌套式矩形显示层次结构，同时通过面积大小显示每个类别数量
- **数据结构**：层次结构数据 + 各子类别数量/数值
- **最佳实践**：紧凑且节省空间的层次结构显示方式
- **关联图表**：圆堆积、不等宽柱状图、旭日图
- **代码工具**：amCharts, Apache ECharts, D3.js, Highcharts, Plotly, Vega

### 旭日图

- **说明**：通过一系列圆环显示层次结构再按类别切割，每环代表一个级别
- **数据结构**：层级关系数据（根节点、子节点、对应数值）
- **最佳实践**：用不同颜色突出显示层次分组或特定类别
- **关联图表**：圆环图、饼形图、树形结构图
- **代码工具**：amCharts, Apache ECharts, FusionCharts, Plotly, D3.js, Vega

### 堆叠式条形图

- **说明**：将多个数据集的条形彼此重迭，显示大类别如何细分为小类别
- **数据结构**：分类维度 + 多组数值型数据
- **最佳实践**：分段越多越难阅读；分段不在同一基线上，跨组比较困难
- **关联图表**：条形图、多组条形图、堆叠式面积图
- **代码工具**：D3.js, Plotly, matplotlib, ggplot2, Vega

### 桑基图

- **说明**：显示流向和数量（彼此之间的比例），通过箭头粗细表现数据流量大小
- **数据结构**：流向数据（源节点、目标节点、流量数值）
- **关联图表**：流向地图、平行集合图
- **代码工具**：Apache ECharts, D3.js, Plotly, Google Charts

### 平行集合图

- **说明**：与桑基图类似，在每个线集划分流程路径，不用箭头
- **数据结构**：每线集对应一个维度，数值/类别由线段表示，线宽由比例决定
- **最佳实践**：每条路径用不同颜色以显示和比较不同类别分布
- **关联图表**：桑基图、平行座标图
- **代码工具**：amCharts, Apache ECharts, D3.js, Plotly

## 层次结构（2 种）

### 树形图

- **说明**：通过树状结构表示层次结构，由根节点经分支连接各节点至末端枝叶
- **数据结构**：树状/层级结构数据（根节点、分支、末端节点）
- **最佳实践**：适用于家谱、分类学、组织管理等
- **代码工具**：Apache ECharts, Google Chart, JSCharting, ZingChart

### 圆堆积

- **说明**：树状图的改版，使用圆形一层又一层地代表整个层次结构
- **数据结构**：树状层级数据 + 数值（控制圆面积）+ 可选分类/连续变量
- **最佳实践**：通过面积量化主指标，色相区分群组，色度表达渐变数值
- **关联图表**：树形结构图
- **代码工具**：AnyChart, D3.js, matplotlib, R(ggraph), Vega

## 财务/交易（2 种）

### 卡吉图

- **说明**：以连续线段追踪资产价格走势，反映市场供需，与时间无直接关系
- **数据结构**：价格数据 + 预设"逆转"水平
- **最佳实践**：认清出现的模式；利用粗幼/阴阳转变作为买卖信号
- **关联图表**：蜡烛图、美国线、点数图
- **代码工具**：D3.js, FusionCharts, React+D3.js, Wolfram

### 点数图（P&F）

- **说明**：用 X/O 号追踪价格涨跌，检测支持/阻力水平和趋势线
- **数据结构**：价格数据 + 每格数值 + 逆转数值
- **最佳实践**：较小逆转值产生更多波动，较大值滤除微小波动；注意突破信号
- **关联图表**：卡吉图

- **数据结构**：流程步骤 + 逻辑连接关系
- **代码工具**：FusionCharts, GoJS, yWorks

### 说明图

- **数据结构**：视觉素材 + 文字批注与符号标识
- **最佳实践**：明确区分主视觉与注解区域；细节密集时可截取局部放大

## 范围/误差（2 种）

### 误差线

- **说明**：显示笛卡尔坐标上数据的估计误差或不确定性（标准差、置信区间等）
- **数据结构**：均值/估计值 + 误差幅度
- **最佳实践**：较短小的误差线表示数值集中、平均值更准确
- **关联图表**：可应用于面积图、条形图、直方图、折线图、散点图等
- **代码工具**：amCharts, AnyChart, FusionCharts, Highcharts, Vega

### 跨度图

- **说明**：又称范围条形图，呈现各分类的最小与最大端点值
- **数据结构**：分类 + 最小值 + 最大值
- **最佳实践**：只集中显示极端数值，不涵盖区间内具体数值或均值
- **关联图表**：箱形图、蜡烛图、甘特图
- **代码工具**：AnyChart, D3.js, JSCharting, Vega-Lite, ZingChart
