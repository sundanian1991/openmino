# A2-约谈准备

> 触发约谈条件（首次预警/连续预警/A级骤降）时，系统化准备约谈内容与要求

## 触发信号
- 首次预警：本月落入后30%，上月不在
- 连续预警：连续2个月在后30%
- A级骤降：上月前30%本月跌到后30%
- 诚信预警/红线：数据质量问题
- 其他异常：周度业绩断崖>20%、主管胜任≤2项

## 核心问题
- 这次约谈的目标是什么？（查明根因/要求整改/施加压力）
- 约谈的框架是什么？（数据呈现→引导提问→要求产出→记录存档）
- 我要准备哪些具体数据和证据？
- 整改要求如何量化？（目标/动作/时间节点）
- 需要哪些人参加？（仅主管/需高层/需军哥在场）

## 思考步骤

**Step 1：明确约谈类型与目标（工具箱§五）**
```
约谈类型矩阵：
┌─────────────┬──────────────┬──────────────┬──────────────┐
│ 触发条件     │ 约谈类型     │ 核心目标     │ 参与人       │
├─────────────┼──────────────┼──────────────┼──────────────┤
│ 首次预警     │ 私下沟通     │ 问原因，了解情况 │ 主管         │
│ 连续预警     │ 正式约谈     │ 要求整改计划   │ 主管+老板    │
│ A级骤降     │ 紧急约谈     │ 查明重大根因   │ 主管+老板+军哥 │
│ 诚信预警     │ 警告谈话     │ 告知后果，整改数据 │ 主管+老板    │
│ 诚信红线     │ 严肃约谈     │ 高层沟通，告知红线 │ 高层+军哥    │
└─────────────┴──────────────┴──────────────┴──────────────┘
```
- 首次预警：单人（主管），30分钟，了解情况为主
- 连续预警：主管+老板，1小时，要求提交整改计划
- A级骤降：24-48h内约谈，主管+老板+军哥（如涉及重大决策）

**Step 2：准备数据呈现（工具箱§五：数据呈现5min）**
```
4个必须准备的数据：
1. 本月排名：第X名（后30%），较上月变化（↓N名）
2. GMV达成率：目标YY%，实际ZZ%，差距[绝对值+百分比]
3. 核心失分项：哪个维度拉低最多？（业绩/合规/配合/组织/成长）
4. 横向对比：同产线均值（低YY%）、同产线第1名（差距ZZ%）

辅助数据（根据约谈类型准备）：
□ 产能双维度：自身峰值比 + 头部差距 → 判断是自身问题还是产线问题
□ 主管胜任5项：哪些不达标？(流失率/出勤/人效趋势/响应/整改)
□ 周度突变信号：出勤/业绩/流失是否异常？
□ 新人拉低幅度：新人占比>30%时计算
```
**数据呈现原则**：
- 只摆事实，不解释（让供应商自己说原因）
- 用对比（环比/对标）而非绝对值
- 图表优于文字（一页纸红绿灯月报或双指标卡片）

**Step 3：设计引导式提问（工具箱§五：引导式提问15min）**
```
问题清单（按约谈类型选用）：

通用问题（每次必问）：
1. "看数据[具体问题]，你怎么看？"
2. "这个问题是什么时候开始的？"
3. "你觉得最根本的原因是什么？（追问3个为什么）"

首次预警（了解情况）：
4. "这个月发生了什么导致排名下滑？"
5. "名单/人员/产品有变化吗？"
6. "需要我们什么支持？"

连续预警/A级骤降（要求整改）：
7. "你已经采取了哪些动作？效果如何？"
8. "下个月目标是什么？具体怎么达成？"
9. "需要哪些资源？我们怎么配合？"
10. "如果继续下滑，你觉得会怎样？"

主管胜任问题：
11. "从数据看你的管理效果在下滑，有没有考虑过管理团队调整？"
12. "流失率这么高，你知道团队在想什么吗？"

诚信问题：
13. "数据异常你怎么解释？"
14. "如果核查发现造假，你知道后果吗？（清退+诚信档案）"
```
**提问原则**：
- 先让供应商说，不打断
- 用"你怎么看"而不是"你错了"
- 追问3个为什么挖根因
- 不替供应商想答案，引导他自己找到

**Step 4：要求产出（工具箱§五：要求产出5min）**
```
首次预警：
□ 口头说明原因（记录存档）
□ 承诺下周起改进动作（3-5条）

连续预警/A级骤降：
□ 提交书面整改计划（必须包含）：
  ✓ 具体目标（量化）：下月排名提升到前XX%
  ✓ 具体动作（3-5条）：每条做什么
  ✓ 时间节点：每条动作完成时间
  ✓ 根因对应：动作针对哪个问题
□ 复盘日期：X月X日复盘进展

诚信问题：
□ 书面说明（48h内）
□ 整改承诺（如有）
```
**整改计划评估标准（工具箱§六）**：
- 目标具体：量化到数字（排名/达成率/出勤率）
- 动作具体：3-5条，每条有具体做法
- 时间节点：每条动作有完成时间
- 根因对应：动作针对问题，不跑偏
- 不合格 → 退回重写 + 说明原因

**Step 5：记录存档（工具箱§五）**
```
必须记录：
□ 约谈时间/地点/参与人
□ 供应商陈述的根因（原话记录）
□ 供应商承诺的改进动作
□ 我提出的要求和标准
□ 约定复盘时间
□ 整改计划（如有）存档位置
```
- 首次预警：记录到沟通档案即可
- 连续预警/A级骤降：整改计划存入供应商空间（工具箱§十八）
- 诚信问题：记录到诚信档案，标记"加倍核查"3个月

## 关键变量
- **约谈类型**：首次/连续/A级骤降/诚信，决定参与人和严肃程度
- **供应商历史**：是否惯犯？是否有整改成功历史？
- **问题性质**：业绩问题/合规问题/配合问题，决定追问方向
- **上级参与**：A级骤降/诚信红线需军哥在场，提前报备
- **时间压力**：立即约谈（24-48h）vs 可延后（3-5天）

## 常见误区
- ❌ 数据呈现超过10分钟：应控制在5分钟内，快速过数据
- ❌ 替供应商想解决办法：应引导式设问，让他自己找答案
- ❌ 目标不量化："会努力的"不合格，必须"下月排名前50%"
- ❌ 没有时间节点：每条整改动作必须明确完成时间
- ❌ 不记录存档：口头沟通无记录，后续无法追溯
- ❌ 首次预警就上强度：首次预警以了解情况为主，不施加过大压力

## 决策输出

**约谈邀约话术（给供应商）**：
```
[供应商名]，你好。系统数据显示[具体问题，如：本月排名后30%]，我们需要当面沟通了解情况。请安排[时间]来办公室，预计1小时。请提前准备：[具体准备内容，如：近一个月数据说明/改进思路]。
```
**约谈跟进话术（约谈后）**：
```
[供应商名]，今日约谈总结：
问题：[一句话总结]
你说明的根因：[供应商说的根因]
我要求：[具体整改要求]
时间节点：[完成时间]
请于[时间]前提交书面计划。
```

**约谈准备清单**（打印或内部记录）：
```
□ 数据准备：排名表 + 双维度表 + 主管胜任表 + 周度异常信号
□ 问题清单：根据约谈类型准备8-10个引导问题
□ 整改要求模板：明确目标/动作/时间节点要求
□ 记录表：约谈记录表（时间/参与人/要点/承诺）
□ 衔接模板：operations-missing.md（约谈邀约/整改通知）
```

## 下一步衔接
- 约谈邀约 → `operations-missing.md`（约谈邀约模板）
- 整改通知 → `operations-missing.md`（整改通知模板，附表格）
- 整改计划评估 → 调用【A7-整改计划评估】
