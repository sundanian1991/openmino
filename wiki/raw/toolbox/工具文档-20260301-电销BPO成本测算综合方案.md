# 电销BPO成本测算方案

> **三种方法，三种场景**：
> - **同行价格参考** → 快速报价、谈判依据
> - **社平工资倍数法** → 成本估算、预算编制
> - **供应商成本推导** → 深度分析、成本谈判

---

## 一、同行结算价格参考

四种主流收费模式，按**业务特性**选择：

| 模式 | 价格 | 适合什么项目 | 注意什么 |
|------|------|-------------|---------|
| **按坐席** | 6000-7000元/月 | 业务稳定、通话均匀 | 坐席利用率低会推高单价 |
| **按时长** | 按实际通话分钟 | 咨询类、重沟通质量 | 需监控效率，防止拖延 |
| **按业绩** | 放款金额1-2% | 销售导向、结果可衡量 | 高利润产品会挤占其他推广 |
| **混合** | 低底薪+低提成 | 最常用，兼顾双方 | 需明确基础服务标准 |

**选型建议**：
- 新项目/不稳定 → 按坐席（成本可控）
- 成熟项目/销售导向 → 按业绩（激励性强）
- 平衡风险 → 混合模式

## 二、社平工资倍数法

### 核心公式

```
单坐席月成本 = 社平工资 × 倍数 + 绩效提成
```

### 为什么用"倍数"？

员工到手工资只是成本的一部分。企业还需承担：

| 成本项 | 占比 | 说明 |
|--------|------|------|
| 社保公积金（单位部分） | 30-40% | 法定支出 |
| 福利费 | 300-800元 | 餐补、交通、节日福利 |
| 管理分摊 | 600-1200元 | 场地、设备、管理人员 |
| **综合成本** | **约工资的1.5倍** | 不含利润 |

### 倍数怎么选？

| 倍数 | 适用场景 | 说明 |
|------|---------|------|
| **1.5-1.8** | 竞争激烈、规模效应 | 利润薄，走量 |
| **2.0-2.5** | 常规项目 | 合理区间，覆盖成本有利润 |
| **>2.5** | 高难度/多语言/品牌溢价 | 特殊项目，需额外投入 |

### 测算示例

**假设**：社平工资 10,343 元/月，倍数 2.0

```
单坐席月成本 = 10,343 × 2.0 = 20,686 元
年度成本 = 20,686 × 12 = 248,232 元
100人团队/月 = 206.9 万元
```

**验证**：市场行情 6000-7000 元/坐席，是供应商报价（含利润）。用倍数法估算的是**成本底线**，可作为谈判参考。

## 三、供应商成本明细

单坐席月度成本构成（不含绩效）：

### 直接人员成本（3500-5800元）

| 项目 | 金额 | 说明 |
|------|------|------|
| 基本工资 | 2500-4000元 | 不低于当地最低工资 |
| 社保公积金（单位） | 工资的30-40% | 法定支出 |
| 福利费 | 300-800元 | 餐补、交通、节日福利 |

### 基础设施（1250-3400元）

| 项目 | 金额 | 说明 |
|------|------|------|
| 场地租赁 | 300-800元 | 按坐席密度分摊 |
| 水电物业 | 100-200元 | 日常运营 |
| 设备折旧 | 150-300元 | 电脑、桌椅、耳麦 |
| 通讯网络 | 500-1500元 | 电话费、专线宽带 |
| IT系统 | 200-600元 | CRM、呼叫中心 |

### 管理支持（900-2100元）

| 项目 | 金额 | 说明 |
|------|------|------|
| 管理人员分摊 | 600-1200元 | 项目经理、主管、质检、培训师 |
| 招聘培训 | 200-500元 | 招聘平台、培训材料 |
| 流失成本 | 100-400元 | 离职补偿、替代招聘、熟练期损失 |

### 利润与税费

| 项目 | 比例 | 说明 |
|------|------|------|
| 增值税及附加 | 6%+ | 法定税费 |
| 风险准备金 | 视情况 | 赔偿、罚款储备 |
| **毛利润** | **10-20%** | 供应商合理利润空间 |

### 成本汇总

```
单坐席月成本 = 人员成本 + 基础设施 + 管理支持 + 利润
            = 4,650-5,800 + 1,250-3,400 + 900-2,100 + 利润
            ≈ 6,000-12,000 元/月（含利润）
```

## 使用指南

| 你的目的 | 用哪个方法 |
|---------|-----------|
| 快速报价、了解市场行情 | 一、同行价格参考 |
| 编制预算、测算成本底线 | 二、社平工资倍数法 |
| 深度分析、与供应商谈判 | 三、供应商成本明细 |

**数据来源**：2024年全国城镇就业人员平均工资（非私营单位月均10,343元、私营单位月均5,789元）

**注意**：实际价格因行业难度（金融/保险通常更高）、项目复杂度、供应商品牌有所浮动。
