# B5-向上汇报准备

> 向军哥或王易人汇报前，系统化组织材料、提炼结论、准备问答

## 触发信号
- 周会汇报：每周一向易人哥做1-2分钟周报
- 月度例会：每月初向军哥/易人哥做月度汇报
- 专项汇报：新项目/重大决策/异常事件需单独汇报
- 临时询问：领导问"XX供应商怎么回事？"（调用【D2-被领导点名】）

## 核心问题
- 汇报对象是谁？（军哥战略层 vs 易人哥执行层）
- 汇报的核心结论是什么？（一句话说清楚）
- 需要哪些数据支撑？（四大战略维度）
- 领导可能问什么？我如何回答？
- 需要领导决策什么？（明确需要审批的事项）

## 思考步骤

**Step 1：识别汇报层级与受众（工具箱§十一）**
```
受众差异矩阵：
┌─────────────┬──────────────┬──────────────┬──────────────┐
│ 汇报对象     │ 关注重点     │ 汇报时长     │ 信息粒度     │
├─────────────┼──────────────┼──────────────┼──────────────┤
│ 军哥（战略层） │ 四大战略维度 │ 5-10分钟     │ 趋势/根因/风险 │
│              │ 合规红线/风险 │ 月度         │ 需决策事项   │
├─────────────┼──────────────┼──────────────┼──────────────┤
│ 易人哥（执行层）│ 执行进度/卡点 │ 1-2分钟      │ 事实/动作/下一步 │
│              │ 需协调事项   │ 周度         │ 快速同步     │
└─────────────┴──────────────┴──────────────┴──────────────┘

汇报类型选择：
周会（易人哥） → 【B4-月度复盘】+ 【17-周度对标月报】
月度例会（军哥） → 【18-月度管理总结】+ 【19-战略维度月报】+ 【20-双指标卡片】
专项汇报（军哥/易人哥） → 【formal-report.md】模板
```

**Step 2：提炼核心结论（工具箱§十一：指标三层翻译）**
```
三层翻译法：
原始数据 → 结构化指标 → 叙事结论

示例：
原始数据：毅航排名第28（后30%），自身峰值比62%，头部差距55%
结构化指标：业绩下滑（峰值比<65%）+ 与头部差距大（<60%）→ 问题型
叙事结论："毅航人效严重未发挥，与头部差距拉大，需立即约谈定位管理问题"

核心结论模板（15字以内）：
"毅航首次预警，新人稀释导致人效下降，需约谈。"
"赛维斯连续预警，整改不力，建议延长观察。"
"头部差距缩小，翰锐可考虑增量。"
```

**Step 3：组织四大战略维度材料（工具箱§十一）**
```
维度1：合规红线
- 质检A类次数：[X]次（涉及供应商[A/B/C]）
- 信息安全：外网管控达标率[XX%]，问题供应商[D/E]
- 数据质量：诚信预警[F]家，诚信红线[G]家
- 持证率：[XX%]（缺证供应商[H]）

维度2：运营效率
- 整体人效：熟手人均GMV [XXX元]（环比↑/↓X%）
- 峰值比分布：>85% [X]家 / 65-85% [X]家 / <65% [X]家
- 头部差距：>80% [X]家 / 60-80% [X]家 / <60% [X]家
- 新人拉低幅度：平均[XX%]，最高[供应商I XX%]

维度3：组织治理
- 主管胜任：5项全达标[X]家 / 3-4项[X]家 / ≤2项[X]家
- 团队稳定：流失率<8% [X]家，≥8% [X]家
- 出勤稳定：>90% [X]家，85-90% [X]家，<85% [X]家
- 整改执行：按期完成率≥70% [X]家，<50% [X]家

维度4：风险控制
- FCI能力指数：≥75（绿）[X]家 / 60-75（黄）[X]家 / <60（红）[X]家
- 巡检结果：红黄绿分布
- 整改闭环率：本月[XX%]（应完成X项/实际Y项）
- 商业健康：G项红色[X]家
```
**叙事模板（工具箱§十一）**：
> "从四大维度看，本月[核心发现]。合规方面[说明]；效率方面[说明]；组织方面[说明]；风险方面[说明]。因此，建议[具体行动]。"

**Step 4：准备问答清单（预判领导问题）**
```
军哥可能问（战略层）：
Q1：D级为什么增加？清退节奏是否足够？
A：连续3个月后30%的从X家增到Y家，原因是[XXX]，已启动清退评估[X家]，延长观察[Y家]。

Q2：头部差距为什么拉大？是不是我们名单问题？
A：头部差距<60%的供应商从X家增到Y家，双维度诊断中[尽力型/问题型]增多，[具体原因]。已要求[具体动作]。

Q3：诚信问题怎么防范？数据真实性能否保证？
A：本月数据质量分≤70的[X]家，已标记3个月加倍核查。已启动[具体防控动作]。

Q4：新供应商引入质量如何？6个月达标率多少？
A：近半年引入[X]家，6个月达标的[Y]家，达标率[ZZ%]。未达标的已[延长/清退]。

Q5：下月重点工作是什么？需要我什么支持？
A：下月P0工作：[1-3项具体工作]。需要您支持：[具体事项，如：清退审批/资源调配]。

───────────────────────────

易人哥可能问（执行层）：
Q1：本周哪些供应商要约谈？原因？
A：首次预警[X家]，连续预警[Y家]，A级骤降[Z家]。已安排约谈时间表。

Q2：整改逾期有多少？怎么跟踪？
A：逾期[X]项，主要供应商[名字]。已升级约谈层级，每周五跟踪。

Q3：新项目对接进度？
A：[项目名]已确定供应商[名字]，T+[X]天上文，当前阶段[寻源/审批/准备]。

Q4：巡检发现什么问题？整改了吗？
A：巡检[X]家，红黄灯[X]家，已发整改通知，截止时间[日期]。
```

**Step 5：明确需要领导决策的事项**
```
需军哥决策：
□ 清退审批：连续3个月后30%的供应商是否启动清退？
□ 重大处罚：诚信红线供应商是否直接清退？
□ 资源调配：新项目是否需要额外预算/资源？
□ 制度变更：规则调整需军哥批准

需易人哥决策：
□ 月度分量：各供应商调整幅度确认
□ 约谈授权：正式约谈/要求换人
□ 清退评估：启动清退评估流程确认
□ 新供应商准入：9维度评估结果审批

决策请求模板：
"需要您确认：[事项]。依据：[数据/规则]。建议：[方案]。如无异议，我将在[时间]执行。"
```

## 关键变量
- **汇报对象**：军哥（战略/风险/决策） vs 易人哥（执行/进度/卡点）
- **汇报类型**：周会（1-2分钟快报） vs 月度（5-10分钟详报） vs 专项（针对性）
- **核心结论**：必须15字以内说清楚（如："毅航首次预警，新人稀释，需约谈"）
- **数据支撑**：四大维度数据必须齐全，不能只讲业绩
- **需决策事项**：必须明确列出，不能模糊说"请领导批示"

## 常见误区
- ❌ 只汇报业绩数据：策略组也能看到，服务组需提供能力数据
- ❌ 指标不翻译：不能只列数字，要写"这说明..."
- ❌ 结论模糊："情况比较复杂" → 需明确判断和结论
- ❌ 不要决策："您看怎么办" → 需给出建议方案让领导选
- ❌ 忽略受众：给易人哥的周报讲战略维度，信息过载
- ❌ 无备答：领导反问时支支吾吾，需预判并准备答案

## 决策输出

**周报模板（给易人哥，1-2分钟）**：
```
【周报】[日期范围]
本周重点：
1. 排名发布：S[X]家/A[X]家/B[X]家/C[X]家/D[X]家（D环比↑/↓X家）
2. 触发预警：首次预警[X家]，连续预警[Y家]，A级骤降[Z家]
3. 巡检完成：[X]家，红黄灯[Y]家，已发整改
4. 新项目：[项目名]确定供应商[名字]，T+[X]天
下周计划：
- 约谈[供应商A/B/C]（X家）
- 整改跟踪[供应商D/E]（X项）
- 巡检[红色FCI供应商F]
需支持：[具体事项，如有]
```
→ 衔接：`weekly-report-supplier.md`

**月报模板（给军哥，5-10分钟）**：
```
【月报】[月份]运营汇报

一、整体格局
S/A/B/C/D分布：S[X]家/A[X]家/B[X]家/C[X]家/D[X]家
层级判断：[S偏少/D偏多/流动健康]
环比变化：D级↑X家（连续预警X家/A级骤降X家）

二、四大维度分析
1. 合规红线：质检A类[X]次（涉及供应商[XXX]）；诚信预警[X]家
2. 运营效率：整体人效[XXX元]，峰值比<65%[X]家（问题型）
3. 组织治理：主管胜任≤2项[X]家，流失率≥8%[X]家
4. 风险控制：FCI红灯[X]家，整改闭环率[XX%]

三、重点事件
- A级骤降：[供应商A]（新人稀释+出勤下滑），已约谈
- 连续预警：[供应商B/C]（连续3个月后30%），启动清退评估
- 新供应商：[D/E]试运行第6月，评估转正/延期

四、下月计划
P0：清退评估[X家] + 约谈[Y家] + 整改跟踪[Z项]
需支持：[清退审批/资源调配/制度变更]
```
→ 衔接：`supplier-monthly-report.md` + `formal-report.md`

**专项汇报模板（formal-report.md）**：
```
标题：[供应商清退专项汇报]
一、背景与问题
二、数据分析（四大维度）
三、根因判断
四、方案对比（2-3个方案）
五、建议方案（含风险评估）
六、需决策事项
七、附录（详细数据）
```
