# Minimize Cognitive Load in Visual Communications

Apply this skill whenever your visual communication includes elements that may impose unnecessary mental effort on the audience without adding sufficient informative value.

## When to Use
- Your audience includes non-experts or has varied technical backgrounds
- The visual contains decorative, redundant, or ambiguous elements
- You aim to improve engagement and reduce misinterpretation
- The subject complexity is not inherently high but appears overwhelming due to poor design

## How to Execute
1. **Audit every visual element**: Ask, “Does this add informative value worth its cognitive cost?” Remove or simplify anything that fails this test.
2. **Eliminate extraneous cognitive load**:
   - Remove decorative borders, excessive colors, redundant labels, and non-essential styling
   - Avoid italics, underlining, and multiple fonts
3. **Ensure accessibility through explicit text**:
   - Include a descriptive, action-oriented title
   - Label all axes clearly
   - Add source footnotes and contextual annotations (e.g., “Peak due to Spring Break”)
   - Use simple language and define acronyms
4. **Use simple text for one or two key numbers** instead of charts—present them prominently with minimal supporting words
5. **Validate**: Could someone understand the main point within 10 seconds without domain expertise?

> **Note**: Never compromise core informative value or essential context (e.g., axis labels needed for accurate interpretation).
