# Stanford CS146S: The Modern Software Developer

> Stanford University • Fall 2025 • Instructor: Mihail Eric

---

## 课程简介

**核心主题**：AI 辅助软件开发 —— 如何用 LLM 10 倍提升生产力

**课程描述**：
> 大语言模型引入了软件开发的新范式。传统开发生命周期正在被 AI 自动化改造。本课程展示现代 AI 工具不仅提升开发者生产力，还让软件工程更大众化。

**新开发模式**：
```
传统：0-1 代码创建
现代：Plan → Generate with AI → Modify → Repeat
```

## 课程大纲（10周）

### Week 1: Introduction to Coding LLMs
- LLM 原理
- Prompt Engineering
- **必看**：Karpathy Deep Dive into LLMs

### Week 2: The Anatomy of Coding Agents ⭐
- Agent 架构
- **MCP 协议**（Model Context Protocol）
- Tool Use & Function Calling

### Week 3: The AI IDE ⭐
- Context 管理
- **Claude Code 最佳实践**
- PRDs for Agents

### Week 4: Coding Agent Patterns ⭐⭐⭐
- **Guest Speaker**: Boris Cherney - **Claude Code 作者**！
- Agent 自主级别管理
- Human-Agent 协作模式

### Week 5: The Modern Terminal ⭐
- **Guest Speaker**: Zach Lloyd - **Warp CEO**
- AI 增强命令行
- Terminal 自动化

### Week 6: AI Testing and Security ⭐
- **Guest Speaker**: Isaac Evans - **Semgrep CEO**
- Secure Vibe Coding
- SAST/DAST with AI

### Week 7: Modern Software Support ⭐
- **Guest Speaker**: Tomas Reimers - **Graphite CPO**
- AI Code Review
- 智能文档生成

- 一键生成 UI/UX
- 快速原型迭代

### Week 9: Agents Post-Deployment ⭐

### Week 10: What's Next ⭐⭐⭐
- **Guest Speaker**: **Martin Casado - a16z General Partner**
- 软件开发的未来
- AI 新范式
