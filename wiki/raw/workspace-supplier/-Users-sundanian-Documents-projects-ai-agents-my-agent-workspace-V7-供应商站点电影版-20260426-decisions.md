# Design Decisions

## 项目背景

**主题**：供应商站点数据汇报 Q1 2026 — 翻页式电影感叙事
**数据**：108 条站点记录，29 家供应商，67 个物理职场，6 条业务线
**Big Idea**：头部 3 家供应商占 31.3% 人力且扩张最快，核心供应商物理集中度过高，单点故障影响面大，需启动分散化评估
**受众**：王易人（服务组负责人）、卞海军（二级部门负责人）
**载体**：单文件 HTML，翻页式叙事（非长滚动），每页一个视觉核心

## Start Questionnaire（已预回答）

- **Entry mode**: Step-by-step（用户已明确方向：翻页式、电影感、数据汇报）
- **Image placeholders**: 否（数据汇报不需要产品图/品牌图，用数据可视化替代）
- **Niche**: 企业数据汇报 / 数据叙事
- **Pages**: 8 页（封面 → 概览 → 风险 → 人力 → 供应商 → 地理 → 业务 → 建议）

## Demo Uniqueness Audit

**Previous-work audit**: 
- 已有版本 1：混合版（看板+汇报混合，有筛选器/Tab/滚动）
- 已有版本 2：杂志报告版（长滚动、白底、McKinsey 风格、66KB）

**Recurring traits to avoid**:
- 顶部固定导航栏
- 滚动式单页面
- IBM/Stripe 风格的数据仪表盘感
- 筛选器/Tab 切换
- 卡片网格堆砌

**Shell-ban list**:
- 禁止固定顶部导航
- 禁止滚动式长页面
- 禁止白色背景+数据表格感
- 禁止卡片均匀网格排列
- 禁止筛选器

**Primary composition family**: **Archive Wall** — 每页像一面档案墙，数据是档案卡片，有主次层级、有留白、有戏剧性的单一大数字/大图表。翻页像翻开一份机密档案。

**Why this differs**: 杂志版是"阅读一份报告"（线性、白底、克制），电影版是"翻开一份机密档案"（深色/氛围感、每页一个视觉冲击、翻页有仪式感）。

## Entry Mode 选择

**Step-by-step** — 用户已明确：翻页式电影感叙事，非长滚动。

## 导演与电影选择

**导演**：David Fincher
**电影**：Zodiac（2007）
**理由**：
- Zodiac 是关于"从海量数据中寻找模式"的故事，和供应商风险分析高度契合
- Fincher 的视觉语言：冷色调、精密构图、信息密集但有层次、克制但有力
- 配色：冷蓝灰基底 + 琥珀色/黄色高亮点，天然适配数据的理性感
- Zodiac 的开场是一整面墙的线索卡片和连线——这就是 Archive Wall 的灵感来源

## Film Palette（从 Fincher/Zodiac 翻译）

| 角色 | 色值 | 用途 |
|------|------|------|
| **主背景** | `#0a0e17` | 深色档案库氛围 |
| **表面层** | `#111827` | 卡片/面板背景 |
| **主文字** | `#e5e7eb` | 正文，近白非纯白 |
| **次要文字** | `#6b7280` | 图注、说明 |
| **琥珀色** | `#d97706` | **高亮/关键发现**（替代红色——更符合 Fincher 冷峻感） |
| **冷蓝** | `#3b82f6` | 次要数据系列 |
| **灰蓝** | `#374151` | 背景数据/参考线 |

## Director Signatures（Fincher）

1. **精密构图**：严格对齐，信息密度高但不乱——每个元素都有位置理由
2. **冷色调基底**：暗蓝灰环境色，营造"深夜档案室"的氛围
3. **克制高亮**：只在关键发现处用琥珀色——不是每页都有红色警告
4. **信息层级**：大数字/大图表作为视觉锚点，小字说明作为支撑

## Film → Web Translation

| 电影元素 | Web 翻译 |
|---------|---------|
| 暗室氛围 | 深色背景 + 微妙渐变 |
| 档案墙 | 数据卡片以矩阵排列，主次分明 |
| 线索连线 | ECharts 图表中的趋势线/流向线 |
| 放大镜下的证据 | 每页一个"放大镜时刻"（大图/大数字） |
| 翻阅档案 | 翻页动画（fade/slide），不是滚动 |
| 冷峻叙事 | 结论式标题，不说废话 |

## Pages 结构

| 页码 | 页面角色 | 场景描述 | 视觉核心 |
|------|---------|---------|---------|
| 1 | **封面** | 暗室灯亮，档案打开 | 标题 + 日期 + 副标题，纯文字，大字重 |
| 2 | **概览** | 第一份档案摊开 | 4 个 KPI 大数字，像档案上的关键数据 |
| 3 | **风险** | 墙上贴满线索 | 10 大风险以线索卡片形式散布，用琥珀色标记 |
| 4 | **人力趋势** | 时间线展开 | ECharts 面积图，大尺寸，冷峻 |
| 5 | **供应商格局** | 档案矩阵 | ECharts 排名图 + 热力矩阵 |
| 6 | **地理集中** | 地图上标记 | ECharts 热力图 + 城市列表 |
| 7 | **业务结构** | 结构图 | ECharts 堆叠图 |
| 8 | **行动建议** | 最终结论 | 3 条建议，像结案报告 |
