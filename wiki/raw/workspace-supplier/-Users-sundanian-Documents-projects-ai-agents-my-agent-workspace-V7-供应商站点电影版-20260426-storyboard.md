# Storyboard

> 供应商站点数据汇报 Q1 2026 — 电影版
> 导演：David Fincher | 电影：Zodiac (2007)
> 构图家族：Archive Wall

---

## Site-wide Cinematic Grammar

**页面外壳**：每页 100vh，垂直居中，翻页式（点击左右箭头或键盘切换），非滚动
**导航**：无顶部导航。底部细进度条（8 个点），当前页琥珀色高亮。右下角翻页箭头
**构图纪律**：每页一个视觉锚点（大数字/大图），其余信息围绕它排列
**密度节奏**：大字报（纯文字）→ 数据矩阵（多卡片）→ 大图表 → 大字报。交替出现
**重复规则**：卡片样式统一，但每页布局不同（避免每页都是卡片网格）

### Restraint Statement

- 不做：动画过渡不超过 2 种类型
- 不做：每页不超过 1 个交互点（翻页是唯一交互）
- 不做：不用红色，用 Fincher 的琥珀色 `#d97706` 作为唯一强调色
- 不做：不加装饰性元素（边框、图标、emoji）

### Material Thesis

- 深色背景（`#0a0e17`）作为档案库
- 卡片背景（`#111827`）作为档案文件
- 1px 细灰线（`#1f2937`）作为分割
- 无阴影、无渐变、无玻璃态——只有纯色块和线条

### Typography Thesis

- 标题：大号、粗字重、近白色
- 数字：超大、等宽字体、琥珀色（关键数字）
- 正文：中等字号、灰色
- 图注：小号、更灰

## Page Scenes

### Page 1: 封面

**Visual Thesis**: 暗室灯亮。纯文字，靠字号层级和留白制造张力。

**Signature Composition**: 左侧大字标题（48px），右侧一个巨大的"29"（供应商数量）用琥珀色显示，占页面 1/3 空间。不对称构图。

**Layout**:
```
┌─────────────────────────────────┐
│                                 │
│   供应商站点数据汇报 Q1 2026     │   29
│   （左侧，大字重）               │   家
│                                 │   供
│   风险信号与行动建议              │   应
│   （灰色小字）                   │   商
│                                 │
│   2026 年 4 月                   │
│                                 │
└─────────────────────────────────┘
```

**Entrance**: 淡入 + 文字从下方微移入（fadeUp）。数字"29"延迟出现，制造停顿感。

## Navigation & Transitions

**翻页交互**：
- 键盘左右箭头翻页
- 底部点击左右箭头
- 底部进度条 8 个点，当前页琥珀色

**页面过渡动画**：
- 页面切换：当前页淡出 → 新页淡入（300ms）
- 页面内元素：依次 fadeUp，间隔 80ms
- 不叠加其他动画效果

## Anti-Convergence Check

**与杂志版的 wireframe 差异**：
| 维度 | 杂志版 | 电影版 |
|------|-------|-------|
| 方向 | 垂直滚动 | 水平翻页 |
| 背景 | 白色 | 深暗色 `#0a0e17` |
| 导航 | 顶部锚点 | 底部进度点 |
| 布局 | 对称网格 | 不对称构图 |
| 强调色 | 红色 `#C13531` | 琥珀色 `#d97706` |
| 封面 | 左对齐纯文字 | 左文右大数字 |
| 图表 | 嵌入式 | 占满 70% 页面 |
