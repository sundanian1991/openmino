# 供应商管理可视化拆解 - infographic-svg 版本

> 基于 infographic-svg 规范的80个可视化单元详细优化
>
> 设计规范：极简信息图 - 暖调学术系
>
> 核心原则：印刷品质感、克制即高级、暖调不冰冷

---

## 设计规范速查（infographic-svg）

### 配色系统

```
结构色（暖调学术系）:
- Canvas: #faf9f7 (米白背景)
- Container Stroke: #1a1a1a (深灰描边 1.5px)
- Card Fill: #f5f4f1 (暖灰卡片)
- Card Stroke: #d4d1c7 (暖灰描边 1px)
- Text Primary: #1a1a1a (深灰文字)
- Text Secondary: #6b6b6b (中灰文字)
- Accent: #c98a6a (暖棕强调色 - 唯一彩色)
- Dashed Stroke: #e8e4dc (虚线描边)

语义色（每图≤2种）:
- 成功/正向: #8f9b7f (橄榄绿)
- 警告/注意: #d4845a (焦橙)
- 用户输入: #6a8fb5 (灰蓝)
- 辅助强调: #a3867a (棕灰)
```

### 字体系统

```
标题: Georgia, "Times New Roman", serif (衬线)
  - H1: 18px, bold
  - H2: 11-12px, 500

正文: system-ui, -apple-system, sans-serif (无衬线)
  - Body: 10-10.5px, 400
  - Caption: 8.5-9.5px, 400
  - Note: 9.5-10px, italic

数值: monospace (等宽)
  - 大号: 11-12px, 600
  - 小号: 10px, 400
```

### 容器系统

```
大容器: rx=10-12, stroke=1.5px #1a1a1a, fill=none
卡片: rx=8, stroke=1px #d4d1c7, fill=#f5f4f1
标签: rx=5-6, stroke=0.5px #d4d1c7
```

### 间距系统

```
4pt基准 (4, 8, 12, 16, 20, 24, 28, 32):
- 卡片间距: 6-8px
- 容器内边距: 10-12px
- 组间距: 20-30px
- 文字行距: 13-14px
```

### 画布系统

```
宽度: 固定680px
高度: 自适应内容
边距: 水平40px, 顶部25-30px, 底部15-20px
```

### 1.1 产能利用率双维度公式

**可视化类型**: 公式说明卡片

**内容结构**:
```
┌──────────────────────────────────────────┐
│ 产能利用率双维度计算                      │
│ ──────────────────────────────────────   │
│                                          │
│ 维度A：自身峰值比 = 当前产能 / 自身历史峰值 │
│                                          │
│ 维度B：头部差距 = (头部峰值 - 当前产能)    │
│              / 头部峰值                   │
│                                          │
│ 评估标准：                                │
│ • 双高：标杆型（维持）                     │
│ • 一高一低：潜力型（分享）/ 问题型（约谈） │
│ • 双低：问题型（立即行动）                 │
└──────────────────────────────────────────┘
```

**配色方案**（infographic-svg暖调系）:
```
卡片背景: #f5f4f1 (暖灰)
卡片描边: 1px, #d4d1c7
文字主色: #1a1a1a (深灰)
文字次色: #6b6b6b (中灰)
强调色: #c98a6a (暖棕，用于维度标识)

语义色（≤2种）:
- 标杆型: #8f9b7f (橄榄绿，20%透明度背景)
- 问题型: #d4845a (焦橙，30%透明度背景)
```

**信息层级**:
```
H1标题: 18px, bold, Georgia, #1a1a1a - "产能利用率双维度计算"
H2维度名: 12px, 500, system-ui, #1a1a1a - "维度A"、"维度B"
Body公式: 10.5px, 400, monospace, #1a1a1a - "当前产能 / 自身历史峰值"
Body说明: 10px, 400, system-ui, #6b6b6b - "双高：标杆型"
Caption标注: 9px, 400, system-ui, #6b6b6b - italic补充说明
```

**视觉引导**:
```
扫描路径: 标题 → 维度A → 维度B → 评估标准
视觉焦点: 维度标识（暖棕色强调）
对比关系:
  - 维度对比: 垂直排列，等宽对齐
  - 公式对比: monospace等宽字体
  - 评估对比: 橄榄绿(标杆) vs 焦橙(问题)
```

**容器自适应**:
```
主卡片容器:
  - 宽度: 100%, max-width: 600px
  - 高度: auto, min-height: 120px
  - 内边距: 12px
  - 圆角: 8px

公式文本:
  - 宽度: 100%
  - 字体: monospace 10.5px
  - 行高: 14px
  - 溢出: ellipsis(超长显示省略号)

评估标准列表:
  - 布局: 垂直列表，左侧8px缩进
  - 符号: • 项目符号，暖棕色 #c98a6a
```

**组件细节**:
```

  - 卡片: 8px
  - 标签(如需要): 6px

边框:
  - 卡片: 1px, #d4d1c7
  - 分割线: 1px, #d4d1c7 (底部)

间距:
  - 维度间距: 12px (垂直)
  - 公式与说明: 8px
  - 评估标准项: 6px

画布:
  - 宽度: 680px (固定)
  - 高度: 自适应 (约160-180px)
  - 边距: 水平40px, 垂直25-30px
```

```
  - 数值对比: monospace等宽
```

**容器自适应**:
```
矩阵容器: width: 100%, max-width: 500px, aspect-ratio: 1/1
象限: 相对定位, 各占50%宽高
气泡: position: absolute
  - 小气泡: width: 48px, height: 48px
  - 中气泡: width: 60px, height: 60px
  - 大气泡: width: 72px, height: 72px
气泡内文字:
  - 供应商名: max-width: 80px, ellipsis溢出
  - 数值: monospace, 居中, 自动换行
```
