## 使用流程

### 会前
1. 读 `00-会前检查清单.md`，逐项检查
2. 打印 `03`、`04`、`05` 材料，人手一份
3. 背熟开场白（在 `00` 里）

### 会中
1. 用 `01-会议议程` 掌握节奏
2. 用 `02-QA 详版` 应对质疑
3. 记录"绝对不能接受"、"愿意妥协"、"愿意共同参与"三点

### 会后
1. 24 小时内发送会议纪要
2. 3 天内更新机制 V1.1
3. 1 周内安排与产线/金分对齐

## 核心姿态

1. **主动暴露短板**："我知道有短板"
2. **邀请共同参与**："你们愿意共同参与哪些部分？"
3. **强调试运行心态**："先建框架，在跑的过程中迭代"
