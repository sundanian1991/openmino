# 日常规范

> 年老师日常工作的具体做法

---

## 一、周报写作规范

**四段式结构**：
```
结果（含数据）→ 根因/问题 → 下一步行动 → 时间线
```

**定量+定性原则**：
用具体数字说话，问题点明确列出

> 示例：翰锐、伽玛325可上线20人，剩余5人因XXX原因延期至4/1

**表格化呈现**：
表格 > 段落，老板扫一眼就能抓重点

**受众思维**：
汇报对象是老板，降低认知成本

---

## 二、月报写作规范

**六段式结构**：
```
整体概况 → 核心指标达成 → 供应商分级分布 → 本月重点事件 → 重点工作进展 → 下月工作计划
```

**量化价值**：
用数据证明工作价值和改进成果

> 示例：月度提效空间~83.5小时，提效比例43%，战略性工作占比22%→38%

---

## 三、会议纪要规范

**五段式结构**：
```
核心思路 → 关键设计 → 特别说明（引用规矩）→ 待讨论 → 时间节点
```

**行动导向**：
会议结论转化为具体行动，明确执行人和时间

**群沟通原则**：
发群里让大家看调整项，核心先行、问题明确

---

## 四、供应商日常沟通

**现场复盘要求**：
要求信用卡和企金供应商现场复盘，同步策略侧可按需参与

**调研确认流程**：
供应商调研数据 → 年老师亲自沟通后确认

**确认内容**：
- 日GMV
- 激励目标
- 核心原因
- 影响判断

---

## 五、公文写作规范

**三段式结构**：
```
事实部分（列表/表格）→ 分析判断 → 行动请求
```

**语气适配**：
同一内容根据对象调整语气

| 对象 | 语气 |
|------|------|
| 向上级 | 正式、简洁 |
| 向下级 | 柔和、指导 |
| 平级 | 协商、协作 |
| 供应商 | 告知、质询 |

**关键词标识**：
每个段落开始用**关键词：**标识，支持快速浏览
