# 供应商管理方法论

> 年老师管理供应商的核心理念和方法

---

## 一、培养悖论定律

**核心洞察**：
培养供应商的必然路径：小 → 给资源 → 做大 → 向外扩张 → 有退路 → 议价/流失

**这不是背叛，是模式的必然结果**

**应对策略**：
| 策略 | 做法 |
|------|------|
| 梯队备份 | 始终培育第二梯队，保持替代选项 |
| BPO联盟 | 把自营培训体系交给中腰部供应商 |
| 新业务优先 | 新业务分配优先给中腰部，不让头部一家独大 |
| 差异化定价 | 头部用市场价，中腰部给成长空间 |

---

## 二、诊断驱动决策

**核心理念**：
用结构化问题替代直觉判断，逼迫性提问获取具体可验证信息

**准入诊断6问**：
| 问题 | 目的 |
|------|------|
| 为什么必须是这家？ | 验证必要性 |
| 没有这家会怎样？ | 验证依赖度 |
| 谁最需要说名字？ | 验证需求来源 |
| 最小试单是什么？ | 验证试错成本 |
| 有成功案例吗？ | 验证可行性 |
| 3年后还需要吗？ | 验证长期价值 |

**业绩诊断6问**：
| 问题 | 目的 |
|------|------|
| 核心指标到底怎样？ | 采集具体信号 |
| 根本原因是什么？ | 追根溯源 |
| 谁受影响最大？ | 识别利益相关方 |
| 能改进吗要什么？ | 评估干预可行性 |
| 去现场看过吗？ | 验证信息来源 |
| 3个月后会怎样？ | 时序思考 |

---

## 三、分层干预原则

**风险信号分级**：
| 信号 | 风险等级 | 干预力度 |
|------|----------|----------|
| 连续2月不达标 | 高风险 | 重点干预 |
| 核心人员离职 | 高风险 | 重点干预 |
| 单月不达标 | 中风险 | 关注跟进 |
| 正常运营 | 低风险 | 日常维护 |

**供应商分类**：
| 类型 | 管理重点 |
|------|----------|
| 核心供应商 | 稳定性维护、深度绑定 |
| 战略供应商 | 长期合作、资源倾斜 |
| 普通供应商 | 标准化管理、按规则执行 |
| 问题供应商 | 风险预警、清退准备 |

---

## 四、沟通框架适配

**借力框架沟通**：
不要直接要权/要资源，用对方KPI框架沟通

> 示例：不说"我需要定价权"，说"共同优化费效比需要决策前置"

**因果链思维**：
追根溯源，理解传导路径

> 示例：结算太低 → 供应商不投入 → 产出下降 → 费效比恶化 → 陷入恶性循环

---

## 五、日常沟通方式

**分类管理**：
不同业务线采用不同沟通深度，策略侧按需参与避免资源浪费

**数据验证原则**：
宏观数据与微观调研结合，避免片面判断

> 调研数据 → 年老师亲自沟通后确认：日GMV、激励目标、核心原因、影响判断
