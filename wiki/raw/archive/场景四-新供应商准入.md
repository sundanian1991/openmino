## 一、准入流程

### 线上初筛

| 硬性条件 | 业务条件 |
|---------|---------|
| 增值电信许可证 | 是否做过同类业务 |
| 注册资金≥1000万 | 人力规模和排名 |
| 站点分布 | 人均结算水平 |
| 人均结算 | 能力项、靠什么达成业绩 |

### 流程环节

```
线上沟通 → 能力评估 → 现场考察 → 试单评估 → 合同签订 → 入场
```

### 流程缩短

| 维度 | 变化 |
|------|------|
| 原周期 | 15天 |
| 现周期 | 7-10天 |
| 压缩环节 | 验厂环节可能不去现场 |
| 操作方式 | 优先级调高，准备工作前置 |

## 二、现场考察判断

### 机房布置（举例）

**高标准特征**：
- 恒温恒湿
- 独立监控
- 消防灭火

**判断逻辑**：
> "机房能做到这个程度，那其他标准就不言而喻，应该是比较高的。"

### 运营管理成体系

**观察重点**：
- 有细致的问题追踪、判断、定位
- 能快速做调整

### 会议质量判断

| 维度 | 好的会议 | 差的会议 |
|------|---------|---------|
| 氛围 | 员工积极参与 | 从上至下强压 |
| 内容 | 准确定位问题 | 只要求"必须怎样" |
| 效果 | 给员工启发 | 情绪凶、不能辅导 |
| 本质 | 辅导型管理 | 强压型管理 |

**观察方法**：看他们如何给员工开会、如何定位员工问题

## 三、试单评估

### 评估标准

| 标准类型 | 内容 |
|---------|------|
| 基本标准 | 综合赛马成绩不为最后一名 |
| 一票否决 | 合规问题 |

### 宽进严出策略

**踩坑背景**：
- 线上数标排第一或第二的供应商
- 实际合作出现"水土不服"
- 经验丰富但在京东业务上仍未做好

**解决方案**：
- 测试完成后更多依据实际效果
- 例如一次性进三家，三家一起PK
- 用实际表现筛选，不唯数标排名

## 四、合同与结算

### 合同谈判

> "合同都是以我们制定的为准，因为是所有供应商都签的。基本上就是同步，供应商基本上都会接受。"

**特点**：
- 京东制定统一标准
- 供应商基本接受
- 标准化程度高

### 结算周期

| 节点 | 时间 |
|------|------|
| 对账 | T+1月 |
| 结算 | T+2月 |

## 五、风险与应对

### 流程缩短带来的风险

| 风险 | 说明 |
|------|------|
| 验厂缺失 | 没去过现场，不知道公司真实情况 |
| 分包转包 | 可能存在分包或转包风险 |
| 挑战点 | 可能被人挑战 |

### 踩坑经验

**案例**：线上数标排第一第二的供应商，实际合作水土服

**原因**：供应商经验丰富，但在京东业务上仍未做好

**教训**：数标排名不完全代表实际适配能力

### 应对措施

| 措施 | 内容 |
|------|------|
| 分包转包 | 发现后直接清退 |
| 预期差异 | 采用宽进严出策略 |
| 筛选方式 | 多家同时试单PK，依据实际效果 |
