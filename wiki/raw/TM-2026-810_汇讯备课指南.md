## 一、先理解本次培训的定位

本次培训不是"经验汇报"，而是 **"从1到10的精进拆解"**。

- 受众：11家供应商的项目经理和直接负责人（约20人），他们已经做过大额税票贷，有经验但深度不够
- 目标：通过汇讯的真实打法和数据，让他们看清"头部到底强在哪"
- 核心要求：**偏实操、话术、客户经营理解**，少讲管理叙事，多给可复制的细节

**你的任务是"打开黑盒"**——把汇讯怎么做到人均结算9000-10000的，拆解到别人能听懂、能模仿的程度。

## 三、模块一：大额税票贷经营体系全景图（60分钟）

### 核心议题
大额税票贷和其他产线（金条、信用卡）到底有什么不同？汇讯是怎么围绕这些差异搭建经营体系的？

### 分享内容清单（请按此结构准备PPT）

**1.1 大额业务特殊性（15分钟）**
- [ ] 税票贷产品画像：额度区间、利率区间、审批条件、目标客群
- [ ] 与个贷产线的本质差异（对照表）：
  - 客户群体（企业主 vs 个人）
  - 决策链（长 vs 短）
  - 转化率（低 vs 相对较高）
  - 客单价/件均（高 vs 低）
  - 成交周期（长 vs 短）
  - 客户顾虑点（企业征信/合规 vs 个人征信）
  - 信任建立（平台专业性 vs 品牌认知）
- [ ] 数据对比：**汇讯 vs 联盟尾部** 的人均通时、转化率、件均、参评率、外耗天数
  - 如果方便，给出具体数字；如果不方便，给比例/倍数关系也可以

**1.2 经营体系五要素（20分钟）**
- [ ] 招商画像 → 培训体系 → 过程管理 → 资源策略 → 激励薪酬
- [ ] 每个要素汇讯目前的做法（简要，详细内容放到模块四讲）
- [ ] 重点讲：**哪个要素是大额业务最关键的？为什么？**

**1.3 用户经营逻辑（20分钟）**
- [ ] 企业主客群特征：行业分布、经营年限、资金需求场景
- [ ] 企业主决策心理：他们最在意什么？（不是利息低，而是...）
- [ ] 信任建立路径：汇讯怎么在电话中让企业主觉得"这个平台靠谱"？
- [ ] 汇讯TOP坐席对"什么样的客户容易转化"的经验总结

**1.4 关键差异点预告（5分钟）**
- [ ] 预告下午案例工作坊要拆解的场景
- [ ] 发布小组任务：带着"最困惑的3个问题"进入下午讨论

### 需准备的素材
- [ ] PPT（不需要花哨，白底黑字+关键数据即可）
- [ ] 大额 vs 个贷差异对照表（可打印发给每人一份）
- [ ] 2-3个"差点做错但及时调整"的真实故事
- [ ] 汇讯核心经营数据（脱敏后可用）

### 备场建议
- PPT页数建议 **15-20页**，每页只讲一个点
- 多用数据，少用形容词（不要说"我们做得很好"，要说"转化率X%，高于均值Y%"）
- 准备好2-3个故事：开场用一个、中间用一个、结尾用一个

## 四、模块二：客户识别、分层与转化策略（60分钟）★ 核心模块

### 核心议题
汇讯是怎么识别高潜力客户的？不同客户怎么打？话术具体怎么说？

### 分享内容清单（请按此结构准备）

**2.1 客户分层模型（15分钟）**
- [ ] 汇讯按什么维度分层：企业规模 / 纳税等级 / 开票金额 / 行业属性 / 经营年限
- [ ] 各层级的转化率数据（如果方便）
- [ ] "伪高潜"识别：哪些标签看起来好、实际上转化率低？
- [ ] 互动设计：准备5个虚拟企业主画像，让现场小组排序

**2.2 高转化客户画像（15分钟）**
- [ ] 行业偏好：哪些行业需求更刚、决策更快？
- [ ] 纳税/开票特征：什么区间的客户质量最好？
- [ ] 资金需求紧迫度信号：哪些行为暗示急用钱？
- [ ] 复贷/存量企业主 vs 新客户的转化策略差异
- [ ] 汇讯TOP10坐席的客户选择策略

**2.3 转化路径设计（15分钟）**
- [ ] 完整转化漏斗：首触 → 需求挖掘 → 信任建立 → 资料收集 → 跟进节奏 → 促成签约
- [ ] 每个环节的关键动作和耗时参考
- [ ] 大额业务特有的"长周期跟进"管理：客户池子分类、跟进频率、放弃标准

**2.4 话术拆解（10分钟）**
- [ ] 开场白：怎么让企业主10秒内愿意继续听？
- [ ] 需求挖掘：怎么问出真实资金需求？
- [ ] 异议处理TOP5（必须有对应话术）：
  - "利息太高" → 实际成本换算 + 渠道对比
  - "要考虑一下" → 真假顾虑识别 + 决策紧迫感
  - "要和家人/股东商量" → 铺垫话术 + 约下次通话
  - "影响企业征信怎么办" → 合规性背书
  - "已经有其他贷款了" → 税票贷差异化价值
- [ ] 促单收尾：阶梯式促成（先试小额、再提额）

**2.5 带行人培养（5分钟）**
- [ ] 怎么识别有潜力的"带行人"
- [ ] 选拔标准：业绩？沟通能力？学习意愿？
- [ ] 培养和激励方式

### 需准备的素材
- [ ] **2-3通真实成单录音**（脱敏处理，去除客户姓名、企业名、具体金额）
  - 录音要求：标注"这是第几通电话""关键转折在哪一句"
  - 每通录音选 **30-60秒关键片段** 播放，不要放完整通话
- [ ] 话术摘要卡片（可打印发给每人一份，作为培训手册附件）
- [ ] 客户分层数据表（脱敏）
- [ ] 如果可能，准备1个"同一客户、两个坐席、两种结果"的对比案例

### 关键要求
- **所有话术必须来自真实成单录音**，拒绝"教科书式"空话
- 每个策略必须匹配具体场景："什么时候用、对谁用、怎么用"
- 不要讲"你们要这样做"，要讲"我们是这样做的，数据是..."

## 五、模块四：汇讯实战拆解（50分钟）

### 核心议题
汇讯是怎么组织的？怎么管的？业绩好背后的管理模式是什么？

### 分享内容清单（请按此结构准备PPT）

**4.1 业绩拆解（10分钟）**
- [ ] 人均结算9000-10000的指标构成：通时、转化率、件均、参评率、外耗天数
- [ ] 与尾部供应商（5000+）的差距分析：主要差在哪几个指标？
- [ ] 注意：京东侧会补充点评合规成本对净利润的影响

**4.2 组织架构与人员结构（10分钟）**
- [ ] 组长管几个人？组长和坐席的比例？
- [ ] 新老坐席比例、流动率控制
- [ ] 晋升通道设计：坐席 → 组长 → ？

**4.3 管理模式（15分钟）**
- [ ] 一天的管理时间线：早会/午间检查/晚会复盘，每个环节做什么
- [ ] 现场巡场：巡什么、怎么发现问题、怎么干预
- [ ] 日报/周报怎么写：哪些数据必看、异常怎么标注、对策怎么写
- [ ] 周复盘/月复盘的固定议程

**4.4 激励与留存（10分钟）**
- [ ] 基本法结构：底薪/提成/奖金占比
- [ ] 提成阶梯设计：按什么指标设？有没有加速机制？
- [ ] 绩优与绩差的收入差距
- [ ] 老员工留存策略：工龄奖？晋升通道？专属权益？
- [ ] 淘汰机制：连续不达标怎么处理？几次谈话、多久改进期？

**4.5 特殊资源条件（如有）（5分钟）**
- [ ] 是否有特殊的招聘渠道、培训资源、技术支持
- [ ] 坦诚说：哪些是可以复制的，哪些受限于特殊条件

**4.6 合规教训（5分钟）★ 必须讲**
- [ ] 高举高达时期的合规问题：具体发生了什么
- [ ] 怎么整改的：哪些做法调整了
- [ ] 现在的平衡策略是什么
- [ ] 对大家的警示：业绩和合规必须并行
- [ ] 注意：京东侧会补充点评，口径提前对齐

### 需准备的素材
- [ ] 组织架构图（脱敏）
- [ ] 日报模板（脱敏后可分享）
- [ ] 管理流程文档（脱敏）
- [ ] 基本法摘要（脱敏后可分享）
- [ ] 合规整改案例（简要，不要过度展开）
