## 一、供应商管理方法论

### 培养悖论定律

**核心洞察**：
培养供应商的必然路径：小 → 给资源 → 做大 → 向外扩张 → 有退路 → 议价/流失

**这不是背叛，是模式的必然结果**

**应对策略**：
| 策略 | 做法 |
|------|------|
| 梯队备份 | 始终培育第二梯队，保持替代选项 |
| BPO联盟 | 把自营培训体系交给中腰部供应商 |
| 新业务优先 | 新业务分配优先给中腰部 |
| 差异化定价 | 头部用市场价，中腰部给成长空间 |

---

### 诊断驱动决策

**核心理念**：
用结构化问题替代直觉判断，逼迫性提问获取具体可验证信息

**准入诊断6问**：
| 问题 | 目的 |
|------|------|
| 为什么必须是这家？ | 验证必要性 |
| 没有这家会怎样？ | 验证依赖度 |
| 谁最需要说名字？ | 验证需求来源 |
| 最小试单是什么？ | 验证试错成本 |
| 有成功案例吗？ | 验证可行性 |
| 3年后还需要吗？ | 验证长期价值 |

**业绩诊断6问**：
| 问题 | 目的 |
|------|------|
| 核心指标到底怎样？ | 采集具体信号 |
| 根本原因是什么？ | 追根溯源 |
| 谁受影响最大？ | 识别利益相关方 |
| 能改进吗要什么？ | 评估干预可行性 |
| 去现场看过吗？ | 验证信息来源 |
| 3个月后会怎样？ | 时序思考 |

---

### 分层干预原则

**风险信号分级**：
| 信号 | 风险等级 | 干预力度 |
|------|----------|----------|
| 连续2月不达标 | 高风险 | 重点干预 |
| 核心人员离职 | 高风险 | 重点干预 |
| 单月不达标 | 中风险 | 关注跟进 |
| 正常运营 | 低风险 | 日常维护 |

## 二、沟通方法论

### 四个表达层级

| 层级 | 定义 | 示例 | 问题 |
|------|------|------|------|
| Level 1 | 纯感受表达 | "我觉得不太行" | 高度主观，信息密度极低 |
| Level 2 | 结构化定性表达 | "问题出在用户路径过长" | 有分析结构，但不可验证 |
| Level 3 | 定量表达 | "转化率下降了12%" | 可验证，但缺乏解释力 |
| Level 4 | 定性+定量融合表达 | "转化率下降12%，因路径从5步增至8步" | 有数据、有解释、有因果结构 |

**目标**：用 Level 4 表达

**场景适配**：
| 场景 | 推荐层级 |
|------|----------|
| 向上汇报 | Level 4 |
| 跨部门协作 | Level 4 |
| 供应商谈判 | Level 4 |
| 日常闲聊 | Level 1-2 |

---

### 向上汇报规范

**四段式结构**：
```
结果（含数据）→ 根因/问题 → 下一步行动 → 时间线
```

**适配军哥穿透式管理风格**：
- 数据在前、问题在中、行动在后
- 结论先行，避免信息罗列
- 有数据先说数据，没数据说进度

---

### 借力框架沟通

不要直接要权/要资源，用对方KPI框架沟通

> 示例：不说"我需要定价权"，说"共同优化费效比需要决策前置"

## 三、问题处理方法论

### 问题诊断三层次

**诊断框架**：
```
现象 → 结构 → 根源
```

找到系统性原因，而非表面问题

**示例**：
新人流失率高的诊断路径：

| 层次 | 分析 |
|------|------|
| 现象 | 新人流失率高 |
| 结构 | 结算低 → 培训投入不足 → 新人体验差 |
| 根源 | 名单结构不合理，不是人力能力问题 |

**避免归因偏差**：区分能力问题 vs 结构问题

---

### 失败复盘机制

**停止反思铁律**：
修补失败2次 → 立刻停下来反思方向

**架构问题不能靠修补解决，必须重构**

**复盘三步**：
```
记录教训 → 提炼铁律 → 形成方法论
```

---

### 风险预防制度化

**提前识别风险场景**：
| 场景 | 风险点 |
|------|------|
| 供应商引入 | 准入标准不明确 |
| 定价调整 | 定价权不在自己 |
| 问题追溯 | 事后规则追溯挖坑 |

**准备标准话术**：
边界清单 + 留痕模板 + 决策速查表

---

### 时序思考原则

**不只看当下，考虑未来趋势**

| 问题 | 时序思考 |
|------|----------|
| 准入评估 | 3年后还需要吗？ |
| 业绩诊断 | 3个月后会怎样？ |
| 问题处理 | 会复发吗？如何预防？ |
