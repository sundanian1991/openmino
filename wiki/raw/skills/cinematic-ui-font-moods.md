## Usage Notes

### Basic implementation
```html
<link href="https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Source+Sans+3:wght@400;600&display=swap" rel="stylesheet">
```
```css
h1, h2, h3 { font-family: 'Bebas Neue', sans-serif; letter-spacing: 0.08em; line-height: 1.1; }
p, li, td { font-family: 'Source Sans 3', sans-serif; letter-spacing: 0.01em; line-height: 1.65; }
```

### Responsive scaling
Adjust letter-spacing at breakpoints. Tighter tracking works better at large sizes, looser at small:
```css
h1 { letter-spacing: 0.08em; }
@media (max-width: 768px) { h1 { letter-spacing: 0.04em; } }
```

### Variable fonts for performance
Where available, use variable font axis to reduce HTTP requests:
```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400..700&display=swap');
```

### Combining with color grades
Match font mood to color grade mood for cohesive design. Example: Pairing #47 (Thriller cold precision) with color grade #11 (Hard Boiled Noir) for a true crime site.
