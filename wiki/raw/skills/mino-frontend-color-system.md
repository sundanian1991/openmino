# 颜色系统规范

> mino-frontend 统一颜色系统 v5.0 Phase 1

---

## 核心原则

1. **统一灰阶** — 所有风格共用6级灰阶
2. **情绪驱动** — Header颜色由内容情绪决定，不是固定
3. **克制用色** — 强调色≤10%，灰阶承担主要信息层级

---

## 统一灰阶系统（6级）

```css
:root {
  /* 核心灰阶（所有风格共用）*/
  --gray-50: #FAFAFA;   /* 斑马纹行、卡片背景 */
  --gray-100: #F5F5F5;  /* hover状态、次要背景 */
  --gray-200: #E5E5E5;  /* 边框、分割线、次要分隔 */
  --gray-400: #999999;  /* 禁用文字、图例、注释 */
  --gray-600: #666666;  /* 次要文字、正文 */
  --gray-900: #111111;  /* 主文字、表头、Header */
}
```

### 灰阶使用规范

| 场景 | 颜色 | 说明 |
|------|------|------|
| 表头背景 | `#111` | 炭黑，最大视觉重量 |
| Header背景 | 见「情绪颜色」| 由内容情绪决定 |
| 底部信息条 | 同Header | 保持一致 |
| 主标题 | `#111` | 炭黑 |
| 正文 | `#666` | 中灰，60%黑 |
| 次要文字 | `#999` | 浅灰，40%黑 |
| 边框 | `#E5E5E5` | 10%黑，存在感低 |
| 分割线 | `#E5E5E5` | 同边框 |
| 斑马纹行 | `#FAFAFA` | 3%黑，微妙区分 |
| hover状态 | `#F5F5F5` | 4%黑，可见但不突兀 |
| 卡片背景 | `#FAFAFA` | 同斑马纹 |

## 情绪颜色系统（5种）

Header颜色由内容情绪决定：

| 情绪 | 关键词 | Header颜色 | 使用场景 |
|------|--------|-----------|---------|
| **紧迫** | 禁止、危险、风险、异常、警告、处罚 | `#8B0000`（深红） | 处罚通报、风险预警、合规要求 |
| **中性** | 数据、说明、计划、常规、通报 | `#111`（炭黑） | 数据通报、常规汇报、排名 |
| **庆祝** | 成功、达成、激励、进步、搞钱 | `#E2725B`（陶土） | 业绩激励、竞赛海报、成果展示 |
| **权威** | 战略、金融、年度、重要、正式 | `#0d7680`（青绿） | 金融数据、战略汇报、年度总结 |
| **奢华** | 颁奖、盛典、高端、里程碑 | `#D4AF37`（金） | 颁奖盛典、年度庆典、高端发布 |

### 情绪识别方法

**Step 1**: 扫描内容关键词
**Step 2**: 统计情绪关键词出现频次
**Step 3**: 以最高频次情绪为准（并列时选「紧迫」>「庆祝」>「权威」>「奢华」>「中性」）

**示例**：
- "供应商处罚通报" → 关键词「处罚」→ 紧迫 → `#8B0000`
- "4月业绩冲刺激励" → 关键词「冲刺」「激励」→ 庆祝 → `#E2725B`
- "金条产能排名" → 无情绪关键词 → 中性 → `#111`

## 强调色使用规范

| 情绪 | 强调色 | 用量限制 |
|------|--------|---------|
| 紧迫 | `#8B0000`（深红） | ≤10% |
| 中性 | `#E2725B`（陶土） | ≤10% |
| 庆祝 | `#D4AF37`（金）或`#E2725B`（陶土） | ≤10% |
| 权威 | `#0d7680`（青绿） | ≤10% |
| 奢华 | `#D4AF37`（金） | ≤15%（可稍多） |

### 强调色使用位置

1. **顶部装饰线** — 2.4px横线（所有情绪）
2. **排名1色块** — 第1名背景（数据表格）
3. **增长徽章** — 正增长标识
4. **关键数据** — 唯一需要突出的数字

**禁止**：
- 大面积强调色背景（>20%）
- 多处同时使用强调色
- 正文文字用强调色

## 通报模式 Header 颜色决策

```
内容 → 提取情绪关键词 → 匹配情绪 → 确定Header颜色 → 选择强调色
```

### 情绪关键词库

| 情绪 | 关键词 |
|------|--------|
| 紧迫 | 禁止、处罚、警告、风险、异常、紧急、立即、必须、严肃、负面、下滑、流失、危机 |
| 中性 | 数据、排名、统计、常规、计划、说明、流程、分析、通报、报告 |
| 庆祝 | 成功、达成、激励、冲刺、竞赛、奖励、进步、增长、突破、搞钱、赚钱、完成 |
| 权威 | 战略、年度、季度、金融、正式、重要、决策、方向、全局、宏观 |
| 奢华 | 颁奖、盛典、庆典、荣誉、冠军、顶级、里程碑、巅峰、王者 |
