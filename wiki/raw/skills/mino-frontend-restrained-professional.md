## 📖 设计哲学

### 运动名称
**Restrained Professionalism** — 克制专业主义

### 设计哲学宣言

**空间与形式**：极简到极致的黑白灰世界，每个像素都为数据服务。无阴影、无渐变、无装饰 — 只有纯粹的网格和边框。顶部 2.4px 陶土色装饰线作为唯一的"大声"元素，右下角圆形页码徽章作为导航锚点。信息密度可以高，但视觉噪音必须为零。

**色彩与情绪**：陶土色 (#E2725B) 是唯一的例外，唯一的温度，唯一的焦点。一个页面的陶土色使用面积控制在 10% 以内 — 顶部装饰线、页码徽章、关键数据强调。这是冷峻理性世界中的唯一温暖，是经过计算的视觉引导。黑白灰不是"无色"，而是让数据说话的沉默背景。

**工艺标准**：每个数字都对齐到像素，每个边框的粗细都经过计算。卡片没有阴影，依靠 1px 边框和充足内边距创造层次。表格 hover 效果只用 `#fafafa`，最微妙的反馈。这是花了无数小时调试间距和对齐的作品，每个像素都在正确位置。

## 🎨 色彩系统

### 六色层级

| 用途 | 色值 | 使用比例 | 说明 |
|------|------|---------|------|
| **主文字** | `#111111` | 50% | 标题、重要数据、表头 |
| **辅助文字** | `#666666` | 25% | 正文、列表项、卡片内容 |
| **第三级文字** | `#999999` | 10% | 图例、注释、次要信息 |
| **背景** | `#FFFFFF` | 100% | 纯白背景（所有页面） |
| **边框/分割线** | `#E5E5E5` | 按需 | 1px 细边框、表格分割线 |
| **强调色** | `#E2725B` | ≤10% | 顶部装饰线、页码徽章、关键数据 |

### 强调色使用规则

**陶土色 #E2725B 的三种用法**：

| 场景 | 用法 | 面积 | 示例 |
|------|------|------|------|
| **顶部装饰线** | 2.4px 横线，贯穿页面顶部 | ~2% | 所有页面顶部 |
| **页码徽章** | 圆形填充，右下角 | ~1% | 所有非封面页面 |
| **关键数据强调** | 文字颜色或背景填充 | ≤5% | KPI 数字、异常值、行动建议 |
| **卡片强调** | 背景填充（浅粉 `#FFF5F2`） | ≤10% | 需要突出的卡片 |

**禁止**：
- ❌ 大面积陶土色背景（超过页面 20%）
- ❌ 陶土色用于正文文字
- ❌ 同时使用多处陶土色强调（一页一处原则）

| 用途 | 字体 | 说明 |
|------|------|------|

### 字体层级（T1-T6）

| 层级 | 用途 | 字号 | 字重 | 行高 | 颜色 |
|------|------|------|------|------|------|
| **T1** | 封面大标题 | 40px / `clamp(2rem, 5vw, 2.5rem)` | 700 | 1.2 | `#111111` |
| **T2** | 页面标题 | 28px / `clamp(1.5rem, 3vw, 1.75rem)` | 700 | 1.3 | `#111111` |
| **T3** | 卡片/模块标题 | 14-16px | 700 | 1.4 | `#111111` |
| **T4** | 正文/列表项 | 12-13px | 400 | 1.6 | `#333333` |
| **T5** | 辅助说明/图例 | 10-11px | 400 | 1.5 | `#666666` |
| **T6** | 页码/面包屑 | 10-12px | 700 | 1.0 | `#FFFFFF`（徽章内） |


## 🧠 认知负荷管理

### Miller's Law 应用

| 页面类型 | 最大视觉块数量 | 说明 |
|---------|--------------|------|
| 封面页 | 3 | 标题 + 副标题 + 日期 |
| 内容页 | 5-7 | 标题 + 4-6 个列表项 |
| 对比页 | 6 | 2 卡片 × 3 要点 |
| 表格页 | 7±2 | 表头 + 5-7 行数据 + 底部总结 |
| 卡片页 | 4-6 | 4-6 个卡片 |

### 渐进式披露原则

1. **第一层**：页面标题（T2）— 告诉用户这是什么
2. **第二层**：视觉块标题（T3）— 快速扫描获取结构
3. **第三层**：正文内容（T4）— 详细阅读
4. **第四层**：辅助说明（T5/T6）— 按需查看

### 视觉分组规则

```
✅ 正确：
├── 相关元素间距：16px
├── 组间间距：32px
├── 使用背景色区分组
└── 使用边框/分割线区分组

❌ 错误：
├── 所有元素等间距
├── 没有视觉层次
└── 过多颜色/装饰干扰
```

## 🎭 情绪曲线

### 8 页演示文稿的情绪设计

| 页码 | 类型 | 情绪 | 视觉重量 | 陶土色使用 |
|------|------|------|---------|-----------|
| 1 | 封面 | 严肃、专业 | 高（大标题） | 装饰线 + 短线 |
| 2 | 对比 | 冲突、张力 | 中（左右分割） | 右侧卡片强调 |
| 3 | 拆解 | 分析、冷静 | 中（列表） | 小标题强调 |
| 4 | 动机 | 洞察、判断 | 中高（表格） | 可能性徽章 |
| 5 | 影响 | 共情、关切 | 中（卡片网格） | 关键卡片强调 |
| 6 | 策略 | 行动、方案 | 中（场景话术） | 行动建议 |

```

冲突/张力 → 对比色（左侧灰/右侧粉）
洞察/判断 → 陶土色徽章（极高/高/中）
预警/风险 → 陶土色强调（唯一焦点）
```

## 🚫 克制原则清单

### 视觉克制

| 做什么 | 不做什么 |
|--------|----------|
| ✅ 纯白背景 | ❌ 渐变背景、纹理背景 |
| ✅ 1px 细边框 | ❌ 阴影、多层边框 |
| ✅ 单一强调色 | ❌ 多色混用 |
| ✅ 几何图形（矩形、圆形） | ❌ 手绘 SVG、装饰插画 |
| ✅ 微妙的 hover（`#fafafa`） | ❌ 动画、过渡效果 |

### 信息克制

| 做什么 | 不做什么 |
|--------|----------|
| ✅ 每页一个核心观点 | ❌ 多观点堆砌 |
| ✅ 4-6 个视觉块 | ❌ 超过 7 个视觉块 |
| ✅ 列表项一行内 | ❌ 多行长段落 |
| ✅ 表格 5-7 行 | ❌ 超过 10 行表格 |
| ✅ 底部总结卡片 | ❌ 无总结的信息罗列 |

### 交互克制

| 做什么 | 不做什么 |
|--------|----------|
| ✅ 静态展示 | ❌ 点击展开、Tab 切换 |
| ✅ 滚动浏览（长内容） | ❌ 弹窗、模态框 |
| ✅ 简单 hover | ❌ 复杂动画 |

### PPTX (PptxGenJS) 模板

```javascript
const pptxgen = require("pptxgenjs");
const pres = new pptxgen();
pres.layout = 'LAYOUT_16x9';

const theme = {
  primary: "111111",
  secondary: "666666",
  tertiary: "999999",
  bg: "FFFFFF",
  border: "E5E5E5",
  accent: "E2725B",
  accentLight: "FFF5F2"
};

function createSlide(pres, theme) {
  const slide = pres.addSlide();
  slide.background = { color: theme.bg };

  // 顶部装饰线
  slide.addShape(pres.shapes.RECTANGLE, {
    x: 0, y: 0, w: 10, h: 0.05,
    fill: { color: theme.accent }
  });

  // 页码（非封面）
  slide.addShape(pres.shapes.OVAL, {
    x: 9.3, y: 5.1, w: 0.4, h: 0.4,
    fill: { color: theme.accent }
  });
  slide.addText("4", {
    x: 9.3, y: 5.1, w: 0.4, h: 0.4,
    fontSize: 12, fontFace: "Arial",
    color: "FFFFFF", bold: true,
    align: "center", valign: "middle"
  });

  return slide;
}
```

## 🔗 与其他风格的关系

### 对比：数据模式 (原版) vs 克制专业主义

| 维度 | 数据模式 (原版) | 克制专业主义 (本风格) |
|------|--------------|---------------------|
| **顶部装饰** | 无 | 2.4px 陶土色线 |
| **页码系统** | 无明确规定 | 右下角圆形徽章 |
| **强调色面积** | "一页一处" | ≤10%（装饰线 + 徽章 + 数据） |
| **版式模板** | 表格为主 | 5 种核心版式 |
| **字体** | 未明确 | Microsoft YaHei + Arial |
| **卡片背景** | 纯白 | 浅灰 `#FAFAFA` / 浅粉 `#FFF5F2` |

### 融合来源

```
克制专业主义 = 数据模式 (骨架) + 金融时报 (权威感) + 品牌模式 (陶土色规范)

核心骨架：
├── 黑白灰三色层级（数据模式）
├── 无阴影无渐变（数据模式）
├── 表头 2px 黑线（数据模式）
└── 单一陶土色焦点（数据模式）

融合元素：
├── 顶部装饰线 → 品牌模式的陶土色使用规范
```
