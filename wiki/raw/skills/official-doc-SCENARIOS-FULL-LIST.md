## 六、按沟通对象分类

- 评估汇报
- 成本分析
- 风险处置方案
- 清退决策
- 事故报告

### 6.2 内部协作（给同事）
- 会议通知
- 进度同步
- 跨部门协作
- 资源协调
- 方案审阅

### 6.3 供应商沟通
- 通知类：准入通知、整改通知、巡检通知、系统升级
- 评估类：评估通报、激励沟通、改进计划
- 谈判类：价格谈判、合同讨论、争议沟通
- 结算类：对账单、付款通知、结算异常
- 清退类：清退通知、交接验收

## 七、按业务阶段分类

### 7.1 引入阶段
- 寻源计划 → 合作邀约 → 准入评估 → 准入通知 → 试运行 → 转正

### 7.2 运营阶段
- 日常运营报告 → 整改通知 → 约谈管理 → 巡检管理 → 故障处理

### 7.3 评估阶段
- 数据收集 → 评估通报 → 激励方案 → 表彰奖励 → 改进计划

### 7.4 清退阶段
- 风险处置 → 清退决策 → 清退通知 → 交接验收 → 完成报告

### 7.5 结算阶段
- 月度对账 → 结算确认 → 付款通知 → 成本分析
