## 二、核心困境

### 2.1 头部供应商"绑架"

```
现状：
- 毅航+毛毛虫合计50%产能
- 字节放心贷结算高40-50%（1.4-1.5万 vs 1万）
- 已流失骨干30人（2-3月）
- 人员置换完成：我们用新人，字节吸收资深员工

风险：
- 外部挖人风险持续存在
- 收紧管理 = 加速流失
- 年老师没有谈判筹码
```

### 2.2 中腰部承接能力不足

```
现状：
- 第三名及以后占比50%，但分散在10家
- 结算水平约8000元/人（未扣成本）
- 扩张意愿不强
- 业绩不稳定，难以承接头部产能

原因：
- 结算低 → 投入意愿低 → 培训资源不足 → 新人流失高
- 50-80人规模冲到第一名时，老员工流失+新人补入 → 业绩下滑
```

### 2.3 新人流失率异常

| 指标 | 正常值 | 供应商实际 | 自营团队 |
|------|--------|-----------|----------|
| 3个月内流失率 | 5% | **30-50%** | 10%+ |
| 成长周期（独立产出） | - | 1个月 | 1个月 |
| 成长周期（养活自己） | - | 3个月 | 3个月 |

**关键洞察**：自营团队流失率是供应商的1/3-1/5，说明培训体系有巨大差异。

## 三、年老师的应对策略

### 3.1 已在执行

| 策略 | 内容 | 进展 |
|------|------|------|
| **BPO联盟** | 把自营团队成熟的培训体系（AI辅助）交给中腰部供应商 | 推进中 |
| **新业务分配** | 新业务优先给中腰部供应商 | 已执行 |
| **第二梯队培育** | 给第三、四、五名设定更高目标，小批量增补 | 规划中 |

### 3.2 目标路径

```
短期：BPO联盟推广 → 降低新人流失率 → 中腰部承接能力提升
中期：第二梯队形成 → 第三、四、五名规模扩大 → 抗风险能力增强
长期：第二梯队挑战头部 → 供应商结构健康化 → 降低头部依赖
```

## 四、组织结构约束

### 4.1 权责分离

| 维度 | 策略组（刘伟佳） | 年老师（服务组） |
|------|------------------|------------------|
| 目标 | 费效比优化 | 供应商稳定 |
| 手段 | 定价权、人力要求 | 业务分配、供应商管理 |
| 风险承担 | 无 | 有 |
| 考核 | 费效比 | 业务连续性 |

**核心矛盾**：定价权在策略组，但供应商稳定责任在年老师。

### 4.2 与策略组的沟通僵局

**年老师的做法**：
- 提出建立定价委员会，要求定价需服务组同意
- 事前与策略组老板、刘伟佳沟通不充分
- 方式太直接，导致对抗情绪

**刘伟佳的反应**（行为模式分析）：
- 表面：说"担心供应商不好管理"
- 实际：
  - 要求头部供应商增人 → 人员仍短缺
  - 批评中腰部业绩不佳 → 但不解决结构性问题
  - 增人重点是头部职场 → 强化头部依赖

**潜台词**：刘伟佳在用"供应商管理问题"来维护自己的定价权，同时把责任推给年老师。

## 五、京东的供应商培养悖论

```
京东的培养路径：
供应商小 → 京东给业务、给资源 → 供应商做大 → 其他甲方发现 → 供应商向外扩张 → 有了退路 → 开始议价/流失
```

**毅航和毛毛虫就是这个路径的产物**：
- 它们是京东培养出来的头部
- 现在有能力接字节的业务
- 字节给出更高结算，自然选择离开

**这不是背叛，是模式的必然结果。**
