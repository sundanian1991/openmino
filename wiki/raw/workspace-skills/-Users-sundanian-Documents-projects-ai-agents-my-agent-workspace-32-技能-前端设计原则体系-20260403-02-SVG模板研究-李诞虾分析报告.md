## 二、结构分析

### 2.1 双层分离

**填充层**（7 个 path）：
- 主体躯干：6 个大块填充（#D6654B 陶土色）
- 眼睛：1 个黑色圆点（#1A1612）

**描边层**（25 个 path）：
- 外轮廓：粗线条（4-5px）
- 细节刻画：中等线条（2-4px）
- 细线纹理：细线条（1.7-2.3px）

### 2.2 stroke-linecap 应用

**所有描边 path 统一应用**：
```xml
stroke-linecap="round"
```

**这是手绘感的核心特征** — 圆头端点让线条看起来像手绘笔画。

## 四、线条粗细分层

### 4.1 粗线条（4-5px）- 外轮廓

```xml
stroke-width="5.206"  <!-- 主轮廓 -->
stroke-width="4.631"  <!-- 次轮廓 -->
```

**用途**：虾的整体外形、身体分段轮廓

### 4.2 中等线条（2-4px）- 细节结构

```xml
stroke-width="4.055"
stroke-width="3.479"
stroke-width="2.907"
```

**用途**：虾节之间的连接、肢体结构

### 4.3 细线条（1.7-2.3px）- 纹理细节

```xml
stroke-width="2.331"
stroke-width="2.907"
stroke-width="1.755"
```

**用途**：虾腿细节、触须、身体纹理

## 五、贝塞尔曲线特征

### 5.1 曲线类型

**大弧线**（身体轮廓）：
```xml
d="m18.72 41.09c11.14 7.83 28.25 9.87 43.46 9.83..."
```
- 长距离平滑曲线
- 控制点间距大（20-30px）

**短弧线**（关节连接）：
```xml
d="m109.9 129.8c3.2 5.32 8.05 8.97 17.98 12.87..."
```
- 短距离转折
- 控制点间距小（5-10px）

### 5.2 手绘感来源

**不完美的曲线**：
- 没有使用完美的圆弧
- 曲线中有"微抖动"（轻微的方向变化）
- 这种"不完美"正是手绘感的核心

## 六、模板化建议

### 6.1 可参数化元素

**可变的**：
- 主体贝塞尔曲线（姿态变化）
- 肢体角度（动态姿势）
- 眼睛位置（表情变化）

**固定的**：
- 色彩系统（#D6654B + #1A1612）
- stroke-linecap="round"
- 线条粗细层级（5px/3px/2px）
- 整体结构（头+身+尾+腿）

### 6.2 建立模板的步骤

**Step 1**：提取结构骨架
```
头部区域 → 身体前段 → 身体中段 → 身体后段 → 尾部
```
