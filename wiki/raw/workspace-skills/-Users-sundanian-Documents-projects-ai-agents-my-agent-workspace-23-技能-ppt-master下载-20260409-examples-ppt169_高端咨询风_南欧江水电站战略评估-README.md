# 南欧江梯级水电站项目战略评估报告 PPT

## 项目概述

本项目为**深度战略评估报告：老挝南欧江梯级水电站项目、主权债务危机与中国电建-老挝电力公司仲裁案全景分析**的PPT演示文稿。

## 项目信息

| 属性 | 说明 |
|------|------|
| **画布格式** | PPT 16:9 (1280×720) |
| **设计风格** | 高端咨询风格 - "流域危机"主题 |
| **页数** | 20页 |
| **生成日期** | 2025-12-07 |

## 设计特色

采用独创的**"流域危机"（River Basin Crisis）**视觉语言：

1. **湄公河色系**：深水蓝 → 湄公青 → 电力蓝的渐变，象征水电能源
2. **危机警示色**：违约红、仲裁金、压力橙表达债务危机主题
3. **水流层叠**：信息如梯级水坝般层层递进
4. **深浅交替**：深色/浅色页面交替使用保持视觉节奏

## 使用方法

### 预览

```bash

cd svg_final && python3 -m http.server 8000
# 浏览器打开 http://localhost:8000

# 预览原始版本
cd svg_output && python3 -m http.server 8001
```

### 导出

SVG文件可以：
- 直接在浏览器中查看
- 使用 Inkscape 转换为 PDF
- 使用在线工具转换为 PPT 格式

## 核心数据

- **总装机容量**：1,272 MW（七级梯级水电站）
- **总投资额**：$27.33亿（中国政策性银行融资）
- **仲裁索赔额**：$5.55亿（占老挝GDP约4%）
- **欠款时间**：2020年1月至2024年12月（4年）

## 核心结论

1. **仲裁即重组序曲**：$5.55亿索赔是谈判筹码，非终局对抗
2. **商业逻辑压倒外交**："一带一路"项目进入法律契约保护阶段
3. **蓄电池模式破产**：美元债务-软货币收入错配必然导致危机
