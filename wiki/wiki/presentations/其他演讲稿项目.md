## 数据炼金师项目

**创建时间**：2026-04-26

数据炼金师（data-alchemist）是一个 AI 驱动的数据叙事工具项目。在 presentations 目录下包含了 SPEC 文档和 README 文件。

产品定位：用户上传真实数据（CSV/Excel），工具通过引导式问答明确受众和目标，AI 自动分析数据并应用 Storytelling with Data 的 30 个可视化技能，生成包含多图表、三幕叙事的完整 HTML 报告。

技术栈：React 18 + TypeScript + Vite + shadcn/ui + ECharts + Framer Motion + Zustand + Claude API

项目架构包含：数据上传模块、AI 对话引擎（5 步状态机）、图表工厂（10 种图表类型）、三幕叙事 UI、图表微调功能、报告导出、评分系统（5 维度 × 20 分 = 100 分）。

这是一个实用工具 + 游戏化外壳的产品，核心是出报告，外壳提供沉浸感。
