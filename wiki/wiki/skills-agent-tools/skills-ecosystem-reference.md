## Visualization Skills — 数据可视化与图表生成

这是技能生态中最大的一组，分三个层次。

### echarts-visualization（29 文件）

使用 Apache ECharts 生成独立交互式 HTML 图表。覆盖 29 种图表类型，是数据可视化的主力技能。

**图表类型覆盖**：

| 类别 | 图表 |
|------|------|
| 基础图表 | 柱状图、折线图、饼图、散点图、面积图 |
| 高级图表 | 桑基图、雷达图、漏斗图、仪表盘、热力图 |
| 地理图表 | 地图、路径图 |
| 层次图表 | 树图、旭日图 |
| 金融图表 | K线图、主题河流图 |
| 特殊图表 | 极坐标图、平行坐标图、象形柱图、线图、效果散点图 |

**特点**：有生成脚本（`scripts/generate-echarts.js`），支持资产和引用参考。

### chart-visualization（27 文件）

通用数据可视化技能，基于 Chart.js。27 种图表类型 + 表格生成。

**与 echarts 的关系**：echarts 能力更全、支持更多高级图表（K线、桑基、热力图），优先使用 echarts。chart-visualization 作为轻量补充和兜底。

### data-visualization（4 文件）

基于 Python matplotlib 的数据可视化。核心能力是根据数据类型智能选择图表类型。

**交叉引用**：与 echarts/chart-visualization 的区别在于输出格式——Python 输出 PNG/SVG 静态图，另外两个输出 HTML 交互式图表。

### AntV S2 Expert（30 文件）

多维交叉分析表格开发助手。支持透视表、明细表、自定义单元格、主题样式、排序、汇总、条件格式、冻结、图标、SSR、React 组件等。

**应用场景**：供应商绩效多维分析报表、KPI 交叉查询表。这是唯一一个面向表格而非图表的技能。

## 技能的安装、管理和清理

### 安装流程

1. **搜索**：通过 clawhub.ai API (`curl -s https://clawhub.ai/api/skills.zh.json`) 或 skillhub.tencent.com 搜索匹配技能
2. **确认**：推荐 1-3 个最匹配的技能，年老师确认后安装
3. **安装**：fetch SKILL.md 源文件，复制到 `.claude/skills/` 目录
4. **按需安装**：不批量安装，用到了才装

### 触发机制

**两级触发**：
- **一级**（轻量）：`TRIGGER-INDEX.md`（~500 tokens）— 从 ~5000 tokens 压缩，按关键词匹配
- **二级**（完整）：命中后加载对应 `SKILL.md` 完整文件（~200 tokens）
