## Overview

供应商管理有四个高频场景，每个场景都有独立的触发条件、处置流程和沟通策略。掌握这四个场景的方法论，覆盖日常80%以上的供应商管理工作。

## 场景一：KPI不达标整改

### 触发条件

| 等级 | 条件 | 动作 |
|------|------|------|
| 黄灯 | 连续1个月不达标 | 提醒关注 |
| 橙灯 | 连续2个月最后一名 | 正式整改 |
| 红灯 | 连续3个月不达标 | 熔断/清退 |

### 处置流程

数据确认 → 原因诊断（外部因素 vs 内部因素）→ 整改要求（三要素法）→ 跟踪验证（周度跟进、月度复盘）

### 沟通策略

**场景区分**：第一个月/情况不差 → 电话10分钟；三个月后/情况严重 → 线下沟通，压力更大。

**开场结构**（五步）：

1. 问现状（"什么情况？上不去了吗？"）
2. 问原因（"为什么连续靠后？"）
3. 内部背书（"业务那边反馈过"）
4. 预警施压（"再做不好业务难了"）
5. 暗示后果（"可以考虑一下"）

**区分共性 vs 个性问题**：名单质量问题是所有人都面临的。如果大家都面临一样的问题，你是最差的，还是得想一下自身的情况。

**支持边界**：新供应商上线派人培训一次，日常运营线上指导，底线是不派人到现场（成本高、不可持续）。

### 目标设定与闭环

| 结果 | 动作 |
|------|------|
| 目标达成 | 私下了解经验 → 公开表扬 → 制造竞争 |
| 达不到 | 最后一次机会，再聊一次 |
| 再达不到+其他问题 | 撤，不再合作 |

## 场景二：供应商"绑架"应对

### 四种绑架类型

| 类型 | 表现 | 危害程度 |
|------|------|---------|
| 跳级绑架 | 不找你，找上级/业管，从上至下施压 | 中等 |
| 规模绑架 | 份额太大（50-60%），大而不能倒 | 高（最难处理） |
| 资源绑架 | 说人力不足、投入不够，威胁无法合作 | 低（可替代） |
| 外部退路绑架 | 有外部客户，威胁退出 | 低（可替代） |

核心判断：只要不是大到不能倒，都有替代方案。

### 五条可复用的管理原则

**不妥协原则**：绑架类问题，妥协一次 = 后续不断妥协。

**问题定义权**：跳级绑架的核心是"谁定义问题"。不被动接问题，自己定义边界和解决方案。

**验证优先**：判断真假用交叉验证：同行 → 同业 → 上下游 → 内部交叉验证。

**筹码盘点**：能给的是稳定（少调整少波动）和规模（以规模换价格）；不能给的是利润和原则性让步；底线是用规则限制涨幅上限 + 培育替代供应商。

**差异化管理**：规模优先的给规模，利润优先的给稳定，稳定优先的承诺少波动。

### 反制跳级绑架公式

> 他不给方案 + 只抛问题 → 你不被动回答 → 自己定位具体问题 + 让供应商提建议 + 快速决策

### "有绑架能力"的判断标准

- 占比 ≥30%
- 业绩前三
- 人力 ≥100人

## 场景三：份额调整

### 三种场景

| 类型 | 触发条件 | 核心考量 |
|------|---------|---------|
| 赛马增减 | 排名结果 | 优胜劣汰 |
| 新增数据分配 | 业务增长 | 业绩优先 + 集中度管控 |
| 业务调整缩减 | 业务收缩/风险 | 保优汰劣 |

### 五因素调整原则

1. 业务业绩情况
2. 增长速度情况
3. 工程配合度
4. 是否具有战略意义
5. 是否有战略扶持

### 增份额流程

策略组发起 → 业务管理 → 年老师拟定方案 → 王易人审批 → 俊哥审批

新增人力增幅上限：一般不超过30%或50%（具体见集中度管控规则）。

### 减份额流程

电话铺垫 → 告知决策依据（为什么减、减多少、时间节点）→ 邮件正式通知

**制度原则**：原则上谁排名最后，就去做相关调整。制度设计层面可以直接退掉这块业务，但至今没遇到过不服闹事的情况。

### 冲劲保持问题

中尾部供应商增加份额后，常因加入新人导致产能不够、业绩反而下滑。历史回看发现稳定前4-5名的供应商有个规律：不管何时加入新人，业绩会下降，但因为新人离开，业绩又能重新排名靠前。这条线的标准大致是30-50个急需的老人、3个月以上的老人占比。

## 场景四：新供应商准入

### 准入流程

线上初筛 → 能力评估 → 现场考察 → 试单评估 → 合同签订 → 入场

流程周期：原15天 → 现7-10天（优先级调高，准备工作前置，验厂环节可能不去现场）。

### 线上初筛标准

| 硬性条件 | 业务条件 |
|---------|---------|
| 增值电信许可证 | 是否做过同类业务 |
| 注册资金≥1000万 | 人力规模和排名 |
| 站点分布 | 人均结算水平 |
| 人均结算 | 能力项、靠什么达成业绩 |

### 现场考察的判断逻辑

**机房布置**：恒温恒湿 + 独立监控 + 消防灭火 → 机房能做到这个程度，其他标准就不言而喻。

**运营管理**：有细致的问题追踪、判断、定位能力，能快速做调整。

**会议质量**：辅导型管理（员工积极参与、准确定位问题、给启发）优于强压型管理（从上至下强压、只要求"必须怎样"、情绪凶不能辅导）。

### 宽进严出策略

踩坑教训：线上数标排第一第二的供应商，实际合作出现"水土不服"。解决方案是用实际效果筛选，不唯数标排名，例如一次性进三家一起PK。

### 一票否决项

- 资质造假
- 重大合规违规记录
- 财务状况严重恶化
- 核心团队不稳定（流失率>50%）

### 风险与应对

| 风险 | 应对 |
|------|------|
| 验厂缺失 | 宽进严出，试单PK |
| 分包转包 | 发现后直接清退 |
| 预期差异 | 多家同时试单，依据实际效果 |
