## 概述

本文整合了两份扩项申请的实际案例：泉州速腾参与采购融资测试项目的扩项申请（2026年2月）和采购融资扩项申请的详细内容。已有wiki文章"供应商引入与退出规则"涵盖了通用的准入/清退规则，本文补充实际执行中的扩项申请流程、定价规则、合规要求和业务整合背景。

## 一、扩项申请流程概览

### 1.1 标准流程

```
业务需求确认 → 寻源策略制定 → 供应商选择 → 扩项申请提交 → 领导审批 → 测试项目启动 → 定期评估 → 正式引入/退出
```

### 1.2 申请文档结构

每份扩项申请包含以下核心部分：
1. **背景与寻源策略** — 为什么引入、如何寻找
2. **供应商选择依据** — 为什么选这家
3. **人力增补计划** — 需要多少人、什么时间
4. **业务整合背景** — 名单来源、电销规划
5. **定价与结算规则** — CPS系数、业绩归因
6. **合规说明** — 流程合规性确认

## 二、案例一：泉州速腾扩项申请

### 2.1 背景与寻源策略

**需求**：引入一家具有竞品合作经验、熟悉大额供应链贷款产品的供应商，完善产品竞争力。

**寻源策略**：从存量大额职场中寻源具备供应链金融经验的供应商。

**寻源结果**：泉州速腾网络科技有限公司。该供应商目前服务网商银行，具备大额项目、供应链金融项目外呼经验，有意愿参与本次测试。

### 2.2 人力增补计划

| 考核指标 | 目标值 |
|---------|--------|
| 月均放款 | 60万 |
| 测试周期 | 3月1日 - 4月30日 |

**人力规划**：
- 3月：现有坐席3人 → 预估需求6人 → 本月增员3人

### 2.3 业务整合背景

**采购融资整合与电销规划对齐**：

| 名单来源 | 数量 |
|---------|------|
| 脱核名单（企业主贷） | 10万 |
| 核企名单（核心企业下游） | 6万 |
| 总名单量 | 16万 |

**电销规划**：
| 周期 | 人力上限 | 目标 |
|------|---------|------|
| 26年Q1 | 6人（2月、3月增至6人） | 产品流程跑通，余额净增420万 |
| 26年全年 | 与名单扩容节奏匹配 | 支持采购融资全年余额净增20.8亿 |

### 2.4 定价与结算规则

**结算系数**：CPS=1.5%，结算金额=GMV×1.5%

**业绩归因口径**：
- 业绩窗口期：0 < 订单生成日期 - 末次外呼接通时间 ≤ 10天（精确到秒）
- 业绩取值：实际放款金额
- 有效性判定：通话时长≥10秒
- 业绩归属坐席：一人多pin场景下，优先查询下单pin的有效话单；若无，则取客户实名下关联pin的末次有效话单
- 职场业绩结算规则：10日内还款部分的金额不结算

### 2.5 合规说明

本次扩项人数在10人以内，符合小额零星采购标准，流程合规。

## 三、案例二：采购融资扩项申请

### 3.1 采购融资业务背景

采购融资是企金业务线的重要产品，通过电销渠道触达供应链上下游企业客户，提供采购环节的融资服务。

### 3.2 业务规模与目标

**全年目标**：采购融资全年余额净增20.8亿

**Q1里程碑**：
- 产品流程跑通
- 余额净增420万
- 人力控制在6人以内

### 3.3 名单策略

**两类名单来源**：
1. **脱核名单**（企业主贷）：10万条，来自已有企业主贷业务的脱核客户
2. **核企名单**（核心企业下游）：6万条，来自核心企业的下游供应商

**名单分配原则**：
- 优先分配给有供应链金融经验的供应商
- 新供应商从少量名单开始测试
- 根据转化效率动态调整名单分配比例

### 3.4 定价机制

**CPS模式**：按实际成交金额（GMV）的1.5%结算

**与传统CPS的区别**：
- 传统CPS按放款金额结算
- 采购融资CPS需考虑还款因素（10日内还款不结算）
- 业绩归因更严格（10天窗口期+有效通话10秒+末次接通）

### 3.5 风险控制

**供应商选择标准**：
- 竞品合作经验（网商银行等）
- 大额项目经验
- 供应链金融理解
- 合作意愿强烈

**测试期管理**：
- 设定明确的时间节点（2个月测试期）
- 设定量化目标（月均放款60万）
- 定期评估（每周跟踪人力和转化）
- 退出机制（未达标则终止测试）

## 四、扩项管理关键经验

### 4.1 寻源经验

| 经验 | 说明 |
|------|------|
| 从存量中找 | 优先从已有供应商中筛选，降低引入成本 |
| 看竞品经验 | 有竞品合作经验的供应商上手更快 |
| 小额测试 | 先小范围测试，验证能力后再扩大 |
| 合规先行 | 确保扩项人数在小额零星采购标准内 |

### 4.2 定价经验

| 经验 | 说明 |
|------|------|
| CPS系数要合理 | 1.5%需确保供应商有合理利润空间 |
| 归因口径要清晰 | 窗口期、有效性、归属规则必须明确 |
| 特殊规则要考虑 | 如10日内还款不结算 |

### 4.3 管理经验

| 经验 | 说明 |
|------|------|
| 人力规划要精确 | 按月制定增员计划，不模糊 |
| 目标要量化 | 月均放款60万，不是"尽量做好" |
| 名单要匹配 | 名单量与人力要匹配，避免浪费 |
| 退出要有依据 | 测试期未达标，有明确的退出标准 |

---

## 五、关键洞察

1. **小额测试降低风险**：10人以内扩项符合小额零星采购标准，流程简化且风险可控
2. **竞品经验是核心筛选标准**：泉州速腾因服务网商银行被选中，有供应链金融经验是关键
3. **CPS归因口径精细化**：10天窗口期+10秒有效通话+末次接通+10日内还款排除，确保归因精准
4. **名单与人力匹配是关键**：16万名单配6人，人均2.7万条，需要合理的转化效率支撑
5. **全年目标20.8亿需分阶段**：Q1仅420万（占全年2%），大部分增长在Q2-Q4
