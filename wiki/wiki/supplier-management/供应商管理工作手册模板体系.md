## Overview

供应商工作手册模板体系是一套标准化的填写模板，覆盖供应商全生命周期的关键操作场景。从准入承接、日常运营、持续提升到退出清退，每个场景都有对应的模板，确保供应商"知道怎么填、知道填什么、知道交给谁"。本文系统梳理工作手册模板的结构、内容和使用指南。

## 工作手册模板架构

### 三大模块

| 模块 | 模板数量 | 覆盖场景 |
|------|---------|---------|
| 准入承接 | 3个 | 人员准入、制度报送、站点报备 |
| 日常运营 | 5个 | 分量确认、排班考勤、数据报送、结算争议、质量合规 |
| 持续提升 | 2个 | 月度复盘、季度对标 |
| 退出清退 | 1个 | 交接清单 |

### 模板设计原则

| 原则 | 说明 |
|------|------|
| 统一格式 | 所有模板使用统一的表格结构 |
| 明确字段 | 每个字段都有填写说明和示例 |
| 必填/选填 | 明确标注必填字段和选填字段 |
| 校验规则 | 内置数据校验规则（如日期格式、数值范围） |
| 版本管理 | 每次更新生成新版本号 |

## 准入承接模板

### 人员准入模板

**用途**：新供应商上线时，填报所有在岗人员信息。

**核心字段**：

| 字段 | 说明 | 校验规则 |
|------|------|---------|
| 姓名 | 员工真实姓名 | 非空 |
| 身份证号 | 18位身份证号码 | 格式校验 |
| 岗位 | 坐席/主管/质检/其他 | 枚举值 |
| 入职日期 | YYYY-MM-DD | 日期格式 |
| 证书编号 | 金融资质证书编号 | 非空（坐席必填） |
| 证书有效期 | YYYY-MM-DD | 日期格式，不能为空 |
| 紧急联系人 | 姓名+电话 | 非空 |

### 制度报送模板

**用途**：供应商报送内部管理制度。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 制度名称 | 如《考勤管理制度》 |
| 制度版本 | 如V1.0 |
| 生效日期 | 制度生效日期 |
| 制度类别 | 人事/合规/运营/安全 |
| 附件 | 制度文件PDF |

### 站点报备模板

**用途**：报备供应商的职场信息。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 站点名称 | 职场名称 |
| 省份/城市/区县 | 地址层级 |
| 详细地址 | 具体地址 |
| 站点负责人 | 姓名+电话 |
| 站点对接人 | 姓名+电话+企微 |
| 工位数量 | 总工位数 |
| 启用日期 | 站点启用日期 |

## 日常运营模板

### 分量确认模板

**用途**：确认月度分量分配结果。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 月份 | 如2026-04 |
| 业务线 | 如金条人工 |
| 分配份额 | 如30% |
| 分配产能 | 如日均1000通 |
| 确认状态 | 确认/有异议 |
| 异议说明 | 如有异议，说明理由 |

### 排班考勤模板

**用途**：填报每日排班和考勤数据。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 日期 | YYYY-MM-DD |
| 班次 | 早班/中班/晚班 |
| 应出勤人数 | 排班人数 |
| 实际出勤人数 | 实际到岗人数 |
| 缺勤人数 | 应出勤-实际出勤 |
| 缺勤原因 | 病假/事假/旷工/其他 |

### 数据报送模板

**用途**：每日业绩数据报送。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 日期 | YYYY-MM-DD |
| 业务线 | 如金条人工 |
| GMV | 当日放款金额 |
| 通话量 | 当日外呼接通量 |
| 接通率 | 接通量/拨打量 |
| 转化率 | 成单量/接通量 |

### 结算争议模板

**用途**：提交结算金额异议。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 结算月份 | 如2026-03 |
| 甲方结算金额 | 甲方计算金额 |
| 供应商核算金额 | 供应商计算金额 |
| 差异金额 | 两者差异 |
| 差异原因 | 详细说明 |
| 支撑材料 | 数据截图、计算过程 |

### 质量合规模板

**用途**：填报质量合规自查结果。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 月份 | 如2026-04 |
| 质检月均分 | 当月质检平均分 |
| A类违规次数 | 严重违规次数 |
| B类违规次数 | 一般违规次数 |
| 话术合规率 | 合规通话占比 |
| 持证率 | 持证人员占比 |
| 信息安全违规 | 0/1（有/无） |

## 持续提升模板

### 月度复盘模板

**用途**：供应商月度复盘总结。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 月份 | 如2026-04 |
| 产能达成 | 实际产能/目标产能 |
| 质检均分 | 当月质检均分 |
| 出勤率 | 月度平均出勤率 |
| 排名 | 当前业务线排名 |
| 亮点 | 本月做得好的方面 |
| 不足 | 本月需要改进的方面 |
| 下月计划 | 下月改进目标和措施 |

### 季度对标模板

**用途**：供应商季度对标分析。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 季度 | 如2026-Q1 |
| 本季度得分 | 季度综合评分 |
| 上季度得分 | 上季度综合评分 |
| 变化趋势 | 上升/下降/持平 |
| 对标供应商 | 对比的供应商名称 |
| 差距分析 | 与对标供应商的差距 |
| 改进计划 | 缩小差距的具体措施 |

## 退出清退模板

### 交接清单模板

**用途**：供应商退出时的交接确认。

**核心字段**：

| 字段 | 说明 |
|------|------|
| 退出类型 | 单业务线退出/全业务线退出/站点缩减 |
| 退出日期 | 正式退出日期 |
| 数据盘点 | 客户信息、业务数据清单 |
| 交接确认 | 数据是否已完成交接 |
| 账号关闭 | 京音/企微/质检账号是否已关闭 |
| 数据删除 | 是否提交《数据删除确认函》 |
| 保密延续 | 是否签署《保密延续协议》 |
| 结算状态 | 结算是否已完成 |

## 模板使用指南

### 填报流程

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 下载对应模板 | 根据场景选择模板 |
| 2 | 填写必填字段 | 按字段说明逐项填写 |
| 3 | 自查数据 | 检查数据完整性和合理性 |
| 4 | 提交 | 通过指定渠道提交（企微群/系统/邮件） |
| 5 | 确认 | 供应商管理岗确认收到并审核 |

### 填报注意事项

| 注意事项 | 说明 |
|----------|------|
| 及时性 | 按规定时间提交，不迟报 |
| 准确性 | 数据与实际情况一致，不错报 |
| 完整性 | 所有必填字段都填写，不漏填 |
| 一致性 | 同一数据在不同模板中数值一致 |
| 可追溯 | 数据有来源和依据 |

## 模板体系的演进方向

- **模板数字化**：将纸质模板转为系统表单，实现在线填报
- **自动校验**：系统自动校验数据格式和合理性
- **模板智能推荐**：根据场景自动推荐对应模板
- **历史数据预填**：系统自动预填历史数据，减少手工填报
