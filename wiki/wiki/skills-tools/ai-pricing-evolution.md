## 概述

AI 产品定价模式研究（2026-04）梳理了国内七大 AI 平台从 Coding Plan（包月订阅）到 Token Plan（按量计费）的演变路径，核心判断是：Coding Plan 是过渡产品，Token Plan 是终态。研究以 25 页演讲的形式，从时间线、竞争格局、三层逻辑、芯片供给等维度展开分析。

## 核心时间线（2024 Q4 — 2026-04）

```
2024年底  DeepSeek带头降价，百万Token从几十块降到几毛钱
    ↓
2025年底  智谱GLM率先推出Coding Plan，固定月费不限量
    ↓
2026-03   阿里百炼以7.9元首月价入局，腾讯云最后入场，六大平台全部上线
    ↓
2026-03   收紧开始：限购、涨价、停售低档、限时抢购
    ↓
2026-03   MiniMax标志性动作：Coding Plan改名Token Plan
    ↓
2026-04   DeepSeek V4发布，1.6万亿参数，单Token推理计算量降至27%
```

## 七家竞争格局

### 两条路线

| 路线 | 代表 | 特点 |
|------|------|------|
| **专卖店** | 智谱、Kimi、MiniMax、小米MiMo | 只卖自家模型，深度优化 |
| **超市** | 阿里百炼、火山方舟、腾讯云 | 聚合多家模型，一个Key自由切换 |

### 值得关注的信号

小米 MiMo 2026 年 4 月入局，V2-Pro 定价 1-3 美元（Claude 的五分之一），但同时公开批评"低价竞争是陷阱"——低价是入场券，不是终局。

## 定价演变的三股力量

| 力量 | 表现 | 影响 |
|------|------|------|
| **模型能力跃升** | Kimi K2.5万亿参数、智谱GLM-5进入第一梯队、DeepSeek V4开源 | 模型越强→Agent越好用→Token消耗越大 |
| **Agent框架爆发** | OpenClaw 355K stars，一个任务等于几十次调用 | Token消耗暴涨6-10倍 |
| **芯片供给变化** | 英伟达中国市占率66%→54%，华为昇腾950PR 4月量产 | 算力成本下降→定价空间打开 |

## 计量单位：定价权之争

五种计量方式同时存在，直接比数字毫无意义：

| 计量方式 | 代表 | 问题 |
|---------|------|------|
| API请求次数 | 阿里百炼 | 数字大但每次调用很轻 |
| Prompt次数 | MiniMax、智谱 | 1个Prompt=15-20次模型调用 |
| Token计量 | MiniMax、Kimi | 最透明，最可比 |
| Credit计量 | 部分平台 | 自定义单位，最不透明 |
| 时间窗口配额 | 少数平台 | 限制使用频率 |

**核心洞察**：谁定义计量单位，谁就掌握定价权。模糊计量有利于早期获客，透明计量有利于长期信任。

## 补贴—收紧—分层：互联网赛道的通用节奏

| 阶段 | 时间 | 表现 |
|------|------|------|
| **补贴期** | 2024-2025上半年 | 百万Token几毛钱，首月7.9元，疯狂圈用户 |
| **收紧期** | 2025下半年 | 限购、停售低档、限时抢购，测试用户价格弹性 |
| **分层期** | 2026年 | MiniMax极速版98-899，Kimi Allegro 559，企业版198-1398 |

## 定价三层逻辑

| 层次 | 驱动力 | 含义 |
|------|--------|------|
| **成本驱动** | 算力成本→Token成本→套餐定价 | 芯片供给直接绑定定价天花板 |
| **竞争驱动** | 七家价格战→补贴圈用户→收紧利润 | 互联网赛道通用逻辑 |
| **价值驱动** | 从"卖Token"到"卖工作流"到"卖生态" | MiniMax Token Plan打包图片/语音/音乐/视频 |

## Agent 成本爆炸

一个开发者用 Claude Code 8 个月消耗 100 亿 Token，按 API 价格值 15000 美元，实际只花 800 美元订阅费（省 93%）。Token 构成：文件读取占 35-45%，代码生成只占 5-15%。大头不是写代码，是读代码。

**结论**：Coding Plan 的"不限量"模式不可持续，Agent 让消耗指数级增长，包月模式撑不住。

## 芯片供给线：定价的底层变量

| 指标 | 数据 |
|------|------|
| 英伟达中国市占率 | 66%→54%（持续下降） |
| 华为昇腾2025交付 | 81.2万片，市占率20% |
| 昇腾950PR | FP4算力是H20的2.87倍，价格是H200的1/3 |
| 国产芯片自给率 | 2023年17%→2027年预计55% |

DeepSeek V4 官方声明："受限于高端算力，目前吞吐有限，预计下半年昇腾950批量上市后价格大幅下调。"现在的价格还不是最低的。

## 开源定价权悖论

DeepSeek 模式：开源、极致低价、不卖订阅。V3.2 在 OpenRouter 月调用 5.35 万亿 Token（全球第三）。

**悖论**：开源模型的免费部署能力持续压制定价上限。定价权不来自"模型独占性"——因为模型可以被开源替代。定价权来自"服务便利性"——用户为省事付费，不是为模型付费。

## 四个确定性趋势

1. **Token 成本将持续下降** — 国产芯片放量 + MoE 效率革命 + 推理专用芯片竞争
2. **国产芯片替代加速** — 华为+寒武纪+阿里平头哥形成国产算力三角
3. **开源模型重塑定价权** — 从"模型独占性"转向"服务便利性"
4. **Agent 时代成本指数级增长不可逆** — 推理需求已达训练的 4-5 倍，所有厂商最终转向 Token 精细计量

## 三个反模式

| 反模式 | 案例 | 教训 |
|--------|------|------|
| 突然切换计费模式 | Cursor退款风波，重度用户一周超支350美元 | 改变计费模式要渐进，不要突然切换 |
| 计量单位混乱 | 五种单位并存，90%用户踩坑 | 短期信息不对称有利，长期伤害信任 |
| 饥饿营销反噬 | 每天10点半抢购 | 短期制造紧迫感，长期伤害品牌 |

## 定价决策清单

1. **定义计量单位** — Token、请求、次数还是 Credit？决定用户感知和利润空间
2. **选择获客策略** — 补贴来得快走得也快，差异化慢但稳
3. **设计分层结构** — 几档？价差多少？太少覆盖不够，太多选择困难
4. **确定透明度** — 模糊有利于短期，透明有利于长期信任
5. **规划演进路径** — Coding Plan 到 Token Plan 的路径已经验证
