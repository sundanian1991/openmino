## 概述

Kami（紙 · かみ）是专业文档排版系统，覆盖 8 种文档类型（简历、一页纸、白皮书、信函、作品集、幻灯片、个股研报、更新日志），统一视觉语言。核心理念："Good content deserves good paper"。不是商务 PPT 也不是消费互联网 UI——像 Monocle 杂志贴上代码后的样子。基于 WeasyPrint（HTML→PDF）和 python-pptx 构建。

## 设计哲学

### 十大不变量

| 序号 | 规则 | 原因 |
|------|------|------|
| 1 | 页面背景 `#f5f4ed` 羊皮纸色，非纯白 | 情绪基础，温暖感 |
| 2 | 单一强调色：墨蓝 `#1B365D`，不超过页面 5% | 克制优先 |
| 3 | 所有灰色暖色调（黄棕底色），无冷蓝灰 | 与羊皮纸一致 |
| 4 | 英文全用衬线，中文标题衬线正文无衬线 | 衬线承载权威 |
| 5 | 衬线字重锁定 400/500，无合成粗体 | 克制排版 |
| 6 | 行高：标题 1.1-1.3 / 紧凑正文 1.4-1.45 / 阅读正文 1.5-1.55 | 印刷节奏 |
| 7 | 中文正文字间距 0.1-0.2pt，英文 0 | 补偿字体密度 |
| 8 | 标签背景用纯 hex，禁止 rgba | WeasyPrint 双矩形 bug |
| 9 | 深度通过环形/微弱阴影，禁用硬投影 | 纸张抬升感 |
| 10 | 禁止斜体 | 设计语言统一 |

### 字体系统

| 语言 | 主字体 | 备选链 |
|------|--------|--------|
| 中文 | TsangerJinKai02（W04/W05 字重 400/500） | 思源宋体 SC → Noto Serif CJK SC → 宋体 SC |
| 英文 | Charter | Georgia → Palatino → Times New Roman |
| 日文 | YuMincho | 俫野明朝 → Noto Serif CJK JP → 思源宋体 JP |

### 间距系统（4pt 基准）

| 级别 | 值 | 用途 |
|------|-----|------|
| xs | 2-3pt | 行内相邻 |
| sm | 4-5pt | 标签内边距 |
| md | 8-10pt | 组件内部 |
| lg | 16-20pt | 组件之间 |
| xl | 24-32pt | 章节标题边距 |
| 2xl | 40-60pt | 大章节之间 |

## 8 种文档类型

| 文档类型 | 模板文件 | 页数约束 | 核心质量规则 |
|----------|----------|----------|--------------|
| 一页纸 | `one-pager.html` | 严格 1 页 | 4 张指标卡片就是标题，30 秒内理解全貌 |
| 白皮书/长文档 | `long-doc.html` | 7±2 页 | 每章开头必须有"主张段落"，通过"那又怎样"测试 |
| 信函 | `letter.html` | 严格 1 页 | 第一段一句话说明目的 |
| 作品集 | `portfolio.html` | 6±2 页 | 以问题开头，不是项目名称 |
| 简历 | `resume.html` | 严格 2 页 | 每条 bullet = 行动+范围+可量化结果+业务影响 |
| 幻灯片 | `slides.py` | 7±3 页 | 标题是完整断言句，不是主题标签 |
| 个股研报 | `equity-report.html` | 2-3 页 | 以差异化认知开头（你看到市场没看到的） |
| 更新日志 | `changelog.html` | 1-2 页 | 每条一句话，动词开头，用户视角 |

## 工作流

```
1. 决定语言（中/英/日）→ 匹配对应模板
2. 选择文档类型 → 复制模板，CSS 不动只改 body
3. 内容提炼 → 原始材料先提取事实/数据/日期 → 分类映射到模板章节 → gap-check → 询问一次
4. 来源校验 → 公司/产品/人物/融资等事实需有来源，冲突时问用户
5. 填充 PDF 元数据 → title/author/description/keywords 四个占位
6. 构建验证 → build.py --verify 检查字体/页数/占位符
7. 视觉检查 → 通读 PDF，检查溢出/重叠/字体加载
```

## 内容写作原则

- **数据优于形容词**：不写"显著增长"，写具体数字和变化率
- **判断优于执行**：不写"做了什么"，写"为什么做这个决定，如何被验证正确"
- **独特表达优于行业陈词**：不说"拥抱 AI 时代"，用自己的话
- **诚实边界**：没做过的不宣称，不知道的数字不编造

## 常见陷阱

| 陷阱 | 症状 | 修复 |
|------|------|------|
| 标签双矩形 bug | 缩放时看到两个同心矩形 | 用 `#EEF2F7` 等纯 hex，禁止 rgba |
| 细边框+圆角双环 | border < 1pt + border-radius 渲染两条弧线 | 用背景填充替代或边框≥1pt |
| 简历 2 页溢出 | 字体回退或行高变化导致超页 | 先砍冗余修饰词，再收紧间距，最后微调字号 |
| break-inside 在 flex 内失效 | 卡片仍被分页截断 | 用 block wrapper 包裹 flex item |
| `height: 100vh` 无效 | WeasyPrint @page 上下文无视口单位 | 用 `min-height: 257mm` |
| SVG 箭头方向错误 | `orient="auto"` 不被支持 | 手动绘制 chevron 箭头路径 |
