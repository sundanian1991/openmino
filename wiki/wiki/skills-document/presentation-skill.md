# Presentation — 演讲脚本 + PPT 一键生成器

> Sources: mino, 2026-04-28

## 概述

Presentation 技能输入一个主题，自动生成演讲脚本和品牌级设计质感的 HTML 幻灯片。支持 62 种世界级品牌设计风格一键切换，输出单文件 HTML（含播放器引擎、键盘控制、自动播放、全屏、PDF 导出）。将内容创建与视觉设计分离——演讲脚本提供完整的口语化讲稿，幻灯片提供极简视觉支撑。

## 五步工作流

### Step 1：引导选择设计风格

用户未指定时展示风格选择菜单，可说"帮我选"触发智能推荐。

### Step 2：获取设计规范

从 GitHub 获取对应风格的 DESIGN.md：`https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/designs/{slug}/DESIGN.md`。提取配色系统、字体层级、阴影系统、圆角规范、间距规范。

### Step 3：生成演讲脚本

按主题和时长生成结构化脚本。质量要求：

- 演讲词必须是**完整的口语句子**，不是关键词列表
- 开头要有**钩子**（故事/数据/提问），不用自我介绍开场
- 每页**一个核心观点**，不堆砌
- 转场自然，页与页之间有过渡句
- 结尾有**行动号召**（CTA）

脚本格式包含：幻灯片内容描述、完整演讲词（引用块格式）、时长估算、节奏和肢体语言提示。

### Step 4：生成 HTML 幻灯片

输出单文件 HTML，包含完整幻灯片播放器。

**幻灯片引擎功能**：
- 键盘控制：方向键/空格翻页、F 全屏、A 自动播放、N 备注、P 导出 PDF、Esc 退出
- 点击控制：右半屏下一页、左半屏上一页
- 触摸滑动：支持移动端
- 自动播放：默认 5 秒/页
- 全屏模式
- PDF 导出：触发浏览器打印

**控制栏 UI**：进度条（页码指示器 + 可点击进度条）、自动播放按钮、全屏按钮、PDF 按钮、备注按钮。样式使用当前设计风格的 accent 色。

**切换动画**：默认淡入滑动效果（opacity 0→1 + translateX 30px→0，0.5s ease）。

### Step 5：输出与交付

保存两个文件到用户当前工作目录：`{主题}_演讲脚本.md` 和 `{主题}_slides.html`。用浏览器打开 HTML 幻灯片预览。

## 幻灯片设计原则

### 文字极简

- 标题：最多 1 行
- 内容：最多 3-5 个要点，每个要点一行
- 绝不放大段文字，演讲词放在备注里

### 视觉层级

- 大标题突出核心观点
- 数字/数据用超大字号
- 图标使用 SVG inline，不依赖外部图片

### 常用页面类型

| 类型 | 布局 | 适用场景 |
|------|------|----------|
| **封面** | 居中大标题 + 副标题 + 演讲者 | 第 1 页 |
| **章节页** | 居中章节标题 + 装饰元素 | 章节分隔 |
| **观点页** | 大字号核心观点 + 小字补充 | 关键论点 |
| **列表页** | 标题 + 3-5 条要点（带图标） | 罗列要点 |
| **数据页** | 超大数字 + 说明文字 | 数据冲击 |
| **对比页** | 左右/上下两栏对比 | Before/After |
| **引用页** | 大段引言 + 出处 | 名言/用户反馈 |
| **CTA 页** | 行动号召 + 联系方式 | 结尾 |

## 演讲者备注面板

侧边滑出面板（350px 宽），显示当前页完整演讲词。黑色半透明背景（rgba(0,0,0,0.95)），白色文字，24px 内边距，可滚动。通过按 N 键或点击备注按钮切换显示。

## 响应式适配

幻灯片容器保持 16:9 比例，最大 1920x1080。内容自适应缩放。768px 以下减小 padding 和字号。

## 打印/PDF 样式

@media print 时隐藏所有控制元素、备注面板，每张幻灯片独占一页，position: relative 使全部显示。

## 降级方案

如果无法从 GitHub 获取 DESIGN.md：使用内置的通用设计变量（基于 Stripe 风格的中性渐变方案），告知用户获取失败，已使用默认风格（深色背景 #0a0a0a，白色文字，渐变 accent）。
