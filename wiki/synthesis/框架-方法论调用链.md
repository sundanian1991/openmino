# 框架-方法论调用链

> 方法论的完整调用链路

## 核心调用链

```
看到信号
    │
    ▼
[[触发规则表（状态机）]] 判定状态
    │
    ├─ N（月度正常）→ 无需动作
    │
    ├─ Y1（首次预警）→ 私下沟通
    │
    ├─ Y2（连续预警）→ 正式约谈
    │
    ├─ E（清退评估）→ 启动清退
    │
    ├─ R（回正）→ 取消预警
    │
    ├─ A（A级骤降）→ 立即约谈
    │
    ├─ H1（诚信预警）→ 警告+加倍核查
    │
    ├─ H2（诚信红线）→ 约谈高层
    │
    └─ T（信任重建）→ 恢复正常
        │
        ▼
[[产能利用率双维度判定]] 定位根因
    │
    ├─ 标杆型 → 标杆分享，优先新业务
    │
    ├─ 尽力型 → 评估产线适配，调预期
    │
    ├─ 问题型 → 立即约谈，启动整改
    │
    └─ 暂时波动 → 观察2周，不下结论
        │
        ▼
[[主管胜任评估5项指标]] 评估团队
    │
    ├─ 胜任 → 维持合作
    │
    ├─ 需关注 → 约谈时针对性提问
    │
    └─ 不胜任 → 谈管理团队调整
        │
        ▼
[[异常信号31个]] 交叉验证
    │
    └─ 输出动作（[[场景SOP]]）
```

## 速查卡

```
【看到排名异常】
  Step1: 9状态判定 → [[触发规则表（状态机）]]
  Step2: 双维度矩阵 → [[产能利用率双维度判定]]
  Step3: 主管胜任5项检查 → [[主管胜任评估5项指标]]
  Step4: 异常信号31个交叉验证 → [[异常信号31个]]
  Step5: 输出动作（A1-A6 SOP）

【看到数据存疑】
  5项信号检查 → [[数据真实性防控]]
  造假类型识别 → [[数据真实性防控]]

【整改后验收】
  4维度验证（目标/动作/持续/根因）→ [[整改闭环验证标准]]

【季度评估后】
  Step1: 计算最终得分 → [[季度评估体系（九宫格ABC）]]
  Step2: 九宫格映射 → [[季度九宫格判定]]
  Step3: ABC档位判定 → [[ABC分级管理动作]]
  Step4: C档+趋势下降 → [[清退规则]]
```
