# B4 月度复盘

> 每月1-5号，用1小时把上月数据变成下月决策依据

## 思考步骤

### Step 1：整体趋势速览（10min）

- 上月总GMV vs 目标达成率
- 4条产线各自达成率
- 近3个月环比趋势
- 供应商整体产能利用率均值

### Step 2：逐产线排名扫描（15min）

| 信号 | 动作 |
|------|------|
| 月度正常（不在后30%） | 无需动作 |
| 首次预警（本月后30%，上月不在） | 标记，下周私下沟通 |
| 连续预警（连续2月后30%） | 正式约谈 |
| A级骤降（上月前30%→本月后30%） | 立即约谈 |

### Step 3：产能利用率定性（10min）

同A1双维度矩阵：标杆型/尽力型/问题型/暂时波动

### Step 4：异常供应商逐一归因（10min）

| 维度 | 看什么 |
|------|--------|
| 产能变化 | 座席数是否减少、有效出勤率是否<85% |
| 人效变化 | 人均GMV趋势、转化率变化 |
| 人员结构 | 新员工占比（>30%要注意）、离职率 |
| 分量变化 | 上月有没有调过分量、产线有没有切换 |
| 质量数据 | 质检得分、客诉率是否异常 |

### Step 5：提炼下月重点（10min）

控制在3-5条：
- 需要约谈的供应商
- 需要调分量的供应商
- 需要重点帮扶的供应商
- 需要向上汇报的风险
- 产线层面的策略调整

### Step 6：写复盘叙事（5min）

用叙事模板串成故事，不是堆数字。
