# 分量触发条件

> 分量增加和减少的触发条件

## 分量增加触发条件

| 触发条件 | 说明 |
|---------|------|
| 业务增长 | 整体业务增长，产能不足 |
| 新增业务线 | 开辟新业务线 |
| 战略调整 | 业务策略调整导致部分供应商业务量增加 |
| 供应商表现优异 | 连续考核优秀、产能提升 |

## 分量减少触发条件

| 触发条件 | 说明 |
|---------|------|
| 业务量收缩 | 整体业务量下降 |
| 绩效惩罚 | 连续考核不达标或排名末位 |
| 风险管控 | 出现一般性违规或潜在风险 |
| 策略调整 | 业务策略调整 |

## 分量调整审批

| 调整幅度 | 审批要求 |
|---------|---------|
| <50% | 供应商管理岗发起 → C3审批 |
| ≥50% | 标准流程 + C2终审 |
| ≥100%（翻倍） | 专项汇报 → C2 |
| ≤10人（快速调整） | 5工作日内完成，向C3京ME备案，无需事前审批 |
