# 季度九宫格判定

> 月度触发规则是"体检雷达"，季度九宫格是"综合诊断"

## 判定流程

```
季度末数据汇总
    │
    ▼
计算最终得分 = 赛马基础分(0-100) - 合规扣分 + 稳定性得分(+4~-4) + 供管自评(+0~+5)
    │
    ▼
X轴：按得分排名分三档（前30%高贡献 / 中40%中贡献 / 后30%低贡献）
Y轴：环比趋势（上升>10% / 持平-10%~+10% / 下降>10%）
    │
    ▼
九宫格定位 → ABC档位 → 管理动作
```

## 九宫格速查表

| | 趋势下降↓ | 趋势持平→ | 趋势上升↑ |
|--|-----------|-----------|-----------|
| **高贡献（前30%）** | A-（需稳住） | A（优秀） | A+（优秀） |
| **中贡献（30%-70%）** | B-（需关注） | B（合格） | B+（合格） |
| **低贡献（后30%）** | **C（淘汰→快车道清退）** | C（不合格→30天整改） | C（不合格→30天整改） |

## 关键判定规则

| 规则 | 说明 |
|------|------|
| X轴决定档位 | 按得分排名分三档，非绝对分数线 |
| Y轴决定动作 | 趋势下降→需稳住/关注/清退；趋势上升→优秀/合格/整改 |
| 低贡献=无论趋势如何 | 均为C档 |
| 新供应商保护期 | 入项不足2季度不参与九宫格 |
