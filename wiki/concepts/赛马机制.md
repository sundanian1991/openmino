# 赛马机制

> 通过数据排名公开化，刺激供应商自我竞争

## 核心规则

| 规则 | 说明 |
|------|------|
| **月度排名** | 按五维总分排名 |
| **公开透明** | 排名数据对供应商公开 |
| **优胜劣汰** | 末位淘汰机制 |
| **动态调整** | 新供应商1-3个月观察期，不参与排名 |

## 实施要点

| 要点 | 说明 |
|------|------|
| **排名维度** | 单维度排名 + 综合排名，多维度对比 |
| **对比基准** | 不只看绝对值，看相对大盘/同类供应商表现 |
| **反馈机制** | 排名变化时主动告知供应商原因 |
| **激励联动** | 排名与分量/价格/新业务挂钩 |
| **避免内卷** | 设定合理目标，防止恶性竞争 |
| **业绩预警** | 单个供应商连续3个工作日核心转化率低于团队平均基准值20%以上 |
| **测试资源** | 赛马期间应为所有参与方分配同质化、等量的数据包，确保起跑线公平。数据量需满足统计学意义（**不少于5000条有效号码**） |
| **豁免条件** | 当新增资源或专项测试的业务量过小，不具备统计学意义时，可豁免赛马。具体标准为：**预计月度业务结算低于5万元** |
