# 自运转飞轮

> 供应商联盟从运营官驱动转向机制驱动的月度自运转节奏

## 月度SOP

```
每月25日：下月轮值供应商确认分享主题并提交提纲
每月28日：运营官审核提纲，提供反馈（15分钟）
每月5日：  线上圆桌（主持人=轮值供应商，运营官仅开场与收尾）
每月7日：  当期纪要由主持供应商输出并归档
每月10日：下下月选题确认并公布
```

## 运营官月度时间分配

| 动作 | 耗时 |
|------|------|
| 审提纲+反馈 | 15分钟 |
| 圆桌开场+收尾 | 6分钟 |
| 行业内容精选 | 30分钟 |
| 群内互动与问题回应 | 30分钟 |
| 临时沟通与协调 | 30分钟 |
| **月合计** | **约2小时** |

## 自治目标

Q1→Q3 三阶段：冷启动（运营官驱动）→ 扩圈（流程固化）→ 自治（机制驱动）。
最终状态：运营官月均投入约2小时，其余由供应商和标准化流程承担。
