# B6 降价谈判准备

> 谈价格之前，先把底牌想清楚——自己的底线、对方的痛点、可交换的筹码

## 触发信号

- 年度/季度合同续签需重新议价
- 供应商成本结构明显变化
- 策略组反馈某产线单客成本偏高
- 王易人要求优化供应商成本
- 新供应商引入导致竞争格局变化
- 供应商主动提涨价（需要反制）

## 思考步骤

### Step 1：确认谈判必要性

| 供应商类型 | 是否值得谈 |
|-----------|----------|
| 标杆型（前30%） | 谨慎——砍价容易伤关系 |
| 中等（排名中间） | 可以谈——有替换空间 |
| 问题型（后30%） | 换思路——不是降价问题是是否继续合作 |

### Step 2：摸清成本结构

| 计价方式 | 降价切入点 | 谈判策略 |
|---------|-----------|---------|
| 按件计费 | 提量换降价 | "加20%的量，单价降5%" |
| 按人计费 | 提效换降价 | "人均GMV达XX后，管理费降Y%" |
| 阶梯价 | 看是否跨了阶梯 | "已进第二阶梯，应用第二阶梯价" |

### Step 3：盘点[[谈判筹码清单]]

| 筹码类型 | 供应商在乎程度 |
|---------|--------------|
| 分量承诺（保证未来N月量不减少） | 高 |
| 增量机会（降价可以给更多分量） | 高 |
| 新产线机会（优先给新产线份额） | 中 |
| 结算周期（缩短） | 高 |
| 排名数据（暗示不降价可能调分量） | 中 |
| 竞品报价（匿名） | 中 |

### Step 4：确定谈判目标区间

| 目标 | 设定方法 |
|------|---------|
| 理想目标 | 同产线均价再低5% |
| 可接受目标 | 同产线均价水平 |
| 底线 | 现状价格降5%且不附带额外条件 |

### Step 5：准备谈判话术框架

- 开场（5min）：先肯定合作→再点明目的→让对方先表态
- 推进：先问对方想法，对方报数字后再还价；每次让步换对方让步
- 收尾（5min）：能确认就确认；不能确认约定时间节点

### Step 6：准备B计划

| 情况 | B计划 |
|------|------|
| 拒绝降价但愿其他让步 | 接受非价格让步（结算周期、服务条款） |
| 拒绝所有让步 | 不续签或减量 |
| 强硬且是唯一供应商 | 先稳住，同时加速培养替代 |
| 要求涨价 | 拿数据和排名说话 |

### Step 7：内部确认

- 降价目标区间→王易人确认
- 分量承诺→王易人+策略组确认
- 结算周期调整→财务确认
- 替代方案→王易人确认
