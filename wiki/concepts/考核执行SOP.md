# 考核执行SOP

> 月度考核流程和报告模板

## 月度考核流程

```
每月1日   → 提取上月数据
每月5日   → 完成自动评分
每月8日   → 人工复核
每月10日  → 结果确认、通知供应商
每季度    → 综合评级、红黄牌判定
```

## 考核报告模板

```
【XX月供应商考核报告】

一、总体情况
- 参评供应商数量
- 各等级分布
- 与上月对比

二、各供应商得分
| 供应商 | 产能 | 效率 | 品质 | 配合度 | 合规 | 总分 | 等级 |

三、异常情况
- 排名大幅下降
- 单项指标异常
- 红黄牌触发

四、重点关注
[需重点关注的供应商及原因]

五、下月重点
[管理动作计划]
```
