# A1 业绩异常判断

> 看到信号后，判断严重程度、定位原因、决定下一步

## 触发信号

| 信号类型 | 具体表现 |
|---------|---------|
| 排名下降 | 月度排名落入后30%，或下降≥5名 |
| 周度断崖 | 本周GMV比近4周均值下降>20% |
| 峰值比低 | 自身峰值利用率<70% |
| 头部差距大 | 与头部供应商峰值利用率差距>15pp |
| A级骤降 | 上月前30%本月跌到后30% |
| 出勤异常 | 有效出勤率<85% |
| 流失加速 | 单周离职超过5人 |

## 思考步骤

**Step 1：确认信号真实性**
- 检查数据质量（口径、完整性、时效性）
- 排除统计口径变化、结算周期等"假异常"

**Step 2：判断异常级别**

| 状态 | 标准 | 级别 |
|------|------|------|
| 首次预警 | 本月落入后30%，上月不在 | 黄色 — 关注 |
| 连续预警 | 连续2个月在后30% | 橙色 — 需干预 |
| 清退评估 | 连续3个月在后30% | 红色 — 需决策 |
| A级骤降 | 上月前30%→本月后30% | 红色 — 立即行动 |

**Step 3：定位原因维度**

| 维度 | 检查数据 | 判断 |
|------|---------|------|
| 产能 | 有效出勤率、座席数变化、新员工占比 | 人够不够？ |
| 人效 | 人均GMV、转化率、与头部对比 | 人效掉没掉？ |
| 质量 | 质检得分、客诉率 | 以量换质？ |
| 业务 | 分量调整、产线切换、产品政策变化 | 外部因素？ |
| 人事 | 离职率、招聘进度、培训投入 | 团队稳不稳？ |

**Step 4：产能利用率定性（[[产能利用率双维度判定]]）**

**Step 5：检查周度突变信号**
- 有效出勤率<85% / GMV降>20% / 单周离职>5人
- 0个→常规处理；1个→本周内沟通；≥2个→48h内沟通

**Step 6：决定下一步**

| 级别 | 动作 | 权限 |
|------|------|------|
| 黄色 | 私下沟通问原因 | 自主执行 |
| 橙色 | 正式约谈，要求整改计划 | 自主执行，事前报备王易人 |
| 红色 | 评估延长或启动清退 / 立即约谈 | 建议权，王易人确认 |
