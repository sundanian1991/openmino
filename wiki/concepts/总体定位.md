# 总体定位

> 构建完整的供应商管理数字人体系

## 定位说明

向职场提供可咨询、可记录、可提醒的供应商管理Agent。

| 维度 | 说明 |
|------|------|
| 范围 | 32+家供应商，3200+人，4大产线（金条/企金/财富/信用卡）、17条细分业务 |
| 初期试跑 | 金条12家 |
| 维护人 | 孙大年 |
| 月工作总量 | ~196h |
| AI可提效 | ~83.5h（43%） |

## 角色转变

从一线操作工到指挥官：数字人处理80%日常运营，人处理20%异常+持续优化系统。

释放时间投向：供应商寻源（+5h/周）、站点布局（+4h/周）、BPO联盟（+4h/周）、深度关系建设（+4h/周）、向上管理（+2.5h/周）、制度体系化（+1.5h/周）。
