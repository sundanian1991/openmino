# 经营分析五类模糊场景

> 通用分析框架：将模糊问题映射到五类场景，每类有对应的思考路径和表达方式

## 五类场景

1. **范围模糊**（"不知道从哪下手"）→ Issue Tree + 二维矩阵 → 五步叙事
2. **路径模糊**（"知道去哪不知道怎么到"）→ 目标画像 + 多维评估 → 对比叙事
3. **优先级模糊**（"事太多先做什么"）→ 评分 + 三层分类 + 甘特图 → 漏斗叙事
4. **因果模糊**（"出了问题不知道为什么"）→ 鱼骨图 + 5-Why + 证据排除 → 侦探叙事
5. **冲突模糊**（"各方诉求不一致"）→ 利益地图 + 三层归因 + trade-off → 桥梁叙事

## 核心原则

- 每个判断交代理由
- 标定边界（什么条件下成立，什么条件下不成立）
- 三角验证（定量+定性+逻辑）
- 排除过程比结论更有说服力
