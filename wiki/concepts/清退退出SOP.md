# 清退退出SOP

> 供应商清退的完整流程

## 流程步骤

```
1. 触发评估（72小时内）
   └─ 确认触发条件成立
   └─ 收集证据材料

2. 替代方案（5个工作日内）
   └─ 评估业务影响
   └─ 制定切量方案
   └─ 确认替代供应商

3. 正式通知
   └─ 书面通知供应商
   └─ 明确退出原因和时间表

4. 数据回收
   └─ 客户资料
   └─ 业务数据
   └─ 录音等资料

5. 费用清算
   └─ 结算至退出当日
   └─ 预留金处理

6. 系统关闭
   └─ 账号注销
   └─ 权限回收
   └─ 系统接入关闭

7. 归档
   └─ 退出报告
   └─ 数据归档
   └─ 经验总结
```
