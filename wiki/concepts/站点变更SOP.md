# 站点变更SOP

> 同城市迁址、新增站点、站点缩减的完整流程

## 场景一：同城市迁址

```
1. 供应商提交变更申请
2. 供应商管理岗审核新址合规性
   └─ 商业用途写字楼
   └─ 非禁设区域
   └─ 产权/租赁证明
   └─ 消防验收合格文件
3. 新址现场核查（关键）
4. 决策
   ├─ 审批通过 → 京音平台更新地址
   └─ 审批驳回 → 明确驳回理由
```

## 场景二：新增站点（新城市）

```
1. 供应商提交新增站点申请
2. 供应商管理岗初审
3. 资源规划岗评估需求匹配度
4. 业务运营岗评估业务承接能力
5. C3审批
6. C2终审（涉及特殊资源/大额投入）
7. 通过后系统配置、人员培训
```

## 场景三：站点缩减

```
1. 供应商提交缩减申请（或管理方触发）
2. 业务影响评估
3. 人员安置方案
4. 数据迁移计划
5. C3审批
6. 执行缩减
7. 后续跟踪
```
