## 一、五维考核评分模型

> [来源：方案框架.md §五维考核 / SM-2026-040_供应商管理SLA.md §考核维度]

### 1.1 维度权重

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| 维度 | 权重 | 说明 |
|------|------|------|
| **业绩** | 35% | GMV、人均产出、增长率 |
| **合规** | 20% | 质检合格率、违规次数、红线触碰 |
| **配合度** | 20% | 响应速度、信息透明度、整改配合 |
| **组织健康** | 15% | 流失率、出勤率、培训完成率 |
| **成长性** | 10% | 产能爬坡速度、新产线适应度 |

> **清洁版赛马机制权重范围参考**（SM-2026-050 §第八章）：过程质量权重范围**15%-25%**，客户体验权重范围**5%-15%**。实际执行中，策略组可根据业务特点在此范围内调整具体权重。

### 1.2 月度赛马评分（季度评估的输入）

> [来源：SM-2026-035_供应商分级清退管理制度.md §第三条 / 方案框架.md §赛马机制]
> ⚠️ **重要变更**：月度不再输出ABCD评级。月度100分制得分在季度末取3个月算术平均作为"赛马基础分"，进入季度九宫格评估。

| 评分区间 | 月度定位 | 季度映射 |
|---------|---------|---------|
| ≥85 | 表现优秀 | 大概率进入A档（高贡献） |
| 75-84 | 表现良好 | 大概率进入B档（中贡献） |
| 65-74 | 需要关注 | 可能进入B-或C档 |
| <65 | 严重不达标 | 大概率进入C档（低贡献） |

### 1.3 红线一票处理

> [来源：SM-2026-035_供应商分级清退管理制度.md §第三章 / 方案框架.md §红线]

| 红线行为 | 处理 |
|---------|------|
| 数据泄露/泄露客户信息 | **直接评级C**（季度评估中不经过计算） |
| 录音篡改/伪造 | **直接评级C** |
| 重大合规事件 | **直接评级C** |
| 红线违规 | 立即启动清退评估 |

## 三、赛马机制

> [来源：方案框架.md §赛马机制 / SM-2026-050_清洁版操作指引.md §赛马]

### 3.2 核心规则

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| 规则 | 说明 |
|------|------|
| **月度排名** | 按五维总分排名 |
| **公开透明** | 排名数据对供应商公开 |
| **优胜劣汰** | 末位淘汰机制 |
| **动态调整** | 新供应商1-3个月观察期，不参与排名 |

### 3.3 实施要点

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| 要点 | 说明 |
|------|------|
| **排名维度** | 单维度排名 + 综合排名，多维度对比 |
| **对比基准** | 不只看绝对值，看相对大盘/同类供应商表现 |
| **反馈机制** | 排名变化时主动告知供应商原因 |
| **激励联动** | 排名与分量/价格/新业务挂钩 |
| **避免内卷** | 设定合理目标，防止恶性竞争 |
| **业绩预警** | 单个供应商连续3个工作日核心转化率低于团队平均基准值20%以上 |
| **测试资源** | 赛马期间应为所有参与方分配同质化、等量的数据包，确保起跑线公平。数据量需满足统计学意义（**不少于5000条有效号码**） |
| **豁免条件** | 当新增资源或专项测试的业务量过小，不具备统计学意义时，可豁免赛马。具体标准为：**预计月度业务结算低于5万元** |

## 六、两套体系的关系说明

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| 维度 | 五维考核 | 赛马机制 | SLA |
|------|---------|---------|-----|
| **定位** | "得多少分"——量化评估与排名 | "资源怎么分"——竞争分配 | "能不能接受"——服务质量底线 |
| **触发** | 每月固定评估 | 新增资源/定期评估/策略变更/申诉 | 日常监控 |
| **逻辑** | 按档打分，不考虑连续性 | 排名竞争，优胜劣汰 | 连续不达标才触发管理动作 |
| **输出** | 100分制得分（季度评估输入） | 分量调整+末位淘汰 | 预警/红线/清退 |
| **使用者** | 供应商管理岗（你） | 策略组+供应商管理岗 | 供应商管理岗+供应商 |

> **关键区分示例**：产能达标率82%
> - SLA视角：<90%即不达标，82%触发**预警**（"需要整改"）
> - 评估方案视角：80-89%得**8分**（"扣2分，但仍有一定产出"）
> - 赛马视角：如果同产线其他供应商平均85%，82%排名靠后→**减量**

## 八、季度评估体系（九宫格ABC）

> [来源：SM-2026-035_供应商分级清退管理制度.md §第二章~第三章]  
> 核心定位：月度赛马是"日常体检"，季度九宫格是"综合诊断"，半年度盘点是"战略校准"

### 8.1 评估频率三层体系

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| 评估类型 | 频率 | 定位 | 输出 |
|---------|------|------|------|
| **月度评估** | 每月 | 日常考核（赛马100分制） | 赛马排名、预警/整改触发 |
| **季度评估** | 每季度末 | 供应商综合评级（九宫格，ABC三档） | 三张清单（保留/培养/优化） |
| **半年度盘点** | 6月、12月 | 深度盘点：战略意义、长期潜力、风险敞口 | 最终决策校准 |

### 8.2 季度评估维度（三维评分+加分项）

> [来源：SM-2026-035_供应商分级清退管理制度.md §季度评估]

```
最终得分 = 赛马基础分 - 合规扣分 + 稳定性得分 + 供管自评加分
```

| 维度 | 说明 | 分值区间 | 数据来源 |
|------|------|---------|---------|
| **赛马基础分** | 季度内三个月月度评估总分的算术平均值 | 0-100 | 月度评估结果汇总 |
| **巡检合规扣分** | 季度内SLA红线/黄牌事件累计扣分 | 0~-∞ | SLA记录+巡检台账 |
| **稳定性得分** | 季度内交付稳定性量化评分 | +4~-4 | 检查表逐项勾选 |
| **供管自评加分** | 战略协作、响应质量、高效执行、共同成长、坦诚透明 | +0~+5 | 供管季度评估 |

**计分规则**：
- 最低分为0，不因扣分出现负分
- 最高分可达110分（赛马100 + 稳定性+5 + 自评+5），不设上限
- 九宫格按**排名分档**而非绝对分数线，高分不影响档位判定

**合规扣分与月度评估的关系**：
- 月度评估中的"合规表现"维度（20分）已对质检合规事件进行了评分
- 季度评估中的合规扣分针对的是**公司级重大事件**（黄牌整改函、红线违规、严重客户投诉等），与月度站点合规**不重复**
- 判定依据：黄牌整改函以巡检正式发文为准，红线违规以SLA触发为准，投诉/一般违规以质检月报异常清单为准

### 8.3 合规扣分标准

> [来源：SM-2026-035_供应商分级清退管理制度.md §合规扣分]

| 事件类型 | 扣分 | 说明 |
|---------|------|------|
| 黄牌整改 | -5分/次 | 巡检发现问题并下发整改函 |
| 舆情有责投诉 | -5分/次 | 重大舆情有责客诉 |
| 红线违规 | **直接评级C** | 安全事件、数据泄露、严重合规违规 |

### 8.4 稳定性评定标准（四项检查表）

> [来源：SM-2026-035_供应商分级清退管理制度.md §稳定性评定]

> 避免"满足任一即降级"的主观判断，逐项打分，系统自动计算。

| 检查项 | A档（+1分） | B档（0分） | C档（-1分） |
|--------|-----------|-----------|-----------|
| **人员变动** | 无大幅变动（流失≤10%，无核心人员变动） | 流失>10%或单月新增>15%新人 | 核心人员离职（负责人/主管级） |
| **产能波动** | 月度产能波动<10% | 月度产能波动10%-20% | 月度产能波动>20%或断崖式下降 |
| **SLA达成** | 季度SLA全部达成 | 季度SLA 1次未达成 | 季度SLA 2次以上未达成 |
| **突发情况** | 无突发情况 | 轻微影响交付（可24小时内恢复） | 严重影响交付（>24小时无法恢复） |

**计算规则**：四项得分相加即为稳定性得分。
- 四项均A = +4分
- 四项均B = 0分
- 四项均C = -4分
- 混合档位按各档项数乘以对应分值相加

**示例**：人员变动A(+1)、产能波动B(0)、SLA达成A(+1)、突发情况B(0) → 稳定性得分 = +2分

### 8.5 供管自评加分标准

> [来源：SM-2026-035_供应商分级清退管理制度.md §供管自评]

> 五项各评"高/中/低"，高=1分，中低=0分。每项必须填写**具体事实依据**，无事实依据的不给"高"。

| 维度 | 高=1分的触发条件 | 事实依据示例 |
|------|----------------|-------------|
| **战略协作** | 主动提出建设性建议、参与规划讨论、有效帮助业务发展 | 主动提出XX建议/参与XX规划讨论 |
| **响应质量** | 24小时内响应、反馈有实质内容 | X月X日XX需求X小时内响应，方案可行 |

## 九、九宫格映射与ABC分级

> [来源：SM-2026-035 §第三章]

### 9.1 九宫格结构

> [来源：SM-2026-035_供应商分级清退管理制度.md §九宫格]

**X轴（贡献度，按最终得分排名）**：

| 档位 | 定义 | 占比 |
|------|------|------|
| 高贡献 | 得分排名在前30% | 四舍五入取整 |
| 中贡献 | 得分排名在30%-70% | 四舍五入取整 |
| 低贡献 | 得分排名在后30% | 四舍五入取整 |

**Y轴（环比趋势）**：

| 趋势 | 定义 |
|------|------|
| 趋势上升↑ | 环比得分增长>10% |
| 趋势持平→ | 环比变化在-10%~+10%之间 |
| 趋势下降↓ | 环比得分下降>10% |

### 9.2 ABC评级映射

> [来源：方案框架.md §相关章节 / SM-2026-040_供应商管理SLA.md §相关章节 / SM-2026-050_清洁版操作指引.md §相关章节 / SM-2026-035_供应商分级清退管理制度.md §相关章节]

| | 趋势下降↓ | 趋势持平→ | 趋势上升↑ |
|--|-----------|-----------|-----------|
| **高贡献** | A-（需稳住） | A（优秀） | A+（优秀） |
| **中贡献** | B-（需关注） | B（合格） | B+（合格） |
| **低贡献** | C（淘汰） | C（不合格） | C（不合格） |

**核心规则**：
- **X轴决定档位**（A/B/C），**Y轴决定后续管理动作**
- 低贡献（后30%）无论趋势如何，均为C档
- 新供应商保护期：入项不足2个季度的供应商不参与九宫格映射，单独列为"观察区"

### 9.3 ABC分级管理动作

> [来源：SM-2026-035_供应商分级清退管理制度.md §ABC分级]

#### A级（优秀）
> [来源：SM-2026-035_供应商分级清退管理制度.md §附录四 A级管理动作]


| 管理维度 | 具体动作 | 频次 |
|---------|---------|------|
| 沟通 | 服务组负责人月度1对1沟通，季度高层拜访 | 月度+季度 |
| 份额 | 优先获得增量分配，新业务试点优先入围 | 按需 |
| 问题响应 | 专属对接群，问题优先内响应 | 实时 |
| 认可 | 季度评优优先候选 | 季度 |
| 约束 | 纳入集中度管控（单一≤70%，合计≤90%） | 持续 |

#### B级（合格）
> [来源：SM-2026-035_供应商分级清退管理制度.md §附录四 B级管理动作]


| 管理维度 | 具体动作 | 频次 |
|---------|---------|------|
| 沟通 | 供管季度例行沟通，有异常时专项约谈 | 季度 |
| 份额 | 表现优秀可获适度增量 | 按需 |
| 培养 | B+趋势上升供应商纳入重点培养名单，给予资源倾斜 | 季度 |
| 改进 | B-趋势下降供应商需提交书面改善计划，月度跟踪 | 月度 |

#### C级（不合格）
> [来源：SM-2026-035_供应商分级清退管理制度.md §附录四 C级管理动作]


| 管理维度 | 具体动作 | 频次 |
|---------|---------|------|
| 整改 | C+趋势持平/上升：30天整改期，提交改善计划 | 一次性+月度跟踪 |
| 清退 | **C+趋势下降：当季启动快车道清退** | 立即 |
| 份额 | 逐步缩减产能，不再新增份额 | 持续 |
| 沟通 | 约谈供应商负责人，书面记录 | 月度 |
| 替代 | 同步启动储备库替代供应商匹配 | 整改期内 |

#### 观察区（新供应商保护期）
> [来源：SM-2026-035_供应商分级清退管理制度.md §第九条 观察区 / §附录四 观察区]


| 管理维度 | 具体动作 | 频次 |
|---------|---------|------|
| 记录 | 正常评分和趋势记录，不映射九宫格 | 季度 |
| 带教 | 按冷启动SOP执行30/60/90天跟进培育 | 按SOP |
| 保护 | 不触发清退，不缩减份额 | 保护期内 |

### 9.4 清退触发路径（季度视角）

> [来源：SM-2026-035_供应商分级清退管理制度.md §清退触发]

```
季度评估结果
    │
    ├─ C档 + 趋势下降 → 【当季启动快车道清退】
    │
    ├─ C档 + 趋势持平/上升 → 30天整改期 → 整改未通过 → 启动清退
    │                                           │
    │                                           └─ 整改通过 → 移出C档
    │
    └─ 红线违规 → 直接评级C + 立即启动清退评估
```

> **与月度触发规则的衔接**：月度"连续3月后30%"和季度"C档+趋势下降"是**互补而非替代**。月度触发用于及时发现异常并干预，季度九宫格用于综合判定供应商生态位。实践中，月度连续预警的供应商大概率会在季度评估中落入C档。
