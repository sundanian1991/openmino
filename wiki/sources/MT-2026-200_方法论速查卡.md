# 方法论速查卡

> 一页纸速查

## 速查内容

```
【看到排名异常】
  Step1: 9状态判定（N/Y1/Y2/E/R/A/H1/H2/T）→ [[触发规则表（状态机）]]
  Step2: 双维度矩阵（自身峰值比×头部差距）→ [[产能利用率双维度判定]]
  Step3: 主管胜任5项检查 → [[主管胜任评估5项指标]]
  Step4: 异常信号31个交叉验证 → [[异常信号31个]]
  Step5: 输出动作（A1-A6 SOP）

【看到数据存疑】
  5项信号检查 → [[数据真实性防控]]
  造假类型识别 → [[数据真实性防控]]

【整改后验收】
  4维度验证（目标/动作/持续/根因）→ [[整改闭环验证标准]]

【季度评估后】
  Step1: 计算最终得分（赛马基础分-合规扣分+稳定性+自评加分）→ [[季度评估体系（九宫格ABC）]]
  Step2: 九宫格映射（X轴贡献度排名×Y轴环比趋势）→ [[季度九宫格判定]]
  Step3: ABC档位判定 + 管理动作选择 → [[ABC分级管理动作]]
  Step4: C档+趋势下降 → 触发评估（不一定立即清退）→ [[清退规则]]
```
