# 供应商管理工作手册-基于HRBP方法论

> 基于HRBP方法论的系统化供应商管理框架

## 六模块全景

```
战略对齐 → 组织诊断 → 赋能提升 → 动态调度 → 梯队建设 → 数据驱动
  (懂业务)   (看真相)   (帮成长)   (调资源)   (建梯队)   (做决策)
```

## 核心内容

1. 战略对齐 — 从"管供应商"到"懂业务的供应商管理者"
2. 组织诊断 — 供应商九宫格与多维评估
3. 赋能提升 — 供应商成长支持体系
4. 动态调度 — 资源分配与调整策略
5. 梯队建设 — 供应商梯队规划
6. 数据驱动 — 数据驱动的决策体系
