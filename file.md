请使用 show-widget 工具生成一个供应商 KPI 管理仪表盘，遵循以下设计规范：

### 设计原则
- **克制美学**：拒绝渐变、阴影、圆角，追求极简高级感
- **留白呼吸感**：模块间距 32-40px，内边距充足
- **单色配色**：仅用黑白灰，通过深浅建立层级
- **信息架构清晰**：每个分区有明确的标题标签

### 配色规范（严格限制）
- 主文字：#111
- 次要文字：#666  
- 辅助文字：#999
- 背景/白：#fff
- 分割线/网格线：#e5e5e5 或 #ccc
- 成功绿（仅用于涨幅）：#2a9d5c
- 警告红（仅用于跌幅）：#d73a49

### 布局结构
1. **Header 区**
   - 顶部边框分隔（border-bottom: 2px solid #111）
   - 小标题标签：KPI DASHBOARD（11px, #666, 字间距 0.5px）
   - 主标题：供应商业绩概览（28px, font-weight: 600）
   - 副标题：年月 + 覆盖范围说明（13px, #666）

2. **核心指标区**
   - 分区标题：核心指标（12px, #666, font-weight: 600, 字间距 0.5px）
   - 4 列网格（grid-template-columns: repeat(4, 1fr)）
   - 每个指标卡片：左侧 3px 实线边框（border-left: 3px solid #111）
   - 指标名：12px, #666
   - 数值：32px, font-weight: 600, #111
   - 环比：11px，绿色(#2a9d5c)或红色(#d73a49)

3. **趋势图表区**
   - 分区标题：趋势分析
   - 使用原生 SVG 绘制（禁止 Chart.js 等外部库）
   - 必须包含：
     - Y 轴刻度标签（如 0, 50, 100, 150, 200）
     - X 轴月份标签（1月-12月）
     - 淡色网格线（stroke: #e5e5e5）
     - 多条折线用不同深浅灰区分（#111, #666, #999）
     - 图例在图表下方水平排列

4. **供应商排名表**
   - 分区标题：供应商排名
   - 表头：2px 底部边框（border-bottom: 2px solid #111）
   - 表格行：1px 底部边框（border-bottom: 1px solid #e5e5e5）
   - 排名列：圆形徽章，黑底白字（background: #111, color: #fff）
   - 评级标签：反色块（A级=黑底白字，B级=白底黑框，C级=灰底）
   - 趋势箭头：↑ ↓ → 配合颜色

### 字体规范
- 字体栈：-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
- 标题字重：600
- 正文：400
- 小号标签：11-12px

### 技术约束
- 仅使用内联 style，禁止外部 CSS 或 class
- SVG 手动计算坐标点绘制 path 和 line
- 容器最大宽度：1200px，水平居中
- 每个元素单独设置 style，不继承
