## 核心职责：供应商选用育留汰

### 选：供应商引入

**目标**：找到合适的供应商，建立合作关系

| 流程 | 关键动作 |
|------|----------|
| **寻源** | 内部需求对接、潜在供应商筛选 |
| **初筛** | 9维度桌面调查（过往业绩、同类型对标、管理方式、人员数量、公司背景、历史交付风格、配合商务、团队架构、资质合规） |
| **准入** | 8阶段流程（寻源→桌面调查→现场考察→评审决策→试运营→正式准入→6个月里程碑→评判点） |
| **跟踪** | 6个月里程碑（引入期→成长期→稳定期→评判点） |

**关键文件**：`concepts/准入评估SOP.md`、`concepts/B1 新供应商准入.md`、`concepts/供应商深度调研方法论.md`

### 用：日常管理

**目标**：确保供应商稳定运营，持续产出价值

| 流程 | 关键动作 |
|------|----------|
| **月度复盘** | 每月1-5号，用1小时把上月数据变成下月决策依据 |
| **巡检** | 出差巡检12项标准检查清单，查出问题有整改 |
| **分量调整** | 动态分配业务量，保持供应商期待 |
| **站点变更** | 同城市迁址、新增站点、站点缩减 |
| **向上汇报** | 周报/月报，穿透式汇报结构 |

**关键文件**：`concepts/日常管理规范.md`、`concepts/B4 月度复盘.md`、`concepts/巡检SOP.md`

### 评：评估考核

**目标**：用数据说话，实现从"人盯人"到"体系管人"

| 维度 | 内容 |
|------|------|
| **五维考核** | 业绩(35%)、合规(20%)、配合度(20%)、组织健康(15%)、成长性(10%) |
| **红黄牌机制** | 单项指标连续2期低于警戒线→黄牌；连续3期/整改不达标/触碰红线→红牌 |
| **赛马机制** | 月度排名、公开透明、优胜劣汰、动态调整 |
| **季度九宫格** | 赛马基础分 - 合规扣分 + 稳定性得分 + 供管自评加分 |
| **三层评估** | 月度评估（日常体检）→季度评估（综合诊断）→半年度盘点（深度盘点） |

**关键文件**：`concepts/评估与考核体系.md`、`concepts/五维考核评分模型.md`、`concepts/红黄牌机制.md`、`concepts/季度评估体系（九宫格ABC）.md`

### 汰：清退管理

**目标**：让供应商心甘情愿自己申请，不闹不撕

| 流程 | 关键动作 |
|------|----------|
| **触发** | 连续考核不达标（连续两期D级）、触碰红线、整改不合格、主动退出、业务策略调整 |
| **评估** | 启动评估(72小时内) |
| **替代** | 替代方案(5个工作日内) |
| **退出** | 正式通知→数据回收→费用清算→系统关闭→归档 |

**退出原则**：有据可依、风险隔离、平稳过渡

**关键文件**：`concepts/清退退出SOP.md`、`concepts/A6 清退评估.md`、`concepts/清退规则.md`

## 关键案例

### 案例1：宿迁自建（最大成就）

**背景**：老板在京东立足的核心项目，如果做不成整个组的价值会被审视

**动作**：
- 团队设置、绩效、结构设计
- 亲自面试近200人，百里挑一选20人
- 在宿迁待一个月，高频争取、协调

**结果**：
- 费效进入健康区间（1.2倍以内）
- 产能环比+110%（vs BPO 45%）
- 成为供应商降价的对标尺

---

### 案例2：降价谈判（直接价值）

**背景**：自建产能+110%，BPO+45%，BPO结算成本有优化空间

**动作**：
- 9月拿着自建数据去谈
- "自建能做到110%增速，你们45%"
- 所有供应商一视同仁降价

**结果**：
- 单价从0.15%降至0.1275%（降幅15%）
- 当月降本100万（671万→571万）

---

### 案例3：线路应急（主动补位）

**背景**：核心业务线路突发中断风险，日影响1300万呼叫、37亿周GMV

**动作**：
- 主动补位承担应急统筹
- 快速找到近10家带指标的供应商
- 每天复盘客诉数据，持续25-30天

**结果**：
- 15天完成恢复（行业30天）
- 避免约80亿GMV损失
- 预案成为部门标准流程

## 管理哲学

### 培养悖论定律

培养供应商的必然路径：小 → 给资源 → 做大 → 向外扩张 → 有退路 → 议价/流失

**应对策略**：
| 策略 | 做法 |
|------|------|
| 梯队备份 | 始终培育第二梯队，保持替代选项 |
| BPO联盟 | 把自营培训体系交给中腰部供应商 |
| 新业务优先 | 新业务分配优先给中腰部，不让头部一家独大 |
| 差异化定价 | 头部用市场价，中腰部给成长空间 |

### 核心产出定位

从"做事情"到"建体系"：
- **不是**写文档、做PPT、跑数据
- **而是**建机制、立规矩、形成可复制的方法论
