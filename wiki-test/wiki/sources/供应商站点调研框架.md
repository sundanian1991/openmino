# 供应商站点调研框架

> 来源：年老师工作区，2026-06-09 创建

## 核心目标
验场 + 业务诊断 + 竞品了解 → 产出汇报文档，推动调整

## 四大模块

### 1. 调研前准备
- 材料：近6个月业绩数据、竞品产品要素表、合同/结算标准
- 人员：供应商老板、现场负责人、一线坐席、培训/质检岗

### 2. 验场（站点验收）
- 硬件环境：职场面积、电脑/耳麦、网络、录音监控、消防
- 团队情况：在岗人数、入职/离职、平均工龄、管理层级
- 合规基础：话术合规、信息保护、培训记录、质检执行

### 3. 业务诊断（核心）
- 数据摸底：人均日拨打量、有效通话量、接通率、转化率
- 问题诊断6问：产能变化、环节定位、共性vs个性、名单变化、产品变化、供应商归因
- 减量诉求评估：量化亏损、固定vs变动成本、盈亏平衡线
- 一线坐席访谈：验证管理层数据、产品竞争力、培训有效性

### 4. 竞品了解
- 产品对比：利率、额度、期限、准入、审批时效
- 电销运营对比：拨打量、接通率、转化率、结算标准
- 关键问题：产品竞争力、名单质量、结算标准、合规要求

## 输出物
1. 调研概况
2. 站点验收结论
3. 业务诊断（数据+问题+根因）
4. 竞品对比
5. 减量诉求评估
6. 行动建议
7. 附录
