COMPARISON

传统 UI vs 电影 UI

TRADITIONAL

传统 UI 范式

  • 组件驱动的网格系统 - 所有页面共享同一套布局
  • 功能优先的信息架构 - 效率最大化，感受最小化
  • 标准化的视觉模板 - 一致但平庸
  • 进度条和数字罗列 - 数据可见但不可感
CINEMATIC

电影 UI 范式

  • 叙事驱动的视觉节奏 - 每个页面有独特的起承转合
  • 情感优先的信息层级 - 三层金字塔引导注意力
  • 自然色板身份系统 - 颜色传达情绪，不是装饰
  • 眯眼测试验证 - 一眼看出最重要的东西
CONCLUSION

每个像素
都在讲故事

01

从叙事开始，不从组件开始。先问"这一页要讲什么故事"，再选骨架。

02

用眯眼测试验证层级。闭上一只眼，如果两个元素在竞争，其中一个需要缩小或褪色。

03

每页恰好一处打破。零打破是平庸，两处以上是失控。一处才叫设计。
