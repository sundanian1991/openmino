CORE ARCHITECTURE

七大核心组件

01

Terminal / REPL

命令行交互界面。支持 Bash 工具调用、文件读写、实时反馈。一切操作的入口。

02

Context Engine

上下文引擎。自动读取 CLAUDE.md、memory 文件，构建项目认知图谱。

03

Tool System

工具系统。Read/Write/Edit/Grep/Glob/Bash，六大原生工具覆盖开发全流程。

04

Skills Framework

技能框架。可复用的专业能力模块，按需加载。从设计系统到数据分析，一键调用。

05

Memory Architecture

记忆架构。跨会话持久化，MEMORY.md 索引 + L1 心跳加载，让 AI 拥有连续记忆。

06

Multi-Agent

多代理协作。主代理协调 + 子代理执行，复杂任务拆解为并行工作流。

07

MCP Protocol

模型上下文协议。标准化外部工具接入，搜索、笔记、GitHub 等能力即插即用。

WORKFLOW

五步工作流

01

配置启动

CLAUDE.md 定义项目规范，memory/ 建立认知基线。会话开始即加载上下文。

02

理解需求

分析任务意图，搜索相关技能。判断是直做还是拆解，选骨架还是选图表。

03

选择工具

匹配技能 + 工具组合。轻量操作直做，复杂任务派子代理或 Workflow。

04

执行迭代

生成内容，自检质量。眯眼测试、校验脚本、三层金字塔检查。不满意就迭代。

05

交付确认

角色卡确认，P0 红线检查。校验脚本通过后交付。记忆捕获供下次复用。
